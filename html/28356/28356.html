「青頭巾」とは何か　第二版 私見『雨月物語』論

 　　　 「 青頭巾」とは何か　第二版  

 　　　――「青」と「頭巾」の象徴についての覚え書き――  

    

 　　　　　　　　　　　　　　　　　 齋藤　吉勝  

    

 　　 　 はじめに  

    

 　上 田 秋あき 成なり の 『 雨月うげつ 物語』巻之五に収められる 「 青あを 頭巾づきん 」は、実 に 後世こうせい の読者 を 悩なや ませ る 魅み 力的りょくてき な 一篇いっぺん となっている。たとえば、鬼 と 化か し た 下野しもつけ の国（現在の栃木県）の山の僧が 、 最期さいご は 快かい 庵あん 禅師ぜんじ が 被かぶ せた「青頭巾」と 「 証しよう 道だう の歌の二句」によっ て 遷化せんげ したのか、という最大の問題があって、これについて は 管見かんけん の及ぶところ有力な定説はなく 、 本篇ほんぺん を 扱あつか う論文もまた少ないように見受けられる。  

 　そこで、かつて筆者は 「 陰陽いんよう 五行ごぎょう と『雨月物語』」な る 拙論せつろん （１） を母校の立正大学國語國文学会誌 に 掲載けいさい したが、そこでは『雨月物語』 に 投影とうえい されると思考する陰陽五行思想を、全九篇にわけ て 概説がいせつ した（表１・表２・図１・図２・図３を参照）。そして、問題の「青頭巾」においては、五行思想の 「 木もく 気き 」から解釈して巻之五を 「 木もく 之巻」 と 捉とら え、五行 ・ 九星きゅうせい の 象しょう 意い と の 共通きょうつう 項こう を 箇条書かじょうが きふう に 記き したのである。しかるにその後、『雨月物語』のある大学の研究者から、特に「青頭巾」について の 見解けんかい は興味深い、もっ と 鋭するど く 論述ろんじゅつ してほしい、との要望をいただいたので、筆者はその後数年新資料を も 探索たんさく し 、 熟察じゅくさつ してきた 。 本稿ほんこう ではその結果を 、 改あらた めて提出しようとするものである。  

    

 　さて、「青頭巾」という怪異・名短篇について、これまで筆者 を 魅了みりょう し、数十 年 悩なや ませ続け た 謎なぞ は、要約すれば次の二点 に 尽つ きる（太字・筆者）。  

    

 　一　「青頭巾」の「青」とは一体何を意味するのか。なぜ頭巾の色は「青」でなければな  

 らなかったのか。  

 　 二　「青頭巾」の「頭巾」とは何 を 象徴しょうちょう し、物語上いかなる役割 を 果は たしているのか。  

    

 　以下、本稿では右の順序 に 従したが って以前の拙論と の 重複ちょうふく もあるが、新た な 知見ちけん およ び 私見しけん を可能なかぎり客観的 に 深化しんか させなが ら 論述ろんじゅつ していきたい。  

    

   

    

   

    

   

    

   

    

   

    

    

    

    

    

 　　 　 一　「青頭巾」の「青」について  

    

 　第一に、実 に 素朴そぼく で 率直そっちょく な疑問から論を展開してみたい。  

 　『雨月物語』巻之五の第八篇「青頭巾」は、全体的なストーリーは理解できるものの、はたして表題の「青頭巾」の「青」とは一体何を意味するのであろうか 。 換言かんげん すれば、頭巾の「青」はなぜ白・黒・赤・黄など の 他た の色で は 不都合ふつごう であったのか、という疑問である。  

 　表題の「青頭巾」の「青」は 、 篇中へんちゅう の初め 「 紺あを 染ぞめ の巾」と表現される。ところが篇の後半、一年 を 経へ て も 未いま だ に 鬼き 僧そう が 「 証しよう 道だう 歌か 二句」 を 唱とな えているのを 、 快かい 庵あん が見かね て 禅杖ぜんじょう で 一喝いっかつ して鬼僧の頭を打ちおろすと、 「 忽たちま ち 氷こほり の朝日にあふがごとくきえうせて、 かの青頭巾 と 骨ほね のみぞ草葉にとどまりける」（太字・筆者） （２） と表現されるのである。一年後に紺染の頭巾が「青」に変色していたことにも秋成の細かい芸を感じるが、ここで重要なことは 、 五行ごぎょう 思想でいう「青」（五色）は、 「 木もく 気き 」 に 該当がいとう する色だということである。その「木気」の重要 な 配当はいとう を簡単 に 記き せば、「東」 「 震しん ・ 巽そん 」 「 寅とら ・ 卯う ・ 辰たつ 」（今の午前四時～午前八時ごろ。）「春」「一・二・三月（旧暦）」 「 仁じん 」 「 貌ぼう 」などである （３） 。ゆえに舞台は「下野の国」（当時の秋成が住む大坂から見てはるか東国。）なのであり 、 阿闍あじゃ 梨り が鬼となったの は 十二支じゅうにし の東北 「 丑うし ・ 寅とら 」 （ 鬼門きもん ） や 八卦はっか の 「 震しん ・ 巽そん 」（雷・風を象徴す る 俵屋宗たわらやそう 達たつ の 「 風神雷神図屛風ふうじんらいじんずびょうぶ 」の鬼型図が有名。）と「東」に関係するのである。また、阿闍梨が鬼 と 化か したのは、「十二三歳な る 童児わらは を（中略）ふか く 愛めで させたまう」ゆえ （ 愛欲あいよく ・ 男色なんしょく ＝ここでは「変態性欲」ではあるが 、 究極きゅうきょく の「仁」と解してもよい と 愚考ぐこう する。）であって 、 巻末かんまつ で「影のやうなる人」（五事の 「 貌ぼう 」の反映か。）が 「 忽たちま ち 氷こほり の朝日にあふがごとくきえうせ」たのは、木気 の 時候じこう が「寅・卯・辰」「春」「一・二・三月」と関係するからであろう。  

 　筆者はこれら の 事柄ことがら について長 年 考察こうさつ し続けているが 、 近世きんせい 当時までの思 想 潮流ちょうりゅう であった五行思想 の 観点かんてん でなければ、作者 が 創作そうさく し た 意図いと やストーリーの細かい骨組み の 配慮はいりょ は、現代の『雨月物語』の読者には理解できないのではないか、と考えているのである。また、作者秋成 も 執筆しっぴつ 当時 は 並外なみはず れた読書家 で 医師くすし でもあり 、 和漢わかん の古典はもとより 、 儒学じゅがく （中国語）や五行思想の知識 も 相当そうとう なものだったことは言うまでもあるまい。以上のような理由から、頭巾の色 は 当然とうぜん 五行思想の「木気」 の 配当はいとう である「青」でなければならず、理論上 、 他た の色では『雨月物語』「青頭巾」の「青」 の 必然性ひつぜんせい は 全まった く 生しょう じないわけである。  

    

 　本論に入る。頭巾の「青」とは何を象徴するのか、という問題である。結論を急がず、テキストを見てみよう。テキストには「氷の朝日」 の 後あと に、次のような一文が続く。すなわち、 「 現げ にも久しき念のここ に 消せう じつきたるにやあらん」である。  

 　浅野三 平 訳やく では、「本当に長い思いがここに消えつきてしまったのであろう」 （４） という 「 推定すいてい 」になっている。また、高 田 衛まもる 訳でも「まことに長い間の執念が、やっとここで消え尽くしたのであろう」 （５） と同じような解釈である。ところが石 川 淳じゅん 訳では、「げにも久しい執念のここに消えつくしたのであろうか」 （６） という「疑問」 の 意い に 解かい する。また 、 円えん 地文子ちふみこ 訳でも「まことにひさしいあいだの執念がこの一喝に消えつきたのであろうか」 （７） と、同じく「疑問」の意となっている。  

 　おそらくこの部分は 、 係かかり 助詞の「や」と文末の推量の助動詞「らむ（ん）」との判断で悩むところだろうが、筆者としては石川・円地両氏の「疑問」型 に 従したが いたいのである。なぜなら、五行説で「木気」の色は「青」（あるいは 「 紺こん 」） に 該当がいとう するのであり、その象徴するところは、まぎれもなく 《 死し か ら 生せい へ 蘇よみがえ る色》、すなわち 「 再生さいせい 」 を 暗示あんじ するからである 。 先さき ほど、木気の配当に「東」「寅・卯・辰」「春」「一・二・三月」 を 挙あ げたが、これら は 皆みな 、太陽 が 昇のぼ る方角・時間・季節などであって、陰陽でいうなら 「 陽気ようき の初め」、生命でいうなら 「 生気せいき の初め」ということになる。  

 　むか し 瀕死ひんし の人に「紺染め の 手拭てぬぐ い」 を 額ひたい に巻い て 快癒かいゆ を 祈いの ったのには、おそらくそういっ た 呪術的じゅじゅつてき な意味合いがあったからである。そして何よりも、日本の天皇の象徴する色は「紫」（青＋赤）の 「 陽気ようき 」なのだ。決して、葬式時に着用するような「白」や「黒」の 「 陰気いんき 」 を 公おおやけ のめでた い 儀式ぎしき （呪術も含めて）において身にまとうことはないのである。  

 　しかしながら、この点に関しては 、 元田與もとだよ 市いち 氏が相当考え抜いた論考を我々に提示している。一部分 を 抜粋ばっすい して紹介しておこう。  

    

 　　　　これにたいして「青頭巾」の「現にも久しき念のこゝに消じつきたるに や あら ん 」とする叙述では、二重傍線で示したとおり「や‐ん（む）」という〈疑問・反語〉の係結びが用いられているが、諸注釈にみえるとおり〈疑問〉にとるのが一般的である。それも当然のことであって、反対に〈反語〉にとった場合、調伏を完全に否定してしまうことになって、仏の道理を讃える後続文との接続のうえから文意が通らなくなるからだ。ただし、〈疑問〉にとった場合でも、調伏が完了したことが明示されるわけではけっしてない。あくまでも疑問文でしかなく、後続文との繋がりはスムーズにみえながら、なにかしら不全感をのこしてもいる （８） 。  

    

 　と、このように元田氏は「なぜ調伏完了はぼかされているのか」と疑問を明確にし、それ  

 はまたこの作品が「シニシズム」（筆者注・冷笑主義） に 繋つな がっているからではないか、と  

 いう予測を我々 に 示唆しさ している。  

    

 　ところで、氏の疑問はもっともだが、わたしの「青頭巾」の読み解きは少し氏と は 異こと なっ  

 ている。それは、係助詞「や」は〈疑問〉にも〈反語〉にも用いられるので、ここは〈反語〉  

 の可能性を も 視座しざ に入れた立場も取るべきだ、というものである。すなわち、「誠に長い間  

 の執念がここに消滅したのであろうか、いや消え た 訳わけ ではない。鬼僧はまたきっとどこかで  

 蘇る可能性がある」、という「人間の執念の不可思議さ」を強調していると思われるのであ  

 る。秋成は実にこの場面で、我々読者に「青頭巾」という《あいまい な 公案こうあん . 》を投 げ 掛か けて  

 いる、と見るべきであろう。  

    

 しかしながら、快庵は、鬼僧と化した阿闍梨が言った言葉、「師はまことに仏なり。かく浅まし き 悪業あくごふ を 頓とみ にわするべきことわり を 教をしへ 給え」 （９） とい う 懇願こんがん を受け入れたからこそ、 「 本源もと の心」へ帰らせようとして、「紺染の巾」と「証道の歌の二句」を鬼僧 に 授さず けたのであった。つまり鬼僧が「再生（蘇り）」 を 希求ききゅう したがゆえに、快庵 は 自みずか ら 被かぶ っていた「青頭巾」 を 提供ていきょう したのである。ところが一年後、鬼僧は「影のやうなる人」として証道歌二句 を 未いま だ に 唱とな え続けていた......。ここに快庵の 「 教化けうげ 」と「本源の心」に帰らせようとする指導 は 破綻はたん を 来きた した、つまりは「失敗」したのであり、鬼僧もまた最後ま で 執着しゅうちゃく か ら 抜ぬ けきれなかっ た 哀あわ れな主人公の姿 を 露呈ろてい した、とみるのである。  

    

 　以下 、 周知しゅうち の問題 に 触ふ れる。この篇で、鬼僧 は 遷化せんげ したのか、という問題である。原文には次のようにある。「かの青頭巾 と 骨ほね のみぞ草葉にとどまりける 。 現げ にも久しき念のここ に 消せう じつきたるにやあらん。たふときことわりあるにこそ。」 （ 10 ） という文章である（図４参照）。実 に 不ぶ 気味きみ である。これは、鬼僧は骨 と 化か した（＝死？）が、「青」は先述した通り「再  

    

   

    

 生」を象徴する色ゆえに、ま た 何時いつ ぞ や 何処どこ かで死肉 を 貪むさぼ り始めるのではないか、とい う 暗示あんじ を読者に印象付けているのではないかとみるのである。さらに、「たふときことわり」も同じく係助詞「こそ」の〈強意型〉で、係りの下に来るべき結びが省略されている、いわゆる《結びの省略》といわれるものが見られるのである。一般の古文の参考書 （ 11 ） によると、「......に＋係り。」の時には、下に「あり・あらむ」 を 補おぎな って訳すとあるから、「たふときことわりあ る に〇 こそ 。」→ 「 に〇 こそ ｟あれ｠」というようにラ変「あり」の 「 已然形いぜんけい 」 を 補おぎな う、という少しややこしい文になるようである。訳は「ここに誠 に 尊とうと い仏教の道理があるのである。」とするのが基本型だが、訳者によっ て 一様いちよう ではない。ちなみに、筆者・齋藤の訳は、「尊い仏教の道理は本当にここにはあるのか、ないのか。」（道理がない可能性もありうる）と、わたし は 皮肉的ひにくてき に今のとこ ろ 解かい している。  

    

 　それゆえ、今ま で 雲水うんすい の身であった快庵禅師が、最後に、この篇の舞台 ・ 大中寺だいちゅうじ に曹洞の霊場を開いたという結末部分に結び付くのである。森山重雄は「真言密教と曹洞禅との宗教的対立葛藤を想定してみることも可能であって」と解釈している （ 12 ） 。しかし筆者・齋藤には、森山氏の宗教的対立説も理解できるが、それよりも先に、快庵による鬼僧への 「 魂鎮たましず め」の必要性が大きかったせいではないか、と思えてならないのである。なぜなら、鬼僧 の 最期さいご で、草葉にとどまった「青頭巾」の「青」は「再生」を象徴する色ゆえに、この場面で は 再ふたた び鬼僧のような 「 猟奇的りょうきてき な人間」の再出現 を 封ふう じ 込こ め、時代的にも風俗・民俗的に も 鎮魂ちんこん をはかる必然性が大いにあった、と思われるからである（なお、このことは、 「 白しら 峯みね 」 の 崇す 徳院とくいん と共に、近世当時までの 「 御霊ごりょう 信仰」のあり方にも関わる大きな問題である）。  

    

 　さらに「五 行 相剋そうこく 」の関係で言えば、「青頭巾」は 「 金剋きんこく 木もく 」の理にかなった短篇である。つまり 、 薩摩さつま の国出身の快庵禅師は「金気」を帯び た 雲水うんすい であり、下野の鬼僧は東国なので「木気」ということになる。「青頭巾」は、「西（五行で金気）の僧侶が東国の鬼僧（五行で木気）に勝つ話」 と 解かい せられる。  

    

    

    

 　　 　 二　「頭巾」が象徴するもの  

    

 　第二に、「青頭巾」の「頭巾」が象徴するものについて 、 若干じゃっかん 考察してみたい 。 先行せんこう の研究書によれば、表題の「青頭巾」とは「僧侶のかぶる紺色の頭巾」 （ 13 ） とのみなっており、筆者 が 執拗しつよう なまでに興味 を 抱いだ く「頭巾」が象徴するものについて は 一切いっさい 触ふ れられていないの が 実状じつじょう である。  

 　ところが 、 閻えん 小妹しょうまい によって、篇中に登場する重要な語句「 心 放ゆる せ ば 妖魔えうま となり 、 収をさ む る 則とき は 仏果ぶつくわ を得る」 （ 14 ） という考え方は、閻氏論文「『西遊記』と秋成の文業」なる論考 （ 15 ） によって、『西遊記』がこの語句の直接の典拠なり、との判断が示されたのである。すなわち、前出の注（２）の文献によれば、「秋成 は 虞集ぐしゅう の『西遊記原序』（『通俗西遊記』宝暦八年・一七五八年刊）によって、この語を作ったと思われる。」としているのである （ 16 ） 。  

 　なるほど、このことは実に興味深いし、筆者に も 異論いろん はない。なぜなら、筆者も学生時代（一九八五年ころ）、『西遊記』と「青頭巾」の比較論文を卒論 で 扱あつか おうとしてい た 時期じき があったからである。今、この二つの小説 の 共通きょうつう 項こう を 全すべ て紹介す る 余裕よゆう はないが、ここでは「青頭巾」の「頭巾」とは一体何を象徴するのか、につい て 焦点しょうてん を 置お いているので、この一点 に 着目ちゃくもく して論を進めたい。  

    

 　『通俗西遊記』は、口木山人（西 田 維則いそく ）の訳本である。『日本古典文学大事典』の土屋順子氏の解説によれば、「中国白話小説を翻案した通俗物の一種。五編三一巻三四冊。明の呉承恩作の『西遊記』を翻案したもの」とある （ 17 ） 。なお刊行年は、（２）の「作者対照略年譜」の資料によれば、宝暦八年（一七五八）、秋成二十五歳の時で、『雨月』刊行より十八年前の作品である。  

    

 　ところで、『西遊記』は五行思想と深い関わりがある、という中野美代子の説があるが （ 18 ） 、特に本書中の「４　旅人の従者たち――桃太郎から『西遊記』まで」 （ 19 ） は 、 論ろん が 明快めいかい で 圧巻あっかん である。中野氏はまず吉 野 裕子ひろこ の桃太郎の従者たちの謎を紹介した上で 、 孫悟空そんごくう たちの役割分担 を 判わか りやすく論じている。そして「三蔵法師と桃太郎の従者たち」の論理性を、「三蔵法師の従者たちは、その役割分担といい、十二支による方位配当といい、まことに整然と論理化されているように思われる。」と主張 （ 20 ） 。最後に結論を「表」で示し、みごとに我々読者 を 納得なっとく させているのである（表３参照）。  

    

   

    

 　さて、ここで筆者 が 述の べたいのは、『雨月物語』と『西遊記』と の 因果いんが 関係である。『雨月物語』と陰陽五行との密接な関係については 、 先さき の筆者の拙論（１）で全体的に述べたつもりだが、今ここで、鬼僧 に 被かぶ せられた「頭巾」のみ に 絞しぼ って主張すれば、秋成は『通俗西遊記』をも「青頭巾」の典拠にしていたのだから、「紺染の巾」は鬼僧にとってさながら孫悟空の例の「金 を 嵌は めた頭巾」 （ 21 ） と同質のもの と 捉とら えることも可能であって、「 心 放ゆる せ ば 妖えう 魔ま と」ならぬよう仕掛けられた重要な小道具でもあったのではないか、と推測されるのである。すなわち、「鬼僧」＝「孫悟空」、「頭巾」＝「金を嵌めた頭巾」という図式がここでは成り立つのではないか 。 死し 肉にく を 喰く らう に 至いた った鬼僧は 、 天空てんくう で 大おお 暴あば れした孫悟空に見立てることもできるのではなかろうか。また、鬼僧を教化し本源の心に帰らしめんとする「頭巾」は、野性的な孫悟空の行動 を 制御コントロール させる「金を嵌めた頭巾」によ く 似に ている。短篇と長篇の違いはあるが、「頭巾」も「金を嵌めた頭巾」も登場人物 の 禁忌きんき の行動をうまく制限し 、 共とも に「人間的な振る舞い」へ と 帰納きのう させる力を持つのである。筆者・齋藤は「青頭巾」の「頭巾」が象徴するものは実はこの点 に 他ほか ならない、と、この場を借りて力説しておきたいのである。  

    

 　また、その他のものとしては、快庵が与えた証道歌の二句と三蔵法師 が 唱とな える呪文 ・ 定心じょうしん  

  真言しんごん も 近きん 似じ しているし、快庵 の 禅杖ぜんじょう と悟空 の 如意にょい 金きん 箍こ 棒ぼう も同様である （ 22 ） 。いずれにせよ、我々読者は、秋成の「青頭巾」という怪異短篇を読むにあたって、『西遊記』という長篇小説を読 み 込こ んでいなければ、その内容は深くは理解できないのではないか、 と 改あらた めて主張す る 次第しだい である。  

    

 　　 　 おわりに  

    

 　 以上、結論を簡単に記せば、「青頭巾」の「青」は陰陽五行の「青」 （ 木もく 気き ） に 該当がいとう する色で、それは「近世の《本歌取り》文学」ともいうべき多数 の 典拠てんきょ 作品群 を 擁よう しながら、さら に 二重奏にじゅうそう 、 三重奏的さんじゅうそうてき に様々 な 重層的じゅうそうてき な意味をも加え、物語中でも「再生（蘇り）」を象徴する、と思われる。と同時に重要なことは、「青頭巾」という短篇は五 行 相剋そうこく の関係 「 金剋きんこく 木もく 」で、西国出身（金気）の僧侶・快庵禅師が東国（木気）の鬼僧に勝つ話、として も 捉とら えられる。また「頭巾」については、『西遊記』のヒーロー・孫悟空の「金を嵌めた頭巾」 に 相通あいつう じ、「人間的な振る舞い」へ と 帰納きのう させる力を持つ重要な小道具であった、ということにもなる。  

    

 　なお、今回の「青頭巾」論に関しては 、 私見しけん が多いものの、なるべく秋成が生きた近世時の読みにさかのぼっ た 見解けんかい に近づいたつもりである。筆者の今後の『雨月物語』研究としては、本論で考察した論を深めて、陰陽五行思想を中心とした、やはり近世という時代の思 想 潮流ちょうりゅう や歴史観 に 沿そ っ た 側面そくめん をさら に 掘ほ り 下さ げてみたい、と考えている。  

 　 諸先学しょせんがく の、 ご 教示きょうじ を願う。
