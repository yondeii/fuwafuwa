ど う い う こ と か と い う と 、 女 性 に 「 女 心 （ 自 分 の こ と ） を 理 解 し て く れ て い る 」 と

１ モ テ る 男 性 と は

と に か く ご ち ゃ ご ち ゃ と 文 句 を 言 わ な い 、 だ ら し な く な い 、 男 は 黙 っ て い て も モ テ ま す 。

一 番 嫌 な の は い や み 、 い じ わ る 系 の 男 の 人 で す 。

一 緒 に い た く な い ど こ ろ か 、 で き れ ば 会 い た く な い 、 話 を し た く な い 、 電 話 も し た く な い と な っ て し ま い ま す 。

で す の で 細 か い こ と を び っ し り と 並 べ て が ち ゃ が ち ゃ 要 求 し て こ な い だ け で も だ い ぶ 器 が 大 き く 感 じ る の で い い ん で は な い か と 思 い ま す 。

あ と そ れ な り に 場 の 雰 囲 気 を 和 ら げ る タ イ プ の 適 当 な 冗 談 （ い や ら し く な い 、 下 品 じ ゃ な い 、 面 白 く な さ す ぎ な い 、 オ ヤ ジ す ぎ な い も の ） を さ っ ぱ り と 言 え る 感 じ だ と 好 感 度 は 上 が る ん で は な い か と 思 っ て い ま す 。

モ テ る 男 性 の 性 格 は 幅 広 い の で 絶 対 こ れ が 正 し い と い う の は な い か も し れ ま せ ん け れ ど も ね 。

と り あ え ず は 男 っ ぽ い 、 真 か ら 男 気 の あ る さ っ ぱ り 、 も し く は 強 い 系 も た く ま し さ が 全 面 に 出 て い て 結 構 い い ん じ ゃ な い か と 思 っ て い ま す が 、 繊 細 で や さ し い 雰 囲 気 が す る 人 も 一 緒 に い て 安 心 ま で で き て し ま っ て 、 モ テ る 度 合 い は ど ち ら で も お よ そ ４ ０ パ ー セ ン ト く ら い は 高 く な っ て い き ま す 。

あ と は 頼 り が い の あ る と こ ろ 、 腹 が 太 い 、 太 っ 腹 な 男 の 人 た ち は 一 応 あ ん な の 子 や 女 性 た ち の 心 の 中 の 恋 人 候 補 リ ス ト 入 り を 果 た す の で は な い か と 思 っ て い ま す 。 と に か く 最 初 か ら 完 璧 を 目 指 す の は 難 し い か も し れ ま せ ん 。

少 し で も リ ス ト に 入 る た め の 努 力 を 普 段 か ら 心 が け て み る と 、 自 分 の 性 格 や 中 身 の 飛 躍 的 な 向 上 に つ な が り ま す の で そ れ が 自 然 な 自 分 の 心 や 考 え 方 の 一 部 に な っ て し ま っ ち ゃ え ば そ の 時 点 か ら は 既 に も う 根 っ か ら の モ テ る 性 格 の 男 性 像 と 変 わ っ て い

く こ と が で き る よ う に な り ま す 。

あ と は 正 し さ を 求 め る 性 格 に な る よ う に 心 が け て み ま し ょ う 。

正 し さ 、 つ ま り 正 義 に 勝 て る 純 白 な キ ャ ン バ ス は あ り ま せ ん 。

そ の 少 年 の よ う な 純 粋 な 瞳 、 い や ら し さ の か け ら も な い ピ ュ ア の 心 や 性 格 の 持 ち 主 は ロ リ 女 で し ょ う が 、 J K で し ょ う が 、 女 子 大 生 で し ょ う が 、 熟 女 で し ょ う が 老 女 で し ょ う が 関 係 な く 安 心 し て 寄 っ て い き ま す 。 自 分 の 心 と 性 格 を 洗 い 流 せ る タ イ プ の 良 い 趣 味 や ア ク テ ィ ビ テ ィ を 見 つ け 出 し て み ま し ょ う 。 そ こ か ら 自 分 の 性 格 の と げ や 角 を 削 ぎ 落 と し て い く ん で し ょ う ね 。

こ れ は 女 子 に と っ っ て も 使 え る 、 必 要 な こ と で す 。

両 方 が モ テ る 性 格 の な っ て し ま え ば そ の 人 た ち は ど ん ど ん リ ス ト 入 り 、 殿 堂 入 り パ ー セ ン テ ー ジ が 上 昇 し て い き ま す 、 あ と モ テ る 男 性 の 性 格 は 親 切 な 方 が 多 い で す 。

ち ょ っ と 気 の 利 い た 人 助 け は 男 女 と も に 弱 い も の で す 。

２ 態 度 に 表 れ る 余 裕

モ テ る 男 性 の 態 度 は 、 最 初 か ら が っ つ り ア ピ ー ル す る の で は な く 、 ど こ か 余 裕 の 表 情 が あ り ま す 。

近 年 、 草 食 系 男 子 と い う 言 葉 を よ く 耳 に し ま す が 、 そ れ は 見 た 目 や し ゃ べ り 方 と み て 態 度 は よ く 観 察 し な い と わ か り ま せ ん 。

例 え ば 、 飲 み 会 や 合 コ ン に 行 っ た 場 合 そ の 中 の 1 人 、 2 人 、 モ テ そ う な 男 性 が い て 、 外 見 か ら 入 る 女 性 も 多 い で す が 顔 で は な く 、 実 際 話 し た 時 に 女 性 の 目 を し っ か り み て 話 し 、 自 分 の 主 張 は し ま せ ん 。

人 間 誰 で も 、 自 分 の 行 動 や し ゃ べ る 癖 、 同 じ 考 え を 持 っ て る 人 に 好 意 を 抱 い て し ま い が ち で す 。

そ こ を 分 か っ て い る 上 で 狙 っ て い る 女 性 に う ま く 話 を 合 わ せ て み る と か 、 逆 に 女 性

と 話 し た い と 思 っ て も わ ざ と 素 っ 気 な い 態 度 を と る 男 性 も 独 特 な 雰 囲 気 で 女 性 の 目 を 引 き ま す 。

プ ラ イ ベ ー ト に 関 わ ら ず 仕 事 で も 皆 が 嫌 う 仕 事 を 率 先 し て 行 い 、 一 生 懸 命 働 く 姿 や 、 部 下 が 仕 事 で ミ ス を し た 時 に も 責 め る だ け で は な く 、 一 緒 に 上 司 に 頭 を 下 げ て く れ る ほ ん の さ さ い な 態 度 で も 、 女 性 の 評 価 を 上 げ る と 考 え ま す 。

本 当 に モ テ る 男 性 は 、 自 分 が モ テ る 人 間 だ と 自 覚 し 行 動 に 移 し て い る の か 全 て は わ か り ま せ ん が 、 も し 全 て 自 覚 が な い の で あ れ ば 生 ま れ つ き の 物 か 、 世 の 賛 成 に 羨 ま し が ら れ る 存 在 で す ね 。

話 が 逸 れ て し ま っ た か も で す が 、 モ テ る 男 性 の 態 度 は 目 で 見 え る 態 度 だ け で は な い で す 。

今 や 世 間 に 普 及 し て い る S N S 、 L I N E な ど 便 利 な も の が た く さ ん あ る 世 の 中 で 普 通 の 男 性 と モ テ る 男 性 の 差 が つ い て し ま う と 感 じ ま す 。

L I N E で や り と り す る 時 は 、 先 ほ ど も 記 し た 通 り 自 分 を 主 張 し す ぎ ず 、 か と い っ て 会 話 が な い わ け で も な く 、 女 性 の テ ン ポ に 合 わ せ て や り と り し て く れ る の で 、 楽 で し ょ う 。 も し 仕 事 で 返 信 が 遅 れ て し ま っ た ら 必 ず 文 頭 に 謝 罪 の 言 葉 を 入 れ た り 相 手 を 気 遣 い ま す 。

大 人 に な れ ば 普 通 に 謝 っ た り 、 感 謝 を 述 べ た り す る も の で す が 今 は 画 面 上 で や り 取 り す る こ と が 多 く な る 一 方 で 、 顔 が 見 え な い か ら と 、 も し 返 信 が 遅 れ た と し て も 何 も 言 わ な い 男 性 が 増 え て い る よ う で な ら な い で す 。

む し ろ そ れ が 当 た り 前 に な っ て し ま い が ち な の で 、 一 言 謝 罪 の 言 葉 を 入 れ る 男 性 が 新 鮮 で 徐 々 に モ テ る 要 素 を 醸 し 出 し て い る か も し れ ま せ ん 。

こ こ ま で 言 う と 、 態 度 で は な く モ テ る 男 性 の 性 格 と 捉 え ら れ て し ま う か も し れ ま せ ん が 、 考 え た り 思 う こ と と 実 際 行 動 に 移 す の は 大 違 い で す 。

素 直 に あ り の ま ま の 自 分 の 行 動 、 態 度 を 表 す 男 性 は モ テ ま す 。

女 性 か ら 見 て 自 分 に は 持 っ て い な い も の を す ん な り こ な す 異 性 は よ り 魅 力 的 に 見 え 、 こ の 人 と も っ と 近 づ き た い と い う 想 い が 出 て く る と 思 い ま す 。

モ テ る 男 性 は 、 周 り の 男 子 か ら も 好 印 象 で き っ と し れ て な く 、 魅 力 的 な 存 在 だ と 感 じ ま す 。

３ モ テ る な と 納 得 し た 男 性 の 特 徴

モ テ る 男 性 の 性 格 を 私 な り の 概 念 で 見 つ け た と 思 い ま す 。
