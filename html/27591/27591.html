２ ０ 代 の 転 職 の 実 情

こ の よ う な こ と は 都 市 伝 説 で あ り 、 意 味 合 い と し て は 転 職 活 動 に お い て は ３ ５ 歳 を オ ー バ ー す れ ば 転 職 が も う で き な い と い う よ う な も の で す 。

３ ５ 歳 で 一 律 に 転 職 が で き な い と い う こ と で は あ り ま せ ん が 、 転 職 の 市 場 に お い て は 即 戦 力 が 要 求 さ れ る の で 、 年 齢 に よ っ て 転 職 の 方 法 や 難 易 度 が あ る 程 度 変 わ る の は 間 違 い な い で し ょ う 。

２ ０ 代 が 転 職 す る 理 由

ま た 、 実 際 に 会 社 に 入 っ て み る と 考 え て い た も の と 違 っ て い た 、 な ど と い う よ う な ミ ス マ ッ チ も 多 く あ る よ う で す 。

日 本 の 現 在 の 新 卒 を 採 用 す る 方 法 が 、 学 生 と 会 社 の ミ ス マ ッ チ を 構 造 的 に 引 き 起 こ す よ う に な っ て い る の で 、 仕 方 が あ る 程 度 な い か も し れ ま せ ん 。

ま た 、 転 職 す る 理 由 と し て は 、 全 て の 世 代 に 共 通 す る 、 仕 事 内 容 や 対 人 関 係 が 合 わ な い 、 と い う も の が よ く あ り ま す 。

職 場 で ず っ と 逃 げ た い と い う よ う な 気 持 ち で 仕 事 を し 続 け る の は 、 非 常 に 辛 い で し ょ う 。

し か し 、 転 職 を こ の よ う な 理 由 の み で す る の は 、 非 常 に リ ス ク が あ り ま す 。

と い う の は 、 ど の よ う な 仕 事 の 場 合 で も 、 辛 い と こ ろ は 少 な か ら ず あ る の で 、 こ の こ と が 転 職 す る こ と に よ っ て 無 く な ら な い と い う こ と が あ る た め で す 。

● ２ ０ 代 が 積 極 的 に 転 職 す る 理 由

２ ０ 代 が 転 職 す る 理 由 と し て は 、 先 に ご 紹 介 し た よ う な 消 極 的 な も の と は 違 っ て 、 次 の よ う な 積 極 的 な も の が 結 構 多 く あ り ま す 。

・ 転 職 す る 際 は ２ ０ 代 の 方 が チ ャ レ ン ジ し や す く 有 利 で あ る

転 職 の 市 場 に お い て は 、 一 般 的 に 若 い 人 ほ ど 有 利 に な る と 言 わ れ て い ま す 。

と い う の は 、 企 業 の 多 く は 、 第 一 に 育 て や す い 若 い 人 材 を 希 望 し て い る た め で す 。

仕 事 を 世 の 中 で し た 経 験 が 無 い 人 を 歓 迎 す る と い う よ う な 求 人 の 場 合 は 、 ほ と ん ど は 若 い ２ ０ 代 の 高 い 学 習 意 欲 が あ る 人 の た め の も の で す 。

ま た 、 人 生 の 大 き な イ ベ ン ト で あ る 転 職 に 対 す る 不 安 や リ ス ク は 、 若 い 人 の 方 が 多 く な い こ と が 理 由 と し て 挙 げ ら れ ま す 。

・ キ ャ リ ア ア ッ プ し た い

将 来 的 に 自 分 が ど の よ う に な っ て い た い か を 考 慮 し た 際 に 、 そ こ に 現 在 の キ ャ リ ア の み で は 到 達 で き な い と 考 え て 、 キ ャ リ ア ア ッ プ す る た め に 転 職 し た い と 思 う 人 が 多 く い ま す 。

２ つ の Ｉ Ｔ と 人 材 な ど と い う よ う な ジ ャ ン ル に お い て 強 み が あ れ ば 、 差 別 化 が 他 の 人 と で き る た め 転 職 の 市 場 に お い て 高 い 価 値 に な る 場 合 が 多 く あ り ま す 。

そ の た め 、 転 職 を こ の よ う な 理 由 で 行 う 場 合 は 、 転 職 を 異 業 種 へ 行 う ケ ー ス が 多 く あ っ た り し ま す 。

ま た 、 ス キ ル ア ッ プ す る た め に 、 業 種 が 同 じ 場 合 で も 高 い 給 料 の と こ ろ 、 案 件 の 大 き な も の が あ る と こ ろ な ど を タ ー ゲ ッ ト に し て 転 職 し た い と 思 う 人 も 多 く い ま す 。

・ 転 職 が 人 生 設 計 に お い て 必 要 で あ る

社 会 人 に な れ ば 、 子 供 を 作 る た め に 結 婚 す る よ う に な り ま す 。

そ の た め 、 自 分 の 将 来 的 な こ と に つ い て も 、 就 活 の 時 期 に は 考 え な く て も 考 慮 す る よ う に な る で し ょ う 。

こ の よ う な 場 合 は 、 現 在 の 仕 事 の 方 法 で は 都 合 が ど う し て も 良 く な い こ と が あ り 、 転 職 し た い と 思 う こ と が 多 く あ る よ う で す 。

２ ０ 代 の 転 職 の メ リ ッ ト

社 会 で 仕 事 を 実 際 に し て み た と こ ろ 、 学 生 時 代 よ り も 自 分 が 希 望 す る 業 界 ・ 企 業 ・ 仕 事 な ど が は っ き り し て 、 軌 道 修 正 を 転 職 す る こ と に よ っ て 図 ろ う と し ま す 。

す ぐ に 転 職 を 決 心 す る の は 早 過 ぎ る の で は な い か 、 も と も と 仕 事 は 辛 い も の で あ る た め 我 慢 す べ き で あ る と 考 え る 人 も 中 に は い る で し ょ う 。

し か し 、 ２ ０ 代 だ け が 転 職 が 異 職 種 ・ 異 業 界 に で き ま す 。

３ ０ 代 前 半 の 場 合 で も 、 異 職 種 ・ 異 業 界 に ま れ に 転 職 で き る 人 も い ま す 。

し か し 、 雇 用 す る 会 社 と し て も 、 仕 事 を 経 験 が 無 い 人 に 任 せ る た め に は 、 年 齢 の 上 限 が あ る 程 度 あ る と 思 っ て い る 場 合 が ほ と ん ど で す 。

新 卒 で 入 っ た 社 員 の 経 験 と 年 齢 の バ ラ ン ス と 、 中 途 の 外 か ら 雇 用 し た 人 の 経 験 と 年 齢 の バ ラ ン ス を 考 慮 し て も 、 判 断 と し て は 妥 当 な も の で し ょ う 。

実 際 に 自 分 に 適 さ な い 仕 事 を 継 続 し て い つ ま で も 悩 ん で い る の で あ れ ば 、 見 切 り を 早 く つ け て 、 や り が い を 本 当 に 自 分 が 感 じ る 仕 事 を し て 、 経 験 を 積 み 重 ね る 方 が い い こ と も あ る で し ょ う 。

２ ０ 代 の 転 職 で 注 意 す る こ と

し か し 、 辛 抱 し て で も ３ 年 間 程 度 は 最 初 に 入 っ た 会 社 で 仕 事 を す る の が お す す め で す 。

３ 年 間 辛 抱 す る と 、 経 験 や ス キ ル も あ る 程 度 マ ス タ ー で き る し 、 そ の 上 で 自 分 が 希 望 す る の は や は り 現 在 の 仕 事 で は な い と 考 え る 場 合 は 、 転 職 を 考 え て も い い の で は な い で し ょ う か 。

雇 用 す る 会 社 と し て も 、 ３ 年 間 辛 抱 し た 人 に 対 し て は 、 ス キ ル や 経 験 だ け で な く 、 頑 張 れ る 人 で あ る と い う 評 価 を す る 場 合 が 多 く あ り ま す 。

● ２ ０ 代 で 転 職 の 経 験 が 無 い 場 合 で も 問 題 な い か ？

転 職 す る 場 合 は 、 現 在 自 分 が 仕 事 を し て い る 職 種 や 業 界 以 外 で も 問 題 な い か と 心 配 す る 人 も い る で し ょ う 。

経 験 が あ ま り な く 、 地 位 な ど が ま だ 無 い 人 で も 、 現 在 の 仕 事 と は 違 っ た 業 界 に 転 職 す る こ と は で き ま す 。

若 手 で 新 卒 し た 後 １ 年 目 ～ ３ 年 目 や 第 二 新 卒 の 場 合 は 成 長 す る 可 能 性 や や る 気 が 大 き い た め 、 新 卒 よ り も 好 ま れ る 場 合 が 多 く あ り ま す 。

自 分 が 目 指 す 目 的 や 方 向 性 が 就 活 し た 際 よ り も 明 確 に な っ て き て い る 場 合 も あ り 、 マ イ ナ ス 思 考 に 経 験 が 無 い と い う こ と で も な ら な い よ う に し ま し ょ う 。

ま た 、 雇 用 す る 会 社 と し て も 、 こ の よ う な こ と を 考 慮 し て 評 価 す る 場 合 も あ り ま す 。

ど の よ う に 自 分 が し て い き た い か と い う こ と を 、 強 く 意 識 す る こ と が 大 切 で す 。

● 第 二 新 卒 は 再 チ ャ レ ン ジ が で き る

第 二 新 卒 は 、 は っ き り し た 定 義 が あ る と い う こ と で は あ り ま せ ん が 、 新 卒 し て か ら 基 本 的 に １ 年 目 ～ ３ 年 目 の 人 で す 。
