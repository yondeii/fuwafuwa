 「キャッシュフローゲームをやってみたい」

キャッシュフローゲームを通して気づくべき事とは？

    

 私は今までに、キャッシュフローゲームを１０００回以上やってきました。数多くキャッシュフローゲームを経験してきた中で、ある重要なことに気づきました。  

    

 それは、キャッシュフローゲームでしか体験できない、右クワドラントを目指す人が必要な教育です。この項では、キャッシュフローゲームでしか体験できない金持ち父さんを目指す人のための重要な教育についてお話します。  

    

 私は、以前はサラリーマンつま りE クワドラントとして十数年働いていました 。E クワドラントに数十年もいると、左側クワドラントの人たちが身に付けてしまっている考え方やマインドが無意識にしみついてしまいます。  

    

 Ｅクワドラントの人たちの考え方・テーマは何だったでしょうか？  

    

 そうです、「安定」や「安心」です。  

    

 安定を目指して一所懸命に働きます。保障がなければ思いきった行動ができません。リスクがある事に挑戦することができません。これはＥクワドラントの人たちのテーマであって宿命です。しかたがありません。  

    

 「安心」や「安定」をテーマに持ってはいけないと言っているわけではありません。問題は、もし、左側のクワドラントから右側のクワドラントへ目指す場合には、邪魔になる考え方ということなのです。  

    

 私たちはすでに、右側のクワドラントになるためには何をすればいいのかはわかっています。お金持ちになるには、どのような考え方を持って行動すればいいのかをすでに知っています。  

    

 なぜなら、３０冊も以上も出版されている「金持ち父さん」シリーズを数冊も読めば、本の中にお金持ちになるにはどうしたらいいのか？金持ち父さんの教えが詳しく書いてあるからです。  

    

 私たちは、お金持ちになるにはどうしたら良いのかは分かっています。しかし、実際にお金持ちになれる人は一握りの人です。  

    

 なぜでしょうか？  

    

 これには、いくつかの理由はあります。１つは、左側のクワドラントでの生活が長過ぎるために、左側のクワドラントの生活が当たり前になっているのです。  

    

 左側のクワドラントの人と右側のクワドラントに人では、マインドが違います。使っている言葉も違います。  

    

 右側のクワドラントに行くには、マインドを右側のクワドラントの人のマインドに変え　なければなりません。言葉も、右側のクワドラントの人が使っている言葉に変えなくてはなりません。  

    

 でも、それがなかなかできないのです。それは左側クワドラントでの生活が長いために、安定や安心など のE クワドラントのマインドがすでに染み付いているからです。  

    

 では、どうすればいいのか？  

    

左クワドラントと右クワドラントのマインドの違いとは？

    

 マインドとは心構えです。自分が信じている考え方です。そのマインドを変えるには、それなりの期間やきっかけが必要です。

キャッシュフローゲームで体験するべき疑似体験とは？

 「本当にこのペースでラットレースが脱出できるのだろうか？」と不安になることがあります。  

    

 例えば、キャッシュフローゲームのプレイ時間を２時間とします。スタートしてから１時間３０分ぐらいまでは、「本当にラットレースを脱出することができるのだろうか」というような不安の状態が続きます。  

    

 ところが、１時間３０分が過ぎたぐらいに、唯一持っていた不動産が高額で売れたりします。すると、たくさんの現金が手元に入ってきます。

 ゲームをスタートしてから４分の３くらいまでは、「本当にラットレースを脱出できるのか？」と不安だったのが嘘のよう、一揆にラットレースを脱出してしまいました。

 この、普通では体験できない感覚を、キャッシュフローゲームで疑似体験できます。現実でもビジネスや投資で体験することはできますが、最初から現実でトライして失敗したら大変です。場合によっては立ち直れないかもしれません。  

    

 まずは、キャッシュフローゲームで何度も疑似体験してこの感覚を身に付けることが、キャッシュフローゲームにしかできない貴重な体験学習の１つなのです。

 いろいろな所で開催されているキャッシュフローゲーム会に参加している経験者さん（だいたい経験数が３回以上の人）は、純粋にキャッシュフローゲームが好きな人もいますが、何かの営業をしたり勧誘することが目的で参加する人が多いです。  

    

 残念なことですが、現実はそうでした。同じキャッシュフローゲーム会の参加者に対して何かの営業をしたり勧誘することが目的で参加します。  

    

 そして、初心者向けキャッシュフローゲーム会の主催者も、参加者の人に対して、何かの営業をしたり勧誘するのが目的でキャッシュフローゲーム会を主催しています。  

    

 と言うことは、何かの営業をしたり勧誘することが目的で参加する経験者と主催者の目的がバッテイングするのです。  

    

 自分が何かの営業をしたり勧誘することが目的でキャッシュフローゲーム会を主催しているのに、経験者に横取りされることを避けたいためなのです。  

    

 だから、初心者向けにして、勧誘や営業活動をする恐れのある経験者を締め出しているのです。  

    

２．有料広告を出している

    

 日本全国で行われている、ほぼ全てのキャッシュフローゲーム会に参加してわかったことは、キャッシュフローゲーム会を主催する目的にはたいていは何かの営業か勧誘が目的でキャッシュフローゲーム会を主催しています。  

    

 キャッシュフローゲーム会に限らず、世の中、ほとんどの業種でそのような流れになっています。例えば書籍やセミナーもその後に何か他の高額な商品、例えば高額スクールや高額なコンサルティンブなどを売るために布石になっています。  

    

 これはそういう仕組みなので仕方がないと思います。しかし、参加 費 50 0 円のキャッシュフローゲーム会の募集で有料広告を出していると言うのはあきらかにその後高額の何かを販売することが目的です。  

    

 ＰＰＣ広告などグーグルやヤフーの検索エンジンの検索結果の上部のスポンサー広告を出しているキャッシュフローゲーム会はその傾向が強いのです。  

    

 なぜなら、例えば「金持ち父さん」とか「キャッシュフローゲーム」と言うキーワードで、継続的にグーグルやヤフーでＰＰＣ広告を出稿するには、月 額 2 0 万円 ～ 3 0 万円もの費用がかかります。  

    

 と言うことは、１ヶ月 に 2 0 万円 ～ 3 0 万円もの広告費を払っても、回収できるという試算があるからできることなのです。キャッシュフローゲーム会の参加費 は 50 0 円と格安です。純粋にキャッシュフローゲーム会の参加費だけでは、それだけの広告費を賄いきれないのです。  

    

３．他のキャッシュフローゲーム会を誹謗中傷

    

 他のキャッシュフローゲーム会についての誹謗中傷しているキャッシュフローゲーム会の主催者は不誠実です。  

    

 私が以前キャッシュフローゲーム会を主催していた時に、他のキャッシュフローゲーム会から名前を好評されて誹謗中傷されました。まったく事実に反することをツラツラ書かれました。あまりに酷かったので削除依頼を出しました。  

    

 なぜ、そのキャッシュフローゲーム会は、他のキャッシュフローゲーム会を誹謗中傷するのでしょうか？  

    

 理由は２つあります。  

    

 １つは、脅威に感じているからです。叩く対象が怖いのです。弱い犬ほどよく吠えるといいます。弱い犬は自分より大きい強そうな犬が怖くて吠えます。実際に戦うと負けるから威嚇しているのです。  

    

 もう１つは、自分が何かの勧誘目的でキャッシュフローゲーム会を開催しているから、他のキャッシュフローゲーム会に参加者を取られたくない。だから先に悪評を流しておいて自分のゲーム会が健全のように見せかけているのです。  

    

 でも、人の悪口を言う人、誹謗中傷する人で、まともな人はいません。自分が不健全だからそのような行為をするのです。  

    

 実際に主催者に会って接してみるとその人の人柄はわかりますが、そのようなキャッシュフローゲーム会には参加したくないですよね。  

    

４．主催者が不誠実

    

 私の友人や知り合い数名が、他のキャッシュフローゲーム会を誹謗中傷しているキャッシュフローゲーム会に参加したことがあります。  

    

 そのキャッシュフローゲーム会は、初心者向けとうたっているのですが、主催者の態度が酷かったそうです。  

    

 「何で、そんなことも分からないんだ！」とか  

 「もう、来るな！」  

    

 などと主催者から言われたそうです。これを言われたらかわいそうです。初心者向けと書いてあったから参加したのに、ゲームシートの書き方を丁寧に教えてくれなかったそうです。  

    

 キャッシュフローゲーム会の運営も、その主催者の人格が出てしまいます。  

    

５．主催者がＥクワドラント・Ｓクワドラント

    

 ラットレースを脱出することが目的のキャッシュフローゲームなのに、運営している主催者がラットレースを抜けていない人がいます。  

    

 キャッシュフローゲームは、単なるボードゲームですが、現実社会をかなりリアルに反映しています。  

    

 キャッシュフローゲームでラットレースを脱出するために必要な考えや知識は、現実社会でもまったく同じです。  

    

 ラットレースを抜け出していいない運営者ですと、キャッシュフローゲームで気づける貴重な気づきポイントの解説ができません。ただゲームをするだけの気づきの無いキャッシュフローゲーム会になります。  

    

 以上の５つのポイントに気を付けて、あなたに合ったキャッシュフローゲーム会を選んでください。
