● 蟹 座 の 女 性 の 恋 愛 特 徴 ポ イ ン ト （ ３ ）

　 蟹 座 は 好 き 嫌 い が 激 し い の が 特 徴 。 好 き な 人 と は 深 く 長 く つ き 合 っ て い く こ と が で き ま す 。 家 族 や 友 人 な ど 、 親 し い 人 を 気 遣 い 、 大 切 に す る 性 質 を 持 っ て い る の で 、 そ れ を 非 難 し た り 傷 つ け る 者 は 許 せ ま せ ん 。 た と え 恋 人 で も 、 一 旦 「 嫌 い 」 と 思 っ た 瞬 間 に 相 手 の す べ て を 嫌 い 、 別 れ よ う と す る で し ょ う 。 そ の た め 、 恋 愛 か ら 遠 ざ か る 場 合 も 少 な く は あ り ま せ ん 。

　 ま た 蟹 座 は 観 察 眼 が 鋭 く 、 人 の 真 似 、 模 倣 が 得 意 で す 。 記 憶 力 が 明 確 で 思 考 回 路 が 常 に 働 い て い る の で 、 魅 力 的 な も の は ぜ ひ 積 極 的 に 真 似 て み ま し ょ う 。
