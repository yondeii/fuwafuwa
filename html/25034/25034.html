「 〇 〇 さ ん の ツ イ ー ト い つ も 面 白 い で す ！ 気 に な り す ぎ

て D M し ち ゃ い ま し た ( 笑 ) 」

い つ も だ っ た ら D M な ん て ほ と ん ど 読 ま な い が 、 こ う し た 誰 か が ち ゃ ん と 書 い て 送 っ て く れ た メ ッ セ ー ジ に は 必 ず 返 信 す る よ う に 心 が け て い る 。

「 あ り が と う ご ざ い ま す ！ 」

気 に な っ て 彼 女 の ア カ ウ ン ト を 見 て み る と 、 「 ミ サ キ @ 丸 の 内 系 女 子 」 と 書 か れ て い た 。 フ ォ ロ ワ ー の 数 も 多 く 、 ど う や ら 彼 女 の S N S 界 隈 で は 少 し 名 の 知 れ た 人 間 で あ っ た 。 顔 は 隠 し て い た が 、 何 故 か 彼 女 の こ と が 気 に な り 、

過 去 の 画 像 付 き の ツ イ ー ト を 見 て い る と 口 元 を 隠 し て は い た が 、 少 し ば か り 顔 が わ か る 写 真 を 発 見 し た 。

「 め っ ち ゃ 美 人 な 気 が す る ・ ・ ・ 」

急 に 彼 女 に 返 し た D M を 適 当 に 書 い て し ま っ た こ と を 反 省 し た が 、 す ぐ に ま た 彼 女 か ら の 返 信 が く る 。

「 〇 〇 さ ん の ツ イ ー ト な ん と な く 私 と 属 性 が 似 て い る か ら 、 勝 手 に 親 近 感 持 っ ち ゃ っ て ま し た ( 笑 ) 」 。

実 は 、 僕 の S N S は た ま に 女 性 の 恋 愛 に 対 す る 毒 を 吐 く ツ

「 次 の 返 信 は 少 し 気 合 い 入 れ る か 」 と 心 の 中 で 決 め て 返 し た 。

「 確 か に 同 じ で す ね ！ T w i t t e r で こ う し て 連 絡 く る の 初 め て だ っ た ん で な ん だ か 嬉 し い で す ！ 」

今 の 僕 に は 最 大 限 の メ ッ セ ー ジ を 送 っ た と 思 う 。

す る と 、 彼 女 か ら の 返 信 で 「 〇 〇 さ ん の ツ イ ー ト 秀 逸 だ

し 結 構 当 た っ て る し 、 い ず れ 会 っ て み た い な 〜 」 と 誘 う よ う な そ ぶ り の 返 信 が き た 。 こ の 場 合 ほ と ん ど は 「 じ ゃ あ 連 絡 先 交 換 し ま す か ！ 」 と な る こ と が 多 い 。

彼 女 に 限 ら ず と も 、 女 性 か ら 男 性 に 連 絡 先 を 聞 け る 人 は 相 当 肝 が 座 っ て い る 。 ほ と ん ど の 女 性 は 男 性 か ら 聞 か せ る よ う な 誘 導 尋 問 の よ う な 誘 い で 、 男 性 か ら 女 性 に 連 絡 先 を 交 換 す る よ う に 促 す 。 今 回 の 彼 女 の メ ッ セ ー ジ も そ れ と 全 く 同 じ も の だ っ た 。 こ れ が 、 合 コ ン や ナ ン パ な ら ス ム ー ズ に 聞 い た り 、 教 え た り で き る が S N S の 場 合 は 少 し 違 う 。

昨 今 で は い ろ ん な 犯 罪 や 個 人 情 報 流 出 な ど の 危 険 性 が あ る た め 、 そ う い っ た も の に 警 戒 心 が 高 い 僕 と し て は S N S を 通 じ て の 連 絡 先 交 換 に は 多 少 な り と も 抵 抗 が あ っ た 。 だ け ど 、 彼 女 が 本 当 に 存 在 す る 人 で あ れ ば 美 人 な 上 に 、 現 実 世 界 の 僕 で は 出 会 え な い だ ろ う と 思 っ た 。

「 こ れ も チ ャ レ ン ジ だ 」 そ う 決 意 し て 、 彼 女 に 連 絡 先 交 換 を し な い か 聞 く 。

す る と 、 「 会 っ て か ら で も 良 い で す か ！ ？ ち な み に 明 日 の 夜 と か っ て 暇 し て ま す ？ 」

思 わ ぬ 返 し に 少 し ば か り び っ く り し た が 、 す ぐ に 返 信 を し て 彼 女 の 誘 い に の る こ と に し て み た 。

・ S N S で の 連 絡 先 交 換 は リ ス ク と 出 会 い の 駆 け 引 き

で す 。 そ の た め 私 が 皆 さ ん に お す す め す る の は 「 L I N E の 連 絡 先 交 換 は で き れ ば し な い こ と 」 で す 。 こ れ は 女 性 た ち に 配 慮 す る わ け で も な く 、 自 ら の 個 人 情 報 流 出 を 防 ぐ 意 味 合 い も あ り ま す 。 L I N E の 方 が 気 軽 に 連 絡 が 取 り や す い と 答 え る 人 は 多 い か も し れ ま せ ん が 、 実 際 に 会 っ て 仲 良 く な る ま で 相 手 の 情 報 は 何 も わ か り ま せ ん 。 さ ら に は 、 彼 女 た ち の 中 に も 「 プ ラ イ ベ ー ト は 知 ら れ た く な い 」 と い う 人 も 多 く い ま す 。 裏 垢 女 子 で オ フ パ コ を す る 人 の 中 に は 平 気 で 彼 氏 が い た り す る こ と も あ る の で 、 無 理 な L I N E な ど の 交 換 は 控 え ま し ょ う 。

2 章 「 じ ゃ あ オ フ パ コ す る ？ 」 か ら 始 ま る ベ ッ ト イ ン

「 い や 〜 こ ん な 綺 麗 な 景 色 が 見 れ る な ん て す ご い ね ！ 」

「 で し ょ ！ こ こ に 来 た 人 た ち は み ん な そ う 言 っ て い く わ ！ 」

彼 女 へ の 謎 は 深 ま っ て い く ば か り だ が 、 「 な ん か 飲 む ？ 」 と 聞 か れ て 「 じ ゃ あ お 茶 で ！ 」 と 答 え る 。

「 お 酒 飲 ん だ ら 勃 た な く な る の ？ 」 と ま た し て も 爆 弾 発 言 が 飛 ぶ 。

お そ ら く 彼 女 は も う し た く て し た く て た ま ら な い ん だ ろ う 。

ふ と 後 ろ に 回 っ て 彼 女 の 腰 か ら 上 半 身 へ と 舐 め 回 す よ う に 触 っ て い き 後 ろ か ら 軽 い ハ グ を す る 。

「 あ ん あ ん あ ー ん ♡ は ぁ ぁ 。 や ば い よ う ♡ 気 持 ち い い 」

彼 女 は 「 〇 〇 さ ん と オ フ パ コ し ち ゃ っ た ね 」 と 言 わ れ た と こ ろ で 、 僕 は 「 こ れ が オ フ パ コ か 」 と 実 感 す る 。
