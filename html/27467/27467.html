４ ０ 代 男 性 が モ テ る 方 法

４ ０ 代 に 突 入 し て か ら 突 然 自 信 が な く な っ て し ま っ た 、 と い う 男 性 は 少 な く あ り ま せ ん 。 ２ ０ 代 は 「 若 い 」 と い う だ け で 声 を か け て く れ た 女 性 も い ま し た 。

同 年 代 の 仲 間 も ま だ 独 身 で ふ ら ふ ら し て い る 人 が 多 い の で 余 裕 が あ り ま し た が 、 ３ ０ 代 に 差 し 掛 か る 頃 か ら 家 庭 を 持 つ 人 が 増 え て い き ま す 。

い よ い よ ４ ０ 代 に な り 「 ま だ 独 身 な の ？ 」 「 彼 女 も い な

い の ？ 」 と い う 目 で 見 ら れ る 機 会 が 増 え る の で 、 モ テ な い こ と に 焦 り を 感 じ る の で は な い で し ょ う か 。

今 回 は 「 ４ ０ 代 男 性 が モ テ る 方 法 」 と い う こ と で 、 モ テ る ４ ０ 代 男 性 に な る た め に 今 か ら で き る こ と を お 伝 え し ま す 。

１ 章 　 ４ ０ 代 に な っ て も モ テ る 男 性 で い る た め に は ？

４ ０ 代 の 男 性 の 中 に は 「 ２ ０ 代 の 頃 は そ れ な り に モ テ た 」 と い う 人 も 多 い の で は な い で し ょ う か 。

し か し 、 年 齢 を 重 ね る に つ れ 、 女 性 か ら の 人 気 を 獲 得 す る ア プ ロ ー チ の 仕 方 を 変 え て い か な け れ ば 、 モ テ る 状 態 を 維 持 す る こ と は で き ま せ ん 。

で は 、 ど う す れ ば 女 性 か ら の 人 気 を 獲 得 す る こ と が で き る の で し ょ う か 。

〇 ４ ０ 代 男 性 は 若 さ や 外 見 に 頼 ら な い

２ ０ 代 の う ち は 、 見 た 目 が ハ ン サ ム だ と い う だ け で モ テ る 時 期 が あ り ま し た 。

ま た 、 特 別 に ハ ン サ ム で な く て も 、 芸 能 人 に 似 て い る な ど の 理 由 で 声 を か け ら れ た り 、 フ レ ン ド リ ー に 会 話 で き る 機 会 が あ っ た の で は な い で し ょ う か 。

４ ０ 代 に な る と 、 そ れ は 通 用 し な く な り ま す 。 顔 が か っ こ い い 、 と い う だ け で は 身 元 の 保 証 に な ら な い た め 、 い く ら イ ケ メ ン で も 住 宅 街 を 歩 い て い る と 怪 し ま れ ま す 。

こ れ は 、 外 見 の 善 し 悪 し よ り も 中 身 の 信 頼 度 へ の 比 重 が 高 く な る か ら で す 。

つ ま り 、 ２ ０ 代 は 若 い の で 「 そ こ に い る だ け で 価 値 が あ る 」 と 周 囲 は 評 価 し ま す が 、 ４ ０ 代 以 降 の 男 性 は 「 犯 罪 を し な い 、 信 頼 で き る か ど う か 」 で 評 価 さ れ る よ う に な り ま す 。

４ ０ 代 と も な る と そ れ な り に 経 験 を 積 ん だ 大 人 な の で 、 若 さ や 外 見 で は 評 価 し て も ら え な く な り ま す 。

〇 女 性 に 信 頼 し て も ら う

「 モ テ る ＝ フ ァ ン が た く さ ん い る 状 態 」 と 仮 定 す る と 、 ４ ０ 代 に な っ て か ら 新 規 の フ ァ ン を 獲 得 す る の は と て も 難 し い で す 。 そ の た め 、 ２ ０ 代 の う ち に す ご く モ テ た 人 は 、 す で に 信 頼 し て く れ て い る フ ァ ン 層 を 大 切 に 守 り な が ら 過 ご す 人 が 多 い で す 。

男 性 が モ テ る と い う こ と に お い て 「 女 性 に 信 頼 し て も ら う 」 と い う こ と が 、 ひ と つ の 大 き な テ ー マ と な り ま す 。 信 頼 し て も ら う た め に は 、 時 間 や コ ス ト を か け る 必 要 が あ り ま す が 、 ２ ０ 代 の う ち は 低 コ ス ト で 済 ん だ こ と も 、

４ ０ 代 に な る と 高 コ ス ト に な る こ と が あ り ま す 。

す で に 友 人 関 係 に あ る 女 性 を 食 事 に 誘 う の は 簡 単 で す 。 し か し 、 は じ め て 出 会 っ た 人 を 食 事 に 誘 う こ と は 難 易 度 が 高 い で す 。 こ の 難 易 度 は 信 頼 度 に 比 例 し ま す 。

そ の た め 「 ４ ０ 代 以 降 モ テ る よ う に な り た い 」 と い っ た 場 合 、 女 性 に 信 頼 し て も ら え る 態 度 、 考 え 方 、 接 し 方 、 外 見 な ど を 磨 く 必 要 が あ り ま す 。

ま た 、 ２ ０ 代 の モ テ て い た 状 態 を 維 持 し た い 場 合 は 、 今 ま で 獲 得 し た フ ァ ン を 大 切 に し 、 こ ま め に 連 絡 を 取 っ て 、

フ ァ ン サ ー ビ ス を 欠 か さ な い こ と で す 。

２ 章 　 モ テ る ４ ０ 代 の 特 徴

４ ０ 代 の 時 点 で モ テ て い る 男 性 に は 、 ど の よ う な 特 徴 が あ る の で し ょ う か 。

ま た 、 モ テ な い ４ ０ 代 と 比 較 す る と ど の よ う な 差 が あ る の で し ょ う か 。

こ こ で は 、 モ テ る ４ ０ 代 の 特 徴 と 、 モ テ な い ４ ０ 代 の 特 徴 を ご 紹 介 し ま す 。

〇 モ テ る ４ ０ 代 ＝ 家 庭 的

今 テ レ ビ に 出 て い る 人 気 俳 優 や ア ー テ ィ ス ト の 中 に は 、 多 く の ４ ０ 代 男 性 が い ま す 。 彼 ら が モ テ る 要 素 は ど こ に あ る の で し ょ う か 。

共 通 し て ト レ ン ド に 上 が っ て く る の は 「 家 庭 的 」 と い う ワ ー ド で す 。

料 理 が で き る 、 一 人 暮 ら し で 家 事 が で き る 、 ま た は 、 良 い お 父 さ ん に な り そ う 、 な ど 「 家 庭 的 な イ メ ー ジ 」 が 印 象 に プ ラ ス さ れ ま す 。

こ れ に よ っ て 、 ２ ０ 代 で は モ テ な か っ た 人 が 注 目 を 浴 び

る こ と も あ り ま す 。

逆 に 、 ち ょ っ と 不 良 な イ メ ー ジ 、 ワ イ ル ド な イ メ ー ジ 、 ビ ジ ュ ア ル 系 ア ー テ ィ ス ト 風 な ど の 家 庭 と は か け 離 れ た イ メ ー ジ の 人 は 、 モ テ か ら 遠 ざ か り ま す 。

「 家 庭 的 な 男 性 」 と は 、 女 性 へ の 理 解 が あ り 、 女 性 蔑 視 は せ ず 、 女 性 の 人 権 を 尊 重 し て く れ る 男 性 の 総 称 で す 。

そ の た め 、 い わ ゆ る 男 ら し い 男 性 よ り も 、 お 母 さ ん の よ う に 母 性 を 持 っ た 男 性 の 方 が モ テ る 傾 向 に あ り ま す 。

〇 ４ ０ 代 の モ テ な い 男 性 の 特 徴

４ ０ 代 の 中 で も と く に モ テ な い 男 性 に は 、 い く つ か 特 徴 が あ り ま す 。

ま ず 女 性 と 関 わ り を 持 た な い 男 性 で す 。

「 コ ミ ュ 障 」 な ど と 呼 ば れ ま す が 、 他 人 と の コ ミ ュ ニ ケ ー シ ョ ン を 拒 む 傾 向 の あ る 男 性 は 、 当 然 な が ら モ テ ま せ ん 。

「 黙 っ て い て も モ テ る 人 」 は お ら ず 、 ど ん な に イ ケ メ ン で あ っ て も 誰 も ち や ほ や し て く れ ま せ ん 。

そ の た め 、 全 く 同 じ 容 姿 の Ａ さ ん と Ｂ さ ん が い て 、 Ａ さ ん が 積 極 的 に 友 達 を 作 ろ う と 努 力 す る の に 対 し 、 Ｂ さ ん

が 声 を か け ら れ る の を た だ 待 っ て い る の だ と し た ら 、 Ａ さ ん の 方 が 圧 倒 的 に モ テ ま す 。

ま た 、 妬 み 嫉 み を 持 っ て い る 人 も モ テ ま せ ん 。 「 俺 の 方 が 絶 対 か っ こ い い の に 、 な ん で ア イ ツ ば っ か り モ テ る ん だ 」 な ど の 気 持 ち を 持 っ て 、 他 人 の 足 を 引 っ 張 ろ う と す る 男 性 は 、 例 外 な く モ テ ま せ ん 。

モ テ な い ば か り か 、 嫌 わ れ て い る 可 能 性 が 高 い で す 。

ラ イ バ ル に 対 し て だ け で な く 、 女 性 に 対 し て 「 女 の く せ に 」 、 年 上 に 対 し て 「 ジ ジ イ （ バ バ ア ） の く せ に 」 、 子 供 に 対 し て 「 ガ キ の く せ に 」 と い っ た 差 別 感 情 を 持 っ て い

る 人 も モ テ ま せ ん 。

〇 モ テ る 特 徴 を 手 に 入 れ る 方 法

モ テ る ４ ０ 代 に な る た め に は 、 差 別 的 な 考 え 方 を 捨 て 、 新 し い 価 値 観 を 手 に 入 れ る こ と が 大 事 で す 。

と は 言 え 、 頭 ご な し に 現 在 の 自 分 を 否 定 す る の で は な く 、 半 年 ほ ど ツ イ ッ タ ー な ど の 女 性 の ア カ ウ ン ト を ロ ム る （ 発 言 せ ず に 眺 め る ） こ と を お す す め し ま す 。

と く に 「 女 性 を 差 別 し な い で ほ し い 」 と い う よ う な 発 言 の 多 い 著 名 人 の ア カ ウ ン ト を 、 た だ ひ た す ら 頭 を ま っ さ ら に し て 、 暇 な 時 間 に 読 ん で み ま し ょ う 。

差 別 を 手 放 す た め に は 、 相 手 の 立 場 に 立 っ て 考 え て み る こ と が 大 事 で す 。 相 手 の 立 場 に 立 つ と は 「 も し 自 分 が 女 性 だ っ た ら 」 と 想 像 す る こ と で す 。

現 代 の 女 性 は 、 自 分 の 感 じ て い る 苦 し み に 共 感 し て く れ る 男 性 を 求 め て い ま す 。

家 庭 的 で 他 人 を 差 別 せ ず 、 同 じ 目 線 で 物 事 を 見 て 一 緒 に 歩 い て く れ る 男 性 が 理 想 で す 。

３ 章 　 モ テ る ４ ０ 代 の 外 見 、 容 姿

モ テ る ４ ０ 代 の 男 性 の 外 見 や 容 姿 に は 、 ど の よ う な 特 徴 が あ る の で し ょ う か 。

モ テ な い ４ ０ 代 男 性 は 、 モ テ る 男 性 と の 間 に ど の よ う な 差 が あ る の で し ょ う か 。

こ こ で は 、 モ テ る ４ ０ 代 、 モ テ な い ４ ０ 代 、 そ れ ぞ れ の 外 見 や 容 姿 と 改 善 方 法 を ご 紹 介 し ま す 。

〇 モ テ る ４ ０ 代 ＝ 眉 毛

モ テ る ４ ０ 代 の 外 見 や 容 姿 に は 、 あ る 共 通 項 が あ り ま す 。 そ れ が 「 優 し い 感 じ だ け ど 芯 が あ る 」 と い う 点 で す 。

優 し い 男 性 は モ テ ま す が 、 た だ ひ た す ら 優 し い 外 見 だ と 「 弱 い 、 個 性 が な い 、 い い 人 で 終 わ る イ メ ー ジ 」 が あ り ま す 。

そ の た め 、 優 し い だ け で は な く 「 芯 が あ る 」 こ と が 重 要 で す 。 芯 が あ る と は 、 周 囲 に 流 さ れ ず 自 分 の 意 見 を し っ か り 持 っ て い る こ と が 外 見 に 現 れ て い る こ と で す 。

こ の 「 芯 が あ り そ う 」 な 様 子 は 、 眉 毛 で 表 現 さ れ ま す 。 ボ サ ボ サ の 眉 毛 を 整 え る だ け で も 印 象 が 変 わ り ま す 。

ま た 、 細 い 眉 、 薄 い 眉 の 人 は 眉 ペ ン シ ル を 使 っ て 書 き 足 す と い い で し ょ う 。

元 か ら 眉 の メ ン テ ナ ン ス を し て い る 男 性 は 目 力 が 強 い の で 「 芯 の 強 さ 」 を ア ピ ー ル す る こ と に 成 功 し て い ま す 。

眉 毛 の 整 え 方 が わ か ら な い 人 は 、 眉 ケ ア を 専 門 に 行 っ て い る お 店 が あ る の で 利 用 し て み ま し ょ う 。

〇 モ テ な い ４ ０ 代 ＝ 不 潔

モ テ な い ４ ０ 代 の 外 見 や 容 姿 で 第 １ 位 な の が 「 不 潔 」 で す 。 普 段 か ら 身 だ し な み に 気 を つ け て い る 人 や 、 フ ァ ッ シ ョ ン 誌 を 参 考 に し て い る 人 は 大 丈 夫 で す が 、 女 性 は 男 性 が 想 像 し て い る 以 上 に 清 潔 さ を チ ェ ッ ク し て い ま す 。

そ の 清 潔 さ と は 「 昨 日 き ち ん と 風 呂 に 入 っ た 」 で あ る と か 「 殺 菌 し た か ら 菌 は い な い 」 な ど の 、 科 学 的 な 清 潔 さ で は な く 、 外 見 の 印 象 の 清 潔 さ で す 。

汚 い か ど う か で は な く 「 汚 く 見 え る か ど う か 」 が 鍵 で す 。

い く ら 実 際 に は 清 潔 に し て い て も 、 不 潔 と 判 断 さ れ る こ と が あ り ま す 。

ま た 、 実 際 に 周 囲 の 人 が 気 付 く ほ ど に 異 臭 を 放 つ 男 性 （ 風 呂 に 入 っ て い な い 、 服 を 洗 濯 し て い な い 、 部 屋 が 汚 い ） も い る の で 、 清 潔 さ の ボ ー ダ ー ラ イ ン が ど こ に あ る の か チ ェ ッ ク し て み る 必 要 が あ る で し ょ う 。

〇 ４ ０ 代 で モ テ る 外 見 に な る た め に

４ ０ 代 で モ テ る 外 見 を 手 に 入 れ る た め に は 、 自 分 が ど ん な 人 間 か 知 る 作 業 が 必 要 で す 。

そ し て 、 自 分 が ど ん な 人 間 か 、 ど ん な 外 見 を し て い る の か 、 よ く わ か ら な い 人 は プ ロ に 任 せ る の が 一 番 良 い 方 法 で す 。

美 容 室 に 行 っ て 「 似 合 う 髪 型 に し て く だ さ い 」 と お 願 い し た り 、 洋 服 も 店 員 さ ん に 「 似 合 う も の を 選 ん で く だ さ い 」 と お 願 い し て み ま し ょ う 。

自 分 で 自 分 の こ と を よ く わ か っ て い な い 人 が 多 い の で 、

全 身 コ ー デ ィ ネ ー ト し て も ら う こ と が 一 番 の 近 道 で す 。

４ 章 　 モ テ る ４ ０ 代 の 性 格

〇 モ テ る ４ ０ 代 男 性 は 「 温 和 」

ズ バ リ 、 モ テ る ４ ０ 代 男 性 の 性 格 は 「 温 和 」 で す 。 大 ら

か で 優 し く 、 余 裕 の あ る 性 格 が も っ と も モ テ ま す 。

し か し 、 男 性 同 士 で 好 感 の モ テ る 印 象 と は 違 う た め 、 路 線 を 間 違 っ て 逆 走 し て い る 人 も 見 か け ま す 。

温 和 な 性 格 は 「 敵 を 作 ら ず 、 味 方 を 増 や し て い く 」 と い う 性 格 で す が 、 勝 負 事 に は 弱 い イ メ ー ジ が あ る た め 毛 嫌 い す る 男 性 も 多 い で す 。

た だ し 、 モ テ を 意 識 す る の で あ れ ば 、 あ く ま で 穏 や か に す べ て を 包 み 込 む よ う な 優 し さ の あ る 対 応 が ベ ス ト で し ょ う 。

〇 モ テ な い ４ ０ 代 男 性 は 「 感 情 的 」

モ テ な い ４ ０ 代 男 性 の 性 格 は 「 感 情 的 」 で す 。 と く に 近 年 は 感 情 的 な 男 性 が 嫌 わ れ る 傾 向 が 強 く 、 喜 怒 哀 楽 が 激 し い 人 は あ ま り モ テ ま せ ん 。

こ の 感 情 的 な 性 格 に も い く つ か 種 類 が あ り ま す が 、 と く に 「 反 射 的 に 怒 っ て し ま う 」 な ど の ネ ガ テ ィ ブ な 性 格 は 毛 嫌 い さ れ ま す 。

感 情 を 表 に 出 す こ と が 悪 い こ と な の で は な く 、 一 度 自 分 の 中 で 考 え て か ら 表 現 す る と 好 感 度 が 上 が る の で 、 反 射

的 に 怒 鳴 る 、 反 射 的 に 泣 い て し ま う 、 大 声 で 笑 っ て し ま う 、 な ど の 感 情 的 な 人 は 、 自 分 の 表 現 の 仕 方 に 注 意 し ま し ょ う 。

５ 章 　 モ テ る ４ ０ 代 の 行 動

モ テ る ４ ０ 代 の 男 性 は 、 ど の よ う な 行 動 を 取 る で し ょ う か 。

ま た 、 モ テ な い ４ ０ 代 の 男 性 は ど の よ う な 行 動 を し て い る の で し ょ う か 。

こ こ で は 、 モ テ る ４ ０ 代 男 性 と モ テ な い ４ ０ 代 男 性 の 行 動 に つ い て ご 紹 介 し ま す 。

〇 モ テ る ４ ０ 代 男 性 の 行 動 ＝ 周 囲 へ の 気 遣 い

モ テ る ４ ０ 代 男 性 は 、 常 に 周 囲 の 動 き を チ ェ ッ ク し て い ま す 。

こ れ は 「 自 分 に 手 助 け で き る こ と は な い か 」 「 困 っ て い る 人 は い な い か 」 と 探 し て い る よ う な 行 動 で す 。

何 か あ れ ば す ぐ に 飛 ん で い っ て 手 助 け す る た め 、 人 か ら 好 か れ ま す 。

言 い 方 を 変 え れ ば 「 恩 を 売 る 」 と い う 行 動 で も あ る の で す が 、 人 か ら 感 謝 さ れ る 行 動 を 多 く す る こ と で 、 モ テ の 範 囲 を 広 げ る こ と が で き ま す 。

４ ０ 代 と も な る と 大 人 の 行 動 を 求 め ら れ る た め 、 人 助 け す る と 「 さ す が 大 人 」 と 思 わ れ て 好 感 度 が 急 上 昇 し ま す 。

〇 モ テ な い ４ ０ 代 男 性 の 行 動 ＝ 独 り よ が り

モ テ な い ４ ０ 代 男 性 の 行 動 は 「 独 り よ が り 」 で す 。

独 り よ が り と は 、 子 供 っ ぽ く 独 善 的 で わ が ま ま 、 と い う 印 象 で す 。

自 分 さ え よ け れ ば そ れ で い い 、 誰 か に 迷 惑 が か か っ て も 構 わ な い 、 と い う 態 度 は 人 を 寄 せ 付 け ま せ ん 。

独 り よ が り な 行 動 を 繰 り 返 し て い る と 「 ４ ０ 代 に も な っ て 子 供 っ ぽ い 」 と 思 わ れ る で し ょ う 。

女 性 は 「 こ ん な 男 性 と 交 際 し た ら 、 面 倒 な こ と を 押 し 付 け ら れ る に 違 い な い 」 と 感 じ 、 距 離 を 置 き ま す 。

女 性 だ け で な く 他 人 か ら 信 用 さ れ な い た め 、 孤 立 し て し ま い ま す 。

６ 章 　 モ テ る ４ ０ 代 の 職 業

モ テ る ４ ０ 代 男 性 は 、 ど の よ う な 職 業 に つ い て い る で し ょ う か 。

ま た 、 モ テ な い ４ ０ 代 男 性 は ど の よ う な 職 業 に つ い て い る の で し ょ う か 。

こ こ で は 、 モ テ る ４ ０ 代 男 性 、 モ テ な い ４ ０ 代 男 性 、 そ れ ぞ れ の 職 業 に つ い て ご 紹 介 し ま す 。

〇 モ テ る ４ ０ 代 男 性 の 職 業 ＝ 安 定

モ テ る ４ ０ 代 男 性 の 職 業 は 「 安 定 し た 職 業 」 で す 。

安 定 と は 、 年 収 が 安 定 し て い る だ け で な く 、 仕 事 内 容 が 安 定 し て い る （ 過 労 の 心 配 が な い ） 、 転 勤 が な い と い う こ と も 含 ま れ ま す 。

ま た 、 ど ん な 職 業 に つ い て い て も 、 今 は 「 一 生 同 じ 企 業 で 働 き 続 け る 」 と い う こ と が 難 し い ご 時 世 な の で 、 手 に 職 が あ る こ と が 重 視 さ れ る 傾 向 も 強 い で す 。

最 終 的 に は 「 独 立 起 業 し て も 食 べ て い け る 柔 軟 性 の あ る 人 」 が モ テ ま す 。

た だ し 、 ４ ０ 代 時 点 で 単 に 女 性 の 友 達 を 増 や し た い 、 彼 女 が ほ し い 、 と い う の で あ れ ば 女 性 が 多 く 働 く 職 場 に 就 職 す る と 出 会 い の き っ か け が 増 え る の で お す す め で す 。

〇 モ テ な い ４ ０ 代 男 性 の 職 業 ＝ 将 来 性 が な い

モ テ な い ４ ０ 代 男 性 の 職 業 は 、 将 来 性 の な い 職 業 で す 。

た と え ば 、 ず っ と コ ン ビ ニ で ア ル バ イ ト し て い た と し て も 、 せ い ぜ い 出 世 で き て 店 長 で す 。

頑 張 れ ば 頑 張 っ た だ け 出 世 す る こ と が で き る 、 と い う 職 業 で な い 場 合 、 あ ま り 将 来 性 は あ り ま せ ん 。

女 性 か ら 見 る と 、 収 入 が 不 安 定 で あ っ て も 、 伸 び 代 が あ る 職 業 は 好 感 度 が 高 い で す 。

今 は 見 習 い だ け ど 、 プ ロ に な っ た ら 大 き な 収 入 の 可 能 性 が あ る 、 と な る と 青 田 買 い し た く な り ま す 。

ま た 、 将 来 に 向 け て の 夢 や 希 望 が あ る 男 性 の 方 が モ テ る の で 、 目 先 の 生 活 に 囚 わ れ て 将 来 の 展 望 を 失 っ て い る と

モ テ ま せ ん 。

ま と め

４ ０ 代 に な っ て か ら モ テ る こ と は 、 不 可 能 な の で し ょ う か 。

年 齢 的 に も 諦 め な け れ ば い け な い の で し ょ う か 。 ４ ０ 代 に な っ て 「 ハ ン サ ム じ ゃ な い か ら 」 「 金 持 ち じ ゃ な い か ら 」 な ど と 言 っ て 、 諦 め て い る 男 性 は 多 い で す が 、 諦 め る の は ま だ 早 い で す 。

努 力 次 第 で は 、 今 か ら で も モ テ る こ と は で き ま す 。

４ ０ 代 デ ビ ュ ー も 夢 で は あ り ま せ ん 。

た だ し 、 モ テ る た め に は こ こ で ご 紹 介 し た い く つ か の ポ イ ン ト を 押 さ え て お く 必 要 が あ り ま す 。

自 分 の で き る 範 囲 で 、 で き る と こ ろ か ら 改 善 し て い け ば 、 多 く の 女 性 と 仲 良 く な る こ と も 、 恋 人 を 作 る こ と も 、 最 終 的 に は 結 婚 す る こ と も 夢 で は あ り ま せ ん 。

少 し ず つ で い い の で 、 モ テ な い 自 分 を 脱 ぎ 捨 て て 、 新 し い 「 モ テ る 自 分 」 を 手 に 入 れ ま し ょ う 。
