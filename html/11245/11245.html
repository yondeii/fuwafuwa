英会話書籍や教材に書かれている「英語のリズム」は、何を指すのでしょうか。
これは英語の「お約束」に当たるアクセントや区切りの位置にあるようです。
例えば「Ｉ　ｎｅｅｄ　ｙｏｕｒ　ｈｅｌｐ」を「アイ　ニード　ユア　ヘルプ」
とカタカナ的に区切ると、本来の英語のリズムとは違ってしまい発音の細部も
分かりません。このアクセントを矢印で表しますと、以下になります。

　〇　「Ｉ（↓）ｎｅｅｄ（↑）ｙｏｕｒ（↓）ｈｅｌｐ（↑）」

この場合の発音は、大まかに「アィ、ニード、ユァ、ヘルプｌになります。
次に、英語本来の「区切り」やリズムで表しますと、以下になります。

　〇　「Ｉ（↓）ｎｅｅ（↑）ｄｙｏｕｒ（↓）ｈｅｌｐ（↑）」

こちらの発音は「アィ、ニィ－ 、ドゥヤ、ヘェル（ッ）」になるのではないで
しょうか（カタカナなので、大まかになっています）このような見方をしますと
「Ｉｓｎ'ｔ　ｉｔ　ｉｎ　ｙｏｕｒ　ｐｏｃｋｅｔ？」は、区切りの位置が
「Ｉｓｎ'、ｔｉ、ｔｉｎ、ｙｏｕｒ、ｐｏｃｋｅｔ？」になるので、発音は
「イズント、イット、イン、ユア、ポケット？」ではなくて－
「イズン、ティッ、ティン、ヤ、ポケッ（トゥ）？」に近くなります。

　●　「Ｉｓｎ'ｔ　　ｉｔ　　ｉｎ　ｙｏｕｒ　ｐｏｃｋｅｔ？」
　　　「イズント　　　イット　イン　ユア　　　ポケット？　　」
　〇　「Ｉｓｎ'　ｔｉ　　ｔｉｎ　ｙｏｕｒ　ｐｏｃｋｅｔ？　」
　　　「イズン　　ティッ　ティン　ヤ　　　　ポケッ（トゥ）？」

また「Ｇｉｖｅ　ｉｔ　ｔｏ　ｍｅ」は「Ｇｉ、ｖｅｉ、ｔｔｏ、ｍｅ」の区切りから
「ギブ、イット、トゥー、ミー」ではなくて「ギ、ヴィ、トゥ、ミィ」です。

４
もちろんカタカナで表しているので正確さは欠けます。
英語のリズムとは、このような自然な区切りや発音が「感覚的に、分かる」状態
を表すのではないかと思います。
この状態は、これまでの区切りでは発音しない個所が「音として」出てくる
（聴こえる）面もあります。すると発音を（音として）見た場合は、英語本来の
正確な「区切り」が聴き取れなければ、英語のリズムや発音は得られないのでは
ないでしょうか。

　○　英語本来の区切りが（音として）聴き取れないと、発音は分からないまま

この状態が「英会話ＣＤを聞き続けても、英語のリズムは得られない」理由では
ないでしょうか。すると聴き取れない状態での発音練習は－
「正確な発音を聞いたことがないのに、発音する」ので相当の難しさがあります。
英会話は、先に英語が聴き取れる状態から始めるのが効率的ではないでしょうか。
そして聴き取れた発音から、自然にアクセントのコツをつかんだ状態が
「英語のリズムが分かる」ことだと思います。

０２　なぜ英会話ＣＤは、聞き続けることが難しい？
－ － － － － － － － － － － － － － － － － － － － － － － －

英会話の習得のためにＣＤ付き書籍を買っても、結局は継続して聞けなかった
方がいらっしゃるのではないでしょうか。その理由はさまざまだと思いますが、
ＣＤ音声の収録時間が「長すぎる」ことが、原因の一つかもしれません。
一見（一聴？）すると音声は７０分近くあった方が「お得」な感じになりますが、
実際に「１日１回」聞こうとした場合は、すぐに飽きてしまいます。

「それなら音声を何日かに分けて、聞き続ければよいのではないか」と思われる
かもしれません。ですがトラックが何１０もある場合は、リスニングが面倒では
ないでしょうか。そこで例えば英会話の音声を「毎日、聞く」習慣を作りたい場合は、
以下にしますと継続しやすいのではないかと思います。

５
　① 　１回に聞く時間は１０分程度。　② 　同じ内容を毎日聞く。

① の時間の理由は、最後まで集中して「聴ける状態」にするためです。
そして② の行為は「ある程度まで」になります。じつは② の状態を続けますと
「以前は聴き取れなかったけど、何度か聞くと意味の分かる」箇所がでてきます。
（リスニング力の向上というよりも、慣れや過去の知識から分かるようです）
もちろん、すべてが理解できるわけではありません。
「知らない単語は何度、正確に聴き取れても、意味は分からないまま」だからです。
ですから同じ音声を聞き続けることは、ある程度の回数にする必要があります。
すると、その「変え時」は、いつになるのでしょうか。
例えば同じ内容を毎日聞いて「以前よりは、聴き取れる個所が増えたな」と思われたり、
会話のセリフや、話の大まかな「流れ」を覚えた時がよいのではないでしょうか。
ほとんどの記憶は、時間の経過により徐々に忘れていきます。
それなら「完全に覚えてから、次へ行く」という、完璧を目指す学習ではなく－
「ある程度を聞いて、いくつか覚えたら、次へ行く」程度でよいのではないでしょうか。
また、書籍を何度も購入するのはもったいないので、テレビ番組を利用するのがよいと
思います。
ちなみに、何度か番組を視聴すると「日本語訳が、直訳になっていない」事に気が付く
場合もあります。
これは日本語のセリフを覚えた状態や、英語が聴き取れて理解していることを証明して
いるので、ここでも「変え時」の目安になります。
このような方法をとりますと、英会話ＣＤが継続して聞きやすくなると思います。

ある英会話の書籍を見ていたら、以下の例文と、その意味が載っていました。

　① 　「Ｉ　ｃａｎ　ｓｐｅａｋ　ａ　ｌｉｔｔｌｅ　Ｅｎｇｌｉｓｈ」
　　　「私は、少し英語が話せます」

６
　② 　「Ｉ　ｃａｎ　ｓｐｅａｋｅ　Ｅｎｇｌｉｓｈ　ａ　ｌｉｔｔｌｅ」
　　　「英語くらい話せるよ！（バカにするな！）」

これは「ａ　ｌｉｔｔｌｅ」の位置の違いで、意味が変わっています。
すると日本語の感覚では「なぜ、こうなってしまうのか」疑問が出てくるのではないで
しょうか。そこで分かりやすいように表しますと、以下になるのではと思いました。

　① 　Ｉ　ｃａｎ　ｓｐｅａｋ「ａ　ｌｉｔｔｌｅ　Ｅｎｇｌｉｓｈ」
　　　私は話せます、「少しの英語」（は）　→私は、少し英語が話せます　　　　　　　　　　　　　　　　　　

　② 　Ｉ　ｃａｎ　ｓｐｅａｋ　Ｅｎｇｌｉｓｈ「ａ　ｌｉｔｔｌｅ」
　　　私は英語が話せます　　　　　　　　　　「（それは）小さなことです」　
　　　私が英語を話せるのは、小さな事です　→英語くらい話せるよ！

このような見方をしますと、① は－
「ａ　ｌｉｔｔｌｅ（形容詞）Ｅｎｇｌｉｓｈ（名詞）」のまとまりになっています。
例えば日本語なら「青い（形容詞）空（名詞）」と同じ形です。以前に読んだ書籍では
「英語で複数の形容詞を使う場合は、およその順番が決まっている」と書かれていた
気がします。そして、記憶が正しければ「状態の前に、先に数を表す」でした。
その一方で副詞の場合は、以下のように付ける形があります。

　「Ｃｏｕｌｄ　ｙｏｕ　ｓｐｅａｋ（ｍｏｒｅ）ｓｌｏｗｌｙ（ｐｌｅａｓｅ）？」
　「できる　　　あなた　話す　　　（もう少し）ゆっくり　　（どうか）　　　？」
　「できる？＝　あなた、話すのを　（もう少し）ゆっくりに　（どうか）　　　　」
　　→「もう少し、ゆっくり話して頂けませんか？」

話を戻しますと、② の形は以下になっているのかもしれません。

　② 'Ｉ　ｃａｎ　ｓｐｅａｋ　Ｅｎｇｌｉｓｈ　＋　Ｉｔ　ｉｓ　ａ　ｌｉｔｔｌｅ

もちろん「ｉｔ」は、前部の「私は、英語が話せます」になります。

７
このように理解すると「英語くらい話せるよ！」（大した事ではありません）は、
分かりやすいのではないでしょうか。（当たっているのかどうか、わかりませんが）
ちなみに、辞典では「ｃａｎ＝基本的な意味は、可能を表す」とありましたが、
ここでは「能力がある」ことを表しています。
このような見方をしますと、① ② の違いは理解しやすいと思います。

０４　なぜ英語の動詞は、さまざまな意味になる？
－ － － － － － － － － － － － － － － － － － － － － － －

英和辞典を引きますと、簡単でよく使われる動詞ほど、多くの意味があるのでは
ないかと思いました。その理由は、英熟語から見えてくるのではないでしょうか。
英語書籍に載っている熟語を見ますと、以下のタイプがあります。

　① 　「動詞＋副詞」　　② 　「動詞＋前置詞＋～」

もちろん他のタイプもありますが、上記が多いのではないでしょうか。
これらは動詞の後に、副詞や前置詞が付くことで、具体的な状態や動作を表しています。
ここから英語で「ある状態や、動作」を表すときは、動詞「だけ」でなく、他の品詞を
使って表すことがあると分かります。
このような見方をしますと① ② の動詞は、単に「大まかな状態や、動作」を表している
のかもしれません。

例えば「（私を）降ろして」と言いたい場合は、英語は「ｐｕｔ　ｍｅ　ｄｏｗｎ」の
言い方があります。これを、先程のような見方にしますと以下になります。

　〇　「置いて（ｐｕｔ）私を（ｍｅ）下に（ｄｏｗｎ）」

すると、動詞の「ｐｕｔ」だけでなく「ｄｏｗｎ」も使って、具体的な動作（降ろして）
を表しています。同時に「何が、どうなる（何を、どうする）」時間も表れます。
そして、この熟語を詳しくしますと以下になるのではないでしょうか。

８
　〇　大まかな状態や動作（ｐｕｔ）＋より具体的な状態や動作（ｄｏｗｎ）

すると先の動詞（ｐｕｔ）は、大まかな動きを表すので、この動詞だけでは、あいまい
な表現になってしまいます。
ちなみに、動詞や単語だけの意思表示では、カタコト英語になるイメージがあります。
そうならないためにも「熟語で覚えることは、重要」ですが、前置詞や副詞の基本を
知ることでも、およその意味の推測や意思表示ができるのではないでしょうか。

例えば、先程の「（私を）降ろして」と言いたい時には－
「降ろして、は分からない。でも動かして、私は下に行くから・・ｐｕｔ　ｄｏｗｎ」
や「ｐｕｔ　ｍｅ　ｄｏｗｎ」が出てくるかもしれません。
すると熟語を「そのまま」覚える理解とは違った、英会話の学習になります。

話を戻しますと、基本的な動詞ほど、さまざまな熟語になるので多くの意味を表します。
この現状は「使われる動詞が限られるから、英会話は、やさしい」の見方もできますし、
「簡単な動詞の熟語が、たくさんの意味になるから難しい」の見方もできてしまいます。

そして、前者から「一般的な英会話なら、中学で習う英語でも十分に使える」と言われ
るのだと思います。
しかし、学校の英語は文法が中心なので、日常で使われる英語を習っても、英会話では
ほとんど使えないようです。ある英語書籍を何冊も出されている方は－
「英会話で使う単語は知っているのに『使い方』を知らないから、英会話ができない」
とおっしゃっています。
このように、簡単な動詞ほど多くの熟語で使われるので、その結果、さまざまな意味に
なっています。
すると日常で使われる英単語の「基本的な意味」を知っておくと、理解しやすいのでは
ないでしょうか。

「なぜ英語の動詞は、様々な意味になる？」では、熟語の動詞が、大まかな状態を表す
ので、多くの意味になるのではないかとご説明しました。
そして動詞を使った熟語を見ますと、前置詞も同様ではないかと思いました。
前置詞の数は動詞よりも限られるのに、同じ動詞を使った熟語は、大きな意味の違いが
出てくるからです。
また同じ単語でも、品詞としては、前置詞や副詞の場合があります。
熟語書籍では「動詞＋副詞」「動詞＋前置詞＋～」のどちらも載っているので、これも
「前置詞（副詞）は、よく使われている＝さまざまな意味を表す」と思ってしまう理由
かもしれません。

そして、日常英会話では、受験に出そうな「ｓｅａｒｃｈ／ｉｎｖｅｓｔｉｇａｔｅ」
よりも、中学英語の「ｌｏｏｋ　ｆｏｒ～」の方が、使われる機会は多いようです。
すると必然的に、前置詞や副詞の出番は増えてしまいます。
もし前置詞を理解したい場合は、多くの熟語から覚えることも１つの方法です。
ですが一緒に使われる動詞によって、意味は大きく変わるので、前置詞や副詞の基本的
な意味を理解することも、必要ではないでしょうか。

例えば「その本は、出版された」を英語で表したいとします。
すると「～された、は受動態で・・出版する・・？の動詞を知らないので、分からない」
と思ったり、真っ先に時制を考えるかもしれません。
ところが書籍では、単に「Ｔｈｅ　ｂｏｏｋ　ｉｓ　ｏｕｔ」と載っていました。
「こんなに簡単な単語で、意味が伝わるの？」と思われるかもしれませんが、辞典でも
「ｃｏｍｅ　ｏｕｔ／ｂｒｉｎｇ　ｏｕｔ＝出版する」と載っていました。ちなみに
「ｃｕｔ　ｉｔ　ｏｕｔ！」は、「（その話は）するな！（外へ出せ）」の意味です。
ここで「ｏｕｔ＝外に出る」のイメージがあれば、先の例文は理解しやすくなります。

このような見方をしますと、学校の英語では「出版する＝ｐｕｂｌｉｓｈ」のように
「１つの動詞＝１つの意味」で覚える機会が多いのではないでしょうか。
ですが英会話では、限られた単語の「組み合わせ」を使うことが多いようです。
この違いが「学校で習った英語は、英会話では上手く使えない」理由かもしれません。

０７　英会話学校の使い方
－ － － － － － － － － － － －

「簡単な英語が使えるようになるには」では、英語を「聞く／読む」のどちらも、
個々のメリットがあるとご紹介しました。

１２
この状態を学校英語に当てはめますと、どのようになるのでしょうか。
すると学校英語は、読む／書く内容が中心なので、聞く機会は多くありません。
その一方で、日常の英会話は「聞く／話す」内容になるので、学校英語では、英会話が
難しいことになってしまいます。
「それなら学校の英語は、役に立たないのではないか」と思われるかもしれませんが、
そうではありません。学校英語と英会話は、文法／会話重視の違いがあるだけです。
それに、学校で英語を習ったからこそ、カタコト英語で意志が伝えられたり、簡単な
洋楽の歌詞やタイトルが理解できてしまいます。

そして英会話は「会話」なので、聞く／話す内容を中心に学習しますと、もっとも効率
がよいのではないかと推測できます。
それなら、聞く／話す内容の、どちらを優先すべきでしょうか。
おそらくは「聞く」ではないかと思われます。
その理由は、リスニング力を上げてから「聞く」と、英会話や洋楽を聞くときには－
「学校で習った英語が、使われている」と気が付けるからです。
これは理解力の向上なので、その後はカタカナ英語でも、意思の疎通が可能になります。

そして、頭のなかの（忘れかけていた）英語を、再び思い出す機会になるので、記憶が
強固になります。
さらにはリスニング力が上がると、正確な発音が聴けますので、発音が耳で覚えられる
自然な状態になります。
このような見方をしますと、最初にリスニング力を上げる「聞く」学習を優先させるの
がよいのではないでしょうか。
そして後から、英語を聞く機会を作ったり、英会話学校などで、聞く／話す機会を作り
ます。すると、時間と費用に見合ったメリットが出てくると思います。

ちなみに英会話学校では、ネイティブの先生が、英語を「ゆっくり」話すようですが、
一般の方は、英語が苦手な方に対しても、普通の「速さ」で話すことが多いようです。
するとネイティブの発音でも、ゆっくりの英語を聞くことは、それほどのメリットは
無いのかもしれません。

１３
そこで一般的な英会話の「速さ」で聞くようにしますと、これまで以上の（レベルの）
英会話学習になります。
そして次の段階（？）で、例えば、英会話学校では「できるだけ、英語が話せる機会」
「正確な発音や意味が分かる機会」「生の英会話の感覚が、聞ける機会」になるので、
さまざまなメリットが出てくるのではないでしょうか。
このようにしますと、英会話学校は上手く活用できるのではないかと思います。

０８　なぜ英会話は、発音しない個所があっても聞き取れる？
－ － － － － － － － － － － － － － － － － － － － － － － － － － － －

英語のリスニング力が上がると、英会話や洋楽の「今までは、聴き取れなかった」個所
が聞こえると同時に、上がっても「聴き取れない（発音していない）」個所があること
にも気が付きます。
これは「スペルの一部にある、特定の発音しないアルファベット」とは違うのですが、
「発音しないのに、なぜネイティブの人は聞き取れる（理解できる）のか」疑問が出て
くるのではないでしょうか。
これには、いくつかの理由があると思いますが、まず以下の状態が考えられます。

　〇　英会話は、熟語や短い会話が繋がっているので、１つの発音として
　　　まとまっている。

例えば、日本人でも「チェキラッ！」を聞くと、たいては「ｃｈｅｃｋ　ｉｔ　ｏｕｔ」
の単語がイメージできます。
そして、この時は、３つの英単語を（個別に）聞いている意識は、ほとんどありません。
確かに「ｃｈｅｃｋ、ｉｔ、ｏｕｔ」を見ますと、３つの単語になっています。
ですが、発音においては「ｃｈｅｃ－ ｋｉ－ ｔｏｕｔ」ので繋がっているので、
１つの「まとまった」発音と言えるのではないでしょうか。
その結果「チェッ（ク）イッ（ト）アウ（ト）」の（カッコ）の個所が聞こえなくても
「チェキラッ（１つの発音）＝１つの意味」として、自然に理解しています。

１４
これは「チェック、イット、アウト」の（個々の）単語や発音とは、別の状態と言える
のかもしれません。

すると、発音の面では「チェック、イット、アウト／チェキラッ」が別々でも、意味は
同じという、お約束（知識の土台）があります。
このように英会話では、使われる単語が繋がっている場合が多いようです。
ですからネイティブの方は「チェキラッ！」のように発音「しない」個所があっても、
普通に理解できるのではないでしょうか。
するとネイティブの方は、普段から、繋がった（一つの単語のような）会話を聞いて
います。この「まとまった」発音は、熟語の「区切り」とは違うのかもしれません。
すると「発音しない個所があるなら、個々に英単語を覚えるのは無意味ではないか」
と思われるかもしれませんが、そうではありません。
どの国の言葉でも、単語の知識が多いほど、会話や文字の理解力が上がるからです。
先程のたとえは「ｃｈｅｃｋ／ｉｔ／ｏｕｔ」の、個々の単語の知識があるからこそ、
後から「ｃｈｅｃ－ ｋｉ－ ｔｏｕｔ」の繋がりに気が付けます。
ちなみに、このような状態は、日本語にも当てはめられます。
例えば、会話のなかで「入試」と聞いても、ほとんどの方は「入学、試験」の単語を
思い出さなくても、意味は分かってしまいます。

かなり強引ですが「チェキラッ！」は、これと同じ状態と言えるのかもしれません。
そして英会話を学習する立場から見ますと、発音「していない」個所に対して－
「自身は聞けないけど、ネイティヴの方は聞き取れている」と誤解している面がある
のではないでしょうか。
もちろん、その場合もあるのですが、本当に発音「していない」場合もあるようです。
つまり「発音しない個所があっても、聞き取れてしまう状態」ではなく、正確には－
「そのままの発音で、そのまま理解している状態」になっています。
ここから「英語は、リスニングが難しい」と誤解されているのかもしれません。

次の理由は、英会話は、アクセントの位置に「お約束」があります。
この状態から、使われる（次の）単語を無意識に予測するので、この「慣れ」から理解
できる（発音しなくても、聞き取れる）のかもしれません。

１５
と同時に、この状態は、英文法がシンプルな構成であることを証明していると思います。
ある英会話書籍では－
「英語は、アクセントのある母音が優先なので、子音は聞こえなくても理解できる」と
書かれていました。
このような見方をしますと「発音は上手くなくても、英会話は成立できる」状態や－
「英語は、発音よりもリズムで理解する」状態が理解できるのではないでしょうか。

　－ 文法がシンプルなら－

先程は「英文法はシンプルな構成になっている」と書きました。
実際に、学生時代は「英語の基本的な文型は、５つ」と習った記憶があります。
もしかしたら記憶違いかもしれませんが、例えば、ある英文を見て「長い」と思っても、
実際は「長い文章のなかに、短い文章（文型）が、いくつかある」こともあります。
これは学校で習う関係代名詞や、英会話の「ｗｈａｔ／ｔｈａｔ」の使い方を見ますと、
分かりやすいと思います。
そして「短い英語は、基本的な文法が分かりやすい」としますと、たとえ長文でも－
「基本的な文型が複数あるだけなので、品詞は分かりやすい」と推測できます。
すると文章に分からない単語が出てきても、何の品詞かは、分かりやすくなります。

このような見方をしますと、ネイティブの方や英会話を習得された方が、文型や品詞を
意識しないのに、普通に話せてしまうのも理解できるのではないでしょうか。
（おそらく、どの国の母国語でも同じです。）
以上の状態が「英会話は、発音しない個所があっても聞き取れる（理解できる）」理由
ではないかと思いました。

０９　「ｌｅｔ」が続く、英会話は
－ － － － － － － － － － － － － － － －

子供向けの英会話番組を見ていたら、学校英語では、あまり見られないと思われる英語
が出てきました。

１６
　〇　「Ｌｅｔ'ｓ　ｌｅｔ（人）ｇｅｔ　ｓｏｍｅ　ｒｅｓｔ」

この時はメモしなかったので、間違っているかもしれませんが、日本語訳は－
「（人）を、しばらく休ませよう」だったと思います。
これは、中学英語でも「ｌｅｔ＋人＋動詞＝人に、動詞させる」は習うので、およその
意味やニュアンスは分かります。
ここでは、どのような内容になっているのか見ていきたいと思います。
まず文法的な見方をしますと、上記の英語は、以下になるのではないかと思います。

　〇　「Ｌｅｔ'ｓ～」＋「ｌｅｔ（人）ｇｅｔ　ｓｏｍｅ　ｒｅｓｔ」

もちろん「～（動詞の原型）＝（後ろの）ｌｅｔ」ですが、このような見方をしますと、
文法的な形が見えてきます。
すると「Ｌｅｔ（人）ｇｅｔ　ｓｏｍｅ　ｒｅｓｔ」との違いは、何かという疑問が
出てくるのではないでしょうか。
これは「Ｌｅｔ'ｓ」を加えることで「ｌｅｔ（人）ｇｅｔ　ｓｏｍｅ　ｒｅｓｔ」の
命令する口調が、同意を求めるようになるのではないかと思いました。
一般的には「ｌｅｔ'ｓ～＝～しよう」です。

この使い方がピッタリな場面は多いので、よく知られていますが、例文は「ｌｅｔ」が
続くことから「是非／今回は（特に）～しよう」のニュアンスになってると思います。

　－ 「休む」から、英語をイメージすると－

ちなみに日本語では「休もう／休憩しよう」が、一般的な言い方になっています。
ですが、ここでの（英語の）「休む」状態は以下になっています。

　〇　「基本的な動詞＋ある状態」→「ｇｅｔする＋ｒｅｓｔの状態」→休憩する

おそらく「ｈａｖｅ／ｔａｋｅ　ａ　ｂｒｅａｋ」も同様だと思いますが、この感覚を
日本語にしますと「休みを、取る」がぴったりです。

１７
つまり、英語で状態や動きを表す場合は「動詞だけ」もありますし、上記のような－
「（基本的な）動詞＋名詞」があります。
ところが、日本語は真っ先に「休む」が出てくるので、「動詞＋名詞」の形の英語は、
思い浮かべることが難しくなってしまいます。
ここに、日本語と英語の「違い」があるのではないでしょうか。

また「ｈａｖｅ／ｔａｋｅ　ａ　ｂｒｅａｋ」を見ますと－
「Ｌｅｔ'ｓ　ｌｅｔ（人）ｇｅｔ　ｓｏｍｅ　ｒｅｓｔ」に「ｇｅｔ　ａ　ｒｅｓｔ」
は使えないのかと、疑問に思われるかもしれません。
これは、ご紹介していない「前の場面」が影響していると思います。
実は「Ｌｅｔ'ｓ　ｌｅｔ（人）ｇｅｔ　ｓｏｍｅ　ｒｅｓｔ」の場面には、（人）に
対して「何日かたてば、風邪は良くなるよ」と言うセリフがありました。
おそらくは、この「何日か」があるので、一定の期間として「ｓｏｍｅ」を使っている
のではないかと思いました。
すると「ａ　ｒｅｓｔ」が（この場面では）使われない理由が分かると思います。

「Ｌｅｔ'ｓ　ｌｅｔ（人）ｇｅｔ　ｓｏｍｅ　ｒｅｓｔ」は、このようになっている
のではないかと思いました。

１０　個人に適した、英会話書籍は
－ － － － － － － － － － － － － － － －

これまでに、さまざまな英会話の書籍を読みましたが（と言っても、十数冊ほどです）
ここでは、その内容から気が付いたことをご紹介したいと思います。

のではないでしょうか。
これは、内容の主旨や、読まれる方が分かりやすいように、さまざまな工夫をしている
からです。なかには日本語の文章が中心になっているタイプと、英会話の文章が中心に
なっているタイプがあります。この場合は、どちらが読みやすいのでしょうか。
あくまで個人的な実感ですが、基本的には、前者ではないかと思いました。

１８
例えば、英会話書籍を購入しても、何度も読み返せなかったり、最後まで一度も読まな
かった場合もあるのではないでしょうか。
これは、もしかしたら、英語の「量」に関係しているのではないかと思いました。
ほとんどの英会話書籍は「英文を、読む」行為が増えてしまいます。
これは、英語を習う目的なので避けられませんが、日本語の書籍に慣れた方でも－
「英語の量が、多すぎる」ために、続けて読むことが難しいのかもしれません。
そこで、少しでも英語（の量）に対するギャップを減らすために、特に読書好きな方は
「日本語の、多い」書籍を選ぶと、読みやすくなるのではないかと思いました。

つまり「日本語の文章だけど、所々に簡単な英文が含まれている」英会話書籍です。
そして、英語を「覚える」ことは意識しないで、繰り返し読みながら、自然に頭に入る
ようにします。
すると、普段から本を読む方や、英語を一から始めたい方は合っているかもしれません。
「それなら、英語の文章が中心になっている書籍は、お勧めできないのか」と思われる
かもしれませんが、そうではありません。
どのような英会話書籍でも、目的や意図に違いがあるからです。
そして「〇〇書籍の××シリーズは、優れている」「××で、誰でも英語が話せる」と
言えるのは、その方にとって合っているのかもしれません。

すると実際は、読まれる方の目的や状態に合った選び方が重要ではないでしょうか。
例えば、最近、読んでいる書籍は「英会話の場面が、中心になっている」内容ですが、
中学レベルでしたので、最初は、それほど読む気がしませんでした。
しかし、実際は「本的な英文法を使って、さまざまな英単語が出てくる」内容でした。
これなら英文も変わりやすく、新しい英単語も、関連付けられることに気が付きました。
また、ある程度の理解力がある方は、一冊を通して会話の「流れ」があると良いのでは
ないかと思いました。
この状態は、実際の英会話に近いですし、繰り返し読みやすいと思います。

書籍によっては、同じ文法が箇条書きに書かれて、一部の英単語だけが違う内容もあり
ました。このようなタイプは、英会話の「流れ」がイメージできないので、
英単語を覚える目的になっているのかもしれません。まとめますと、以下になります。

１１　英語を「繰り返し話す」行為は、発音練習になる？
－ － － － － － － － － － － － － － － － － － － － － － － － － －

これまでに、さまざまな英会話の書籍を読みましたが（と言っても、十数冊ほどです）
ここでは、その内容から気が付いたことを、またまたご紹介したいと思います。
書籍のなかには「付属ＣＤなどの英語を聞いて、繰り返し発音する」内容が、発音練習
として載っている場合もあります。
これは単に発音練習だったり、何か特別な技術のように、名前が付けられていることも
ありますが、その目的は「発音を良くするため」です。

そして、この練習を英会話に置き換えますと、相手の言ったことを「まねて」発音する
状態になります。
これは、日常英会話では、ほとんど「あり得ない」と言えるのではないでしょうか。
この状態は、特定の内容「だけ」発音しているからです。

しかし実際の英会話は、さまざまな言葉の組み合わせになっています。
たとえ「まねる」ことがあっても、相手の言ったことを確認する時に、何度か聞き返す
くらいではないでしょうか。
このような見方をしますと、特定の単語やフレーズを「繰り返して、発音する」練習は、
「なんでもあり」の会話から見ますと、不自然になってしまいます。

もちろん「口を使って、発音する」練習は必要です。
ですが、実際に会話で話すときは、自身で言葉を選んでいる（意思を表す）状態です。

２０
すると「同じ内容を、繰り返し発音する」状態とは、違ってしまいます。
そこで「同じ内容を、繰り返す」発音練習よりも－
「耳で聞き取れる、さまざまな英語を発音する」ほうが効率的ではないかと思いました。
ただし、例によって、正確な発音を「音として」聴き取れる状態が前提です。
そして、聞き取れた英語を「なんでも（どこでも）真似て、発音する」ことが、自然な
発音練習になるのではないでしょうか。順序立てますと、以下になります。

　① 　英語のリスニング力を上げる。（正確な発音が、聴き取れる状態になる）
　② 　聞き取れる正確な発音を真似て、発音する。（自然な、発音練習になる）

さらにはリスニング力が上がると、知っている英語を「読む」行為でも、発音の練習に
なります。これは、アクセントの位置が感覚的に分かるからです。
このような見方をしますと、発音練習は、先にリスニング力を上げてから、さまざまな
英語を発音する方が効率的ではないかと思いました。

１２　英語の前置詞は　その１
－ － － － － － － － － － － － － －

英語が苦手な学生さんは「英語の前置詞は、使い方が難しい」と、思われているのでは
ないでしょうか。
これは、学校で前置詞を習ったときに、熟語として、大まかな意味で覚える傾向がある
からだと思います。また英会話ができる方でも「使い方を、感覚的に覚える」までには、
ある程度の時間が必要だと思います。
もしかしたら、使い方の丸覚えだけで、基本的な意味は、それほど意識していない方も
いらっしゃるかもしれません。
ところで、熟語といえば「動詞＋前置詞」の形が多くあります。
そして動詞は同じでも、後の前置詞によって、大きく意味が変わってしまいます。
これは先の動詞よりも、後の前置詞の影響が強いと言えるのではないでしょうか。
つまり熟語は、前置詞によって具体的な意味になるので、動詞を中心に覚えてしまうと、
理解が難しくなってしまうかもしれません。

２１
例えば、英語のアニメでは、ハンドルから手を離した運転手に対して－
「両手で（ハンドルを）持って！」と、注意する場面がありました。
この時の英会話が分かりますでしょうか。聞き取りが正しければ、使われたのは－
「ｈａｎｄｓ　ｏｎ　ｔｈｅ　ｗｈｅｅｌ！」でした。
また、運転している人は「ａ　ｐｒｅｓｏｎ　ａｔ　ｔｈｅ　ｗｈｅｅｌ」と表せます。
「これだけで意味が伝わるの？」と思われるかもしれませんが、英会話では、動詞すら
使わない、ちょっとした表現があるようです。

ちなみに、このような「日常の、何気ない表現」は、英会話書籍には、ほとんど載って
いないと思います。
たいていの英会話書籍は、意思の疎通が前提なので、Ｑ＆ Ａのような内容だからです。
このような見方をしますと、英会話書籍の内容が文法的だったり、同じ印象になるのは、
ここに理由があるのかもしれません。
すると書籍の内容は、英会話の一面になってしまいます。

先程の「ｈａｎｄｓ　ｏｎ　ｔｈｅ　ｗｈｅｅｌ！」
「ａ　ｐｅｒｓｏｎ　ａｔ　ｔｈｅ　ｗｈｅｅｌ」に話を戻しますと、ここで－
「ｏｎ／ａｔ」の基本的な意味が分かれば、およその理解が出来てしまいます。
また「コストダウン／コストカット」で使われる「ｃｕｔ／ｄｏｗｎ」違いも、分かり
ますし、日本語の「ピックアップ」が、本来の意味と違っていることも理解できます。
すると、前置詞の基本的な意味を知ることは、英会話でも役立つと言えるのではないで
しょうか。

１３　会話を学習するときの覚え方は
－ － － － － － － － － － － － － － － － －

例えば、日常で使っている日本語でも「その単語の意味は？」と聞かれたら、説明する
のが難しい場合もあるのではないでしょうか。
クイズ番組では、意味や説明の内容から単語を当てる問題もありますが、これは使い方
は分かっていても、説明できない状態を表しています。

２２
このように、母国語でも正確な意味を言えない状況から、言語は、以下の特徴があるの
かもしれません。

　〇　言葉（単語）の知識は「あいまい」でも、使えてしまう。

この状態を、言葉の学習である英会話に当てはめると、従来とは違う覚え方が出てくる
のではないでしょうか。
一般的な覚える学習は、知らない状態から、知る（分かる）状態になることが目的です。
そして新しい知識を持つと、さらに高度な学習へ繋がるので、覚える内容は、正確性が
必要になります。その一方で、言語の場合はどうでしょうか。
先程の言語の特徴から見ますと、あいまいに覚えても、大きな問題は無いと言えるの
かもしれません。一口に「会話」と言っても、単語だけの意思表示や流暢な場合など、
さまざまなレベルがあるからです。
ここから一般的な覚え方とは違う「あいまいに、覚える」行為は、「あり」ではない
かと推測できます。

例えば、作家の方は、一般の方以上に語彙力が必要です。
それなら、一般の方以上に「言葉を、覚える」学習をしたのでしょうか。
おそらくは、たくさんの本を読む行為や、書く行為から自然に言葉を覚えて、作品に
生かしていると思います。
このような現状を見ますと語学の学習は、正確な知識を増やす内容でなくても、自然
に覚えられるのではと推測できます。

これは慣れで覚える内容なので、覚えたかどうかの確認は、ほとんど行ないません。
「それじゃあ、覚えたかどうか分からないので、何の役にも立たないのではないか」
と思われるかもしれませんが、そうではありません。

このような新しい発見や気付きはインパクトがありますので、完璧に覚えなくても、
頭のどこかに「引っかかる」くらいの知識にはなります。
そこで「知識になっていない（まだ使えない）までの、記憶する」学習を重ねたり、
思い出す機会を作ることで、潜在的な向上があるのではないでしょうか。

２３
もちろん、覚える学習としては正確性に欠けるので、テストや資格など一定の結果が
必要な場合は使えません。
ですが会話（言葉）学習なら、レベルを決める必要もないので、この覚え方は使える
のではないでしょうか。さらには知識に「なっていない」記憶だからこそ、応用する
解釈や、柔軟な発想が出るかもしれません。
このように言葉の学習は「知識にならないまま、慣れから少しずつ覚えるられる」事
もあるのではないかと思いました。

－ － － － － － － － － － － － － － － － － －
１４　英会話を「話す」時のポイントは
－ － － － － － － － － － － － － － － － － －

「英会話は、使われる単語が限られている」「英会話は、中学英語でも理解できる」

これまでに読んだ書籍やインターネット情報には、こう書かれていましたが、ここから
分かることは何でしょうか。
それは「限られた英語が、さまざまな意味を持つので、いろいろな場面で使われる」事
だと思います。ここから「基本的な動詞や前置詞は、多くの意味を持つ」へ繋がります。
そして、英語が母国語の方は、自然に「限られた英単語を使った、さまざまな意味」を
覚えてしまいます。その一方で（母国語でない）英語を習う場合は、まったく同じ条件
や環境を得ることは、難しいのかもしれません。
例えば、留学しても、すでに母国語を持っていますし、年齢的にも、ネイティヴと同じ
ように覚える過程とは、違うのではないかと推測できます。

ところで、英会話は「会話」なので、読む／書くよりも、聞く／話すことが重要です。
そして話すよりも、先に「聞く」（音として、聞ける）状態が重要になっています。
その理由は、英会話学校の使い方に書いたのですが、それなら聞けるようになった後は、
話す時のポイントやコツはあるのでしょうか。

　〇　中学校で覚えた「基本的な英単語や文法」を使って、できるだけ
　　「簡単な状態で表わそうと」試みる。

２４
学校英語は「１つの英単語や表現＝１つの意味で覚える」傾向があると、以前にご紹介
しました。もし英会話でも、このように学習しますと－
「限られた単語や短いフレーズが、さまざまな意味になる」現状には、適していません。
「それなら１つの英語から、さまざまな意味を覚えればよいのではないか」と思われる
かもしれません。ですが、一度に複数の意味を覚えようとすると、量も多くなるので、
学習としては難しくなってしまいます。

例えば英和辞典を見ますと、１つの単語でも、数多くの意味が載っています。
ここから「さまざまな意味を、覚える」学習は、難しいのではないかと推測できます。
それに辞典に載っている意味は、すべてが重要だとも限りません。
そこで、話す機会があった時は「できるだけ基本的な英語を使って、表現する」ことを
意識します。例えば、何かが腐った状態を表したいとします。
この時に、腐る（ｓｐｏｉｌ）の単語を知らなくても、時間の経過から悪くなるので－
「ｇｏ　ｏｆｆ／ｇｏ　ｂａｄ」で表します。
（フレッシュじゃない、悪い臭い、食べられないなどをイメージした英語を使います）

すると最初は間違えたり、不自然な英会話に思われるかもしれません。
ですが、相手に通じたときは「こんなに簡単な表現で、伝わるんだ」という体験から、
（ある場面での）使い方は覚えやすいと思います。
このような「話す」機会を積み重ねますと、そのうちに－
「この単語は、こういうニュアンスを持っているのでは」と、感覚的に分かるのでは
ないでしょうか。（その単語の使い方が、説明できなくてもです）
そして、通じた英語（英会話の知識）は蓄積していくので、単語だけの会話から－
「一つ、上の」状態になるかもしれません。
ちなみに、英会話を「聞く」の面では、子供向けのアニメなどを利用します。
使われている単語は、比較的、優しいので（音としての）聞き取りも難しくありません。
すると、ある場面と英会話が関連付けられるので、覚えやすくなります。

－ － － － － － － － － － － － － － －
１５　偶然、必然（？）な英単語
－ － － － － － － － － － － － － － －

２５
ある程度の英単語を覚えたり、英会話で出てきた単語を見ますと－
「２つの単語は、スペルが似ている」「意味が近いのは、偶然だろうか」と思うことも
あるのではないでしょうか。
そして、後の学習から「アタリ（ハズレ）」が分かることもあります。
このような気付きや関連性は、英単語が覚えやすいので、少しだけご紹介したいと思い
ます。ですが、英語番組から引用した個所もあるので、間違っている可能性も否定でき
ません。

　〇　「ｄｅｖｉｌ＝（名詞）悪魔、悪い子」
　〇　「　ｅｖｉｌ＝（名詞）邪悪、（形容詞）悪質な」

語源が同じでも、品詞が違うと、たいていはスペルの後半が変わります。
ですが、この２つは前半になっています。実際に、同じ語源かどうかは分かりませんが、
意味は似ています。「ｅｖｉｌ」の発音は「イーヴィ（ル）」が近いです。
ちなみに、英語のアニメでは「不思議だね／ちょっと変だね」くらいの意味で－
「Ｔｈａｔ'ｓ　ｗｅｉｒｄ」が使われていました。

　〇　「ｉｎｎｏｃｅｎｔ ＝（形容詞）無罪の、天真らんまんな」
　　　「ｉｎｎｏｃｅｎｃｅ＝（名詞）無罪」

　〇　「ｉｇｎｏｒａｎｔ＝（形容詞）無知の、無学な」
　　　「ｉｇｎｏｒｅ　　＝（動詞）無視する、顧みない」

「ｉｎｎｏｃｅｎｔ／ｉｎｎｏｃｅｎｃｅ」は歌詞で使われたり、英単語としても重要
になっています。「ｉｇｎｏｒａｎｔ／ｉｇｎｏｒｅ」も、同じ「無」ですが、中身は
大きく違っています。
スペルの流れや、当たらずとも遠からずな意味（？）なので、ご紹介しました。

　〇　「ｖｅｎｄｏｒ」＝「ｖｅｎｄｅｒ」＝「（名詞）行商人、売る人」

なんとスペルが違っているのに、どちらも同じ意味になっています。

２６
これは携帯の辞書機能にも、紙の書籍にも載っていました。

　〇　「Ｔｈａｔ'ｓ　ｉｎｔｅｒｅｓｔｉｎｇ＝それは面白いね／興味があるね」
　　　
学校英語で有名な熟語は「ｂｅ　ｉｎｔｅｒｅｓｔｅｄ　ｉｎ～＝～に興味がある」
がありますが、上記は、ちょっとした関心がある時に使うようです。
また相手に対する相槌や、言いにくいことを「ぼかす」時にも使われるようです。

　〇　「Ｄｏ　ｙｏｕ　ｈａｖｅ　ａ　ｃｕｒｉｏｓｉｔｙ？」
　　　「あなたは関心があるの／気にならないの？」

ほとんどの辞典では「ｃｕｒｉｏｓｉｔｙ＝（名詞）好奇心」が載っています。
学生時は、このように覚えましたが、英語番組を見ますと、そこまでの堅い意味より
「興味／関心がある」程度のニュアンスになっていました。

　〇　「Ｙｏｕ'ｒｅ　ｉｎ　ｃｈａｒｇｅ」
　　　「後は、よろしくね／頼んだよ／お任せするね」

英語のアニメでは、留守番をお願いする相手に対して言っていました。
日本では、カタカナのチャージ／チャージャーがありますので、真っ先に「充電する」
意味が出てくるのではないでしょうか。
辞典で調べますと「ｃｈａｒｇｅ」の基本的な意味は「（動詞）請求する／負わせる」
でした。ここから電力を負わせるので、充電する意味が出てきたのかもしれません。
文頭から訳しますと「あなたは、そういう状態、負っている状態に入っている」から－
「あなたは、すでに責任を負う状態になっています」になって、最後は、上記の意味に
なります。

　〇　「Ｗｅ　ａｒｅ（ｆｕｌｌｙ）ｂｏｏｋｅｄ　ｔｏｎｉｇｈｔ」
　　　「今晩は、予約でいっぱいです」

英語番組では、お店の人（ｗｅ）が「予約で、いっぱいです」と言っている時に

２７
「ｒｅｓｅｒｖ／ｒｅｓｅｒｖａｔｉｏｎ」の発音がありませんでした。
その後、英語の書籍を見ている時に「ｂｏｏｋ＝（動詞）予約する」が載っていたので、
これが使われたのだと思いました。最初は「本」ではない意味に驚いたのですが、
よく考えますと、カタカナになっている「ダブルブッキング」があります。

　〇　「鯉の、池＝ｋｏｉ　ｐｏｎｄ」

「鯉は、広島カープの『ｃａｒｐ』じゃないの？」と思われるかもしれません。
ですが、インターネットの辞書では「ｃａｒｐ」を紹介している場合と、「ｋｏｉ」も
併記している場合がありました。お気付きかと思いますが、発音は「鯉」と同じです。
以前に見たテレビでは、観賞用のニシキゴイが、海外に輸出される場面があったので、
ここから英語の「ｋｏｉ」が生まれたのかもしれません。
まとめて覚えますと「池＝ｐｏｎｄ」も覚えやすいのではないでしょうか。
ちなみに、最近見たアニメでは、折り紙が「オリ（→）ガ（↑）ミ（↓）」の発音で、
英単語になっていました。

１６　英会話書籍の使い方
－ － － － － － － － － － － －

英会話学習のＣＭを見ますと、実際に、英語を話せるようになった方が登場しています。
すると「自身も利用すれば、話せるようになれるかもしれない」と思ってしまいますが、
なぜか英会話の書籍では、同様の内容を見たことがありません。
とちらも、同じ「英会話を、習得する」目的にもかかわらず、なぜ、英会話の書籍では
「英語が話せるようになった」内容のＰＲが無いのでしょうか。

また、これらを使う側から見ますと、大きな出費になるにもかかわらず、習得率を公表
している学校や教材は、あまり聞いたことがありません。
このような現状なので、手軽な英会話書籍を購入する方も多いではないでしょうか。
そして、その中では、ＣＤ付き書籍の人気が高いようです。
本章では、ＣＤブックが効率的に使えるかもしれない方法を、ご紹介したいと思います。

２８
　① 　書籍に載っている英会話の日本語訳を、先に覚えてしまいます。
　② 　付属の音声ＣＤから、英会話の場面「だけ」を、後から繰り返し聞きます。

英会話ブックの書籍は、ＣＤ音声の日本語訳や英文、詳細な説明など、内容が充実して
いますが、よく見ますと（その内容が）かぶっている個所もあります。
もし、このような書籍を説明書通りに使いますと、英語の量が多すぎるので、継続が
難しくなってしまうのではないでしょうか。そこで最初は、内容を絞ってしまいます。

① は、誰もが習得できる速読を使いますと、効率的になります。
そして、② は「日本語訳を見ながら、音声を聞く」状態ではありません。
このようにしますと、意識が「見る」日本語に偏ってしまうので、聞くだけにします。
そして、後から「なぜ日本語訳は、このような意味になるのだろう」
「単語のスペルが、知りたい」などの疑問や知りたい個所が出てきたら、補足として
書籍を見るのがよいのではないでしょうか。
すると、シンプルな学習方法になるので、英会話書籍が長く使えるのではないかと思い
ました。

１７　「Ｄｏ　ｙｏｕ　ｍｉｎｄ（ｉｆ）～」は「～してもよいですか」？
－ － － － － － － － － － － － － － － － － － － － － － － － － － － － － － － － － －

以前にも書きましたが、日本語と英語は「動詞の感覚」が違うので、日本人にとって、
英語の理解は難しいのではないかとご紹介しました。
例えば、日本語では動詞１つで表現できることが、英語では「ｂｅ動詞＋形容詞」や、
「動詞＋副詞（前置詞）」など、２語になっている場合があります。
そこで、英語の作り方／－ 前置詞の作り方では、動きを表す場合は「何が、どうなる」
の感覚で理解すればよいのではないかとご紹介しました。

もちろん、英語でも「１つの動詞＝１つの意味」の場合もあります。
ですが日本語とは違う感覚なので、理解や解釈が難しいときもあります。
例えば「ｍｅａｎ」を調べますと、辞典では「～を意味する」動詞（！）でした。

２９
これが動詞というのは、日本人にとって不思議ですが、おそらく日常的には－
「Ｉ　ｍｅａｎ～＝私（の意思、言いたい事）は、～です」の使い方や、相手の真意を
確認するために「（ｄｏ）ｙｏｕ　ｍｅａｎ～？＝あなた（の意思）は～ですか？」の
ように使われます。そして主語の「誰が（何が）」の個所が、はっきりしています。

この状態は、主語が曖昧でも通じてしまう日本語と違うので、ここでは、英語の感覚を
意識します。具体的には、自身から話すときに「Ｙｏｕ　ｍｅａｎ～」と言う場合は－
「相手からの、矢印」をイメージします。
すると、相手に「Ｄｏ　ｙｏｕ　～？」と質問しても、矢印の支点は相手にあるので、
直訳は「あなたは～ですか？」になります。
このような見方をしますと、堅苦しい日本語訳になりますが、英語の理解がしやすい
場合もあります。例えば、手元の辞典では、下記の例文が載っていました。

　〇　「Ｄｏ　ｙｏｕ　ｍｉｎｄ　ｉｆ　Ｉ　ｓｍｏｋｅ　ｈｅｒｅ　？」
　　　「（私が）ここで煙草を吸っても、構いませんか？」

ここで英文の矢印を意識しますと、直訳は「あなたは～？」になります。
ですが、上記の日本語訳は「私が」吸っても構いませんか？になっています。
この状態は、英語の学習としては分かりにくいので、矢印の支点を揃えてしまいます。

　〇　「Ｄｏ　ｙｏｕ　ｍｉｎｄ　　　　ｉｆ　Ｉ　ｓｍｏｋｅ　ｈｅｒｅ　？」
　　　「あなたは気にする状態ですか？　もし、私がここで煙草を吸ったら」
　　　「あなたは、ここで私が煙草を吸うのを、気にする状態ですか？」
　　　「あなたは、私の煙草が気になりますか？」

つまり「Ｄｏ　ｙｏｕ　ｍｉｎｄ～？＝あなたは～を気になりますか？」になります。
すると、相手が煙草を気にする（吸ってほしくない）場合は、自然な流れの返答は、
以下になります。

　〇　ＹＥＳ！＝Ｉ　ｍｉｎｄ＝私は、煙草が気になります＝吸わないでほしい

３０
もし先程の「（私が）ここで煙草を吸っても、構いませんか？」の、上手な日本語訳で
覚えたとします。すると「ＹＥＳ！」の返答は「構わない」という逆の意味をイメージ
してしまいます。
そして煙草が気にならないなら「Ｎｏ，ｐｒｏｂｌｅｍ＝問題ない」が、分かりやすく
なります。同様の内容は以下になります。

　〇　ＮＯ！＝Ｉ　ｄｏｎ'ｔ　ｍｉｎｄ＝私は、煙草は気になりません。
　　　　　　　　　　　　　　　　　　　＝吸っても構いません

ちなみに「ｓｍｏｋｅ　ｆｒｅｅ」の場所は、煙草を「吸ってもよい」場所ではありま
せん。日本語のように見ますと「ｓｍｏｋｅがｆｒｅｅな場所＝喫煙所」になりますが、
実際は、その逆になります。
これは「喫煙すること（ｓｍｏｋｅ）に、拘束されない（ｆｒｅｅの、本来の意味）」
から、「ｓｍｏｋｅ」から解放されている、煙草が「吸えない」場所（禁煙所）です。
このような「ｆｒｅｅ」は、最近では「ＳＩＭ（ロック）フリー」があります。

１８　なぜ「ｕｓｅｄ　ｔｏ～＝以前は（昔は）～だった」の意味になる？
－ － － － － － － － － － － － － － － － － － － － － － － － － － － － － － － － － －

辞典を見ますと「ｕｓｅｄ　ｔｏ～＝以前は～だった」の意味で、主に、過去の習慣や
状態を表しています。そして使い方は、以下の例文があります。

　〇　「Ｉ　ｕｓｅｄ　ｔｏ　ｄｒｉｎｋ」
　　　「私は以前に、お酒を飲んでいました（飲む習慣がありました）」

これは、それほど難しい単語ではありません。
ですから、そのまま「ｕｓｅｄ　ｔｏ～＝以前は～だった」と覚えても構いませんが、
ちょっとした疑問が出てきてしまいます。
それは「なぜ過去の状態を表すのに、過去形を使わないのか」です。
例文では、過去に飲んでいた習慣が「ｔｏ＋不定詞（動詞の原型）」になっています。

３１
そして「ｕｓｅｄ　ｔｏ～」は、辞典では助動詞とされている「ｈａｖｅ　ｔｏ～」に
似ています。（英語の作り方では、「ｈａｖｅ」は動詞ではないかと書きました）
すると文法的な見方で日本語に訳しますと、理解が難しくなるのではないでしょうか。
そこで、本章では「ｕｓｅｄ　ｔｏ～＝以前は（昔は）～だった」になるまでの流れを、
ご紹介したいと思います。

　－ 「ｕｓｅｄ」が過去形なら－

ところで動詞の「ｕｓｅ」は、ほとんどの方が「使う」の意味をご存じだと思います。
これは「習慣的に、使う」という、基本的な意味からきているそうです。
（「くせもの英単語帳」より）
すると「ｕｓｅｄ　ｔｏ～」の「ｕｓｅｄ」は、動詞である「ｕｓｅ」の過去形では
ないかと思いました。
このような見方をしますと、「ｕｓｅ　ｔｏ～」は以下の流れになります。

　〇　「Ｉ　ｕｓｅ　ｔｏ　ｄｒｉｎｋ」
　　　「私は習慣的に、お酒を飲むほうを使います」
　　　「私は習慣的に、お酒を飲みます」

そして、これが過去形の場合は、以下の流れになります。

　〇　「Ｉ　ｕｓｅｄ　ｔｏ　ｄｒｉｎｋ」
　　　「私は習慣的に、お酒を飲むほうを使いました」
　　　「私は習慣的に、お酒を飲んでいました」
　　　「私は以前に、お酒を飲んでいました（飲む習慣がありました）」

このような見方をしますと「ｕｓｅｄ　ｔｏ～」は、習慣的な意味である動詞
（ｕｓｅ）の過去形という解釈ができます。そして、この流れから－
「ｕｓｅｄ　ｔｏ～＝習慣的に、～を使った＝以前は～だった」の意味になった
のではないでしょうか。すると日本語訳の「（以前に）お酒を飲んでいた」は、
理解しやすくなります。

３２
そして「ｕｓｅ　ｔｏ～」の形は、あまり使われなくなったのかもしれません。
ただし、このような解釈では、発音に矛盾が出てきてしまいます。
動詞である「ｕｓｅ」の発音は「ユーズ」に近いですが、過去形は「ユーズドゥ」です。
すると、先程の解釈では「ｕｓｅｄ　ｔｏ～＝ユーズドゥ　トゥ～」になるはずですが、
実際は「ユーストゥ」に近い発音になっています。
しかし、これでは発音が難しいので「ユーズドゥ　トゥ～」が「ユーストゥ」に変わっ
てしまったのではないかと思いました。（間違っているかもしれませんが）
このような見方をしますと、過去を表す「ｕｓｅｄ　ｔｏ～＝以前は～だった」は理解
しやすくなります。

－ － － － － － － － － － － － － － － －
１９　英語の「上手な日本語訳」は
－ － － － － － － － － － － － － － － －

本書や英語の作り方では、上手な日本語訳になるまでの「流れ」を作りながら－
「何故この英語は、このような日本語訳や意味になるのか」をご紹介しました。
ですが「もしかしたら、こうなってるのではないか」と気付いてから、推測で書いた
場合もあるので、正確性には欠けてしまいます。
「何故あいまいな内容にもかかわらず、紹介したのか」と思われるかもしれません。
その理由は、本書のネタを考えたり英語書籍を読んでいたら、上手な日本語訳「だけ」
書かれている内容では、理解が難しくなるのではないかと思ったからです。
上手な日本語訳になるまでの流れは、以下になります。

　〇　英語（英文）→前から訳した、不自然な日本語訳（一応は直訳）
　　　→（直訳を並び替えた）上手な／自然な日本語訳

一般的な書籍では、上手な日本語訳だけ書かれている場合もあります。

ところで最後の上手な日本語は、一体何のためにあるのでしょうか。
もちろん「英語を、理解するため」ですが、英会話の学習なら、かならずしも必要とは
言えないのではないでしょうか。

３３
その理由は、英会話を聞いたときに、頭のなかで意味さえ分かれば問題ないからです。
すると、英語の理解で重要なのは、直訳や上手な日本語訳になるまでの「流れ」にある
のかもしれません。
例えば、ある英会話書籍には「ａｆｔｅｒ　ｙｏｕ＝お先に、どうぞ」と書かれていま
した。これは簡単な熟語なので、丸覚えが効率的になります。
ですが「なぜ、その意味になるのか」の理解としては「お先に、どうぞ」になるまでの
流れがあると、分かりやすくなります。

　〇　「（Ｉ'ｌｌ　ｂｅ）ａｆｔｅｒ　ｙｏｕ」
　　　「私は、あなたの後ろにいるつもり（の状態）です」
　　　「（私が、後ろなので）あなたが、先になります」
　　　「お先に、どうぞ」

ちなみに、同じ丸覚えでも「ｙｏｕ　ｆｉｒｓｔ＝（あなたが）お先に、どうぞ」は、
直訳に近いので、そのままでも構いません。
そして最後の上手な日本語訳は、英語関係のお仕事や試験で必要な、書く／話す内容
ではないでしょうか。
例えば学校英語では、試験や入試の関わりがあるので（英会話の学習ではないので）
試験では、上手な日本語訳を求められます。
これでは、より重要な「英語の理解」が見えにくいのかもしれません。
そこで、テストの採点は「模範解答に近いか（上手な日本語訳に、なっているか）」
よりも「内容を、理解しているか」を基準にしたほうがよいのではないでしょうか。

また、これまで読んだ英会話書籍は「先に英語、その後（や下行）に、日本語訳」が
書かれていることが多くありました。
「英語の内容だから、当たり前のことではないか」と思われるかもしれませんが、
書籍を読むのは、日本人です。
感覚的に覚えることが目的なら、先に日本語訳を書いたほうが分かりやすくなります。

２０　「英語の矢印」を詳しくすると
－ － － － － － － － － － － － － － － － －

「英語の（前置詞の）作り方」では、英語の「支点」や、方向を意識した「矢印」を
意識すると理解しやすいと書きました。
そして、たいていは一つの例文でしたが、より詳しく説明できるかもしれないことに
気が付きましたので、ご紹介したいと思います。
ここでは、会話の前後の場面から説明します。（逆に、ややこしいかも知れません）

　〇　（ｙｏｕ）ｇｅｔ　ｂａｃｋ！→（支点）動作　方向→
　　　（あなた）得る　　後ろ　　　→戻れ／下がれ！

これは、以前に「命令形は、このようになっているのでは？」と書いた内容です。
そして、前の状態は「Ｉとｙｏｕが、一緒にいた」場面なら、以下の時間の流れになり
ます。

←過去　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　未来→

　Ｉとｙｏｕが一緒→ｙｏｕが、Ｉから離れた→Ｉが「ｇｅｔ　ｂａｃｋ」と言った
　　　　　　　　　（ｙｏｕ　ｇｏ　ｏｕｔ）

中央の「ｙｏｕが（Ｉから）離れた場面」は－
「Ｉとｙｏｕの支点から、ｙｏｕが、外に行った」矢印があると言えるのではないで
しょうか。この矢印があるので、「ｙｏｕ」が（Ｉに）戻ってほしい場合は－
「外に行ったＩ→もとの支点」の、戻る矢印が出てきます。
「（ｙｏｕ）ｇｅｔ　ｂａｃｋ！」は、このような矢印があるのではないでしょうか。
また以下の会話からも、矢印が見えてきます。

３５
　〇　母親　「Ｄｉｎｎｅｒ　ｉｓ　ｒｅａｄｙ、（ｙｏｕ）ｃｏｍｅ　ｏｎ！」
　　　子供　「Ｉ　ａｍ　ｃｏｍｉｎｇ！」

最初の母親のセリフは「子供（がいる場所）から、母親へ向かう」矢印があります。
そして、子供の「今、行く！」の行動は、母親の「来て」の矢印に沿いますので、
「母親の居る場所に、来る＝ｃｏｍｅ」を使うのではないでしょうか。
ちなみに、ここで子供が「Ｉ　ａｍ　ｇｏｉｎｇ！」と答えると「（外に）出かける」
（夕食は、食べない）意味になるようです。
これは、「ｇｏ／ｃｏｍｅ」が「対」のニュアンスを持っているからかもしれません。
また、ここから「ｇｏ／ｃｏｍｅ」の日本語訳が、そのまま「行く／来る」の意味に
ならないことが分かります。
このような英語の矢印を意識しますと「ｇｏ／ｃｏｍｅ」「ｔａｋｅ／ｂｒｉｎｇ」
の使い方や意味の違いは、理解しやすいのではないでしょうか。

２１　英会話の「消える」発音とスペルは
－ － － － － － － － － － － － － － － － － － －

英語のリスニング力が上がると、今までは聞き取れなかった個所が、聴き取れるように
なります。
そして同時に、リスニング力が上がっても、やっぱり聞き取れない（発音していない）
個所があることにも気が付きます。
例えば、日常英会話で使われる「Ｉ　ｃａｎ'ｔ（～）」は、学校の英語では－
「アイ　キャント（ゥ）」と、しっかり発音しているのではないでしょうか。
（カタカナなので、大まかになっています）
そして、スペルは「ｎｏｔ（ノット）」の「ｎｏ（ノッ）」が省略されているので、
「アイ　キャント（ゥ）」は、自然な言い方になっています。

しかし、実際の英会話は「アイ　キャン」しか聞こえない場合もあります。
「それでは、Ｉ　ｃａｎ～と区別できないではないか」と思われるかもしれませんが、
これは、その個所（場面）だけを見た場合です。

３６
例えば、英語のアニメ番組では「Ｉ　ｃａｎ'ｔ」の意味のところで「アイ　キャン」
と聞こえる場面（「ｔ」は、聞こえない場面）がありました。
この時に気が付いたのですが、話の流れや、セリフを言ったときの表情で推測すると、
「Ｉ　ｃａｎ／Ｉ　ｃａｎ'ｔ」は区別できるのではないかと思いました。
するとネイティブの方は、無意識的にトータルな判断をしているのかもしれません。
また、学生時は－
「Ｉ　ｃａｎは、Ｉを強く発音して、Ｉ　ｃａｎ'ｔは、ｃａｎ'ｔを強く発音する」
と習った記憶があります。ある英会話書籍には、以下の内容が書かれていましたが、
この状態がよく表れていると思います。

　〇　英語は、実際の発音よりも、アクセントで聞き取る（判断する）傾向がある。

「Ｄｏｎ'ｔ　ｙｏｕ　ｋｎｏｗ？」は「ドン　チュゥ　ノゥ？」に聞こえますが、
この発音は、スペルに残っている「ｔ」は、ほとんど聞き取れません。
ですが「ｎ'（ン）」の発音があるので、「Ｄｏ（ドゥ）ｙｏｕ　ｋｎｏｗ？」と
区別できるのではないでしょうか。
ちなみに「Ｔｈａｎｋｓ　ａ　ｌｏｔ」は「色々と、ありがとう」になります。
ここで、相手（のおせっかい）を皮肉るなら「ａ　ｌｏｔ」が強い発音になるようです。
このような例を見ますと、英語は直訳で理解するのでなく、相手の表情や状況などで、
判断する必要があるようです。

そこで、それらしき単語を探したところ、以下ではないかと思いました。

３８

２３　英語の前置詞は　その２
－ － － － － － － － － － － － － －

英語書籍を見ていますと、英単語は、言葉が分かれたり引っ付いたりしているのでは
ないかと思いました。
例えば「ｗｅｌｌ（良い）＋ｃｏｍｅ（来ること）」で「ｗｅｌｃｏｍｅ（歓迎）」

３９
かと思っていましたが、実際は「ｗｉｌｌ（喜び）＋ｃｏｍｅ」と書かれていました。
予想通りだったのは「ａｌｌ＋ｒｅａｄｙ」から「ａｌｒｅａｄｙ」でした。
そして「ｔｉｌｌ」は「ｕｎｔｉｌ」の省略で、使い方が分けられる程度の、同じ意味
でした。さらに「ｏｎ／ｕｐｏｎ」は、同様の意味ですが、「ｕｐｏｎ」から分裂した
（？）「ｕｐ」もありますし、前置詞が合体した（？）「ｉｎｔｏ」もあります。
また「ｉｎ＝～の中に」「ｔｏ＝～の方へ」の意味が有名ですが、２つを合わせて－
「ｉｎｔｏ～＝～（の方）へ入っていって」と理解すると、動的な状態がイメージしや
すいのではないでしょうか。

辞典に書かれていて驚いたのは、「ｏｆ」と「ｏｆｆ」の語源が同じことでした。
「ｏｆ」と言えば、「ＡｏｆＢ」（ＢのＡ）の日本語訳が多いのですが、辞典を引くと
「ｏｆｆ」は「離れて」の意味がありました。
日本では「ｏｎ／ｏｆｆ」のイメージが強いので、これは意外に知られていません。
その他には「ａｒｏｕｎｄ」「ｍａｙｂｅ」「ｆｏｒｅｖｅｒ」「ｗｉｔｈｉｎ」は、
その意味から推測すると、２つの単語が引っ付いたのかもしれません。
イギリス英語では「ｆｏｒ－ ｅｖｅｒ」と２語のように書かれるようですし、辞典でも
このように書かれています。そして、語源といえば「ｓｈａｒｐ（鋭い）」－
「ｋｅｅｎ（鋭敏）」「ｂｏｍｂ（爆撃する）」「ｂｕｒｎ（燃やす）」
「ｂｕｒｓｔ（破裂する）」などは、擬音から生まれたのではと推測できます。
このように語源を意識しますと、より英単語が理解しやすいのではないでしょうか。
そして辞典を見ているだけでも、何らかの発見があるかもしれません。

２４　「ｂｅ動詞」と「ｄｏｅｓ」と基本的な英会話は
－ － － － － － － － － － － － － － － － － － － － － － － － －

英会話で使われる「（Ｉｔ）ｌｏｏｋｓ／ｓｏｕｎｄｓ　ｌｉｋｅ～」は、日本人でも
分かりやすいのではないでしょうか。前から訳しますと、以下の流れになります。

　〇　「（Ｉｔ）　ｌｏｏｋｓ／ｓｏｕｎｄｓ　ｌｉｋｅ～」
　　　「（それは）　　見える／聞こえる　　　～のように」

４０
　　　「それは、見える（聞こえる）ね、～のように」
　　　「（それは）～のように見える（聞こえる）」

これは、単語を当てはめるように訳せてしまいます。
本章では、日本語訳を並べ変えながら、基本的な英会話を理解したいと思います。
（例文は「ＰＯＰＥＹＥ超簡単ブロークン英会話」より抜粋）

　〇　「Ｈｏｗ　ｌｏｎｇ　ｄｏｅｓ　ｉｔ　ｔａｋｅ　ｂｙ～？」

　　　「Ｈｏｗ　　　ｌｏｎｇ＝ｄｏｅｓ　＝ｉｔ　　ｔａｋｅ　　ｂｙ～？」
　　　「どれくらい　長い　　＝動的な状態＝それを　移動すると　～で？」
　　　「どれくらい長い動的な状態になる、それを～で移動すると？」
　　　「（それを）～で移動すると、どれくらい長い状態？」
　　　「～を利用して、どのくらい時間がかかりますか？」

行き先や目的地までの所要時間を聞く、基本的な形です。
そして所要時間は、実際の移動による「かかった時間」なので、日本語訳は、単に－
「時間を聞いている」内容にしか見えません。
ですが英語の内容は、動的な状態（移動）から、結果どうなる（どの時間の長さになる）
かを聞いているのではないでしょうか。
そして、そうなっている状態の疑問文なので「ｄｏｅｓ」が使われています。
次の例文は、そういう状態を表す「ｂｅ動詞」が使われます。

　〇　「Ｈｏｗ　ｆａｒ　ｉｓ　ｔｈｅ　ａｉｒｐｏｒｔ　ｆｒｏｍ　ｈｅｒｅ？」

　　「Ｈｏｗ　　　ｆａｒ＝ｉｓ＝ｔｈｅ　ａｉｒｐｏｒｔ　ｆｒｏｍ　ｈｅｒｅ？」
　　「どれくらい　遠さ　＝状態＝その　　空港　　　　　　　　　　　ここから？」
　　「どれくらいの遠い状態ですか、その空港は、ここから？」
　　「ここから空港まで、どのくらい（の距離が）ありますか？」

空港までの（物理的な）距離を聞いているので、動的な状態ではありません。

４１
ですから「ｂｅ動詞」が使われます。次は、ふたたび「ｄｏｅｓ」の例文です。

　〇　「Ｈｏｗ　　　ｍｕｃｈ　　ｄｏｅｓ　　　　ｉｔ　ｃｏｓｔ　　？」
　　　「どれくらい　量（程度）　動的な状態　　　それ　費用をかける？」
　　　「どの程度の量の状態になりますか、　それに費用をかけると？」
　　　「（それは）いくらになりますか？」

「値段は動的な状態では無いので、ｂｅ動詞が使われるのではないか」と思われるかも
しれません。
もしかしたら、２通りの「ｈｏｗ　ｍｕｃｈ～？」があるのではないでしょうか。
１つは、上記の「ｄｏｅｓ」が使われている「いくらか？」です。
これは「ｉｔ」に「ｃｏｓｔ」をかける状態なので、例えば、何かを造るときの－
「いくらか？」を表わします。
すると「ｉｔ」の費用は（何かを、造る）動的な状態にあたるので「ｄｏｅｓ」が使わ
れます。そして、もう１つは買い物で「モノの値段（状態）」を聞く以下になります。

　〇　「Ｈｏｗ　　　ｍｕｃｈ　ｉｓ　ｔｈｉｓ？」
　　　「どれくらい　値段　　　状態　これ？」
　　　「どれくらいの値段の状態なの、これは？」
　　　「（これは）おいくらでしょうか？」

このような見方をしますと「ｂｅ動詞」と「ｄｏ／ｄｏｅｓ」の違いが見えてきます。

これは、発音練習に見えるかもしれませんが、子供の年齢から見ますと、おそらくは
「英単語の意味を、覚える／英語に親しむ」ことが目的ではないでしょうか。
そしてカードを使った学習と、文字や記号を使った発音練習は、共通した要素を持って
います。それは、上記の「１単語ずつ、覚えていく」内容です。
しかし実際の英会話では、複数の単語が「繋がるように」発音したり、単語の一部は、
発音しないこともあります。

すると１つの単語の正しい発音を覚えても、そのまま使える機会は少ないので、英会話
の発音練習としては非効率になっています。
それに発音が（音として）聞き取れない状態なら、正解な発音は自身でも分かりません。

ちなみに、ある英会話書籍では「英語の発音は、音声学を習うと理解できる」と載って
いましたが、ネイティブの方は習わなくても、正しい発音を「耳で」覚えてしまいます。
このような見方をしますと、母国語は、以下のように覚えるのかもしれません。

　〇　英語の発音が「聴ける」耳の状態になる。
　〇　自然に「聞こえる」発音を真似て、言葉を覚えながら話せるようになる。

４３
そして（母国語でない）英会話を学習する場合は、できるだけ上記の状態に近づけると
よいのではないでしょうか。つまり、発音は「耳で」覚えてしまいます。
そして言葉（単語）を覚えるときも、会話や、書籍を使った内容になります。

２６　日常英会話で必要な発音レベルは
－ － － － － － － － － － － － － － － － － －

本書で参考書籍とさせて頂いた「ＰＯＰＥＹＥ超簡単ブロークン英会話」では－
「英語は母音が優先されるので、そこを押さえれば意味は通じる」と書かれていました。
これは、前章でご紹介した音声学を習われた、秋尾沙戸子さんが書かれた内容です。
そして、例えば「ｂａｎａｎａ」を発音するときは、真ん中の「ｎａ」を強調すると、
前後の発音は、あいまいでも通じてしまう（！）そうです。
このように、発音のポイントを押さえると聞き取れてしまうのは何故でしょうか。
その理由は英文法がそれほど難しくない事と、おそらくは前後の文脈から、意味を判断
しているからではないでしょうか。

ところが、日本人が英会話を習うときは、あまりにも正確な発音にこだわり過ぎている
ような気がします。そして「正確な発音でないと、意味は通じない」と思ってしまい、
この先入観から、英単語だけの発音練習が出てきたのかもしれません。
もちろんネイティブ並みの英語力を求めるなら、高い正確性も必要になります。
ですが、多くの方が望まれているであろう、日常的な英会話は－
「発音のポイントを押さえて、意味が通じる」程度で十分ではないでしょうか。
これは、先程の「ｎａ」を強調した「ｂａｎａｎａ」と同じレベルです。

すると、日常的な英会話の発音は「１つの英単語の、正確な発音ができる状態」よりも、
「相手に理解してもらえる程度の、発音レベル」で十分だと思います。
「１つの英単語の、正確な発音」は、英会話で使う機会が多くありません。
ですから、このような発音練習は、英会話の練習としては適していないと思います。

２７　「Ｗｈｅｒｅ　ｄｏ　ｙｏｕ　ｇｏ？」に「ｔｏ」が付いたら
－ － － － － － － － － － － － － － － － － － － － － － － － － － － － － － －

　〇　「Ｗｈｅｒｅ　ｄｏ　　　　ｙｏｕ　ｇｏ　？」
　　　「どこの　　　動的な状態　あなた　行く　？」
　　　「どこなの、あなたが行く（動的な）状態は？」「どこへ行くの？」

以前にも言葉を並べ変えながら、基本的な英会話をご説明しました。
本章では「ｔｏ」が付いたら、どのような意味やニュアンスになるのかをご紹介したい
と思います。
「ｔｏが付いても付かなくても、それほど意味は変わらないのでは」と、思われるかも
しれません。もちろん、そのような場合もあると思いますが、「ｔｏ」の有無で意味が
違ってしまう例文がありました。

　〇　「Ｗｈｅｒｅ？」「どこ？」
　〇　「Ｗｈｅｒｅ　ｔｏ？（Ｗｈｅｒｅ　ｗｏｕｌｄ　ｙｏｕ　ｌｉｋｅ　ｔｏ？）」
　　　「どちらへ、向かわれますか？」

前者は「どこ（どの場所か）？」を聞いている意味になりますが、後者は、どこへ－
「向かうか」を聞く意味になっています。
すると「ｔｏ」だけでも、強い意味を持つこともあるのではないかと推測できます。
そして、冒頭の「Ｗｈｅｒｅ　ｄｏ　ｙｏｕ　ｇｏ　？」に「ｔｏ」を付けますと、
以下の違いが出てくるのではないかと思いました。

　① 　「Ｗｈｅｒｅ　ｄｏ　ｙｏｕ　ｇｏ？」「どこへ行くの？」
　② 　「Ｗｈｅｒｅ　ｄｏ　ｙｏｕ　ｇｏ　ｔｏ？」「どこの目的地へ行くの？」

一般的に「ｔｏ（～）」は、「～へ／～のために」と訳されます。
これは、行き先や目的を示す重要な役割になっていますが、もし、ここでも同様の意味
なら、① ② は「違い」があるのではないかと推測できます。

４５
順番にご説明しますと、① は（行き先や目的を示す）「ｔｏ」がありません。
すると「ｙｏｕ」に対して「具体的な行き先を、聞いている」内容ではなく－
「何故、ここを離れるのか？」のニュアンスがあるのではないでしょうか。
そして、次の② は「ｔｏ」の行き先が「ｗｈｅｒｅ」に当たるので－
「具体的な行き先を、聞いている」内容になっていると思われます。
このように① ② は「ｔｏ」が付くことで、微妙にニュアンスが変わってしまいます。
（当たっているかどうか分かりませんが）

２８　英語のスペルが、少しだけ覚えやすくなる方法
－ － － － － － － － － － － － － － － － － － － － － － － －

一般的に英語の学習は、小文字で覚えることが多いと思います。
子供が習う「英単語カード」は大文字かもしれませんが、これは、英語に親しむ目的に
なっています。
ところで英単語を覚える時は、大文字、小文字のどちらが覚えやすいのでしょうか。
インターネットでは、ほとんどのアドレスが「半角、小文字」ですし、英文を見ても、
たいていは小文字が使われています。

そして、ここで英文のすべてを大文字にすると、どのような印象になるのでしょうか。
おそらくは小文字が使われた英文よりも、読みにくくなります。
例えば「ＡＰＰＬＥ」と「ａｐｐｌｅ」を、１つの形として見たとしますと、小文字は
「凸凹が、大きい」特徴があります。
もしかしたら英語圏の方は、このような印象を持たない（どちらも同じ文字に見える）
のかもしれませんが、そうでない方は、理解する時間に差が出てくると思います。
何故このような差が出てくるのでしょうか。

以前に本で読んだのですが、脳内で文字（視覚的）情報を処理するときは、形を認識
する特定の場所があるそうです。
すると、ある程度は特徴があったほうが認識しやすいので、小文字のほうが覚えやすい
のではないでしょうか。

４６
そこで、この特徴を利用しますと、英語のスペルは覚えやすくなると思います。

　〇　小文字で英単語を覚えるときは、文字の凸凹を意識します。

例えば、カタカナで「プレイ」と書くと、多くの方は「ｐｌａｙ」を思い浮かべます。
ですが、英語では「ｐｒａｙ＝（動詞）祈る」もあります。
もちろん発音は違いますが、ここで小文字の「ｌ」の出っ張りを「バットが立っている」
イメージを持ちます。
すると「ｐｌａｙ」と「バット」が関連付けられて、覚えやすいと思います。
ここで、単に「出っ張りのある単語が、〇〇」でもよいかもしれません。
ちなみに、出っ張りのない方（ｐｒａｙ）は、そうでない方として覚えてしまいます。

２９　ディクテーションの効果は
－ － － － － － － － － － － － － － －

英語書籍を見ますと「ディクテーション」という、学習方法が載っていました。
現在でもスペルや語源を知らないのですが、書籍によりますと－
「聞いた英語を、できるだけ文章や文字として書き起こす」内容のようです。
この内容を英会話の学習に当てはめますと、どうなるでしょうか。
英会話は「耳で聞いて（理解して）その場で話す」内容になっています。
すると聞いたときに理解できればよいので、書く作業は、それほど必要性が無いことに
なります。
「ディクテーションは書けない英単語が分かるので、英会話でも意味があるのでは」と
思われるかもしれません。英会話書籍にも、同様なことが書かれていましたが、確かに
スペルの分からない英単語は分かります。
しかし英会話は、聞く／話す行為です。
聞くときに必要なのは「発音が聴き取れて、意味が分かる」状態ですし、話すときに
必要なのは「会話に必要な英単語の知識を、持っている」状態が基本になります。
そして、この状態は上手な発音や正しい文法、単語のスペルは知らなくても問題ない
のではないでしょうか。

４７
例えば、英会話で意味が分からなかったら－
「Ｗｈａｔ ｄｏｅｓ～ｍｅａｎ？」「Ｗｈａｔ　ｉｓ　～？」と聞けますし、相手の
意図なら「Ｗｈａｔ　ｄｏ　ｙｏｕ　ｍｅａｎ？」で対応できてしまいます。
そして、この時に「～（の発音）」と意味が分かるので、英会話の練習にもなります。
（英会話を話す学習としては、一番のメリットかもしれません）
するとディクテーションは、受験や試験、テスト対策に向いているのかもしれません。
単語のスペルは、リスニングの解答になっている場合が多いからです。
それに文法が重視されているので、スペルや語彙力は重要になっています。

－ － － － － － － － － － － －
３０　英会話教材の会話は
－ － － － － － － － － － － －

英会話書籍を見ているうちに、あることに気が付きました。
書籍で使われる会話は「目的を持って話す内容が、多い」ので、必然的に旅行や買い物
などの「その場面だけで、使える」内容になってしまいます。
すると、さまざまなタイプの書籍でも似たような内容になるので、実際の英会話の一面
になっているのかもしれません。
それでは英会話教材に「載っていない」会話は、どのような内容でしょうか。
英語に限らず「会話」は、他愛のない話や雑談など、特定の場面で使える（役に立つ）
とは言えない場合があります。これは、それほどの意志や明確な目的はありません。
しかし、このような「他愛のない（何気ない）」会話こそが、英会話書籍で見られない
内容だと思います。
「それなら他愛のない会話も、英会話教材で使えばよいのではないか」と思われるかも
しれません。
ですが、あまりに幅広い内容になるので、書籍やＣＤブックに収録するのは難しい状態
です。このような現状では、書籍「以外の」会話を聞くことが、学習になるのではない
でしょうか。

　〇　日常会話が含まれる、海外のアニメやドラマを観たり、英会話学校以外で、
　　　ネイティブの方の話を「聞く」機会を持つ。

４８
「話す」機会になりますと「何か、話さなければいけない」ストレスや焦りが出るので、
聞く機会としました。そして聞く行為だけでも、ちょっとした会話から－
「このような場面で、このフレーズは使えるのか」と気が付けます。
このような見方をしますと、英会話「以外の」ソフトでも、学習に使えるのでは
ないでしょうか。

－ － － － － － － － － － － － － －
３１　耳から英単語を覚えると
－ － － － － － － － － － － － － －

二ヶ国語放送の番組を視聴しますと、比較的、簡単な英単語なら、正確な発音と意味を
覚えてしまいます。（リスニング力を上げている状態が前提です）
これは聞く行為からの記憶なので、覚えた英単語は、会話（聞く／話す）で使える状態
です。ですから読む／書くに関する筆記テストでは、役に立たないかもしれません。

しかし、それでも役立つ面はあると言えるのではないでしょうか。
例えば聞いて覚えた単語は、後から学校で習うと、すでに発音は「ばっちり」です。
この状態はリスニング（テスト）に役立ちますし、予備知識でもあるので、覚えやすい
と思います。

ある英会話書籍では「英単語は、文字を使うよりも、音（発音）のほうが覚えやすい」
と書かれていました。
すると、英会話に必要な英単語を覚えたい時は、このような（耳から）覚える方法でも
よいのではないでしょうか。覚えるまでの「流れ」は、以下になります。

　〇　ある程度の回数で、同じ内容の英会話を、繰り返し聞きます。
　　　→簡単な英単語だけでも、正確な発音と意味が分かるようになります。
　　　→学校で習えば覚えやすいですし、覚えた発音からスペルも調べられます。

一般的に、覚える行為は「覚えたか、覚えていないか」で判断されてしまいます。
ですが、その間には「あやふやに／一部でも覚えた」状態があるのではないでしょうか。

４９
そして英会話の学習は、あやふやな状態を繰り返す（記憶を、塗り重ねる）事によって、
そのうちに覚えられる（会話で使える）こともあるのではないかと思います。

－ － － － － － － － － － － － － － － － － － － － －
３２　覚えやすい英単語、意外な意味の英単語
－ － － － － － － － － － － － － － － － － － － － －

ここでは、印象に残りやすかったり「こんな意味が、あるの？」と思ってしまいそうな、
比較的、覚えやすい単語をご紹介したいと思います。

　－ １－
例えば、英会話で「アイ、スゥー、ユー！」と聞こえたとします。
ここで場面や状況が分からない状態なら、どこまで理解できるでしょうか。
「基本的な文型なら、スゥーは動詞で『私は、あなたをスゥーする』ではないか」と
思われるかもしれません。それなら「スゥー」の動詞はあるのでしょうか？
辞典では、以下の英単語が載っていました。

　〇　「ｓｕｅ＝（動詞）訴訟を起こす／懇願する」

相手に強く「Ｉ　ｓｕｅ　ｙｏｕ！」と言えば、バラエティ番組でおなじみの－
「（あなたを）訴えてやる！」があります。これは、英語ドラマで使われていましたが、
カタカナの「スゥー」に近い発音でした。
ところで「ｓｕｅ」は、どこか聞き慣れない印象があるのではないでしょうか。
これは、スペルの「短さ」や、それほど身近でないという理由もありますが、ほかにも
以下の状態が考えられます。

　〇　たいていの英単語は、一つの単語にアクセント（発音の強弱）が付いている。

おそらく「ｓｕｅ」は、ほとんど「強」しかありません。
このような特徴もあるので、覚えやすい面があるのではないかと思います。

５０
　－ ２－
すでに日本では、カタカナの「レシピ」が調理法として使われています。
これは英語の「ｒｅｃｉｐｅ」が語源になっていますが、辞典では、名詞として載って
いました。そして、その意味も「調理法／処方箋」と書かれています。
つまり「レシピ」は、英語と同じ使い方ですが、もし、この料理法が一般的でない
「秘伝の／内密の」内容になりますと、どうなるでしょうか。
英語番組では「（ｓｅｃｒｅｔ）ｆｏｒｍｕｌａ」を使っていました。

　〇　「ｆｏｒｍｕｌａ＝（名詞）決まった言い方／習慣／方式」

おそらく、カーレースの「Ｆ１」（フォーミュラ１）と同じです。
すると「ｒｅｃｉｐｅ＝（名詞）調理法」よりも厳格で、形式的な状態を表わしている
のかもしれません。

　－ ３－
最近では「百貨店／デパート」という言葉は、それほど使われません。
辞典で調べますと「百貨店／デパート」は、「ｄｅｐａｒｔｍｅｎｔ　ｓｔｏｒｅ」が
載っていました。
すると「ｄｅｐａｒｔｍｅｎｔ＝多い／たくさん」をイメージしそうですが、実際は、
逆の意味になっています。

　〇　「ｄｅｐａｒｔｍｅｎｔ＝（名詞）部門／省／課」

ビジネス英語では「〇〇課」の「課」に当たります。
「デパートの英語が、一部を表わす意味とは・・」と意外に思われるかもしれません。
ですが以下のように理解しますと、覚えやすいと思います。

　〇　さまざまな「ｄｅｐａｒｔｍｅｎｔ」（部門）が集まった「ｓｔｏｒｅ」
　　　（お店）が、百貨店（デパート）になっている。

５１
－ － － － － － － － － － － － － － － －
３３　旅行時の英語を覚えるときは
－ － － － － － － － － － － － － － － －

「書籍の英会話は、目的を持って話すので、ある場面だけで使える内容ではないか」と
以前にご紹介しました。そして特定の場面なので、使われる単語は限られてしまいます。
すると例えば「ある場面」である旅行時の英語は、使われる単語を完璧に覚えなくても
よいのではないかと思いました。
このような場面では、会話が限られますので、想定問答のようになっているからです。
ですから「旅行のために、英会話を覚えないといけない」時は、書籍の内容を「完璧に」
覚えるのではなく、「繰り返し、軽く見る」程度にしてもよいのではないでしょうか。
たとえ完璧に覚えていなくても、実際の英会話を聞くことで思い出しやすくなります。

このような意識を持ちますと、旅行前の「覚えないといけない」意識は、少しだけ楽に
なるのではないかと思います。
また、この状態を「日常英会話の学習」に当てはめますと、以下になります。

　〇　英会話に必要な単語や英文を、覚える意識は持たないで「繰り返し、見る」
　　　（繰り返し、読む）機会を作る。

　〇　日常英会話を「繰り返し、聞く」機会を作る。
　
このような状態の積み重ねによって、少しずつ、英会話の理解力が上がっていくのでは
ないかと思いました。

－ － － － － － － － － － － － － － － － － －
３４　洋楽の英語と、カバー曲の違いは
－ － － － － － － － － － － － － － － － － －

別著「～英語のリスニング力が上がる方法」の効果を実感されますと、英語の
聞こえ方が変わってしまいます。

５２
すると、英会話や洋楽を聞くときはプラスになりますが、反対の場合もあります。
例えば、邦楽の英詞（の部分）や、カバー曲を聞くときに表れます。
カバー曲を聞くと、オリジナル曲は発音していないのに、歌っている場合があります。
すると、発音する個所が増えますので、この状態から－
「リズムに乗って、自然に歌っている」よりも「タイミングを、合わせているように」
聞こえてしまいます。
また、歌っている方が「どれくらい聞き取れているか」およその推測ができます。

もしかしたら丁寧に歌っていることが理由かもしれませんが、結果的に、オリジナル曲
とは違う印象になってしまいます。
すると、これまで聞いていたカバー曲でも違和感を感じるので、聞かなくなってしまう
場合もあるのではないでしょうか。
リスニング力が上がると、オリジナルのアクセントと違うことに気が付いてしまいます。
洋楽／邦楽のどちらも聴かれる方は、ここではマイナスの要素になるかもしれません。

そして、この状態は、音楽でなくても言えるのではないでしょうか。
以前に「英語は母音が優先されるので、そこを押さえれば意味は通じる」とご紹介しま
した。つまり英会話は、単語の正確な発音よりも、リズムやアクセントから意味を判断
する言語のようです。すると「英語の発音が、よくなった状態」は、正確には－
「英語のリズムで、自然な発音ができる状態」と言えるのかもしれません。
このような見方をしますと、発音練習は「正確に、発音しようとする」よりも－
「英語のリズムを得てから、聴き取れる発音を真似する」ほうが効果的になります。
つまり、先にリスニング力が必要になっています。
リスニング練習と発音練習は、それぞれ「聞く」「話す」内容なので、一見しては別々
の内容に見えてしまいます。
ですが実際は、先にリスニング力を上げて、後から、発音練習ができる「流れ」がある
ようです。これは、発音の自然な覚え方といえるのではないでしょうか。

－ － － － － － － － － － － － － － － － － － － － － － － －
３５　英単語の意味が、少しだけ覚えやすくなる方法
－ － － － － － － － － － － － － － － － － － － － － － － －

５３
英和辞典を見ているうちに、あることに気が付きました。
それは、１つの英単語が「・」で区切られている場合があることです。
例えば、以下のようになっています。

　〇　「ｆｏｒｅｖｅｒ＝（副詞）　永久に／いつまでも」
　　　→「ｆｏｒ　ｅｖｅｒ」「ｆｏｒ・ｅｖｅｒ」

　〇　「ａ・ｂｒｏａｄ＝（副詞）　広く、海外へ」

このような区切りは電子書籍に載っていませんでしたが、何を表わすのでしょうか。
紙の辞典では「切れ目」と書かれていましたが、おそらくはアクセントに関係している
と思います。すると発音が分かりやすい表記になりますが、同時に、単語の語源や意味
がイメージしやすい面もあるのではないでしょうか。
例えば「ｄｅｐａｒｔｍｅｎｔ」は、手持ちの辞典では「ｄｅ・ｐａｒｔ・ｍｅｎｔ」
になっています。そして、その意味は「（名詞）部門／省／課」です。
ここで「・」の区切りを見ますと、「ｐａｒｔ（名詞）部分」が含まれていますので、
「ｄｅｐａｒｔｍｅｎｔ」の意味がイメージしやすくなります。
また、冒頭の「ａ・ｂｒｏａｄ」も「ｂｒｏａｄ＝（形容詞）　幅広い」があります。
（おそらく、カタカナになっている「ブロード」バンドです）
ここでも「ｂｒｏａｄ」の意味から、「ａ・ｂｒｏａｄ」はイメージしやすいです。
すると初めて見る単語や、意味を知らない単語でも、「・」の区切りを意識しますと、
およその推測ができる場合もあるのではないでしょうか。
これは、もしかしたら「紙の」辞典だけで使える方法（メリット）かもしれません。

－ － － － － － － － － － － －
３６　英会話学習の順序は
－ － － － － － － － － － － －

例えば、英会話で簡単な意思などを伝えたい場合は、単語だけで出来てしまいます。
これは簡単な文法さえも必要ないので、もっとも基本的な意思の疎通は、以下になって
います。

５４
　① 　英会話が（音として）聞き取れる「リスニング力を持った」状態。
　② 　英会話で使われる単語の「知識を持っている」状態。　

本章では、この状態を「はじまり」として、英会話の学習を順序立てたいと思います。
まず、上記の内容を見ますと－
「英単語だけの会話（意思の疎通）なら、真っ先に必要なのは② ではないか」と思われ
るかもしれません。
ですが単語の知識を持っていても、聞き取れなければ、会話の「聞く」ができないので、
先に必要なのは、① のリスニング力になります。
手前味噌ですが「～英語、英会話のリスニング力が上がる方法」では、この状態に
なれる方法が載っています。
そして、次に聞き取った単語を理解するためには、② の状態が必要になります。

ところで② を見ますと、英単語／熟語書籍を思い浮かべる方もいらっしゃるのではない
でしょうか。ですが英会話の学習なら、これらの書籍は、かならずしも必要ではないの
かもしれません。その理由は英単語／熟語が中心なので、会話の内容は少なく、数や量
を重視しているからです。
すると、これらの書籍は、受験や資格、専門的知識に対応した内容ではないでしょうか。
（もちろん英会話の単語、熟語書籍もあると思いますが）
例えば、これらの書籍に「ｓｔｏｒｙ／階」や「ｂａｎｇ／前髪」が載っていることは
多くありませんが、どちらも日常的に使われる単語になっています。
それでは② の状態に適した、効率的な学習方法はあるのでしょうか。

もし① の状態をクリアしたら、日常的な（それほど目的のない）英会話を聞きながら、
単語を覚えていくのがよいのではないでしょうか。
ここで重要なのは、この時の「覚える」は、完全に記憶するわけではありません。
同じ英会話を何度か聞いて「あの発音は、〇〇の意味かな／この会話は～の意味かな」
と分かれば、別の内容に変えてしまいます。
そして日本語は、一度聞くだけでも構いません。
後は、英会話を聞いて（日本語の）内容を思い出すようにすると、理解する行為になり
ます。

５５
すると、同じ英単語を何度も聞く機会になるので、記憶しやすいのではないでしょうか。
これは「その時に記憶するのではなく、何度も聞きながら自然に覚える」学習です。
そして簡単な英会話なら、日本語なしでも意味が分かるので、意味やニュアンスなどが
実践的に覚えられます。
その後は、聞く内容のレベルを上げますと、② の状態になるのではないでしょうか。

－ － － － － － － － － － － － － － － － －
３７　英会話を「話す」適切な時期は
－ － － － － － － － － － － － － － － － －

ほとんどの日本人は、学校で何年間も英語を習っています。
そして、カタカナ英語や音楽も入れますと、英単語の知識（語彙）は、十分にあると
いえるのではないでしょうか。
にもかかわらず「英会話学校に通ったら、英語が話せるようになった」方は、それほど
多くありません。では何故、このような結果になってしまうのでしょうか。
その理由は、英単語の知識を持っていても聞けない、リスニングが出来ないからです。
しかし、逆にいいますと「聞く」ができれば、その後の学習が効率的になります。
このような見方で英会話学習の流れを見ますと、以下になります。

　① 　リスニング力を上げる。
　② 　たくさんの英単語を覚える。
　③ 　聞ける（理解できる）英会話に対して「話す」学習をする。

「英会話学習の順序は」では、② の状態に対して「聞く」学習をご紹介しました。
これは「英会話は、聞く要素があるから」というのが理由ですが、短時間でたくさんの
英単語を覚えたいなら「読む」学習が使えます。
「書籍を見て、覚える」内容は、読む時間を短縮することで効率的になります。
その一方で「少しずつ英会話力をつけたい」場合は、聞く内容に比重を置きます。
① のリスニング力が上がった状態は、正確な発音を「耳から」覚えられます。
すると① ② の状態の後が、話す学習の適切な時期と言えるのではないでしょうか。

５６
例えば子供が言葉を話すときは、覚えた単語を知っている後なので、① ② に近い状態に
なります。
このような英会話の学習は、「聞く」の後から「話す」内容に進むことになります。
「① の状態の後に英会話学校に通えば、② ③ が同時にできるのではないか」と思われる
かもしれません。ですが、学校での学習内容を想像してみて下さい。
例えば一対一の会話なら、一度に聞く／話す／覚える複数の内容になるので、レベルが
高くなってしまいます。
そこで「英会話学校に通ったけど、習得できなかった」方は、上記のように学習内容を
順序立てます。
つまり① ② の学習によって「話す」（話したい）気持ちが出てきたら、③ を始めます。
この時に英会話学校を利用することで、費用が節約できるのではないでしょうか。

－ － － － － － － － － － － － － － － － － － － － －
３８　時間を尋ねる「Ｈｏｗ　ｌｏｎｇ～」は
－ － － － － － － － － － － － － － － － － － － － －

英語番組を見ていたら「Ｈｏｗ　ｆａｒ～」は、距離を尋ねるときに使って－
「Ｈｏｗ　ｌｏｎｇ～」は、モノの長さや所要時間を尋ねるときの内容でした。
ちなみに「Ｈｏｗ　ｆａｒ～」は「どれくらい、離れてる」の意味が基本です。
似たような内容は「ｂｅ動詞とｄｏｅｓと基本的な英会話は」に載せています。
ここでは「Ｈｏｗ　ｌｏｎｇ～」の根本的な意味をご紹介したいと思います。

まず最初に「Ｈｏｗ　ｌｏｎｇ　ｉｓ　ｔｈｉｓ～？」は、モノの長さを尋ねるときに
使います。同時に、所要時間も「Ｈｏｗ　ｌｏｎｇ～？」が使えてしまいます。
すると「なぜ（物理的な）長さと時間を尋ねるのが、同じ形なのか」と不思議に思われ
る方もいらっしゃるのではないでしょうか。
しかし見方を変えますと、何らかの共通項があるのではないかと推測できます。

実は、最近見た英文から、あることに気が付きました。
それは「英語は、現在を支点とした時間軸を『長さとして』とらえている」ことです。
以前に、ご紹介した例文を見てみましょう。

５７
　〇　「Ｈｏｗ　　　ｌｏｎｇ＝ｄｏｅｓ　＝ｉｔ　　ｔａｋｅ　　ｂｙ～？」
　　　「どれくらい　長い　　＝動的な状態＝それを　移動すると　～で？」
　　　「どれくらい長い動的な状態になる、それを～で移動すると？」
　　　「それを～で移動すると、どれくらい長い状態になる？」
　　　「～を利用して、どのくらい時間がかかりますか？」

日本語訳は「どれくらい時間がかかる」なので、時間を尋ねる内容に見えてしまいます。
ですが、英文は「～で移動する（動的な）状態」を尋ねています。
ここから「時間の長さ」を聞く内容になっていると言えるのではないでしょうか。
そして、その「長さ」から所要時間を聞く日本語訳になっています。

「Ｈｏｗ　　　ｌｏｎｇ　ａｇｏ　ｄｉｄ　　　ｙｏｕ　ｇｏ　ｔｏ　Ｔｏｋｙｏ？」
「どれくらい　長さ　　　前　　　状態だった　あなた　行く　～へ　東京　　　？」
「どれくらい前の長さでしたか、あなたが東京へ行った（動的な）状態は？」
「どれくらい前に、東京へ行きましたか？」

こちらの「ｈｏｗ　ｌｏｎｇ」は「～した状態」を聞いています。
すると「～したら、どうなる」は聞いていないので、所要時間の内容ではありません。
ですが現在を支点とした「どの時（時期）か」という、時間の長さになっています。
ところで「いつ」を聞くなら、単に「ｗｈｅｎ」が使えます。
すると「ｈｏｗ　ｌｏｎｇ　ａｇｏ～」との違いは、どこにあるのでしょうか。
おそらく「ｗｈｅｎ」は、現在、未来の予定でも尋ねることが出来ますが－
「ｈｏｗ　ｌｏｎｇ　ａｇｏ～」は「ａｇｏ」があります。
こちらは過去だけを聞けると同時に、より正確な時期を聞く意味なのかもしれません。
まとめますと以下になります。

　〇　Ｈｏｗ　ｆａｒ～？　＝どらくらい離れてる？＝物理的な距離はどれくらい？
　　　　　　　　　　　　　→どれくらいの距離ですか？

　〇　Ｈｏｗ　ｌｏｎｇ～？＝どれくらいの（モノの）長さですか？
　　　　　　　　　　　　　＝「動的な状態」は、どれくらいの長さですか？
　　　　　　　　　　　　　　→所要時間は、どれくらいですか？
５８
　〇 Ｈｏｗ　ｌｏｎｇ　ａｇｏ～？＝どれくらい前の「時間の長さ」ですか？
　　　　　　　　　　　　　　　　　→どれくらい前のことですか？

このような見方をしますと、所要時間を聞く「ｈｏｗ　ｌｏｎｇ～」は理解しやすいの
ではないでしょうか。

－ － － － － － － － － － － － － － － － －
３９　「ｔｈｅｒｅ」の距離と範囲は
－ － － － － － － － － － － － － － － － －

英会話書籍を見ると、店員さんに尋ねる「～はありますか？」は２つありました。

　① 　「Ｄｏ　ｙｏｕ　ｈａｖｅ～？」　　② 　「Ｉｓ　ｔｈｅｒｅ～？」

① の直訳は「（店員さんは）～を持っていますか？」なので、ここから意味はイメージ
できてしまいます。
これは① の形で、単に商品の有無が聞けると覚えるのが良いのではないでしょうか。
そして② は、学校英語の「ｔｈｅｒｅ　ｉｓ～＝～がある／いる」と同じです。
すると、これが疑問文になって「～（の商品）は、ありますか？」と聞いていると理解
します。

別著「英語の作り方」では「ｔｈｉｓ／ｔｈｅｒｅ」は、物理的距離の違いがあると
ご紹介しました。
ここでは、もう一歩踏み込んで「ｔｈｅｒｅ」の、範囲的な（平面、空間的な）違いが
あるとしますと、さらに分かりやすいのではないでしょうか。

まず「ｔｈｅｒｅ　ｉｓ～＝～がある／いる」を詳しくしますと－
「ｔｈｅｒｅ　ｉｓ～＝こここらへん／この辺りに、～がある状態（ｂｅ動詞）です」
のような、一定の範囲になります。
すると店員さんに尋ねるときは「ｔｈｅｒｅ＝お店の範囲」なので、以下になります。

５９
　〇　「Ｉｓ　ｔｈｅｒｅ～？」→「ｉｓの状態　お店の範囲　～？」
　　　「そういう状態、お店の範囲は、～ですか？」
　　　「お店の範囲は、～がある状態ですか？」
　　　「～は、お店にありますか？」

そして距離的な見方は「ｔｈｉｓ＜ ｔｈｅｒｅ」なので、範囲的な見方をしますと－
「これ（このへん）＜ この辺り」の違いが出てきます。
すると「ｔｈｅｒｅ」の範囲的な（平面、空間的な）違いが分かりやすいと思います。

　－ 「ｔｈｅｒｅ」に「ｉｓ／ａｒｅ」が使える理由は－

ここでは、日本人の誰もがイメージする英語から「ｔｈｅｒｅ」を見たいと思います。
以下の違いは、何処にあるのでしょうか。

　③ 　「Ｔｈｉｓ　ｉｓ　ａ　ｐｅｎ」「これは、ペンです」
　④ 　「Ｔｈｅｒｅ　ｉｓ　ａ　ｐｅｎ　ｏｎ　ｔｈｅ　ｄｅｓｋ」
　　　「机の上には、ペンがあります」

③ の「ｔｈｉｓ」は、「このペン（の距離、範囲）」が対象です。
それに対して、④ の「ｔｈｅｒｅ」は「この辺りの、ペン（の範囲）」を表わしている
のではないでしょうか。
そして「ｔｈｉｓ」よりも広い範囲になるので、「ｏｎ　ｔｈｅ　ｄｅｓｋ」の状態
まで表せます。このような見方をしますと、以下になります。

　〇　「Ｔｈｅｒｅ　ｉｓ　　　　ａ　ｐｅｎ　ｏｎ　ｔｈｅ　ｄｅｓｋ」
　　　「この辺りは＝ｉｓの状態＝１本のペンは、机の上です」
　　　「この辺りは、１本のペンが、机の上にある状態です」
　　　「机の上には、ペンがあります」

また「ｔｈｅｒｅ」の範囲は広いので、モノや状態が単数、複数のどちらでも収まって
しまいます。

４０　英会話が聞ける番組の使い方
－ － － － － － － － － － － － － － － －

「リスニングが上がった後に、たくさんの英語を聞き続けると、話せるようになる？」

英会話の学習としては「話す」の前に、聞く（聞き取れる）必要があります。
同時に意味が分かる必要もあるので、英単語の知識や、感覚的な文法の理解も必要では
ないでしょうか。
「内容の分かる」英語を聞き続けると、英単語（発音）と、その意味が一致してきます。
ですから聞き続ける行為は、このような学習効果があると言えるのかもしれません。
そして、この後に「話したい」状態が出てくることで、徐々に英会話力も上がるのでは
ないでしょうか。

ところで英単語を覚える行為といえば、書籍を使った方法が一般的になっています。
特に、学生さんは試験や受験があるので、スペルや意味を「完全に」覚える必要があり
ます。（完全に覚えないと、得点できません）

ですが英会話の学習なら、そこまで厳密である必要はないのではないでしょうか。
例えば、自身や周りに「英会話を、習得したい」方は多くいらっしゃると思いますが、
「いつまでに、英会話を習得しなければいけない」方は、それほど多くありません。
すると、いつかは英語が話せるようになりたい場合は、少しずつ覚える方法も、選択肢
の一つになります。

ここでは英会話が「聞ける」番組を使って、英単語を覚える方法をご紹介したいと思い
ます。
逆にいいますと「いつまでに、英会話を習得しなければいけない」などの、期限がある
場合は対応できません。

６１
　〇　毎週に放送している英会話が「聞ける」番組を、１日１回視聴します。

ここでの番組は、英会話を「学習する」番組ではありません。
実際の英会話の場面が多い、海外ドラマやアニメーションを使います。
そして「番組を見ながら、内容を覚える（覚えようとする）」のではなく、１日１回、
同じ番組を視聴し続けて、頭に自然に入ることを目的としています。
これらの番組は、二カ国語放送や字幕がありますので、生の英会話が聞けると同時に、
日本語訳の内容も分かります。
そこで日本語の視聴は最低限にして、英会話から内容を思い出すようにします。
すると回を重ねるごとに、少しずつ英語が頭に入るのではないでしょうか。
また、忘れていた英単語を思い出す「きっかけ」にもなります。

実際に行ないますと、１回目は聞き取れなかった個所が、後の視聴で－
「この単語が使われている／こういう文法になっている」と分かるようになります。
これは「慣れ」からくるものと、頭のなかの英語と一致した（リスニングできた）時や、
思い出した場合など、さまさまなパターンがあります。
そして同じ番組を１週間視聴したら、新しい放送に変えてしまいます。

「どれくらい英単語を覚えたのか、確認しないのか」と思われるかもしれません。
ですが少しずつ英会話力を上げる学習なので、覚えた状態とまではいかない、潜在的な
知識を頭に入れる内容になっています。
すると、これまで聞いた英会話の「あやふやな」知識によって、語彙力が増えたり－
「こういう内容ではないか」と推測できたり、理解力が上がるのではないでしょうか。

４１　それぞれの長い英語は　
－ － － － － － － － － － － － －

これまでに何度か、実際の英会話と学校英語の違いをご紹介しました。
例えば、英会話は「限られた英単語が、多く使われる」のに対して、学校の英語は－
「覚える単語が増える一方なので、英会話では、上手く使えないのではないか」と思い

ました。
ここでは、長めの英会話や文章から、気が付いたことがありましたので、ご紹介したい
と思います。まず日常的な英会話は、以下の特徴があります。

　〇　一つの会話は長くても、短い文法が連続している場合がある。
　　　また主語は一つでも、複数の動詞が使われている場合がある。

基本的に英会話を聞くときは、長ければ長いほど、理解が難しくなってしまいます。
そこで、上記の特徴を踏まえて聞くとよいのではないかと思いました。
具体的には、会話のなかの「～、ェン～」に聞こえる「～、ａｎｄ～」や、息継ぎの
「区切り」や動詞を意識します。
この時に「〇〇して、それから××して」のような、動詞が続いている状態をイメージ
すると、分かりやすくなります。

特に動詞が聞き取れると、助動詞の発音は聞き取れなくても、理解できる場合は多いの
ではないでしょうか。
例えば「Ｉ'ｌｌ　ｂｅ～」は「ｂｅ」が分かれば「ｗｉｌｌ」の存在は推測できます。
（動詞の原型になっているからです）
ある英会話番組では「Ｉｔ　ｗｉｌｌ～」の場面がありました。
そして、この時の発音は、聞き取りが難しい「ｨッ、ゥィ～」でしたが、動詞の発音は
はっきりしていました。
ここでの助動詞は、「そうなるでしょう」程度の意味なので、発音が小さいのかもしれ
ません。

ちなみに「～出来ない／やらない（つもりです）」は「Ｉ　ｗｏｎ'ｔ」が使われます。
そして、この場合は、助動詞でも「ｗｏｎ」の発音は強くなります。
これは「ｎｏｔ」の意味を表す、重要な役割があるからではないでしょうか。
話を戻しますと、学校や受験の長い英語は、以下の特徴があります。

　〇　たくさんの単語が「一つの文法」に使われて、長い英語になっている。

６３
すると一行で書かかれている場合は、文章全体が分かりやすい（見えやすい）のですが、
紙面の都合で難しい場合もあるのではないでしょうか。
そこで、英語の書籍では以下の書き方にしますと、分かりやすくなります。

　〇　文章中の動詞や、関係代名詞の「区切り」で改行して、前から訳しても
　　　意味が分かるようにします。

以下の文章は、英語番組で見られた例文ですが「区切り」を分かりやすくしました。
（何となく、話し下手な日本語にも見えます）

　〇　Ｉｔ　ｉｓ　ｓｏ　ｉｍｐｏｒｔａｎｔ／ｔｏ　ｈａｖｅ　ｐｌａｃｅｓ
　　　それは、　　とても重要です　　　　　／（重要なのは）場所を持つことです

　　（ｗｈｅｒｅ）ｗｅ　ｃａｎ　ｌｅａｖｅ／ａｓ　ｉｔ　ｉｓ
　　（その場所は）私たちが、残しておける　／それの状態のままです。

実際の英会話では「ｗｈｅｒｅ」が省略されても、通じるかもしれません。
そして当然ながら、上手な日本語ではありません。
ですが、上記の区切は、文章全体が「こうなっている」と分かるメリットがありますし、
日本語のイメージも出やすいのではないでしょうか。
日本語訳は「（私たちが）そのままの状態の場所を残しておくことは、とても重要です」
です。このような見方をしますと、長い英語でも分かりやすくなります。

４２　なぜ「～に住んでいる」を表すときは、進行形にならない？
－ － － － － － － － － － － － － － － － － － － － － － － － － － － － － －

例えば、今現在、東京に「住んでいる」ことを表す場合は－
「Ｉ'ｍ　ｌｉｖｉｎｇ　ｉｎ　Ｔｏｋｙｏ」とは言いません。この時は－
「Ｉ　ｌｉｖｅ　ｉｎ　ｔｏｋｙｏ」のように、現在形を使います。

６４
すると「なぜ今現在も東京に住んでいるのに、進行形が使われないのか」と、不思議に
思われるのではないでしょうか。
これは「～に住んでいる」状態は、「何かを、している」状態よりも、時間の幅が広い
からだと思います。つまり進行形の領域を超えているので、今現在の「住んでいる」を
表す場合は使えません。
書籍では「現在進行形は、今現在の行為を表すときに使われる」と書かれていました。
（「知ったかぶりの英会話」より）
すると東京に住んでいる状態は「何かを、している行為」ではないので、現在形を使っ
た「Ｉ　ｌｉｖｅ　ｉｎ　ｔｏｋｙｏ」になります。

もう少し現在形の「状態」と、現在進行形の「行為」を見ていきますと－
例えば以前に車を購入すると、今現在でも、所有している状態は続きますが、行為では
ありません。
ですから「私は、車を所有している」は「Ｉ　ｈａｖｅ　ａ　ｃａｒ」になります。
今現在、所有していても「Ｉ'ｍ　ｈａｖｉｎｇ　ａ　ｃａｒ」ではありません。
まとめますと、以下になります。

　〇　現在形は「状態」を表すときに使われるので、広い時間の幅になっている。
　〇　現在進行形は、今現在の進行している「行為」に使われる。

先程の参考書籍には「進行形は、短い期間の事が、今現在でも続いている状態を表す」
とも書かれていました。すると、今現在の「行為」（短い時間の幅）に当たります。

　－ 進行形の「時間の幅」が短い理由は－

このように現在進行形は行為を表すので、時間の幅は短くなってしまいます。
そして形を見ますと、その理由が見えてくるのではないでしょうか。

ご存じの通り、現在進行形は「ｂｅ動詞＋～ｉｎｇ」で表します。
これは、現在形である「ｂｅ動詞」の状態が「～ｉｎｇ」という「条件付き」になって
いる見方もできます。（文法的には「ｂｅ動詞」を形容するような状態です）

６５
つまり「ｂｅ動詞」の領域を「～ｉｎｇ」で表すときは（現在形の）一部になるので－
「～している」という、短い時間の幅（行為）を表します。
ちなみに未来形（ｂｅ動詞＋ｇｏｉｎｇ　ｔｏ～）も、「ｂｅ動詞」の一部になるので、
未来の内容（予定）を「現在に」表しています。
このような見方をしますと、進行形は「～している（ｂｅ動詞の）状態＝今現在の行為」
と解釈したほうが分かりやすいのではないでしょうか。
最後に、現在進行形にぴったりな日本語を思い付いたのでご紹介します。

　〇　現在進行形「ｂｅ動詞＋～ｉｎｇ」は「～している最中です」の意味になる。

すると、冒頭の例文は「私は、東京に住んでいる最中です」
「私は、車を持っている最中です」になるので、不自然になってしまいます。
しかし、逆に言うと「～している最中です」が自然なら、現在進行形が適していること
になります。（すべてに当てはまるかどうかは、分かりませんが）

ですから聞こえる状態になっても、突然に英会話が話せるわけではありません。

ですが、その後にスペルを確認したら－
「あの単語が、使われている」「この単語の発音は、こうなのか」と、気が付く機会に

すると「聞く／話す」ことが重要な英会話でも、テキストを使った学習は役立つと言え
るのではないでしょうか。
特に英単語や英会話のフレーズは、学校の学習経験があるので、見るほうが覚えやすい
（思い出しやすい）面があります。
それでは「テキストを見ながら、英語を聞く」練習をすると、どうなるでしょうか。
おそらくは、重要な「聞く」ことが難しくなってしまいます。

これは多くの方が実感していると思いますが、日本語は理解できるので、意識が、見る
（文字を、読む）へ向かってしまうことが原因です。
英語のテキストでも、やはりスペルを追うことに集中してしまいます。
このように「聞く／読む」を同時に行なうと、難しくなるのは仕方がありません。
すると英会話のテキストは、どのように使うのが効率的でしょうか。

　〇　二ヶ国語番組を見るときは、会話があるときは「聞く」だけにします。
　　　そして、できるだけ会話の無いときに日本語（字幕）を見ます。

例えば、英語のドラマなら、セリフを話していないときに字幕を読むようにします。
他の書籍では「先に日本語訳を見て、後から英会話を聞く」内容をご紹介しましたが、
この場合は、日本語を見る（読む）時間が必要になってしまいます。
そこでリスニング力が上がったら、英会話を聞く時間のなかで日本語を見るのがよい
のではないかと気が付きました。
すると「聞く／読む」時間が同じになるので、さらに効率的になります。

「そんな短時間では、日本語が読めないのではないか」と思われるかもしれませんが、
そのままで構いません。

英会話では、相手からの「お礼」に対しての言葉は、いくつかあります。
例えば、英語書籍や教材では、以下のように訳されているかもしれません。

　〇　「Ｎｏｔ　ａｔ　ａｌｌ／Ｎｏ　ｐｒｏｂｌｅｍ」
　　　「Ｙｏｕ　ａｒｅ　ｗｅｌｃｏｍｅ」「どういたしまして（気にしないで）」

「Ｎｏ　ｐｒｏｂｌｅｍ」は、相手への行為が、お礼を言われるほどではないので－
「問題ないです／大した事ではありません／お安いご用です」のニュアンスになります。
そして「Ｙｏｕ　ａｒｅ　ｗｅｌｃｏｍｅ」は、相手を歓迎しているので、こちらも－
「相手を助けられて、嬉しい」気持ちを表しています。

ですが「Ｎｏｔ　ａｔ　ａｌｌ」だけは、文字を見ますと、完全否定なので、直訳は－
「すべてでは、無い」になります。
何故このようなフレーズが、お礼に対する受け答えになっているのでしょうか。

　〇　「（Ｉｔ　ｉｓが省略？）ｎｏｔ　ａｔ　ａｌｌ」
　　　「（それは）すべてでは、無い」「大きなことは、していない」
　　　「ちょっとしたこと（行為）です」→「どういたしまして」

おそらくは上記の「流れ」があるのではないかと思いました。
すると「Ｎｏ　ｐｒｏｂｌｅｍ」と同じ解釈ができるので、分かりやすくなります。

６８
ところが「Ｎｏｔ　ａｔ　ａｌｌ」の基本は、否定のニュアンスです。
すると「Ｎｏｔ　ａｔ　ａｌｌ＝どういたしまして」とは別に、本来の使い方も知って
おいた方がよいのではないでしょうか。
ここでは、その例文を見つけたので、ご紹介したいと思います。

　〇　「Ｉ　ａｍ　　　ｎｏｔ　ａｔ　ａｌｌ　ｃｏｎｖｉｎｃｅｄ」
　　　　　　ｂｅ動詞　　　　　　　　　　　　過去分詞

「ｃｏｎｖｉｎｃｅ＝（動）納得させる／確信させる」が受動態になっています。
例えば、何かの結果や報告で納得いかないときに、使われるのではないでしょうか。
上手な日本語訳になるまでの流れは、以下になります。

　〇　「Ｉ　　ａｍ　　　　　ｎｏｔ　　　ａｔ　ａｌｌ　ｃｏｎｖｉｎｃｅｄ」
　　　「私は　そういう状態　～ではない　全てにおいて　納得させる」
　　　「私は、全てにおいて、納得させられる状態ではない」
　　　　→「（私は）まったく納得できません」

日常英会話では、ほとんど使われないと思いますが、ビジネスでは耳にする機会がある
かもしれません。
ここでの「ｎｏｔ　ａｔ　ａｌｌ」は、本来の意味ではないでしょうか。
また、受動態の形に目を向けますと－
「Ｉ　ａｍ　ｎｏｔ　ｃｏｎｖｉｎｃｅｄ　ａｔ　ａｌｌ」でも、通じるのではないか
と思いました。（間違っているかもしれません）
お礼に対してだけなら「Ｎｏｔ　ａｔ　ａｌｌ＝どういたしまして」でも使えます。
ここでは、本来の使い方をご紹介しました。

そして「ｆａｖｏｒｉｔｅ＝（形）お気に入りの～」は、同じ語源ではないでしょうか。
ここでは「ｆａｖｏｒｉｔｅ」と同様にイメージしやすい、もう一つの「ｆａｖｏｒ」を

　　「もし、あなた方で、そういう状態、賛成なら、手を上げて下さい」

ここでは「ｉｎ　ｆａｖｏｒ＝賛成（の状態）」の意味になっています。
これは「ｉｎ　ｆａｖｏｒ」が「すでに、好意の状態になっている」から、賛成の流れ

　　　① 　「Ｉ　ｗａｓ　ｓｏ　ｅｘｃｉｔｅｄ」
　　　② 　「Ｉ　ｃｏｕｌｄｎ'ｔ　ｓｌｅｅｐ　ｌａｓｔ　ｎｉｇｈｔ」
　　　③ 　「ｔｈａｔ＝Ｉ　ｃｏｕｌｄｎ'ｔ　ｓｌｅｅｐ　ｌａｓｔ　ｎｉｇｈｔ」

これは英会話番組で使われていた例文に、書き加えた内容になっています。
番組では「このような形は、ｖｅｒｙではなくｓｏを使う」と説明していたので－
「ｖｅｒｙ」は使えないようです。
そして、その理由は「ｖｅｒｙ」の意味から推測できるのではないでしょうか。

まず最初に、上記の内容で強調したい（重要な）個所は、① ② のどちらでしょうか。
日本語を見ますと② は具体的な結果で、① は原因を表しています。
すると、② の「昨夜は、寝られなかった（結果）」が重要ではないでしょうか。
もし、ここで「ｓｏ」の代わりに「ｖｅｒｙ」を使ったとします。
すると（② の原因である）① が、強調され「過ぎて」しまい、重要である－
「昨夜は、寝られなかった」との整合性が取れないのかもしれません。
また、ここから「ｖｅｒｙ／ｓｏ」は、以下の違いがあるのではないでしょうか。

　〇　「ｖｅｒｙ」を使うと、その個所が（内容の）中心になっている。
　〇　「ｓｏ」を使うと、ある程度は強調しつつも、他が中心だったり、
　　　　控えめな（謙虚な？）表現になる。

ちなみに① の「あまりに興奮していた」が重要な場合は、どう表すのでしょうか。
一例としては、以下になるのかもしれません。

　〇　「Ｉ　ｃｏｕｌｄｎ'ｔ　ｓｌｅｅｐ　ｌａｓｔ　ｎｉｇｈｔ
　　　「昨夜は、寝られませんでした。

４７　なぜ英会話は、簡単な単語を使うのに難しい？
－ － － － － － － － － － － － － － － － － － － － － － － －

例えば洋楽の歌詞を見ますと、それほど難しい英単語は使われていないことに気が付き
ます。
それなら６年間も英語を習った日本人は、意味が分かるのかというと、理解できるのは
一部になってしまいます。
このように簡単な英単語を使っているのに、なぜ理解できないのでしょうか。
これは、簡単な英単語でも組み合わせると、幅広い意味になることが原因ではないかと
思います。

英会話では、できるだけ「簡単な単語を使って、話す」傾向があるようです。
つまり簡単な英単語の組み合わせが、さまざまな意味を表すので、英単語の語彙力では、
英会話の理解は難しいことになってしまいます。
たくさんの英単語を覚えても、英会話力が上がらないのは、このような面があるのでは
ないでしょうか。

「学生時は、英単語だけでなく熟語も覚えていた」と思われるかもしれません。
ですが実際の英会話で使われる熟語は、それほど多くなかったのではないでしょうか。
それに、たいていは「１つの熟語＝１つの意味」で覚えるので－
（試験やテストの解答は１つなので、そのほうが都合がよい？）、これらを使う機会は
少なくなってしまいます。

７２
ちなみに個々の単語の組み合わせで、それぞれの語とは、違う意味になることを－
「イディオム」というそうです。（「ゼロから始める英単語」３０００より）
ここでは簡単な英単語を使った熟語の解釈を、いくつかご紹介したいと思います。

　〇　「ｌｏｏｋ　ｆｏｒｗａｒｄ　ｔｏ～（～は、不定詞ではなく名詞）」
　　　「～を楽しみにして待つ」

辞典で調べますと「ｆｏｒｗａｒｄ」は、なんと「楽しみ」の意味がありませんでした。
基本的な意味は、カタカナと同じ「前方」でしたので、楽しみの意味は、どこから出て
くるのでしょうか。
そこで上手に（強引に？）解釈しますと、以下になるのではないでしょうか。

　〇　「ｌｏｏｋ　ｆｏｒｗａｒｄ　ｔｏ～」
　　　「見る　　　前方を　　　　　～のほう」
　　　「～の前方を、つい先走って、見て（期待して）しまう」
　　　　→「～を、楽しみにして待つ」

ところで「～」が不定詞でないのは、何故でしょうか。
英語を学習すると、つい「ｔｏ」の後ろは不定詞をイメージしてしまいます。
ですが不定詞にしますと「～する前方を（楽しみに）見る」になるので、不自然になっ
てしまいます。ですから名詞が来るのではないでしょうか。

ちなみに「楽しむ」といえば、動詞の「ｅｎｊｏｙ」があります。
ですが、辞典の例文は「ｅｎｊｏｙ」の後に「ｔｏ不定詞」が付いていません。
熟語の「ｅｎｊｏｙ～ｉｎｇ＝～を楽しむ」を、まとめて覚えた方も多いのではないで
しょうか。どちらも「ｔｏ不定詞」を使わないのは、偶然でしょうか。

　〇　「ｃｏｍｅ　ａｃｒｏｓｓ～＝～に、偶然に会う」

「会う、はともかく、なぜ偶然の意味が出てくるのか」と思われるかもしれません。
例えば「ｃｏｍｅ」「ａｃｒｏｓｓ」の個々の意味は、学生さんでもイメージできます。

７３
ですが、これが組み合わされると、何故このような意味になるのでしょうか。
まず「ｃｏｍｅ」は、「向こうから（何かが）来る」状態がイメージできます。
そして「ａｃｒｏｓｓ」も「（何かが）横切る」状態なので、どちらも主語が「～する」
イメージはありません。
つまり能動的ではないので、上手に解釈しますと以下の流れになります。

　〇　「（突然に）何かが、横切ってくる」→「（何かが人なら）偶然に、会う」

手持ちの辞典では、イメージしやすい「～を横切る」の意味もありました。
こちらは、主語が「～する」能動的なイメージがしやすいのではないでしょうか。
ちなみに「ａｃｒｏｓｓ」は、意外にも動詞の意味はありませんでした。

　〇　「ｔａｋｅ　ａｆｔｅｒ～＝～に、似ている」

こちらを上手に解釈して「流れ」を作りますと、以下になります。

　〇　「ｔａｋｅ　ａｆｔｅｒ～」「～の後から、取って付けたようになる」
　　　　→「～に、似ている」

英語圏の方は、このような使い方や意味を（母国語なので）自然に覚えるようです。

４９　「ｍａｋｅ」の根本的な意味は
－ － － － － － － － － － － － － － － － －

　〇　「ｍａｋｅ＝（動詞）作る」「ｍａｋｅ＋人＋動詞＝人に動詞させる」

上記は、どちらも学校英語で習う、おなじみの意味になっています。
辞典でも「ｍａｋｅ＝作る、～させるの意味を表す」とありましたが、何故こんなにも
違うのでしょうか。ここでは、そのルーツかもしれない内容をご紹介したいと思います。

　〇　「ｍａｋｅ＝（動詞）～になる」

７５
上記は「ｍａｋｅ」の「一つの意味」として、辞典に載っていた内容です。
英語番組では「～の要素を持っている場合は『～になる』が使える」と、紹介していま
したが、これは「ｍａｋｅ」の根本的な意味ではないかと思いました。

　〇　「ｍａｋｅ＝条件、素養があるときに、～になる（の意味になる）」

上記の「ｍａｋｅ」が根本的な意味として、例文を見ていきましょう。

　〇　「Ｋａｔｅ　ｍａｄｅ　ａ　ｐｒｅｔｔｙ　ｄｏｌｌ」
　　　「ケイトは、かわいい人形を作った」

ケイトは、かわいい人形を作る条件や素養（作る能力）があるからこそ、ここから－
「～を作った（～になれた）」という見方ができます。（やや無理がありますが）

　〇　「Ｉ　ｗｉｌｌ　ｍａｋｅ　ｈｉｍ　ｄｏ　ｉｔ　ａｔ　ｏｎｃｅ」
　　　「（私が）彼に、それをすぐにやらせるつもりだ」

学校英語の「人に～させる」の理解は、置いておきます。
そして「ｍａｋｅ＝条件、素養があるときに、～になる」を基本にしますと、ここでは
主語が「彼が、それをする」（彼に、そうさせる）条件を持っています。
ですから「彼に、そうさせる」という、使役の意味が出てくるのではないでしょうか。
（やや無理がありますが）
辞典には「ｍａｋｅ＝進む、行く」の意味が載っていました。

　〇　「Ｔｈｉｓ　ｒｏｃｋｅｔ　ｃａｎ　ｍａｋｅ　２０００　ｍｅｔｅｒｓ
　　　　ａ　ｓｅｃｏｎｄ」
　　　「このロケットは、１秒に２０００メートルで飛びます」

このロケットも、秒速２０００メートルで飛べる素養を持っています。
ですから「飛ぶことになる」（そうすることが出来る）意味になります。
（これは無理がありません）

７６
　－ 「ｍａｋｅ　ｕｐ　ｍｙ　ｍｉｎｄ＝決心する」の内容は－

また「ｍｉｎｄ」を調べますと、以下の熟語が載っていました。

　〇　「ｍａｋｅ　ｕｐ　ｍｙ（ｏｎｅ'ｓ）ｍｉｎｄ＝決心する」

何故このような意味になるのか？の前に、同じく辞典に載っていた－
「ｍａｋｅ　ｕｐ＝作り上げる、化粧する、完全にする」を見ていきましょう。
この３つの意味は「仕上がる（達する）」状態になっている、共通項があります。
すると基本的には「ｍａｋｅ　ｕｐ＝仕上がる（達する）状態」になりますが、これは
冒頭でご紹介した「ｍａｋｅ」と「ｕｐ」の基本的な意味から、このようになります。

　〇　「ｍａｋｅ＝条件、素養があるときに、～になる（の意味になる）」
　　　「ｕｐ＝上がる、達する」

「ｕｐ」は、単に「上がる」（上がったときの状態）を表すだけではありません。
「ｕｐ＝上がる、達する」から分かるように、「そうでない時～そうなった時」までの
時間の幅があります。
すると「ｍａｋｅ　ｕｐ　ｍｙ　ｍｉｎｄ」は、気持ちが達するまでの状態から
「決心する」の意味が出てきます。
このような見方をしますと「ｍａｋｅ」の根本的な意味は、条件、素養を持っている
「～になる」ではないかと思いました。

５０　「ｓｏｍｅ」と「ａｎｙ」の使い方は
－ － － － － － － － － － － － － － － － － － － －

英語の授業では「ｓｏｍｅは肯定文、ａｎｙは疑問文や否定文で使われる」と、学生の
頃に習いました（現在では、違っているかもしれません）
その一方で別の英語書籍では、以下の内容を載せました。

７７
　○　「ｓｏｍｅ」は「いくつか（少し）の、量や数」に比重が置かれている。
　○　「ａｎｙ」は、量や数よりも「あるか、どうか」に比重が置かれている。

ここでは上記の意味に近い例文を見つけたので、ご紹介したいと思います。

　① 　「Ｗｈｙ　ｄｏｎ'ｔ　ｗｅ　ｇｅｔ　ｓｏｍｅ　ｄｅｓｓｅｒｔ」
　② 　「Ｗｈｙ　ｄｏｎ'ｔ　ｗｅ　ｇｅｔ　ａｎｙ　　ｄｅｓｓｅｒｔ」

実は、① ② のニュアンスは違う（！）ので、それぞれに使われる場面があります。
順番に見ていきましょう。
まず「ｓｏｍｅ」が使われている① は、英会話で多く使われる「～しませんか？」に
なっています。
ですから、一緒にいる方を誘うような「（私達で）デザートを、食べませんか？」の
意味です。
そして、ここでの「ｓｏｍｅ」は、その時の（人数分の）デザートになるので、厳密
な数は表していません。（３人いれば３個、５人のときは５個です）
これは「いくつか（少し）の、量や数」を表していると言えるのではないでしょうか。

次の「ａｎｙ」が使われる② は、直訳に近いニュアンスになっています。
すると、日本語訳は「なぜ私達は、デザートが無いんだ！」（手に入れられないんだ）
になるので、こちらの「ａｎｙ」も、厳密な数や量までは表していないと同時に－
「あるべきものが（一つも）ない」状態になっています。
そして、こちらは「あるか、どうか」の「無い」になっています。

このような見方をしますと「ｓｏｍｅは肯定文、ａｎｙは疑問文や否定文で使われる」
とは言えない面もあるのではないでしょうか。
ただし、① は「～しませんか」のニュアンスです。
そして否定疑問文（？）だと思いますが、日本語は、肯定文のような聞き方です。
すると話は最初に戻ってしまい、やはり冒頭でご紹介した－
「ｓｏｍｅは肯定文『のニュアンス』、ａｎｙは疑問文や否定文で使われる」になって
しまいます。（ややこしい）

７８

５１　「お元気ですか？／お元気でしたか？」の違いは
－ － － － － － － － － － － － － － － － － － － － － － － － －

あいさつ代わりになっている「お元気ですか？＝ｈｏｗ　ａｒｅ　ｙｏｕ？」は
「あたなは、どう」のような、現在の状態を聞いているニュアンスがあります。
これは「ｂｅ動詞」から推測できますが、しばらく会ってない相手に使うと、不自然に
なってしまうのではないでしょうか。
この場合は「最近は、どうですか／どうしてましたか？」のような、近況を伺うことが、
あいさつ代わりになるからです。
例えば、日本語では「お久しぶりです／ご無沙汰しています」が、それに当たります。
すると英会話でも、このような挨拶があるのではないでしょうか。

　〇　「Ｈｏｗ　ｈａｖｅ　ｙｏｕ　ｂｅｅｎ？」

過去分詞は、広い時間の幅があります。
そこで、ここから「お元気でしたか？（お久しぶりです）」のニュアンスが出てきます。

　〇　「Ｈｏｗ　　　ｈａｖｅ　ｙｏｕ　　ｂｅｅｎ？」
　　　「どれくらい　持ってる　あなた　　ずっとｂｅｅｎな状態は？」
　　　「あなたの持つ、ずっとｂｅｅｎな状態は、どうでしたか？」
　　　「あなたの広い時間の幅は、どうでしたか？」
　　　「会っていない期間は、どうでしたか？」→「お元気でしたか？」

このような過去分詞の見方は「英語の作り方－ ２０１３－ 」で、ご紹介しました。　

←過去　「過去分詞の領域」　「過去形」　　「現在形」　　「現在進行形」　未来→
　　　　「ｂｅｅｎの領域」　「ｗａｓ」　　「ａｒｅ」　　「ｂｅｉｎｇ」

そして、このように聞かれたときは、答えるときにも現在完了を使います。

７９
　〇　「Ｉ　　　　ｈａｖｅ　　ｂｅｅｎ　　　　ｇｏｏｄ（ｆｉｎｅ）」
　　　「わたしは　持っている　ｂｅｅｎな状態　よい」
　　　「わたしは、ずっとｂｅｅｎな、よい状態を持っています」
　　　「（しばらく会っていない間も）元気にしていました。」

先程の「Ｈｏｗ　ｈａｖｅ　ｙｏｕ　ｂｅｅｎ？」は疑問文なので、相手の方は、答えが
必要です。ですが、挨拶としては「お久しぶりです」だけの場合もあります。

　〇　「Ｉｔ　　ｉｓ　　　ｂｅｅｎ　　　ａ　　　　ｗｈｉｌｅ」
　　　「それは　状態です　広い時間の幅　ひとつの　間（時間）」
　　　「それは（しばらく会っていない）広い時間の幅（の状態）ですね」
　　　「それは、長い間がありましたね」→「お久しぶりです」

ここでの「ｉｔ」は、久しぶりに会うまでの期間に当たります。
そして「その期間が、長かったですね」の言葉が「お久しぶりです」になっています。
疑問文ではないので、相手に聞いているわけではありません。
このように過去分詞は「しばらくの間（これまで会っていない期間）」を表していると
言えるのではないでしょうか。
ここから現在完了形は「お元気でしたか？／お久しぶりです」の挨拶として使えます。

５２　前から訳す英会話
－ － － － － － － － － － －

英会話書籍や教材、英語番組を見ますと、当たり前のように英語の例文と
上手な日本語訳が見られます。もちろん英語の意味を知ることは重要ですが、
上手な日本語訳である必要はないのではないでしょうか。
英会話の習得は、英語を聞いたときに理解したり話すことが目的です。
つまり（日本語とは違う）英文法の理解になるので、上手な日本語に変換
すると「英文法は、このような並びになってる」と理解できるチャンスが
なくなってしまいます。以下は英語書籍に載っていた英会話フレーズです。

８０

　○　Ｆｏｕｒ ｔｅｎｓ ａｎｄ ｔｅｎ ｏｎｅｓ ｐｌｅａｓｅ

　　　１０ドル札４枚と、 １ドル札１０枚でお願い致します。

両替や現金化のときの一例ですが、日本語訳は以下でも分かります。

　○　Ｆｏｕｒ ｔｅｎｓ ａｎｄ ｔｅｎ ｏｎｅｓ　ｐｌｅａｓｅ

　　４枚（は）１０ドル で、　 １０枚（は）１ドル札で お願いします。

このようにしますと「英語は枚数を先に言うのか」と学習できますし、
「Ｆｏｕｒ ｔｅｎ ｄｏｌｌａｒｓ ａｎｄ～が省略されてるのか」
と推測したり「慣れないうちはｄｏｌｌａｒｓを付けたほうがいいかも」
など様々な思考が働きます。
すると（日本語としては）多少の違和感があっても、できるだけ直訳や
前から訳した日本語のほうが合っているのかもしれません。

５３　たった今電話で聞いた内容を話すときは
－ － － － － － － － － － － － － － － － － － － － －

　○　Ａさん　（ｊｕｓｔ）ｃａｌｌｅｄ　ｔｏ　ｓａｙ「具体的な内容」
　○　Ａさんが（ちょうど）電話してきて、言ってました「具体的な内容」を。

Ａさんから電話がかかってきて、話し終わったとします。
そしてＡさんが話した具体的な内容を、誰かに話すときに使えます。
これは英文法の基礎であろう「何が～どうなる（どうする）」から
「（Ａさんが電話して）言ってた、具体的な内容を。」の流れになっています。
ここでの「ｊｕｓｔ」は時制的な意味なので「たった今あった」電話になります。
「電話の単語がないのではないか」と思われるかもしれませんが、コールは
「呼ぶ」だけではなく「電話する／電話をかける」の動詞になっています。

８１
そしてＡさんから電話があって、本人からの内容をシンプルに話すときは
以下があります。

　○　Ｔｈａｔ　ｗａｓ　Ａさん（ｈｅ／ｓｈｅ　ｗａｓ）～ｉｎｇ
　　（電話は）　Ａさんからで　（彼／彼女は）　　　　　～だって。

「電話はＡさんだった」「彼／彼女が～していた」の２つの文法です。
電話があった時と、その時にＡさんが話した内容なので、どちらも過去形に
なります。そして英会話では（ｈｅ／ｓｈｅ　ｗａｓ）を省略するため、
見た目は簡潔になりますが、文法が見えにくい状態です。この状態が日本人の
英会話習得を難しくさせているのではないでしょうか。
先に英文法を習うと、どうしても不自然な英語に見えてしまいます。
また他の人が言っていたことを話す機会は、電話だけではありません。
単に「Ａさんが～って言ってました」の例文は以下になります。

　○　Ａさん　ｓａｉｄ（ｔｈａｔ）ｈｅ／ｓｈｅ　ｗａｎｔｓ　ｔｏ～
　　　Ａさんが言ってたけど、　　　彼／彼女はしたいって、～を。

「Ａさん＝ｈｅ／ｓｈｅ」で、その場にいないＡさんの要求を伝えられます。
時制が揃っていないのは、セリフのように「その時に～って言ってる」状態を
伝えているからかもしれません。このような（時制の揃わない）言い方は、
日本語訳でも自然になります。
違う形としては「Ａさんが相手（ｙｏｕ）に～して欲しいと言ってた」場合は
以下になります。
　
　○　Ａさん　ｓａｉｄ 　 「ｈｅ／ｓｈｅ」 ｗａｎｔｓ ｙｏｕ　ｔｏ～
　　　Ａさんが言ってたけど彼／彼女がして欲しいって、あなたが～するのを。

Ａさんが「あなたに～して欲しいと言ってた」の意味になります。
このように直後に「何を」言っていたか続く場合は、関係代名詞「ｔｈａｔ」を
使う機会は少ないそうです。その理由は次の章に続きます。

８２
－ － － － － － － － － － － － － － － － － －
５４　関係代名詞「ｔｈａｔ」の作り方
－ － － － － － － － － － － － － － － － － －

英会話の学習を始めると、学校で習った関係代名詞はあまり使われないし、
省略されていると思われる方もいらっしゃるかもしれません。これは文法で
「Ｉｔ ｉｓ～ｔｈａｔ（具体的な内容）＝（具体的な内容）は～です」で
覚えたのに、実際の英会話では「ｔｈａｔ」があったり無かったりするので、
このような印象になるのかもしれません。何故このようになってしまうので
しょうか。ここでは大まかに「ｔｈａｔ」が必要な時とそうでない時を
ご紹介したいと思います。

　○　「ｔｈａｔ」の後に続く（具体的な）内容が短い（少ない）ときは
　　　　省略しがちで、長い（多い）ときは「ｔｈａｔ」が必要になってる。

「Ｉｔ ｉｓ～ｔｈａｔ（何がどうなる）＝（何がどうなる）ことは～です」
は、「ｔｈａｔ」後半の（何がどうなる）が短い内容です。そして、この時は
「ｉｔ」の指す（何がどうなる）が短い（少ない）ので、全体が分かりやすい
状態です。すると「ｔｈａｔ」で前後を繋ぐ必要性がないので省略が可能です。
もちろん「ｔｈａｔ」が使われても、後半の内容が短いときもありますが、
やや形式的な言い方になるようです。

「Ｉｔ ｉｓ～ｔｈａｔ（内容がいっぱい）＝（内容がいっぱい）の事は～です」
の場合は、「ｔｈａｔ」後半の内容が長くなってしまいます。すると「ｉｔ」の
対象が分かりにくいので、代用的な「ｔｈａｔ」（あれなんだけど）を使う必要性
がでてきます（「あれ」の内容は、その後の内容がいっぱいです）
また「Ｉｔ」と「具体的な内容」の間に、別の単語がいくつか入ることでも
「Ｉｔ」と「具体的な内容」が離れてしまいます。この場合も代用的な「ｔｈａｔ」
（話を戻すと、あれのことなんだけど）のニュアンスで入れると－
「Ｉｔは、あれのことなんだけど、具体的な内容は～」と続くので「Ｉｔ」の対象が
分かりやすくなります（例えは分かりにくいですが）

８３
このような見方をしますと「ｔｈａｔ」の使い方が見えてくるのではないで
しょうか。

　○　Ｉｔ ｉｓ～　　　　　　　　　　 　ｔｈａｔ「何が、どうなる」
　　　それは～な状態で（それと言うのは）ｔｈａｔの「何が、どうなる」です。

　○　途中に「ｔｈａｔ」を入れることで、具体的な長い内容は～と繋ぐことが
　　　できる。
　○　関係代名詞＝前半と後半の「関係」を繋ぐ「代名詞」

－ － － － － － － － － － － － － － － － － －
５５　英文法にワンクッション入れると
－ － － － － － － － － － － － － － － － － －

例えば、相手に「Ｙｏｕ　ｗｏｕｌｄ　ｂｅｔｔｅｒ　ｔｏ～」と言うと
主語が相手（ｙｏｕ）なので、決め付けるような言い方になってしまいます。
もし相手よりも立場が上だったり、強く言う必要性があるなら構いませんが、
相手のためを思って言っても（相手は）命令されてると誤解するかもしれません。
これは主語が相手（ｙｏｕ）なので「あなたは～したほうがいいよ」から
「～しないとまずいんじゃないの」のような言い方にも聞こえるからです。
そこで相手のために言うときは、主語を自身にすることで正確に表わすことが
できます。

　○　Ｙｏｕ　　ｗｏｕｌｄ　ｂｅｔｔｅｒ　ｔｏ～
　　　あなたは　したほうがいいよ、　　　　～を。

ではなく、相手のために言いたいときは以下になります。

　○　「Ｉ　ｔｈｉｎｋ」　ｙｏｕ　　ｗｏｕｌｄ　ｂｅｔｔｅｒ　ｔｏ～
　　　「私は思うんだけど」あなたは　したほうがいいよ、　　　　～をね。

８４
日本語は主語を省略することも多いので問題ありませんが、英語では相手を
主語にすると、強く促すニュアンスにもなってしまいます。
（例えば命令形は、主語の「ｙｏｕ」が省略されています）
そこでワンクッション入れて「私」を主語にした言い方にします。
すると「～したほうがいいと、私は思うよ」になることで、控えめに意見を
言えたり、相手の立場に立った言い方になります。
また「ｗｏｕｌｄ」だけでなく、ほかの助動詞の過去形を使うことも、相手に
対する配慮になります。

　○　「Ｉ　ｔｈｉｎｋ」　ｔｈａｔ「ｍｉｇｈｔ」ｂｅ～
　　　「私は思うんだけど」あれは～たぶん「じゃないかな／だろうね」

このように自身の意見を言いたいときは、先に「私は～と思うんだけど」を
入れると柔らかい言い方になりますが、相手の意見を聞きたいときは、以下の
言い方もあります。

　○　Ｗｈａｔ ｄｏ ｙｏｕ ｔｈｉｎｋ ａｂｏｕｔ（意見やアイデア）
　　　何だと思いますか、　　　　　　　意見やアイデアについてですが－

一見すると相手に「どう思っているの？」と催促してるように見えます。
ですが実際は、相手にアイデアだけを「どう思うか」聞いているので、自身の
提案を控えめに言えたり、相手の立場への配慮にもなっています。
このように日本語ではあまり意識しない「主語（主体）は、何か」ですが、
英会話ではニュアンスが変わるので注意する必要があります。

ところで英会話では「Ｉ　ｗａｎｔ　ｔｏ～＝～したい」と言うと「私」の
強い要求になるので「Ｉ　ｗｏｕｌｄ　ｌｉｋｅ　ｔｏ～＝～したいのですが」
のほうが丁寧な言い方になると習います。
すると自身の意思や要求で「Ｉ　ｗａｎｔ　ｔｏ～」を使うのは合わないことに
なりますが、強いからこそ相手を気遣える言い方もあります。

８５
　○　Ｉ　　ｗａｎｔ　ｔｏ　ｒｅｍｉｎｄ　ｙｏｕ（英文法）
　　　私は、より気にしたいんです、　　　　あなたが（英文法）するのを。

相手が（英文法）することを、主語である「私が、注意してる／気にしてる」
ことで、相手を気遣う言い方になります。
もし、ここで「Ｉ　ｗａｎｔ　ｙｏｕ　ｔｏ　ｒｅｍｉｎｄ～」だと、単なる
相手への注意になるのかもしれません。
ちなみに「ｒｅｍｉｎｄ」は「ｍｉｎｄ＝気にする」に、繰り返す「ｒｅ」が
付くので「ｒｅｍｉｎｄ＝繰り返し気にする＝注意する」の意味が出てきます。

このように「ｗａｎｔ ｔｏ～」に続く動詞によっては、必ずしも強い要求や強制
になるわけではありません。例えば感謝の気持を表わしたいときは
「ｔｈａｎｋ　ｙｏｕ　ｆｏｒ～」が真っ先に思い浮かびますが、以下のような
言い方もあります。

　○　Ｉ ｗａｎｔ ｔｏ ｔｈａｎｋ ｙｏｕ ｆｏｒ～
　　　私はしたいです、 感謝をあなたに、 ～のことで。

これは日本語の「感謝したいと思います／感謝の意を表わしたいと思います」に
近いのかもしれません。

５６　なぜ「ｎａｍｅ」は「名前を付ける」の意味になる？
－ － － － － － － － － － － － － － － － － － － － － － － － － － －

辞典で調べますと「ｎａｍｅ」は名詞と動詞の意味があります。
しかし日本語では「名前」「名前を付ける」と書き分かけられるので
動詞の「ｎａｍｅ」に馴染めない方もいらっしゃるのではないでしょうか。
これは以下の特徴から理解することができます。

　○　英語は、主語の後に続く単語が「動詞になる」

８６
英会話番組で聞いてびっくりした内容です。
英文法では一番簡単な型として「Ｉ　動詞＝私は、動詞する」がありますが、
正確に表わしますと、主語の後に続く単語が「動詞になる」そうです。

　○　「Ｓ　＋　Ｖ」　＝　「ＳはＶする」ではなく
　　　「Ｓ（主語）」の後に続く単語が「単語する」の動詞になる。

このような見方で主語の後に「ｎａｍｅ」が入ると、どうなるでしょうか。

　○　「Ｉ（入った単語が動詞になるスペース）」
　　　「Ｉ　ｎａｍｅ」
　　　「私は、名前する」
　　　「私は、名前を付ける」

このような流れは「何に／なんて名前を」の流れがイメージできるので、
英文法を意識しなくても後に単語が続きやすくなります。
このような見方をしますと「ｎａｍｅ＝名前を付ける、動詞の意味」ではなく
「名詞を（主語の後に）置くと、動詞になる」という解釈のほうが分かりやすい
のではないでしょうか。

例えばカタカナの「リクエスト」は、主に名詞で使われています。
この感覚で主語の後に置くと「Ｉ ｒｅｑｕｅｓｔ＝私はリクエストする」から
「要求／懇願／起用する／頼む」動詞のイメージが出てきます。
これは「ｏｒｄｅｒ／ｃｏｓｔ／ｒａｉｎ」でも同じ見方ができます。
また最近ではネットの普及から「メールする＝ｅ－ ｍａｉｌ」も動詞で使えますし
「Ａさん ｆｒｉｅｎｄｅｄ ｍｅ」なら
「Ａさんが（ｓｎｓなどで）フレンド申請してきた」という使い方もできます。

そして主語の後の動詞のスペースだけでなく、動詞のところに単語を置くと、
動詞になると覚えておくと、カタコトの英会話でも伝えやすくなります。
例えば「カラオケに行こう！」と言いたいときでも、文法的に

「Ｌｅｔ'ｓ ｇｏ..ｏｕｔ..ｆｏｒ ｋａｒａｏｋｅ？」と考えなくても

５７　なぜ「ｏｒｄｅｒＡｆｒｏｍＢ」は「ＡをＢに依頼する」の意味になる？
－ － － － － － － － － － － － － － － － － － － － － － － － － － － － － － － － － － － －

例えば、ある会社に仕事を依頼するときに
「ｏｒｄｅｒ仕事ｆｒｏｍ Ｂ会社 ＝ 仕事をＢ会社に依頼する」が使えます。
英会話番組で見かけた内容ですが、生徒の方は「Ｂ会社から依頼を受けた」と
解釈しました。何故このように思ったのでしょうか。
ご存知「ｆｒｏｍ」といえば「～から」の日本語訳が頭に染み付いています。
ですから「Ｂ会社から」と思ったようですが、実際は「Ｂ会社に」なので
詳しくご紹介したいと思います。

じつは「ｏｒｄｅｒ」は「注文する／取り寄せる」の意味もあります。
これはカタカナの「オーダー」で使っていますが、以下の熟語が出てきます。

　○　ｏｒｄｅｒ 商品 ｆｒｏｍ Ｂ会社 ＝ 商品をＢ会社に注文する

するとＢ会社は注文の商品を納めないといけないので、後日にＢ会社「から」
商品が届けられます。
このような流れを見ますと「ｆｒｏｍ Ｂ会社 ＝ Ｂ会社に」の日本語訳が
理解しやすいのではないでしょうか。

８８

　○　ｏｒｄｅｒ 商品 ｆｒｏｍ Ｂ会社
　　　オーダーする、商品を（納品で）返ってくるようにＢ会社から
　　 ＝ 商品をＢ会社に注文する

５９　なぜ「ａｄｄｒｅｓｓ」は「住所」の意味になる？
－ － － － － － － － － － － － － － － － － － － － － － － － － －

以前に「ｎａｍｅ」は主語の後に置くことで「名前を付ける」動詞になると
ご紹介しました。ここに「ａｄｄｒｅｓｓ」を置くとどうなるでしょうか。
日本語では「ａｄｄｒｅｓｓ＝住所」の名詞が有名なので、当てはめますと
「住所する＝～へ住所する＝～へ送る／宛てる」意味が出てきます。

９０
　○　Ｔｈｅ　ｅｍａｉｌ　ｗａｓ　ａｄｄｒｅｓｓｅｄ　ｔｏ　Ａさん
　　　その　　ｅメールは　送られました　　　　　　　　Ａさんへ。

受動態の形ですが、メールが「ｗａｓ ａｄｄｒｅｓｓｅｄ＝～へ住所される」
つまり「送られる／宛てられる」動詞の意味になっています。
これは主語がメールなので比較的（動詞の）イメージがしやすいと思います。
ところで何故「ａｄｄｒｅｓｓ」は名詞で「住所」の意味があるのでしょうか。
これは「ａｄｄｒｅｓｓ」が「向かう」意味があるからです。

　○　Ａさん ａｄｄｒｅｓｓｅｄ ｔｈｅ ｃｏｎｆｅｒｅｎｃｅ
Ａさんは 向かった その会議に。
　　　Ａさんは 演説を行なった 会議で。

演説は単に話すだけではありません。
「ａｄｄｒｅｓｓ」の「向かう」意味があるからこそ（演説などを）行なう
動詞になります。そして（動詞として）向かうと－
「相手に対して、どう対処するか」聞くことができます。

　○　Ｈｏｗ ｓｈａｌｌ 　 Ｉ　 ａｄｄｒｅｓｓ ｙｏｕ？
　　　どうやって しましょうか　　　　私は 対処を　　　　 あなたに？
どうしましょうか　　　　　　　 私は 呼びかけを　 　あなたに？
　　　どのような肩書きで（私は）お呼びすればよろしいでしょうか？

初対面や目上の方などに対して「どう向かうか／どう対処するか」聞くことで
「どのような名前で呼べばよいか」を尋ねることができます。
また対処の意味は人だけではありません。
例えば「ａｄｄｒｅｓｓ ｔｈｅ ｐｒｏｂｌｅｍ＝問題に対処する」として
使えます。このように「ａｄｄｒｅｓｓ」で向かうと（向かった）先について
分かることが出てくるので、ここから「向かって分かった先の情報＝住所」の
意味が出てきます。

６０　ビジネス「ｓｏ」の使い方
－ － － － － － － － － － － － － － －

英会話の「ｓｏ」はどこか軽いというか、会話を埋めたり接続詞で覚えた方も
多いのではないでしょうか。
そのため「ｓｏ＝そう」みたいな訳もありますが、この感覚で「ｓｏ」を使うと
ビジネスでは上手く意味が伝わらない場面も出てきます。
「日常もビジネスも同じ英会話なので、ｓｏのニュアンスも同じでは」と思われる
かもしれません。ですがビジネスならではの常用単語もありますし、相手との交渉
や契約など、建設的な内容が多いのではないでしょうか。
そこで日常英会話とビジネスの「ｓｏ」をご紹介したいと思います。

　○　他社も開発をしています「ｓｏ」われわれも開発を急ぐ必要があります。

英文としてイメージしてください。
この「ｓｏ」は、前後の内容が「確実に」繋がっています。
つまり他社が開発しているので「（そう）だからこそ」開発を急ぐ必要がある
内容になっています。
重要な後半を伝える「ｓｏ＝そうだからこそ～」のニュアンスは、日常英会話で
あまり使われないのではないでしょうか。

　　　ビジネス英会話の「ｓｏ」
　○　前半の内容「そうなので／そうだからこそ」後半は重要です。
　　　前半の内容「＜ 」後半は重要。

　　　日常英会話の「ｓｏ」
　○　前半の内容「それから／それで」後半は～。
　　　前半の内容「→」後半の内容。

９２
ビジネスの「ｓｏ」は「ｂｅｃａｕｓｅ」の前後の繋がりをイメージすると
分かりやすいかもしれません。続いては内容からイメージしやすい「ｓｏ」です。

　○　Ｉ'ｍ 「ｓｏ」 ｇｌａｄ ｙｏｕ ｌｉｋｅｄ ｉｔ
　　　私は　　そんなにも 嬉しい あなたが 好きで それを
私は　　それほど 嬉しい 　あなたが好きになって、それを
　　　私はとても（それくらい）嬉しいです、あなたが気に入ったので。

「ｇｌａｄ」を強調するので直前に置かれますが「いいじゃん／いい感じ」
程度だと思います。同じような「ｓｏ」の位置なら、日常英会話では
「Ｉ'ｍ ｓｏ ｈａｐｐｙ～」「Ｔｈａｎｋ ｙｏｕ ｓｏ ｍｕｃｈ」
があります。また例文の後半は「ｙｏｕ ｌｉｋｅｄ ｉｔ」の過去形なので、
すでに（プレゼントか何かを）渡した後の状態（現在）ではないでしょうか。
もし渡す前なら「Ｉ ｈｏｐｅ ｙｏｕ ｌｉｋｅ ｉｔ」の表現があるからです。

　○　「ｓｏ」～＝「そんなにも／それくらい／それほど」～です。

次の例文は、前後の内容が繋がった「ｓｏ」です。

○　話の前半「ｓｏ」ｗｈｅｎ ａｒｅ ｙｏｕ ｃｏｍｉｎｇ ｔｏ 場所？
　　前半「それなら」いつの 状態 あなたが　 来るのは　 場所に？
　話の前半「そうなら」いつあなたは来るの、場所に？

例えば、話の前半に「相手が、どこかの場所に行く予定」を言ったとします。
そして前半の内容から「そういうことなら／それなら」という繋ぎの「ｓｏ」
になっていますが、ビジネスの「ｓｏ」ほど後半を強調する感じはありません。
これは繋がりというよりも、続きを表す「ｓｏ」ではないでしょうか。
すると、ここから「ｓｏ＝そう」の意味が出てきたのかもしれません。

　○　「ｓｏ」～＝「前半なら／それなら」～＝「そう／それで」～

６１　「ｔｏｏ」と「ａｌｓｏ」の違いは
－ － － － － － － － － － － － － － － － － － －

先ほどご紹介した「ｓｏ ｇｌａｄ」「ｓｏ ｈａｐｐｙ」「ｓｏ ｍｕｃｈ」の
「ｓｏ」は、直後の単語を強調している面もありますが、英語では、単語が先に
出ることで、強い言い方にできる特徴があります。

　○（相手が意見を言った後）Ｉ ｔｈｉｎｋ ～ ｔｏｏ　私も～だと思います。

例文は、相手の考えと同じ意見を持つときにも使えると思いますが、日本語の
「そうですね」のように相槌にも使えそうで、それほど強いわけではありません。
しかし強い気持ちを表わしたいときは、以下の言い方があります。

　○　Ｉ ａｌｓｏ ｔｈｉｎｋ ～　私もまた～だと思います。

相手の考え「～」に同調すると言うよりも、はじめから自身が同じ意見を持って
いたので、結果的に（自身も同じ考えだと）強調できる言い方になります。
ニュアンスとしては以下になります。

　○（相手が意見を言った後）
　　　Ｉ ａｌｓｏ ｔｈｉｎｋ～ ＝ 私（が持っていた意見）もまた～です。

先に「ａｌｓｏ」を置くことでも強い言い方になります。
対して「ｔｏｏ」は後出しの「～も」なので、ここまでの強いニュアンスは
ありません。このように「ａｌｓｏ」は強く言える（前に置かれる）面がありますが、
何故このようなニュアンスがあるのでしょうか。

　○　ａｌｓｏ ＝ ａｌｌ ｓｏ ＝ すべてそうです ＝ ～でもあります

９４
「ａｌｌ」が含まれているので、相手の意見に「すべて同じです」の意味が
あります。そして「ａｌｓｏ」以降の内容がすべてなので、後から追加説明
したような「～でもある／さらには～」の意味も出てきます。

６２　「ｏｆ」の使い方
－ － － － － － － － － － －

拙著「英語の前置詞の作り方」では、なぜオフコースは「もちろん」の意味
なのかご説明しました。簡単に言うと、直前の内容や単語が－
「そのコースになる（コースしかない）」からと書きましたが、この意味から
「ｏｆ」が使いやすくなるのではないかと思いました。

　○　（内容や単語）「ｏｆ」 ～ ＝ 内容や単語で「もちろん」～のことです。

「Ａ ｏｆ Ｂ」は、ＡＢが近い状態であると同時に一体を表しています。
ここでは先にある「内容や単語」を、近い状態である「ｏｆ～」で補足するイメージ
で理解します。すると例えばレストランの予約で「５人です」と言いたいときは
「Ｔｈｅｒｅ ｗｉｌｌ ｂｅ ｆｉｖｅ」で伝わりますが、「ｏｆ」を使うことで
より正確になります。

　○　Ｔｈｅｒｅ ｗｉｌｌ ｂｅ ｆｉｖｅ ｏｆ ｕｓ
　　（予約は）五人で、　　　　　　　　　もちろん私たちのことです。

何らかの好意を受けたときは

　○　Ｉｔ ｉｓ ｖｅｒｙ ｋｉｎｄ ｔｏ ｈｅｌｐ ｍｅ
　　　それは親切ですね、　　　　　 私を助けてくれるなんて。

でも伝わりますが、以下なら「誰の」優しさか正確になります。

９５
　○　Ｉｔ ｉｓ ｖｅｒｙ ｋｉｎｄ ｏｆ ｙｏｕ ｔｏ ｈｅｌｐ ｍｅ
　それは親切ですね、　 もちろんあなたです、　 私を助けてくれるなんて。

また「ｏｆ」の直前の単語だけにかからない使い方でもできます。

　○　Ｗｅ ｎｅｅｄ ｔｏ ｂｅ ｔｈｏｕｇｈｔｆｕｌ
　　　私たちは必要があります、配慮する

　　　ｏｆ ｅｒｅｒｙｏｎｅ ｈｅｒｅ
　　　もちろんみんなです、ここに居る。

一行目だけでも意味は伝わります。
ここに二行目を加えると「特定の誰か」に対してではなく
「ここに居るみなさんで～しましょう」のニュアンスになるので、婉曲的に
注意することもできます。
前後の内容から「ｏｆ」の使い方が見えてくるのではないでしょうか。
