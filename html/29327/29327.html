第一章　［食］は修行の要――禅寺と宮沢賢治



　日本人にとって、菜食とは粗食のこと


　日本人は仏教伝来以降、あまり肉食はしていない。ただしそれは四足獣に限ってのことで、鳥類の肉は食べているし、魚も食べている。

　しかし、量としてはわずかで、欧米とは比較にならない。何か特別なお祝いごとのある日や客が訪れた日に、飼っている鶏を殺して食卓に供する。あるいは偶然キジを捕まえた時、ごちそうとして食卓に上ることもあっただろう。日常は穀物や野菜を主として食べていた。

　だから特に強調して〝菜食〟ということもなく、自然に菜食に近い食事をしていた。

　ここで西洋で言われる菜食者、つまりベジタリアンという言葉の語源について説明しておきたい。それはラテン語の Vegetus からきたもので、本来は「完全な」「新鮮な」「元気のよい」「活気に満ちた」という意味である。特に野菜のみを食べるという意味ではないが、肉食中心の欧米にあっては一般に野菜を食べる人と解されている。しかし、もともとが菜食に近い日本では「肉食→菜食」という欧米並みの落差を出すには、〝粗食〟ということになるのではないかと思う。


　粗食の実践者


　粗食の実践者をみると、霊的に高い人が多い。衣や住に対してもそれほどとんちゃくしていない。

　また、霊的向上を目指して修行する人のために用意される食事も粗食である。永平寺のような禅寺にこもって修行する人々の食事がその代表的なものだ。修験道の行者のように山にこもって道を求める人はさらに粗食である。日本を精神的に導いてきた空海や道元などの求道者は、そういった粗食に耐え、他の修行もしていくなかで、霊的開眼と悟りに至る神秘体験をしている。粗食と霊的開眼、さらに偉大な仕事を成し遂げることのあいだには、深いつながりがありそうである。


　禅寺では食事も重要な修行のひとつ


　禅は日本人にとって身近な修行である。心に悩みを抱えている時や、壁にぶつかった時、あるいは人生について疑問を持ち道を求める時‥‥、禅寺にこもってその答えをみつけようとする。それは現在もなお行なわれている解決法である。有名な禅寺・永平寺での修行の様子は、時折、テレビでも紹介されるため、ご存じの方も多いであろう。

　永平寺は、約七四〇年前に道元禅師が修行のための道場として創建した寺である。福井県の山中にあり、冬は雪に閉ざされるという自然環境の厳しい寺である。ここでの修行は道元みずからがこと細かく定めた『永平清規( えいへいしんぎ ) 』という修行システムにのっとって行なわれている。この修行システムは道元の生前より今に至るまで受け継がれていて、まったく変わっていないという。

『永平清規』は六編からなっているのだが、そのうちの一編まるまる全部が食事のことに充てられている。修行をすすめるうえで、食物がいかに大切かという証( あかし ) である。また修行者のために食事を用意する人を典座( てんぞ ) というのだが、この役は高徳な僧にしか許されない重要を役職である。典座に抜擢( ばってき ) されるのは、僧としてたいへん名誉なことなのだ。

　この典座の心得や、食物を調理するうえでの、心がけについても、道元はこと細かに指示を与えている。なぜそうまで食を重要視したか。それは道元自身が、食に対峙することから悟りへの道を切り拓いたからである。だから禅の世界では、食事に対する心の持ち方が、すべての修行の出発点であり、土台であるとされているのだ。

　さて、では永平寺では、どんな食事が修行者に出されているのだろうか。

　まず朝食は早朝五時四○分である。そのため典座は三時には台所に入る。食事の内容は天井粥( てんじょうがゆ ) と呼ばれる薄い麦粥が一椀、それに万年漬けと呼ばれる沢庵が二切、そしてごま塩である。これだけで、もちろんお代わりなど許されない。この朝食は粥座( しゅくざ ) と呼ばれている。

　午前四時に起床し、素早く寝具を片づけ、身支度を整えた修行者は、それから座禅を組む。そうした後、この質素な、しかし典座が心をこめて作った食事を食べる。食べ方も修行のひとつで、音をたてず、口に運ぶのである。

　昼食は斎座( さいざ ) といい、麦七に米三の割合の麦飯にみそ汁、そして漬け物が少しついているだけの、文字どおり一汁一菜である。夕食は薬石( やくせき ) といい、献立は粥座とほとんど同じである。この三食が一日の食物であり、修行者は早朝から床につくまで休む間もない生活を送りつつ、これだけの粗食で心身を保つ。そうして自身の成長を目指すのである。

　しかしこれでもまだ多いほうで、昔の修行者は一日に一食しか食べなかったという。午前中に一度食べるきりで、あとは何も口にしない。その大切な一度の食事でさえも、時には食べずにすませることもあった。こんな話が残っている。

　中国に百丈禅師( ひゃくじょうぜんじ ) という僧がいて、かなり高齢であったが、若い僧に混じって同じように作務をしていた。しかし、若い僧からみると老師が作務をしているのを気の毒で見ていられない。そこである朝、老師の掃除道具を隠してしまった。老師のほうは道具をさんざん探したものの見つからない。しかたがなく座禅を組んで時をすごした。そうして朝食の時間となったが、老師は食事に手をつけようとせず、こう言った。

《私は働かずに食べてよいという徳は持ち合わせていない。だから一日仕事をしなかったので、一日食べないのは当然のことだ》（『禅と食事』上田祖峯）

　食物が目の前に、自分のために用意されていても、それを食すことが徳を減じ、修行の妨げになるというのである。そこまで食べること（あるいは食べないこと）を大切な修行にしているわけであるが、その意義が集約されているのが「五観( ごかん ) の偈( げ ) 」という経文である。食事の前に必ず唱えるもので、内容はこんなものだ。前掲書を参照しながら紹介していこう。


　修行としての食事のとり方・五項目


　① 　一つには 功( こう ) の多少を計り 彼( か ) の 来処( らいしょ ) を 量( はか ) る。

（意味→功を計るとは、食事が自分の前に出されるまでにどれほど多くの人々の手が加わっているかという、その苦労を推し量りなさいということであり、彼の来処を量るとは、その食べ物が天地の恵みによるものであることを自覚しなさいということ。食事は日常的なものなので、食事を支えているものについて鈍感になりがちである。しかしそのような態度は人を怠慢の極みに引きずりおろし、煩悩の赴くがままの生活に埋没させてしまう。人は誰でも仏性を備えているが、それを目覚めさせるためには日常の物事に対して感謝の心をもって接することが肝要なのである）

　② 　二つには己が徳行の全欠を 忖( はか ) って 供( く ) に応ず。

（意味→供とは食事のことで、自分の修行のあり方が食事をいただくにふさわしいかどうかを反省しなさいという意味。自分は一個の生命体であるが、食物も同じく尊い生命体である。それを意識し、自分がその生命体を食べるのにふさわしい生き方をしているかどうかを考えて、行動し反省していくなら、それは自己を高めていくことになる）

　③ 　三つには 心( しん ) を 妨( ふせ ) ぎ 過( とが ) を離るる事は 貪等( とんとう ) を 宗( しゅう ) とす。

（意味→心の汚れや乱れを取り去って、貪( とん ) ・瞋( じん ) ・痴( ち ) の三毒を克服する修行をするためにこの食をいただきますという意。貪は貪( むさぼ ) り、瞋は怒り、痴は無知のことである）

　④ 　四つには正に良薬を事とするは 形枯( ぎょうこ ) を 療( りょう ) せんがためなり。

（意味→人間としての自分の生命を保って修行を続けるために、この食事を命の良薬として食べさせていただきます）

　⑤ 　五つには 成道( じょうどう ) のため 故( ゆえ ) に今この 食( じき ) を受く。

（意味→悟りを開くためにこの食事をいただきますという決意表明）


　以上である。

　これらは修行僧だけでなく、普通の生活を送るわれわれにとっても大切な心がけだといえよう。道元はこうした日常的な食事に関する行為を、真心をこめて行なっていくことが、自分を悟りへと導いてくれる近道であると教えている。瞑想や苦行のような一見修行っぽいものよりも、日常生活のなかに道があると言っているわけで、それを極限にまで押し進めて昇華したものが『永平清規』であり、永平寺で行なわれている修行だと言えるだろう。


　食べることを修行法にまで昇華した木喰( もくじき ) 行


　禅の食事は修行食として完成されたものであるが、日本にはもうひとつ修行食といっていいものがあり、これが木喰( もくじき ) である。修行食としてはもっと野性味があるもので、体系づけられてもいない。

　日本は国土の八割が山である。そういう国なので、世界でもまれにみる山岳宗教が起こった。民俗学者の五来重( ごらいしげる ) 氏によると、これが陰陽道や宗教と結合して修験道という特殊な宗教を作りあげ、それは日本人のあらゆる宗教の原点になっているという。

　山という一種の異次元世界にこもって修行し、道をきわめようとする修行者（修験者、あるいは山伏）はその食物も限られざるを得なかった。主食である米を携行できないことから、日々の食物を工夫する必要があった。そんな不自由さから生まれたのが、食物による修行法、木喰( もくじき ) 行である。これは五穀（米・麦・大豆・小豆・胡麻）、あるいは十穀（五穀にそば・きび・稗( ひえ ) ・唐きび・粟を加えたもの）を断つというものである。実は、この五穀十穀の種類は一定していないのだが、とにかく人間に必要な穀類を断ってしまう。火食 もしない。だから食べるのは木の実や草根木皮であり、五穀断ちだとそば粉を水で練ったものが主食となる。このような食生活を行として行なう修行者を木喰という。木喰( もくじき ) 応其( おうご ) （一五三六～一六〇八年）や木喰( もくじき ) 五行( ごぎょう ) （一七一八～一八一〇年）という行者がいる。

　この木喰行がさらに進むと、生きながらにして仏となる即身成仏となる。この行ははじめの三年三カ月を五穀断ちし、次の三年三カ月を十穀断ちする。この間ずっと水行を続ける。やがて死期が近づくと、穴を掘って石室を作り、空気穴だけあけておく。そうしてなかに入ると土をかけてもらって、みずから生き埋めになって餓死するのである。これを「入定」と呼び、たいへん尊い行だとされている。食物を抑制するといっても、ここまでくるといいのか悪いのかわからないが、すごいことであるには違いない。

　話が前後するが、木喰の食事法についてもう少し詳しく述べる。

　火食をしないというのは火を通したものを食べないことで、煮たり焼いたりせずに生( なま ) のままを食べる。味つけもいっさいしない。精白したものも食べない。主に食べるのは、そば粉や山野に自生する植物、すなわちカヤの実や松の実、クルミやギンナン、それに果物であろう。粗食といえばこれほどの粗食もない。だいいち、自生するものを見つけることができなければ、何も食べないですごさねばならないのだ。そうして山を歩き、崖をよじのぼり、眠るのも山のなかという生活をする。

　このような食事が人をどんなふうに変えるかというと、そば粉だけを食べて生活した体験者によると、血液の活躍が火食をする人よりはるかに旺盛になって、蚊も刺さなくなる。血液のにおいが違うためらしい。また身体が温まって山に入るには好都合になるということである。


　役行者( えんのぎょうじゃ ) が食べていたのは山菜と木の実


　木喰の先駆者で、修験道の創始者が役行者（役小角( えんのおづぬ ) ）である。いまから一三〇〇年前（七世紀後半）、大化の改新の時代に活躍した人物なので、伝説的な周辺情報が多いのであるが、山にこもって粗食で修行し、数々の奇跡的能力を発揮したことは確かであるらしい。祈りの力で民衆の病気を治し、雨を降らし、飢饉から救い、争いごとを解決するなど、民衆の救済を中心とした活躍をしている。後に「神変大菩薩( じんぺんだいぼさつ ) 」と仰がれ、民衆の信仰の対象となった。夜になると富士山を庭のごとくに踏破したことからか、山梨県富士吉田の北口本宮富士淺間神社には神変大菩薩が祀られている。

　役小角が食べていたのは、山のなかでとれるワラビ、ゼンマイなどの山菜や、木の実である。いつも松葉や松の実を口に入れて噛んでいたが、松葉や実を食べると暗い闇のなかでも眼が見えるようになり、また五〇〇メートル先を行く人の足音を聞きわけられるようになる、ほんのかすかなにおいもかぎわけることができるようになるという。仙人の食物のなかでも重要なもので、五感が鋭くなり六感も冴えるという効果をもたらす。

　彼が着ていたものも山にある植物で作ったものだ。藤の蔓( つる ) の繊維で編んだ布を体に巻きつけ、着物としていた。背は当時にしては大男で、一七〇センチくらいあり、筋肉は引き締まり、日に焼けた顔はヒゲ面であった。

　こうして厳しい修行を続けるうちに数々の能力を身につけ、その能力を使って民衆を救い始める。そしてまた、弟子に教えを説いたり、修行のしかたを教えたり、法を伝授したりしている。それが集大成されたのが修験道であり、木喰である。


　悟りに至る一〇段階の修行法


　修験道では、山に入る時に服装を変える。カンマン姿というのに着替えるのだが、これは〝修行による変身〟を象徴しているという。山のなかで厳しい修行をすることで、凡夫から悟った人へと変身するからである。そしてやがては即身成仏へといく。そこに至る修行法について役行者はこういっている。役行者は、ソクラテスや孔子同様、何も書き残していないが、その教えは弟子から弟子へと語りつがれている。


　わがあとを継がんとするものは、十界頓超( じっかいとんちょう ) の行をしなさい。

　行をすると、心身につきまとって心を乱していたものが、すべて離れてしまう。

　そして、かならず悟りに入ることができる。（『役行者ものがたり』銭谷武平）


「十界頓超( じっかいとんちょう ) の行」は、現代では「十界修行」と呼ばれている。十界というのは、人間が悟りを得てブッダ（目覚めた人）に至るまでにたどる一〇の段階のことである。この「十界修行」には、それぞれの段階をクリアして次のレベルにいく方法が示されている。それは、次のようなものである。


　一、床堅( とこがため ) （地獄）　正座した修行者を打つことで、自己に仏性があることを観じさせる修行法。

　二、懺悔( ざんげ ) （餓鬼）　密室で正先達に五体投地の三礼をし、自分の罪を懺悔する。読経もなされる。

　三、業秤( ごうばかり ) （畜生）　秤に下げて修行者の罪あるいは業を秤る。命綱を付けて断崖絶壁から身を乗り出して谷底を覗く修行が、これにあたるといわれる。

　四、水断( みずだち ) （修羅( しゅら ) ）　初夜勤行の時、正先達よりそれ以後の水の使用と飲むことが一切禁じられる。

　五、閼伽( あか ) （人間）　水断終了後、修行者の頭上に閼伽水が注がれ、煩悩が洗い流される。

　六、相撲( すもう ) （天）　修行者が左右から出て相撲をとる。後、読経がなされる。ここまでが六道の修行。

　七、延年( えんねん ) （声聞）　修行者が長寿を祝う舞をまってみせる。

　八、小木( こぎ ) （縁覚）　修行者が採燈( さいとう ) の先達に小木をおさめる。

　九、穀断( こくだち ) （菩薩）　七日間、穀類を一切断って修行すること。成満の日の未( ひつじ ) の刻に、十に入る。

　十、正灌頂( しょうかんじょう ) （仏）　修行者は正先達から大日如来の秘印が授けられ、即身即仏する。（前掲書）


　この修行では食については修羅界での水断ちと菩薩界での穀断ちが示されている。ブッダのレベルに到達するためには食の節制が修行のひとつになっていることに注目しておきたい。

　役行者が死ぬとき、最後までつき従ってきた弟子の本行( ほんぎょう ) に向かって、自分の死を嘆いたり悲しんだりしないようにと言い残している。


「わたしは、すでに、年も六十八歳になった。

　わたしには、仙人としての永久の寿命であるところの本寿がある。

　したがって、いつまでも歳には限りがない。

　しかしながら、人間としての寿命は、今年で尽きてしまう。

　おまえたちは、これを決して嘆いたり、悲しんだりしないでほしい」（前掲書）


　そしてまた、自分に会いたければ一心に修行をするように、そうすればいつでも会うことができる、自分の魂はいつまでも大峯にとどまって決して他に移らないから、という。大峯は、役行者が霊感に打たれて登った山である。大日如来の浄土であるといわれる霊山で、険しい山だ。役行者はここに過去六回転生してきた、今回が七回目の転生であると伝えられている。

　そこに魂を残し、修行する人を導いていくと約束している。だからだろうか、役行者の死後も、彼に会ったという人が何人もいたという。また埋葬した場所を掘り返すと、遺体が消えてなくなっていた。このようなエピソードは弘法大師空海とよく似ている。

　真言密教を日本に伝え、高野山を開いた空海も、長安に渡る前は山のなかにこもって一三年間も修行している。役行者と同様に、山野をかけめぐり、滝に打たれ、草根木皮を食しての厳しい修行である。この基礎ともいえる修行生活ののちに、超人的活躍をし、奇跡としか思えない開運に巡り合っている。


　山にこもって粗食で修行した空海


　空海（七七四～八三五年）の生家は四国の豪族の名門である。幼い頃より神童の誉れ高く、一五歳の時に伯父の阿刀大足( あとのおおたり ) に連れられて都に上り、一八歳で大学の明経科に入学してエリートコースを歩み始める。一族はそんな空海に大きな期待を寄せていた。

　ところが、大学では求めているものを得ることができず、中退してしまう。そうして山にこもって修行を始めるのであるが、この後の空海の足取りは不明である。三一歳になって遣唐船に乗って中国に行くまでの一三年間、山にこもっていたのは確かであるが、確実な足取りはわかっていない。ただ山から降りてきた時には、とてつもない成長を遂げていた。

　空海の著書である『三教指帰( さんごうしいき ) 』によると、大学をやめるきっかけになったのは一人の沙門が教えてくれた「虚空蔵( こくうぞう ) 求聞持( ぐもんじ ) 法」という行法を成就したためだという。求聞持法とは、法にしたがって虚空蔵菩薩の真言を百万遍唱えれば、いっさいの教法の文句が暗記できるという、記憶力が飛躍的に増大する行法である。

　空海はこの法を信じて山に入り、必死の思いで精進努力した。この山は阿波( あわ ) の大瀧嶽( たいりょうのたけ ) や土佐の室戸岬( むろとみさき ) であると記されている。そして室戸岬で修行している時に求聞持法を成就し、その結果、明星が飛んできて口のなかに入ったという。この法の成就により、空海は、一度、目にし、耳にしたものは決して忘れることがないという超天才になった。

　こうして一八歳から三一歳までの一三年間を、無名の山岳修行者として黙々と修行していた。この間の食べ物は、先ほど述べたような粗食であっただろう。また、真夏の暑い盛りに何も口にしないで修行に励んだという記述も残されている。

　この頃の空海は、海のものとも山のものとも知れぬ存在である。大学は中退しているし、正式の僧でもない。当時は僧になるにはお上の許可が必要で、自分で勝手に出家している者は私度僧といわれ、何の資格も特権もなかった。空海はその私度僧の状態で山にこもったわけで、無名・無資格の何の力もない存在である。その彼が山を降りるや、突然、歴史の表舞台に姿を現わす。それも遣唐船に乗るというとてつもない現われ方である。


　一三年間の山ごもりが開運を呼ぶ


　それ以後の空海の運の開け方というのは、まさに奇跡というほかはない。遣唐船に乗るのは難しいうえに、もし首尾よく唐に渡れたとしても最低二〇年はとどまらなくてはならない。また嵐にあって途中で船が沈む可能性もある。しかし空海が乗った船は嵐に見舞われたものの、何とか無事に中国に着き、都の長安に上ることができた。

　長安には空海を待っている人がいた。恵果( けいか ) 阿闍梨( あじゃり ) という密教の高僧である。阿闍梨はすべての弟子をさしおいて、空海に密教のさまぎまな法を伝授する。空海は語学に通じており、サンスクリット語もまたたく間に覚えてしまったから、阿闍梨からの伝法に何の支障もなかった。

　そのようにして求めていたものを得ると、もう唐にいる意味はない。そこで日本に帰るのだが、何十年に一度という船がその時に出航したことも奇跡なら、留学期限が満了していないにもかかわらず、何のとがめもなかったのも奇跡である。しばらく太宰府に留め置かれたものの、その間に朝廷が代わり、嵯峨天皇の時代になったために、重用されて大いに活躍している。

　山にこもって粗食で修行に励んだその結果が、こうした奇跡としか思えないことを次々に起こしているのである。相当高いレベルでの霊的覚醒があり、奇跡を引きつける力を持ったためだろう。

　山を降りた後の空海も粗食であった。おそらくずっと粗食で通していたと考えられる。それは空海が、あとに続く人々のために建立した修行道場・高野山金剛峯寺( こんごうぶじ ) をみれば理解できる。ここで修行する人に出される食事も、粗食である。またここでは日に一度、いまも生きて活躍している空海のために食事が作られ、空海がいるとされている奥の院に運ばれるのだが、その食事の献立も粗食である。

　弘法大師空海にいまでも食事が運ばれるのは、彼の死に方がたいへん特異なものだったからである。いや、空海は死んだとはいわれない。入定( にゅうじょう ) したといわれる。五穀を断ち、やがて水すらも飲まなくなり、ついに入定した。それについて空海が書いたとされている「二十五箇条の御遺告( ごゆいごう ) 」にはこう記されている。


　わたしは去る天長九年十一月十二日から五穀を極端に厭( いと ) い避けて、専ら座禅を好むようになった。弟子たちよ、よく聞くがよい。わたしの余命はあといくばくもなくなった。汝( なんじ ) たちはわが教えを忠実に守ってもらいたい。わたしは永く山に帰ろうと思う。（『空海』松長有慶）


　晩年の空海は座禅に集中していた。そしてそのまま入定したという。空海の肉体は腐りもひからびもせず、生前そのままの姿でいまも奥の院の石窟のなかにおられるという。その衣には、あたかもどこかに出かけてきたかのように、時折、泥がついている。また、髪もひげも伸びているというのだ。だから現在でも毎日食事が運ばれ、衣も年に一度は新調して古い衣と取り替えているのである。


　以上、禅と修験道、そして密教を中心に食のあり方を見てきたが、日本特有のこれらの霊的向上の道が、ともに食を修行のひとつにしていることがよくわかる。


　大本教の出口ナオ・王仁三郎も粗食


　こうしたことから、日本では精神的、霊的に大きな仕事をしている人には粗食の人が多いということが言える。特に宗教的理念をもって民衆を教化した人は、必ずといっていいほど粗食である。

　たとえば、近代日本の宗教のなかでもっとも大きな展開をして、現代でも多くの教団に影響を与え続けている大本教は、開祖の出口ナオ（一八三六～一九一八年）、会長の出口王仁三郎（一八七一～一九四八年）ともに粗食である。特にナオのほうは生涯一度もおなかいっぱいに食べたことがなかったという。若い頃は貧乏のため、晩年にあっては行のために少食を守った。その行も自分のための行ではなく、世の大難を小難にするためにみずからに課したものであった。

　大本教の三代目教祖であり、ナオの孫にあたる直日氏は、祖母の食事の様子についてこんなふうに語っている。


　祖母の食事の様子はきわめてしずかなものでした。朱塗りの膳に向って、朱の三椀を使って、音もかそかに箸をとっていました。（中略）椀に少しご飯をよそって、湯をかけてあがっている姿のしずかさはもっとも印象的です。いただくものの一切は、お上から上ったものとして大切にし、厳格に一汁一菜を守っていました。野菜の煮もの、浸しが多く、豆腐を味噌汁に入れ、月に一度の生湯葉などは、ごちそうになっていました。いまの人が想像もできない粗食ではありましたが、天地のご恩を感謝して、有難く美味しくいただいた点では、問題にならないひらきがあります。祖母は、食事をあがっている時も、その姿はまことにしずかな、清い、高いものを感じさせる空間の中にいました。（『大本、出口なお・出口王仁三郎の生涯』伊藤栄蔵）


　王仁三郎が日常どんな食事をとっていたかは、大本教本部に王仁三郎を訪ねた浅野和三郎の話から推測できる。浅野和三郎は心霊学の研究者として有名であるが、もともとは東大出のインテリで、小泉八雲（ラフカディオ・ハーン）に師事し、英訳書も何冊かある英文学者である。大本教との出会いによってその世界にのめりこむのであるが、彼が初めて京都府・綾部の大本教本部を訪ね、出された夕食はこんなものである。

　王仁三郎は「大本にはたいしたご馳走はありませんで」と言ったが、まったくそのとおりで、膳の上にのっていたのは二切れの沢庵と焼いたするめ、それにポロポロの冷たい麦飯であった。もちろん王仁三郎も同じものである。和三郎はそのやけに固いするめに驚き、のどを通りにくい麦飯に苦労しながら、麦飯を湯漬け（茶漬けではないので、茶も出なかったのだろう）にし、沢庵の塩味でごまかして何とかのどへ流しこんだ。王仁三郎は食べなれているらしく、平然と食べていたという。


　宮沢賢治の『ビヂテリアン大祭』


　もう一人、粗食の人をあげておこう。やはり宗教的な雰囲気をもった人物で、宮沢賢治（一八九六～一九三三年）である。法華経に帰依し、その経典の説く教えを日常で実行しようと懸命になっていた。

　その生涯は〝利他の精神〟に貫かれている。自分の生活を省みずに、人に対してお金や物を施し、自身は質素な暮らしをしていた。賢治の思想をもっともよく表現しているのが、有名な「雨ニモマケズ」という詩である。彼が理想とした生き方を描いたものだ。

　賢治は生涯結婚せずに不犯を守り、食に対してもたいそうこだわった。その食についてのこだわりを一編の小説にまとめあげたのが『ビヂテリアン大祭』である。「ビヂテリアン」とは「ベジタリアン」、つまり菜食者のことである。まだ一般的なカタカナ表記が決まっておらず、賢治は発音から「ビヂテリアン」と表記している。

　この作品のなかに菜食者についての説明と、同じ菜食者のなかにも二つのタイプがあることが書かれている。的確な説明、及び分類である。以下、参考になる部分を紹介する。


　元来ご承知のごとくビヂテリアンといふのは動物質のものを食べないといふ考のものの団結であり、日本では菜食主義者と訳するけれども主義者といふよりは、むしろ菜食信者といふ方が、よく実際に適( かな ) つてゐるとも思はれる。

　その中にはいろいろ派があるやうであるが、大体そのテーゼについて大きくわけると、同情派と予防派との二つになるらしい。

　同情派と云ふのは、恰度( ちやうど ) 仏教の中でのやうに、あらゆる動物はみな生命を惜むこと、我々と少しも変りはない、それを一人が生きるために、ほかの動物の命を奪って食べるそれもー日に一つどころではなく、百や千や万のこともある、これを何とも思はないでゐるのは全く我々の考が足らないのでよくよく喰べられる方になつて考へて見ると、とてもかあいさうでそんなことはできないとかう云( い ) ふ思想をもつものであり予防派の方はじぶんの病気予防のために、なるべく動物質をたべないといふのであくまで利己的な連中の集りである。（以上、改稿形より『宮澤賢治全集』第一〇巻）


　賢治はもちろん〝同情派〟である。そして、なぜ自分がそうなのかという根拠についても、この作品のなかに述べられている。それは仏教的な考えで、慈悲と智慧と愛、そして輪廻転生( りんねてんしょう ) によるものである。まるで経典そのもののような賢治の言い分に、少々長くはあるが、耳を傾けてみたい。


　仏教の精神によるならば慈悲である、如来の慈悲である完全なる智慧( ちえ ) を具( そな ) へたる愛である、仏教の出発点は一切の生物がこのやうに苦しくこのやうにかなしい我等とこれら一切の生物と諸共( もろとも ) にこの苦の状態を離れたいと斯( か ) う云ふのである。この生物とは何であるか、そのことあまりに深刻にして諸氏の顔を傷つけるであらうがこれ真理であるから避け得ない、率直に述べようと思ふ。総( すべ ) ての生物はみな無量の劫( カルパ ) の昔から流転に流転を重ねて来た。流転の階段は大きく分けて九つある。われらはまのあたりにその二つを見る。一つのたましひはある時は人を感ずる。ある時は畜生、則( すなわ ) ち我等が呼ぶ所の動物中に生れる。ある時は天上にも生れる。その間にはいろいろの他のたましひと近づいたり離れたりする。則ち友人や恋人や兄弟や親子やである。それらが互にはなれ又生を隔ててはもうお互に見知らない。無限の間には無限の組合せが可能である。だから我々のまはりの生物はみな永い間の親子兄弟である。異教の諸氏はこの考をあまり真剣で恐ろしいと思ふだらう。恐ろしいまでこの世界は真剣な世界なのだ。私はこれだけを述べようと思ったのである。（前掲書）


　こういう考え方であるから、実際に日常生活のなかでは食にこだわっている。いや、こだわるというよりも、考え込んで食べる、というほうが当たっているかもしれない。その食に込められている命について考え、自分が食べないでこの命が生かされていればどんなふうだったろうと想像するのである。こんな賢治であるから、食についてのエピソードがいくつか残っている。それは限りなく心やさしく、自分に対して厳格な賢治の姿を彷彿とさせるものである。


　賢治の食についての二、三のエピソード


　弟の宮沢清六氏によると、賢治が家族とともに食事をする時の様子はこんなふうであった。


　兄は食事のときなどに、自分の家で家族と一緒に物を食べるのに、いつも恥ずかしがって恐縮しながら食べたもので、物を噛( か ) むにしてもなるべく音をたてないようにしていたのでありました。（『雨ニモマケズ』）


　大正七年、二二歳の時から五年間、完全菜食の生活をしていた。これは宗教的な修行のためであったが、この頃の彼の食事は冷たい麦飯に納豆をかけたものだったり、漬物と麦飯のみだったりしたという。そばがきも時折食べている。このような菜食をするにあたっての決意を、知人の保阪嘉内に宛てた手紙のなかでこう述べている。大正七年五月に書かれたものだ。


　私は春から生物のからだを食ふのをやめました。けれども先日「社会」と「連絡」を「とる」おまじなゑ に、まぐろのさしみを数切食べました。又茶椀むしをさじでかきまわしました。食はれるさかながもし私のうしろに居て見てゐたら何と思ふでしょうか。「この人は私の唯一の命をすてたそのからだをまづそうに食つてゐる。」「怒りながら食つてゐる。」「やけくそで食つている。」「私のことを考へてしずかにそのあぶらを舌に味わひながら、さかなよおまへもいつか私のつれになって一緒に行かうと祈つてゐる。」「何だ、おらのからだを食つてゐる。」まあさかなによつて色々に考へるでせう（後略）。（傍点は原文のまま、『宮沢賢治序説』菅谷規矩雄）


　この後に、もし自分がこの食われている魚だったらどうだろうという文章が続くのであるが、こんな手紙を読むと、賢治にとっての食べるという行為は、単に腹を満たすため、食欲のためというのではなく、多分に哲学的・宗教的なものだということがみてとれる。目の前に置かれた食べ物について思いをはせ、それの生きてきた人生や考えまでを想像している。さかなを食べるということは、そのさかなの生きてきた人生全部を自分の内に取り入れることだとでも考えているようだ。

　またもう一方では、食物を選ぶことが、自分のその時の状態の表明でもある。つまり魚を食べることが社会との連絡をとるおまじないであるということだ。両方とも極端といえばそのとおりであるが、賢治においては、食べることがそこまで精神的なものだったのだろう。

　しかし、肉はまったく食べなかったのではなく、必要を感じた時には口にすることもあった。こういう手紙が残されている。


　七月の始め頃から二十五日頃へかけて一寸肉食をしたのです。それは第一は私の感情があまり冬のやうな工合になつてしまつて、燃えるやうな生理の衝動なんかを感じないやうに思はれたので、こんな事では一人の心をも理解し兼ねると思つて、断然幾片かの豚の脂、塩鱈の干物などを食べた為にそれをきつかけにして脚が悪くなつたのでした（この頃賢治は脚気にかかっていた――筆者注）。然るに肉食をしたつて別段感情が変るでもありません。いまはもうすつかり（菜食に）逆戻りをしました。（『賢治随聞』関登久也）


　これは大正一〇年八月二日の日付で、宛て先は関登久也氏である。菜食生活の最中に書かれた手紙であるから、賢治は菜食そのものにはそれほどの執着はなく、あくまで修行ということを中心に、自分の状態をみながら行なっていたのであろう。しかし結局は菜食に逆戻りしている。

　賢治が亡くなったのは三八歳の時である。早死にしたことから、賢治に対して弱々しいイメージをお持ちの方も多いと思うのだが、実際はかなりのハードスケジュールをこなす精力的な人であった。まず歩くスピードがたいへん速く、普通人の二倍以上。原稿を書くのも速く、ある時には一カ月に三〇〇〇枚もの枚数を書いた。朝は暗いうちに起き、夜も遅くまで勉強や仕事をした。これは禅僧の生活と一致するものだといえるだろう。


　日本人の食に対するあり方は、このように独特のものである。

　しかしこれとは別に、もっとユニークな食に対しての視点がある。それは食事によって人の運命を観るという観相術であり、さらにそれを進めて、食を変えることで運命をも改善する運命転換法である。その方法は特別なものではなく、誰にでも実行可能なものだ。次にこれをみていくことにする。


第二章　［食］と運命転換――水野南北の〝人の運は食にあり〟



　観相家・水野南北のひらめき


　南北が生まれたのは宝暦七年（一七五七年）、江戸中期である。幼い頃に孤児となり、一〇歳の時には飲酒を始め、けんかばかりして生傷が絶えなかった。一八歳頃、酒代ほしさに犯罪を犯して入牢するのだが、人生のどん底ともいえるこの時期に、彼は人生に開眼している。すなわち牢に入ったことで、牢内にいる人の人相と、普通の人の人相のあいだにきわだった違いを発見し、観相に興味を持つのである。

　出獄後、彼はさっそく易者のところに行って自分の運命を見てもらうのだが、その結果は彼にショックを与えるものであった。剣難の相であと一年の命であるといわれたのだ。そして難を逃れる唯一の方法は出家であると教えられる。

　命には代えられないと、南北（当時はまだ熊太という名だった）は禅寺を訪れて入門を乞う。しかし、住職としてはヤクザまがいの南北をそのまま受け入れるわけにはいかない。そこで一年間、麦と大豆だけの食事を続けることができたら入門させると約束する。体( てい ) よく追い払われたようなものだが、南北のほうは真剣にそれを実行する。麦と大豆だけの粗食で、港湾労働という肉体労働に従事して生活の糧( かて ) を得た。

　そうして一年がたった。剣難にもあわず、命もある。南北は約束どおり禅寺を訪ねる前に、先の易者のところに行ってみた。すると易者は南北を見るや驚いて、剣難の相が消えたという。何か大きな功徳を積んだに違いないがどうだと聞く。そこで食事を変えたことを話すと、それが陰徳を積んだことになって、相まで変えたのだということであった。

　禅寺に入る必然性がなくなった彼は、それよりも観相家を志すべきだと決心して、諸国遍歴の旅に出る。二一歳の時であった。

　その後、南北は、人間の全身の相を研究するために銭湯で働いたり、骨格や死人の相を確かめるために火葬場で働いたりして、観相を研究した。しかしいくら研鑽を積んでも百発百中というわけにはいかない。なぜだろうと悩んだあげく、解答を求めて伊勢神宮に参詣し、断食と水ごりの修行をする。その必死の思いが天に通じて、

「人の運は食にあり」

　という真理を、霊感として授けられるのである。伊勢外宮( げくう ) の祭神は豊受( とようけの ) 大御神( おおみかみ ) で、五穀をはじめとするいっさいの食物を司る神であるから、その神から授けられた答えだろうと言われている。

　南北はこの霊感を受けてから、「我れ衆人のために食を節す」という決意をして、生涯粗食で過ごした。彼が食べていたのは、主食は麦飯で、一日に麦を一合五勺、副食は一汁一菜であった。米は断って、餅すらも食べなかった。酒は何よりも好きであったが、一日に一合と決め、それ以上は飲まなかった。こんな粗食を、盆も正月もなくずっと続けた。

　南北自身は貧相で、決して成功する相ではなく、短命の相なのだが、このように食を慎んだために、七八歳まで健康に生き、大きな財をなしたという。


　以上が南北の大まかな経歴であるが、以下、南北の著書『相法極意修身録』より食について書かれたところを抜粋、要約して紹介していこう（なお、現代語訳は玉井禮一郎氏のものを参考にした）。


　運がよくなる食事のとり方一二カ条


　① 　運命の吉凶は食で決まる

　・食事量の多少によって、人の貧富や寿命や未来の運命を予知することができる。人間には天より与えられた一定の食事量がある。これをみだりに貪( むさぼ ) り食うものは、天の規律を破るものである。

　・食事の量が少ないものは、人相上不吉な相であっても、運勢は吉であり、それなりに恵まれた人生を送り、早死にしない。特に晩年吉。

　・食事が常に適量を超えている者は、人相学上から見ると吉相であっても、ものごとが調( ととの ) いにくい。手がもつれたり、生涯心労がたえなかったりして、晩年は凶である。

　② 　粗食の者は貧相でも幸運をつかむ

　・食事が常に適量である者の運勢の吉凶は、相として表面に表われているとおりである。

　・常に大食・暴食の者は、たとえ人相上はよくても運勢は一定しない。もしその人が貧乏であればますます困窮し、財産家であっても家を傾ける。大食・暴食して人相も凶であれば、死後入るべき棺もないほど落ちぶれる。

　・常に身のほど以上の美食をしている者は、たとえ人相が吉であっても運勢は凶である。美食を慎しまなければ、家を没落させ、出世も成功もおぼつかない。まして貧乏人で美食する者は、働いても働いても楽にならず、一生苦労する。

　・常に自分の生活水準より低い程度の粗食をしている者は、人相が貧相であっても、いずれは財産を形成して長寿を得、晩年は楽になる。

　③ 　粗食でも大食すれば大凶

　・粗食で少食を守っている者は、たとえ極悪貧窮の相であっても財をなし、出世する。子孫にも財を遺す。しかし粗食にしているとはいっても、大食で、量が一定していない者は大凶である。

　・少食で、量も厳重に決めている者は、相が悪くても福分に恵まれ、万事思うようになる。こういう人は外見弱々しく見えても、病気になることはない。

　④ 　食事時間が不規則な者は吉相でも凶

　・食事の時間が不規則な者は、吉相であっても運勢は凶である。何事も成就しそうで成就せず、生涯、生活が不安定である。貧しい人の場合は思うように事が運ばず、八、九分どおりまでできていても、完成間際で崩壊する。

　・大食で、しかも食事時間が不規則な者は論外で、慎しまなければ、ついには家を潰し病気で倒れる。もし凶相であれば、死に場所さえ定かでない境涯に落ち込む。

　⑤ 　少食の者には死病の苦しみや長患いがない

　・常に大食の者は病みつきから食欲をなくしてあまり食べない。が、少食の者は病気になることはまず少ないが、仮になっても食べないということはない。大食の者は天与の食が尽きてもまだ生命があるために、食べられない状態で長く苦しんだ後に死ぬ。

　少食の者は天与の食がまだ尽きていないために、生命が終わっても食が終わっていない状態にある。食のあるところには生命があるので、その食が尽きる時に生命も自然に亡くなる。だから少食の者には、死病の苦しみや長患いがない。しかし少食であっても、食事が不規則であれば病気にかかりやすい。

　・食を慎み、欠乏に耐える者は、晩年の凶運をまぬがれる。

　⑥ 　大いに成功・発展する相があってもその相どおりにいかないことがある

　・怠け者でずるく、酒肉を楽しんで精進しない者には成功はない。成功・発展しようと思うならば、自分が望むところの一業をきわめて、毎日の食事を厳重に節制し、大願成就まで美食を慎しみ、自分の仕事を楽しみに変える時には自然に成功するであろう。食を楽しむというような根性では成功は望めない。食は成功の基礎であり、その基礎をみだりに食いつぶすものは、その基礎を失うものである。食は恐ろしいまでに大切なことがらなのである。

　⑦ 　美食を続けると消化器系の病気になる

　・無病の相であっても、若い頃から美味を好み、毎日のように美食するものは、老いては消化器系の病気にかかり、食べたくても食べられない業病となる。貧しいにもかかわらず三年間美食を続けると、生命まで失うことになる。

　・「食」というものは「気」に準ずるものである。食が定まらないから、気も定まらない。食が定まらなければ、万事定まらず、損失や災いをなすものである。食が定まると気もおのずから鎮静し、心が治まる。心が不動であれば、災いを生じることはない。

　・収入が定まっている者は、三度の食事も定まる。収入が定まらない者は、食も定まらない。ゆえに収入を定めようとするなら、まず三度の食事を定めるべきである。しかも高い収入を得ようとするならば、少食に定めるべきである。たとえ衣食住の定まらぬ相であっても、三年間食を慎しみ人徳を積むなら、仕事は自然に授かるものである。

　⑧ 　肉体労働者は大食してもよい

　・肉体労働者は一見、自分のために働いているようであるが、実はそうではなく、世のため人のために菩薩のはたらきを働いているのである。ゆえに自分の分以上に大食しなければそのはたらきをなすことができない。だから、そのはたらきの強弱にしたがって食の大小を定めるべきである。そのように食を慎む肉体労働者は、やがて肉体労働から解放されて立身出世する。

　⑨ 　乱心（精神病）の相があっても、常に食を正しく定めている者には、乱心が生じない

　・しかし乱心の相があるうえに、食を定めなくみだりに食べるものは、必ず乱心を生じ、低次元の霊がつきやすくなる。そういう場合は日に三度の食事の他はいっさいの飲食を与えず、これを一〇〇日間続ければ憑きは落ちる。数年間にわたってそういう状態の者は、三年間このような食事法を続ければ自然と治る。これは食事を厳重に定めることで精神が正しくなり、肝気が治まるために治るのである。

　⑩ 　人格は飲食の慎しみによって決まる

　・人が貴く上品になるのも、人品いやしくなるのも、皆、飲食の慎しみによる。名僧知僧といわれる人が貴いのも、慎しみ深く少食にして、食べたいだけ食べるということをしないからである。たとえ博学の出家僧だといっても、食を慎しまずに食べたい放題だと、この人を尊敬しようという気にはならない。

　⑪ 　酒肉を多く食べて太っている者は、生涯出世栄達なし

　・酒肉をたくさん食べる者は太っているように見えるけれども、実はそうではない。酒肉のために血の量が増し、心気がゆるんでいるために太っているように見えるのである。人は心気を本としている。心気がゆるんでいる者で出世栄達した者はいない。

　・大食をした後は気が重くなって眠くなる。目覚めた時も体がだるく、頭が重い。これらは皆、心気がゆるんで全身の肉が締まらなくなるからである。よって身のほど以上の大食をなす者は、生涯いい仕事はできない。

　⑫ 　高齢者の肉食は害が少ない

　・かなりの高齢になれば、体力が衰えるので肉食でそれを補っても害はない。しかし節度をもって食べることが大切である。

　・都会では美食や肉食が多く、動物を殺してその肉を食べる。そのため意識が高慢になり悪心が生じる。常に粗食に耐える者は万事控えめで、悪心が生じることがない。田舎や山村に悪人が少なく、都会に多いのはそのためである。

　・鳥や魚も食物であるが、その命を奪って食べるのだから、その報いを受けて長命を短命に変える。しかし肉食でも正しく、しかも少なく食べる時は、生命を養い長寿を保てる。これは鳥や魚は人間に食べられることでその役目を果たすので、食べることが一種の慈悲行となるからである。


「われ、衆人のために食を節す」という精神を貫く


　この南北の教えの根底に流れているのは、彼が霊感を受けて決意した、

「われ、衆人のために食を節す」

　という精神である。

　自分の食を減らして、その減らした分を神に捧げ、また人に施す。自分の食分を少なくし、空腹をガマンして、食べるはずであったものを、人に与える。

　そうすると陰徳を積んだことになり、少食を続けることでその徳が積もり積もって、開運につながっていく。徳が貯金となり、その貯金を使うことで運が開けていくということである。

「神への祈りをかなえる法」として南北が教えている方法に、このことが語られている。

　運が悪くて難儀ばかりしているが、神に祈れば運が開くかという質問に答えたものである。

「真心をこめて祈らなければ、神は感知してはくれない。真心をもって祈るとは、自分の命を神に献じることである。食は自分の命を養う基本であるから、食を神に献じることは、自分の命を献じるのと同じことである。

　献じる方法は、いつもご飯を三膳食べる人なら、二膳だけにしておいて、一膳を神に献じる。といっても、その一膳分を神棚などにお供えする必要はなく、心のなかで念じればよい。自分が祈りを捧げたい神仏を思いうかべて、その神仏に向かって『三膳の食のうち一膳を捧げ奉ります』という。そうしてから、二膳を食べると、その一膳は神仏が受けとってくれる。

　自分が十分に食べて、そのうえで神仏に上等な食を献じても、神仏は喜ばれない。自分が我慢して献じる、その真心を受けてくれるからである。菓子でもおかずでも何でも、三つ食べたいと思ったら二つにしておいて、我慢したひとつを神に捧げる。食べる時には、いつでもそうするのである。

　そうすれば、どんな願いごとでも叶えられる。小さい願いごとなら一年で、ふつうの願いごとなら三年、そして大望は一〇年で、叶うのである」

　また、こういう言い方もしている。

「自分の食分をけずって、生あるものに施すことは大きな陰徳で、心がこもっている。別のものを用意して施したのでは罪になってしまう。自分の食分を施すことが、本当の陰徳である。そういう人は、とくに祈らなくても、神は守ってくれる。またそうやって食を慎むことで、おなかのなかは楽であるし、気持ちもすっきりしている。病気にかかる心配はないし、災難にあうこともない。

　そんなふうにして、三度の食事を慎むならば、一日に米一合分の陰徳を積んだことになり、一年たてば三斗六升、一〇年で三石六斗になる。それだけ自分の天の扶持をのばしているわけで、それを土台として立身出世ができる。ひたすら自分から徳をのばさないと、天から福徳がまわってくることはない」

　自分の食分をけずる、ということが強調されている。これは、自分のいまの経済状態で食べられる献立より、いくぶん低めのものを食べようということだろう。グルメ志向で、不相応なものを食べると、その分、自分の持ち分の徳を使ってしまうことになる。

　反対に、大金持ちが粗食を続けると、ますます運は開けていくことになる。この代表的な人物が、第三章でみる徳川家康である。


　早起きをして太陽を拝する開運法


　南北はこのように、食を基本として何でも判断していくわけだが、その関連で、その他の開運法も示している。いずれも食と不即不離にある開運法である。食を慎しむことはいいことだと理解できても、実行が難しい。その実行をしやすくするのに、この方法からアプローチすると効果的である。


　① 　毎朝昇る太陽を拝む

　これは長寿法として紹介されている。病気がちで短命の相がある人は、毎日、朝日を拝むことで、健康で長生きすることができる。南北自身、短命の相で、三〇歳までの寿命しかなかったが、この方法を実践することで乗り越えた。

　② 　早起きすると運がよくなる

　早朝の太陽には〝少陽発達〟の気がある。この気を受けると、心気がすこやか、活発になり、そうなると自然に運命もすこやか、すなわち好転する。運は気につれて巡るものだからである。

　朝寝をする人はその気が受けられないため、心身がすこやかではないし、顔色もわるい。好運を保とうとするなら、早起きをして仕事に精進し、飲食を慎しむほかはないのである。


　これと関連しているが、以下のような言及もある。


　③ 　夜、仕事をすることは大凶である

　人相などが悪くても、朝寝をせずに、早起きして仕事をすると、運は人相どおりにはいかず、よくなっていく。しかし、夜に仕事をすると、その逆のことが起こる。だから、夜やらなければならない仕事があったら、その分、朝暗いうちから起きて、夜仕事をするときと同様、あかりをつけてやればよいのである。そうすれば、太陽の運行とともに生命がめぐり、自然に運気がよくなり、裕福になっていく。


　ほかにも太陽と運気について言及している項目があるので、太陽と運気とは深い関係にあるようだ。


　④ 　衣服や住まいもぜいたくすぎるものは大凶である

　これについても食と同じで、自分の経済状態にふさわしいものをよしとする。またそれよりも質素にしている人は、その徳が内面に満ちているので、苦労が少ない。

　だいたい飲食を慎む人は、衣服や住まい、家具なども質素にしているものである。食がきちんとしていれば、他の要素もそれに従うからで、すべて食が基本である。

　⑤ 　貧乏人が金持ちの趣味をまねるのはよくない

　貧乏人でありながら、金持ちの楽しみをまねる者を〈求貧〉という。自分から貧乏を求めているということだ。そんな暇があったら、自分の仕事を楽しみとして努力することである。そしてそれができる身分になったら、楽しめばよい。

　⑥ 　倹約は吉であるが、ケチは凶である

　真の倹約とは、出すべきところには出して、余計なものを省くことである。使用人のための飲食をケチったり、世間の付き合いで出すべき金を出さなかったり、施すべきところを施さなかったりするのは、ケチであり、運も悪くなる。使用人が店の金をごまかしたり、他人からの信用がなくなって仕事が減ったりするからである。

　そんな心の狭いことはせずに、使用人にケチったりせず、自分一人の飲食を慎むことである。自分がまず率先して質素にすることで、おのずと家族や使用人もそうなっていく。

　⑦ 　精神を治めるために食を慎む仏法

　いくら念仏を唱えても、飲食を慎まねば仏意にそわないと（南北先生は）いうが、どんな高徳の僧の話を聞いてもそんなことはいわない、（先生の）独断と偏見ではないのかという質問に答えたものである。

　南北はもともと法華経を重んじているが、こう答えている。

「仏法は精神を治めるための根本である。だからまず飲食を慎むことをもって、その修行のはじめとする。心が乱れることはすべて、飲食がその原因となっているからである。飲食を慎めば心は静かになり、不動心を得る。そうなれば、道をきわめることはたやすい。

　酒や肉をたくさん食べると、意識が活発になり、心が乱れ、思わぬ不善をなしがちである。食べすぎたり飲みすぎると、心がにごり、気分は自然と重くなって迷いの心を生じ、道をきわめることが難しくなるものである」


　以上、現代のわれわれにとって参考になりそうなものを中心に紹介した。南北はこの方法で多くの人を救ったということである。


　井上正鉄の発願「われ、一飯を残して、人の飢えを救わん」


　南北の弟子に、吐菩加美( とほかみ ) 神道の創始者である井上正鉄( まさかね ) がいる。禅の修行から神仏の世界に入り、その後、漢方、易学と相学、指圧、神道の修行･･････と、当時の〝精神世界〟ともいうべき世界を遍歴している。そして吐菩加美神道を創始したのだが、このなかには南北の思想を神道霊学的に展開したものが息づいている。

　正鉄が生きていたのは江戸時代のなかでも、飢饉があいついでいた頃である。多くの人が飢え死にし、子供は間引かれていくという状況にあって、南北の教えそのままに、

「われ、一飯を残して、人の飢えを救わん」

　と発願し、神道による救済を始める。

　しかし、この運動は幕府の弾圧にあい、正鉄は三宅島に流され、そこで亡くなっている。

　宗教家の場合は、運・不運を、ふつうの人間の尺度では計れないので、ここで正鉄が亡くなっていることについて何ともいえないのだが、神道もまた食を重視していたことは、南北が霊示を受けたのが伊勢神宮であったことをみても明らかである。


　天変地異と食の関係を説いた梅辻規清


　やはり同時代に生きていた梅辻( うめつじ ) 規清( のりきよ ) という神道家も食に対して独自の見解をもっていた。彼は南北とは視点の異なる見方をしている。

　食、そして衣と住という人間の営みが、他者との〝競い〟のなかでエスカレートしていくと、天変地異や大飢饉が起こるというのである。

　それは人々の心のあり方、つまり思念のなせる仕業である。感謝を忘れて、人の衣や住居、食べ物をうらやみ、嫉妬する。またそれを手に入れようと醜い行ないをする。そうすると道徳的な常識（良識）がなくなって物欲のとりこになり、「もっとほしい」とか「あれも不足、これも不足」という気持ちで心がいっぱいになる。そういう「欲しい、欲しい」という思念のエネルギーが莫大なものになり、現実化してしまう。

　思念の力ばかりでなく、物理的な因果関係によっても起こる。たとえば天保七年と八年は大飢饉が全国を襲い、一〇万人の死者が出た。「天保の大飢饉」である。

　この原因は、人々が衣のぜいたくに走ったための人災であるというのが、規清の見方である。

　木綿より絹を好み、絹ばかりを人々がほしがると、商人は当然増産を考える。絹は蚕という生きものを殺して得るものである。サナギになって繭( まゆ ) に入っている蚕を煮沸して、その繭をとり、それが絹糸となる。

　養蚕家はお金がほしいから、絹を増産せよといわれれば、そうする。とくに蚕の飼育は換金作物振興ということで、大いに勧められたので、ヒエやアワ、豆、芋、野菜など、食物を作っていた畑を、蚕の食料である桑細にし、それだけでは足りず、森林を焼き払って桑畑にしている。

　そのため、森林に住んでいた動物が住み処と食物を失って、田畑や民家を襲う。また森林がなくなると、土の保湿能力が著しくなくなって、ちょっとした雨でも洪水が起こり、民家はむろんのこと、田畑をも押し流してしまう。せっかく作った作物がなくなって食糧不足になるのである。

　人間のぜいたく志向が、異常気象と結びついて、天保の大飢饉が起こったと、規清は主張し、そうならないための生活の改善や、江戸市中を改造する建築プランを具体的に提案している。

　しかし、こうした活動は幕府にとって目ざわりなもので、井上正鉄と同様、囚われの身となり、八丈島に島流しされている。

　規清を乗せた御用船は八丈島に行く途中、三宅島に寄り、そこで風待ちをしているが、そのとき三宅島に流されていた正鉄と対面したということである。このとき二人のあいだでどんな会話が交わされたか、何となく想像できる。


　健康法の元祖・西勝造も粗食・少食を提唱


　こうした粗食・少食の教えは、その後も宗教の世界で生き続けているが、もうひとつの流れとして、現在、大ブームの健康法への流れがある。健康法の元祖ともいうべき西勝造の西式健康法が、その代表的なものである。西式健康法は現在出ているさまざまな健康法の源流であるから、健康法のベースには粗食・少食があるといっても間違いではない。

　西勝造は、粗食・少食に加え、温冷浴や生食療法（生野菜を食べ、生水をたくさん飲む）を提唱し、医者が見放した病人を次々に治している。なかには末期ガン患者の治癒例もある。この時代はまだ水も野菜もいまほど汚染されていなかったから、西式健康法は効果抜群だったのであろう。

　ヨーロッパやアメリカで多くの人の支持を得た桜沢如一の健康法・マクロビオティックも、粗食・少食を旨としている。

　しかし〝開運〟という視点から粗食・少食がいわれることはほとんどない。そういう意味で、水野南北という人物は、もう少し脚光をあびてもよいのではないかと思う。

　次の章では時代を少々さかのぼって、戦国時代の武将の食を調べてみる。この結果も、意外や意外で、最終的勝利者となった人物は禅僧のように粗食なのである。


第三章　戦を勝ち抜いた武将の［食］――徳川家康は頑固な麦飯愛好者



　季節はずれのものは食べなかった家康


「織田がつき、羽柴がこねし天下餅、寝ているままで食うは徳川」

　という有名な狂歌がある。同じ戦国時代を生き抜き、数々の戦( いくさ ) で戦場をかけめぐりながら、最後の勝利者となったのは徳川家康（一五四二～一六一六年）である。信長にしろ秀吉にしろ、天下はとったものの一代限りで終わり、世襲させることはできなかった。

　なぜ、家康にはそれができたのだろう。

　長寿にめぐまれたこと、子供がたくさんいたこと、すでに時代の流れがそうなっていたこと、など、さまざまな理由をあげることができる。そんな理由のひとつとして指摘したいのが、家康の食のあり方である。

　家康の食は、天下をとった人間にしてはたいへん質素なものである。民、百姓なみの食事といっていいのではないだろうか。

　家康の主食は麦飯である。白米は決して食べなかった。そして、副食として必ず食べていたのがみそである。三河はみその産地として有名で、それも大豆だけで作った豆みその産地である。家康はそのみそで作ったみそ汁を常食していたが、しかし椀が透けてみえるほど薄いものであったという。塩分を気にしたのではなく、みそを倹約するという粗食主義からくるものであった。

　おかずは野菜の煮物や小魚などで、それも一品か二品、つくだけである。一汁一菜、あるいは二菜が日常の食であった。ごちそうや季節はずれのものは決して食べず、誰かからめずらしい到来物があっても、家臣に分け与えて、自分はひと口も口にしなかった。

　こういうエピソードがある。

　信長と同盟を結んでいた時のことである。一一月だというのに、信長から見事な桃が届けられた。近臣たちはこのめずらしい贈りものに大はしゃぎだったが、家康はぜんぜんうれしそうではない。それどころか、浮かれている家臣たちに向かって、こう言ったという。

「この桃はたいへんめずらしいものではあるが、右府（信長のこと）とわしとでは領国の大きさが違い、したがって身分が格段に違う。わしらのような小身が、このようなめずらしいものを好めば百害あって一利なしである。食べ物にせよ何にせよ、めずらしいものを好むようになると、それに無用な金を費やして家臣を養うこともできなくなる。志のあるものは、珍物は好まないものである」

　そういって自分はひとつも食べず、すべて家臣に分け与えた。

　この話を後になって聞いた武田信玄は、

「家康は大望あるゆえ、養生を至上として、季節はずれのものは食わないとみえる」

　と、大いに感心したという。

　信玄も粗食で玄米とみそが中心の食事をしていたから、家康の心がわかったのである。

　また信玄は、ここいちばんという大事な戦の前には、肉食を断って、神に勝利を祈願した。その願文が、信玄が軍神としてうやまった諏訪大社に残されている。病死さえしなければ天下取りの有力候補であった人物であり、禅の修行もしていたから、粗食と開運についてあるいは知っていたのかもしれない。


　白米厳禁、庶民の食・粟飯は食べる


　家康には、食についてのこんなエピソードもある。

　あるとき近臣のひとりが、家康が麦飯ばかり食べているのを見て、気をきかせたつもりで、下に白米をいれ、上に麦飯をほんの少しかぶせるようにのせて、差し出した。気のきく奴と喜んでもらえるかと思ったのである。ところが大違いで、家康は一目で見破ると不機嫌になった。そして自分の心がわかっていないとなげいた。

「わしが単なる倹約で麦飯を食べていると思うのか。いまは毎月のように戦がある。戦場に行っている者たちは安心して寝食すらできないというのに、どうしてわしだけぜいたくができよう。わしが先に立って倹約すれば、いささかなりとも戦費の役に立つと同時に、下下( しもじも ) の者をいたわることにもなるのだ」

　また、こんな話もある。

　本能寺の変のすぐあとのこと、家康は天下をとった明智光秀の追っ手から逃れて、伊勢の白子浦から船に乗り、領国・三河へと向かった。その船のなかで家康は食事を所望した。逃避行なので、ろくに食事もとっておらず、おなかがすいていたのだ。

　しかし、突然の乗船だったので、船のほうでも何の用意もしていない。そこで船子（船のなかで働いている乗組員）が自分の用意してきた弁当を差し出した。船子の弁当だから、中身はたいしたものではない。おかずなどなく、主食のみだが、その主食も粟と麦が大部分、米はほんの少し入っているだけであった。

　家康が、何かおかずをといったので、船にあったニナというタニシに似た貝を塩辛にして出した。

　そうしてこんな粗末なものを食べるのだろうかと見守っていると、家康は上機嫌で、風味がよいといって三椀もおかわりしたという。粟は上流階級の食卓には決してのぼらないものであるし、ニナも同様である。それを空腹であったとはいえ、おいしく食べられるのは、家康の日頃の食事が質素だったためである。


　家康の粗食主義、一〇の特徴


　家康は粗食を自分の主義、あるいは哲学としているが、それにはこんな特徴がある。


　① 　えりごのみしないで何でも食べる。一色（ひとつのもの、一種類のもの）ばかりをたくさん食べると、積もり積もって毒となるので用心すべきだ。患ってからでは遅い。

　② 　食べ物には火を通し、温かいうちに食べる。

　家康は、食べ物は温かいうちに食べたほうがいいという、孔子がいったことを実行した。また、当代きっての名医・曲直瀬( まなせ ) 道三( どうさん ) は、胃腸を温めることが健康の秘訣、といっており、これも知っていた。

　当時、発病率がもっとも高かったのが胃腸の病気で、病気の半分を占めていたのでとくに注意していた。火を通すのは食中毒の予防にもなる。

　③ 　食べたいという欲をおさえて、簡素・節約する。口が食べたがっても、その欲を抑えて食べすぎないようにする。これが養生の要である。

　④ 　いつも美味ばかり食べていると、それはおいしいものではなくなってしまう。平日の食べ物は軽味なものがよく、美味はせいぜい月に二、三度でよい。

　⑤ 　季節はずれのものは食べない。

　⑥ 　味よりも健康を優先する。その根本は「粗食」である。

　夏には消化がいいものをといって、うどんを時たま食べている。その頃のうどんは全粒粉で打ってあるので栄養価は高い。これに必ずしょうが汁を入れさせたというから、夏バテ防止と健胃を意識したものであろう。鷹狩りに出かけて、主食がなくなったときは、近所の農家でイモを調達し、それですませている。

　⑦ 　酒については、こういう意見であった。「酒は元気をかきたてるが、量をすごせば争いを起こす。慎しむべきである」と、百薬の長としての酒は奨励したが、大酒を飲むのは厳として禁じ、家臣にも徹底させた。

　⑧ 　医に通じ、漢方薬を研究して、必要と思う薬はみずからが指揮を執って調合させている。そして、それを常備薬としたが、たいへんよく効いたという。時折、諸大名にも分け与えている。

　⑨ 　医者も自分で選んでいた。が、医者のいうことを鵜呑みにせず、身体の反応を観察して養生を心がけていた。

　⑩ 　運動とストレス解消をかねて、鷹狩りによく行っていた。晩年になるにつれて頻度も多くなった。鷹狩りはよく歩くし、野宿することもあるので運動や気分転換にはもってこいである。


　以上である。

　このような食を実践していたからか、家康は家臣の受けがよく、「上様のためなら命を投げだしても惜しくない」というものが五〇〇人はいた。それを聞いた豊臣秀吉がたいへんうらやましがったという。

　家康が、これほどまできちんとした健康管理ができたのは、強力なブレーンがいたからである。それが天海僧正だ。


　健康法のブレーンは天海僧正


　天海僧正（一五三六頃～一六四三年）は、家康・秀忠・家光と、三代にわたって将軍の相談役になった人物である。一〇八歳まで生きたというが、一説には一四〇歳だったとも言われ、平均寿命が三〇歳という時代にあって驚異的な長寿を保った。

　家康が七五歳で死んだ後、さらに二七年も生き、家康を祀る日光東照宮の造営では陣頭指揮を執った。上野の寛永寺を開いたのも天海であるが、このとき天海は八四歳だったというから、長寿社会の現代に生きるわれわれからみても驚異的である。

　天海が家康に教えた健康法は、

「粗食・正直・日湯・だらり」

　であった。

　日湯というのは、毎日入浴することで、だらりとはリラックスして日々を過ごすことである。

　また、こんな健康法の歌も作っている。

「気は長く　つとめはかたく　色うすく　食は細うして　心ひろかれ」

　天海自身が好んで食べたのは、きな粉や納豆、豆腐などの大豆食品であった。最近、建脳食として注目を浴びている食品である。

　天海と家康が出会ったのは天正一七年（一五八八年）である。天海が五二歳のときのことだ。しかしそれ以前に、天海は武田信玄と出会い、信玄のもとでしばらく客になっていたという。信玄が粗食であったのも、天海の影響があるのかもしれない。

　ともあれ、一介の僧侶にすぎなかった天海が、天下の徳川家の相談役となるまでに出世したのも、粗食は開運の秘訣という法則を証明しているようである。


　春日局( かすがのつぼね ) は玄米とぬかみそ汁で、莱( さい ) はなし


　徳川家の話が続くが、もう一人、粗食主義で開運した人を追ってみたい。春日局( かすがのつぼね ) （一五七九～一六四三年）である。

　浪人の妻から、三代将軍・家光の乳母となり、大奥をとりしきるようになったばかりか、政治の表 にも強い権力を示した女性である。

　この人もまた、粗食を通し、着るものも、どんな上等の着物でも着られる身分にありながら、特別な場合を除いては木綿しか身につけなかった。

　春日局の食事は、玄米とぬかみそ汁である。それにごくたまに赤イワシを食べる程度であった。ぬかみそ汁というのは、漬物の床であるぬかみそをすりつぶして、みそのかわりに使った汁物である。赤イワシとは塩漬けにしたイワシのことだ。

　まさに一汁一菜、しかもその一菜も毎日ではなかったというのだから、本当に粗食である。そのかわり、飯と汁だけはたっぷりと作らせ、残りを下働きのものに下げるようにしていた。彼らは三度の食事をふるまわれるので大助かりだったということである。

　あるとき家光が、春日局のつましい食事を見て、もう少しよいものを食べるように勧めると、局はこのほうが胸につかえず、いただきよいのだと答えている。乳母をしていた頃は毎日ごちそうを食べた。それは家光の乳のためと思ってのことで、その用がなくなったいまでは、もうそんなごちそうは必要ない。そこで以前の食事に戻したのだと答え、変えようとしなかった。

　そこで家光は、以後、自分の膳に出たものから一、二品とると、これは婆( ばば ) にやってくれ、と下げ渡したという。家光と局とのあいだの絆は、母子以上に強かったから、そうせずにはいられなかったのだ。

　春日局が生涯にわたって尊敬し続け、また頼りともしたのが、家康である。乳母として採用される時、面接試験で家康に会っているから、それ以来の付き合いである。何か事があれば、家康に相談している。

　家康のほうも局の政治力を買って、進んで後ろ楯になっている。実の母のお江( ごう ) の方にうとまれた家光は、嫡男でありながら、軽くみられていた。次男に家督をつがせ、将軍にしようとお江の方が動いていたからである。局はそれを家康に直訴して、改めさせたいきさつがある。

　家康の死後も、局は彼が祀られた東照宮を信仰している。したがって天海僧正にも会っているわけで、僧正も局のよき相談相手であっただろう。身分が高くなっても、粗衣・粗食を貫いたのは、天海の忠告もあってのことであろう。

　しかし局は、健康でありたいとは願っただろうが、どうしても長生きしたいとは考えていなかった。それは薬断ちをしていたことから推察できる。病に伏したときに、周囲のものが薬を勧めたが決して飲もうとしなかった。

　というのも、家光が二六歳で天然痘にかかって重態となり、生死の境をさまよったとき、局は水ごりをとって、こう祈念したからである。「私が身がわりとなって命を差し出しますから、どうか上様の命を助けてください」と。

　そう祈念しておきながら、いまさら薬など飲んで助かろうとするのは、神仏に対して申しわけないというのだ。そうしてそのまま亡くなってしまった。六五歳であった。

　家光はその死を哀しみ、七日間の喪に服し、江戸中の鳴りものを停止させた。これは異例の扱いであった。


　家来と同じものを食べた大久保彦左衛門


　家康の臣家で「天下のご意見番」として有名な大久保彦左衛門（一五六〇～一六三九年）も粗食の人である。

　戦国時代に生まれ、数々の戦で戦場をかけめぐった彼は、平和な世になってからは、「化石」扱いである。武士としてよりも役人として優秀な人間が出世する世の中になったからだ。しかしそんな扱いなどものともせず、ガンコじいさんとして言いたいことを言ったため、昔ながらの気骨をもつ武士には圧倒的な人気があった。

　彼が書いた『三河物語』は、自分の子孫のために書かれた私家版だが、手書きで写されて旗本のあいだで回し読みされ、隠れたベストセラーとなった。

　彦左衛門の食事は、粟( あわ ) ・稗( ひえ ) ・麦で作ったかゆが中心である。戦が終わって生活が多少楽になっても、変えなかった。家来と同じものを食べるためである。自分の家族だけぜいたくをするということはしていない。

　怖いもの知らずで、相手が家康であろうが家光であろうが、筋が通らないことは、きちんと意見している。家康でさえ、その頑固さにはカブトを脱いだほどである。無欲で、保身など考えない性格だったからであろうか、八〇歳という天寿を全うした。


　水戸黄門はものごとに執着しない粗食主義者


　永戸黄門の名で知られる徳川光圀( みつくに ) （一六二八～一七〇〇年）も、粗衣・粗食の人である。父親が家康の一一人目の息子にあたるので、光圀は家康の孫ということになるのだが、その食事は一汁二菜、ごくまれに一汁三菜であった。内容は、みそ・塩・野菜、そして地元でとれるイワシやサケなどである。

　結婚して四年で妻に先立たれたが、生涯、側室はおかなかった。光圀には兄の子供を跡継ぎにしたいという意図があったから、そのためとも考えられる。父親の意志とはいえ、兄がいながら自分が跡取りになっているのは心苦しいと思い続けていたのだ。

　光圀は背が高く、色白で、鼻すじが通って、魅力的な眼差しをした美男子だった。そのうえ、若い頃はおしゃれで遊び好きだったので、彼が登城する日には、その姿をひと目見ようと、群衆が殺到して大騒ぎになったという。

　幼い時に、庶民と接することが多い環境で育ったため、藩主になってからも庶民感覚を失うことなく領地を治めた。そのため人望が篤かった。藩主の地位を、余力を残したまま兄の子にゆずると、自身は城を出て質素な隠居所に移り住んでいる。

　引退してからの生活が「水戸黄門漫遊記」として知られている。日本全国というわけにはいかなかったが、住んでいる周辺は、あのように徒歩で歩き回り、親孝行の者にはほうびを与え、横暴な役人はこらしめたという。巡るときの服装は粗末な薄い着物と、古びた頭巾であった。

　ふとんの上げおろしから何から、身の回りのことは自分でし、時には料理を作ることもあった。江戸からの使いが来たときなど、カツオを一本買ってこさせ、それを自分でサシミにし、中落( なかおち ) は汁の実にして、一汁一菜の手料理でもてなしたという。

　光圀の信条は「物事に執着しない」ということであった。その結果として「色・欲・酒を避けた生活」を志していた。若い頃は将来が危ぶまれたほど遊んで、芸人のようななりをした不良少年だったというから、何か大きな出来事に遭遇して、変わったのだろう。だから少々厳しい生活を自分に課し、実行している。この方針は七三歳で亡くなるまで貫かれている。


　みそ造りを奨励し、保護した武田信玄


　時代が前後するが、これまでに何度か登場した武田信玄（一五二一～一五七三年）と、敵であり盟友でもあった上杉謙信（一五三〇～一五七八年）をみてみたい。

　粗食の信玄が重要視した食物は、前述したようにみそである。とくに戦のときには、兵糧として十分のみそを用意させた。

　武田家文書のなかにこういう項目がある。


　一、川中島をはじめ、信濃国（長野県）全域の河沿い左右五里（約二〇キロ）の奥地まで、各戸必ずみそ作りを奨励すること。

　一、軍がみそを徴発するときは、三年みそから高額で買い上げ、自然備蓄量を増加させること。

　一、混入米麦は、現地の実情に応じて、百姓の好みを採用すること。


　このなかで、川中島近くの河沿いの農家にみそを作るよう奨励しているのは注目に価する。川中島での合戦は、信玄が謙信とのあいだで何度となく繰り広げた戦であるが、その合戦地のすぐ近くで、みそ作りを強く勧めている。戦のさなかでもすぐにみそが手に入るようにという配慮で、それだけ兵糧としてのみそを重視していたということだろう。

　また、城を守る際の食糧として信玄が重視していたのが、米・大豆・みそ・塩・ぬか・乾魚・海藻である。

　また、甲斐の名物であるほうとうも重視していた。ほうとうは、いろいろな野菜といっしょに生うどんを煮込み、みそで味をつけた料理である。甲斐は山国で、水田も少なかったから、小麦の栽培がさかんで、粉食が発達した。ほうとうはそういう地で生まれた食べ物で、ありあわせの野菜とみそさえあれば作れるし、腹もちのよいバランスのとれた料理だったから、珍重された。


　一生、禁欲僧のような生活を送った上杉謙信


　一方、上杉謙信は、宗教的理由から粗食をした人である。年少の頃より求道の志が篤く、仏門に入って修行三昧の生活をしたいと念願していた。しかし時代がそれを許さず、何度か出家の決意をして行動を起こしながら、実現していない。

　そこで二七歳のとき、在宅、有髪のまま臨済宗に帰依している。法号は宗心である。以後、長尾入道宗心と名乗り、生活も禅僧と同様に厳しくしている。妻帯もせず、一生、不犯である。

　謙信がよく食べていたのが、梅干である。主食は玄米であった。酒が好きだったが、騒いで飲む酒ではなく、一人静かに盃を傾けるのが好みであった。四九歳で亡くなった彼の死因は脳卒中である。酒と、肴の梅干のせいとも考えられるが、人生五〇年といわれた時のことだから、年に不足はないだろう。

　現代でも人気があるのは、そんな禁欲的な生活と、正義のための戦しかしなかったことだ。領土を広げるための戦はしたことがなく、ただ攻めてくるものは防がねばならないから、そういうやむにやまれぬ事情があった時のみ戦った。

　しかし、いざ戦となるとたいへん強かった。守り神として、いつも護摩をたいて祈願をこめていた毘沙門天( びしゃもんてん ) を象徴した旗印のもと、勇敢に戦った。

　江戸時代にはやった、戦国武将の軍兵の強さをうたった歌によると「一が上杉、二が武田、三が徳川、四が斎藤、五が北条、六が今川、七が織田･･････」となっている。謙信の軍兵は後世に伝えられるほど強かったのである。

　しかもその軍兵はどこに進軍しても、土地の人には歓迎された。土地の住民の食料を略奪したり女性を連れ去ったりする兵がいなかったからである。軍の規律がたいへん厳しく、しかも徹底していたのである。そればかりか、行く先々にある神社に砂金を奉納したので、喜ばれたということだ。


　一汁一菜と木綿の着物で財政を立て直した上杉鷹山


　この謙信の精神を受け継いでいるのが、上杉家七代目藩主・上杉鷹山( ようざん ) （一七五一～一八二二年）である。時は江戸中期（一七六〇年頃）、田沼意次( おきづぐ ) が権勢をふるい、ワイロが公然とまかり通っていた時代である。先代からのぜいたくがたたって、この頃の上杉家は財政が苦しかった。鷹山はその立て直しのため、藩をあげての大倹約令を出した。


　一、ふだんは木綿の着物を着ること

　一、毎日の食事は一汁一菜にかぎること、ただし歳暮は一汁二菜で祝うこと


　というものだが、これは家臣から猛反対にあう。とくに老臣の反対が強く、彼らはいままでのぜいたくな生活を捨てるのを嫌がった。それを主人である鷹山が実行するよう頼み、自分はいちばん最初にはじめている。しかもそれは、その倹約令よりさらに質素であったという。

　朝食はおかゆが二膳と漬物のみ、昼食と夕食は一汁一菜、ごくまれに干魚がつく程度。こんな食事を、一七歳で藩主になり、三五歳で引退し、七二歳で死ぬまで続けている。時折、うどんを食べることもあった。酒はいっさい飲まなかった。

　庭園も泉水や庭石で調えるようなことはせず、雑木、雑草の茂る自然な状態にしている。楽しみは読書であった。

　そんな倹約は、しかし、下下( しもじも ) の者には求めてはいない。もっぱら身分の高い家臣や裕福な商人に対してである。庶民はただでさえ苦しい生活をしているのだから、それ以上苦しめてはいけないといっている。だから家臣にほうびを与えるときも、減額したりはしていない。

　鷹山も光圀同様、生涯側室をもたず、一〇歳年上の妻と仲睦まじく暮らしている。妻は八一歳で亡くなり、鷹山もその後しばらくして、眠ったまま、朝、目がさめなかったという、患うことのない安らかな死を迎えている。


「目が食べたがるだけ、二食で十分」といった徳川吉宗


　江戸幕府八代目将軍・吉宗（一六八四～二七五一年）は、享保の改革を行なって破綻しかけていた幕府を立ち直らせた人物である。テレビで名高い大岡忠相( ただすけ ) を町奉行に抜擢した将軍といえばなじみがあるかもしれない。

　吉宗は一八〇センチもある偉丈夫で、エネルギッシュによく働く人物であったが、粗食であった。一汁三菜だが、一日二食しか食べない。

「人を養うには二食で十分である。その他はみな、目が食べたがるだけで、ただ腹のおごりである。太平の世での飽食になれてしまうと、いざという時がきたらどうするのか。昼夜の関係なく奔走し、兵糧も乏しくなるという状態のなかで役に立たなくなるのではないか」

　吉宗のこの考え方は父親ゆずりである。父・紀州大納言光貞は、常に彼にこう言いきかせていたという。

「自分がごちそうを腹いっぱい食べて、ぬくぬくとした上等の温かい衣服にくるまっているようではだめだ。まず自分から節約して、一飯でも多く家来に分けてやる、着物も一枚でも多く着せてやるというようにしなければならない。自分が腹いっぱい食べたいと願うのならば、家来の者全員にまず施し、それから国に、それから天下に施すように考えよ」

　吉宗もそれをよく心にとどめ、食にかぎらず着るものも質素にしていた。天下の将軍が質素なものを着ていれば、家臣も将軍よりはいいものを着るわけにいかず、全体的に華美な服装が改まる。

　吉宗は衣服の色は表は黒、裏は茶と決めており、冬の寒い日でも襦袢は着ず、下がさねは二枚にしていた。冬の暖房も節約して、着物を火で温めて着るという、歴代将軍がやっていたようなことはしていない。

「昔は上の者も下の者もぜいたくを知らず、何につけても簡素、質素であった。だから費用もちゃんと足りて、万民、安穏でいた。それなのに、太平の世が続くうちに人々は華美を好み、ぜいたくをするようになった。それで世の中が貧しくなったのだ。衣服や調度品の数が増えて、多くなっていくのはよいことではない」

　そういって、実行したため、幕府の財政は奇跡的に持ち直し、彼が将軍職を引退する前年（一七四四年）には、幕領の総石高と収納高が江戸時代を通じて最高になるという快挙を成し遂げている。

　吉宗はまた、家康同様、医療にも造詣が深かった。医学書をかたわらにおいて熱心に勉強している。行政としては、辺地にいる貧しい人々が、お金がないゆえに医者にかかれず、むざむざ死んでいくのを助けるため、施薬院を設けた。文字どおり、薬を施す場所で、そこへ行って頼めばすぐに薬がもらえた。これが後に小石川養生所という、貧しい人を助ける病院となる。


　以上、粗食主義の武士およびその周辺人物を取り上げてきたが、この他にも食にこだわった人はいる。自分の献立を自分で決めていた伊達政宗（一五六七～一六三六年）、「この身ひとつに美食は好まず」といって、玄米と赤イワシで過ごした宮本武蔵（一五八四～一六四五年）、秀吉の妻であり、豊臣が滅びたあとは尼となった北政所( きたのまんどころ ) （一五四八～一六二四年）も粗食であった。

　こうした人々の傾向（共通点）を、少々乱暴ではあるが箇条書きでくくって、この章のまとめとしたい。


　① 　若いときに厳しい環境にあって苦労している。

　② 　精神的バックボーンをもっている。それは宗教的なものであったり、師である人物であったりする。

　③ 　身分が下の人を粗末にせず、大切にしている。

　④ 　食とともに、衣も質素である。


　以上だが、ここから人生成功の秘訣が垣間見えてくるようである。

　いずれも平均死亡年齢が五〇歳にみたない時代、ときによっては二〇歳、三〇歳という時代においての、この年齢であるから、どれほどの長寿か想像できると思う。

　このなかで、日常の食べ物について記録が残っている人物で、ひとつの〝哲学〟をもって食べ物に対している人を選んで述べていく。


　若いとき、艱難にあって精神を鍛えてきた海舟


　江戸から明治へと移る、日本の革命時代に活躍した人物・勝海舟、この人の食べ物への対し方は、その性格同様、豪快である。海舟の談話を編集した『氷川清話』を参考にして紹介していこう。

　まず七一歳のとき、長生きの秘訣はと聞かれて、

「若い頃からいろいろな艱難にあって、精神を鍛えてきたことだ」

　と答えている。

　その艱難のはじまりは、家柄は悪くないのに、父親の小吉が遊び人のため、貧乏だったことである。正月の餅を買う金さえなかった。

　小吉は遊び好きだったが、悪人ではなく、海舟のこともそれなりにかわいがった。その小吉が『夢酔独言』という自伝のなかで海舟のことをこう書いている。

　良友を友として悪友と付き合わず、武芸に励み、おれには孝行してくれて、よく兄弟をあわれみ、倹素して物を使わず、粗服をもって恥じず、粗食した、という。

　粗衣・粗食というのは、以後もずっと海舟が守った規律だが、苦にしていない。出世して何でも食べられ、着られる身分になっても、これで通している。

「牛肉とか牛乳とか、スープで長生きできるものか。ただこれらの滋養物を食べると、身体が強壮になると思いこんでいるから、その思いこんでいることで、強壮になるだけのことだ。身体の養生などべつにむつかしいものがあるものか。すべて精神の作用によるものさ」

「なにごとも根気がもとだ。いまの人は牛肉の、滋養物の、と騒ぐくせに、根気がないのはおかしなことである。人間の身体は憲法政治ではいけないよ。やれ睡眠が何時間で、働く時間が何時間、やれ食物は朝は何を食って、昼は何、などという面倒くさいことは、わずらわしいだけで意味がない。法律ずくめでは身体も困るわい。人間は気を養うことが第一だ。気さえ萎えなけりゃ、何を食っても大丈夫なものだ」

　当時は洋食がはやりはじめたころで、みながそれを必要以上に珍重するものだから、こう言ったのである。

　海舟のところにはよく客が来たが、そのとき出される食事も質素なもので、一汁一菜であった。相手の身分が高かろうが低かろうが食事のメニューは変わらなかった。

　時間が食事どきでなければ、お茶が二回出て、それからお菓子、そのあとコーヒーである。これも相手によってメニューが変わるということはない。

　海舟自身は、酒は飲まず、食事は一日二食、その量も少なく、二膳と決めていた。夏には魚をぜんぜん食べなかった。

　それでいて暑さ寒さに強く、冬でも火にあたって暖をとることはないし、寝るときは晩年まで素っ裸であった。


　勝海舟が熱意をもって学んだ剣術と禅と蘭学


　こんな海舟のバックボーンとなっているのは、若いころの苦労と、剣術への精進、そして蘭学への情熱である。剣術は四歳のころからはじめ、二〇歳になると大名屋敷や旗本に教えにいく腕前になっている。

　それは壮絶な稽古の賜物である。なにしろ早朝の朝稽古はもちろんのこと、夕方、浅草から歩いて北区の王子権現まで行き、そこで瞑想したのち木刀の素振りをするという修行を毎日続けていた。帰ってくるのは朝で、それからすぐに朝稽古に出るのである。これは師範である島田虎之介の命によるものだが、一日も休んだことがなかったという。

　この修行はたいへんだったが、それだけに実りも多かった。後年、勝はこう述懐している。

「この修行の効は徳川幕府が瓦解するときに表われた。あのような艱難辛苦に耐え得て、かつひるむことがなかったのは、まったくこの修行のおかげである」

　またこの時期、寒中でも足袋をはかず、袷( あわせ ) 一枚で平気であった。寒さ、暑さでまいったことはなく、年をとっても身体がじょうぶで足も弱らず、根気も衰えることがなかったのも、このときの稽古のおかげであったとしている。

　この精進と切り離せないのが、禅の修行である。剣術の奥義をきわめるには禅学をやるようにとやはり島田師にいわれて、牛島（墨田区）の広福寺で禅学をはじめ、四年間にわたって熱心に励んでいる。

　この禅と剣術が、海舟のあのように痛快で豪胆な生き方を生んだ。海舟は後年、多くの剣客に命を狙われるが、そのように死がつねに身近にあっても泰然自若としていられ、天寿をまっとうすることができたのは、まったくこのためであったという。

　ここでも禅が出てきているが、禅での食のあり方を学んだ海舟は、やむをえず粗食になっていた当時の食に対して自信をもったであろう。以後もそれを実践し、明治時代になって急に洋食にうつつをぬかす人が出てきたのに対して、さきに述べたような批判をしているわけである。だからといって洋食を毛嫌いしたのではなく、海舟も好きであったという。しかし、なにか特別なときに食べるものであった。

　蘭学をやるようにと勧めたのも島田虎之介であるから、海舟はたいへんよい師に恵まれたことになる。これがまた海舟にあっていて、熱心に勉強した結果、出世の糸口になっている。

　このときの努力も尋常なものではない。蘭学修行時代は、勉強中、疲れるとそのまま机に伏せて寝てしまい、布団で寝ることはしなかったというほどである。

　蘭学の本を手に入れるのにも苦労している。これも人並みではない努力で乗り切っている。行きつけの本屋で見つけた日蘭辞書『ヅーフハルマ』全五八巻、これはどうしても必要な本であった。しかし六〇両という値段で、海舟にはとても手が届かない。そこを何とか借金してお金を工面して行ってみると、売れたあとであった。海舟はあきらめず、買い主のところへ行って、「あなたが寝ていて本を読まないときに来ますから、写させてくれ」と頼みこんでいる。そうして半年間、毎夜毎夜通うのである。持ち主はそんな海舟に感心して、結局、それを譲っている。


　海舟の悟りは「至誠」をもって事にあたること


　彼がどうしてこんなに努力することができたのか。それには一人の行者との出会いが大きく影響している。父親の親しい友人で、きせん院という人である。

「おれが一生のお師匠さまだ」

　と、海舟にいわしめた人物だ。流行の富くじを当てる名人で、たいへん人気があったが、ある時期からパタッと当たらなくなり、貧におちていた。

　その行者が、ある日海舟に、

「あなたはまだお若いが、なかなか根気が強くって末頼もしい。そこでわたしがひとことお話ししておきますから、ぜひ覚えておいてください」

　といって、富くじが当たらなくなった理由を話してくれた。それは心にやましいこと、気にかかることができたためだという。

「自分の心にとがめるところがあれば、いつとはなく気が萎えてくる。そうすると鬼神とともに働くところの至誠が乏しくなってくるのです。そこで人間は、平生踏むところの筋道というのが大切ですよ」

　そう教えた。

　海舟はそれを聞いて、悟った。すなわち、生きる方針をつかんだ。それは「至誠」をもってことにあたる、ということである。以後、ずっとそれを守り続けたので、人の道を踏みはずすことは一度もなかった。

　自分が年をとってもこれだけ壮健でいられるのも、心にまったくやましいことがないからである、と海舟は言い、行者の忠言に感謝している。

　海舟は部下をもたず、どんなに大きい仕事でも一人でやった。外交交渉も、完全な全権委任（全部、海舟に任せるということ）をもらい、一人で交渉して決めているが、それらがすべてうまくいっている。開国したばかりで世間知らずの日本が、アメリカや西欧諸国の植民地にならないですんだのは、海舟の手腕があってこそだといわれている。

　どうしてそんなことが、たった一人でできたのか、なぜいつでも孤独でやれたのか――。

「味方などだれもいなくてもいい、味方というもの非常にやりにくいものだ。味方は必要ない」

　と、海舟はいっていた。

　誰にも、何にも依存しない、自立した人間であった。その根底にあるのが、「誠を尽くすこと」で、結局、それがことを成就させる秘訣となっている。

　また海舟は、決して弁解や言い訳をしなかった。「だって･･････」ということはいっさいいわない。弁解は、自分を正当化し、自分を認めてもらう、あるいは責任逃れをすることである。依存心が強い人ほど、自分を守りたい気持ちも強いから、弁解する。

　しかし海舟は、人が自分のことを何といおうが、どう思われようが気にしていない。人に愛されたい、好かれたいなどと考えないのである。一本、筋を通して生き、その生き方に自信をもっているから、他人の思惑など気にならない。同時に、自分の損得などという狭い範囲でものを考えず、日本や世界という視野に立って判断している。

　こうしたことが、結局、健康につながっている。粗衣・粗食で過ごしたことも、この精神が根底にあってのことだろう。

　海舟の生涯は波乱に満ちており、たいへん痛快なものであるが、その理由はこんなところにもうかがえる。


「自分のことは語らない」親鸞


　こんな勝海舟と、生き方の上でよく似ているのが、親鸞である。九〇歳という、当時にあっては奇蹟的な長寿を貫いた人物で、浄土真宗の開祖である。

　自分について語らない、という潔い人だったので、生涯の正確な記録は残っていない。弟子が残した記録や、親鸞の著書である『教行信証』などから推察していくしかない。そのようにして、現代に至るまで親鸞の研究が行われているのは、それだけ魅力的な人物だということである。

　九歳で出家し、比叡山で修行するが、悟りを得られず、二九歳で法然( ほうねん ) の弟子となる。しかし法然の説く専修念仏は迫害され、親鸞も寒さの厳しい越後に流されてしまう。そこでは衣食住は保障されず、すべて自分で調達して生きていかねばならない。そのサバイバル能力がなければ野垂れ死にするしかない。

　それ以後も親鸞の生活は、弾圧と貧困にさらされているが、それに負けることなく、かえってそれをバネにして、生きている。

　ぜいたくをしたくてもできない生活であるし、親鸞がそんなものを望んだとは考えられないから、粗衣・粗食で通した生涯であったろう。

　強い人であったが、これは努力してそうなったので、はじめから強かったわけではない。何があろうと切り抜けてやろうという意志をもって、仏とともに歩むことで培ったものである。

　親鸞がとくに精力的に活躍したのは八三歳から八五歳までの三年間である。この間に主要な著作が書かれ、また、宗教家として出した手紙も多い。現在でもこの年齢で創造約な仕事をしている人はわずかであるから、驚異的なことである。しかし、人間は本来、こういう力を内に秘めていて、引き出すことなく生涯を終えてしまうのではないかと考えることもできる。それらを引き出すものが、粗衣・粗食であり、誠実に自分の天職をやり続けていくことではないのだろうか。

　長寿や健康を求め、そのために運動や食事に気をつける、というのは実は逆で、それらは天職をやり続けていくなかで与えられるのではないか。

　勝海舟や親鸞の生涯をみていると、そんなふうに感じられる。


　杉田玄白の養生七ヵ条


　蘭方医で、日本ではじめて西洋解剖学の本『解体新書』を訳した人物・杉田玄白も、食について一家言あった。

　虚弱体質であったにもかかわらず、八五歳まで生き、死ぬまで現役で仕事をしていたが、その秘訣はみずからに課した七項目の養生法にあった。このなかに食に関する項目が二つ含まれている。

　まず、七項目をあげる。


　一、昨日の非を悔恨するな

　一、明日のことを思いわずらうな

　一、飲と食とは度を過ごしてはいけない

　一、正物にあらざればいやしくも食すべからず

　一、事なきときは薬を服すべからず

　一、壮実をたのんで房を過ごすべからず

　一、動作をつとめて安を好むべからず


　心の持ち方・食のとり方・日常の行動のしかたをいっているもので、現在にも十分通用する。今も昔も養生法の基本はそれほど変わらないことがわかる。

　食についていわれているのは三番目と四番目で、飲食は度を過ごしてはいけないというのは、すなわち粗食・少食ということである。次の、正物にあらざれば、というのは、季節はずれのものは食べるなということだ。

　注目すべきは五番目の、薬についての注意である。事なきときは、というのであるから、病気でないときは薬を飲むなということである。この薬というのはおそらくいまでいうビタミン剤、栄養剤、強壮剤のたぐいであろう。また、寝ていれば治る程度の軽い病気のときは飲まなくていい、ということも含まれているだろう。

　そんなときは薬に頼らず、他の六項目を守ることで健康を保つようにというのが、開業医としても活躍していた医師・玄白の意見である。わざわざ項目に入れているのは、それだけ薬に頼る人が多かったのではないかと推測できる。

　体の弱かった玄白も、薬に頼りがちになる自分を抑えて、

　・気持ちを常に前向きにもち、

　・粗食を守り、

　・性欲をコントロールし、

　・楽をせずに身体をよく動かす、

　ことを実践して、蘭学の集大成に精進することで、長寿をまっとうした。


「食は飢えない程度で十分」といった千利休


　千利休が死んだのは七〇歳である。当時にしては長寿であっても、他の長命者のなかでみるといささか若すぎる気がしないでもない。これは秀吉の命である自刃による死だからである。そうでなければ八〇歳、九〇歳まで生きただろうといわれている。

　というのも、利休が茶人として認められたのが織田信長の茶頭となった五〇歳のとき、そしてその地位を確立したのが、豊臣秀吉の茶会の後見をつとめて「天下一の茶匠」という評判を得た六四歳のときだからである。以後も活動はますます精力的になり、死の前年には「利休百会記」という茶会を主催し始めている。

　この茶会は毎月行なうもので、はじめた月の八月には三回、九月には一四回も行なわれている。時によっては一日に四回行なうこともあった。〝一期一会〟といわれるように、茶会は露地入りから退出まで、生涯にただ一度だけのこと、という心で臨む。それだけに気力を充実してかからねばならない。それを六九歳で、一日四回こなしたのだから、気力・体力ともに強くなければできるものではない。自刃していなければ八〇、九〇歳までといわれるゆえんである。その体力と気力を培う一助となったのが粗食である。

　利休は、家は雨がもらなければよいし、食事は飢えない程度で十分である、という。住居の立派さや食事の珍味を楽しみとするのはくだらないというのだ。

　この精神は、利休が主催した茶会での懐石料理によく表われている。一汁二菜、多くて一汁三菜で、素材にあまり手を加えないで、その持ち味を引き出す調理法が尊ばれた。まさしく粗食である。どんなに身分の高い客人が来ようと、いわゆるごちそうを出すことはなかった。

　それまでの茶会で出されていたのは本膳料理といって、豪華で品数も多いぜいたくなものであった。それを利休の師が一汁三菜の腹八分とし、利休はさらにそれを質素にして、飢えなければよいとした。

　利休もまた禅の修行をしており、それによって「茶禅一致」を感得していた。日々の暮らしもこの心がけにのっとったもので、酒と好色を慎しみ、慢心と我執を克服せよと弟子たちに説いている。食物としてはとくに大豆食品を好んだという。

　茶の湯でたてる抹茶も、利休の健康に貢献している。心を落ち着かせるし、ビタミンＣが大量に含まれているので、体の免疫力を高めるからだ。お茶の世界で生きている人に健康で長寿な人が多いのは、抹茶だけでなく、煎茶や玉露などのお茶をたくさん飲むからだ、というのは定説である。


　栄養は錠剤より食物から


　ところで最近、健康ブームで、錠剤で栄養をとることが流行している。カルシウムやビタミンＥやＢ、あるいは何かのエキスなどの錠剤を飲むのである。ビタミンＣもそんななかのひとつで、いくらとっても一定量以上は排泄されるので何の害もないといわれている。

　しかし、これまでの長命者でそんなものに頼っている人はいないし、なんとなく（直感的に）不自然な感じがする。杉田玄白も、七カ条の養生法のなかにわざわざ入れているくらいである。やはり栄養にしろビタミンにしろ、食物からとるのが正統なのではないだろうか。

　たとえば、カルシウムについてはこういう話がある。これは西勝造氏が『長命の生理』というエッセイ集のなかで述べていることだ。

　カルシウムの研究で有名で、とにかくカルシウム万能を唱える阪大の医学博士がいた。自分の患者にはまずカルシウムを与える。自身は、自分で開発したカルシウム剤を毎日服用するという具合である。確かに病人の身体を検査してみると、カルシウムが不足しているのは事実である。

　しかしだからといって、カルシウム剤ばかり飲んでいると、肺結核になるのだという。自分の肺を珪肺に導き、風邪でもひくとぽっくりいってしまう。その博士も風邪をひいて寝こむと、そのまま亡くなってしまった。西氏は人を介して忠告したが、間に合わなかったという。

　西氏は、カルシウムをはじめとする無機質は、それが含有されている食物からとるべきで、無機質そのものから作ったものと、食物に含まれているものとは根本的に異なっていることを知るべきである、といっている。

　もうひとつ、似たような話として氏があげているのは、呼吸器系疾患を治すために与えられていた肝油で、胃ガンになった例である。

　京都の柊家という旅館の、一五歳になる令嬢が、危篤状態だから、すぐ来てくれという依頼があった。頼んできた人が西氏の断れない相手であったため、何とか時間をつくって出かけた。病人を見ると、なんとなく眼の玉が外側に寄っている。これはガン系統の病気だとすぐわかった。そこで治療法を指示してから部屋に引きあげた。

　そうすると、しばらくして三人の主治医が西氏の部屋までやってきて、西氏の指示した治療法などとんでもないことだという。そして胃ガンだという西氏の判断にも、断固違うと言い張るのである。

「あのお嬢さんは肋膜で、一二歳のときから三年間、わたしが診てきたのです。そして肋膜は完全に治ったのですが、今度は食欲不振からこんな病気になられたのです」

「あなたが呼吸器のほうを治したからガンになったのですよ。いったいあなたは肝油を一日にどれくらい飲ませたのですか」

「一日三〇グラムです」

「そんなに肝油を与えると、ビタミンＡの過剰からガンができるのです」

　結局、医者たちは西氏の意見を受け入れず、この少女は亡くなった。たいへん残念なことであったと、氏は嘆いている。


「断食という食べ方」を選んだ中山みき


　九〇歳まで生きた中山みきは、天理教の開祖である。この人の食べ方の特徴は〝断食〟、すなわち食べないことにある。神がかりになってからそれは始まり、その後、教団が何か大きな節目にぶつかるたびに断食している。その日数は長いときで三八日間に及んだ。

　みきが教祖になった出発点は、長男・秀司の足の病気だった。本人は立っていられないくらい辛いのに、医者にみせても原因がわからない。したがって治療の手立ても立てられない。そこで最後の手段として、加持祈禱を頼む。そのとき霊媒として身体を貸したみきに神が降りて、天理教が始まるのである。江戸時代、天保九年のことである。

　中山家は土地の庄屋で、大富豪である。ところがみきにかかった神は「貧に落ちきれ」という。中山家の蔵にある財産をどんどん人に与えよといい、豪壮な屋敷まで解体せよというのだ。家族のものがこの命に従わないとみきの身体に異変が起こる。食をとらなくなるのである。それも二、三日というのではなく、家族たちがその命に従うまで、二〇日でも一カ月でも断食が続く。

　身体はやせ衰え、声が出なくなり、目も見えず、耳も聞こえない。いまにも死ぬのではないかという状態になる。こうなると家族も放ってはおけず、命令どおり、たとえば家を解体しはじめると、みきの病気はケロリと治る。そんなことの繰り返しで、先祖代々守ってきた中山家の財産はあらかたなくなり、家族が食べていくことすら困難になる。しかしみき自身には何の迷いもない。

　慶応二年には、一カ月の断食をしている。そのころは信者も増え、遠くから助けを求めて多くの人がやってくるようになっていた。そのなかに、眼病を患った助造という男がいた。病気が治ってからもしばらく熱心に通ってきていたが、やがて来なくなったと思ったら、自分のほうが天理教の本部だと言い始めた。新しい信者のなかにはそれを信じて、助造のほうにお参りする人も出てくる。みきはこのことを知ったとたん、食事をとらなくなった。

「水さえ飲んでいれば、やせもせぬ、弱りもせぬ」

　といって、食べようとしない。

　周りの人が心配して、何度も何かを食べてくれるよう懇願すると、ほんの少しのみりんと野菜を口にしたものの、穀物はまったく食べない。この断食は一カ月続いたが、このときはみきの身体は少しも衰弱せず、いつもどおりに生活していた。

　そうして断食行があけると、三、四人の信者とともに助造のもとに赴いた。助造は自分の非を認めて謝罪、一件落着となった。


　心をこめて作った食しか受けつけない中山みきの身体


　大本教同様、天理教も警察の弾圧を受けており、みきもしばしば拘留されている。この拘留期間中も、みきは断食をしている。そればかりか、水すら飲まない。警察から支給されるものは、いっさい口にしないのである。

　明治一五年一〇月に奈良警察に連行されたときは、一〇日間の拘留期間中、断食・断水ですごし、しかも夜になっても横にならず、終日端座して過ごした。みきは八五歳である。

　明治一九年、なんと八九歳のときにも拘留されている。二月一八日から一二日間というから、厳寒のさなかである。拘留されたのは古い商家を改造して警察分署とした粗末な建物なので、とにかく寒い。土間にじかに板を敷いただけの床からは、地面から直接寒さが伝わってくるし、窓はガラスのはまっていない格子なので、吹きっさらしである。布団はもちろんのこと、座布団一枚もない。

　みきはここで、夜中の二時まで取り調べを受けた。この間、ずっと断食である。孫のひさが付き添いとしてそばにいた。ひさには食事は支給されないので、信者から弁当が届けられる。それを、みきに出される食事と取り替えようとするのだが、見張りの警官が四六時中ついて監視の目を光らせているので、できない。湯は、ひさが自分でわかしたのを持ってきていて、それだけは飲むのが許された。

　みきは心をこめて作ったものしか食べないのである。それはどこにいようと同じで、そんな食物が手に入らないときは、食べないでいることになる。

　しかし、これはもしかしたら、「食べない」のではなく、「食べられない」のかもしれない。神を身体に降ろす人であるから、自分をそれにふさわしい状態にいつも保っておかなければならない。「神のやしろ」としてふさわしくない身体と心の状態であれば、神は降りてこられないだろう。いくら神のほうで助けてやりたいと思ったところで、受けるほうの人間が神を受け入れない状態を作っているなら、どうしようもない。みきの役割は、神の言葉を人々に伝えることにあったから、自分をそれだけ純粋に保つ必要があった。

　また、長期間の断食をしても死ぬことも衰弱することもなかったのは、神をわが身に降ろすことで、神から何らかの生命力を与えられていたのではないかと考えられる。

　みきが亡くなったのは翌年の二月二八日であるが、この間、拘留が四回、行としての断食はたびたび行ない、いちばん長いもので三八日間、そしてまた七五日間に及ぶ穀物断ちをしている。家族が迷いをもっていた頃は、みきの身体に異変が起こっているが、それ以後はずっと、断食中でもはつらつとして行動している。これは注目すべきことである。開祖となるだけの魂が、食べないで生き続けること、これこそ「究極の粗食」ではないだろうか。

　　　

　人間は〝気〟でもつもの、ストレスをうまく発散させる生活を


　江戸幕府の最後の将軍・徳川慶喜に仕え、明治になってからは大蔵省や銀行で活躍、民間企業の創立を指揮して日本経済の基礎を築いたのは、渋沢栄一である。「質素倹約」と「生涯現役・隠居反対」をモットーとして、九二歳まで実際に現役で活動した人物だ。

　生家は富裕な大農家であったが、父親が質素を信条としており、それを彼にたたきこんだ。その食事は、ご飯に豆腐のみそ汁と漬物程度、肉はめったに食べなかった。肉を食べないのは生涯を通じてのことで、付き合いで口にすることはあっても、自分から積極的に食べることはなかった。

　食事は一日に二食、昼食は食べず、間食もしなかった。二五、六歳の頃、毎晩のように諸藩との宴会外交が続いたが、「酒はもとより飲まず、婦人にもいっさい接さぬ」という堅固な態度で臨んだ。

　タフな人で、三日三晩、完全徹夜で仕事をしても平気であったし、一時期は五〇〇社以上の会社の責任者として八面六臂( ろっぴ ) の働きをしていた。どの会社もうまく経営できていたという。自分で常々いっていた健康法は、こうだ。


　一体人間は〝気〟で持つもので、気すなわち精神の作用如何により身体はある程度まで左右されるものだ。（渋沢著『青淵百話』）


　困難なことにぶつかって食欲がなくなったり、眠れなくなったりして衰弱するのは「身体が精神のために衰弱した」からである。だから、常に精神さえ確固としていれば、身体のほうも自然と壮健になる、自分もそうつとめてきたので、こんなに元気なのだという。

　五五歳のときに、口のなかにガンができる。煙草が好きだったが、それ以来やめ、以後再発することはなかった。死因は腸閉塞であった。

　さて、篤農家・二宮尊徳も一汁一菜の人である。家族や弟子といっしょに、にぎやかに食事をするのが好きで、多いときには一〇〇人くらいで食卓を囲んだ。健康は守るものではなく作るものであり、その方法は日々の暮らしのなかにある、という考え方であった。健康を損なうものとしてもっとも警戒していたのが、ぜいたくであった。

「飯と汁、木綿の着物は身を助く　その余のものは我を責むるのみ」

　これは尊徳が、故郷に帰る弟子に対して詠んだ歌で、これを〝安全のお守り〟にするようにとのことだったという。

　一休宗純、良寛、鳥羽僧正、沢庵、白隠慧鶴などは、いずれも禅の修行を積んだ僧侶である。ある時期、徹底的に厳しい修行をして、ひとつの境地に至っている。禅寺での食事を身体にしみこませているので、以後も逸脱することなく過ごし、長寿に恵まれている。もちろんその要因は、食だけではないだろうが。

　芸術家・画家に長寿が多いのは、ストレスを仕事に発散していたからであろう。寝食を忘れて打ちこむ、とよくいわれるように、ひとつの道に精進する人は、本能的に節制している。飲みたいだけ飲み、食べたいだけ食べているようでは、本当にいい仕事はできないからである。

　食がすべてとはいえないが、いい仕事をすることと食との関係は、かなり深いようである。


〝食〟をコントロールするのは成長のためのひとつの行


　以上、「食」というものを、人間の成長、人格の完成、さらに人生での成功（霊的な意味でも物質的な意味でも）ということを軸にみてきた。

　すでに読者は多くのことを感じられたことと思うが、ここでひとつだけ付け加えておきたいのは、食にこだわりをもたない人で、霊的にかなり高いレベルにいる人もいるということである。

　こういう人はもちろん、食べすぎることも飲みすぎることもない。食べるものがなければ断食しても平気である。そしてことさらに肉を忌みきらうこともないし、菜食にも添加物にも頓着しない。皆で集まってのティータイムには、甘い菓子も食べるという、見たところ普通の人である。

　しかし、その霊的成長の度合はまったく群を抜いており、今生での肉体を捨てたら（つまり死んでしまったら）次の段階へと生まれ変わるか、あるいは神や高次元存在の手助けをして、われわれ人間を導く仕事をするという人である。

　つまり「食」による修行というのは、それが必要な段階にいる人のためのひとつの修行法なのである。絶対的なものではないし、すでにそんな修行を必要としていない人もいる。

　が、肉体という、食によって養わなければならないものをもって生まれてきた以上、まずやるべき修行が食のコントロールだということもできる。それがいまの自分に与えられている修行であり、越えなくてはならないハードルであるということだ。

　しかしそのハードルを越えたら、また次なるハードルがあり、私たちの魂はそうやってひとつひとつのハードルを、自分の力で越えながら先に進んでいく。

　そういった大きな視野から食をとらえ、食のコントロールを少しずつでもうまくできるようになるよう、努めていけばいいのではないかと思う。そしてその出来具合によって、自分のいまいる位置を認識すればいいのではないだろうか。

　忘れてはならないのは、食以外にも、やらなければならないことはたくさんあり、それらが同時進行で発展していくということである。これは、本書でご紹介した人たちの人生をみれば、一目瞭然のことであろう。

　　　西田みどり著

　世界に大きな影響を及ぼした歴史上の偉人は、どんなものを食べていたのか？　その食事内容を調査研究したレアな労作。菜食を実行している人たちのなかには偉大な仕事を成した人が多く、多分に精神的傾向を持つ。マイケル・ジャクソン、ジョン・レノンもベジタリアンだった。ピタゴラス、プラトン、レオナルド・ダ・ヴィンチ、ヘシオドス、シェイクスピア、ニュートン、ダーウィン、ヴォルテール、ミルトン、ルソー。さらにソロー、エマーソン、オルコット、ブラバツキー夫人、アニー・ペザント、バーナード・ショー、ベンジャミン・フランクリン、トマス・モア、ガンジー、トルストイ、クリシュナムルティなどについて、菜食に焦点を当て、その真相を解明。意外にも菜食者であったヒトラーの食事メニュー、さらに魔界の入り口に入りかけていたヒトラーを正そうと説得した覚者（偉大なる聖者）がいたという知られざる事実も明らかに。その覚者とは何者か？　「食」のテーマで精神世界、神智学にまで迫ったユニークで奥が深い一冊。知玄舎電子書籍で発売。


◎異界見聞録シリーズ、第３弾内容紹介

［異界見聞録３］バリの呪術師――イワヤン・シュリアナに会う

　　　西田みどり著

　世界のシステムは目に見えるものだけではありません。未だ科学や医学では解明できない、目に見えない別次元のシステムが背後で蠢いて私たちの世界を動かしています。得体の知れない不思議や謎、奇跡的な現象がなぜ起こるのか？　『マトリックス』や『ハリーポッター』の異界は、映画や小説のなかだけでなく、いまここに私たちが接している現実に重なって存在しています。本シリーズは、その異界のシステムを知り現実に奇跡を起こす存在や不可思議な現象に迫り、異界の謎と秘密を解き明かします。深層世界、精霊や祖霊、生死の臨界、神域や魔界、メタフィジカルなエネルギーフィールドとは何か？　今回の異界の取材地は、今なお病気平癒や願望成就の祈禱などが行われているインドネシア・バリ島。バリには「バリアン」と呼ばれる呪術師が存在します。本書は、バリアンの役割とその正体に踏み込んだ貴重なルポルタージュです。知玄舎電子書籍で発売！
