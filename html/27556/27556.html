 <一場>  

    

 人里離れた山奥に建つ古城。日本だというのに、ここだけ突然ルーマニア。舞台は古城のサンルーム。客席側は全面、窓とする。  

 存在感のある階段がある。バラが生けられた花瓶、磁器人形やスタンド照明などの欧風アンティーク雑貨が、小テーブルや飾り棚に置かれている。カウチ、椅子が数脚。  

 上手は玄関ホールへ、下手は台所へ、階段は寝室へと続く。

 瞬間、鮮烈な光が一騎を捉える。  

    

 峠・萩子「サー！」  

    

 一騎、両腕を挙げ、降り注ぐ光を抱え込む。  

 ～暗転～  

    


 <エピローグ>  

    

 萩子がクリスマスツリーを片付けている。そばで見ている峠。  

 想平、遊子登場。  

    

 遊子「なあに峠 、 惚ほう けた顔して。萩子も」  

 想平「帰る準備はできているのか」  

 峠「うん」  

 萩子「できてる」  

 遊子「汐子さんは？」  

    

 汐子登場。  

    

 汐子「眠ーい。五時間しか寝ていない。下山は午後にしようよ」  

 遊子「おばあ様がお待ちかねよ」  

 汐子「せめてブランチとってから。樹助さんのサンドイッチ、もう一回食べたい」  

 想平「こちらのブランチは二十三時。今は十一時。吸血鬼の皆さんはおやすみだ」  

 汐子「そうだった」  

 想平「まずいな、ご挨拶ができない」  

 遊子「昨夜済ませておくんだった。頭がまるっきりまわらなくなってしまって」  

 汐子「仕方ないわよ、あの状況じゃあ」  

    

 空彦、豆子登場。  

    

 豆子「あー、くたびれた。ヘロヘロ。昼間に飛ぶのは疲れるのよね」  

 空彦「素晴らしい青空だよ。見てごらんよ」  

 遊子「お散歩？　ほらほら峠。あなた達もお友達を見習ってシャキッとなさい」  

 豆子「お散歩じゃないわ。豆のコウモリレーダーにひっかかったからちょっとね。萩子、ご挨拶してきたわよ」  

 萩子「え？」  

 空彦「登ってきてるよ。おばあちゃんが一人とおじさんおばさんがいっぱいで、ティーンエイジャーと子供がもっといっぱい」  

 峠 「 ... ... 」  

 遊子「もし や ... ... 」  

 想平「おいおいまさ か ... ... 」  

 萩子「大おばあちゃまが？」  

 汐子「一族郎党大集合ってわけね」  

 想平「早くも連絡網が回ったのか。抜け目のない奴らだ」  

 汐子「お葬式で盛り上がるつもりだったのが水注されたんでこっちで取り戻そうとしてんのよ」  

 遊子「まあっ、ガッカリさせてしまったわね」  

 想平「そんなとこで恐縮してどうすんだ」  

 遊子「とにかくお迎えに」  

 想平「おお、そうだ」  

    

 想平、遊子、汐子、あたふたと退場。  

    

 峠「サーのことは？」  

 豆子「ご存じだったわよ」  

 空彦「おばあちゃん、一騎もう居ないよって言ったら、知ってるって」  

 萩子 「 ... ... 」  

 豆子「知ってるわ、ありがとう。そうおっしゃって穏やかに笑っていらした」  

 萩子「そ う ... ... 」  

 峠「今夜はここで過ごしてもらおう」  

 豆子「やった！」  

 萩子「ええ。お話することがいっぱいある」  

 峠「話してほしいことも」  

 豆子「豆はあれが訊きたい。一騎のラブレターにお返事書いたきっかけ」  

 空彦「バカだな豆子。そんなの最初っから書きたかったに決まってるじゃないか」  

 豆子「えっ、そうなの？」  

 空彦「最初っからサーが好きだったんだ。僕にはわかる。昔の人は慎み深いのさ」  

 豆子「一年以上も引っ張ったの？　面倒臭いのねえ」  

 萩子「私もそう思う。大おばあちゃまは最初からサーが好きだったんだと思う」  

    

 一騎、階段上に登場。（子供達の会話に台詞を挟み込んでいく。）  

    

 萩子「だからね、私は大おばあちゃまにお願いしてその最初のお手紙を読ませていただくの」  

 一騎「拝啓。僕は吸血鬼です」  

 峠「まだ持ってるかなあ」  

 一騎「名前はヘンリー・ド・ローズ」  

 萩子「当たり前じゃない」  

 一騎「その昔、両親、姉と共に三本マストの帆船に乗って海を渡ってきました」  

 萩子「サーのことだから度肝を抜くテを使ってアピールしたと想うな」  

 一騎「その夜は奇しくも七夕で」  

 空彦「どんな？」  

 一騎「初めて聞く美しい東洋の伝説は、外国の少年の心にまったくもって深い印象を残すことになりました」  

 峠「いきなり『吸血鬼です』なんて告白したりして」  

 空彦「まさか」  

 一騎「今は春」  

 萩子「ありうる」  

 一騎「霞の空に、星はやや輝きを欠いていますが」  

 豆子「うん、ありうるね」  

 一騎「この度地上に、美しき一等星を発見。まだあかりの残る夕方に、それはなんとも言えぬ優美な光を放っておりました」  

 豆子「真実の告白を明るさでくるん で ... ... 」  

 一騎「突然の手紙、失礼の段、お許しください」  

 萩子「真実を物語に変え て ... ... 私な ら ... ... 」  

 豆子「あ、みんなの声。近くまで来てる」  

 峠「迎えに行こう」  

 萩子「うん！」  

    

 空彦、豆子、萩子退場。  

 峠、花瓶から薔薇の枝を抜く。退場。  

    

 一騎「先にも述べたように僕は吸血鬼です。馬鹿なことを言ってとあなたがこの手紙を破いてしまうか、それともえくぼを見せて笑ってくれる か ... ... 僕はどうも後者のような気がするのですがいかがでしょうか。僕の希望的観測というやつでしょうか。いえいえ破っていただいてもいっこうにかまいません。どうせ僕はお返事をいただくまで永久に書き続ける決心なのですから。僕の永久は長いですよ。なにせ寿命は一千年もあろうかと思われます。どんと腰を据えて、のんびりあなたのお返事を待つことといたしましょう。六十歳、八十歳のあなたに手紙を書き続けているのを想像するのは楽しい。僕は断言しますが、今と変わらぬ愛を、八十歳のあなたに捧げているでしょう」  

    

 音楽徐々に台詞をかき消していく。  

    

 一騎「さて、僕は英国に生まれ、年少時をかの地で過ごしましたが、祖父はルーマニア人で、遠くは十五世紀、侵入してきたオスマン帝国軍と戦った将軍として勇名を馳せ、常用した赤い甲冑と薔薇の紋章からレッドローズと人々に謡われた騎士でありま す ... ... 」  

    

 音楽大きく盛り上がり、一騎の笑顔を残しつつ照明ゆっくりおちていく。  

    

 ～暗転～
