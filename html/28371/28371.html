問題定義

3 0 代前半、見た目は中の上、崖っぷちＯＬの結婚

今この本を読んでいる貴女は、きっと結婚に関心があるのだろう。

ある程度、自分の容姿に自信はあり、それなりの社会経験も積み、いわゆる「大人のいい女」の部類に入ると自負している事だろう。

恐らくそれは間違っていない。

しかし、それだけでは幸せな結婚はできない。

なぜそんなことが言えるか？

何を隠そう、私自身がそれを体感した張本人だからだ。

 

3 0 代前半、見た目は中の上、ある程度は今風の服装をし、会社と一人暮らしのマンションを毎日往復する、崖っぷちのただ の O L であった。まさに貴女と同じような立場である。

 

そんな日々の中で、ある日、私を婚活に駆り立てる出来事が起こった。

 

その出来事とは、失恋である。

気が向いた時だけ連絡してくる彼氏（今思い返すと完全にセフレ扱いされていた）に結婚の話を持ちかけたら、一目散に逃げられてしまった。

私はしばらくその男への怒りの念に苦しめられた。

 

「都合のいい時だけ連絡してきて、私から連絡をしたら返信もあったりなかったり。

デート代もほぼほぼ割り勘（なんなら私の方が多めに出している）。

これってよく言う、都合のいい女ではないのか？！

女として生まれた以上、まず愛する男性に愛され大切にされ、その人に尽くし、幸せな家庭を築きたい。

このまま都合のいい女として一生を終えるのはあまりにみじめである。

女として悔しくないのか？！」

 

様々な思いが錯綜したのち、私は婚活することを決意した。

 

毎週末婚活に挑み、真剣に男性と向き合った。それは、人生と向き合う闘いであった。

1 日にお見合 い3 件という日や、午前中にお見合いパーティに参加し午後 は2 件お見合いというハードな日もあった。

一方で交際成立しながらも、別の案件（お見合い）も予定に入れるという日々を暫く送った。

それはまさに戦場である。

 

3 0 代前半、見た目は中の上、崖っぷ ち O L は、半年で身 長 18 2 センチ、細身、年 収 65 0 万円、優しくて私の事を一番に考えてくれる「普通の男」と結婚した。

 

現在は、優しい夫と幸せな新婚生活を送っている。


 

本当に幸せな結婚は貴女の人生を最高にする。その逆もある。

結婚をするかしないかは自由である。

しかし、結婚をすれば世間から見た貴女へ見方が変わるのは良くも悪くも事実である。

 

打算的ではあるが、社会的信用度が上がる。

女として、「イタい人」には見られにくくなる。

 

色んな人に影響を与え、与えられる。

 

本当に貴女を大事にする男性と出会えば、最高の人生を送ることができる。

心優しいあなたは、優しく誠実な彼を命を張ってでも支えようと心の中で密かに誓う。

 

それは本当に幸せな結婚と言えるだろう。

 

人生に一度くらいは経験しておいていいだろう。

 

しかし、間違った心構えで過ごしている間にくだらない男に引っ掛かり、それと結婚してしまったのでは、本末転倒である。

不幸な結婚は、貴女の人生を最悪にする。

不幸な結婚は、女性から男性にアプローチして始まる男性優位の結婚に多い（男性からのアプローチなしに成り立つ結婚自体がそもそもアウトだが）。

その場合、夫にぞんざいに扱われて地獄の結婚生活を送ることになるか、男が浮気をして結局離婚し（しかも子持ちの状態などで）家計が火の車のシングルマザー（養育費の仕送りなしなど）になるのが関の山である。


 

貴女にとって結婚を決断しない男は何の価値もない

「結婚したいけど、彼氏がなかなか話を切り出してくれない」

「友達や家族に、そろそろ私を紹介してくれてもいい頃なのに」

などとヤキモキしている女も中にはいるだろう。

 

結論から言おう。

「そんな男とはとっとと縁を切り、次へ行け。」

 

本気で貴女に惚れている男であれば、貴女が次に行こうとしたら慌てて結婚を決断するはずだ。

 

まず確認しておくことがある。

貴女が今、結婚したいのに、していないのであれば結婚への意識が根本的に間違っている可能性がある。

 

そう。結婚したいなら、結婚するための心構えが必要なのである。

 

その前に、貴女が現状において幸せな結婚が出来る女かどうかの簡単なチェックを行う。


 

幸せな結婚ができる女？できない女？チェックリスト

 

次の項目に貴女はいくつ当てはまるだろうか。

 

・なぜかドＳな男に惹かれる。

・自分を好きになる人は面白味がないと感じることが多い

・不倫をしたことがある

・母親の意見が気になる、または母親に反対されるとそれに従う

・本音を言うと嫌われそうで怖い

・ぶりっこや女の武器（愛嬌を振りまく等）を使うことに抵抗がある

・付き合っている男に冷たくされても媚びるなどして追いかけてしまう

 

どれかの項目に当てはまった貴女。

 

可哀想に。

 

これらの項目にひとつでも当てはまったら、絶対に幸せな結婚はできない。

貴女には少々荒治療が必要である。

 

この本を読み、真に幸せになる心構えと行動指針を身に付けよ。

 

この本では自称「恋愛スペシャリスト」などがホザくうわべの幸福論やくだらない恋愛一般論など微塵も論じない。

常に本質と向き合う。中には認めたくない残酷な現実もあるだろう。

しかし、それと向き合う事こそが、貴女が幸せになるための第一歩なのである。


貴女の結婚を阻む意外なもの

「母親」

母親は貴女の呪縛でしかない。

特に、夫からあまり愛されていない母親は娘が自分よりずっと幸せになることを無意識で阻止しようとする。

しかし、娘がとても不幸になることを歓迎はしない。

あまりにも不幸な娘の存在は、世間から母親としての評価を下げられるからである。

それ以外の理由はない。

 

クセモノなのは、世間と自分達が持っている固定概念である。

 

その固定概念とは、

「母親は娘へ無条件に愛を注ぐもの」

という戯言である。

 

当事者も意識レベルではその固定概念を信じて疑わない。

 

幼少期は母親に嫌われるのは死活問題である。そのため、子どもは必死に母親に好かれようとする。

 

その名残で年頃になった娘は、母親から発信される無意識の嫉妬を知らずのうちに感知し、自分自身をスゴく不幸ではないが母親よりは幸せではない範囲に温存しようとする。

 

それほどまでに子どもは母親に愛されたいのである。

 

しかし、貴女はもう子どもではない。

母親がいなくなっても生きていけるはずである。

 

母親を実際に殺さなくてもいいが、貴女の心の中で母親を殺すくらいの決意をしなければいけない。

貴女は、母親より幸せになる権利がある。

いや、母親よりずっと幸せになる義務がある。

 

今、ここに誓え。

｢私は母親よりも、いや、世界で一番幸せになる｣

さあ、誓え！

そして、結婚披露宴最後の両親への手紙では、うわべだけ母親へ感謝の言葉を吐いていればいいのである。


 

「謙遜や優しさ」

断言する。

謙遜や優しさによって自己評価の低い女はくだらない男を引き寄せる。

現状の日本社会は、何かにつけて謙遜することが美徳とされ、それが歓迎される場面が多い。

特に女性に対してのそれは顕著である。

男性中心社会の日本において、女性は謙虚であるほうが男にとって都合がいいのである。

 

奥ゆかしい日本女性のイメージを作っているのも謙遜の文化である。

しかし、それが過ぎるとただの安売り女になってしまう事を認識しなければならない。

 

貴女は、そこらへんの女よりも格段にランクが上なのである。ある程度美人で、ある程度のキャリアを積み、社会的信用を得て、自立した女性なのである。近づいてくる男は、貴女なりの基準を設けて選別しなければならない。

 

 

優しいという事も美徳とされるが、それも過ぎるとくだらない男を引き寄せる要因となる。

 

貴女は、

彼氏がデートに大幅に遅れた場合、

「今晩行く」と言って何時に行けるかわからないなどと電話で伝えられた場合、

メールなどで連絡しても何日も返事を返してこない場合、

どのような対応を取っているだろうか。

 

脳裏に膨大なわだかまりを感じながら、無駄な優しさで「彼も忙しいし、仕方ないよね」などと無理にそれを打ち消そうとしていないだろうか。

 

その行為は、ただの時間の無駄である。

 

そんな事をしているくらいなら、自分磨きに時間をかけるべきである。

貴女を一番に考えない男は、貴女にとって男ではない。

 


 

婚活女の間違い

飲み会は愚の骨頂

コンパ、出会い目的の飲み会は愚の骨頂。

まさに、石ころが圧倒的に多い玉石混合なのである。石ころならまだマシ、地雷が混じっている可能性がある。

合コンで言う地雷とは、遊びの女を見つける為に参加している既婚者（もちろん既婚である事は隠している）の事である。

 

そんな輩に引っ掛かり、体を許し、嵌まってしまおうものなら、もう目も当てられない。

嵌まった後に既婚者であることを｢もう夫婦仲は冷めきっている｣などくだらない枕詞を添えて暴露され、実現することのない男の離婚を待っていようものなら、人生における貴重な時間を醜くくだらないものにしているだけである。

そんな時間を過ごしているくらいなら、家でおとなしく爆睡している方 が 100 0 倍有意義である。

 

追う恋愛をする女 は 200 0 ％幸せになれない

追う恋愛をする女は煮えくり返るような地獄へ自ら飛び込んでいるようなものだ。

スリルでドキドキする恋愛は、一見魅力的に映るが、女を不幸にする。

愛されるより愛したい女 は 200 0 ％世間様の言う「幸せな女」にはなれない。

「恋する女はキレイになる」などの言葉を吐いた輩は少々言葉足らずである。

「幸せな恋をする女だけがキレイになる」のである。

女が我慢を強いられる不幸な恋愛や女が男を追う恋愛は、女を疲弊させ、心も身体も蝕んでいく。

 

結婚したいなら恋愛をするな

「いろいろなことがあったけれど、紆余曲折の末、結婚」

「大恋愛の末、結婚」

「偶然の出会いから、結婚」

これらのドラマチックな展開を妄想し、現状の彼氏にしがみついている女は滝行でもして神経を研ぎ澄ませて考えよ。

 

貴女にはもう時間がない。

出産を考えている場合は身体的なタイムリミットだってある。

是非そのドラマチックな幻想を頭から切り離し、効率的な出会いに目を向けてほしい。

 

結婚したいなら、恋愛をするな。


具体的には、結婚相談所に登録すべきである。

間違っても、無料の出会い系サイトに登録をしたり、街コンなどに参加してはいけない。それは婚活ではなく時間の無駄である。

 

入会には以下のような証明書提出を求められる相談所に登録すべきである。

・独身証明書（既婚でないことの証明）

・源泉徴収票、課税証明書（年収の証明、男性のみ求められることが多い）

・卒業証明書（学歴の証明）

 

そういった敷居の高い結婚相談所は、もちろん入会費も高額となる。

お金とは、信用である。入会費の支払いは本気度の証明である。

すなわち、結婚相談所に登録している人たちは結婚したい気持ちの本気度の証明として高額な入会費を払っている人たちばかりなのである。

 

この支出は一時的には痛みを伴うかもしれないが、玉石混合の飲み会に参加して地雷に引っ掛かることを考えたら、圧倒的に安いものである。

 


 

婚活市場での心構え

容姿・服装を整える

髪は基本的にロングヘアーにすること。

身長に対して体重が重すぎるなら適正体重まで絞ること。

目が一重なら二重にプチ整形も考えよう。

婚活用の服を買いそろえる。お手本は女子アナ。清楚なお嬢様風を心がけよ。

 

婚活用の服は似合っているかどうかは関係ない。

 

婚活現場は、戦場である。

 

私には清楚な格好は似合わないとか、

黒のカッコいいパンツスーツが好きだから清楚でひらひらな服を着るのは嫌とか言う女は、

滝行でもして神経を研ぎ澄ませて考えよ。

 

戦場に、似合わないとか、夏場で暑いからという理由で戦闘服ではなくタンクトップで挑む奴がいるだろうか？

 

戦場は現実とかけ離れているので職場に置き換えて考えてみよう。

 

職場に、似合わないとか、夏場で暑いからという理由でスーツではなくタンクトップで来る奴がいるだろうか？

 

そんな事をしようものなら、

｢あの人は職場にタンクトップで来て頭がおかしいのではないか？｣

｢あの人に大きなプロジェクトは絶対に任せられない｣

必ずそう思われるであろう。

一発でアウトである。

 

婚活現場も全くもって同じである。

清楚な服は、婚活での戦闘服である。

 

婚活という戦場に、黒いモード系のファッションや、パンツスーツで登場するのは、先のタンクトップの件と同じである。ずば抜けて容姿の評価が高いとかでないと、アウトである。

 

それでは没個性であるとか自身の魅力が伝わらないなど余計な事を考える必要は皆無である。

魅力や個性は｢出す｣ものではなく｢出る｣ものである。

 

特に何もしなくても貴女の魅力や個性は自然に｢出る｣ので、留意する必要はない。

 

ずるがしこい女であれ

婚活現場において貴女は男たちにとって 「 One of the m （大勢の中のひとり）」の一商品である。

年齢、身長、体重、容姿、離婚歴、家族状況、その他バックグラウンドで容赦なく品定めされる。

その逆も然りである。

貴女も容赦なく、男という商品を年収や身長、その他の要因などで品定めするのである。

女の場合はどうしても年齢や容姿が品定めの基準となる事が多い。

そこにショックを受けることなく、現実を受け止める心づもりが必要である。

なぜなら、それは男も女もお互い様なのであるから。

 

例として、婚活での年齢について一般的な話をする。

どうしても 、 3 0 代前半男性 は 2 0 代女性を対象として検索をかけてくる。

3 0 代前半男性にとって 、 3 0 代女性はその時点で対象外なのである。

3 0 代後半 、 4 0 代前半男性であれ ば 3 0 代前半女性も対象に入れる場合があるので、こちらもできる範囲で年齢については幅広くとらえる必要がある。

 

大事にされる女であることを自覚する

邪険にしてくるヤツは相手にしない。

貴女を大切にしない男は全員、あなたにとって男ではない。

 

貴女は、宝物のように大事にされなければならない。

それを自覚し、そのように振る舞わなくてはならない。

 

交際成立後、彼が貴女を一番に考えてくれるか、大事にしてくれるかを見る必要がある。

デートの待ち合わせ時間に無断で遅れないか。

連絡をしたらすぐにレスポンスがあるか。

食事では貴女の注文を気遣ってくれるか。

行動ひとつひとつを言いだすときりがないが、行動の端々に、貴女への優しさや思いやりが感じられるかということである。


 

お見合いでの立振舞い

お見合いパーティ行動指南

パーティは早め（開始の 約 1 5 分前）に行ってプロフィールカードをしっかり書く。

なぜなら、相手が貴女を知るための情報はプロフィールカードしかないからである。

ギリギリに来てカードを書く時間がなく、空欄が多い、書いていても殴り書きといった最悪の事態は避けるべきである。

プロフィールカードは相手との会話を広げるために存在するものと心得よ。

自分が得意な会話や最近行った観光名所、趣味などの話題に繋がる事柄を端々に散りばめるのだ。

プロフィールカードをしっかり書くだけで、婚活パーティでの成功度 は4 割上がる。

 

お見合いの心構え

どちらかがプロフィールを確認して申し込みをし、もう一方がそれを承認し成立するのがお見合いである。

パーティなどとはちがい、事前にお互いのプロフィールをしっかり確認するので、ある程度お互いのバックグラウンドや趣味などは認識している。

あとはホテルの喫茶店などで会って 約1 時間ほど話をし、フィーリングが合えば交際成立となる。

 

お見合いでの心構えは、プロフィールやバックグラウンドを事前にしっかり確認し、会話では自然に共通点を引き出し楽しく会話をすることである。

自分に関係がない、興味のないことを小一時間にわたって聞くのはストレス以外の何物でもない。

 

何かの縁でお見合いに至ったのであるから、結局縁がなかったとしてもこの 約1 時間だけは楽しく会話に花を咲かせようではないか。


 

交際後の心構え

男からアクションを起こさせる

デートの誘いは男からアクションを起こさせる。

 

女からデートの約束を仕掛けてはいけない。

 

また、男からの急な誘いには乗るべきではない。

急な誘いにいつでも対応していたら、それこそ安い女と認識されてしまう。

 

少なくともデート日 の3 日前までの誘いしか受けてはいけない。

 

友達・家族に紹介されるか

貴女が彼にとって大切な人であれば、絶対に自分の友達・家族に紹介するはずである。

ある程度の時間が経ってもその行動を起こさなければ、彼との関係清算を頭の片隅に置いておくべきである。

ある程度の時間とは、お見合いの場合 は3 ヶ月である。

それくらいの期間交際すれば 、6、7 回デートを重ねただろう。

その中で、お互いにフィーリングが合うかどうかは十分わかるだろう。

3 ヶ月経っても彼が結婚への具体的な行動を起こさなければ、貴女は彼との関係清算を念頭に置き、別のお見合いのアポイントをどんどん入れていくべきなのである。

 

結婚を決断させるま で SE X をしない

結婚を決断させないうちに男性 と SE X をするのは、牛丼屋が毎日タダで牛丼を食べさせるようなものである。最初は「旨い、旨い」などと言い食っているかもしれないが、そのうち「牛丼は飽きた」などと抜かしファミレスに行くかもしれない。

 

このように、結婚前 に SE X をすれば、そのままズルズルと関係が続き、そのうち飽きられ別の女に乗り換えられ、残るモノは傷ついた貴女の心と身体である。

 

結婚を決断させないうち に SE X 　≒　タダ飯である。

タダ飯が格別に美味しかったとしても、人はタダの物に価値を置かない。

 

だから、貴女との結婚を決断していない男と絶対 に SE X をしてはいけない。

腹を極限状態まで空かし、ついにありつけた牛丼は格別に美味しいということを忘れてはいけない。

また、結婚を決断する行為とは、日本文化において具体的には結納式である。

ホテルの会場でふたりと両家両親が揃い、男は女に結納品と婚約指輪、結納金などを納める。

結納金の相場は大 体 10 0 万円が一般的である。そのお金は新居の家具などのために使われる。

 

それが日本の伝統的なプロポーズである。

女の大事な人生とふたりの結婚生活がかかっているというのに、たった の 10 0 万円くらいを納められない、または納めようとしない男はただ単に貴女と結婚しようと思っていないだけである。さっさと次へ行くべし。


おわりに

貴女はもっと幸せになっていい！

いや、幸せになる義務がある！

 

なぜこんなに何回も叫ぶように訴えるのか？

この本を書いた理由は、本質的には頑張り屋で心優しいけれど、様々な要因で芯を通し切れていない「都合のいい女」にその要因を払拭し、幸せになってもらいたいからである。

 

日本には「都合のいい女」が結構な割合で存在し、

「都合のいい女」とセフレだけの関係を求めている「ヤリチン男」も結構な割合で存在する。

 

仕事帰りにでも本屋の男性雑誌コーナーを見渡してみよ。

信じられないような表紙のリード文言を頻繁に見つける。例えば、

「職業別！ヤれる女の職業ランキング」

「ヤれる女 の 1 0 つの特徴」

「ヤれる女の出没地域」

などなど、女としては怒りを通り越して悲しい。

（ちなみに、その本によれば、一番ヤれる職業はネイリストで、一番ヤれない職業は銀行員らしい。ネイリストの貴女、気をつけよ！）

 

「ヤリチン男」と「都合のいい女」の組み合わせは女にとって地獄絵図である。

女はどんどん疲弊していくのが外部からも目に見えてわかる。

 

しかし、女性として、婚活に向けて正しい心構えを認識し、行動指針を実行すれば貴女は必ず幸せな結婚が出来るはずなのだ。
