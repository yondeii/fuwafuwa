第 １ 章 ご あ い さ つ

（ １ ） ダ ウ ン ロ ー ド あ り が と う ご ざ い ま す

こ の た び は 、 数 あ る 本 よ り 本 本 を ダ ウ ン ロ ー ド し て い た だ き ま し て 、 ま こ と に あ り が と う ご ざ い ま す 。

本 本 は 、 女 性 に 限 ら ず 、 他 人 と 話 す こ と が 苦 手 な 人 が 少 し で も 、 多 く 、 長 く 会 話 が つ づ け ら れ る よ う に 基 本 的 な コ ツ み た い な も の を ま と め ま し た 。

あ な た が ど ん な に イ ケ メ ン で も 、 ど ん な に お 金 持 ち で も

や は り 、 会 話 と い う コ ミ ュ ニ ケ ー シ ョ ン が で き な け れ ば 、 そ れ ら は す べ て 、 「 宝 の 持 ち 腐 れ 」 と な っ て し ま い ま す 。

会 話 を し て い て も 、 相 手 の 反 応 が 悪 か っ た り 、 次 の 言 葉 に 詰 ま っ た り 、 盛 り 上 が れ な か っ た り と い う 、 悩 み を 少 し で も お も ち な ら 、 こ の 本 が 役 に 立 つ こ と で し ょ う 。

本 本 で は 、 聞 く 、 話 す 、 質 問 す る 、 受 け 答 え を す る 、 複 数 の 人 と 会 話 す る 、 と い う こ と 具 体 的 な 事 例 な ど を 挙 げ て 、 お 話 を し て い き ま す 。 あ な た の た め に な る こ と ば か り で す の で 、 最 後 ま で じ っ く り と お 読 み く だ さ い 。

そ し て 、 読 み 終 え ま し た ら 、 必 ず 実 行 す る よ う に し て く だ さ い 。 読 ん で イ ン プ ッ ト し た だ け で は 、 結 果 は あ ら わ れ ま せ ん 。 も ち ろ ん 、 一 発 で う ま く い く こ と は 少 な い か

も し れ ま せ ん が 、 努 力 は 裏 切 り ま せ ん の で 、 繰 り 返 し 実 行 し て く だ さ い 。

あ な た の 恋 愛 が 成 就 し ま す よ う に 。

第 ２ 章 聞 く

（ １ ） 自 分 の 話 を し て い る と き は 気 持 ち よ く 話 さ せ る

人 は 、 意 外 と 自 分 の こ と を 話 し た が る も の で す 。 ナ ン パ を さ れ て も 「 私 、 彼 氏 い る か ら 」 と い っ て 、 聞 い て も い な い こ と を 自 分 か ら 先 に 話 す 女 性 も い ま す 。

そ の く ら い 、 自 分 の こ と を 話 し た が る 生 き 物 な の で す 。 で す か ら 、 話 す 技 術 を 身 に つ け る 前 に 、 「 聞 く 」 こ と に 焦 点 を お き 、 相 手 に 気 持 ち よ く 話 を さ せ る よ う に し ま す 。

た と え ば 、 相 手 が エ グ ザ イ ル の こ と が 好 き で 、 そ の 話 題 を 話 し 始 め た と し ま す 。

た ま た ま 、 あ な た が エ グ ザ イ ル の デ ビ ュ ー 前 の こ と を 知 っ て い た 場 合 、 そ の こ と を 話 し た く て ど う し よ う も な く な り ま す が 、 こ こ は 、 グ ッ と こ ら え て 、 相 手 の 話 に 耳 を 傾 け ま す 。

も し 、 あ な た が 話 し 始 め て し ま う と 、 会 話 の 主 導 権 が あ な た に 移 っ て し ま い 、 話 題 が 途 切 れ て し ま い ま す 。 相 手 に 不 愉 快 な 思 い を 与 え る と 同 時 に 、 自 分 の 話 が で き な か っ た 、 す な わ ち 、 受 け 入 れ て も ら え な か っ た と 感 じ さ せ て し ま い 、 あ な た の 前 か ら 立 ち 去 ろ う と し て し ま い ま す 。

で す か ら 、 相 手 が 話 し て い る と き は 最 後 ま で 聞 く よ う に し ま す 。

ま た 、 「 会 社 辞 め た い 」 と 相 談 を う け る と 、 つ い 、 「 辞 め た 後 は ど う す る の ？ 」

と か 、 「 真 剣 に 考 え た の か ？ 」 な ど と 、 会 社 を 辞 め た い と い う 行 動 に つ い て 、 質 問 を す る 人 が 多 い で す が 、 ま ず は 、 相 手 の 気 持 ち を 聞 い て あ げ ま す 。 た と え ば 、 「 辞 め た い く ら い 、 き つ い の か ？ 」 な ど と 、 心 を 開 き や す く し て あ げ る こ と が 重 要 で す 。

心 を 開 い て も ら え ば 、 自 然 に 会 話 が 進 み ま す 。 相 手 が い ろ ん な こ と を 話 し 始 め る の で 、 あ な た は 、 う な づ き な が ら よ う く 話 を 聞 い て 上 げ る だ け で 、 好 感 度 が ア ッ プ し ま

す 。

（ ２ ） し っ か り と 反 応 し て あ げ る

相 手 の 話 を じ っ と 聞 い て あ げ れ ば い い の で は な く 、 そ こ に は 、 あ な た か ら の 無

言 の メ ッ セ ー ジ を 送 っ て あ げ ま す 。 つ ま り 、 会 話 の 相 槌 を う っ て あ げ る の で す 。

し か し 、 た だ 相 手 の 会 話 の タ イ ミ ン グ に 合 わ せ て 「 う ん 、 う ん 」 と う な ず く の で は な く 、 バ リ エ ー シ ョ ン 豊 か に 反 応 し て あ げ ま す 。

た と え ば 、

相 手 ： 「 先 週 、 い き な り ゲ リ ラ 豪 雨 に あ っ て び し ょ 濡 れ

に な っ ち ゃ っ た 」

あ な た ： 「 あ あ 、 そ れ は 、 災 難 だ っ た ね 」

相 手 ： 「 も ー 最 悪 で し た 」

あ な た ： 「 わ か る 、 わ か る 」

相 手 ： 「 わ か り ま す ？ 」

あ な た ： 「 も ち ろ ん 、 俺 も 何 回 か ゲ リ ラ 豪 雨 に あ っ て 濡 れ た こ と あ る も ん 」

と 、 相 手 の 気 持 ち を 先 回 り し て 、 こ ち ら が そ の 気 持 を 話 し て あ げ れ ば 、 相 手 に

は 話 し や す い ひ と と 認 識 し て く れ ま す 。

反 応 は バ リ エ ー シ ョ ン と 共 感 で す 。 共 感 す る と き は 、 相 手 は マ イ ナ ス の 感 情 な

の か 、 プ ラ ス の 感 情 な の か を 判 断 し ま す 。 年 数 や 回 数 の よ う な 数 字 に も 注 意 を

し ま す 。

数 字 な ど の 場 合 は 、 そ の 数 字 オ ウ ム 返 し で う な づ き ま す 。

相 手 ： 「 私 は 、 ７ 年 も こ の 店 に 通 っ て い る ん で す 」

あ な た ： 「 ほ ぉ 、 ７ 年 も で す か 」

と い う 具 合 に 、 数 字 に 反 応 し て 、 う な づ く と 、 「 聞 い て く れ て い る 」 と 思 わ れ 、 相 手 に は 、 感 じ の い い 人 と 思 わ れ ま す 。

第 ３ 章 ： は な す

（ １ ） 話 題 の ネ タ は 「 天 気 の は な し 」 か ら

日 本 は ご 存 知 の よ う に 四 季 が は っ き り と し て い ま す 。 そ し て 、 日 本 人 は 古 来 よ り 農 耕 民 族 が 多 い た め 、 天 気 に は つ ね に 気 を と ら れ て い ま し た 。 そ の た め 、 日 本 人 の あ い さ つ は 「 こ ん に ち は 、 今 日 は い い 天 気 で す ね 」 と い う 、 あ い さ つ が 交 わ さ れ る こ と が 多 い の で す 。 で す か ら 、 天 気 の 話 題 は 誰 で も 、 受 け 入 れ や す い と 言 え ま す 。 こ れ を 使 わ な い 手 は あ り ま せ ん 。

話 題 の ネ タ は 天 気 の 話 か ら 始 め ま す 。 「 今 日 は 降 水 確 率 が 1 0 ％ と 言 っ て い た の で 、 折 り た た み 傘 を も っ て き ま し た 。 あ な た は 、 何 ％ な ら 傘 を 用 意 し ま す か ？ 」

な ど と 、 す こ し ク イ ズ っ ぽ く す る の も い い で し ょ う 。

こ こ か ら 、 話 が は じ ま り 、 「 電 車 に 傘 を 忘 れ て し ま っ た 」

エ ピ ソ ー ド な ど が 聞 け た り し て 、 話 が は ず む か も し れ ま せ ん 。

（ ２ ） モ テ な い 自 己 ア ピ ー ル が モ テ る よ う に な る

女 性 と の 会 話 の 基 本 で 「 自 己 開 示 」 と い う の が あ り ま す 。 自 分 の こ と を 話 す と い う こ と な の で す 。 そ う す る こ と に よ り 、 相 手 に 自 分 が ど ん な 人 間 な の か を 教 え る こ と で 、 安 心 感 を 与 え ま す 。

ナ ン パ 師 は こ の 自 己 開 示 を う ま く 使 い な が ら 、 タ ー ゲ ッ ト の 心 の 壁 を な く し ま す 。

女 性 と 会 話 で 早 く 親 し く な る に は 、 や は り 、 自 己 開 示 が 必 要 で す 。 だ か ら と 言 っ て 、 弱 弱 し い 自 分 を 泣 き な が ら 話 し て は な り ま せ ん 。 ユ ー モ ア を 交 え な が ら 話 す こ と が

大 事 で す 。

よ く 、 女 性 と 会 話 す る と 、 仕 事 の 話 し に な る こ と が あ り ま す 。

相 手 ： 「 お 仕 事 は 何 を さ れ て い る の で す か ？ 」

あ な た ： 「 I T 関 係 の 仕 事 で す 」

相 手 ： 「 え え 、 I T 関 係 だ と 毎 日 帰 り が 遅 い の で は な い で す か ？ 」

あ な た ： 「 え え 、 も う 、 ほ と ん ど 会 社 に 泊 ま っ て ま す 。 毎 日 が ホ ー ム シ ッ ク で す 」

相 手 ： 「 は は 、 わ か り ま す 、 そ れ 」

こ の よ う に 、 " ホ ー ム シ ッ ク " と い う 言 葉 で 家 に 帰 り た い 、 つ ま り 仕 事 を 投 げ だ し た い 気 持 ち を 表 し て い ま す 。

本 来 な ら 、 「 仕 事 が 楽 し く て 」 と 言 う べ き と こ ろ な の で す が 、 あ え て 、 " 仕 事 が つ ら い " と い う こ と を 言 い ま す 。 こ こ に 、 人 間 ら し さ を 醸 し 出 し 、 「 こ の 人 な ら 、 な ん で も 話 し て も い い か な 」 と 思 わ せ ま す 。

ま た 、 本 心 を 語 っ て く れ た と い う 安 心 感 か ら 、 「 私 も 話 し て も い い か な 」 と 女 性 の 内 心 を 話 し て く れ る よ う に な り ま す 。 つ ま り 、 打 ち 解 け る た め の 第 一 段 階 に な る の で す 。

こ の 例 は 少 し 、 高 度 で す が 、 日 ご ろ か ら 、 そ の よ う な セ ン ス を 磨 い て く だ さ い 。

ほ か に は 、 「 金 が な い （ な く て ） 」 か ら ほ し い も の が 買 え な い と か 、 食 べ た い も の が 食 べ れ な い 、 女 性 受 け が い い

の は 、 旅 行 に 行 き た い け ど 、 行 け な い と い う 話 を す る と 、 会 話 が 弾 み ま す 。

た だ し 、 あ ま り に 強 調 し す ぎ な い よ う に し ま し ょ う 。 経 済 観 念 の な い 男 と 思 わ れ ま す 。 こ こ は 、 あ く ま で も 平 均 的 な 「 そ う い う 人 っ て 多 い で し ょ 」 と い う ラ イ ン で 話 を し て く だ さ い 。 で き れ ば 、 「 実 は わ た し も そ う な の で す 」 と い う 言 葉 を 引 き 出 す た め に 、 自 己 開 示 を す る と い う こ と が ポ イ ン ト で す 。

第 ４ 章 ： 質 問 す る

（ １ ） ま ず は 情 報 質 問 か ら

相 手 の こ と が ま だ 、 よ く 知 ら な い 、 あ る い は 、 い ま 話 題

に な っ て い る こ と に 知 識 な い と き は 、 情 報 に 関 す る こ と を 質 問 し ま す 。

情 報 に 関 す る 質 問 と は 、 俗 に い う 「 ５ W １ H 」 の こ と で す 。 「 い つ 」 「 ど こ で 」

「 だ れ が 」 「 何 を 」 「 ど の よ う に 」 「 ど の く ら い 」 を 会 話 の な か で 質 問 を し な が ら 会 話 を 弾 ま せ ま す 。

た だ し 、 簡 単 な 質 問 な の で 拷 問 の よ う に 質 問 攻 め に な ら な い よ う に 注 意 を し ま す 。

相 手 ： 「 先 日 、 温 泉 旅 行 に 行 き ま し た 」

あ な た ： 「 ほ ぉ ～ 、 ど ち ら に ？ 」

相 手 ： 「 長 野 県 で す 」

あ な た ： 「 ど な た と 行 か れ た の で す か ？ 」

相 手 ： 「 会 社 の 慰 安 旅 行 で す 」

あ な た ： 「 何 泊 し た の で す か ？ 」

相 手 ： 「 2 泊 で す 」

こ の よ う に 、 話 題 に な っ て い る こ と の 情 報 を 次 々 に 聞 き 出 し て い き ま す 。 た だ し 、 例 文 か ら も お 分 か り に な り ま す よ う に 、 質 問 攻 め に な っ て い る の で 、 拷 問 の よ う に 感 じ ら れ ま す 。

そ れ を 回 避 す る た め に は 、 や は り 、 感 情 を 入 れ る こ と が 重 要 に な っ て き ま す 。

相 手 ： 「 先 日 、 温 泉 旅 行 に 行 き ま し た 」

あ な た ： 「 ほ ぉ ～ 、 温 泉 で す か ？ い い で す ね 。 そ れ で ど ち ら に ？ 」

相 手 ： 「 長 野 県 で す 」

あ な た ： 「 そ う で す か 、 空 気 が き れ い な と こ ろ だ か ら 気 持 ち が 落 ち つ き ま す よ ね 。 ど な た と 行 か れ た の で す か ？ 」

相 手 ： 「 会 社 の 慰 安 旅 行 で す 」

あ な た ： 「 そ れ は 、 大 賑 わ い で す ね 。 何 泊 し た の で す か ？ 」

相 手 ： 「 2 泊 で す 」

先 ほ ど の 例 文 と 比 べ て い た だ け れ ば お 分 か り に な る と 思 い ま す 。 感 情 を 入 れ る こ と に よ っ て 、 相 手 に は 、 気 持 ち よ く し ゃ べ っ て も ら え る よ う に な り ま す 。 そ う な れ ば 、 相 手 に は 、 「 話 し や す い 人 」 と い う 印 象 が 残 り 、 ま た あ な た と 話 が し た い 、 そ の よ う な 気 持 ち に な る の で 、 好 感 度 が 上 が っ て い き ま す 。

（ ２ ） マ イ ナ ス 感 情 を 引 き 出 す

マ イ ナ ス 感 情 （ ネ ガ テ ィ ブ ） を 引 き 出 す と 相 手 は 、 ど ん ど ん 話 し 始 め ま す 。 と く に 、 相 手 が 女 性 の 場 合 は 、 悩 み や 苦 労 を し て い る こ と を 聞 き 出 す こ と に 成 功 す る と そ の 女 性 は 、 た く さ ん の 話 を し は じ め 、 あ な た と い る こ と が 楽 し い と 感 じ る よ う に な り ま す 。

引 き 出 す コ ツ は 、 相 手 が 思 っ て い る で あ ろ う 感 情 や 気 持 ち を 先 読 み し て 、 あ な た が そ れ を 言 葉 に し て 、 相 手 に 質 問 し て あ げ れ ば い い の で す 。
