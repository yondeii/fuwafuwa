血 液 型 占 い は 日 本 で は 有 名 で す ね 。

で も 、 科 学 的 に は 立 証 さ れ て い ま せ ん 。

で も 、 な ん と な く 当 て は ま っ て い る し 、 確 か に 当 た っ て い る と 思 う 時 が あ り ま す ね ？

こ れ は 、 心 理 的 法 則 な ん だ と 思 い ま す 。

つ ま り 、 周 り の 評 価 が 自 分 を 形 造 る 。

と い う 心 理 的 法 則 が あ て は ま る の で す 。

例 え ば 、 A 型 な ら ば 幼 い こ ろ か ら 周 り か ら レ ッ テ ル を 張 ら れ ま す 。

『 あ な た は A 型 だ か ら ・ ・ ○ ○ な 性 格 な の ね 』

と い う 感 じ で す 。

あ な た も 周 り の 友 人 に 言 わ れ た こ と が あ り ま せ ん か ？

『 お 前 は ● 型 だ か ら 、 ● ● ● ● ● な ん だ な 』

と 言 う 感 じ で す 。

こ れ は 心 理 学 的 に は 『 レ ッ テ ル 法 』 と 言 わ れ ま す 。

人 は 、 周 り の 意 見 で 自 分 を そ の よ う に 思 い 込 ん で い く の で す 。

（ 日 本 人 は 周 り の 意 見 に も 左 右 さ れ や す い で す よ ね ）

■ Ｏ 型 の 一 般 的 特 徴

仲 間 意 識 が 強 く 統 率 力 も あ る の で 、 リ ー ダ ー と し て の 資 質 が あ る 。

親 し み や す く 性 格 も 表 裏 が な い た め 、 人 を ひ き つ け る 魅 力 が あ る 。

自 分 が 損 し て も 相 手 の た め な ら と 、 頼 ま れ る と 何 で も 引 き 受 け る ほ ど の 社 交 性 を 持 っ て い る 。

人 間 性 豊 か で 仲 間 を 大 切 に し 、 心 と 心 の つ な が り を 求 め 、 と も に 喜 び と

も に 涙 す る 。

よ く 考 え た 上 で 決 め た こ と は 、 成 し 遂 げ る ま で 頑 張 ろ う と す る 強 い 意 志 を も っ て い る 。

負 け ず 嫌 い で ち ょ っ と の こ と で は へ こ た れ な い 。

小 さ な こ と は 気 に せ ず 、 い つ ま で も 過 去 の 失 敗 に こ だ わ っ て ク ヨ ク ヨ す る こ と は な い 。

現 実 的 で あ る と 同 時 に ロ マ ン チ ス ト で も あ る 。

ど ち ら か と 言 え ば 聞 き 役 よ り 話 し 手 に な る ほ う が 多 い 。

好 き 嫌 い を は っ き り 出 し 、 嫌 い な 人 に は 無 頓 着 に な る こ と が あ る の で 敵

も 多 い 。

自 分 を 隠 し て お け な い の で で 、 本 心 を す ぐ に 出 し て し ま う 。

対 人

人 と 人 の つ な が り を 大 切 に す る た め 、 友 人 が た く さ ん い る 。

初 め て 会 っ た 人 と も 気 兼 ね な く 、 リ ラ ッ ク ス し て 話 が で き る 。

（ Ｏ 型 女 は 尽 く す よ り も 、 尽 く さ せ た 方 が 幸 せ を 感 じ る タ イ プ ）
