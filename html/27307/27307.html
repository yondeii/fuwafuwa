はじめに

 

私は、これまで、若い日本語教師の悩みをたくさん聞いてきました。

中国のある大学で授業をしていたときの話です。ある教師が「生徒が授業に興味も関心も示さないんですよ！」と相談に来ました。もちろん私にも、多少そのような経験があります。最初は、あまり意識しなかったのですが、その教師の授業をよく観察してみると、学生たちは私たちが中学や高校でよくやった「内職」 をしていました。志望校の受験科目にあまり関係ないと思ったとき、 受験の教科である 英語の単語を机の下で覚えたり、さしあたり、今 、 行われているの授業よりも、優先度が高いと考える科目の勉強を したりする 、教師の視線を気にしながら 行う 、いわゆる「内職」 です。

「なにやってんだ！？」 と 、 教師から一喝されたことも一度ならずあったはずです。 しかし、面白くない授業に対する学生たちの反応は、どこの国でも同じです。

中国の大学で日本語を学んでいる学生の多くは 、卒業後、日系の会社で通訳や翻訳の仕事 に就くことを 希望しています。そして、日本語能力試験 一 級に合格することは、 そのような 企業に就職するための登竜門なのです。ですから、彼らは授業中、面白くない授業はほとんど聞いていなくて、日本語能力試験一級の文法編や文章読解の問題を解きまくっているのです。その光景に出くわした私は決心しました。

これからは、 「学生との真剣勝負！」 だと。 『やるか、やられるか』 です。

『行列のできる授業をするぞ！ 』 と決意しました。

『 学生が、身を乗り出して聞くような授業です ( ・・・・・・・・・・・・・・・・・・・・ ) 』

もともと、大学の授業 で は、教師が学生のモチベーションを高める ことを意識した 授業はあまり見かけません。しかし、日本では、カリスマ講師と呼ばれる教師が予備校にはいますよね。『日本語学習者』に行う授業に限っては、そのような魅力ある学習が必要だと 私は 思います。

『体験的にそう思います』

彼らの当面の目標である、日本語能力試験の教材開発を研究しようと思ったのは 、 そのような動機からです。私の手作り教材は、新聞、雑誌からの記事は一切使用していません。著作権の問題もあるのですが、自分の思想、 大げさ に言えば哲学でしょうか。常日頃、 目にする 政治・経済・文化・人間関係や自然との触れ合いの中で自分が感じてきたことを文章にしてみました。

 

【一文の文章でも授業が一コマになることがあります。】

たとえば、授業で、日本料理について文章問題があったとします。 読者の 皆さんも、ご存知のように、日本料理は味だけでなく、『見た目』も重んじます。

そのことから、『日本の食文化』 についての話になります。普通、「今夜、 赤提灯 ( あかちょうちん ) にいくか！？」　 という文章があったら、ほとんどの留学生は 理解できません 。大人の日本人だったら、すぐに、「居酒屋で飲もうよ！」という 『 ものの 喩 ( たと ) え 』 としての 比喩 ( ひゆ ) 表現は分かりますよね。 また 、私は日本 人の サラリーマン を例にとり、 アフターファイヴの説明から、授業に出席している留学生のお国の事情を聞いたり し ます。

それが一つの「意見交流の場」 になります。

この 日本語能力試験問題集は、日本の文化や日本人の考え方を、日本語教師が日本語学習者と対話 しながら学習 できるような内容を考え、教材作成しました。そして、実際の日本語能力試験よりも難しめに作成しています。それは、日本語教師が日本語能力試験を指導する時に、教師自身も学生とともに考える面白い授業ができるように、との思いがあったからです。

実際、私が大学で留学生の日本語能力試験の指導をしていて、面白い授業 とは何かということを常に考えてきました。私の教材は、そのような授業を日本語教師にしてもらいたいと願って作成した教科書です。

 

本書の試験問題は、「言語知識・読解」分野が実際の N1 試験と同様の形式と問題数でつくられています。なお、本書では 聴 ( ちょう ) 解 ( かい ) は 取 ( と ) り 扱 ( あつか ) っていません。

　　　　著者　中村一男

「我が国において、外国語教育の【　 41 　】 が指摘されているのは、明治時代の 文明 ( ぶんめい ) 開化 ( かいか ) （注 1 ） 以来のことである。確かに江戸時代の 鎖国 ( さこく ) （注 2 ） 体制による 一国 ( いっこく ) 主義 ( しゅぎ ) の政策が 破綻 ( はたん ) し、西洋文明に適合するためには、日本国家としても外国語教育は 【　 42 　】 ことのできない重要課題となったのは間違いない。事実、 後 ( のち ) に 大文豪 ( だいぶんごう ) となった 夏目 ( なつめ ) 漱石 ( そうせき ) 、 森鴎外 ( もりおうがい ) などが西欧の学問を導入する先駆者となり、 今日 ( こんにち ) の経済 【　 43 　】 日本の 礎 ( いしずえ ) になったのは、国民の誰もが理解している。

近年、外国語の早期教育の一環として、日本の多くの小学校で英語が教えられている。確かに、言語習得の視点から 捉 ( とら ) えると， 臨界期 ( りんかいき ) というものがあり、早く始めれば、それだけの 【　 44 　】 はある。特に発音の面において 顕著 ( けんちょ ) である。以前、私は研究のために、小学六年生に、英語とスペイン語を同時に学習指導したことがあるのだが、彼らは「ネイティヴ」と同じような発音ができるようになった。

しかし、英語の早期教育といっても、現実には教える方の教師の負担感は大変なものである。外国語を指導するのは、専門でもない現場の教師にとって迷惑な話である。教えられる生徒も、 見切 ( みき ) り 発車 ( はっしゃ ) の号令に 戸惑 ( とまど ) いはないのか。外国語教育は、 教える側と教えられる側との明確な目的の共有がないと、目標達成は極めて困難であるというのが、これまで外国語教育に 携 ( たずさ ) わってきた者としての実感である。

外国語教育は急ぐ必要がないというのが、私の持論である。多くの先人が、中学校から文法訳読法で、たくさんの英文に触れ、高度な専門書を訳読するようになったのである。このような高度な外国語読解ができる人物が、世界のビジネスマンと 互 ( ご ) する言語運用能力を身につけてきたのである。

外国語教育は 慌 ( あわ ) てる必要は ぜんぜんないと思う。むしろ、母語能力を高めることが先決である。なぜならば、外国語と母語は基底部においては、【　 45 　】 ので、母語を鍛えておくことは、外国語を鍛えることになるのだから。

（注 1 ） 文明開化：日本が開国した際、諸外国から多くの文化・文物が流入したことをいう。

（注 2 ）鎖国：外国との交流を絶つこと。（当時、日本はオランダ・中国以外の国との国交を持たなかった）

【複数のテキスト（合計 600 字程度）を読み比べて比較・統合しながら理解できるかを問う問題】

 

 

問題　 A と B はそれぞれ 別の雑誌のコラムである。 A と B の両方を読んで、次の問いに対する答えとして、最もよいもの を 1 ・ 2 ・ 3 ・ 4 か ら一つ選びなさい。

3.         器用貧乏の人の成功に関する、具体的事例について

2.         A も B も、ともに「賢明な人」に対する定義が明確である。

3.         A も B も、「器用貧乏な人」は大成する人が少ないと考えている。

4.         A で は学生時代の「共通性」が指摘されているが、 B では指摘されて いない。

 

65 　 A 、 B の記事で書かれている、成功しない人の特徴として、正しいものはどれか。

1.         物事に成功しない人は、成功した人のものまねばかりする。

2.         物事に成功しない人は、目先の困難や障害に目をそむける。

3.         物事に成功しない人は、どんな人にもいい顔で返事をする。

4.         物事に 成功しない人は、仕事に役立つ新しい商品を購入する 。


A

私は何をするにも、人に頼んだり、新しい商品を購入したりして「 煩 ( わずら ) わしさから逃げる」傾向のある人間です。とにかく、面倒くさいことが嫌いで、あれこれと 手間 ( てま ) のかかることに時間をかける気持ちになれないのです。いつも一つのことに集中していたいと思っているからです。しかし、「器用貧乏」な人もいます。いろいろなことに関心を持ち、何をしても器用にこなす人です。でも私個人としては、あまりうらやましいという気持ちはありません。どうも彼らが持っている「共通性」を考えると、一つのことに熱中できないということだと思います。仕事を見つけるのは早いですが、辞めるのも早いという人が多いような気がします。そういう人は職業の物の奥深さを味わうことなく転職を繰り返す人が多いように思います。

私は毎日の新聞で楽しみにしている記事があります。それは、長い間にわたって一つのことを追求してきた人物の「紹介 欄 ( らん ) 」です。どの人も「いい顔」をしています。 付 ( つ ) け 焼刃 ( やきば ) 的に身につけた人にはない表情があります。一つのことを何十年も続けることの才能がある人のことです。プロとしての倫理観と 匠 ( たくみ ) の技を身につけた人のことです。器用貧乏の人は、何をしても普通の人よりも上手にできる器用さがありますが自分の関心が分散する傾向があることから、何をやっても 大成 ( たいせい ) することが難しいようです。

 

B

器用貧乏という言葉があります。これは、「たまたま器用なために、あれこれと気が多く、人に都合よく使われて成功しない」という意味で使われているようです。

そういえば、私の身の回りにもいます。そういう人は学生時代、どんな教科でも満点近くは必ず取れたという人が多いようです。しかし、こういう人は、普通の 地道 ( じみち ) なやり方では満足できず、いろいろ新しいことに興味を示しますが、なにか途中で障害に気づくと、すぐにやめる理由を探し出し、次の仕事に取りかかろうとします。また、自分の失敗を他人のせいにする傾向があり、「負け惜しみ （注１） の減らず口 （注２） 」をよく言います。大失敗をしないという点では「賢い人」と言えるかもしれません。

しかし、長い人生は、結局、苦労や困難の繰り返しです。このような人は、 度重 ( たびかさ ) なる 苦難に立ち向かうことができません。そう考えると、器用貧乏は成功しない人だと思います。

（注 1 ）負け惜しみ：実際には、負けたり、失敗したりしているのに、必死に言いつくろうこと。

（注 2 ）減らず口：状況が悪いのに、反論を言い続けること。
