あらすじ：

　地球は 、 204 0 年代に入り 、 201 0 年代初頭に実用化段階に入ったシリコンカーバイドの技術により、省エネ化が進むと共に、原子力の制御技術は、飛躍的な進歩を見た。

204 0 年代には、人類は、原子力の恩恵に欲し、原子力全盛の時代を迎えていた。

　 205 0年1月3 日、日本で正月休みを過ごした後 、 NAS A に戻る途中で、大学生のタケルとルミはアメリカのサンフランシスコ空港に到着した。彼らは宇宙パイロットの卵であった。空港ロビーで休憩している最中に、巨大地震の揺れを数秒感じたかと思うと、その瞬間、耳をつんざくような巨大な爆発音が発生した。空を飛んでいる飛行機どうしが空中で衝突し出したのだ。さらに、地上では、列車同士の衝突や、自動車同士の衝突が至るところで発生し、まさに、地獄を見るような様相を呈していた。しかも 、5 分後には、地球の時点速度が徐々に低下し始めていた。太陽の直経が、異常磁場の影響により、徐々に収縮し始め、従来、バランスを保っていた太陽系の惑星間に存在するダークマターの分布が異常を来たし始めたからだ。

　米国本土にめぐらした超伝導ケーブルのラインに沿って、巨大な異常磁場が発生し、その影響で 、 I T 機器を搭載した航空機、列車、自動車などが全てコントロール不能となり、米国全土が巨大なパニック状態に陥ったということであった。

　同時に、太陽系の宇宙空間が極度に歪み 、4 次元空間から、地球とそっくりな青く美しい天体が出現した。いっぽう、地球の自転速度が次第に下がってくるのに伴って、人類の新陳代謝が低下し、睡眠時間が伸びてきたことが社会的パニックを引き起こしつつあった。

　 205 0年1月3 日サンフランシスコで発生した強大なマグニチュー ド 9. 0 の地震は、全米各地の原子炉が、地震の影響でメルトダウンを発生させた。その後、爆発した原発の送電線を通じて、巨大な原子爆発のエネルギーが全米本土に張り巡らされた超伝導ケーブルにリークし、巨大な異常磁場の原因となったという衝撃的な事実が浮かび上がった。

　世界中のコンピューターシステム や I C チップを埋め込んだマシンのほとんどは、無残にも破壊され、人、モノ、金のあらゆる情報は、今回の巨大磁場の発生により、消滅してしまった。しかも、全世界中には放射能物質の死の灰が拡散し、人類は、地中深くに用意していたシェルターに退避して、穴蔵生活に甘んじざるを得ない状況となった。

　異常磁場による世界的な壊滅の翌日、全世界の首脳が国連に集まり、今後の世界のあり方を検討した。その結果、人類のシェルター生活は、長く見積もっても 、6 箇月の猶予しかなく、他の惑星に移住するしか選択肢はないとの結論に至った。双子の地球への移住である。

　その後、全世界の主要各都市のはるか上空に、巨大な宇宙船と思われる編隊が姿を表した。主要都市上空にとどまっていた宇宙船の編隊の中から地上に現れたのは、人類と瓜二つの姿をした宇宙人の集団であった。彼らの話によると 、 20 0 万年前、地球人類は、銀河系で非常に進歩した文明を保有していたが、銀河系における惑星間の大戦争によって、人類の大半は消滅し、残った人類の一部が、当時から存在してい た 1 0 万光年かなたの双子惑星の地球に移住したということらしい。

　当時、地球に残った人類の一部は、原始生活に入っていったが、地球を脱出した人類の一部は、高度な科学文明を継承し、発展させ 、4 次元宇宙の世界で、遥か彼方から、兄弟の地球を見守ってきたのだと語った。彼らが住んでいる双子惑星の地球を、彼らはミューズと呼んでいた 。 20 0 万年を経て、今回の地球の危機的な状況を察知して、手を差し伸べに来たのだといった。先方の偵察をするチームメンバーとして、タケルとルミは選抜メンバーに選ばれ、双子惑星の地球 で1 ヶ月ほど、滞在することになった。

　ミューズ人の話では、彼らは、時間のスピードの伸縮を自由自在にコントロールできる能力を個人レベルで備えているらしい。そのため、年齢は自由自在に選択でき、若い年代に戻りたければ、パワーズという若返りの薬を一錠飲めば、一晩で若返ることができる。同じく老人に戻りたければ、老化用のパワーズを飲めば、老人に変身できる。現在、過去、未来を行き来するタイムマシン機能を体内に備えているとのことだ。

　遺伝子工学が極度に発展し、生命現象の謎の全て を 1 0 万年かけて解き明かしたミューズでは、生命を完璧にコントロールできる社会である。恋愛は自由恋愛であり、結婚という概念は存在しない。生命孵化装置を使って、自宅にいながら自分のコピーを培養させて誕生させることもできる。生まれた分身は、既に自分と同じような経験や記憶を移植されている。

　ミューズでは、時間のスピードをコントロールできるため、生産性や価値、効率といった概念もなく、お金というような諸悪の根源となるものも存在しない。

　しかし、ミューズでは人口は爆発的に増加し、地球 の 10 0 倍にも達していた。彼らが密かに狙っていたのは、豊かな地球の水資源とタンパク源であった。地球人類は、ミューズに移住し、ミューズで楽園の生活に浸っていたが、違和感を感じ始めていた。そこでは 、4 次元宇宙の銀河系における全知全能の悪の神が、地球人類抹殺のホロコースト計画を密かに進行させていた。

「タケル、大丈夫なの？しっかりして」ルミは、真っ白に凍りついて地面に倒れたタケルを揺り起こそうとしたが、あまりの冷たさに飛び上がってしまった。

タケルの心臓は止まっていた。

「タケル、私を一人にしないで、お願い、目を覚まして」

ルミの瞳に浮かんだ涙は、タケルの頬を幾度となく濡らしたが、タケルは再び目を開けることはなかった。

タケルの体が空中に浮いたかと思うと青白く輝き出して、空高く舞い上がり、螺旋の軌跡を描きながら天空に消えていった。

「ルミといったな。タケルは冥界の人となってもう帰ってくることはないだろう」そう言うなり、悪の神はルミを手下のロボットに捕らえさせて、高らかな声で笑い出した。ルミはあまりのショックに気絶してしました。

冥界に連れて行かれたタケルは、幼いころに死に別れたおじいさんとおばあさんのなつかしい声を聞いて目を覚ました。

「タケル、どうしてこんなところに来たのですか」

タケルは、あの懐かしい声は幼い時に自分をかわいがってくれたおじいさんとおばあさんの声に違いないと思いながらも、真っ暗闇をさまよっていた。

「今日は、空は澄み切って青空が広がっているというのに、暖かい雨が降っている。これは、タケルの恋人の悲しみの涙ですね」おばあさんは、暖かい涙の雨でやっと目を覚ましたタケルを見つめながらにっこり笑った。

「かわいそうなタケル、若くして冥界にくるというのは、タケルらしくない」

「ルミさんが待っているから、タケルは元の世界にお帰りなさい」おばあさんはタケルの頭をなでた後、両腕の人差し指で空中に円弧を描いたかと思うと、タケルの前には、ピンク色と黄金色で輝く、直 径2 メートルほどの大き な2 重のリングが現れた。タケルはそのまばゆい美しさを放 つ2 重のリングに魅入られていた。

「タケル、このリングの中を潜りなさい。そうすれば、あなたは不死鳥となって、全宇宙のエネルギーを一身に集めることができるから。その力で悪の神に打ち勝つことができるはずだから。」おばあさんの教えに従って、タケルは光り輝 く2 重のリングを潜り抜けると、タケルは、全 長 10 0 メートルはあると思われる、まばゆい光で輝く巨大な不死鳥に変身していた。

「さあ、タケル、ルミさんと人類を助けにいきなさい。」おじいさんとおばあさんはタケルを励ましながら、タケルを大空に送り出した。

不死鳥となって甦ったタケルは、大空高く舞い上がって、おじいさんとおばあさんとの別れを惜しむ間もなく、冥界 の7 次元のタイム・トンネルを通過し、悪の神が存在す る6 次元に舞い戻ってきた。

ルミをにらみつけていた悪の神は、巨大な不死鳥の出現に驚いた。

「お前は何者だ。」

「私はヤマトタケルです。冥界から舞い戻ってきたのです。ルミ、大丈夫？」

「タケル、大丈夫だから、心配しないで。」

タケルは 、 20 0 メートルはあるとも思われる大きな翼で大きな風を起こし始めた。その風は、全宇宙のエネルギーを瞬間の移動によって集めたもので、その巨大なハリケーンのような風に当たった悪の神は、全身が赤く輝き始め、燃え始めた。「止めてくれ。」

悪の神が、そう言い終わるや否や、それまで燦然とした輝きを放っていたレインボー色に輝く光が、悪の拠点であったピラミッドから消え、回転が停止した。それと同時に、悪の神は、のたうち始めた。

「タケル、何が起こったの？」

「僕にも、わからない。」不死鳥から元の人間の姿に戻っていたタケルは、何となく白髪の科学者がくれた腕時計に目をやった。そのとき 、 1 0 センチ四方に浮かび上がったスクリーンには、メッセージが映し出されていた。そのメッセージは、「ヤマト・タケルより」と書かれていた。さらに、次のようなメッセージが映し出された。

「アカシックレコードのデータ・ベース・システムに全宇宙最強のウイルスを埋め込んでおきました。まもなく、悪の神も消滅するでしょう。さようなら」

このメッセージを読み終わったタケルは、眼前でのたうち回っていた悪の神が宇宙から消滅していることに気づいた。アカシックレコードの記憶データの源であるゼリー状の海は、全て蒸発し、巨大なピラミッドには一滴も残っていなかった。

「その通りね、万事休すを脱したわ、タケル」

　「地球 の NAS A 司令部から、連絡が入っている、ルミ」

「どんなメッセージなの？タケル」

「地球は放射能汚染から開放され、磁気嵐も去った。全人類はノアの方舟に乗って地球に戻ってくるようにって、ルミ」

「痛いわ！」

二人の笑顔には、神話に出てくる神々の顔が宿っていた。

 

　ホロコースト惑星に地球人類が旅立つカウントダウンの１日前は、アリゾナ州 の NAS A 司令部では、天と地がひっくり返るような慌ただしい状況であった。

　そこでは、世界の優秀な科学者の叡智を結集して、地球全土に拡散した放射能物質を消滅させる超ナノ粒子を短期間で開発し終えていた。放射能のパワーを減衰させ、中和して消滅させる機能を備えたその粒子は、「アース」と名付けられていた。「アース」は、放射能物質を吸収して、消滅させる際に、ピンク色の光を放つ特性を有していた。

　 NAS A 司令部は直ちに地球浄化計画を実行に移した。その計画とは、超ナノ粒子「アース」を地球全土に向けて、遥 か 1000 0 メートルの上空から散布して「アース」のドームを形成し、放射能物質を消滅させるというものだ。

　その日の午後 、 NAS A 司令部では、巨大なロケッ ト 1 0 基にそのドーム形成マシンを搭載して、赤道直下の経路 を1 週させた。全地球に「アース」を散布することに成功し た NAS A の職員は、安堵で胸をなでおろしていた。人工衛星から送信されてくる映像には、鮮やかなピンク色に光るドームで覆われた地球の姿があったからだ。それは、「アース」が形成するドームが、全地球の放射能汚染から開放してくれることを明らかにしていた。

　翌朝には、地球の異常磁場は収束し、磁気嵐の猛威は消え去っていた。地球の自転速度も正常に戻り、本来の地球の姿を取り戻していた。（完）
