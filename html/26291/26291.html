誰 に で も 送 れ る 、 コ ピ ペ の よ う な 文 章 で は 駄 目 だ ！ と い う こ と で す 。

そ ん な 文 章 は 、 そ の 人 に 会 わ な く て も 送 れ て し ま う 内 容 で す 。

そ ん な 文 章 が 人 の 心 を 動 か す と 思 い ま す か ？

ど ん な 口 下 手 な 人 で も L I N E I D 交 換 が 出 来 る 関 係 が 作 れ た の な ら ば 、 何 個 か の 話 題 が 出 て い た は ず で す 。

そ の 話 題 を 拾 っ て な ん で も 良 い の で 文 章 に 入 れ て い き ま し ょ う 。

そ う す れ ば 相 手 は 、 「 私 に 興 味 を 持 っ て く れ て い る の だ な 」 と 感 じ て く れ る は ず で す 。

あ な た の 少 な か ら ず あ る で あ ろ う 、 下 心 す ら 上 手 く 隠 し て く れ ま す よ 。

な の で 、 相 手 と の 会 話 は 面 倒 く さ く て も 聞 き 流 さ ず に き ち ん と 対 処 す る こ と を 心 が

け ま し ょ う 。

何 度 も 言 う と お り 、 フ ァ ー ス ト メ ー ル は 本 当 に 重 要 で す 。

時 間 が た て ば た つ ほ ど 、 相 手 は あ な た の 事 を 忘 れ る し テ ン シ ョ ン も 下 が っ て き て し ま い ま す 。

そ う な ら な い た め に も 、 フ ァ ー ス ト メ ー ル の 重 要 性 を 感 じ て 頭 を フ ル 回 転 さ せ て 送 っ て く だ さ い 。

メ ー ル の 特 徴 は 後 か ら 読 み 返 す こ と が 出 来 る と い う こ と で す 。

良 い 内 容 の 文 章 が 作 れ れ ば 、 相 手 が 勝 手 に 何 度 も 読 み 返 し て 、 そ の 文 章 は 相 手 の 心 を 何 度 も 揺 さ ぶ っ て く れ る こ と で し ょ う 。

人 間 は 感 情 が 行 動 を 引 き 起 こ す 生 き 物 で す 。

相 手 の 心 を 揺 さ ぶ る 文 章 を 作 っ て し ま え ば 、 あ と は そ の 感 情 が 体 を 動 か し て く れ ま す 。

そ れ で は 実 験 を す る 程 度 で も 良 い の で 、 こ の 本 に 書 か れ て い る 内 容 を 試 し て み て く だ さ い 。

き っ と 以 前 よ り も 深 い 関 係 や 、 良 い 反 応 が 返 っ て く る と 思 い ま す 。

あ と は あ な た が 行 動 す る だ け で す 。
