毒見師イレーナ

［著］マリア・Ｖ・スナイダー
［翻訳］渡辺由佳里
　ある殺人を犯した罪で死刑囚となった少女イレーナ。ついに絞首台へと送られる日を迎えるも、そこで思わぬ選択肢を与えられる――今すぐ絞首刑か、それとも、国の最高司令官の毒見役になるか。だが毒見役を選んだイレーナを待ち受けていたのは、逃走防止の猛毒だった。かくして少女は毎日与えられる解毒剤なしには生きられぬ身体に。わずかな生きる希望に賭け壮絶な日々に立ち向かうが......。

　　　1


　生きながらにして棺の中に埋葬されたような暗闇だ。希望の光が一筋も差しこまないこの場所では、少しでも気を緩めると、思い出したくない過去がまるで現実のように襲いかかってくる。
　突然、白い炎が目の前に現れた。頬にはあのときの灼熱( しやくねつ ) 感が蘇( よみがえ ) る。突きつけられた松明( たいまつ ) から逃れようとしたが、身体( からだ ) を柱にくくりつけられているせいで、それ以上あとに退( ひ ) けない。
　のけぞらせた背中に、後ろ手に縛られた両手が食いこむ。炎がさらに近づき、額が耐えがたいほど熱くなり、何かが燃える匂いがした。顔を焼かれる......そう覚悟した瞬間、炎が遠ざかった。だが、眉と睫毛( まつげ ) はすでに焦げ落ちていた。
"松明の火を消してみろ！"男の荒々しい声が命じる。わたしはひび割れた唇から必死で息を吹いた。炎の熱と恐怖で口の中はからからに乾いていて、歯はまるでかまどで焼かれたかのように熱い。
"ばかもん！"男は悪態をついた。"口で消すんじゃない、意志を使え。精神力で火を消すんだ"
　慌てて目を閉じ、炎が消えてなくなるよう念じる。それがどれほどばかげたことでも、この男を満足させるためなら、なんでもするつもりだった。
"意識を集中させろ！"ふたたび炎が顔に近づいてくる。
"髪に火をつけてやればどうですか？"別の男が提案した。先の男より若くて、貪欲な声だ。"そうすればこいつも、もっとやる気を出しますよ。ほら、父上、わたしにお任せください"
　その声の主が誰なのかに気づき、恐ろしさで全身が引きつった。拘束を解こうとしてもがくうちに、どこからともなく振動音が聞こえてくる。その低い音は、どうやらわたしの喉の奥から出ているようだ。音響の中、意識が散り散りになっていく。響きはしだいに音量を増し、室内を満たし、ついに炎をかき消した――。
　そのとき、鍵を開ける金属音が鳴り響き、悪夢のような記憶から現実へとわたしを引き戻した。暗闇をほのかな黄色の光が切り裂き、独房の重い扉が開くにつれて、光の帯が石の壁を伝ってくる。灯火( ともしび ) に顔を照らされ、眩( まぶ ) しさに目をきつく閉じ、独房の隅で身をすくめた。
「さっさと来い、ドブネズミ。でないと鞭( むち ) が飛ぶぞ！」
　看守がわたしの首についている金属の首輪に鎖をつけ、身体ごと引き起こした。躓( つまず ) いて前のめりになり、喉が潰( つぶ ) れそうになる。震える足でなんとか立ち上がると、看守たちはわたしを後ろ手にし、手際よく手錠と足枷( あしかせ ) をつけた。
　ふたりの看守に連れられて両脇に檻( おり ) が連なる地下牢( ちかろう ) の廊下を歩く間、角灯( ランタン ) のちらつく光から目を背けた。得体の知れない汚物がたまった牢獄の床を裸足( はだし ) で小刻みに歩いていると、四方から囚人の野次( やじ ) が飛んでくる。
「は、は、は。誰かさんが吊( つ ) られるようだぞ」
「餌を食い潰すドブネズミが一匹減って、いいこった」
　看守は囚人たちの声には耳を貸さず歩み続けるが、わたしの心臓は投げかけられる言葉にいちいちびくりと反応する。
「ボキッ！　ポキッ！　そして、最後の晩餐( ばんさん ) が足を伝って流れ落ちる」
「俺も、俺も連れていってくれ！　俺も死なせてくれ！」
　看守が立ち止まった。うっすらと開けた目に見えたのは階段だ。最初の段に上ろうとしたが鎖に躓いて倒れ、首を引っ張られた。石階段の尖( とが ) った角が肉に食いこみ、むき出しになっている腕や脚の皮膚が削( そ ) ぎ取られる。
　いくつかの扉を引きずられるようにして通り抜けたあと、看守はわたしを乱暴に放り投げた。ぶざまな格好で床に転がったわたしの目に、まばゆい光が降りかかる。陽( ひ ) の光を見るのは本当に久しぶりだった。固く目を閉じると、涙が頬を伝った。
　ついに最期のときがきたのだ。そう思ったとたん、パニックに陥りかけた。けれども、死刑になれば地下牢での惨めな日々も終わるのだと考えたら、心が静まった。
　ふたたび看守に引き起こされ、従順にあとをついていった。汚れた藁( わら ) の上で寝起きしていたので、全身が虫刺されだらけで痒( かゆ ) く、ドブネズミのような悪臭を放っている。与えられた水では喉の渇きを癒すのがやっとで、身体を洗うことができなかったのだ。
　目が明るさに慣れてきたので、周囲を見渡した。城の主要な廊下は、かつて金の燭台( しよくだい ) や豪華なタペストリーで飾られていたと聞くが、今歩いている廊下の壁には何もかかっていない。冷たい石畳の床は、中央の部分だけがすり減って滑らかになっている。たぶん、使用人や護衛専用の隠れ廊下なのだろう。
　開け放たれた窓の前を通り過ぎるとき、どんな食べ物でも満たすことができない飢えを覚え、外の風景を見やった。鮮やかなエメラルド色の草が、目に染みた。木々は豊かな葉をまとい、草花は歩道を縁取り、木樽( きだる ) に植えられた花はこぼれ落ちんばかりだ。新鮮な風は高級な香水のようにかぐわしくて、思いきり胸に吸いこんだ。排泄物( はいせつぶつ ) と体臭の酸っぱい匂いばかりを嗅いでいたから、澄んだ空気はまるで上質のワインのような味がした。暖かい空気が肌を優しく撫( な ) でる。常にじっとりと湿って肌寒い地下牢とは違い、癒される心地よさだった。
　どうやら、暑い季節が始まったばかりらしい。ここイクシア領には六つの季節がある。わたしが囚( とら ) われたのは暑さが和らいだ頃だったから、牢獄に閉じこめられていたのは五つの季節ということになる。あれから、もう少しで一年が経( た ) つ......死刑が決まっている囚人を閉じこめておくには、あまりにも長すぎはしないか。
　鎖に繋( つな ) がれたまま歩くのは辛( つら ) く、息を切らしていると、広い執務室に連れこまれた。イクシア領とそれを取り囲む領土の地図が壁を覆っている。床には本を積み上げた山がいくつもあり、まっすぐに歩くことができない。
　部屋のあちこちに短長異なる使いかけの蝋燭( ろうそく ) が散らばっていて、蝋燭の炎を近づけすぎたのか焦げあとができた書類もあった。部屋の中央を占領している木製のテーブルには書類が散乱し、六脚の椅子がまわりを取り囲んでいる。
　執務室の奥にある机に、男が座っていた。その背後にある大きな窓から吹きこむ風が、肩まで届く男の髪をなびかせている。
　死刑囚は処刑の前に、役人の前で自分の罪を告白しなければならない――独房の囚人が、ささやき声でそんな噂話( うわさばなし ) をしていた。ここがその部屋だとしたら、告白を終えたら次は絞首台だ。全身が震えおののき、身体につけられた鎖がその振動で音をたてた。
　机に座っている男は、襟にふたつの赤いダイヤ模様が縫いつけられた黒いシャツと、黒いズボンを身にまとっている。最高司令官の顧問官の身分を示す制服だ。無表情の顔には疲労がにじんでいたが、サファイアを思わせる青い目は、わたしを射抜くように鋭く観察した。
　突然自分の身なりを意識して、恥ずかしさがこみあげた。汚れた足にはタコができていて、長い黒髪は脂ぎってもつれた塊だ。薄っぺらな赤い囚人服はぼろぼろに破け、垢( あか ) が染みついた素肌が透けて見えている。羞恥と鎖の重みで、気が遠くなった。
「女じゃないか」男の目が驚きで見開かれた。「次に処刑される死刑囚は女なのか？」凍りつくような冷たい声だ。
"処刑"とはっきり言葉に出されて、さっき取り戻した冷静さが吹き飛んだ。もし看守たちがそこにいなかったら、床に崩れ落ちてすすり泣いていただろう。だが、弱みを少しでも見せたら看守から痛い目にあわされる。
　男は黒い巻き毛を耳にかけて、誰ともなしにつぶやいた。「もう一度、事件の関係書類を読んでおくべきだったな」それから看守たちを追い払った。「下がっていい」
　看守らが退出したあと、男は身振りで、机の前にある椅子に座るよう命じた。わたしは鎖をガチャガチャと鳴らしながら椅子の端に腰かけた。
　男は机の上にあるフォルダを開け、じっくりと中を読んだ。「イレーナ、今日は君にとってとても幸運な日かもしれないぞ」
　思わず口にしそうになった皮肉をのみこんだ。地下牢で学んだのは、絶対に反抗してはならないという教訓だ。口答えせず、頭を垂れて男と目が合うのを避けた。
　男はしばしの間無言でいた。「品行方正で礼儀正しい、か。だんだん君が適任者に見えてきたな」
　部屋の乱雑さにもかかわらず、机の上は整頓されていた。フォルダと筆記具のほかには小さな彫像しかない。銀の斑紋が光り輝く黒豹( くろひよう ) 二頭は、本物そっくりの精密さだ。
「君はブラゼル将軍のひとり息子レヤードを殺害した容疑で裁判を受け、有罪判決を受けた」男はそこで言葉を止め、指でこめかみを撫でた。「ブラゼルがなぜ城にいるのか不思議だったが、理由はこれだったのか。異常なほど処刑に興味を持っているのも納得できるな」わたしというよりも、自分に話しかけているようだ。
　ブラゼルの名前を耳にして、胃がこわばった。けれど、どうせもうじき彼の手が届かない場所に行くのだと自分に言い聞かせて、心を落ち着かせた。
　イクシア領の軍隊が政権を握ったのはほんの一世代前のことで、その統治者は『行動規範』という厳しい法律を作った。戦時中の例外を除き、人を殺したら死刑になるのが定めだ。自己防衛や過失致死であっても弁解にはならない。有罪の判決が下ったら、殺人犯は城の地下牢に送られ、公衆の面前で絞首刑になる。
「おそらく君は罪状に抗議するつもりでいるんだろう？　濡( ぬ ) れ衣( ぎぬ ) を着せられたとか、自己防衛だったとか」男は椅子の背に反り返り、うんざりしたようにわたしの答えを待った。
「いいえ、そのつもりはありません」小声でささやいた。長い間使っていなかった声帯では、声を出すのもやっとだ。「わたしはレヤードを殺しました」
　黒い制服の男は背筋を伸ばして座り直し、厳しい視線でわたしを見据えた。それから、「これは、思ったよりうまくいくかもしれないな」と声を出して笑った。
「イレーナ、君に選択肢を与えよう。死刑か、最高司令官アンブローズの新しい毒見役になるか、どちらかを選びなさい。つい先日、最高司令官の毒見役が死んだから、後継者が必要なんだ」
　唖然( あぜん ) として男を見つめた。胸が高鳴った。でも、冗談を言っているに違いない。たぶん、からかって楽しんでいるのだ。笑うのには最高の方法じゃないか。囚人の表情が喜びに輝くのを見てから、絞首台に送って希望を打ち砕くなんて。
　わたしは騙( だま ) されたふりをして調子を合わせた。「愚か者でない限り、毒見役になるのを断ったりはしないでしょう」わたしのかすれ声はさっきより大きくなっていた。
「だが、これは一生辞められない仕事だぞ。それに、毒見役の訓練は厳しい。毒の味を知らずして最高司令官の食事に毒が入っているかどうか判断できないからな。訓練中に死ぬこともある」男は書類をフォルダの中に片付けた。
「城の中に寝る部屋は与えるが、日中のほとんどを最高司令官と過ごすことになる。休日はない。結婚も、子どもを産むことも許されない。死刑を選ぶ囚人もいる。次の一口で死ぬかもしれないと怯( おび ) え続けるよりも、処刑の場合は少なくともいつ死ぬかわかるからな」男は獰猛( どうもう ) な笑みを浮かべた。
　この人は本気だ。興奮で全身が震えた。生き延びるチャンスがある！　最高司令官のために働くのは地下牢に戻るよりましだし、もちろん絞首刑よりずっといい。だが、頭に疑問の数々が浮かんできた。なぜ人を殺したわたしに、こんなにも重要な仕事を与えるのか？　最高司令官を殺すかもしれないし、逃げ出すかもしれない。それを、どうやって防ぐつもりなのか？
　けれども、そんな質問をしたら、この人は自分の過ちに気づいてわたしを絞首台に送るかもしれない。代わりに別のことを尋ねた。
「今は誰が最高司令官の毒見をしているんですか？」
「わたしだ。だから、代役を早く見つけたい。それに『行動規範』では、空席ができたら次に処刑される予定の者が埋めることになっている。命を奪った罪の代償として、自分の命を提供するわけだ」
　じっと座っていられなくなり、立ち上がった。壁に貼られた地図に目をやると、軍の戦略的な配置が記されている。散らばっている本の題名は国防と諜報( ちようほう ) 技術に関するものだ。蝋燭の数と溶け具合は、彼が深夜遅くまで働いていることを示している。
　あらためて、顧問官の制服を着た男を見つめた。この男はヴァレクに違いない。最高司令官に直接仕える防衛長官で、イクシア領の膨大な諜報ネットワークの指揮官だ。
「死刑執行人にどう伝えようか？」ヴァレクが尋ねた。
　きっぱりと答えた。「わたしは愚か者ではありません」




　　　2


　ヴァレクはフォルダをおもむろに閉じて入り口に向かった。薄氷を横切る雪豹( ゆきひよう ) のように、優雅で軽やかな動きだ。扉が開いたとたん、廊下で待っていた看守たちは、ぴしっと姿勢を正した。
　それからヴァレクは何やら看守たちに指示を出し、頷( うなず ) いた看守のひとりがこちらに近づいてきた。わたしは身構えた。また地下牢( ちかろう ) に戻されるのだろうか？　思わず逃げ道を探して部屋を見渡す。だが、看守はわたしを後ろに向けて手足の枷( かせ ) と首の鎖を外しただけだった。囚( とら ) われたときからずっと手枷がついていたため、手首の皮は帯状に擦( す ) りむけている。同じく金属の首輪がつけられていた首筋に指で触れると、ぬるりとした血が付着した。
　ふらつき、とっさに椅子の背を掴( つか ) んだ。鎖の重みから解放されたとたん、身体( からだ ) が浮き上がるような、気が遠くなるような、奇妙な感覚に襲われたのだ。卒倒しないように目を閉じて、何度も深呼吸を繰り返した。
　ようやく気分が落ち着いて目を開けると、ヴァレクはすでに机に戻り、立ったままふたつのゴブレットに飲み物を注( つ ) いでいた。扉が開いた戸棚にはさまざまな形の瓶と色とりどりの壺( つぼ ) がぎっしりと並んでいる。ヴァレクは手に持っていた瓶を戸棚に戻し、扉に鍵をかけた。
「マージが来るのを待っている間、飲み物でもどうかと思ってね」ヴァレクは、琥珀色( こはくいろ ) の液体が入った大きな錫( すず ) のゴブレットをわたしに手渡した。そして、自分のゴブレットを掲げて乾杯をした。「新任の毒見役イレーナへ。前任者より長続きしますように」
　ゴブレットに口をつけかけていたわたしは、そこで手を止めた。
「気にするな。いつもどおりの祝杯の言葉だ」
　喉が渇いていたので、ごくごくと飲んだ。滑らかな液体は喉元を過ぎるとき、わずかにひりつくような痛みをもたらした。水以外の液体を飲むのは久しぶりだったので、つかのま、胃が受けつけないのではないかと案じたけれど、どうやら取り越し苦労だったようだ。
　前任の毒見役がどうなったのか尋ねかけたとき、飲み物の成分を当てるように命じられた。そこで、あらためて少量を飲んでから答えた。
「蜂蜜で甘みをつけた桃ですね」
「よろしい。ではもう一度。今度は飲みこむ前に液体を舌でかき混ぜてみなさい」
　驚いたことに、今度は柑橘系( かんきつけい ) の味がかすかにする。「オレンジですか？」
「そのとおり。次はうがいをしてみなさい」
「うがい？」尋ね返すと、ヴァレクは頷いた。ばかばかしいと思いながら残りの液体でうがいをして、つい吐き出しそうになった。「腐ったオレンジ！」
　笑うヴァレクの目のまわりに笑い皺( じわ ) ができていた。さっきまでは金属でできているみたいに硬くて冷たい顔だったのに、微笑( ほほえ ) むと柔らかくなる。ヴァレクは自分のゴブレットをわたしに手渡すと、もう一度実験を繰り返すよう命じた。
　やや狼狽( ろうばい ) しながら一口すすると、さっきのようにわずかにオレンジの味がした。腐った味に身構えつつうがいをしたところ、今度はオレンジのエッセンスが強まっただけだったので、ほっとした。
「さっきよりましか？」ヴァレクは空っぽになったゴブレットを受け取りながら尋ねた。
「はい」
　ヴァレクは机に座って、ふたたびさきほどのフォルダを開けた。羽根ペンを取り上げ、書類に記入しながらわたしに話しかけた。「これは最初の毒見の訓練だ。君の飲み物には《蝶の塵( バタフライズ・ダスト ) 》と呼ばれる毒が盛られていたが、わたしのものには入っていなかった。この毒が入っているかどうかを察知する唯一の方法は、うがいをすることだ。君が味わったあの腐ったオレンジのような風味が《蝶の塵》だ」
　反射的に立ち上がった――目の前が真っ暗になった。「致死的な毒ですか？」
「量が多ければ二日以内に死ぬ。二日目まで症状は出ないが、出たときには手遅れだ」
「わたしが飲んだのは致死量ですか？」息を止めて、その答えを待った。
「もちろん。それ以下だったら毒の味には気づかなかったはずだ」
　とたんに胃が痙攣( けいれん ) して、吐き気をもよおした。だが、この男の前で嘔吐( おうと ) する屈辱を避けるために、必死に胆汁を飲みこんだ。
　ヴァレクは書類の山から顔を上げ、わたしの表情を観察した。
「訓練は危険だと言っただろう。しかし、君の身体が栄養失調で弱っているうちは身体が持ちこたえられない毒を与えるわけにはいかない。《蝶の塵》には解毒剤がある」そう言って、白い液体が入った小さなガラス瓶を見せた。
　椅子に崩れ落ちて、ほっと安堵( あんど ) のため息をついた。だが、他人を寄せつけない冷たい表情に戻っているヴァレクを見て、彼が解毒剤をくれると言っていないことに気づいた。
「君が訊( き ) くべきだったのに訊かなかった質問への答えが、これだ」ヴァレクは小さなガラス瓶を持ち上げて振った。「最高司令官の毒見役が逃亡できないようにする対策だよ」
　わたしはその言葉の意味を理解しようとしてヴァレクをじっと見つめた。
「イレーナ、君は人を殺した人間だ。抑止対策なしに最高司令官に仕えさせると思うのか？　護衛が厳重に守っているから君が最高司令官を襲うのは無理だが、《蝶の塵》はそれ以上に制御の効果がある」ヴァレクは解毒剤の瓶を、窓から差しこむ陽( ひ ) の光の中でくるくるとまわした。「君が生き延びるためには、毒が身体にまわって死なないようにしてくれる解毒剤を毎日飲む必要がある。毎朝君がわたしの執務室に赴く限りは解毒剤を与えるが、一日それを怠ったら、翌日君は死ぬ。犯罪や謀反をおかせば、毒が君の身体を蝕( むしば ) みつくすまで地下牢で過ごすことになる。わたしなら逃亡も犯罪も考えないな。激しい腹痛と嘔吐に見舞われ、いっそ殺してくれと懇願しながら死にゆく毒だから」
　自分の置かれた状況をまだしっかり把握しきれずにいると、ヴァレクの視線がわたしの背後に向けられた。振り向くと、家政婦の制服を着た恰幅( かつぷく ) のよい女が扉を開けて入ってくるところだった。ヴァレクは、わたしの世話をしてくれるマージだと紹介した。マージは、ついてこいとも言わずに、さっさと大股で外に出ていった。
　わたしはヴァレクの机の上にある瓶に目をやった。
「明日の朝、またわたしの執務室に来るように。マージが連れてきてくれる」
　退出を命じられているのは明らかだったが、尋ねそこねたたくさんの質問をしたくて戸口で足を止めた。けれども、それらをのみこんだ。数々の疑問は重い石のように胃に沈み、わたしは扉を閉めて、どんどん先に歩いていくマージのあとを急いで追った。
　マージはまったく歩みを緩めようとしなかったので、ついていこうとすると息が切れる。道中の廊下や曲がり角を覚えようとしてみたものの、マージの大きな背中と早足に集中するのに精一杯で、途中で諦めた。マージの黒いスカートは床に届くほど長かった。家政婦の制服はウエストがぎゅっと締まった黒いスカートと首から踝( くるぶし ) まである白いエプロンで、エプロンには縦に二列の赤いダイヤ模様の縞( しま ) がついている。
　ようやくマージが大きな浴場の入り口で立ち止まったので、眩暈( めまい ) がおさまるまで床に座りこんだ。
「あんた、臭いよ」マージは嫌悪感をむき出しにして、肉付きのいい顔をしかめた。人に指図するのに慣れているらしく、奥にある浴槽( よくそう ) を指差して命令した。「湯船に浸( つ ) かるのは身体を二度洗ってからだぞ。制服はその間に持ってきてやる」
　身体を洗えるとわかって、ふいに元気になった。囚人服を脱ぎ捨てて洗い場に急ぐ。頭上にある蛇口をひねると、熱い湯が流れ落ちてきた。城の浴場には湯が入ったタンクが設備されているらしい。ブラゼルの豪華な館にすらなかった贅沢品( ぜいたくひん ) だ。
　マージに命じられたように、髪と身体を二度洗った。擦りむけている首や手首や足首に石鹸( せつけん ) がひどくしみたけれど、気にはならなかった。二度洗い終わったあとで、また二度洗う。なかなか落ちないしつこい汚れをこすり続け、それが痣( あざ ) だと気づいてようやく洗うのをやめた。
　身体は綺麗( きれい ) になったが、今、この瞬間にも《蝶の塵》がわたしの内部を蝕んでいる。明日の朝、解毒剤を飲まなければ、明後日には毒が全身にまわって死ぬ。明日を生き延びても、死ぬまで《蝶の塵》から逃れられないのだ。
　毒のことを忘れたくて、頭に叩( たた ) きつける熱い湯の中にじっと立った。なぜか、湯の滝に打たれているのが自分の身体だという実感が湧いてこない。肉体は投獄された苦痛と屈辱を覚えているけれど、遠くから他人を眺めているような気もする。ブラゼルの館で過ごした最後の二年間に、わたしの霊魂( ソウル ) がどこかに行ってしまったからだろうか？
　いきなり、目の前にブラゼルの息子レヤードの姿が浮かんだ。怒りで歪( ゆが ) む端整な顔の生々しさに反射的にあとずさりし、両手を盾にして自分をかばう。目を閉じて鞭( むち ) が飛んでくるのを待ち構えたが、何も起こらなかった。こわごわ目を開けると、レヤードの姿は消えていた。
　だが、震えは止まらない。幻覚にしては、あまりにも鮮明だった。レヤードの亡霊が呼び起こした醜い記憶を忘れるために、何かしていようと、櫛( くし ) を探すことに神経を集中させた。
　ようやく見つけたものの、もつれきった髪にはなかなか櫛が通らない。今度は髪を切る鋏( はさみ ) がないかと探している最中、視界の隅に誰かがいることに気づいた。そちらに目を向けると、屍( しかばね ) が立っていた。頬がこけ、肋骨( ろつこつ ) と骨盤が浮き上がっている。棒のように細い脚が上半身を支えているのが不思議だったが、緑色の瞳は、まだ生きている人間のものだ。
　突然、凍りつくような恐怖とともに気づいた――あれは、わたしじゃないか。変わりはてた自分を正視できず、慌てて鏡から目を背ける。でも、弱虫、と自分を叱りつけてもう一度じっくりと見た。わたしはこんなに変わってしまったのだ。外見だけでなく、内面も。十六歳までの無垢( むく ) なわたしは、もう存在しない。
　けれどレヤードが死んだことで、わたしの霊魂は隠れていた場所から戻ってきたかもしれない。そこで霊魂を呼び戻そうとしてみたが、無駄だった。仕方がないことだろう。わたしの身体は、毒見をして安全な食事だけをアンブローズ最高司令官に与える道具であり、彼の所有物なのだ。わたし自身のものではない身体に霊魂が戻ってくるはずもない。諦めて、鏡に映る自分から目を離した。
　もつれた髪をなんとか解きほぐし、後ろで長い三つ編みにする。ほんの数時間前まで唯一の望みは死刑の前に清潔な囚人服に着替えることだったのに、今はこうして城の風呂に浸かっているのだから、幸運だと思わなければ。
「長風呂はもう十分だよ！」
　湯の中でうつらうつらとしていたわたしは、マージの大声にびくりとした。
「制服を持ってきたから、着替えな」マージの表情は硬く、非難たっぷりだった。わたしが身体を拭いている間も、苛立( いらだ ) ちを隠さない。
　下着と一緒に手渡されたのは毒見役の制服だった。毒見役の制服は、幅広の帯がついた赤いサテンのシャツと黒いズボンで、シャツの両袖には肩から袖まで小さな黒いダイヤ模様がついている。でも、マージがくれた制服は明らかに男性用だ。身長が一メートル六十二センチで栄養失調のわたしは、ままごとでお父さんの役をしている子どものように見える。仕方がないので帯を三重に巻き、シャツの腕とズボンの裾をまくった。
　マージは鼻で笑った。「ヴァレクには食事を与えたら部屋へ連れていけと言われたけど、これじゃあ、先に裁縫師のところに行かなくちゃだめだな」浴室を出かけたマージは立ち止まって口をへの字に曲げた。「ああ、ブーツもいるのか」
　わたしは迷った子犬のように従順にマージについていった。
　裁縫師のディラナは、わたしの格好を見て朗らかに笑った。ふっくらした頬と小さな顎をブロンドの巻き毛が包み、蜂蜜色の瞳と長い睫毛( まつげ ) が彼女の美しさを際立たせている。
　ようやく笑いを押し殺すと、ディラナは言った。「厩舎( きゆうしや ) の職員のズボンと同じ色だし、厨房( ちゆうぼう ) の女給も赤いシャツを着るから、ちゃんと探せばサイズにあった制服が見つかるわ」
　その努力すらしなかったのをディラナにたしなめられ、マージはへの字の唇をさらに曲げた。
　ディラナは若いくせに、まるで孫の相手をするお婆( ばあ ) さんのようにこまごまと世話を焼いてくれる。心が温まり、彼女と友だちになれたらと思った。でも、ディラナにはたくさん知り合いがいて、交際を求める男性にもいつも取り囲まれているだろう。
　わたしのサイズを測り終えると、ディラナは部屋にある赤、黒、白の服の山からわたしに合った制服を探し始めた。
　イクシアでは、最高司令官が政権を握ったときから制服制度が義務になり、ひと目で身分がわかるようになっていた。ここでは成人全員が働かねばならず、働く者は制服を着なければならないのだ。
　イクシア領は八つの軍管区に分かれており、それぞれの区域を将軍が統治している。将軍を含む高級官僚と顧問官の制服はふつう全身が黒で、襟には階級を示すダイヤ模様が縫いつけてある。そのほかの職種についても、制服のデザインは領内で統一されていて、シャツの袖かズボンの脇に並んだダイヤ模様で職種や階級がわかるようになっている。
　所属する軍管区は使う色で識別できる。黒と赤は最高司令官の色で、城の兵士と使用人の制服は、白、黒、赤三色のいずれかを組み合わせたもの。ほかの軍管区ではそれぞれ異なる色の組み合わせを使っていて、たとえば家政婦の黒い制服のエプロンに紫色のダイヤ模様がついていたら、第三軍管区の者だとわかる仕組みだ。
　ディラナは探してきた服を手渡して部屋の端にある更衣場を示した。「さっきのよりはサイズが合うと思うわ」
　着替えているとき、「ブーツも必要よ」というディラナの声が聞こえた。自分に合った服に着替えて少し威厳を取り戻したわたしは、さっきまで着ていた制服を彼女に手渡した。
「これ、以前毒見役だったオスコヴのだわ」ディラナの顔は一瞬悲しげに曇ったが、辛( つら ) い思い出を捨て去るかのように首を横に振った。
　最高司令官の毒見役と親しくなるのは、相手にとって感情的に大きなリスクなのだ。そう気づいて、ディラナと友だちになるという夢は砕けた。
　ふと、ブラゼルの館に残ったメイとカーラの姿がまぶたに浮かび、寂しさで胸が潰れた。カーラの三つ編みを直したりメイのスカートの歪みを直したりしたくて、指が震える。
　けれども、今手の中にあるのはカーラの絹糸のような赤毛ではなく、ただの制服だ。ディラナはわたしを椅子に座らせてその前に屈( かが ) みこみ、靴下の次にブーツを履かせた。ブーツは柔らかな黒の革製で、ふくらはぎの途中で折り返していた。ディラナは、ズボンの裾をブーツにたくしこみ、わたしを助け起こした。
　ずっと靴を履いていなかったので靴擦れができるのではないかと思ったが、ブーツはクッションがきいていて、足にぴったりと合っていた。これまで履いた中で、最高のブーツだ。寂しさを少しの間だけ忘れて、にっこりと笑った。
　ディラナもそれを見て微笑み返した。「わたしは足を測らなくても必ずぴったりのブーツを見つけられるのよ」
　マージはわざとらしく咳払( せきばら ) いした。「ランドのブーツのサイズは間違えたくせに。かわいそうなランドときたら、あんたにぞっこんだから文句も言えないでいるんだ。そのせいで厨房で足を引きずってるじゃないか」
「マージの言うことは気にしなくていいのよ」わたしに向かってそう言うと、ディラナはマージのほうを向いた。「ほかに仕事があるんじゃないの？　意地悪してると部屋に行ってスカートを全部短くするわよ」そして、気立てよくわたしたちを追い払った。
　それからマージに連れられて使用人専用の食堂に行った。与えられたのは少量のスープとパン。スープがあまりにもおいしくて、いっきに平らげておかわりを求めた。
「だめだ。たくさん食べたら胃が受けつけない」というマージの冷たい返事に心残りを覚えながらも器をテーブルに置き、彼女のあとについてわたしが使う部屋に向かった。
「日の出には出勤できるようにしておきな」
　遠ざかるマージの背中を見送ったあと、あてがわれた部屋をじっくり眺めた。
　マットレスに染みがついた金属製のベッド、飾り気のない木製の机と椅子、室内用便器、衣装箪笥( だんす ) 、角灯( ランタン ) 、ストーブ。ひとつだけある小さな窓は閉じていて、灰色の石壁にはなんの飾りもなく、殺風景だ。硬いマットレスは、座ってもほとんど凹( くぼ ) まなかった。それでも地下牢とは比べものにならないほど居心地がいいのに、なぜか満足できない。
　柔らかいものが何もないからだ。心に浮かんでくるのは、ヴァレクの金属のように冷たい表情や、マージの批判的な態度や、制服の強い色彩と無味乾燥なデザインだけ。ふかふかした枕や毛布の手触りが恋しい。わたしを傷つけない、柔らかなものにしがみつきたい。
　衣装箪笥に替えの制服をかけると、窓に向かった。窓台は腰をかけられるほど広く、鎧戸( よろいど ) には鍵がかかっていたが、掛け金は内側にある。震える手で掛け金を外して鎧戸を大きく開くと、眩( まぶ ) しい光がいっきに差しこんできた。手で覆い、目を細めて外を見て、目の前の光景に驚いた。わたしの部屋は城の一階なのだ！　それも、窓から地面までたったの一メートル五十センチほどだ。
　この部屋と厩舎の間には、犬舎と馬のための訓練場がある。厩舎の職員と犬の訓練員はわたしが逃亡しても注意を払わないだろう。簡単に窓から飛びおりて逃げることができる。二日以内に死ぬという現実を別にすれば、心そそられる。
　今ではなくとも、いつかきっと。二日間の自由に死ぬ価値があると思えるときがきたら。
　それくらいの希望は持ってもいいだろう。




　　　3


「もたもたするな！」レヤードの鞭( むち ) が肩に食いこんだ。
　焼けつくような痛みによろめきながら、間髪いれずに襲いかかる革の触手から身をかわした。けれども、手首をロープで柱に繋( つな ) がれているので自由に動けない。続けざまに鞭がしなり、腕や脇腹の皮膚が裂ける。ぼろぼろに破れたシャツは肌をかばってはくれない。
「止まるな、もっと速く動け！」
　そのとき、頭の中で誰かが慰めるようにささやいた。"逃げなさい。暖かくて、思いやりに満ちた場所に。心を遠くに送って、肉体を解放しておやり"
　レヤードやブラゼルのものではない滑らかな声だ。もしかして、救いの神だろうか？　死を受け入れてしまえば、拷問から逃れて楽になれる。心引かれるが、生きる望みを捨てたくはない。
　飛んでくる鞭を避けることだけに神経を集中させ、革紐( かわひも ) が音をたてて空気を切るたびにハミングバードのように素早く右へ、左へ走りまわる。けれども、疲労は限界に達していた。これ以上は動けない......。そう思ったとき、わたしの全身が細かく震え始めた。

　目覚めたら、周囲は真っ暗だった。汗で身体( からだ ) がぐっしょり濡( ぬ ) れ、制服が絡みついている。夢の中の震動は、何かを打ちつける音に変わっていた。誰かが扉を叩( たた ) いているようだ。そういえば、寝ている間に侵入されないように、扉の取っ手を椅子の背で固定しておいたのだ。扉が叩かれるたびに椅子がカタカタと揺れる。
「起きてます！」大声で答えると、ようやく喧騒( けんそう ) が静まった。ドアを開けると、角灯( ランタン ) を掲げたマージがしかめっ面で立っている。大急ぎで制服を着替えて廊下に出た。
「出勤は日の出の時間かと......」言いかけて、非難たっぷりのマージの視線に口をつぐんだ。
「もう日の出だよ」
　マージについて迷路のような城の廊下を歩いているうちに、しだいに空が白んできた。わたしの部屋は西に面しているから、朝の気配を感じるのが遅いのだ。明日からは暗いうちに起きるようにしよう。マージが角灯を吹き消したとき、パンが焼ける匂いが漂ってきた。久しぶりに嗅ぐ甘い香りに、自分がいかに飢えているか思い知らされた。
「朝食はいただけるんですか？」物乞いをしているような自分の声が情けない。
「あんたの食事はヴァレクが与えることになってる」
　毒が盛られた朝食を想像したとたん、食欲は吹き飛んだ。《蝶の塵》の解毒剤をすぐに飲まないと、毒が全身に行き渡って手遅れになってしまうかもしれない。不安と焦りで胃がこわばり、ヴァレクの執務室に着いたときには気を失いそうだった。
　部屋に入ると、ヴァレクが湯気を立てる料理の皿を並べているところだった。昨日見たときにテーブルの上に散らばっていた書類は、隅に積み上げられている。ヴァレクが無言で指し示した椅子に座り、目で解毒剤を探した。
「ところで君は......」ヴァレクは言葉を止めて、わたしの顔を凝視した。
　わたしも、しっかりと見つめ返した。絶大な権力を持つ防衛長官の視線に怯( おび ) えて、冷静さを失わないように。
「風呂と制服だけで、これほど見違えるものなのか」ヴァレクは上の空でベーコンを齧( かじ ) りながら、ひとりごとをつぶやいた。「将来役に立つかもしれない情報だな。覚えておこう」
　すぐさま事務的な態度に戻ったヴァレクは卵とハムがのった皿をふたつ並べた。
「さあ、訓練だ」
　出勤したときに解毒剤をくれる約束では？　ヴァレクにはどうでもいいことかもしれないが、わたしにとっては命にかかわる問題だ。「解毒剤を先にください」不安と焦りにかられて、うっかり口を滑らせた。
　ヴァレクは何も答えない。
　怒らせてしまったのだ。どうしよう？　絞首台に戻されるのだろうか？
　長引く沈黙に身をすくめていると、ようやくヴァレクが口を開いた。
「症状は今日の夕方まで現れないから大丈夫だ」
　軽くあしらいながらも、ヴァレクは戸棚から大きな瓶を取り出してピペットで白い液体を量り始めた。その間に解毒剤が収納されている場所をそっと観察しようとしたが、ヴァレクはわたしの視線に気づいていて、奇術師のような手ぶりで一瞬にして戸棚の鍵を隠してしまった。ピペットをわたしに手渡すと、ヴァレクはテーブルの向かい側に座った。
「それを飲み終えたら、すぐに今日の訓練を始める」
　一滴も残さないように解毒剤を口に絞り出した。身震いするほど苦い。わたしの手からピペットを取り上げると、ヴァレクは青い瓶を差し出した。
「嗅いでみなさい」
　瓶の中には白い粉末が入っていた。見かけは砂糖に似ているが、紫檀( したん ) のような匂いがする。ヴァレクは目の前にあるふたつの皿を指し示し、どちらに毒が盛られているか当てるよう命じた。猟犬が獲物の匂いを探すときのように料理に鼻を近づけると、左の皿のほうからかすかに紫檀の香りがした。
「左ですね」
「よろしい。最高司令官への食事からこの匂いがしたら手をつけずに、拒否しなさい。これは《チグタス》という名の毒で、たとえ一粒でも摂( と ) ったら一時間以内に死ぬ」
　ヴァレクは毒入りの皿を取り去り、残った皿を指差した。
「君の朝食だ。まずは職務のために体力を取り戻す必要があるから、しっかり食べなさい」
　何時間もこの調子で毒を嗅ぎ分けたので、一日の終わりには頭痛と眩暈( めまい ) がしてきた。一度にたくさんの香りと名前を教わると、間違って覚えてしまうかもしれない。学んだことを記録しておきたかった。
「紙と書くものを貸してもらえませんか？」
　ヴァレクは動きを止めて、わたしをまじまじと見た。
「君には驚かされてばかりだな。これまでの毒見役はろくに読み書きができなかったから考えもしなかった。そういえば、ブラゼル将軍は孤児にも教育を与えることで知られていたな。忘れていたよ」ヴァレクは雑記帳と筆記具をわたしの前に放り投げた。
「部屋に持って帰っていい。今日の訓練はここまでだ」
　無言で雑記帳と筆記具をかき集めながら、自分の迂闊( うかつ ) さを呪った。ヴァレクの冷たい表情を見れば、何を考えているのかすぐわかる。ブラゼル将軍は、親を失って路頭に迷っている孤児を救い出して衣食住だけでなく教育まで与えたのに、わたしは恩に報いるどころか、彼のひとり息子を殺したのだ。外面とはまったく異なるブラゼルとレヤードの真の姿をヴァレクに話しても、絶対に信じてはくれないだろう。
　ブラゼル将軍の孤児院は、ほかの軍管区の将軍たちの笑い種( ぐさ ) なのだ。軍隊が政権を握ったのは十五年前だが、「あれからブラゼルはボケて甘くなった」と揶揄( やゆ ) されている。この誤解は、ブラゼルの思う壺( つぼ ) だった。慈善家の好々爺( こうこうや ) を危険な競争相手とみなす者はいないので、第五軍管区の指導者としてのブラゼルの地位は安定している。
　執務室を出ていくとき、分厚い扉に複雑な仕組みの錠前が三つもついているのに気づいた。上の空で鍵を触りながら戸口でもじもじしていると、ようやくヴァレクが尋ねた。
「今度はなんだ？」
「部屋への戻り方がわからないんです」
　ヴァレクは、飲みこみが悪い子どもに話しかけるように言った。
「最初に見かけた家政婦か厨房( ちゆうぼう ) の給仕に訊( き ) けばいい。この時間帯なら、そのあたりにたくさんいるから。西翼一階の使用人宿舎だと言えば、道順を教えてくれる」
　道を尋ねた厨房の女給は、マージとは違って、気立てがよくておしゃべりだった。洗濯室に立ち寄ってベッドのシーツまで探してくれる面倒見のよさに甘えて、浴場とディラナがいる裁縫部屋への道順も尋ねておいた。あの部屋にある大量の制服が、いつか役立つチャンスが来るかもしれない。
　部屋に戻ってすぐ窓の鎧戸( よろいど ) を開けた。すでに日は沈もうとしている。大急ぎで机に向かい、薄暗くなっていく夕日を角灯代わりにして、今日学んだことをつぶさに書きとめた。歩いた道順も書いてみたが、地図にするには情報が足りない。城の内部を探索したかったけれど、もうくたくただ。ヴァレクが言うとおり、まずは体力を取り戻さなくては。偵察はそれからだ。でも、機会ができるまで生き延びることができるのだろうか？

　早朝にヴァレクの執務室に赴いて一日中訓練をする日がしばらく続いた。二週間も毒の匂いを嗅ぎ分けたおかげで、わたしの臭覚は以前よりずっと鋭くなっている。平穏な日常に慣れてきた十五日目の朝、なんの前触れもなくヴァレクが言い渡した。
「体力がついたようだから、そろそろ毒見の実践訓練に移る」
　顔から血の気が引いていくのがわかった。
「最初に試すのは最も致命的な毒だ。この毒を乗り越えることができたら、ほかの毒では死にはしない。弱い毒から始めて最後に毒見役が死んだら、それまで訓練に費やしたわたしの時間が無駄になる。合理的なシステムだ」
　ヴァレクは、赤い色の細長い瓶を取り出した。中身を眺める彼の目は、賛美するように輝いている。
「昔、夫に裏切られた妻がよく使ったので《あなた、一杯いかが( ハブ・ア・ドリンク・マイ・ラブ ) ？》という名前がついた猛毒だが、ふだんは《マイ・ラブ》と呼ばれている。即効性があって、飲むとすぐさま全身をおかし、著しい幻覚とパラノイアを引き起こす」
　ヴァレクは、湯気の立つティーカップに二滴だけ液体を落とした。
「これより多い量を与えたら、確実に死ぬ。少量の場合には生存の可能性はあるが、生き延びても数日間は見当識障害から回復しない」
　なぜ、毒見役に選んで生きる希望を与え、二週間も訓練したうえで命を奪うのか？　すでに毎日死と背中合わせの職務なのに、理屈に合わない。わたしは怒りに震えながら反論した。
「最高司令官が食事を口に入れる前に試すのが毒見役の役割ですよね。即座に効果を現す猛毒の《マイ・ラブ》が盛られていたら、わたしが一口食べて、その場に倒れて死ぬだけじゃないですか。なぜ、訓練で死ぬかどうか試さなくちゃならないんですか？」
　もっと言いたいことはあったが、ヴァレクの厳しい視線に圧倒されて口をつぐんだ。
「毒見役の仕事は、もっと複雑なものだ。最高司令官の食事に含まれた毒物を識別すれば、毒殺を企( たくら ) んだ者を推測するのも容易になる。口に含んでから気を失うまでの短時間に君が《マイ・ラブ》と叫ぶことができたら、わたしはその情報を元に容疑者を絞りこめる。まず、この毒を好んで使う暗殺者は知られている。次に、《マイ・ラブ》は南部のシティア領でしか採れない。軍事政権になる前はイクシア領でもかなり自由に入手できたが、国境が閉鎖されてからは、闇市で買う財力がある者は限られている」
　ヴァレクの言葉に、壁に貼られた地図に目をやった。わたしがいた第五軍管区はシティア領に隣接しているが、ふたつの領土が交流していた時代は知らない。シティア領も、そこだけに生えている毒草も、想像できなかった。
「イレーナ、君の職務は極めて重要なんだ。そうでなければ、わたしは訓練にこれほどの時間をかけたりはしない。抜け目がない暗殺者は、毒見役の行動を何日も克明に観察して一定のパターンを発見する。例えば、肉を試食するときにいつも左側の端だけを切り取るとか、飲み物をかき混ぜないとか、そういった癖だ。液体の表面だけをすする毒見役がいたら、底に沈むタイプの毒を使ってターゲットを殺すことができる。君が学ぶべきことはたくさんあるが、《マイ・ラブ》の味見はその導入編にすぎない」
　ヴァレクは書類の山の間を優雅に動きながら説明した。踊りの名手と間違えそうな流麗な身振りは、実績を積んだ暗殺者のものだ。机に飾られた豹( ひよう ) の獰猛( どうもう ) な美しさとヴァレクの姿が重なり、身体の芯まで冷たくなった。
「毒を飲んだあとはマージが部屋に連れていって君の世話をしてくれる。《蝶の塵》の解毒剤も毎日マージに渡すから心配するな」
　いつの間にか執務室に来ていたマージは、あたかも死刑執行人のようにわたしが毒を飲むのを待ち構えている。
　わたしはお茶が入ったカップを両手で持ち上げた。緊張で冷たくなった指先が少しだけ温まる。飲む前に座ったほうがいいのだろうか？　それとも、横になったほうがいいのか？　まわりを見渡したが、何も目に入ってこない。腕がぴりぴりと痺( しび ) れる。どうやら、息をするのを忘れていたようだ。
　深く息をついて決意した。逃れる術( すべ ) はないのだから、せめて威厳だけは保とう。
　ヴァレクに向かって乾杯するようにカップを掲げ、いっきに中身を飲み干した。
「酸っぱい林檎( サワーアツプル ) の味ですね」
　ヴァレクが頷( うなず ) くのは見えたが、カップを置く暇もなく、まわりの世界が溶け始めた。ぐにゃりと歪( ゆが ) んだマージがわたしに向かって波打つ。巨大に膨らんだマージの眼窩( がんか ) から花が咲いたかと思ったら、今度は頭がどんどん縮んで身体のほうが大きくなり、部屋いっぱいに膨らんだ。灰色の壁からは腕が生えてきて、その腕に鷲掴( わしづか ) みにされ身動きできずにいると、床から現れた幽霊に小突かれて嘲笑( あざわら ) われた。マージの化け物を押しやろうとしたが、全身に絡みつき、耳から穴を掘って脳を叩く。化け物が耳元でささやく。「人殺し。あばずれ女。どうせ、寝ている間に首をかき切ったんだろう。そのほうが簡単だからな。血がシーツに染みていくのを見て、楽しんだのかい？　おまえなんか人間じゃない。汚らしいドブネズミだ」
　忌まわしい声を止めたくてあがくと、化け物はふたつに分裂して黒と緑の玩具( おもちや ) の兵隊になり、わたしを押さえつけた。
「どうせこいつは毒死するさ。もし死ななかったら、おまえたちが好きにしていいよ」
　マージの化け物がそう言うと、兵隊はわたしを闇に突き落とした。

［著］ジェイソン・モット
［翻訳］新井ひろみ
　アメリカの静かな田舎町。50 年前に8 歳の息子を亡くし、悲しみに寄り添いながら暮らすハロルドとルシールの前に、ある日突然、亡くなった息子ジェイコブが当時の姿のまま現れる。そばかすの散る頬、癖のついた髪、そして忘れる日のなかった茶色の瞳を涙で潤ませたこの子は――本当に息子なのか？　戸惑いを隠せないなか、世界では次々に"死者が生き返る"という謎の事件が起き始め......。

　　　1


　その日、ハロルドが玄関のドアを開けると、仕立てのいいスーツを着た黒人男性が微笑( ほほえ ) んでいた。とっさに猟銃に手を伸ばそうとして、ハロルドは思いだした。あれはもう何年も前に売り払ったのだった。巡回牧師が巻きこまれる事件が起きたり、猟犬の虐待が世間を騒がせたりしたために、ルシールに説き伏せられたのだ。
「どちら様？」ハロルドはまぶしさに目をすがめた。強い日差しが、男の肌の色をいっそう濃く見せている。
「ミスター・ハーグレイブですね？」
「そうだが」
「どなた、ハロルド？」ルシールが声をかけた。彼女はリビングルームにいて、テレビ相手にいらついていた。ニュースキャスターは、初の帰還者であるエドマンド・ブライズの話をしている。ふたたびの生を得た彼の暮らしが、以前とどんなふうに違っているかについて。
「二度目のほうが幸せでしょうか？」キャスターはまっすぐカメラを見据えて問いかけ、それに答えるという重荷を視聴者に負わせる。
　庭のオークの葉は風にそよいでいるが、太陽が低いところにあるために、枝の隙間から陽光が水平にハロルドの目に飛びこんできた。額に手をかざしてもなお、男性と男の子の姿は緑と青の背景に浮かぶシルエットでしかなかった。緑は敷地のはずれに並ぶ松、青はさらに先まで広がる雲ひとつない空だ。子どもは八つか九つといったところだろうが、そのわりには小柄だった。
　ハロルドは瞬( まばた ) きをした。いくらか目が明るさに慣れてきた。
「どなたなの、ハロルド？」最初の問いに答えが返ってこないとわかると、ルシールはもう一度尋ねた。
　戸口に立ちつくしたハロルドは、少年を見つめたままハザードランプのように瞬きを繰り返した。頭の奥のほうでシナプスが働きだす。目覚めた脳は、男性の連れている少年が誰であるかをハロルドに伝えてくる。いや、それは間違いだ、計算してみろ、とハロルドは脳に命じる。それでもやはり、出てくる答えは同じだった。
　リビングルームではテレビカメラが、振り立てられるこぶしや、叫ぶ口や、プラカードを掲げ声を張りあげる人々を映しだしている。そして、居並ぶ兵士たち。権力と武器、両方を持つ者に特有の、厳( いか ) めしい姿だ。画面の中央はエドマンド・ブライズが住む小さなタウンハウスだが、カーテンが引かれている。本人が家の中のどこかにいるのは、確かなようだ。
　ルシールはかぶりを振った。「信じられない」つぶやいてから、また夫に言う。「ねえ、お客様はどなた？」
　ハロルドは少年をしげしげと見た。小柄で、色白で、頬にはそばかす。ぼさぼさの茶色い髪。古くさいデザインのＴシャツとジーンズ。瞳には深い安堵( あんど ) の色が宿っている。動かない瞳ではない。きらきら光って、うっすら涙が浮かんでいる。
「四本足で、ぐすんぐすんいうもの、なーんだ？」少年は震える声で言った。
　ハロルドは咳払( せきばら ) いをした――自分の咳すら、現実味がなかった。「わからないな」
「風邪をひいた牛！」
　老人の腰に飛びつくなり、「パパ！　パパ！」と少年は泣きじゃくりはじめた。
　肯定も否定もできないまま、ハロルドは背中をドア枠にしたたかぶつけ、転倒しかけた。が、長く眠っていた父性本能から、とっさに少年の頭をぽんぽんたたき、「よしよし」とささやいた。「よしよし」
「ハロルド？」ルシールはついにテレビから目を離した。玄関先で何かよからぬ事態が持ちあがったに違いないと思った。「ハロルド、どうしたの？　誰が来たの？」
　ハロルドは唇を舐( な ) めた。「ああ......その......」
"ジョセフだ"とでも言えればよかった。
「ジェイコブだ」彼はようやく言った。
　ルシールにとって幸いだったのは、そばにソファがあったことだった。気絶した彼女の体をそれが支えてくれたのだから。

　ジェイコブ・ウィリアム・ハーグレイブは一九六六年八月十五日に死亡した。奇( く ) しくもその日は、彼の八回目の誕生日だった。それからの数年間、町の人々は眠れぬ夜などジェイコブの死についてささやきあった。寝返りを打って連れ合いを起こし、人の命の儚( はかな ) さや、神に生かされていることのありがたさについて、夫婦でひそひそと語りあった。ときには一緒にベッドを出て子ども部屋の戸口にたたずみ、眠るわが子を眺めながら、幼い命を早々にこの世から連れ去る神の真意を推し量ったりもした。なんといっても、小さな町で生まれ育った南部人たちだ。その種の悲劇を神と結びつけて考えるのは当然だった。
　ジェイコブの死後、彼の母であるルシールは何度も言った。前の晩にあんな夢を見たから、あの日は何か起きるような気がしてならなかった、と。
　それは、自分の歯がぼろぼろ抜け落ちる夢だった。それは近いうちに誰かが死ぬってことだよと、母親から教えられていた。
　誕生日パーティーでのルシールは、終始、息子とその友だちをはじめとする招待客全員に気を配っていた。落ち着かない雀( すずめ ) のようにあちこちへ移動しては、楽しんでいる？　おなかはいっぱいになった？　と尋ね、スマートになったわね、お子さん大きくなったわね、と言葉をかけてまわった。そして、なんていいお天気なのかしらと、何度も言った。その日はあらゆるものが輝き、あらゆるものが生き生きと息づいていた。
　不安ゆえに、ルシールのホステスぶりは完璧なものになった。子どもたちはごちそうを堪能し、大人はひとり残らず会話を楽しんだ。控えめなメアリー・グリーンまで、ルシールの説得を受けて一曲歌う約束をした。彼女の歌声は砂糖菓子よりも甘くつややかだった。おまえも大きくなったらきっと惚( ほ ) れるぞ、と夫のハロルドはジェイコブをよくからかったものだった。いい一日だった、あの日は。本当にいい日だった。ジェイコブが消えるまでは。
　あんなふうに誰にも気づかれずに消えられるのは、子どもならではだった。三時から三時半のあいだに――のちにハロルドとルシールは警察にそう語った――本人にしかわからない理由から、ジェイコブは庭を出て南へ向かい、松の並木を過ぎて森へ入り、川まで行った。そうして、親に断りもなく溺れ死んだ。

　管理局の係官が現れる数日前に、ハロルドとルシールは話しあったばかりだった。もしもジェイコブが"帰還者"として戻ってきたらどうするか、と。
「彼らは人間じゃないのよ」ルシールは両手をもみあわせて言った。ふたりはポーチにいた。大事なことはすべてポーチの上で起きる。
「突っぱねるわけにいくか」ハロルドは語気荒く言った。話し合いは瞬く間に口論になった。
「だって、人間じゃないのよ」彼女は繰り返した。
「人間じゃないなら、なんなんだ？　植物か？　鉱物か？」ハロルドの唇が煙草( たばこ ) を求めてむずむずした。煙草を吸いながらだと、妻と口論をしても優位に立てる気がするのだ。彼女が夫の喫煙をとがめる本当の理由は、それではないかとハロルドはにらんでいた。
「ふざけるのはやめて、ハロルド・ナサニエル・ハーグレイブ。これは真面目な話なんだから」
「おれがふざけてる？」
「ええ、そうよ！　あなたは軽薄なのよ！　考えなしに、ぺらぺらぺらぺら、無駄口をたたいてばかり」
「忘れたんだな？　きみは昔、おれのことを"能弁"と言ったんだ。それがいつから"無駄口"になった？」
「ばかにしないで。まだ惚( ぼ ) けちゃいないわ。むしろ、前より冴( さ ) えてるかもしれない。話をそらさないでちょうだい」
「ふん！　軽薄とはな」吐き捨てるようにハロルドが言うと、ポーチの手すりに唾が飛んだ。
　かまわずルシールは話を続けた。「あれがなんなのか、わたしにもわからない」立ちあがり、また座る。「あなたやわたしと違うのだけは確かよ。あれは......あれは......」間を置いて、頭の中で言葉を準備してから、彼女は慎重に口を開いた。「悪魔よ」言ったとたんに自身がひるんだ。出した言葉がくるりと向きを変えて、襲いかかってきたかのようだった。「わたしたちを苦しめるために現れたんだわ。あるいは、わたしたちをそそのかして邪悪な行いをさせるためかもしれない。この世の終わりよ。"死者がよみがえるとき"が来たのよ。聖書にあるとおりに！」
　軽薄と言われたことにこだわっているハロルドは、鼻を鳴らした。手がひとりでにポケットへ行く。「悪魔？」ライターに手が触れると頭が働きだした。「悪魔なんて迷信だ。狭い心と貧相な想像力の産物にすぎない。辞書から駆逐されるべき単語がひとつある――悪魔だ。ふん！　それこそ軽薄な言葉だ。あの"帰還者"とやらは、まったくそんなんじゃない。いいか、ルシール・アビゲイル・ダニエルズ・ハーグレイブ、彼らは人間だ。歩きもすれば、キスもする。そんなことができる悪魔には、ついぞお目にかかったことがないね。そうそう、きみと結婚する前、とある土曜の夜にタルサで出会った金髪娘。うん、いまから思えばあれは悪魔だったかもしれない。少なくとも、悪女ではあった」
「いいかげんにして！」ルシール自身が驚くほどの大声だった。「そんな言い方をされて、黙っているわけにはいかないわ」
「どんな言い方だ？」
「現れたとしても、それはジェイコブじゃないのよ」ことの深刻さがふたたび胸に迫り、ルシールの口調はゆっくりになった。「あの子は神に召されたの」膝に並べた両手が、青白い小さな握りこぶしになる。
　静寂が訪れた。
　ほどなくそれは破られた。
「どこに書いてあるんだ？」
「なんの話？」
「聖書の、どこだ？」
「だから、何が？」
「死者が墓からよみがえるとかなんとかってのは、どこに書いてあるんだ？」
「ヨハネの黙示録じゃない！」あまりにもばかげた問いにルシールはあきれ、両手を広げた。「黙示録にちゃんと書かれているわ！　この世の終わりに死者がよみがえるって！」両手がこぶしになっているのがちょうどよかった。映画の登場人物たちがするように、ルシールは誰にともなくこぶしを振りまわした。
　ハロルドは笑った。「黙示録のどこだ？　何章、何節に書いてある？」
「うるさいわね。書かれているのは確かなんだから、それでいいじゃないの。もう、黙っていて！」
「はいはい、わかりましたよ。無駄口ばかりたたく軽薄な男にはなりたくないからね」

　ところが、いざ悪魔が――昔と変わらぬ愛らしい姿をして、茶色い瞳を涙で潤ませて、長く親と離れていた心細さから解放された喜びと安堵をあらわにして――玄関口に現れてみると、ルシールはいったんは気を失ったものの、そのあと彼女の心は蝋燭( ろうそく ) さながら見る見る溶けてしまった。役所から来た、きちんとした身なりの男は冷静だった。このところ立てつづけに同じような場面を目撃してきた彼は、いつもの微笑をその顔に浮かべた。
「支援グループがあります。帰還者のための支援グループです。帰還者の家族向けのもありますよ」そう言って、また微笑む。
「発見されたのは――」彼はさらに言った。すでに名乗ってはいたが、ハロルドもルシールも人の名前をなかなか覚えられなくなっているところへもってきて、死んだ息子が突然、戻ってきたのだ。だからふたりにとって、彼は単に"役所の人"だった。「中国の北京にほど近い小さな漁村でした。魚を獲( と ) ろうとしていたのかなんなのか、川べりにひざまずいていたそうです。彼が理解できるような英語をしゃべれる村人はいませんから、みんな北京語であれこれ尋ねました。名前はとか、どうやってここまで来たのかとか、家はどこかとか。迷子を見つけたら誰もが尋ねるようなことですね。言葉は通じませんでしたが、女の人たちが彼を慰めました。泣きだしていましたからね。そりゃあ、泣きもしますよ」男はまたにっこり笑った。「全然知らない場所なんですから。ともかく村人たちは彼をなだめ、英語のできる役人を探してきて......」彼はダークスーツに包まれた肩をすくめ、あとは大した話ではないことを示した。そして、つけ加えた。「似たようなことがあちこちで起きているんです」
　男はまた言葉を切ると、生き返った息子を撫( な ) でまわすルシールを、作りものではない笑みを浮かべて見守った。彼女はジェイコブを胸に抱きしめ、頭のてっぺんに唇をつけた。それから両手で彼の頬を包みこむと、泣き笑いをしながらその顔にキスの雨を降らせた。
　ジェイコブは愛らしく笑い、身をよじりはするものの、されるがままになっている。母親にキスされたところを手でごしごし拭ってもおかしくない年ごろなのに、彼はそうしなかった。
「今日は記念すべき日ですね」役所から来た男が言った。

　カムイ・ヤマモト

　真鍮( しんちゆう ) のベルを軽やかに鳴らして、男が店のドアをくぐった。外の車はガソリンを入れ終えて出ていくところだったから、運転手は彼に気づいていなかった。カウンターの向こうにいた赤ら顔の太った男が、ひょろりと背の高い男との会話をやめ、ふたりして彼を見つめた。フリーザーの作動音だけが低く響いている。深々とお辞儀をするカムイの後ろで、今度は閉まるドアのベルが鳴った。
　カウンターの中の男たちはまだ何も言わない。
　カムイは微笑みを浮かべてもう一度お辞儀をした。「許してください( フオーギブミー ) 」彼がそう言ったので、男たちはひどく驚いた。「降伏します( アイサレンダー ) 」カムイは両手をあげた。
　赤ら顔の男が何か言ったが、カムイには理解できなかった。男ふたりは外を気にするそぶりを見せながらぺらぺらとまくしたて、やがて赤ら顔がドアを指差した。カムイはそちらを向いたが、見えるのはひとけのない道路と、昇りはじめた太陽だけだった。「アイサレンダー」彼は繰り返した。
　拳銃は、森のはずれの木の根元に埋めてきた。ほんの二時間ほど前、カムイはそこで目を覚ましたのだった。軍服の上着と戦闘帽も置いてきたから、早朝のガソリンスタンドに現れた彼は、下着のシャツに、磨きこまれた軍靴という格好だった。すべてはアメリカ人に殺されないようにするためだった。「山本です」日本語で名乗り、また言った。「アイサレンダー」
　赤ら顔の男がいちだんと大きな声を出した。痩せたほうも加わって、ふたりでわめきながらドアを指差す。「アイサレンダー」カムイは繰り返した。男たちの口調が激しくなっていくのが恐ろしかった。痩せたほうが、カウンターにあった炭酸飲料の缶をつかんでカムイに投げつけた。狙いがはずれると男はまた怒鳴り、ドアを指差し、次に投げるものを物色しはじめた。
「サンキュー」カムイは必死に言ったが、それが自分の言いたいことを表す英語でないのはわかっていた。英語はほとんど知らないのだ。彼は後ずさりした。カウンターの下に手を伸ばした赤ら顔が何かを見つけ、うなりながらそれを投げてきた。缶が左のこめかみに当たり、カムイは背中からドアに倒れこんだ。真鍮のベルが鳴る。
　続け様に缶を投げる赤ら顔の横で、痩せたほうも投げるものを探しながら怒鳴りつづける。カムイはふらつく足でガソリンスタンドをあとにした。自分は丸腰であって、単に投降したいだけであることを示すために、両手を高く掲げた。耳の奥で心臓がどくどくと脈打っていた。
　すっかり姿を現した太陽が、町並みを柔らかなオレンジ色に染めている。平和な光景だ。
　こめかみから血を滴らせながら、カムイは両手をあげて通りを進んだ。「アイサレンダー！」人々よ目覚めよとばかりに、叫んだ。願わくは、その人々が自分を生かしてくれますように。

　たとえ死からよみがえった者であっても、転入者があれば、役所では当然ながら事務作業が発生する。国際帰還者管理局には、各国から財政援助の申し出が引きも切らなかった。管理局とのコネ を確かなものにするために、国という国が予備費を費やし、それでも足りなければ債務を負うことさえ辞さなかった。帰還者関連の人や物を調整できる組織は地球上にここしかないからだった。
　だが皮肉なことに、管理局の職員にも一般人以上のことはわかっていなかった。彼らの実質的な活動といえば、帰還者の数を数えることと、家へ送り届けることだけだったのだ。

　ハーグレイブ家の玄関先でそれぞれの気持ちがある程度落ち着き、抱きあったり笑いあったりが収まると――それには三十分近くかかったが――ジェイコブはキッチンへ移され、不在だった年月の分を取り戻そうとするかのように旺盛な食欲を見せた。役所の男はリビングルームでハロルド、ルシールと向きあうと、茶色い革のブリーフケースから書類の束を取りだし、仕事に取りかかった。
「そもそも、この帰還者が死んだのはいつでしたか？」役所の男は――マーティン・ベラミーですともう一度名乗った上で――尋ねた。
「そういう言い方をしないといけないんですか？」ルシールは大きく息を吸い、背筋を伸ばした。息子を抱いたり撫( な ) でたりで乱れてしまった長い銀髪を整え終えた彼女は、急に威厳も見識もある女性に見えはじめた。
「そういう言い方って？」ハロルドが訊( き ) いた。
「"死んだ"という言い方でしょう」ベラミーが答えた。
　ルシールがうなずく。
「あの子が死んだと言って、なぜいけない？」ハロルド自身が思っていた以上に大きな声になった。ジェイコブには聞こえなかったかもしれないが、じゅうぶんに近いところにいる。
「しいっ！」
「あの子は死んだんだ。そうじゃないみたいに取り繕ってもしかたないだろう」ハロルドは無意識のうちに声を落とした。
「ベラミー係官はわかってくださってるわ」ルシールは膝の上で両手をもみ絞り、しきりにジェイコブをうかがい見た。隙間風のひどい部屋に灯( とも ) した蝋燭( ろうそく ) を見るような目で。
　ベラミーは微笑( ほほえ ) んだ。「わかりますよ。皆さん、同じことをおっしゃいますから。わたしがうっかりしていました。では、もう一度、はじめから」彼は質問票に目を落とした。「この帰還者が――」
「出身は？」
「はい？」
「きみはどこの生まれだ？」ハロルドは窓辺で青い空を見あげていた。「その訛( なま ) りから察するに、ニューヨークだな？」
「だとすると、いいんですか？　悪いんですか？」初めて訊かれたような顔をしてベラミーは言ったが、実際はノースカロライナ南支局配属となって以来、アクセントのことで何度同じ質問を受けたかわからない。
「最悪だ」ハロルドは答えた。「しかしおれは寛大な男だ」
「ジェイコブです」ルシールが横から言った。「ジェイコブと呼んでください。あの子の名前はジェイコブです」
「わかりました。申し訳ありませんでした。最初からそうするべきでした」
「ありがとうございます、ミスター・マーティン・ベラミー」なぜか膝の両手がまたこぶしになっていた。ルシールは深呼吸をし、意識して力を抜いた。「ありがとう、マーティン・ベラミー」もう一度言った。
「ジェイコブがいなくなったのはいつでしたか？」ベラミーは穏やかな口調で質問を繰り返した。
「一九六六年八月十五日だ」ハロルドが答えた。そわそわと玄関のほうへ歩きだし、唇を舐( な ) める。両手が、年季の入ったズボンのポケットを探り、年季の入った唇へと移る。安らぎは――煙草( たばこ ) も――見つからない。
　ベラミーはメモを取った。
「どういう状況でした？」

　人々が彼を捜しまわったその日、ジェイコブという言葉は呪文のように繰り返された。一定の間隔を置いて声があがる。「ジェイコブ！　ジェイコブ・ハーグレイブ！」すると別の声がして、さらにそれが次へと伝わっていく。「ジェイコブ！　ジェイコブ！」
　はじめのうちは、いくつもの声が重なりあって恐怖と不安の不協和音を奏でていたが、少年が見つからないまま時間が過ぎるにつれ、喉を無駄に使わないためにも、それぞれが代わる代わる叫ぶようになった。そうするうちに太陽は黄金色になって地平線に近づいていき、まず高い木々の梢( こずえ ) に、それから低木の茂みへとのみこまれていった。
　やがて、誰もがのろのろと歩くようになった。イバラの茂みに分け入って体力を消耗し、少年の身を案じて心労を募らせていた。ハロルドと組んでいるのはフレッド・グリーンだった。「絶対に見つかるさ」フレッドは何度も言った。「おれが贈ったＢＢガンの包みを開けたときのあの子の顔を見たか？　子どものあんなに嬉( うれ ) しそうな顔、見たことあったか？」フレッドは肩で息をした。足は棒のようだ。「絶対に見つかる」また言って、自分でうなずく。「おれたちが見つける」
　すっかり夜になり、灌木( かんぼく ) の茂みと松林に覆われたアルカディアのあちこちで懐中電灯の光が瞬いた。
　川のそばまで来たハロルドは、ルシールを家に置いてきたのは――"ひょっこり帰ってくるかもしれない。ママがいなかったら心細いじゃないか"と言い含めて――正解だったと思った。彼はこのとき直感した。この川で息子は見つかる、と。どういうわけか、それがわかった。
　ハロルドは浅瀬に膝までつかると、ゆっくり歩きだした。息子の名を呼び、足を止め、そのあたりから返事が返ってくるのではないかと耳を澄まし、また進んでは大声で呼んだ。そうやって一歩一歩、前進した。
　ついに発見したとき、息子の体は月と水の光を反射して、美しいまでの銀色に輝いていた。水面の輝きと同じ色だった。
「ああ、神よ」ハロルドがこの台詞( せりふ ) を口にしたのは、このときが最後になった。

　語りながらハロルドは、急に自分の声が年寄りじみたものに思えてきた。感情のこもらない、しわがれた、まさに老人の声。ときおり、しわだらけの分厚い手で、頭皮にかろうじてへばりついている灰色の髪を触るが、その手はしみだらけだし、無様に腫れた指の節にはときどき痛みが生じる。同世代にはもっとひどい関節炎持ちがいるとはいえ、ハロルドの痛みも、若さという財産を失ってしまったことを思い知るにはじゅうぶんだった。いま、こうしてしゃべっているあいだも、腰に軽い痛みがある。
　ないに等しい頭髪。しみで斑( まだら ) になった肌。丸い大きな頭。しわの寄った大きな耳。少しでもましに見えるようにとルシールが選び抜いた服も、ハロルドにはぶかぶかで借り着のようだ。何もかも、当然だった。彼はいまや老人なのだから。
　ジェイコブが――活発な子どものままで――戻ってきたことは、ハロルド・ハーグレイブにおのれの老いを気づかせるきっかけになった。
　夫同様に年を取り、白髪頭になっているルシールは、話すハロルドには目もくれずに八歳の息子を見つめていた。キッチンの食卓でピーカンパイを頬張るジェイコブを見ていると、いまは一九六六年で、変わったことは何も起きていない、これからだって何も起こらない、と思えてくる。ときどき彼女は顔にかかる銀髪を手で払ったが、しみの浮いた薄い手が目に入っても、特になんとも感じなかった。
　ハロルドとルシールは痩せたつがいの鳥のようだったが、ここ数年はルシールのほうがいくらか大きい。というか、ハロルドの縮み方が早いのだ。いまでは夫婦喧嘩( げんか ) をするときには、彼のほうが妻を見あげるし、目方もルシールは夫ほど減っていなかった。そして、そんなに痩せるのは煙草のせいだと夫を責める。彼女は服のサイズがいまもぴったりで、細くて長い腕の動きは敏捷( びんしよう ) だ。それに比べてハロルドは、だぶだぶの袖に腕が隠れている分、よけい貧相な体つきに見えるのだった。そういうわけで、このごろはルシールのほうが何かと有利だ。
　夫ほど老けていないのをルシールは誇りに思い、そう思うことに罪悪感も抱かなかったが、やはり少しは抱くべきかと、ときどきは感じていた。
　メモを取るベラミーは手が痛くなるまで書きつづけたが、それでもまだやめなかった。面談相手の目の前でメモを取ることに抵抗を感じていた時期もあったが、やはりこれは効果的な手法なのだった。世間の人々は、役人に何か話してそれが記録されないとなると、気分を害するものらしい。ベラミーにとっては好都合だった。彼の脳は聞くことよりも見ることのほうが得意だったから、その場で書いておかないとあとで思いだすのに苦労するのだ。
　ベラミーは、一九六六年のその日、誕生日パーティーが始まった時刻をまず記した。続いて、涙と後悔に満ちたルシールの語りをすべて書き取った。生きているジェイコブを見た最後の人物が彼女だったこと。しかし覚えているのは、ほかの子を追いかけて角を曲がっていった息子の、白い腕だけだということ。葬儀には、教会に入りきれないぐらいの人が集まったこと。
　だが、書かなかった部分もあった。官僚的な文書にするのははばかられ、自分の記憶だけにとどめておくことにした細かな部分だ。
　ハロルドとルシールはかろうじて息子の死を乗り越えたが、それから五十年あまり、ある種の虚( むな ) しさがふたりの日々から消えることはなかった。日曜のディナーのときなど、不意にそれが頭をもたげて険悪な空気になった。名状しがたいその感情について、夫婦で話しあうことはめったになかった。毎日息を潜め、それのまわりを忍び足で歩いた。たとえて言えば、リビングルームの真ん中に突如として原子核破壊装置が出現したようなものだった。サイズこそ小さいものの精巧きわまりないその装置は、いかに忌まわしく信じがたいことがこの世で起きうるかを彼らに見せつけるのだった。
　月日がたつにつれ、ふたりとも虚しさを直視しないことに慣れていった。目をそらすのが上手になっていった。それはほとんどゲームのようなものだった。ストロベリー・フェスティバルのことを話してはいけない。あの子が大好きだったお祭りだから。すごい建物だと思ってもじっと見つめてはいけない。あの子はきっと建築家になるねとふたりで話しあったのを思いだすから。あの子に似た面差しの子どもがいても、無視しなければいけない。
　ジェイコブの誕生日が来るとふたりの気分は沈み、会話もぎこちなくなった。ルシールがわけもなく泣きだしたり、ハロルドが前の日より多めに煙草を吸ったりした。
　しかし、これまでずっと毎年そうだったわけではない。心の傷跡が生々しい、はじめの数年だけだった。
　彼らは年を取った。
　ドアは閉ざされた。
　悲嘆に暮れた日から遠く離れたところまで来たときに、玄関先にわが子がふたたび現れたのだ。神に召されたはずの子が、にこにこと、元気に、八歳のままで。

　ハロルドとルシールの話が終わり、静けさが訪れた。重い沈黙はしかし、長くは続かなかった。キッチンのテーブルでジェイコブがフォークを使う音がする。彼は喉を鳴らしてレモネードを飲み、満足そうにげっぷをした。「ごめんなさい」と、はきはきした声で両親に謝る。
　ルシールは微笑んだ。
「不躾( ぶしつけ ) なことをうかがいますが」ベラミーが言った。「決して責めるとかそういうんじゃありません。この......特殊な現象をより深く理解するために、どうしてもお尋ねしなくてはならないんです」
「ほうら、おいでなすった」ハロルドの両手がついに幻の煙草を探るのをやめ、ポケットの中で静止した。ルシールがたしなめるように手で払う仕草をした。
「あなたがたとジェイコブとの関係はどのようなものでしたか？」
　ハロルドは鼻を鳴らした。右足に体重をかけたほうが逆よりも安定する、と体がようやく決断を下した。彼はルシールを見た。「わたしたちはあの子を邪魔者扱いしていましたとかなんとか、言わなきゃいけない場面だぞ。テレビでよくやってるだろう。きつく叱ったり夕飯抜きの罰を与えたり、テレビで見る虐待の数々をわたしたちもやりましたと告白するわけだ」ハロルドは玄関ホールの小さなテーブルの前へ行った。そのいちばん上の引き出しに、新しい煙草が入っている。
　彼がリビングルームへ戻るより先に、ルシールが叫んだ。「吸わないで！」
　両手が彼のものではなくなったとでもいうように、ハロルドは淡々と機械的にパッケージを開けた。火のついていない煙草を唇に挟むと、しわの多い顔をぽりぽりと掻( か ) き、長くゆっくりと息を吐いた。「こうしていればいいんだ。これだけで」
　ベラミーが穏やかに言った。「あなたがたや、ほかの誰かのせいで息子さんが......ああ、婉曲( えんきよく ) な表現がもう浮かびません」彼は苦笑した。「とにかく、ご両親を追求したりするつもりは毛頭ありません。ただ事実を教えていただきたいだけなんです。管理局はなんとしてもこの現象を解明したいと考えています。皆さんと同じです。われわれは人と人を結びつける役割を担ってはいますが、だからといって、このことに関して詳しいわけではまったくないんです」彼は肩をすくめた。「われわれとしては、皆さんからお話を聞くことが解明の手がかりになるに違いないと考えています。答えるのを渋る方が出てくるような質問をしてでも、手に負えなくなる前に事態を把握したいのです」
　ルシールは古いソファの上で身を乗りだした。「手に負えなくなる？　そんなことになるんですか？」
「そりゃ、なるさ」ハロルドが言った。「間違いない」
　ベラミーはゆるゆるとかぶりを振っただけで、先の質問にまた戻った。「あなたがたとジェイコブの関係はどうでしたか？」
　ハロルドが答えようとしているのを察知したルシールは、夫を黙らせるために口を開いた。「良好でした。なんの問題もない親子関係でした。世の中の親がみんなそうであるように、わたしたちも息子を心から愛( いと ) おしく思っていました。あの子もわたしたちを慕ってくれていました。あのころもそうでしたし、いまもそうです。わたしたちはあの子を愛していて、あの子もわたしたちを愛してくれている。そして、今日、神の恩寵( おんちよう ) によってわたしたちはふたたび一緒になることができたんです」ルシールは首筋をこすり、それから両手を広げた。「これは奇跡です」
　マーティン・ベラミーはメモを取った。
「いかがです、ご主人？」
　ハロルドは火のついていない煙草を口から離すと、頭を掻き、うなずいた。「家内の言ったことがすべてだ」
　ベラミーがペンを走らせる。
「つかぬことをうかがいますが、おふたりのうちのどちらかでも、信仰深くていらっしゃる？」
「ええ！」いきなりルシールが背筋を伸ばした。「イエス様はわたしの師であり友人です！　それがわたしの誇りです。アーメン」ルシールはハロルドのほうへ顎をしゃくった。「あちらの方はね、不信心なんですよ。神様の恩寵に甘えきっているんです。悔い改めなさいとわたしは言いつづけているんですけど、とても頑固で」
　ハロルドが、古い芝刈り機にも似た耳障りな声で笑った。「うちは交代で信心するんだよ。五十年ちょっとたっても、まだおれの番は来てないってわけだ。ありがたいことにね」
　ばかみたいというように、ルシールが両手をひらひらさせた。
「宗派は？」ペンを走らせながらベラミーが訊く。
「バプテストです」ルシールが答える。
「いつから？」
「生まれたときから」
　記録される。
「まあ、正確に言えば違いますけど」
　ベラミーが手を止める。
「メソジストだった時期もありました。でも聖書の解釈でところどころ牧師様とどうしても意見が合わない点があって。ホーリネス教会ものぞいてみましたけれど、あれにはついていけませんでした。叫んだり、大声で歌ったり踊ったり。教会というよりパーティー会場という感じでした。キリスト教徒として、あれはないと思いましたよ」ルシールは伸びあがるようにして、ジェイコブがちゃんとそこにいる――椅子に座ったまま舟を漕( こ ) いでいる。昔からいつもこうだった――のを確認してから、言葉を継いだ。「それからまた別の――」
「この人はきみの信仰遍歴を長々と聞きたいわけじゃない」ハロルドがさえぎった。
「あなたは黙ってて！　わたしが質問されてるのよ！　そうでしょう、ミスター・マーティン・ベラミー？」
　ベラミーはうなずいた。「ええ、おっしゃるとおりです。どんなお話も重要性を帯びてくる可能性があります。わたしのこれまでの経験からすると、鍵は細部に隠されていることが多いんです。とりわけ、今回のような大きな出来事においては」
「大きいって、どれぐらい？」ルシールが即座に尋ねた。相手がこれを言いだすのを待っていたかのようだった。
「何人ぐらいかという意味ですか？」
　ルシールはうなずいた。
「とんでもなく多いというわけではありません」ベラミーは慎重な口調で答えた。「正確な数を公にすることは許されていませんが、まあ、そこそこですね」
「数百？」ルシールは食いさがった。「数千？　そこそこって、どれぐらい？」
「気になさる必要はありませんよ、ミセス・ハーグレイブ」ベラミーはかぶりを振った。「じゅうぶん奇跡的と言える数ですから」
　ハロルドがくつくつと笑った。「彼はきみの心をお見通しだ」
　ルシールは無言で微笑んだ。

　ベラミー係官が聴取を終えるころには、太陽は密( ひそ ) やかに姿を隠し、窓の外でコオロギが鳴き、ジェイコブは両親のベッドで静かな寝息をたてていた。キッチンの椅子から息子を抱きあげて寝室へ運ぶとき、ルシールは嬉しくてならなかった。この年で、こんな状態の腰で、誰の力も借りずに息子を運べるなんて、思いもよらなかった。
　ところがそのときが来て、そっと身をかがめ両腕を息子の下へ差し入れて、自分の体に動けと命じると、ジェイコブは軽々と持ちあがり、彼女の胸元に収まったのだ。まるで、二十代のころに戻ったかのようだった。若くて機敏だったあのころに。年を取ったのも腰が痛むのも、気のせいだったのではなかろうか。
　なんなく階段をあがり、息子をベッドにおろしたルシールは、かつてそうしたようにかたわらに腰かけて小さくハミングをした。だが、早く眠りにつかせなければいけないとは思わなかった。
　この子はじゅうぶん長く眠っていたのだから。
　ルシールはしばらくじっと息子を見つめていた。上下する胸を見守りながら、魔法――あるいは奇跡――が、突然終わってしまうのではないかと恐れた。しかし、そうはならず、ルシールは神に感謝した。
　リビングルームへ戻ってみると、ハロルドとベラミーがぎこちない沈黙の中にいた。戸口に立つハロルドは、火のついた煙草をスパスパ吸っては、網戸越しに夜の闇に向かって煙を吐きだしている。ベラミーは、さっきまで座っていた椅子のそばに立っている。急に、喉の渇きと疲れがあらわになったように見える。このとき初めてルシールは、彼を迎えてからお茶の一杯も出していなかったことに気づいた。そしてそれが妙にショックだった。けれど、ハロルドとベラミーの様子からして、これとはまた別のショックを受けることになりそうだと、なぜか予想できた。
「きみに質問があるそうだ、ルシール」煙草を口へ持っていくハロルドの手は震えていた。だからルシールは、黙って吸わせておこうと決めた。
「何かしら？」
「座っていただいたほうがいいでしょう」椅子に導くべく、ベラミーが彼女に近づこうとした。
　ルシールは後ずさりした。「なんなんですか？」
「デリケートな問題なので」
「それはわかります。でも、そんなに怖い質問なの？　違うわよね？」
　ハロルドは妻に背を向けると、うつむいたまま黙って煙草をふかした。
「どなたも」ベラミーが口を開いた。「最初は、単純な質問だと思われるんです。しかし、実のところ、非常に複雑で重大な意味を持つ質問なのです。ですから、答える前にじゅうぶん考えていただきたい。答えるチャンスが一度しかないというわけではありません。ただ、考え抜いた末に下した決断だとご自身で納得していただきたいんです。難しいかもしれませんが、できれば感情に流されることのないように」
　ルシールの頬が紅潮した。「まあ、ミスター・マーティン・ベラミー！　まさかあなたが性差別者だったなんて。わたしが女だからって、感情的になると決めつけないでいただきたいわ」
「違うんだ、ルシール」ハロルドが語気を荒くした。が、その声はどこか自信なさげだった。「いいから、彼の話を聞くんだ」
　ルシールは椅子に座った。
　マーティン・ベラミーも腰をおろした。ズボンの前についた、見えないものを手で払い、その手をしばらくじっと見つめた。
「あの」ルシールが言った。「はっきりおっしゃってくださいな。前置きはもうたくさん」
「今日、最後の質問です。答えていただくのはいますぐでなくてもかまいませんが、早いに越したことはありません。早く答えてくだされば、それだけ物事が複雑にならずにすみますから」
「ですから、早くおっしゃってください」ルシールは懇願する口調になった。
　マーティン・ベラミーは息を吸った。「ジェイコブをこのまま手元に置いておきたいですか？」
