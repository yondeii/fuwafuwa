第 一 章 　 葬 儀 の 流 れ

な ぜ 、 ご 葬 儀 を 行 う の で し ょ う

「 ご 葬 儀 を 上 げ な く て は い け な い 。 」 と い う こ と は あ り ま せ ん 。

現 在 は 「 直 葬 」 と い う よ う に 、 葬 儀 を 行 わ ず 故 人 を 見 送 る こ と も あ り ま す 。

ご 家 族 の 考 え や 、 諸 事 情 も あ る と お も う の で 、 大 切 な 方 と の お 別 れ を 、 気 持 ち を も っ て 送 っ て い た だ け れ ば 、 ど の 様 な 形 を と っ て も 良 い の で は な い か と 思 い ま す 。

で は な ぜ 葬 儀 を 行 う こ と を す る の で し ょ う 。

「 葬 儀 」 の 正 式 な 呼 び 方 は 「 葬 送 儀 礼 」 と い い 、 儀 式 を 通 し て 大 切 な 方 と の お 別 れ を し 、 気 持 ち の 面 で も 社 会 的 な 意 味 で も 一 つ の 区 切 り が で き る よ う に 葬 儀 を 行 う と い う 考 え 方 も で き る か と 思 い ま す 。

生 前 の 故 人 と の 数 あ る 思 い 出 を 振 り か え る と と も に 、 感 謝 の 気 持 ち を 伝 え る た め の 時 間 で は な い で し ょ う か 。

（ お 寺 な ど の 宗 教 的 考 え と は 異 な る と 思 う の で 、 そ の 点 は ご 了 承 く だ さ い ）

喪 主 様 の 心 構 え

大 切 な 方 を 亡 く さ れ た と き 、 悲 し み に 浸 る こ と や 平 静 を 保

つ の が 困 難 な な か で も 、 現 実 は 葬 儀 の 段 取 り を 進 め な け れ ば な り ま せ ん 。

大 変 　 お 辛 い か と 思 い ま す が 、 こ の 様 な 時 こ そ お 亡 く な り に な ら れ た 大 切 な 方 を 思 っ て 、 ど の 様 に 見 送 る か の 対 応 が 必 要 に な り ま す 。

重 要 ！ 知 ら な か っ た で は 済 ま な い お 金 の 話

故 人 の 遺 産 を 使 っ て 葬 儀 を あ げ よ う と お 考 え の 方 も い ら っ し ゃ る か と 思 い ま す が 、 金 融 機 関 で は 、 名 義 人 の 死 亡 を 知 っ た 時 点 か ら 預 金 の 口 座 を 停 止 す る 義 務 が あ り 、 個 人 名 義 の 取 り 扱 い の 全 て が ロ ッ ク さ れ 引 き 出 せ な く な る の で す 。

キ ャ ッ シ ュ カ ー ド は も ち ろ ん 窓 口 で も 取 引 は で き な く な り ま す 。

理 由 は 、 亡 く な っ た 時 点 で 、 全 て 財 産 と み な さ れ る の で 、 相 続 の 対 象 に な る か ら で す 。

し か し 、 葬 儀 は 突 然 の 予 期 せ ぬ 出 来 事 な の で 、 費 用 に つ い て 窓 口 に 申 し 出 る と 引 き 落 と し に 対 応 し て く れ る こ と も あ り ま す 。

こ の 場 合 、 口 座 の 名 義 人 以 外 の 人 が 引 き 落 と し を す る こ と に な る の で 、 金 融 機 関 に よ っ て は 保 証 人 や 書 類 な ど の 提 出 が あ る か も し れ ま せ ん 。

相 続 人 が 複 数 い る 場 合 な ど は 、 全 て の 人 の 委 任 状 な ど が 必 要 と な る ケ ー ス も あ る た め 、 も し 、 ご 兄 弟 が 遠 く に 離 れ て 住 ま れ て い る 場 合 な ど は と て も 不 便 と な り ま す 。 （ 委 任 状 と 一 緒 に 、 本 人 確 認 の た め に 戸 籍 謄 本 や 印 鑑 証 明 も 要 求 さ

れ る こ と も あ り ま す 。 ）

も し 、 亡 く な っ た ご 本 人 以 外 で 、 銀 行 な ど の 通 帳 を 管 理 さ れ て い る 方 が い ら っ し ゃ る の な ら ば 、 葬 儀 費 用 分 だ け は 相 続 さ れ る 方 の 了 承 を 得 た 上 で 、 前 も っ て 降 ろ さ れ て い た 方 が よ ろ し い か と 思 わ れ ま す 。

通 常 　 一 週 間 以 内 、 長 く て も 十 日 の う ち に 葬 儀 が 行 わ れ て い ま す 。

パ ン フ レ ッ ト に 「 直 送 」 よ く か い て あ り ま す が 、

病 院 が 当 日 ま で 預 か っ て く れ る 。

火 葬 場 の 空 き が あ る 、 火 葬 許 可 書 の 書 類 が 揃 っ て い る 。

な ど 条 件 が そ ろ わ な い 限 り 、 病 院 か ら 直 に 火 葬 場 に 行 け る こ と は ま ず あ り ま せ ん 。

( 直 ぐ に 火 葬 を 希 望 さ れ て も 、 法 律 に よ っ て 死 亡 後 ２ ４ 時 間 は 火 葬 で き な い か ら で す 。 ）

昔 は 自 宅 で お 亡 く な り に な ら れ る 場 合 も ご ざ い ま し た が 、 現 在 、 殆 ど の 方 が 病 院 か デ イ ホ ー ム な ど の 施 設 で お 亡 く な り に な り ま す 。

危 篤 の 連 絡 を 受 け て か ら 、 ご 逝 去 さ れ る ま で お 時 間 が あ れ ば 、 検 討 し て か ら 手 配 す る 時 間 の 余 裕 も あ る か と 思 い ま す が 、 殆 ど の 場 合 、 悲 し み に 浸 る こ と も 許 さ れ な い く ら い に 、 次 か ら 次 へ と 物 事 は 進 ん で い き ま す 。

ま ず ご 逝 去 さ れ る と 、 病 院 の 安 置 室 へ と 場 所 が 移 動 し ま す 。

一 度 動 き 始 め る と 時 間 が な い の で 、 変 更 す る こ と も で き ず 、 よ く わ か ら な い ま ま 、 葬 儀 社 の 言 い な り に な っ て 、 葬 儀 を あ げ る こ と に な る の で す 。

葬 儀 社 に よ っ て は 、 パ ッ ク で 金 額 を 決 め て い る と こ ろ や 、 後 か ら 考 え る と 不 必 要 な も の ま で 、 買 わ せ ら れ 金 額 を 跳 ね 上 げ て し ま う 所 、 亡 く な っ た 方 が 、 葬 儀 の 積 み 立 て を し て い た が 、 実 際 に 葬 儀 を あ げ る と 差 額 分 を 請 求 さ れ た 。

こ の よ う な ケ ー ス が 起 き て い ま す 。

急 な こ と で 、 他 社 と の 比 較 や 相 場 が わ か ら な い も の を よ し と し て 、 「 皆 様 は 普 通 こ の く ら い の 葬 儀 を 上 げ て い ま す 」 な ど と い い 高 額 に な る こ と あ り ま す 。

で す の で 、 「 生 前 に 葬 儀 の 話 を す る の は 縁 起 で も な い 」 と よ く 言 う 方 も い ら っ し ゃ い ま す が 、 「 終 活 」 と い う 言 葉 が あ る よ う に 、 で き る こ と で あ れ ば 、 ま だ お 元 気 な と き や 具 合 が 悪 く 入 院 し た 時 を 機 会 に 、 金 銭 面 を 含 め て 数 件 、

近 く の 葬 儀 社 に 見 積 も り を 取 っ て お く こ と を お 勧 め し ま す 。

ネ ッ ト で 安 さ を 売 り に し て い る と こ ろ が あ り ま す が 、

火 葬 費 用 な ど 最 低 必 要 に な る は 金 額 が 決 ま っ て い ま す 。

そ の よ う な 所 は 、 人 件 費 が 別 や 、 消 耗 品 が 入 っ て い な か っ た 。 な ど 別 途 あ と か ら 請 求 さ れ る こ と が あ る の で 、 よ く 訊 ね て か ら お 願 い す る よ う に す る と 良 い で し ょ う 。

葬 儀 社 が 決 ま っ て い る 場 合 は 、 も し 病 院 で 紹 介 さ れ た と し て も 、 「 も う お 願 い し て い る 葬 儀 社 が あ る 」 と 伝 え れ ば 、 紹 介 す る 葬 儀 社 で な け れ ば い け な い 。 と い う こ と は あ り ま せ ん の で ご 安 心 く だ さ い 。

お 願 い し て い る 葬 儀 社 に 亡 く な ら れ た ら 連 絡 を し 、 迎 え に き て 頂 け れ ば 良 い で し ょ う 。
