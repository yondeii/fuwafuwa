この仕事が、「私の世界」でどういう意味を持つのか。


千鶴：

（人の傷を癒す、命を救う。苦しんでいる人を、助けてあげたくて、私はこの仕事を選んだんだったわ......）


じゃあ、今一番癒してあげたい人は？

私は自問自答を繰り返した。



〜広目組組長宅・組長の部屋〜


私は、狸右衛門さんの屋敷を訪れた。


狸右衛門：

「――話ァある程度聞いてるぜ。悪く思わんでくれ。あれはあれなりに、この組と、――嬢ちゃん先生のことを考えてんだ」


千鶴：

「判ってます......でも、正直......どれが正解なのか、判らないんです」


狸右衛門：

「......てめえの心に聞くんだ、そういう時ァよ。そうやって出した答えに、間違いなんてねえやな」


千鶴：

「......心」


狸右衛門さんに言われたように、目を閉じて未来に想いを馳せる。

私と龍牙さんの未来。

どうあるべきか、ではなくて。

どうありたいか、と心に問いかけた。


そこに見えたのは、二人。

私と、龍牙さん。

付かず離れず寄り添って、陰も日向も変わらずに......


千鶴：

「――狸右衛門さん」


狸右衛門：

「......その顔、心は決まったか？」


千鶴：

「はい......龍牙さんはいますか？」



〜広目組組長宅・中庭〜。


龍牙さんは、一人庭の池を眺めていた。

私はその背中に腕を伸ばし、そっと抱きしめる。


龍牙：

「――千鶴」


千鶴：

「......私、決めました。私の答え......聞いてくれますか？」


龍牙：

「――ああ」


千鶴：

「......私は......」

〜広目組組長宅・大広間〜


――紋付袴を着た重鎮たちが序列純に座っている。

その世界に詳しい者が見れば、そうそうたる顔ぶれだということが判るだろう。

一番上座に座っているのは現・組長。

そして、――


媒酌人：

「......それでは、この襲名に異論がなければ、各々方( おのおのがた ) お手元の盃を飲み干してください」


媒酌人の言葉に、次々と盃が干されていく。

次いで現組長が、次期組長へ手ずから真っ白い盃を差し出す。

中に満たされるのは酒。


媒酌人：

「これを飲み干した瞬間から、四代目広目組組長襲名となります。異論がなければ、盃を飲み干して、その盃を懐に収めてください」



媒酌人の言葉の通り、満ちていた盃が空になり、男の着物の懐に収められた。


媒酌人：

「四代目広目組組長、護崎龍牙殿。おめでとうございます」


わぁっ、と歓声が上がる。

口々に彼を寿( ことほ ) ぐ言葉が飛び交った――


　　　◇


私は、その光景を部屋の外で感慨深く見ていた。

これから始まる幾多の困難も、彼なら......護崎龍牙と言う人ならきっと乗り越えられる。


結局、半井さんは「単独犯」ということで片が付き、関西阿修羅会はそれを誅( ちゅう ) することで、メンツを保った。

本意がどこにあるにせよ、和解を受け入れた関東帝釈会。

次にまた諍いが起きない限り、この静かな時間はまだ続くだろう。


千鶴：

「さ、宴会だわ。お酒、運んでください」


感動している場合ではない。

もう、四代目広目組は始まったのだから。

私はそこにいた組員に指示をすると、黒い留袖の裾を払いながら、厨房へ向かった。


　　　◇



跡目相続の盃事を終え、関東帝釈会のめぼしい幹部達は帰っていった。

飲めや騒げやで乱れまくった大広間も、広目組の若衆( わかしゅ ) 総出で片付けるとあっという間に元通り。


千鶴：

「ご苦労様。あとは私やるから、皆もう行っていいわよ」


若衆：

「いいんですか！」


千鶴：

「あんまり羽目外しちゃだめよ。堅気( かたぎ ) の人に迷惑かけてもだめ。......はい、これ皆のお小遣い」


若衆：

「姐さん、ありがとうございます！」



広目組を上げての晴れの日。

皆飲みに行きたくてウズウズしてたらしく、すぐに街へ繰り出していった。


千鶴：

「......さて、と。私はもうひと頑張り」


龍牙：

「千鶴」



千鶴：

「龍牙さん。――今日は、お疲れ様でした。そして、襲名おめでとうございます」


龍牙：

「ああ。......お前も、今日は色々采配して疲れたろう」


千鶴：

「私は手配だけだもの。あとは業者さんと、若衆のみんなのおかげ。......でも一応、こんな格好はしてるの。おかしな格好じゃ、あなたのメンツに関わるでしょ？」


龍牙：

「......ふ」


千鶴：

「ふふっ。お屋敷のお片付け終わったら帰るから。龍牙さんも今日は色々お店に顔を出すんでしょ？　私のことは気にしないで行ってください」


龍牙：

「――後でもいいさ」


龍牙さんは私をぎゅっと抱きしめた。


千鶴：

「なぁに......どうしたの？」


龍牙：

「お前を労( ねぎら ) おうと思ってな。部屋に酒を用意した。――相手してくれ、千鶴」



〜広目組組長宅・組長の部屋〜


お祭り騒ぎの後の静けさを感じながら、私と龍牙さんは差しつ差されつお酒を飲む。


千鶴：

「......ぷは、いいお酒......」


龍牙：

「頬が赤い」


千鶴：

「うふ、もういい気持ちだからかしら。はい、龍牙さんにもご返杯」


龍牙：

「おっと......。.........ああ、美味いな」


千鶴：

「ふふ」


そのうち、ほろ酔いも進んできた頃、龍牙さんがテーブルになにかを置いた。

心地よくぼやける視界に、それを捉える。


千鶴：

「......？　なあに、この箱」


龍牙：

「開けていい」


言われるままに箱を開けると、そこには、


千鶴：

大粒のダイヤモンド、それを囲むようにまたダイヤ。

プラチナの輝きを持った指輪が、ケースの中で立っていた。


千鶴：

（これって......）

「龍牙さん」


一気に酔いも吹き飛んだ。

ケースの中で光る指輪と、その向こうの龍牙さんを見ていると、いつの間にか両目から涙が溢れ出す。

着物の袖で押し拭っても、後から流れて止めどない。

龍牙さんに左手を取られ、薬指に指輪が滑りこむ。

ぴったりなその指輪は、私のために作られたものなのだ。


千鶴：

「......綺麗」


龍牙：

「――今日この日、俺は広目組を背負った。今まで以上に苦労も心配もかけるだろうが......俺はお前と所帯を持ちたい。――結婚してくれ、千鶴」



パン、パン、パーン！


派手な音のクラッカーが鳴ったかと思うと、水流( つる ) さん、牧之瀬( まきのせ ) さんを始め、飲みに繰り出したはずの広目組の若衆のみんなが立っていた。


一同：

「おめでとうございます！」


龍牙：

「......てめえら......いつから......！」


蠍彦：

「もう、勘弁して下さいよ組長。本気でサシ飲みだけかと思ってヒヤヒヤしましたよ」


豹：

「姐さん、......これからは、名実ともに、広目組の姐さんだな！」


千鶴：

「......みんな、......ありがとう......」


蠍彦：

「その指輪、組長が半年くらい悩んでたんです。私が代わりに選んで取りに行こうかって言っても、一人でやる！......って頑( かたくな ) なで頑なで」


龍牙：

「――カツ！　てめえ！」


豹：

「渡すタイミングとか、リハーサルしてたり」


龍牙：

「牧之瀬ェ！」


千鶴：

「......あははっ。本当？龍牙さん」


龍牙：

「......ッッ...............まあ、その」


千鶴：

「嬉しい......」


薬指の指輪の幸せな重み。

嬉しさで流す涙は、いくらでも惜しくない。

龍牙さんが咳払いをして、私に改めて向かう。


龍牙：

「......この馬鹿どものせいで、返事を聞いてねえ。......千鶴」


千鶴：

「......はい。謹んで、お受けします」


再び、若衆のみんなが歓声を上げた。


蠍彦：

「はいはい、皆さんそこまで。これからお二人は愛の語らいをしますので、可及的速やかに撤収してください」


一同：

「はい！　若頭！」


蠍彦：

「というわけで、以降はお二人ごゆっくり」


水流さんはそう言って若衆を連れて、屋敷を出ていった。


龍牙：

「......まったく、あいつら......」


千鶴：

「ふふっ。愛されてるわね」


龍牙：

「......それはお前もだ。あいつらも、嬉しくてしょうがねえんだ」


千鶴：

「本当？　そうなら、嬉しい」


龍牙：

「――千鶴」


龍牙さんが、私の手をとった。


龍牙：

「改めて......よろしく、頼む」


千鶴：

「――はいっ......」
