第 1 章 ミ ス タ ー ・ ジ ョ ニ ー ！

「 は ... ... ふ 、 ん ♡ ん ん っ ♡ 」

の ぷ っ く り 熟 れ た 乳 首 を く に く に ♡ と 弄 る こ と も 忘 れ な い 。

「 あ っ ♡ ん っ 、 ふ ♡ は ぁ っ ♡ き も ち 、 イ イ ... ... ！ ♡ 」

　 漏 れ る 声 も 抑 え る こ と な く 、 く ぅ っ 、 と 少 女 の 手 が 強 く バ イ ブ を 押 し 付 け た と 思 っ た ら 、 ビ ク ビ ク と 体 を 震 わ せ る 。 や が て ピ ン ♡ と 伸 び た 脚 先 が ふ る ふ る と 震 え て 、 絶 頂 に 至 っ た こ と を 知 ら せ て い た 。

第 2 章 通 学 も 一 緒 ♡

　 私 の 日 常 は ご く ご く 平 凡 。 学 校 へ の 通 学 は 電 車 。 今 日

も 制 服 を キ ュ ッ と 着 て 、 荷 物 の 準 備 も バ ッ チ リ 、 車 内 で 聞 く た め の イ ヤ ホ ン も セ ッ ト し て 。 ... ... そ し て 、 リ モ コ ン バ イ ブ の ス イ ッ チ を 入 れ る 。

　 私 の 密 か な 楽 し み は こ れ 。 楽 し み と い う か 、 日 課 ？ 　 と い う か 何 だ ろ う 、 も う 酸 素 ？ 　 い つ も ナ カ に は バ イ ブ を 入 れ て 楽 し ん で い る 。 今 日 は 小 さ め で 、 リ モ コ ン で 強 弱 を 楽 し め る 代 物 。 ク リ ト リ ス ま で 刺 激 し て く れ る も の に 比 べ た ら 刺 激 は 足 り な い け ど 、 で も こ う い う の も た ま に は い い 。

　 私 は ぐ ー っ 、 と ス イ ッ チ を 「 強 」 ま で 押 し 上 げ て 、 ナ カ の 振 動 が ぶ る ぶ る と 強 度 を 増 し て 行 く の を 楽 し ん だ 。 ゾ ク ゾ ク し て 嬉 し い 。 気 持 ち い い こ と は 大 好 き だ 。 い つ

か ら だ っ た か 忘 れ た 。 も ち ろ ん 私 の 処 女 は こ の 「 お 友 達 」 に 捧 げ て あ る 。 ど の 子 か 忘 れ ち ゃ っ た け ど 。 で も 、 と に か く バ イ ブ を 入 れ て 生 活 す る の が 私 の 日 常 な の だ 。

　 私 が そ ん な も の を 入 れ て い る な ん て 気 付 か な い 周 り の 人 た ち 。 男 の 人 も 女 の 人 も い て 。 み ん な 疲 れ た 顔 で 電 車 に 揺 ら れ て い る 。

「 ん ん っ ... ... ♡ 」

　 満 員 電 車 の 中 で 誰 か の 鞄 が い い 具 合 に 当 た っ て 快 感 が 増 し た 。 思 わ ず 声 を 上 げ そ う に な る け ど 、 我 慢 我 慢 。 だ っ て 、 我 慢 す る の が 気 持 ち い い か ら 。 誰 に も バ レ な い よ う に 、 涼 し い 顔 し て こ れ を 入 れ て い る こ と 、 そ れ 自 体 が 快 感 。

　 誰 か 痴 漢 し て く れ る 人 が い た ら い い の に 、 な ん て 思 う 。 だ け ど な か な か 運 命 の 人 は 現 れ な い 。 さ っ き 鞄 が 当 た っ た 時 、 も し か し て 、 と 思 っ た け れ ど 、 そ れ は 偶 然 だ っ た ら し い 。 あ れ か ら 刺 激 ら し い 刺 激 は な い 。

　 で も 、 こ の 状 況 で 誰 か が 痴 漢 し て く れ て 。 胸 ま で 弄 っ て 、 バ イ ブ を 動 か し な が ら 、 耳 元 で 囁 か れ た り 、 息 を 荒 げ て く れ た り 、 硬 く な っ た も の を 押 し 当 て て く れ た り ... ... そ ん な こ と を 想 像 す る と 、 妄 想 だ け で も う 幸 せ 。

　 私 は ま た 、 登 校 前 に ふ る ふ る と 人 知 れ ず 電 車 の 中 で イ っ て し ま っ た 。 こ れ が 幸 せ だ か ら 私 は 周 り の 人 み た い に ど ん よ り し た 顔 な ん て し な い 。 い つ で も 元 気 で い ら れ る の は こ れ の お か げ か も 知 れ な い 。

　 で も 。 い つ か 、 運 命 の 人 （ 痴 漢 さ ん ） に 会 い た い な 、 な ん て ... ... 思 っ て し ま う の だ 。

第 3 章 運 命 の 人 ♡

　 そ れ は あ る 日 突 然 や っ て 来 た 。 今 日 の 相 棒 は ク リ ト リ ス も 刺 激 し て ナ カ で 左 右 に 震 え る 素 敵 な バ イ ブ 。 名 前 は デ ニ ー 。 そ う 、 一 人 一 人 名 前 が つ い て い る の だ 。 そ ん な 愛 し て る デ ニ ー は 今 日 も 好 調 で 、 私 の イ イ と こ ろ を 容 赦 無 く 叩 き 潰 し て く れ る 。

　 ふ っ く ら と し た 乳 暈 に 添 え ら れ る 温 か い 手 の ひ ら 。 下 か ら た わ わ な 胸 を 支 え る よ う に 包 み 込 ん で 、 た ゆ た ゆ ♡ と 揺 ら し て く れ る 手 が 、 あ っ た 。

「 は ぁ っ ♡ は っ ♡ は ー っ ... ... ♡ 」

「 ... ... 」

る 。

　 も っ と 、 と い う ア ピ ー ル に 私 は 、 す り ♡ と そ の 手 を 撫 で て 、 乳 首 を 触 っ て く れ る よ う に ア ピ ー ル し て み た 。 痴 漢 さ ん は び っ く り し た の か 、 体 を ギ ク リ と こ わ ば ら せ た け れ ど 、 そ の 意 味 を 理 解 し た よ う に 、 ほ う っ 、 と た め 息 を つ い て 、 さ す さ す ♡ と 乳 首 を 指 の 腹 で な ぞ り 始 め た 。

「 は っ ♡ ぅ ♡ 」

「 ... ... 気 持 ち い い の ？ 」

　 耳 元 で 囁 か れ る 。 あ あ こ れ ！ 　 こ れ が 私 の 求 め て た 運 命 の 痴 漢 さ ん ♡ 嬉 し く て ち ょ っ と は に か ん で し ま う 。 自 然 に 緩 ん で し ま う 口 元 と 赤 く な る 頬 を 止 め ら れ な い 。

　 私 は コ ク 、 と あ く ま で も 「 恥 ず か し い 」 素 振 り を し て 頷 い た 。

　 胸 を い じ っ て い た 手 が ど ん ど ん 下 が っ て 、 腰 に 回 さ れ る 。 く び れ を 執 拗 に 撫 で 回 さ れ て 、 私 は び く び く す る 体 を 止 め ら れ な か っ た 。 そ の 度 に デ ニ ー く ん の ク リ 責 め が 私 を も っ と 責 め 立 て て 、 ヒ ク ヒ ク す る カ ラ ダ を 抑 え る こ と が で き な い 。

「 あ ぁ っ ... ... ！ ！ ♡ 」

腰 と 同 時 に 乳 首 を く に く に ♡ と 弄 ら れ た 時 、 私 の 体 は 大 き く 震 え て 、 途 端 に 声 が 漏 れ て し ま っ た 。 フ 、 と 痴 漢 さ ん が 嬉 し そ う に 微 笑 む 気 配 を 感 じ る 。

「 そ ん な 声 出 し ち ゃ っ た ら 、 バ レ ち ゃ う よ ？ 　 そ れ に 何

か な ？ 　 コ ・ レ 」

　 そ う 、 バ レ な い わ け が な か っ た ！ 　 バ レ て 欲 し く な か っ た わ け じ ゃ な い 。 む し ろ 、 バ レ て 欲 し か っ た く ら い だ け ど 。

「 や ぁ っ ♡ ん 、 ぅ ぅ っ ... ... ！ ！ ♡ ♡ 」

　 痴 漢 さ ん は 容 赦 な く 私 の バ イ ブ を つ か ん で 、 ご ち ゅ ご ち ゅ ♡ と 子 宮 口 を ノ ッ ク し 始 め る 。 自 然 と 内 股 に な っ て 力 が 入 ら な く な っ た 足 。 ガ ク ガ ク 震 え て も う イ っ ち ゃ い そ う 。 ク リ も む ぎ ゅ む ぎ ゅ ♡ と 力 強 く 刺 激 さ れ た ら 、 も う ど う し よ う も な い 。

「 ッ 、 は っ 、 ぁ ぁ ッ ... ... 〜 〜 〜 〜 〜 〜 ！ ！ ！ ！ ！ ♡ ♡ ♡ ♡ 」

　 私 は で き る だ け 息 を 潜 め た け れ ど 、 ど れ だ け 蕩 け た 顔

を 晒 し て い た か 分 か ら な い 。 そ の く ら い の 勢 い で 、 イ っ て し ま っ た 。

「 ... ... 続 き 、 シ な い ？ 」

第 4 章 私 と 痴 漢 さ ん と 、 バ イ ブ さ ん

「 君 、 痴 女 な の ？ 」

　 私 も 登 校 は 諦 め た し 、 青 木 さ ん も 出 社 な ん て 知 っ た こ と か と 言 う 風 に 座 っ て い る 。

　 寄 っ て 来 る 鳩 た ち に 、 お 昼 ご 飯 の パ ン の か け ら を あ げ て い た 。

「 ち じ ょ ... ... ？ 」

「 ダ メ な の ？ 」

「 で 、 で も 」

「 ん ？ 」

「 青 木 、 さ ん さ え 良 け れ ば ... ... 毎 日 、 痴 漢 、 し て く れ た ら 嬉 し い で す ... ... 」

　 バ イ ブ 、 色 々 選 び ま す か ら ！ 　 と 私 は 力 強 く 言 っ た 。

第 5 章 痴 漢 デ イ ズ

　 青 木 さ ん と の 約 束 か ら 一 週 間 。 私 た ち は 毎 日 同 じ 時 刻 、 同 じ 車 両 に 乗 り 合 わ せ て 、 痴 漢 プ レ イ を 楽 し ん で い る 。
