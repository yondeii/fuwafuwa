それでも、今の"あなた"にピンとこないものは、気にせず受け流してください。

今はその時でないかも知れませんし、好みの問題かもしれません。

　

このLIGHT ６ について、タイトルをつけずに執筆を始めたとき、よく目に飛び込んできたのが中国で成功している人びとのTV番組です。

彼らは口をそろえています。

　

いい時代に生まれたことを感謝していると。

「この 時代 じだい の 恩恵 おんけい 」を受けたことで、今の成功があると。

　

また、 IT アイティー 長者 ちょうじゃ のTV番組でも、彼らは私に語りかけてきます。

「この 時代 じだい 」のチャンスをつかんだことで、今の成功した人生があると。

　

まるで、「この 時代 じだい の 恩恵 おんけい 」ということをテーマにしろ！と語りかけてくるように Synchronicity 偶然の一致 がチャネリングメッセージを運んできます。

　

そのようなわけで、今回のタイトルは「この 時代 じだい の 恩恵 おんけい 」でした。

これが今、私の中でさまざまな 他 ほか のチャネリングメッセージとつながりながら、進行していました。

シンプルすぎてわかりません？

この人々は、チャネリングメッセージなどを知らなくても、テクノロジーの 発達 はったつ による 恩恵 おんけい とともに、" 独自 どくじ の 進化 しんか "を 遂 と げている 新人類 しんじんるい です。

Hackerの道

Programmerにとって「Hack（ハック）」が最速化のすべてです。

「Hack」という言葉は、工夫する・役立てるという意味でProgrammerは使っています。

Hacker（ハッカー）の本来の意味は、プログラミングのエキスパートみたいな意味です。

Hackerの仕事は、 事物 じぶつ を正面からだけではなく、多面的にとらえ、目的 を達成する最も効率的な方法を見つけ出すことです。

Hackerは、物事の本質を突き詰め、多面的にとらえることで、シンプルな原理まで最速で辿り着きます。

この性質はEntityたちが語る"進化した人間の性質"をよく表しています。

　

Facebook創業者のマーク・ザッカーバーグが株式公開時に株主へ宛てた手紙があります。

「Hackerの道とは、持続的な改善と反復を積み重ねていくことです。Hackerとは、物事は常に向上できる余地があり、完璧なものは何もないと信じている人のこと です。」

　

ここで「言葉の混乱の環」の話です。

"完璧なものは何もない"とEntityたちの語る"あらゆるものが完璧です"は、文章的には全く違いますが、同じ内容をマーク・ザッカーバーグは語っています。

またもや、 禅 ぜん 問答 もんどう のような表現になってしまいます。

ものごとは"完璧なものは何もない"ことで完璧で、それは変化していくので改良を加えていく余地があるということです。

Programmerは完璧に完璧を重ねて進化していくと信じている人のこと となります。

　

ユビキタス・キャプチャー

Programmerは情報処理の達人です。

Programmerが身につけている情報処理の 技術 テクニック に「ユビキタス・キャプチャー」と呼ばれる習慣があります。。

ユビキタスは 偏在 へんざい 、つまりあらゆる場所と時間を意味し、キャプチャーはその通り、 捕 つか まえるという意味です。

WebやTVを見ているとき、朝ベッドで目覚めたとき、散歩をしているとき、とにかくありとあらゆるタイミングでピン と来たら、とりあえず情報を捕まえて（キャプチャーしておく）という習慣です。

今は、スマートフォンやICレコーダーでボイスメモができたりします。

アナログの時代にインスピレーションを大事にする人は、枕元に、ペンとメモ帳で「ユビキタス・キャプチャー」をおこなっていました。

私だったら、こんな 洒落 しゃれ た 名前 ネーミング にはならず、「いますぐメモ」とでもNaming（ネーミング）してしまいます。

やっていることは同じでも、カッコイイ 名前 ネーミング の方が、やること自体にワクワクします。

情報をキャプチャーするということは、同時に気に 留 と めていることを、「 忘 わす れる」ということでもあります。

人は忘れることで、より多くのことを 覚 おぼ えたり、意識を他のことに向けることができます。

「ユビキタス・キャプチャー」を行い、 頭 マインド の中のメモリー（ 短期 たんき 記憶 きおく ）を軽くしておくことが 頭 マインド を 明晰 めいせき に 保 たも つコツです。

　

情報をつかまえる姿勢・態度

「ユビキタス・キャプチャー」は、情報をつかまえる 姿勢 しせい ・ 態度 たいど を保持することが重要です。

これは Inspiration 直観 （自分の経験値から、湧いてくる情報）やチャネリングメッセージ（見えない存在からの共感 Telepathy テレパシー 、または思考の 塊 かたまり の受信）をすぐにつかまえる 姿勢 しせい ・ 態度 たいど を保持することが重要なのです 。

昔、読んだ本の中にこんな 暴論 ぼうろん がありました。

「メモなんか必要ない。すぐに忘れてしまうようなアイディアや情報は重要なものではないから～」という作家さんがいました。

危 あや うく、納得しそうな 論理 ロジック ですが、私はそうは思いません 。

これも私の経験則ですが、基本は「いますぐメモ」です。

投機的実行

コンピューターを高速に動作させるための工夫のひとつに、「 投機的 とうきてき 実行 じっこう 」というものがあります。

コンピューターのプログラムは高速で「 分岐 ぶんき 」を起こしながら、実行されています。

この予測不能な「 分岐 ぶんき 」の影響を極力減らす方法として、「 投機的 とうきてき 実行 じっこう 」という機能が組み込まれています。

　

ここで、人の日常生活を例にとり、極端に簡単な説明をします。

通常の実行とは、人が何か問題に取り組むために一つの仮説を立てて実行します。

その仮説が 誤 あやま りだとわかったところで、また別の仮説を立てて別の実行を行います。

これでは一つひとつ仮設・実行を行っているので時間がかかります。

　

「 投機的 とうきてき 実行 じっこう 」とは、考えられる仮説を同時に実行するということです。

そして、複数の仮説の中で、途中に 誤 あやま りだと分かったところで、うまくいかない仮設を捨てながら進んでいくというプログラムの手法です。

この「 Synchronism シンクロニズム 」という概念は、私の今持っていると信じ込んでいる「願望実現・現実創造」の 技術 テクニック をさらに進化させてくれます。

　

「チャネリング人生マニュアル」は、チャネリングメッセージによってもたらされる高度に進み過ぎた「知識」と、自分の今の進化レベルで機能する「知識」とのハイブリッド（異なった要素を組合わせたもの）が、人生では、最も適した利用方法だと思います。

　

その 象徴的 しょうちょうてき なAnalogy（類推・類比）現象として、現在では、自動車もガソリンとが電気モーターのハイブリッド車がうまく適応しています。

水素自動車や電気モーター車だけの方が理想的なことはわかっていても、現在のインフラが追いついていないのです。

そこで、車メーカーは 次 じ 世代 せだい への 移行期 いこうき に、ハイブリッド車を提供しています。

　

このような社会現象でさえ、人類の進化の象徴的な流れ として理解の中に関連させることは可能です。

現在、世の中にあるものごとは、あらゆる点でつながっているのです。

　

Old Ageの中でもっともよく機能していたものごとは、この時代New Ageへの移行期の中で、徐々に機能しなくなるものがあります。

　

だからと言って、Old Ageの中で機能していたものごとを突然拒絶したり、利用しなくなること自体が痛みや苦しみをともないます。

　

たとえば、いくら時間が幻想だと言ったところで、私の肉体はきっちりとした充分な睡眠時間を要求します。

肉体はOld Ageそのものなのです。

　

チャネリングメッセージ「知識」と、自分進化レベルで機能する「知識」との融合が、人生にバランスをもたらす「チャネリング人生マニュアル」と言えます。
