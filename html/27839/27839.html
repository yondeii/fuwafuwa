恋 愛 業 界 の ３ Ｇ 。 モ テ る 男 に な る た め の マ イ ン ド と 行 動

合 コ ン や 街 コ ン 、 婚 活 な ど 様 々 な 出 会 い の 場 が あ り ま す が モ テ る 男 っ て 必 ず い ま す よ ね 。

そ の 男 に 女 性 か ら の 人 気 が 集 中 し て し ま い 、 自 分 は 相 手 に さ れ な い な ん て 経 験 お 持 ち で は な い で し ょ う か 。

そ の モ テ る 男 が 実 際 に 大 切 に し て い る の が 恋 愛 業 界 の ３ Ｇ で す 。

す な わ ち 、

ジ ェ ン ト ル 、 ギ ャ ッ プ 、 強 引 。

ち ょ っ と そ れ ぞ れ が 相 反 す る よ う な 要 素 の 気 が し て 、 混 乱 し て し ま っ た と い う 方 も い ら っ し ゃ る か も し れ ま せ ん 。

実 際 に は 、 こ の 3 つ は 矛 盾 し ま せ ん 。 ま ず は ジ ェ ン ト ル に ふ る ま っ て 、 女 性 に 安 心 感 を 与 え 、 信 頼 を 獲 得 す る こ と が 大 切 。 た だ し 、 そ れ だ け で は い い 人 で 終 わ っ て し ま う 可 能 性 も あ り ま す 。

そ こ に 魅 力 を 加 え る の が ギ ャ ッ プ 。 本 文 で 詳 し く 解 説 し て い ま す が 、 ギ ャ ッ プ を 見 せ つ け る こ と に よ っ て 、 た だ の い い 人 だ け で な く 、 よ り あ な た に 興 味 を も っ て も ら う こ と が で き ま す 。

そ し て 、 最 後 に 、 押 し 切 れ な い 場 合 に 必 要 に な っ て く る の が 「 強 引 」 で す 。 紳 士 的 で 、 ギ ャ ッ プ が あ っ て 魅 力 的 で あ っ て も 、 最 後 の 一 押 し が で き な い 男 に 幻 滅 し て し ま う と い う 女 性 も 実 は 少 な く あ り ま せ ん 。

こ の ３ つ を バ ラ ン ス よ く 、 そ し て 段 階 を 追 っ て 使 い こ な す こ と に よ っ て 、 自 分 も そ の 男 の よ う に 女 性 か ら 人 気 を

得 る こ と が 可 能 で す 。

今 回 は 恋 愛 業 界 の ３ Ｇ を ベ ー ス に モ テ る 男 の マ イ ン ド と 行 動 を ご 紹 介 い た し ま す 。

1 章 ジ ェ ン ト ル ： 女 性 は お 客 様

紳 士 的 な 男 性 は 、 落 ち 着 い て い て 余 裕 が あ り 大 人 っ ぽ い の で 女 性 は 安 心 す る こ と が で き ま す 。 遊 ん で い る 男 に 振 り 回 さ れ て 疲 れ て し ま っ た 、 な ん て 経 験 を 持 っ て い る 女 性 は よ り そ の 傾 向 が 強 い か も し れ ま せ ん 。

紳 士 的 に 振 舞 う こ と で 、 そ れ 自 体 が マ イ ナ ス に な る と い う こ と は ま ず あ り え ま せ ん 。 ま ず は 紳 士 的 に 女 性 に 接 す る こ と で 好 印 象 を 与 え ま し ょ う 。

紳 士 的 な 男 性 の 特 徴 と し て 挙 げ ら れ る の は 、

・ 落 ち 着 い て い る

・ 気 遣 い が で き る

・ 聞 き 上 手

・ 褒 め 上 手

・ 清 潔 感 が あ る

な ど と い っ た 特 徴 。 順 番 に 見 て い き ま し ょ う 。

ま ず 、 紳 士 的 な 男 性 は 落 ち 着 い て い ま す 。 余 裕 が あ り 焦 り が な い た め 、 女 性 は 安 心 す る こ と が で き ま す 。 落 ち 着 き の な い 人 が 無 理 に エ ス コ ー ト し て し ま う と 女 性 も 焦 っ

て 気 を 遣 っ て し ま い お 互 い 疲 れ て し ま う の で す 。 そ う な ら な い た め に も 落 ち 着 い て 接 す る こ と は と て も 大 切 で す 。

そ し て 気 遣 い が で き る の も 紳 士 的 な 男 性 の 特 徴 。 極 端 に 言 え ば 女 性 は お 客 様 で す 。 食 事 の 際 は お 箸 や お し ぼ り を と っ て あ げ た り 、 女 性 の 飲 み 物 が 少 な く な っ た 時 に は 声 を か け た り な ど 、 ち ょ っ と し た 気 遣 い が で き る と 印 象 が か な り 良 く な り ま す 。

次 に 、 聞 き 上 手 、 褒 め 上 手 で あ る こ と 。 女 性 は 「 今 日 〇 〇 に 行 っ て き た 」 「 今 日 こ ん な こ と が あ っ た の ！ 」 と

い っ た お 話 を す る の が 大 好 き で す 。 興 味 が な い こ と で も 真 摯 に 聞 い て あ げ る と 女 性 は 嬉 し い で す し も っ と 話 し た く な る の で す 。 し っ か り 相 槌 を 打 っ て 聞 い て あ げ ま し ょ う 。 ま た 、 紳 士 的 な 男 性 は 褒 め 上 手 で も あ り ま す 。

た と え ば デ ー ト の 日 、 待 ち 合 わ せ 場 所 に 着 い た と き に 「 そ の 服 似 合 っ て る じ ゃ ん 」 と 言 わ れ た ら 女 性 は か な り 嬉 し い で す 。 そ れ が 買 っ た ば か り の 服 、 そ れ も そ の 日 の た め に 買 っ た 服 だ と し た ら な お さ ら 。 デ ー ト の ス タ ー ト で テ ン シ ョ ン が 一 気 に 上 が る で し ょ う 。 紳 士 的 な 男 性 は 褒 め 上 手 で も あ る の で す 。

も ち ろ ん 、 自 分 自 身 の 身 だ し な み も 忘 れ て は い け ま せ ん 。 そ も そ も 清 潔 感 が な け れ ば 第 一 印 象 で 紳 士 的 と は 思 わ れ な い で し ょ う 。 自 分 に 合 っ た 服 の サ イ ズ 、 色 合 い 、 髪 型 や 髭 、 爪 。 細 か い と こ ろ に 意 識 で き る 男 性 は 女 性 か ら 見 て も 「 マ メ な 人 だ な 、 き っ と 仕 事 で も 細 か い と こ ろ を 大 切 に で き る 人 な ん だ 」 と 好 印 象 を 与 え る こ と が で き ま す 。

紳 士 的 な 男 性 は 「 自 分 は 紳 士 的 だ 」 と ア ピ ー ル す る の で は な く 、 女 性 は 何 を し た ら 喜 ぶ の か を 理 解 し て い る の で す 。 そ し て 常 に 見 ら れ て い る 意 識 を し て い る た め 、 清 潔 感 に そ れ が 現 れ て い ま す 。 紳 士 的 と い う の は 誠 実 さ で も あ り ま す 。 女 性 に 対 し て 紳 士 的 に 接 す る こ と で 誠 実 さ が

伝 わ り 、 信 頼 を 勝 ち 取 る こ と が で き る の で す 。

3 G の 最 初 の G 、 ジ ェ ン ト ル 。 紳 士 的 、 と い う と ハ ー ド ル 高 く 感 じ て し ま っ た か も し れ ま せ ん が 、 実 際 に 読 ん で み る と 人 と し て 当 た り 前 の こ と の 延 長 線 上 に あ る こ と に も お 気 づ き で は な い で し ょ う か ？ ま ず は で き る と こ ろ か ら 、 実 践 し て み て く だ さ い 。

2 章 ギ ャ ッ プ ： 紳 士 的 だ け で は だ め ！ ギ ャ ッ プ で 女 性 の 気 を 惹 き つ け ろ ！

紳 士 的 で あ る こ と は 、 女 性 に 信 用 し て も ら う た め に 必 須 の プ ロ セ ス で は あ る の で す が 、 紳 士 的 な だ け で は た だ の い い 人 で 終 わ っ て し ま う 可 能 性 も あ り ま す 。

そ こ で 、 紳 士 的 に 振 る 舞 い 、 と き に ギ ャ ッ プ を 見 せ る こ と に よ っ て 女 性 の 興 味 を 引 く こ と が で き る の で す 。 紳 士

的 に 振 舞 う だ け で は な か な か 女 性 と の 距 離 は 縮 ま り ま せ ん 。 ギ ャ ッ プ を 見 せ る こ と に よ っ て 女 性 に 対 し て 強 烈 な 印 象 を 与 え る こ と が で き ま す 。

例 え ば 、

・ 俺 様 タ イ プ な 人 が 繊 細 に 。

普 段 は 上 か ら 目 線 で 俺 様 タ イ プ の 人 が 急 に 繊 細 に な っ た ら 女 性 か ら し て み れ ば 可 愛 く 思 え る の で す 。 母 性 本 能 が く す ぐ ら れ 、 「 守 っ て あ げ た い 」 「 こ の 人 は 私 が い な く ち ゃ ダ メ な ん だ 」

と さ え 思 う の が 女 性 の 心 理 で す 。

つ い つ い 人 に 対 し て 強 く 出 が ち だ 、 と い う 方 は 意 中 の 女 性 の 前 で だ け 少 し だ け 自 分 の 弱 い 所 も さ ら け 出 し て み る と 良 い か も し れ ま せ ん 。

・ 静 か で 穏 や か な 人 が い ざ と い う と き に は 頼 れ る

普 段 は 静 か で 穏 や か な 人 は 普 段 は 安 心 感 が あ り ま す が 、 そ の 反 面 で 何 か ト ラ ブ ル が 起 き た 時 に 大 丈 夫 な の だ ろ う か 、 な ん て 心 配 を 持 た れ が ち な の も 否 め ま せ ん 。 基 本 的 に は 穏 や か で 優 し い け れ ど 、 必 要 な 時 に は 強 く て 、 頼 れ

る 。

そ ん な ギ ャ ッ プ に 女 性 は 惹 か れ て し ま う も の で す 。

そ ん な 男 性 で あ れ ば 、 日 常 で も 安 心 。 い ざ と い う と き も 安 心 で 、 本 当 に 頼 り が い の あ る パ ー ト ナ ー に な り ま す よ ね 。

・ チ ャ ラ い の に 優 し い

普 段 は チ ャ ラ チ ャ ラ し て い る 人 が お 年 寄 り に 電 車 の 席 を 譲 っ て い た り し た ら 女 性 は 「 意 外 と 優 し い ん じ ゃ ん 」 と

思 う で し ょ う 。 チ ャ ラ い 人 は 一 般 的 に は あ ま り 女 性 受 け が 良 く あ り ま せ ん が 、 内 面 と の ギ ャ ッ プ で 女 性 に 強 く 印 象 を 与 え る こ と が で き ま す 。

見 た 目 を チ ャ ラ チ ャ ラ し て い た い 人 は 、 行 動 面 で は 優 し く ふ る ま う よ う に す る と ギ ャ ッ プ を 意 図 的 に 作 り 出 す こ と が で き ま す 。

当 然 で す が 、 ギ ャ ッ プ が あ れ ば よ い わ け で は あ り ま せ ん 。 優 し そ う な の に 優 先 席 に 率 先 し て 座 り 、 立 っ て い る お 年 寄 り を 無 視 、 な ん て い う 「 マ イ ナ ス ギ ャ ッ プ 」 は 評 価 を 下 げ る だ け で す 。

・ チ ャ ラ い の に 一 途

続 い て 、 チ ャ ラ い 人 の ギ ャ ッ プ 第 二 弾 と し て 「 チ ャ ラ そ う な の に 、 実 は 一 途 」 と い う の も 有 利 な ギ ャ ッ プ で す 。 誰 に で も 声 を か け る 。 い ろ ん な 女 の 子 と だ ら だ ら 、 フ ラ フ ラ 遊 ん で い る 。 告 白 す る 時 は 壁 ド ン で 上 か ら 目 線 。

そ ん な 印 象 を 持 た れ が ち な チ ャ ラ い 人 が 、 特 定 の 女 の 子 に 対 し て の み 一 直 線 。 そ し て 告 白 す る 時 は 顔 を 赤 ら め て 噛 み 噛 み 。 そ ん な 一 途 さ も ギ ャ ッ プ に す る こ と が で き ま す 。

も ち ろ ん 、 実 際 に チ ャ ラ い の に 一 途 な フ リ を す る の は 逆 効 果 で す よ 。

・ 真 面 目 な の に お 茶 目

普 段 は す ご く 真 面 目 で 、 仕 事 も プ ラ イ ベ ー ト も テ キ パ キ こ な す し っ か り し て そ う な 人 が ど こ か 少 し 抜 け て い た ら 女 性 か ら し て み れ ば か わ い く 思 え る の で す 。 寝 癖 が つ い て い た り 、 忘 れ 物 を し た り 、 小 さ い 子 供 と 無 邪 気 に 遊 ぶ 姿 が あ れ ば 女 性 は キ ュ ン と し ま す 。

2 つ 目 に G は 、 ギ ャ ッ プ 。 ギ ャ ッ プ に よ っ て 「 こ ん な 一 面 あ っ た ん だ 」 と 女 性 を と き め か せ 興 味 を 引 く こ と が で き ま す 。 女 性 が と き め く ギ ャ ッ プ を 見 せ る こ と で 恋 愛 へ 発 展 さ せ る こ と が で き る の で す 。

大 切 な の は ギ ャ ッ プ の 方 向 性 で す 。 大 切 な の は ギ ャ ッ プ 、 と は い っ て も 「 紳 士 的 な の に 実 は 粗 暴 」 「 細 や か に み え て 気 が 利 か な い 」 な ど の ギ ャ ッ プ は 、 ひ た す ら 女 性 を 幻 滅 さ せ る だ け 。

そ う い う 時 は 、 あ る 程 度 は 改 善 し つ つ 、 軌 道 修 正 を し て

い き ま し ょ う 。 粗 暴 な の は 「 オ レ サ マ 」 に 。 気 が 利 か な い の は 「 お お ざ っ ぱ で 豪 快 」 に 。 弱 み を ど の よ う に 強 み に 転 換 し て い く か 、 と い う の も 大 切 な 視 点 に な っ て き ま す 。

ギ ャ ッ プ の 見 せ 方 が わ か ら な い 、 と い う 方 は 、 中 の よ い 友 人 、 異 性 が 望 ま し い で す が 、 い な け れ ば 同 性 で も 構 い ま せ ん 。 複 数 の 友 人 か ら み て 、 自 分 の 印 象 や 長 所 、 短 所 が ど こ な の か を 素 直 に 教 え て も ら う と い う の が 大 切 で す 。 そ こ か ら 導 き 出 さ れ る 「 人 か ら ど う 見 ら れ て い る の か 」 と い う 情 報 を も と に 、 ギ ャ ッ プ を 作 っ て い け ば よ い の で す 。

な お 、 こ の 自 分 を 知 る と い う プ ロ セ ス 、 単 に ギ ャ ッ プ づ く り だ け で な く 、 恋 愛 に お い て も そ う で す し 、 恋 愛 か ら 離 れ た 人 生 全 般 に お い て も す ご く 役 に 立 ち ま す 。

最 初 は 少 々 面 倒 な 作 業 な の で 渋 ら れ る か も し れ な い で す が 、 カ フ ェ 代 、 ラ ン チ 代 く ら い を ご 馳 走 し て で も や る 価 値 の あ る 行 為 で す よ 。 も し く は お 互 い の メ リ ッ ト を 説 明 し て 、 お 返 し に 分 析 し て あ げ る 、 と い う の も 有 効 か も し れ ま せ ん 。

3 章 強 引 ： こ れ が で き る 男 性 は ご く わ ず か 、 い ざ と い う 時 は 強 引 に

モ テ る 男 は こ こ ぞ と い う と き に 強 引 に 女 性 を 誘 い ま す 。 紳 士 的 に ふ る ま い 、 ギ ャ ッ プ を 見 せ 、 最 後 は 強 引 に 誘 い 出 す こ と で 女 性 は キ ュ ン キ ュ ン し ま す 。

例 え ば 草 食 系 男 子 と の デ ー ト の 時 は 「 私 が リ ー ド し な き ゃ 」 と 女 性 は 内 心 疲 れ て い ま す 。 で す が 、 そ ん な 時 に ぐ っ と 引 っ 張 っ て く れ る 人 だ と 女 性 は キ ュ ン と し ま す し な に よ り 安 心 す る の で す 。

女 性 は ど こ か で 男 性 に リ ー ド さ れ た い と い う 気 持 ち が あ り ま す 。 女 性 を 誘 う と き は 少 し 強 引 く ら い が 丁 度 良 い の で す 。 と り わ け 、 ベ ッ ド イ ン す る 時 は よ っ ぽ ど の 肉 食 女 子 で な い 限 り 、 女 性 の 側 か ら 誘 い 出 す と い う こ と は 、 中 々 あ り ま せ ん 。

と り わ け 、 最 初 は 恥 ず か し さ に 加 え て 、 プ ラ イ ド も あ る は ず 。 お い そ れ と 身 体 は 許 さ な い と い う ス タ ン ス で 、 一 度 は 拒 否 し て く る も の と 考 え 「 押 し て ダ メ な ら 押 し 倒 せ 」 の 精 神 で 、 強 引 さ を 見 せ ま し ょ う 。 （ と は い え 、 本 気 で 拒 否 し て い る 場 合 も あ る の で 程 度 問 題 で も あ り ま す 。 ）

ま た 、 女 性 は 恋 愛 に お い て 言 い 訳 を し た が り ま す 。

「 彼 が ど う し て も っ て 言 う か ら 」 「 彼 が 強 引 に 誘 っ て く る か ら 」 な ど 、 自 分 か ら 積 極 的 に ア タ ッ ク し た く な い と い う 女 性 が 多 い の で す 。 そ ん な 消 極 的 な 女 性 に と っ て 強 引 に 誘 っ て く る 男 性 は か な り 都 合 が 良 い の で す 。

こ こ 最 近 で は 草 食 系 男 子 が 増 え て い ま す 。 2 0 代 の 4 割 が 童 貞 と も い わ れ て い ま す 。 普 段 仲 良 く し て い て も 、 そ こ か ら ま っ た く 進 展 さ せ る 気 が な け れ ば 「 恋 愛 感 情 で は な い の か な ？ 」 と 思 わ れ て し ま っ た り 、 恋 愛 感 情 を 持 っ て い る こ と に は 気 づ い て も ら っ て い て も 、 「 最 後 の 一 押

し が で き な い 男 は 対 象 外 だ 」 と 思 わ れ て し ま っ た り す る か も し れ ま せ ん 。 つ ま り 、 紳 士 的 で ギ ャ ッ プ を 見 せ る こ と が で き た と し て も 最 後 の 強 引 さ が な け れ ば 、 た だ の 良 い 人 、 男 友 達 で 終 わ っ て し ま う の で す 。

実 際 の と こ ろ 、 こ の 「 強 引 」 が で き な い 男 が か な り 多 い の が 現 実 で す 。 と て も 勇 気 が 必 要 で す が 、 最 後 に 強 引 に い け る か い け な い か で 結 果 は が ら り と 変 わ っ て き ま す 。

も ち ろ ん 脈 が な い の に 強 引 に い っ て し ま う と 、 引 か れ て し ま い ま す の で 見 極 め は 大 切 で す 。 「 軽 い 女 だ と 思 わ れ た く な い か ら 拒 否 し て い る 」 の か 「 本 当 に 嫌 な の か 」

は 判 断 し な け れ ば な り ま せ ん 。 こ の あ た り は 、 「 い よ い よ ・ ・ ・ 」 と い う 段 階 に な れ ば よ り 明 ら か に な る は ず で す 。

な の で 、 後 戻 り が で き る 段 階 の 内 は 「 当 た っ て 砕 け ろ 」 「 嫌 よ 嫌 よ も 好 き の う ち 」 く ら い の マ イ ン ド で ア プ ロ ー チ を か け て 行 っ て も よ い の で は な い で し ょ う か ？

そ し て 最 後 は 、 「 押 し て ダ メ な ら 押 し 倒 せ 」 で す 。 も ち ろ ん 、 こ こ ま で 来 て 、 相 手 が 本 気 で 拒 否 し て い る よ う で あ れ ば 、 そ こ で 退 く こ と も 大 切 で す よ 。

ま と め

合 コ ン や 婚 活 、 学 校 や 職 場 な ど 様 々 な 出 会 い の 場 が あ る 中 で 必 ず 1 人 は い る モ テ る 男 。

そ ん な モ テ る 男 に な る た め に は ま ず 今 回 ご 紹 介 し た 恋 愛 業 界 の ３ Ｇ を 実 践 し て み ま し ょ う 。 ま ず は 紳 士 的 に 振 舞 い 、 相 手 の 信 用 を 高 め る こ と 。 こ こ で 安 心 感 と 信 頼 を 持 っ て も ら わ な け れ ば 、 次 の ス テ ッ プ に 進 め ま せ ん 。

次 に 重 要 に な っ て く る の が 、 ギ ャ ッ プ 。 「 こ の 人 、 こ ん な 側 面 も 持 っ て た ん だ 」 と い う ギ ャ ッ プ が 、 女 性 に と っ

て あ な た に よ り 興 味 を も つ き っ か け に な っ て く れ ま す 。 も ち ろ ん 、 ど ん な ギ ャ ッ プ で も 良 い わ け で は な い の で 、 方 向 性 に は 注 意 で す 。

そ し て 、 お 互 い に 深 い 興 味 を 持 ち 、 よ り 親 密 に な っ た ら 、 最 後 は 「 強 引 」 で す 。 こ こ を 決 め な け れ ば 、 友 達 で 終 わ っ て し ま う と い う ケ ー ス も 多 い の で 、 最 後 の 最 後 、 あ と 少 し だ け 勇 気 を 振 り 絞 り ま し ょ う 。 も ち ろ ん 、 本 当 に 嫌 が れ て い る の か は 見 極 め が 大 切 で す 。
