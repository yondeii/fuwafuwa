埴輪 はにわ と 文字 もじ のあいだ


― 底 そこ なし 補遺 ほい ―

まえがき


　 埴輪 はにわ とは 何 なに か。 埴輪 はにわ 時代 じだい 、 埴輪 はにわ 地域 ちいき で 何 なに があったのか。


　 埴輪 はにわ を 文字 もじ からさぐりたい。 文字 もじ は 頼 たよ れる 情報源 じょうほうげん である。
　とはいえ 文字 もじ を 正 ただ しく 読 よ み 取 と るのはそう 簡単 かんたん ではない。 記 しる されていることが 事実 じじつ とは 限 かぎ らない。 偏 かたよ った 主観 しゅかん が 入 はい っていたり、だます 意図 いと が 隠 かく されていたり、 形式的 けいしきてき なあいさつに 過 す ぎなかったりする。 文字 もじ どおりに 受 う け 取 と るべきかどうか、 慎重 しんちょう さが 必要 ひつよう である。だがいったん 文字 もじ を 目 め にしたら、あとで 間違 まちが いとわかっても、 頭 あたま に 刻 きざ まれた 文字 もじ を 消 け し 去 さ ることはむずかしい。それでも、そこに 文字 もじ があれば、 人 ひと は 意味 いみ を 確 たし かめずにはいられない。そして、 手掛 てが かりが 文字 もじ だけなら、 人 ひと は 文字 もじ を 深読 ふかよ みしてしまう。
　 以上 いじょう を 心 こころ にとめ、 埴輪 はにわ と 文字 もじ との 関係 かんけい をさぐる。まず、 埴輪 はにわ 時代 じだい の 文字 もじ を 探 さが す。それから、 時代 じだい を 下 くだ りながら 文字 もじ 「 埴輪 はにわ 」を 探 さが す。 本書 ほんしょ では、 埴輪 はにわ 地域 ちいき の 文字 もじ に 絞 しぼ った。


　なお 本書 ほんしょ には、 拙著 せっちょ 『 埴輪 はにわ には 底 そこ がない』に 記載 きさい するほどでもないと 判断 はんだん したものをまとめた。もちろん 埴輪 はにわ との 関連 かんれん がないわけではない。

凡例 はんれい
　 多 おお くは、 考古学 こうこがく で 定着 ていちゃく している 用語 ようご を 用 もち いた。ただし、よりふさわしいと 思 おも われる 用語 ようご があればそちらを 選択 せんたく した。
（１） 埴輪 はにわ を 作 つく っていた 時代 じだい を「 埴輪時代 はにわじだい 」とした。おおむね 三世紀 さんせいき 半 なか ば～ 六世紀 ろくせいき 末 すえ である。
（２） 埴輪時代 はにわじだい に 埴輪 はにわ を 作 つく っていた 地域 ちいき を「 埴輪地域 はにわちいき 」とした。
（３） 土 つち を 盛 も り 上 あ げて 作 つく った 墓 はか を「 墳墓 ふんぼ 」とした。
（４） 墳墓 ふんぼ 名 めい の 後 あと に、 以下 いか の 記号 きごう で 墳形 ふんけい を 表 あらわ した。
　〇△ ： 前方後円墳 ぜんぽうこうえんふん
　〇： 円墳 えんふん
　〇▲ ： 帆立貝形墳 ほたてがいけいふん （ 前方部 ぜんぽうぶ の 短 みじか い 前方後円墳 ぜんぽうこうえんふん 、もしくは 造 つく り 出 だ しのついた 円墳 えんふん ）
　□： 方墳 ほうふん
　□△ ： 前方後方墳 ぜんぽうこうほうふん
（５） 各 かく 墳墓 ふんぼ の 名 な は、 現在 げんざい もっとも 通 とお りがよいと 思 おも われる 名称 めいしょう で 記載 きさい した。 複数 ふくすう の 名 な を 持 も つ 墳墓 ふんぼ は 別名 べつめい を 括弧 かっこ 内 ない に 示 しめ した。 例 れい ： 岡田山 おかだやま １ 号墳 ごうふん （ 出雲 いずも 岡田山 おかだやま 古墳 こふん ）□△
（６）「 文字 もじ を 象嵌 ぞうがん すること」と「 錯 さく 銘 めい 」とを 同 おな じ 意味 いみ で 用 もち いる。
　 読 よ みやすいよう、 本文 ほんぶん ほぼすべての 漢字 かんじ にルビ（ふりがな）を 振 ふ った。ただし、 引用 いんよう 文献 ぶんけん のルビは 一部 いちぶ のみで、ルビを 振 ふ ったものも 確定 かくてい した 読 よ みではない。 文責 ぶんせき はもちろん 著者 ちょしゃ にある。

１　 埴輪 はにわ 時代 じだい の 文字 もじ


　 埴輪 はにわ 時代 じだい の 文字 もじ のうち、 埴輪 はにわ 地域 ちいき の 文字 もじ を 探 さが す。


１‐ １　 埴輪 はにわ に 刻 きざ まれた 文字 もじ

　 誰 だれ かが 何 なに かを 埴輪 はにわ に 書 か いた。


　 愛知県 あいちけん 春日井市 かすがいし の 勝川遺跡 かちがわいせき から、 文字 もじ が 刻 きざ まれた 円筒埴輪片 えんとうはにわへん が 出土 しゅつど している。 内側 うちがわ に 三文字 さんもじ が 確認 かくにん された。 第一文字 だいいちもじ は 縦方向 たてほうこう の 直線 ちょくせん 一画 いっかく のみである。 第二文字 だいにもじ は「子」か「干」、 第三文字 だいさんもじ は「日」か「目」のような 形 かたち である。 意味 いみ は 解明 かいめい されていないが、 埴輪 はにわ と 文字 もじ が 間違 まちが いなく 同 おな じ 時代 じだい の 同 おな じ 地域 ちいき に 存在 そんざい していた 証拠 しょうこ である。


１‐ ２　 埴輪 はにわ 以外 いがい に 記 しる された 文字 もじ

　 埴輪 はにわ より 多 おお く、 金属 きんぞく 製品 せいひん や 土器 どき に 文字 もじ が 記 しる された。


１‐ ２‐ １　 刀剣 とうけん

　 埴輪 はにわ 時代 じだい の 遺跡 いせき から、 文字 もじ を 記 しる した 刀剣 とうけん が 出土 しゅつど している。 出土 しゅつど 遺跡 いせき は 埴輪 はにわ 地域 ちいき の 東西 とうざい に 及 およ ぶ。


花形環頭飾 はながたかんとうかざり 金象嵌銘 きんぞうがんめい 大刀 たち
　 奈良県 ならけん 天理市 てんりし の 東大寺山 とうだいじやま 古墳 こふん 〇△ （ 四世紀 よんせいき 築造 ちくぞう と 推定 すいてい されている）の 粘土槨 ねんどかく の 中 なか 、 棺 ひつぎ の 外 そと から 出土 しゅつど した。 鉄刀 てっとう の 背 せ に 金 きん で 象嵌 ぞうがん された 隷書体 れいしょたい 二十四文字 にじゅうよんもじ のうち、 二十文字 にじゅうもじ が 読 よ み 取 と れる。 字間 じかん は 四文字 よんもじ 分 ぶん ほどである。 吉祥句 きっしょうく を 連 つら ねているらしい。 以下 いか に 簡単 かんたん にまとめる。


「 中平 ちゅうへい 」（ 後漢 ごかん の 中平 ちゅうへい 年間 ねんかん は 西暦 せいれき 一八四～一九〇）に 作 つく ったこの 良 よ い 刀 かたな は、「上應星宿」（ 天 てん の 星座 せいざ に 呼応 こおう している）、「下辟不祥」（ 下界 げかい の 災 わざわ いを 避 さ けられる）。


　 全長 ぜんちょう 一一〇㎝ 、やや 内湾 ないわん 、つまり 刃 は のほうにわずかに 反 そ る。 刀身 とうしん は 年号 ねんごう や 象嵌 ぞうがん の 技術 ぎじゅつ から 中国製 ちゅうごくせい とみなされている。 埴輪 はにわ 地域外 ちいきがい でつくられたことになる。 青銅製 せいどうせい の 柄頭 つかがしら は、 埴輪 はにわ 地域 ちいき 特有 とくゆう の 直弧紋 ちょっこもん があることから、 埴輪 はにわ 地域 ちいき で 作 つく られたと 考 かんが えられている。
　 東大寺山 とうだいじやま 古墳 こふん 〇△ からは、 円筒 えんとう 埴輪 はにわ 、 鰭 ひれ つき 朝顔形 あさがおがた 円筒 えんとう 埴輪 はにわ 、 朝顔形 あさがおがた 円筒 えんとう 埴輪 はにわ 、 家形 いえがた 埴輪 はにわ 、 盾 たて 形 がた 埴輪 はにわ 、ユギ 形 がた 埴輪 はにわ 、 甲冑 かっちゅう 形 がた 埴輪 はにわ が 出土 しゅつど している。 盾 たて 形 がた 埴輪 はにわ とユギ 形 がた 埴輪 はにわ には 直弧紋 ちょっこもん が 刻 きざ まれたものがある。 今 いま のところ 大刀 たち 形 がた 埴輪 はにわ は 確認 かくにん されていない。
　 銘 めい と 埴輪 はにわ とのつながりを 検討 けんとう する。 刀身 とうしん が 銘文 めいぶん 通 どお り 中平 ちゅうへい 年間 ねんかん に 作 つく られたならは、 銘 めい は 埴輪 はにわ 時代 じだい に 先立 さきだ つ。 銘 めい の 内容 ないよう が 埴輪 はにわ に 影響 えいきょう した 可能性 かのうせい がある。 災 わざわ いから 守 まも ってくれるものとして、 家 いえ 、 盾 たて 、ユギ、 甲冑 かっちゅう をモデルに 埴輪 はにわ を 作 つく ったかもしれない。また、この 大刀 たち が 以降 いこう の 大刀 たち 形 がた 埴輪 はにわ の 盛行 せいこう のきっかけかもしれない。

銀象嵌 ぎんぞうがん 大刀 たち
　 熊本県 くまもとけん 玉名郡 たまなぐん の 清原古墳群 せいばるこふんぐん の 江田船山古墳 えたふなやまこふん 〇△ の 石室 せきしつ から 出土 しゅつど した 直刀 ちょくとう である。 柄 つか は 残 のこ っていなかった。 直刀 ちょくとう の 背 せ に 七十五 ななじゅうご 文字 もじ が 象嵌 ぞうがん されている。 字間 じかん は 一文字 ひともじ 分 ぷん ほどである。 字 じ はほぼ 水平 すいへい に 書 か かれ、 隷書 れいしょ と 楷書 かいしょ が 混 ま じる 書体 しょたい である。 以下 いか 、 銘文 めいぶん を 簡単 かんたん にまとめる。


とある「 大王 おおきみ 」が「台（治）天下」の 時代 じだい に「 典曹人 てんそうじん 」として 仕 つか えた「 无利弖 ムリテ 」がたいへん 苦労 くろう してこの 刀 かたな を 作 つく らせた。 立派 りっぱ な 刀 かたな である。これを 持 も てば、 長生 ながい きでき 子孫 しそん は 繁栄 はんえい し、よいことがあり、 統治 とうち する 地 ち を 失 うしな うこともない。 刀 かたな を 作 つく った 者 もの は「伊太和」、 銘 めい を 書 か いた 者 もの は「張安」である。


　ところどころ 読 よ めないが、 常套句 じょうとうく 以外 いがい は 名前 なまえ が 多 おお い。「 大王 おおきみ 」は 第 だい 二十一 にじゅういち 代 だい 天皇 てんのう の 雄略 ゆうりゃく とみる 説 せつ が 有力 ゆうりょく である。「 典曹人 てんそうじん 」は 文書 ぶんしょ を 扱 あつか う 役人 やくにん らしい。「 典曹人 てんそうじん 」の 无利弖 ムリテ が 作 つく らせたことと、うたわれている 大刀 たち の 効能 こうのう からして、この 大刀 たち は 攻撃用 こうげきよう の 武器 ぶき ではなく、お 守 まも りのようである。
　 江田船山古墳 えたふなやまこふん 〇△ からは 円筒 えんとう 埴輪 はにわ と 朝顔形 あさがおがた 円筒 えんとう 埴輪 はにわ が 出土 しゅつど しているのみで、 形象 けいしょう 埴輪 はにわ は 確認 かくにん されていない。 銘文 めいぶん と 埴輪 はにわ の 関係 かんけい は 読 よ み 取 と れないが、 埴輪 はにわ と 文字 もじ を 記 しる した 物 もの とが 同 おな じところに 集 あつ められたひとつの 例 れい である。

第二 だいに 章 しょう の 結 むす び
　 埴輪 はにわ が 記紀 きき に 影響 えいきょう を 及 およ ぼした 感 かん がつよい。 成立 せいりつ 時期 じき の 順序 じゅんじょ から 言 い えば 当然 とうぜん である。また、かみ 合 あ わない 部分 ぶぶん もあるものの、 床 ゆか の 高 たか い 建物 たてもの があること、 鷹 たか の 尾 お に 鈴 すず をつけ 腕 うで にとまらせること、 琴 こと を 弾 ひ くのは 男子 だんし で 弾 ひ く 際 さい は 椅子 いす に 座 すわ ることなど、 記紀 きき と 埴輪 はにわ の 表現 ひょうげん が 一致 いっち するところもある。そういった 風習 ふうしゅう や 風俗 ふうぞく の 実在 じつざい 性 せい は 高 たか いと 言 い えよう。

　また、ついでに 陵墓 りょうぼ の 埴輪 はにわ も 検討 けんとう したが、 現在 げんざい の 陵墓 りょうぼ の 治定 じじょう は 日本 にっぽん 考古学 こうこがく が 発展 はってん する 以前 いぜん になされたものなので 根拠 こんきょ が 弱 よわ く、 記紀 きき その 他 ほか の 文字 もじ と 埴輪 はにわ をつなぐものとしては 頼 たよ りない。せめて 埴輪 はにわ についての 詳細 しょうさい な 情報 じょうほう がほしい。
