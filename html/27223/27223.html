突 然 で す が あ な た は 何 型 で す か ？

・ ・ ・ ・ ・

血 液 型 占 い は 日 本 で は 有 名 で す ね 。

で も 、 科 学 的 に は 立 証 さ れ て い ま せ ん 。

で も 、 な ん と な く 当 て は ま っ て い る し 、 確 か に 当 た っ て い る と 思 う 時 が あ り ま す ね ？

こ れ は 、 心 理 的 法 則 な ん だ と 思 い ま す 。

つ ま り 、 周 り の 評 価 が 自 分 を 形 造 る 。

と い う 心 理 的 法 則 が あ て は ま る の で す 。

例 え ば 、 A 型 な ら ば 幼 い こ ろ か ら 周 り か ら レ ッ テ ル を 張 ら れ ま す 。

『 あ な た は A 型 だ か ら ・ ・ ○ ○ な 性 格 な の ね 』

と い う 感 じ で す 。

あ な た も 周 り の 友 人 に 言 わ れ た こ と が あ り ま せ ん か ？

『 お 前 は ● 型 だ か ら 、 ● ● ● ● ● な ん だ な 』

と 言 う 感 じ で す 。

こ れ は 心 理 学 的 に は 『 レ ッ テ ル 法 』 と 言 わ れ ま す 。

人 は 、 周 り の 意 見 で 自 分 を そ の よ う に 思 い 込 ん で い く の で す 。

で す か ら 、 こ と 日 本 に お い て は あ な が ち 血 液 型 占 い も 当 た る よ う に な っ て い る の で す ！

（ 日 本 人 は 周 り の 意 見 に も 左 右 さ れ や す い で す よ ね ）

で は 、 以 上 を 踏 ま え て Ａ Ｂ 型 女 の 特 徴 を 見 て 行 き ま し ょ う ！

■ Ａ Ｂ 型 の 一 般 的 特 徴

Ａ 型 の 慎 重 さ と Ｂ 型 の 行 動 力 を 持 ち 合 わ せ て い る た め 、 た い て い の こ と は 上 手 に こ な せ る 。

感 受 性 が 強 く 感 激 家 、 そ の 反 面 、 物 事 に 動 じ な い ク ー ル な と こ ろ も あ る 。

物 事 を 多 角 的 に と ら え 冷 静 に 分 析 す る こ と が で き る 。

何 か を 始 め る と き は し っ か り シ ミ ュ レ ー ト し て 検 討 し て か ら 行 動 に 移 す 。

頭 の 回 転 が 速 く 、 ど う し た ら 一 番 ト ク か を 瞬 時 に し て 見

極 め る 力 が あ る 。

頭 の 中 に は い ろ い ろ な ゕ ゗ デ ゕ や プ ラ ン が た く さ ん つ ま っ て い る 。

難 問 に ぶ つ か る と そ れ ま で の や り 方 は あ き ら め て 、 新 し い 方 法 で 対

応 す る こ と が で き る 。

よ く 気 が つ き 、 如 才 な く 何 事 も ソ ツ な く や っ て の け る 。

一 般 的 に 社 会 に 参 加 し よ う と す る 意 欲 や 奉 仕 の 気 持 ち が 強 い 。

聞 き 上 手 で 、 情 報 を 引 き 出 す の が う ま い 。

短 所

優 し く て 思 い や り が 深 い と 思 っ た ら 、 突 然 意 地 の 悪 い こ

と を 言 っ た り と 、 周 囲 の 人 が 驚 く 態 度 を と る 。

よ く 考 え て か ら 行 動 を 起 こ す こ と も あ る し 、 思 い 立 っ た ら 即 行 動 と い う 具 合 に そ の 時 々 で 変 わ る 。

礼 儀 な ど の 社 会 通 念 に は こ だ わ ら ず 、 単 な る 慣 習 的 な こ と や ベ タ ベ タ し た 人 情 を 嫌 う 。

批 評 批 判 が 鋭 く 、 ま わ り の 人 に は ゗ ヤ ミ に 聞 こ え る こ と が あ る 。

や や 根 気 に 欠 け 面 倒 な こ と が 嫌 い で 優 柔 不 断 な と こ ろ も あ る 。

責 任 を 持 つ の を 嫌 が り 、 第 三 者 的 な 立 場 を 保 と う と す る 。

打 算 的 な と こ ろ が あ り 、 ト ラ ブ ル が 起 き る と さ っ と 一 人 だ け 逃 げ 出 し て し ま う 。

物 事 を 分 析 す る 能 力 に 長 け て い る が 懐 疑 的 に 考 え が ち 。

問 題 意 識 や 探 究 心 が 旺 盛 で 、 少 し の こ と で も 疑 問 が 残 る と 納 得 し な い 。

仕 事 は 速 い が 、 少 し で も 邪 魔 が 入 る と 集 中 で き な く な る 。

対 人

人 と の 間 に 距 離 を 置 い て い る よ う で い な が ら 、 い ろ ん な 人 と 豊 富 な つ き 合 い を す る 。

大 勢 の 人 の 前 で も 臆 せ ず 自 分 の 意 見 を 述 べ る こ と が で き る 。

自 信 家 の 部 分 が あ る の で 、 上 司 や 先 輩 と も 対 等 に 接 し 物 怖 じ し な い 。

恋 愛

ム ー ド の あ る 大 人 の つ き 合 い を 望 み 、 我 を 忘 れ て し ま う こ と は あ ま り な い 。

燃 え る よ う な 恋 と は あ ま り 縁 が な く 、 交 際 が 始 ま っ て も 恋 人 と は つ ね に 一 定 の 距 離 を お き た が る 。

自 分 の 感 情 を 極 力 抑 え る の で 、 恋 人 に さ え う ま く 伝 わ ら な い こ と が あ る 。

ス マ ー ト に 恋 を 楽 し み た い タ ゗ プ で 、 相 手 に 対 す る 理 想 が 高 い 。

お し ゃ れ で ス マ ー ト な 雰 囲 気 が あ る の で 異 性 に よ く も て る 。

職 業

頭 の 回 転 が よ く 、 飲 み 込 み が 早 い の で 、 ど ん な 職 業 で も ソ ツ な く こ な せ る こ と が で き る 。

合 理 性 に 富 み 、 物 事 を 多 角 的 に と ら え る こ と が で き る の で 、 ビ ジ ネ ス 感 覚 は な か な か の も の 。

少 し 忍 耐 力 が な く 何 が 何 で も や っ て や ろ う と い う 気 迫 に 欠 け る 。

自 分 を 出 さ な い

Ａ Ｂ 型 の 女 性 は 、 自 分 の 気 持 ち を 素 直 に 表 す こ と が 苦 手 。

悪 気 は な い ん だ け ど 、 表 現 が 下 手 だ か ら 、 自 分 自 身 の こ と を 人 ご と の よ う に 話 し た り す る か ら 、 他 人 に は 自 分 で 何 も で き な い よ う に 感 じ ら れ 、 冷 た い 感 じ に う つ る 場 合 も 。

人 と の つ き あ い が 苦 手

自 然 の 流 れ に 逆 ら わ ず 生 き て い く タ ゗ プ で も あ る け ど 、 自 分 の 意 志 を 持 っ て 否 定 的 な 一 面 も あ る 。

一 見 、 穏 や か な 人 格 者 に 見 え る け ど 、 不 意 に 反 抗 的 な 態

度 を と っ た り す る こ と も 。

あ と 、 物 事 の 本 質 を は っ き り と 見 極 め る の が 苦 手 で 判 断 力 に 欠 け る と こ ろ も あ る 。

や さ し さ を も つ

Ａ Ｂ 型 女 性 は 、 繊 細 で 鋭 敏 な 面 を 持 っ て い る 。

思 い や り も あ っ て ボ ラ ン テ ゖ ゕ 精 神 が 旺 盛 だ か ら 、 人 か ら 尊 敬 さ れ る 事 が 多 い 。

■ Ａ Ｂ 型 女 を 落 と す Ｌ Ｉ Ｎ Ｅ テ ク

Ａ Ｂ 型 女 を 一 言 で 言 う と 、

合 理 主 義 で 、 熱 し 過 ぎ な い タ イ プ 。 恋 愛 に 関 し て は 一 目 惚 れ よ り も 、 そ ば に い る こ と で 徐 々 に 好 き に な っ て い く 。

で は 、 Ａ Ｂ 型 女 に は ど ん な Ｌ Ｉ Ｎ Ｅ テ ク が 有 効 か ？

ま ず 、 一 目 ぼ れ は し な い タ ゗ プ な の で 、 Ｌ Ｉ Ｎ Ｅ も 根 気 よ く 付 か ず 離 れ ず の 感 覚 を 持 っ て 臨 も う ！

ま た 、 Ａ Ｂ 型 は お し ゃ れ で セ ン ス が 良 い 。

だ か ら 男 で あ っ て も 、 お し ゃ れ セ ン ス を 磨 く こ と を 忘 れ て は い け な い ！

基 本 的 に 『 合 理 的 』 で 、 『 燃 え に く い 』 か ら 、 燃 え る よ う な 恋 よ り も 、 友 達 か ら 恋 人 に 昇 格 ・ ・ と い う 落 ち 着 い た 恋 愛 が 向 い て い る 。

だ か ら Ａ Ｂ 型 女 に Ｌ Ｉ Ｎ Ｅ す る な ら 、 友 達 感 覚 を 忘 れ ず に 、 男 友 達 に 送 る よ う な Ｌ Ｉ Ｎ Ｅ で 誘 っ て し ま お う ！
