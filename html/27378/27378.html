　 人 は 誰 も が 生 徒 で あ り 先 生 で す 。 親 は 子 を 育 て な が ら 、 子 か ら 大 切 な 事 を 学 び ま す 。

教 え る 事 と 、 教 わ る 事 の 間 に は 理 屈 も 立 場 も 存 在 し ま せ ん 。 盲 目 の 乞 食 の 一 日 に 大 企 業 の 社 長 が 学 び 、 女 王 蟻 の た め に 死 ぬ ま で 行 進 を 止 め な い 蟻 を 見 て 愛 と 忠 義 を 学 ぶ ヤ ン キ ー が い る 。

　 食 う 者 も 食 わ れ る 者 も 、 奪 う 者 も 奪 わ れ る 者 も 。 殺 す 側 も 殺 さ れ る 側 も 、 宿 命 づ け ら れ た 立 場 や 環 境 の 一 切 を 切 り 離 し 、 学 び た い 。 そ う 思 っ た 瞬 間 か ら 人 は 世 界 中 に 散 ら ば る 全 て の も の か ら 学 ぶ こ と が で き る の で す 。

　 さ て 、 多 少 話 が 脱 線 し て し ま い ま し た が 、 あ な た を 極 上 の 人 生 へ と 導 く 先 生 の 条 件 に つ い て 。

　 そ の 先 生 は 、 生 き て い よ う が 死 ん で い よ う が 、 生 物 で あ ろ う と 無 生 物 で あ ろ う と 、 犯 罪 者 で あ ろ う が 坊 主 で あ ろ う が 年 齢 、 性 別 、 社 会 的 地 位 に か か わ ら ず た っ た １ つ 、 た だ そ れ だ け を あ な た に 実 践 さ せ る 事 さ え で き れ ば そ の 先 生 は あ な た に と っ て 最 高 の 先 生 に な る 。

　 な ぜ な ら ば 、 そ れ が あ な た の 人 生 を 極 上 へ と 導 く こ と に な る か ら で す 。 そ の た っ た １ つ の 教 え 、 そ れ は 【 あ な た に 手 書 き を さ せ る こ と 】 。

　 あ な た の 嬉 し か っ た こ と 、 悲 し か っ た こ と 。 あ な た が 隠 し 続 け て き た 本 当 の 気 持 ち 。

あ な た の 人 生 そ の も の を 、 パ ソ コ ン の キ ー ボ ー ド で は な く 、 ボ イ ス レ コ ー ダ ー で も な く 、

紙 や ノ ー ト に 、 か け が え の な い 、 あ な た の

「 手 や 指 」

を 使 っ て 記 録 を さ せ る と い う こ と 、 た だ そ れ だ け を あ な た に 習 慣 づ け さ せ る そ れ さ え で き れ ば 先 生 の 勝 ち で す 。 後 は 教 え 子 で あ る 、 あ な た が 勝 手 に 答 え を 見 つ け る 事 が 分 か っ て い る か ら で す 。

　 あ な た が こ の 先 で 出 逢 う 「 誰 か 」 、 あ る い は 「 何 か 」 。 そ こ で も し も あ な た が

「 手 書 き し て み て も い い か な ・ ・ ・ 」

そ う 思 っ た 時 は 、 必 ず そ の チ ャ ン ス を モ ノ に し て 下 さ い 。 そ れ が あ な た の 人 生 を 極 上 へ と 導 く こ と に な り ま す 。

　 世 の 中 に は 素 晴 ら し い 「 実 績 」 を 持 っ た 人 達 が た く さ ん 存 在 し て い ま す 。

で す が 、 彼 ら の 言 葉 は 私 の 耳 を 素 通 り し ま し た 。

　 も ち ろ ん 私 に も 問 題 は あ っ た の で し ょ う 。 私 は 今 、 手 書 き の 習 慣 化 を 身 に 着 け た お か げ で 、 自 分 の 気 持 ち だ け で な く あ り と あ ら ゆ る こ と を 手 書 き す る よ う に な っ た 。 そ の 結 果 、 今 ま で バ ラ バ ラ に 学 ん で き た 事 が パ ズ ル の ピ ー ス を 埋 め て い く よ う に 、 自 分 の 中 に し っ く り 落 ち て く る 感 触 を 掴 め る そ ん な 体 験 を し ま し た 。

自 分 を 向 上 さ せ る 方 法

人 は 生 き て い る 限 り 向 上 し 続 け て 行 く べ き モ ノ だ と 思 い ま す 。

ど の よ う な 形 で あ れ 「 向 上 し た い 」 と 願 う 人 は 多 い の で は な い で し ょ う か 。

　 で す が 、 実 際 に ど の よ う に 向 上 し て い っ て い い の か 分 か ら な い 。 と い う 人 が 多 い の で 、 そ の 方 法 は た く さ ん あ り ま す し 私 が 知 ら な い 方 法 も あ る と 思 い ま す 。

　 で す か ら こ こ で は 「 自 分 を 向 上 さ せ る 方 法 の 一 つ 」 を 書 い て 行 き ま す 。

そ の 方 法 と は 「 好 奇 心 を 持 つ 」 と い う 事 で す 。 好 奇 心 は 人 間 の 本 能 的 欲 求 の 一 つ で す 。

・ 生 存 本 能

・ 群 居 衝 動

・ 自 己 重 要 感

・ 性 欲

・ 好 奇 心

が 「 本 能 的 欲 求 」 に あ た り ま す 。 マ ズ ロ ー の 欲 求 階 層 説 と は 違 い ま す 。 話 を 戻 し ま し て

「 好 奇 心 」 を 持 つ と い う 事 は 本 能 的 な 欲 求 で あ る と 共 に 人 を 成 長 さ せ る 要 素 で も あ る の で す 。

　 少 し 大 げ さ か も し れ ま せ ん が 人 間 が こ こ ま で 「 進 化 」 し た の は 「 好 奇 心 」 が あ っ た か ら で は な い で し ょ う か 。

　 仮 に 私 た ち の 祖 先 が 「 火 」 に 興 味 を 示 さ な か っ た と し た ら こ こ ま で の 進 化 は あ っ た の で し ょ う か 。

　 私 は 学 者 で は な い の で 一 方 的 な モ ノ の 見 方 は 控 え ま す が も し か し た ら そ う な ら な か っ た の か も 知 れ ま せ ん 。 そ れ と 同 じ よ う に 「 好 奇 心 」 を 持 つ 事 で 学 ん だ り 経 験 し た り し て い く 「 原 動 力 」 に な る ハ ズ で す 。 そ の 学 ん だ 事 や 経 験 の 多 く は あ な た を よ り 一 層 の 自 己 成 長 に 導 い て く れ る で し ょ う 。

　 よ く 言 わ れ る 言 葉 で す が 仮 に 何 か に 対 し て 失 敗 し た と し て も そ の 経 験 と 向 か い 合 う 事 で 多 く の 事 を あ な た に 教 え て く れ ま す 。

そ も そ も 「 失 敗 を し て は い け な い 」 と い う 方 が 間 違 っ て い る の で す 。

「 失 敗 は あ っ て 当 た り 前 」 な の で す 。 人 生 で 一 度 も 失 敗 が な い 人 な ん て お そ ら く 、 こ の 世 に は い な い の で は な い の で し ょ う か 。 多 く の 人 は ト ラ イ ＆ エ ラ ー を 繰 り 返 し て 成 長 し て い く の で す 。

　 ま ず 「 ト ラ イ 」 が な い 事 に は 成 長 し て い く 事 は 難 し い で す 。 そ の 「 ト ラ イ 」 を す る に は

ま ず 「 興 味 を 持 つ 事 」 な の で す 。 そ う し て い く 事 で き っ と あ な た は 多 く の 経 験 か ら 多 く の 事 を 学 ん で い け る ハ ズ で す 。

ど う や っ て 興 味 を 持 つ か ？

じ ゃ あ ど う や っ て 興 味 を 持 つ べ き な の か ？ そ も そ も や り た い こ と な ん て な い の に 、 興 味 を 持 て と 言 わ れ て も い き な り パ ッ と 思 い つ か な か っ た り す る で し ょ う 。

　 も し か た ら す で に 興 味 の あ る も の や 、 行 っ て み た い 場 所 な ど あ る か も し れ ま せ ん 。 そ れ な ら そ れ で い い の で す が 、 今 回 話 す の は 、 「 自 分 が ど ん な こ と に 興 味 が あ る か ？ 」 に つ い て お 話 し ま す 。

　 そ れ は 様 々 な 方 法 が あ り ま す 。 例 え ば 人 に 話 を 聞 い て み る こ と 。 友 人 や 親 、 仕 事 仲 間 に 「 最 近 ど ん な こ と を し た か 」 を 聞 い て み た り す る と い い か も し れ ま せ ん 。 周 り の 人 が ど ん な こ と を し て 、 ど ん な こ と に 興 味 を 持 っ て い る の か 聞 い て み る と 、 案 外 自 分 で も 、 「 お 、 そ ん な も の が あ る の か 」 と 知 ら な い 情 報 が た く さ ん 入 っ て く る も の で す 。 も し 興 味 を 持 て ば 、 話 を 聞 い て 一 緒 に 手 伝 っ て く れ た り 付 い て き て く れ た り す る と 、 仲 良 く も な る の で 一 石 二 鳥 で す よ ね 。

　 で す が 、 そ も そ も そ う い っ た 人 た ち が い な か っ た り 、 も し く は そ う い っ た 人 間 関 係 が め ん ど く さ い 、 ま た は 、 プ ラ イ ベ ー ト で ゆ っ く り そ う い う の を 見 つ け た い と い う 方 も い る で し ょ う 。 そ ん な あ な た に お 勧 め す る の は 、 「 本 屋 に 行 く こ と 」 で す 。

　 な ぜ 本 屋 な の か ？ 本 屋 に は 様 々 な 種 類 の も の が あ り ま す よ ね 。 料 理 、 旅 行 、 漫 画 、 雑 誌 、 フ ァ ッ シ ョ ン 、 生 活 な ど な ど 、 何 種 類 の も の が あ り ま す 。 本 屋 に 行 っ た と き に 、 わ ざ わ ざ す べ て の ジ ャ ン ル を 見 て か ら 買 う 人 な ん て い ま せ ん 。 対 外 、 も う こ れ ！ と い っ た も の が あ り 、 そ れ を 買 い に 行 く か 、 な に か お も し ろ い も の な い か な 、 と 行 く も の も で す 。

こ の 「 な に か お も し ろ い も の な い か な 」 と い う の が 、 自 分 の 興 味 の あ る も の で す 。 本 と い う の は 自 分 の 興 味 の あ る も の を 移 す 鏡 だ と 私 は 思 っ て い ま す 。 そ も そ も 興 味 の な い も の な ん て 、 手 に と り ま せ ん 。 ジ ャ ン ル や 表 紙 、 タ イ ト ル な ど が 目 に 入 っ て 初 め て 手 に 取 る も の で す 。 教 科 書 の よ う に 配 ら れ た も の を だ ら だ ら と 読 む も の で は な く 、 「 本 当 に 興 味 の あ る も の 」 し か 手 に と ら な い は ず 。 な の で 、 な に か 新 し い も の を 探 し た い 、 興 味 を 持 ち た い と 思 う の で あ れ ば 、 フ ラ ッ と 本 屋 に 立 ち 寄 っ て み て は い か が で し ょ う か ？
