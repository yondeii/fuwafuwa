第１章　　 「宗教VS 世間」という対立

あべこべですよという結論

「あなたは、ふだん、まったくあべこべの感覚で生きていますね」と言われたら、どう思いますか？

　突然そんなことを言われても、「それってどういうこと？」と感じるでしょう。では、「あべこべ」という言葉を辞書をひいて調べてみましょう。「さかさま」「逆」という意味がでてきます。つまり、「あなたはさかさまの感覚で生きています」ということです。

「そんなことはありません。私は正常な感覚で生きていますよ」と思うでしょう。「この世の中で自分たちの考えることが、ほとんどあべこべだ」などとは、誰も思っていません。

　しかし、お釈迦さまから見れば、実際は皆、あべこべ感覚で生きています。

　では、なにとなにが、あべこべなのでしょうか？

「本当のこと」と「世の中の考え」が、あべこべです。「お釈迦さまが言うこの世の真理」と「あなたを含めた世間一般の認識」と言い換えてもよいでしょう。

　具体的な例を挙げてみましょう。

　たとえば、人生は苦しいものです。これは真理です。しかし、皆、「人生は楽しい」と思って暮らしています。

　人間はいつか死にます。これも当たり前のことです。それなのに世の中の人々は「私は死なない」という幻想を抱いて生きています。

　これがこの本の結論です。冒頭なのに、もう結論を話してしまいました。この本自体も、あべこべ感覚で進んでみようかなと思い、結論を最初にお話ししてみました。

「あべこべ」が生きにくさの原因

　今の日本は、便利さも自由さも平等さも、申し分なくそろっているといえるでしょう。しかし、皆、なんとなく生きるのがつらかったり、苦しかったり、不安だったりしていませんか？あるいは、「こんなはずではなかった」と毎日のように思ったり、「どう生きたらよいのかわからない」と悩んだりしていませんか？また、突然、自分や身内が「癌ですよ」と言われて「そんなこと、信じられない」と、悲嘆に暮れたりということも、そう珍しくはないでしょう。

　これらの生きにくさは、ぜんぶ「あべこべ感覚」が関係しています。

　しかし、まずはそれ以前に、皆、「正しい生き方」がわからなくなってしまっていることが問題です。本来、人々に「正しい生き方」を説く宗教や、学校などで教えてもらうはずの道徳が、とくに日本の社会では、ぜんぜん機能していません。

一般社会と隔絶される宗教の教え

　日本は、宗教と世間一般の暮らしとの区別がはっきりしています。宗教の教えを、ほとんど社会の中に取り入れません。

「宗教」と聞くと、皆さん、自分とは遠い世界のことだと思ってはいませんか？特別な信仰のある人だけがやるものと思っていたり、定年になって退職したあとに山に篭( こも ) ってやるようなものととらえているでしょう。一般の生活と宗教を、はっきりと区別しています。

　他の国は、ある程度は宗教が文化として社会に根づいていて、宗教を完全に無視するということはありません。たとえば、あの自由の国アメリカであっても、社会に宗教が浸透しています。一応、憲法上は政教分離ですが、誰一人として政治と宗教を分離していません。大統領も、堂々と自分の宗教を宣言します。考えてみてください。大統領に就任する宣誓のときに、聖書を持って宣誓するのですから。しっかりと「so help me god （ですから神よ、私に力を与えてください）」という言葉で、誓い文句を締めくくります。これ一つとっても、「政教分離」という感じではないことがわかるでしょう。

　しかし、日本では、総理大臣がアメリカの大統領と同じように宗教の言葉を用いて宣誓するのは無理です。宗教の重要性など、これっぽっちも考えていません。皆、厳密なまでに唯物論者で、「今さえ食べて生きていければいい」という生き方が、はっきりしています。「金儲( もう ) けこそ人生だ」ということが、社会の中ではっきりしています。宗教の教えは、少しも日本人の中に浸透していません。

学校も精神や心について教えない

　宗教を社会と厳密なまでに区別するのに加え、日本は学校の教育でも、道徳・正しい生き方について教えません。例外はありますが、精神のことや心については語ってはならない、基本的にそれらを語ることはタブーだ、とされています。

　精神や心についての教育を避けていたとしても、子どもが問題を起こしたときには道徳を教える必要がでてきます。しかし、いざ教えようとしても、道徳とはなんなのかがわかりません。そして、本当は正しい生き方を教えるのが道徳なのですが、会社で教えている規則と同じように考えていたりします。

「生き方」と「ルール」は、ちがうでしょう？それがわからないため、世間で笑われるほどおかしなことを、奇妙なことを、日本の学校では道徳や倫理として教えてしまいます。下級生には優しくしましょう、目上の人にはていねいな言葉で話しましょう、大きな声であいさつをしましょう、友達を仲間外れにするのはやめて仲良くしましょう、家に帰ったら手を洗いましょう、花壇の花をとってはいけません、電車ではお年寄りに座席を譲りましょう、街や公園にごみをポイ捨てするのはやめましょう、などなど、いろいろなことを道徳の授業で教えます。これらは要するに、日常社会のマナーとルールのことです。どうも皆さん、道徳とは犬にも猿にもわかる、単純なことだと理解しているようです。

　マナーとルールというのは、特別に教えられなくても自然に身についたりするものです。先に挙げたような内容は、道徳といえるものではありません。社会人として欠かせないマナーの羅列です。それに加えて、大きな問題があります。道徳授業には、ぜんぜん説得力がないのです。しかも、教える先生も学ぶ生徒も、道徳授業は本業に関係ない、学んでも学ばなくても進学には影響を与えない、とりあえずやらなくてはいけない副業だと思っています。

正しい生き方を教えるのが道徳

　説得力がない道徳授業では、「なぜこのような決まりを守らなくてはいけないのか」という理由を教えてくれません。おそらく理由はわからずじまいでしょう。火遊びをしてはいけません、赤信号のときに道路を渡ってはいけません、などというルールは、その理由がはっきりしているので、なんの躊( ちゅう ) 躇( ちょ ) もなく守られます。この例のように、すべての道徳の項目に対しても、はっきりとした理由がないと「守らなくてはいけない」という気持ちは起こりません。たとえば、「下級生に優しくしなさい」と言われた生徒が、「私は勉強や部活でとても忙しいです。下級生の面倒をみる暇はありません」と反論したなら、どのように答え、指導すればよいでしょうか。ただ「よいことだからやりなさい」と言っただけでは、やる気にはなりません。

　道徳とは、本来は「幸福に生きる」という、すべての生命の切望を実現できる正しい生き方のことです。それは副業ではなく、教育の本業でなくてはなりません。学校で学ぶ知識は、退職した時点でいらなくなります。しかし、人は死ぬまで生きています。死ぬまで正しい生き方をしなくてはなりません。

「間違った生き方はよくない」という言葉さえも、同語反復です。では、正しい生き方とはなんですか？正しくない生き方とはなんですか？それが道徳の課題なのです。その柱となる問題を置いておいて、いくらかのマナーを教えても道徳を学んだことにはなりません。マナーとは、普遍的な真理ではありません。国によって、地方によって、マナーはさまざまです。場所によっては、守っても守らなくてもよいものです。そのような中途半端な教えは、道徳ではありません。

　このように、学校の道徳授業は大きな間違いだらけですが、子どもはそれが正しいと思い込んでしまいます。学校で教えてもらっているときに「間違ったことを学んでいる」とは誰も思わないでしょう。「学校で教えてもらったことこそ正しい」と思いながら、間違ったことを学んでいきます。

「そのうちやります」は永遠にやらない

　道徳・正しい生き方について、皆、ほとんど考えずに生活しているという証拠があります。

　考えてみてください。たとえば、なにか信仰のある人が、「道徳を守ろう」「嘘をつかない人間になろう」と真剣に訴えかけたとしたら、皆さんはどうするでしょう？

「では、そのうちやります」と思うのではありませんか。

　今すぐそれをやろうとする人は、ほとんどいないと思います。その理由は、「道徳を守ることはよいことかもしれないけど、それほど必要に迫られていないし、まぁ、すぐにやらなくてもいいか」という気持ちになるからだと思います。「今は仕事をしてお金を儲けて、やりたいことをぜんぶやって、それが終わったらそのうちやります」と考えるのです。

　しかし、「そのうち」という時間は、永遠にやってきません。本当に大切だと心からわかっているなら、皆、今すぐに行動するはずです。そうでしょう？それをやらないと本当に困ってしまうことであれば、どんなことを差し置いてもやるはずです。

　つまり、道徳のある生き方や正しい生き方をすぐに実践しないということは、それを必要としていない証拠です。

欲を満たすことが正しいと思っている

　では、世間ではどういう生き方を正しいと思っているのでしょうか？

　世間でどんなことを言っているか、よく考えてみればわかります。

　おいしいもの、楽しいもの、快適なもの、ステキなものばかりが溢( あふ ) れていて、宣伝されています。そして、健康によいもの、長生きできる情報などが、とても好まれています。ほとんどが、欲を満たしながら長生きをすることばかり、言っているのです。つまり、「欲を満たしながら長生きをすることが正しい生き方だ」と、本心から、真面目に思っています。

　道徳、つまり正しい生き方に関することなどは、ふだんの生活にまったく登場しません。誰もそんなことは望んでいないのでしょう。ふだんの暮らしの中での優先順位でいえば、倫理・道徳などの価値は見事なまでに最下位に君臨しています。

倫理・道徳に説得力を感じない

　また、誰かが「嘘をつくな」と呼びかけて「まぁ、じゃあそのうちしましょう」と思うのは、嘘をつかずに生きることが、ピンとこないからです。皆さん、「嘘をつかずに生きましょう」と言われたとしたら、心の芯まで「その通りだ。そうしよう」と、すぐには思わないのではありませんか？嘘をつかないで生きることが、どうして正しいのか、よくわからなかったりするでしょう？

　しかし、「嘘でもなんでもついて、とにかくお金を儲けるんだ」、「カンニングでも裏口入学でもなんでもして、あの学校の試験に合格するんだ」という考えを聞くと、やるかやらないかは別として、「そう思うのは、そうだろうなぁ。わかるよ」と思うでしょう？先の「嘘をつくな」という正しい意見より、はるかにわかりやすく感じます。「替え玉でもして試験に受かってくださいよ」と言われたら、「ほんとに。できることならそうしたいよ」と思うでしょうね。そこが人間の本音です。

「とことん儲けなさい」「金持ちになりなさい」「ライバルに勝ちなさい」「成功を収めなさい」などと言われると、「わかりました。幸せになるためには当然そうすべきですね」と、思います。しかし、「欲張ってはならない」「嘘をついてはならない」「敵も許すべきです」という話は、「どうして？」「そうすると、なにかよいことがあるの？」と疑問に思うでしょう？つまり、世間でいう倫理・道徳には説得力を感じないのです。

「殺したい」という自分の願望しか考えられない

　今、日本の社会でとても残酷な事件が多いといわれていますね。たいへんな事件が、いくらでも日本の社会で起きています。

　先日も、駅で無差別に一般市民を襲った殺人事件がありました。ごくふつうの日曜日を過ごしている人たちが、たまたま駅に行ったら、突然、見知らぬ人に包丁で刺されてしまったのです。

　犯人はすぐに逮捕されました。警察が「どうしてそんなことをしたのか？この人のことを知っていて、刺したくなるような恨みでもあったのか？」と聞くと、犯人は、「いいえ。まったく知らない人です」と答えています。「では、どうしてそんなことを？」と聞くと、「誰でもいいから人を殺したかった」と供述しているといいます。まったく知らない人を、特別な理由もなく、「殺したい」というそれだけの理由で刺してしまうのです。

　マスメディアは、その動機を理解できないものだと考え、「それは実に不可解だ。理由なき殺人が社会に蔓( まん ) 延( えん ) する時代になってしまった。世の中どうなっているのか」などと騒いで報道しています。

　しかし、本当は騒ぐようなことではありません。「刺したいから刺した」「殺したいから殺した」という動機は、決して不可解なことではありません。

　人間の本音というのは、自分の願望だけ叶えば、その後のことやその他のことは考えないのです。人間の本性はそんなものです。

　ですから、たまたまどこかの子どもが人を殺したからといって、そんなに驚くことではありません。よく働いて、真面目に仕事をしていた人が、ある日突然、隣の家に強盗に入ったとしても、それは不思議なことではありません。

　報道で騒いでいる人たちも、端( はな ) から道徳を否定している世間一般の人々です。残念ながら、議論をしたからといって真理にたどりつくことはありません。しかし、落ち着いて考えてみればわかります。犯人は、ただ、テレビを見たり、ゲームをしたり、漫画を読んだり、映画を見たりしているうちに、「刺してみたいなぁ」という気持ちがでてきて実行しただけのことです。つまり、人間とはそういうものなのです。

残酷で、排他的な人の本音

　人間というのは、自分の願望さえ叶うなら、いかなる残酷なことにも労を惜しみません。これが人間の本音です。ぜひ、覚えておきましょう。

「人を刺してみたいなぁ」と思ったら、包丁を持って誰かを刺します。犯人は、「私はただ人を刺して、人を殺してみたかっただけです」と言います。これは、ただ自分の願望を叶えた生き方で、いってみれば「ふつう」の世間の生き方です。

　その人を「おかしい」と言うのは、言うほうがおかしいのであって、その人はどこもおかしなところはありません。ふつうです。

　この私の意見を奇妙に感じるかもしれません。しかし、よく考えてみてください。人を刺そうとはしない他の人々であっても、「自分の期待・希望・願望が叶うならば、なんでもやってやる。これこそが生き方だ」と思って生きているでしょう。そこには、「道徳」はありません。「正しい生き方」という指針もありません。自分の願望が人を殺すことなのかどうかだけの違いであって、根本的には同じ生き方なのです。

　このような世間一般の道徳なき生き方は、問題です。

「自分さえよければいいや」という考えは、排除や差別を生みだします。たとえば、ある家族の誰かが罪を犯したら、その家族はその地域から跡形もなく消えて、逃げなくてはなりません。罪を犯した人は一人ですが、親や兄弟や、年をとったおじいさんやおばあさんまで引っ越さなくてはならなくなります。その街ではうしろ指を指されて、もう生活できません。なぜかというと、罪を犯していない他の人は、罪を犯していない自分たちのことを「なにも悪いことをしない、とてもよい人間」と思っているからです。ですから、犯人の家族まで差別して排除しようとするのです。

　罪を犯した人とその家族は、ほかの地方に行ったとしても、行った先で名前を変えないと暮らせません。罪を犯した人のいる家族だとわかったら、また、そこにはいられなくなります。「とにかく悪者は排除しろ」、という社会です。

問題が起こってはじめて宗教に走る

　世間一般で生活している皆さんは、ふだんは、宗教の話や戒( いまし ) めには、あまり興味がありません。では、教育によって道徳・正しい生き方を教えてもらっているかというと、それもありません。

　これはつまり、どういう生き方かわかりますか？ふだんの生活に、倫理と心の安らぎの支えになるものが、まるでないということです。倫理や心の安らぎが危険な状態にあるのです。しかし、皆、そのことに気づいていません。

　そして、本当に危機感をもちはじめたとき、やっと宗教に走るわけです。自分の身になにかとんでもないことが起こって、はじめて宗教を学ぼうとします。一般社会の皆さんと宗教との関係というのは「たいへんなことが起こらない限り、関わりません」というものです。

道徳を気にしない、苦しい世の中

　誰もが道徳や生き方を気にしないために、世の中が苦しくなっています。

　たとえば、私は自分勝手に自分のために生きています。他の人も自分勝手に、自分の都合で生きています。そんな世の中ですから、私が、ふだんの生活の中で誰かの勝手な都合で殺される可能性はあります。殺されるまでいかなくても、誰かが私の部屋に空き巣や、強盗に入る可能性はあります。誰かが私の住まいの玄関キーをピッキングで壊す可能性はあります。ですから、心の安らぎがありません。誰もが心の安らぎが得られない状態で生きています。

　もし危険がまったくないのであれば、部屋に鍵をかけなくてもよいでしょう？私が外出しているときには、その時間だけでも部屋の窓を開けっ放しにして、空気がいっぱい通るようにしておけば身体にもよいです。しかし、それはできません。開けっ放しにしておいたら、空き巣に入られてしまいますからね。

　仮に窓を開け放しておいて、その窓から泥棒が入ったら、どうなるでしょう？まず、警察を呼びますね。警察はなんと言うでしょう？「どうして窓を開けておいたんだ！」「どうして鍵をかけないんだ！」「気をつけないお前が悪い！」と言って、詳しく調べようとはしないでしょう。万が一、泥棒が「私がやりました」と自首すれば犯人は捕まり、警察の方々も誇らしげに知らせてくるでしょうが、開けてあった窓から入った空き巣のことを、自分たちで必死に調べて捜すなどということは、よっぽどのことがない限りはやらないでしょう。結局、「窓を開けておいた自分が悪い」と思うしかなくなります。

　ですから、空気の入れ換えをしたいのであれば、留守のときではなく、自分が部屋にいるときに窓を開けなくてはいけません。しかし、それは寒かったり、暑かったりで、十分にはできません。部屋の空気は、なんだかすっきりしないまま生活します。これは一例ですが、すべてのことがこのように、なんだか息苦しいことになっています。

この世は幻と説く、宗教の世界

　ふだんから心の安らぎはありませんが、なにかいつもとちがうことが起こって、いよいよ心が危険な状態になると、人は宗教にすがります。

　では、宗教の世界では、どのようなことを教えているのでしょうか？

　世の中にはいろいろな宗教があって、この世のことや生き方について、いろいろなことを語っています。

　たとえばヒンドゥー教では、「この世の中は、幻想・幻覚（マーヤーmāyā ）である」、「本当の世界というのは唯我一なる魂である」と説きます。それはヒンドゥー教のかなり有名な哲学で、「アドヴァヤ／アドヴァイタadvaya ／advaita 」といわれるものです。「唯我一論」、「不二」という考えです。

　私たちには、この生きている世界で呼ばれる名前もあり、家族もいます。世の中には人間もいるし、友達もいるし、動植物もいます。そういう世界なのですが、ヒンドゥー教では「そのようなものは幻覚だよ」と教えます。「あるのは、たった一つだけ」で、それは「神の魂そのものしか、なにもない」と、そう教えています。

　ヒンドゥー教以外の一神論を語る宗教も、「この世は本物ではなく、本当の世界は別なところに神によって用意されている」と信じています。イスラム教やキリスト教など、他の一神論を語る宗教でも「この世は本物ではない」と言います。「本当の世界は、我々が永遠に生きる世界は、神様が別なところに用意している」と説きます。

「この世は仮の世界」と言われても理解できない

「この世は仮の世界です」「神として生きましょう」という宗教の教えを聞いても、世間一般の人はふつうの暮らしの中でそれを実感することはないでしょう。ヒンドゥー教で「世の中は幻覚である」と言っていても、一般人にとっては「本当にそうかなぁ」と思うばかりで、なにもわからないでしょう。

「あなたは、神の魂そのものである」と言われても、神の魂はなかなか見えませんから、「私も神で、あなたも神だ」というヒンドゥー教の有名な言葉を聞いても、どうも理解できないのです。

　だって、そうでしょう？たとえ「あなたは神である」と言われても、自分のことはふつうの人間にしか見えませんね。困っていることがあっても奇跡が起こせるわけではなく、不満やケンカも、なかなか解決できません。「私は神だなんて言われても、なにもできない人間ではないか」と思い、宗教の説く意味を理解することはありません。

　この世は「仮のもの」と言われても、世間一般の人にとってこの世は「事実そのもの」です。

一般の人にとってはこの世が本当のこと

「この世が本当」という一般社会の見方からすると、「この世は幻で、本当の世界は別にある」という宗教の教えは、へんてこでわかりません。「そんなことを思うよりは、もっとこの世の中の現実や事実を見たほうがよいのではないでしょうか」と思ったりします。

　このように、宗教と世間一般の対立というのがあります。宗教のこの世に対する考えと、世間一般でのこの世に対する考えは、まるで「あべこべ」です。ですから、なかなかかみ合いません。この、宗教VS 世間の「あべこべ」は、私がこの本でお伝えする皆さんの「あべこべ感覚」とは、趣のちがうことなのですが、わかりやすい「あべこべ」の例として、挙げておきます。

　宗教の世界は「世間一般があべこべだ」と言うし、世間は「いいえ、宗教があべこべです」と思っています。宗教心のある誰かに「嘘をつくな」と言われると、「あなた、あべこべなことを言わないでください。この世の中で、嘘をつかないで生活できるわけがないでしょう」と、世間一般の人は反論します。

　どこまでいっても、宗教と世間一般の折り合いはつきません。お互いさまです。お互いがライバル同士になって、争っています。

　お互いに「あべこべだ」と言い合うことで、結局どうなっているのでしょうか？実は「倫理的・道徳的な生き方を誰もしない」ということになっています。誰もが「とにかく、自分さえよければいい」という生き方をしています。ですから当然、世の中はたいへん危なくて、息苦しくて、心の安らぎがありません。

宗教の非論理性は問題

　宗教と世間一般の考えが「あべこべ」に隔絶してしまう原因は、「非論理性」にあります。

　宗教にはたくさんの言葉があります。例に挙げたヒンドゥー教の「すべては幻だ、幻覚だ」というのもその一つです。

　皆さん、「すべては幻だ、幻覚だ」と言われたら、その意味がよくわかりますか？世間で暮らす人にとって「事実そのもの」である世界が、すべて幻だなんて、突然言われても、それはどういうことなのかわからないでしょう？ちゃんと論理的に説明してもらわなくては困ってしまうと思います。「こうで、こうで、こうだからこうなっているのです」と論理的に説明されなければ、理解することをあきらめて「言われてもわからないよ。勝手に言ってなさいよ」と思ってしまうのも当然です。

　それなのに、宗教は論理的な説明をしません。ですから、宗教にも責任があるのです。言葉の意味を一つひとつ、きちんと論理的に説明することが大事なのに、それを行わず一方的に「これが真理だ。信じなさい」と説くだけです。

宗教の教えをたとえると

　病気になっている人がいるとします。その人は、自分がどうして病気なのか、どんな病気なのか、さっぱりわかりません。しかし、「とにかく治りたい」とだけは、強く思っています。

　すると誰かが来て、「あなた、納豆を食べなさい。そうすると病気が治りますよ」と言ったとします。言われたその人はとにかく言われた通り、納豆を食べます。しかし、病気は治りません。

　するともう一人が、「あのね、納豆を食べてはいけませんよ。あなたは病気だから、今日からほうれん草を食べなさい」と言ったとします。そこで、言われた通り、その日からほうれん草を食べることになります。しかし、病気は治りません。

　さらにもう一人が来て、「ほうれん草で病気が治っていないでしょう。あんなもの、だめです。ほうれん草ではなくてブロッコリーを食べてみなさい」と言います。ブロッコリーを食べて、それでもまだ病気は治りません。

　さらにまた、他の人が来て、「ブロッコリーとか野菜には毒が入っていますからね。魚を食べなさい」と言います。そして、魚を食べても、病気が治りません。

　するとまた一人が「魚はやめなさい」と言います。「野菜にしなさい。野菜にはいろいろよい成分が入っていますから。それから納豆を食べなさい」と言います。それで、また納豆を食べる羽目になります。

　......こんなことを、どこまで続ける気だろうかと思いませんか？しかし、世の中の宗教が言っていることは、このアドバイスとそんなに変わりません。

お釈迦さまは「一切皆空」とは言わない

　では、宗教が説く「世界は幻覚だ、嘘だ」などという世界観について、いったいお釈迦さまはどうおっしゃっているのでしょうか。仏教も、現世は「幻覚」だと説いているのでしょうか？これは、本来、結論となるテーマですが、この本はあべこべに進むことにして、まず、仏教の世界観から説明します。

　皆さん、「一( いっ ) 切( さい ) 皆( かい ) 空( くう ) 」という般若心経の言葉はご存じでしょう。大乗仏教では、よく出てくる謳( うた ) い文句で、暗記している方も多いでしょう。「一切皆空」とは、「世の中は空だ」「世の中は存在しないのだ」という意味です。

　お釈迦さまは、無条件に、なんの説明もなく「一切皆空」「世の中は存在しないのだ」とおっしゃることはありません。実は「一切皆空」というのはお釈迦さまの言葉ではなく、お釈迦さまの言葉を勝手に解釈した他の人の言葉です。お釈迦さまがおっしゃるのは、いつでもいたって理性的なことだけです。

問題は「捏造」という心のはたらきにある

　では、お釈迦さまはなんとおっしゃるでしょうか？それは、「ただ生命は、情報・対象を、そのまま認識しないで、捏( ねつ ) 造( ぞう ) して（パパンチェーティpapañceti ）認識するのだ」と説きます。

　お釈迦さまは「我々に入る情報を、そのまま、その事実を知ることをしないで、捏造して知ったことにするのですよ」とおっしゃいます。そして、「捏造することで、本当のことがわからなくなっているのですよ」と説きます。つまり、世の中が事実か幻かは関係なく、「問題は捏造することにある」とおっしゃるのです。そして「捏造」についても、証拠を挙げながら論理的に説明しています。

論理的な証拠がないものは妄想

「捏造」については、次の章で詳しく解説することにします。その前に、ここで宗教の非論理性と、論理的なお釈迦さまの教えについて、先にお話しします。

　宗教には、「有名でもよく理解できない」というような言葉はたくさんあります。先ほど挙げた「一切皆空」という言葉もその一つかもしれませんが、大乗仏教のある宗派はそれに徹して、「世の中は存在しないのだ」という考えでいます。

　しかし、大乗仏教の他の宗派は現世に否定的ではありません。キリスト教と同じように現世をとことん正当化します。それらの宗派では「すべての人々は、本( ほん ) 覚( がく ) である」と言います。つまり「生まれつき悟っている」ということです。「皆、ブッダだ」「皆、神である」というように、ヒンドゥー教と同じことを言っています。

　では、これは本当なのでしょうか？ここにも疑問符が生まれます。本覚思想を語る人たちは、私たちが神であることの論理的な証拠はなにも示していないからです。そうなると、自分の感情を語っているのと同じです。一つも事実でなく、証明もできない、だらだらとした妄想に過ぎません。

　他人の批判ばかりをするのはよくありませんが、これはとても大事なところなので言っておきます。「解脱に達する」ということは、我々の立場から言えばすごく具体的なことで、神や光になるとか、そういう大げさなことではありません。

　日本では、「我は光である」などと言う人もいますが、そんなことは経典のどこにも書いてありません。一かけらの証拠もないのに、どうしてそう言い切ることができるでしょう。そんなことを言うのは、あとで詳しく説明しますが、「私は死なない」という邪見に基づいている証拠です。「皆、神である」というのは、「私は死なない」という邪見を、言葉を変えて、なんとか守ろうとしているだけのことです。

論理的で実際的なお釈迦さまの教え

　ちょっと考えてみてください。たとえば、お寺のお坊さんに「あなたは本覚的な如( にょ ) 来( らい ) 仏( ぶつ ) だ」と言われたらどうですか？ちょっとはありがたい気分になるかもしれませんが、なにか素晴らしいよいことがあるでしょうか？言われた通り、自分が如来仏だったとしても、ふだんの生活はどうですか？試験に落第することもある、仕事も失敗する、奥さんとケンカする、友達とケンカする、いろいろうまくいかなかったりするわけです。そんなとき、お寺のお坊さんに「あなたは如来そのものだ」と言われても、なんとも思わないでしょう。いくら自分が如来だと言われても、日常の問題はなにも解決できないのですから、よいことはなんにもありません。

　いい加減なこと、無責任なことは、いくらでも言えます。いってみれば、妄想を喋( しゃべ ) るわけですから。誰にでも妄想を喋る資格はあります。

　しかし、本当のことを喋るのは、稀( まれ ) な人間にしかできません。お釈迦さまは、「世界は幻だ」とか「一切皆空」とは言いません。その代わり、「我々の認識システムは、こうなっているのだよ」と、論理的に解説します。調べればわかることを説いています。

　調べればわかることを宗教が説くなら、それは一般社会でも通用するものでしょう？お釈迦さまの論理性をもってすれば、宗教VS 世間一般という対立は解消します。

論理的でなければ、発展しない

　病気を治したい場合、皆さんはどうしますか？

　誰かがなんの証拠もなくただ言っていることを鵜呑みにはしませんね。

　そうではなく、私はなぜ病気なのか、それはどういう病気なのか、どんな原因で病気になったのか、具体的に、科学的に調べるでしょう？「あぁ、こういう原因だ。じゃあ、この原因を取り除こう」と考え、適切な処置をして病気が治ります。それがお釈迦さまの道です。

「生きることは苦」という真理には達していなくても、自分がとにかく苦しいことは誰でも知っています。苦しいから宗教の教えを乞う場合も多いです。では、苦しいからといって、神に祈れば幸福になる、般若心経を唱えれば幸福になる、護摩供養をすれば幸福になる、音楽を歌ったら幸福になる......などということを、ただ言われた通りにするなんて、そんなことをいつまで続ける気なのでしょうか？そんなことは今までずっとやってきたでしょう？しかし、苦しみはぜんぜんなくなっていません。どれだけ無駄で意味のないことか、皆さん、よくわかっているはずです。

　論理的に証拠を示せない世の宗教の教えは、あべこべ感覚に基づく邪見です。そのことをお釈迦さまは、「人間に考えられる邪見は六二あります」とおっしゃいました。今の時代に考えられる邪見は六二もありません。現代の邪見は、全知全能の神を信仰することと、「我はお釈迦さまである」と思っている愚かな教えと、神の考え方を真似ているものだけです。

　大乗仏教は日本においても伝統的なもので、日本の文化に長い間、多大な貢献をしてきています。ですから、一概に悪口を言うつもりはありません。しかし、真理を発見して世に教えた本師であるお釈迦さまの教えではありません。むしろ、お釈迦さまの教えをないがしろにして、開祖様たちが自分たちの考えに基づいて語ったものです。自分の宗派の開祖様を絶対的に信仰しますが、お釈迦さまには立場がないのです。お釈迦さまの教えを実践する人は、「小乗仏教」だと非難までしました。仏教からその教えを発見した人を追い出して、自分で乗っ取ったのです。しかし、大きな穴があきました。それで、人間のお釈迦さまの代わりに、架空の「如来」という絶対神を勝手につくりました。

　如来の特色をみると、不思議なことになにも新たな発見はないのです。他宗教で（お釈迦さまが邪見だと否定していた）信仰する絶対的神様そのものです。森羅万象を創造して支配管理する絶対的神という概念と、すべての現象は因果法則によって成り立つというお釈迦さまの教えは相反します。大乗仏教は、原始時代から無知な人々は信仰してきた絶対神に「如来」というレッテルを貼っただけです。大乗仏教の如来というのは、ただの絶対神の名義変更です。

信じなければいけないものは、嘘

　日本では、宗教を社会生活から切り離していることは先に述べました。しかし、一方では、やけに真剣に信仰しているようです。どうもちぐはぐで滑稽なのですが、たとえば、般若心経だけはお守りに使ったり、うちわに書いてあったり、湯のみに書いてあったり、しまいにはネクタイにまで般若心経がずらっとプリントされていたりと、けっこう生活に取り入れられています。

　そのような信仰は、はたして意味があるのでしょうか？

　私からみれば、「そこまで信仰が厚いのは、それが嘘だから」なのです。

　嘘は、信じなくてはだめなのです。事実は、信じる必要がありません。たとえば、「地球は丸い」ということを「信仰」している人はいますか？信仰する必要はないでしょう。それは事実ですからね。

　般若心経もそれと同じことです。「般若心経は、みんながものすごく信仰しているものです」と言っています。そして、「こんなに一生懸命、信仰しているのだから、そこにはなにか意味があるはずです」と思っています。

　しかし、あまりにも信仰が強くて、信仰すること自体にしか価値を見出せないのであれば、それは嘘に決まっています。この世の中に、なにか変わらないものがあるという嘘に、ひたすらしがみついているだけです。私に対しても「こんなにも日本人が信じているのですが。文句をつけないでください」という抗議がきます。しかし、私は事実を言っているだけです。

仏教は自由な宗教

　仏教は「こうしなさい、そうすれば大丈夫です」とは、決して言いません。自由な宗教ですから「あなたはこれを信じなければだめです」ということは、まったくありません。お釈迦さまの説く真理は、世間一般の考えとまったくあべこべですから、最初からすべてを理解して納得できるはずもありません。それでよいのです。「これくらいなら、なんとなくわかる」「これは、少しはそう思える」というように、少しずつ吸収していけばよいのです。わからないことは保留にしておいても、気にすることはありません。

　たとえば涅槃や解脱について、「どうしてもわかりません」という人もいます。そのような人に、私は「わからないものがあったら、放っておきなさい」と言います。納得がいかないものは、「私には納得がいかない」というところさえ、きちんと正直におさえておけば、答えは自( おの ) ずと出てきます。しっかりと修行を続けていけば、最終的に目指す「解脱」まで、論理的に着々と進んでいきます。最初は理解できなくても、心配することはありません。

論理的に考えることで発展できる

　これからも、日本に、世の中に、いろいろな宗教が現れるでしょう。

　しかし、どんな宗教が現れても、皆、同じことを言うと思います。そのどの宗教であっても、誰かを発展させることはないと思います。なぜなら、すべての宗教が「あべこべ感覚」に基づいた思考で、「私はいる、私は死なない」という実感しかもたないからです。これは、だらだらといくらでも喋れる、ただの妄想概念です。それでは発展できません。

　証拠を分析して、論理的に考えて、真理に達しなくてはいけません。修行によって、いろいろなものを破って、破って、最後に真理に達します。修行もなにもしないで、べらべら語られるものは、「私はそう思う」というだけの話です。そういう話にとらわれていたのでは、悟りも、解脱も、人の努力も、大切なものがすべて無になってしまいます。

　だからこそ、あべこべ思考ではない、どこまでも論理的なお釈迦さまの話に真剣に耳を傾けるべきなのです。

　さぁ、次の章から、真に発展的なお釈迦さまの世界に行きましょう。これから紹介していくのは、皆さんを成功と幸福に導く、お釈迦さまの道です。あべこべ感覚でものごとをさかさまに理解していたら、いくら努力しても幸福になるはずはありません。ものごとをありのままに知る人こそ、成功します。ありのままに知る方法を教えるお釈迦さまの道は、「勝利の道」と名づけられます。「勝利」とは、世間のあべこべ感覚で考えるとライバルに勝つことになりますね。それは勝ったとしても、負けた相手が自分の敵になるので、決して自分に安らぎを与えません。しかし、お釈迦さまが説かれる勝利の世界は異なります。誰にも奪うことができない絶対的な勝利です。俗世間の思考とちがう現実的な真理なので、よく考えて理解したほうがよいと思います。

第２章　　 捏造のメカニズム

問題は「捏造」して認識すること

「生命は、ありのままのことを知りません。ものごとを捏造して認識します」

　そうお釈迦さまがおっしゃいました。このことは前章で少し触れましたが、この生命の事実のメカニズムと問題について、詳しくお話ししていきます。

　これは大事なことで何回も繰り返しますが、問題は「捏造」なのです。世の中が事実か幻かは問題ではありません。

　ちなみに、この「捏造」という言葉は、もともとは「パパンチャpapañca 」という言葉が使われています。日本語で本来は「戯( け ) 論( ろん ) 」という言葉で訳されていますが、私は、あえてその言葉を使わないで「捏造」という単語を使っています。皆さん、ぜひ「捏造」の仕組みを勉強してください。お釈迦さまの真理を理解するためには、理解する側が捏造をしていては、うまくいかないので、一つひとつしっかりと理解していただきたいと思います。

　私がなにかを見たときに、「これは素晴らしいものですね」「これは嫌なものですね」「これはつまらないものですね」などと言ったとすると、お釈迦さまは「それはちょっとちがいますよ」とおっしゃいます。「それはちがいますよ」というのは、「あなたが見ているのは幻ですよ、幻覚ですよ」という意味ではなく、「あなたはデータを捏造したのですよ」という意味です。

捏造とは、自分の都合でつくる情報

「捏造」の意味は、データを誤( ご ) 魔( ま ) 化( か ) すことです。「捏造」するためには、データが必要です。なにもないところで、なにかを捏造することはできません。

　では、なんのために捏造をするのでしょうか？

　それは、ほかでもない、自分のためです。

「捏造」の意味を、研究者を例に説明してみます。ある研究者が「すごく素晴らしい研究開発をして、人々に称賛され、特許を取得して、お金をたくさん儲けたい」と考えたとします。そこで、目的のために手段を選ばないで、自分ではあまり努力をしないで欲望を叶えようとしたとします。そのために、他人のデータを盗んだり、盗んだデータをさらに捏造して、「こういう証拠が見つかりました」「画期的な証拠が見つかりました」など、お金儲けにつながりそうな部分を大げさにつくりあげます。いい加減でデタラメなことを論文に書いて、研究を発表します。つまり、データはあるのですが、捏造したものなのです。

　捏造するためには、自分にとって都合がよいデータだけを集めます。しかも、事実にはないことを書き加えて、インパクトのある結論をつくりあげます。これが捏造です。真実とはかけ離れた結論が導き出されます。

データがないと捏造はできない

「データ」がないと「捏造」はできません。

　たとえば、私は考古学をやったことがありませんし、考古学の世界のことはなにも知らないので、なにが重要なデータかはわかりません。考古学の学生にしても、私のことを考古学に詳しい人として認めません。そんな私が、考古学の分野で称賛されたいと思い、デマでもなんでもいいから人々があっと驚くようなことを発表しようと思ったところで、どうなるでしょう？そんなに知識がなく、重要なデータを集めることができないのですから、人を誤魔化せるほどの捏造ができません。データを集めないで、考古学についての文章をなんとなく書いて発表しても、「そんな馬鹿げた間違ったことを、なに言っているの？」と言われるだけでしょう。

　しかし、仮に縄文時代について研究しようとしている人が、縄文時代の新発見について、なにかを捏造して称賛を得ようと思った場合には、事情がちがってきます。知識がありますから、自分に都合のよいデータを集めたり、論理の飛躍を埋めるようなデータを集めることができます。捏造しようと思えば、捏造できるのです。

　つまり、捏造するためにはまずデータが必要です。そして、捏造する人は、ほかの誰でもなく、自分の都合のために捏造をします。これが捏造の二大特徴です。

すべての生命は捏造するという真理

　今、挙げた例のように、一人の人の話ではなく、もう少し幅をひろげて「すべての生命が捏造しています」と言ったら、それはどういう意味でしょう？それぞれの生命が、データを自分の都合に合わせて、好都合に書き換えているという意味になります。

　お釈迦さまは、「すべての生命は皆、データを捏造して認識してしまう。それが問題だ」とおっしゃっています。これは「一切皆空」のように、論理的な説明ができない宗教の言葉のことだけを言っているのではありません。もっと生命としての根本的なことです。

「すべての生命は捏造する」というのは真理です。その理由も、一つひとつ、論理的に説明できます。

捏造を止めれば、真理がわかる

「世界は幻だ」「一切皆空」のように、「外の世界は、存在しません」とお釈迦さまはおっしゃいません。そして、「存在する」とも言いません。

　では、どのように言うかというと「あなた方は捏造しているから、外の世界についてあれこれ考えても話になりませんよ」とおっしゃいます。

　だからといって、お釈迦さまは外の世界についての説明をしていないのではありません。外の世界についての真理を、お釈迦さまは一通り説明しています。しかし、いくら説明しても、聞き手側が捏造しているために理解しないのです。

　私たちテーラワーダ仏教がやっていることは、「お釈迦さまの言葉を、とにかくそのまま教えてあげましょう」ということです。

　そこで、まず「捏造を止めましょう」ということになります。捏造していては、いろいろな真理を正しく理解できません。

　お釈迦さまは、「理性ある人間として、なんでもかんでも捏造するのは情けないことです。捏造している限りは、あなたに私の話はわかりませんよ」とおっしゃいます。捏造を止めれば、お釈迦さまが当たり前のことを言っていることがよくわかります。事実を語っているかどうか、よくわかります。

世界は瞬間、瞬間、変化する流れ

　では、お釈迦さまが、外の世界をどのように説明したのかを紹介します。

「生命の六感も、外の情報・対象も絶えず生( しょう ) 滅( めつ ) 変化していく、流れである」。

　これがお釈迦さまの教えです。これは捏造しない人にわかる真理です。

　最初の「生命の六感」というのは「目・耳・鼻・舌・体・心」のことです。我々には六感があり、そこでものを知っています。「自分がいるんだ」ということは、六感があるから知っているのです。感覚がなければ、「自分がいる」ということもなにもありません。

　私たちには感覚があります。その感覚に、対象というデータが触れます。触れると、「知る」ということが起こります。そして、それは瞬間、瞬間、変化していく「流れ」です。お釈迦さまはそう説明しています。

　さらに、その「流れ」も、原因により生じ、原因により滅します。したがって、「ある」と言えるような、止まっている「実体」はないのだと、お釈迦さまはおっしゃいます。

　瞬間で消えるということは、「ある」とは言えないということです。流れていきますから「ある」「ある」「ある」と言うことはできません。「花がある」とか「牛がいる」などと言うことはできません。

　たとえば牛が見えたとしても、目に見える牛の現象というのは、ずっと流れているのです。牛さえもずっと流れています。私の身体さえもずっと流れているものです。この瞬間の牛と、次の瞬間の牛は、同じ牛ではありません。それぞれ、その瞬間で生まれて消えます。

　その瞬間の牛は、その瞬間だけのものです。この瞬間の私の身体も、この瞬間で生まれて消えています。この瞬間の私の身体と、次の瞬間の私の身体は、それぞれの瞬間だけのものです。これが、瞬間、瞬間に生まれては滅し、つねに変化して流れているということです。つまり「実体は成り立たない」という意味で「空」なのです。お釈迦さまが最初に「空」という単語を使いました。それは、「『実体がある』というものはないのですよ」という意味です。

捏造の仕組み

　捏造の仕組みに関する経典での説明は、仏教の知識がない人が理解するのは難しいです。ですから、とても俗的なわかりやすいたとえで説明します。

「草の束はおいしいですか？」と聞かれたら、どのように答えますか？

　　

①「おいしいです」（あなたは草を食べる人になります）

②「おいしくない、まずいです」（あなたは草を食べてみました）

③「わからない」（出された質問に答えようとする努力さえもしない人です。または食べてみたが鈍感で判断できませんでした）

　　

　どのように答えてもだめみたいですね。

　牛にも同じ質問が出されたと想像してみましょう。牛なら、プロのように詳しく答えると思います。人間に置き換えてみれば、牛はこのように答えるでしょう。

「この草の束はまぁまぁ、おいしいです。しかし、切ってから時間がたっているので、みずみずしさがありません。ほかに食べるものがない場合は一応食べますが、芝生に生えている新鮮でみずみずしい草がいちばんおいしいですから、本当はそちらを食べたいです」

　草の束はおいしいですか？という質問は、人間に対しては成り立ちません。ですから、人間が出すべき正解は、「答えは成り立たない」です。なぜなら人間は、草に対して「食べ物」という感覚がありません。同じ草の束を、人間にも牛にも見てもらったとしましょう。人間は「草だ」「家の中に持ってくるものではない」「切ったものはごみだ」「芝生なら気持ちのよいものだ」というような認識をします。牛はそれとまったくちがって、「ごちそうだ」と認識します。同じ草の束を見たところで、人間には一つの認識、牛にはそれとは異なる別の認識が生まれたのです。

　では、人間にとって「ごみだ」「芝生なら気持ちがよいものだ」と思わせたり、牛にとって「ごちそうだ」と思わせたりというカラクリをしたのは、その草の束でしょうか？そうではありません。草の束は、人間の目にも牛の目にも平等にデータを与えました。人間に一つの顔、牛に別の顔を見せたわけではありません。

　では、この認識の差がどこで生まれたのでしょうか。わかりやすいでしょう？それぞれの生命の心の中に認識の差が生まれたのです。草の束をカラスが見ます。犬が見ます。または猫が見ます。ネズミが見ます......。それぞれの生命は、自分に限った、他とちがった認識をします。それぞれの生命が自分の都合にあった認識をします。これが「捏造」ということです。

「捏造」はとても悪い、不幸の源

　人間が「草の束はおいしい」と思ったら、都合が悪いです。食べられませんから。牛が「美しいものだ」と思って草の束を鑑賞することになったら、それもまた困ります。牛にとっては、食べたほうが都合がよいです。捏造というのは、この世のデータが目・耳・鼻・舌・体・心という六感（六根）に触れた瞬間、各生命が各生命の都合によって合成して認識することです。これはとても自然に、瞬時に起こることです。

　ですから、「ありのままに見る、知る」ということは、自然には起こりません。誰一人として、ありのままの真理は知らないのです。ここまで理解すると、「じゃあ捏造は自然なことではないか、なにが悪いのか」と思われるでしょう。しかし、捏造はとても悪いのです。

「あの人は美しい、この人は不細工」「あの人は敵、この人は味方」「猫はかわいい、蛇は気持ち悪い」「お金があれば最高だ、貧乏は不幸だ」......などの思考もすべて捏造です。怒り、嫉妬、憎しみ、恨み、敵意、落ち込み、欲、傲( ごう ) 慢( まん ) 、自我（エゴ）などの煩( ぼん ) 悩( のう ) が、捏造にからんでいます。心が汚れます。

　自分が「悪い」と捏造した生命を殺したり、「欲しい」と捏造したものを独り占めしたり、また、さまざまな悪業をして不幸のほうへ進むことになります。捏造のせいで発展が止まるのです。「魚が大好物だ」と捏造する人は、魚ばかり食べます。「お金さえあれば最高に幸福だ」と捏造する人は、金儲けばかりに人生をかけます。

　ここで、捏造によって生命が束縛されてしまうことが理解できると思います。捏造が生命にとって束縛なのです。足かせなのです。智慧が発展しません。自由が得られません。幸福にはなれません。生きることに固執して無限に輪廻転生して苦しみます。ですから、お釈迦さまは、「少々努力して六感に触れるデータを一回でもよいから、ありのままに認識するようにしなさい」と推薦するのです。

　一回でも、捏造しないで、データをありのままに認識すると、悟りに達します。これこそ、生きるものにとって、この上ない幸福です。

第３章　　 「あべこべ」＝「顛倒」のメカニズム

お釈迦さまが説いた「顛倒」という真理

　この本のはじめに「皆さん、本当のこととぜんぶあべこべですよ」と言いました。「あべこべ」については、経典で「顛( てん ) 倒( どう ) 」という、もう少しかしこまった言葉で説明されています。

　お釈迦さまは、「人の心には、顛倒という問題がありますよ」と、それについて詳しく説明しました。「顛倒」の意味は、「あべこべ」と同じです。高いのに、低いと理解する。低いのに、高いと理解する。汚いのに、きれいだと勘違いする。なんでもさかさまに理解することを「顛倒」といいます。年をとっているのに、若いと思ったりすることなどが「顛倒」です。

　経典には、「比丘( びく ) （お釈迦さまの弟子たちのこと）たちよ、概念の顛倒・心の顛倒・見解の顛倒は四つある」とあります。

　お釈迦さまの言葉というのは、ずいぶんと短い中に、たくさんの意味が盛り込まれ過ぎるほどに詰まっています。この意味は、「『顛倒』には、『概念の顛倒』・『心の顛倒』・『見解の顛倒』と、三つある。そしてそれぞれの『顛倒』が、四種類ずつある」ということです。つまり、

　　

・顛倒が起こる場所は「概念の顛倒」・「心の顛倒」・「見解の顛倒」の三ヵ所である。

・「概念の顛倒」・「心の顛倒」・「見解の顛倒」それぞれの段階に、「常住」という顛倒、「楽・善」という顛倒、「我はいる」という顛倒、「身体はきれい」という四種類の顛倒がある。

　　

　ということになります。この章では、これについて、一つ、一つ説明していきます。

顛倒が起こる場所　①「概念の顛倒」

▼サンニャー・ヴィパッラーサ[sa ññā-vipallāsa] ：

　なにかを見たり、聞いたり、嗅いだり、味わったり、触れたり、思い浮かんだりした瞬間に、「それはなにか」という「想・概念」が起こります。それが、「概念の顛倒」です。

　　

　人は「なにかな？」と目で見た瞬間に、「あっ、これは○○だ」と知っています。聞いた瞬間に、それはなにか、もう知っています。ものすごく速いです。ふだん意識していませんから、「そんなに瞬間でわかるなんて、本当かな」と思うかもしれませんね。しかし、実際、音が耳に触れた瞬間に、もうなんの音かも知っています。この理解だけは、ものすごく速いです。

　そして、外のデータが六感「目・耳・鼻・舌・体・心」に触れた瞬間に知っている概念というのは、ありのままの事実ではなく、顛倒したものです。

　たとえば、「カー」という声が聞こえた瞬間に、「あっ、カラスだ」と瞬時に知っているでしょう？「ブー」と聞こえた瞬間に、「あっ、クルマだ」と知っています。そして、この瞬時に知ることは、決してかっこよいことではありません。お釈迦さまに言わせれば「これは顛倒ですよ、あなた、本当にそうかどうか調べたのですか？」と、なります。しかし、調べることもなにもしないで、瞬時に知ることが自動的に起こります。

やってはいけない顛倒　①「常住」

いつも「ある」とばかり認識する

　平家物語の「無常」が、仏教の「無常」と違う理由には、時間をとり過ぎているということもあります。鐘が鳴って「あぁ、また鐘の音が聞こえる」だとかなんだとか、考えていますね。その間に、煩悩がたくさん生まれています。仏教では、ものごとは瞬間、瞬間、変わっているとします。それが「無常」です。鐘が鳴って、どうたらこうたら、考えて煩悩を増やしたりするのは、仏教の世界とは異なります。

無常を知れば、煩悩が消える

　自分の家で生けた花が「もう枯れた」とわかるまでに、だいたい一週間ぐらいたっているでしょう？時間のとり過ぎです。時間をとり過ぎているので、その間、いっぱい煩悩が生まれています。「やっぱりこの花はきれいだなぁ」「今度は、よく生けることができた」など、いっぱい煩悩が生まれています。

　人は、「なるほど、世界は変わっているな」と気づくまでに、何年もの時間を必要とします。「花が散っているな」と気づくために、二日も三日も必要とします。

「無常」を知ると、煩悩が消えるはずです。「無常」を知った時点で、「悟り」に達します。ですから、一年かけて、一〇年かけて、四〇年かけて、「やっぱり私は変わるんだ」と気づいても遅過ぎです。その四〇年間にいっぱい煩悩が生まれているのですから。

　生まれたばかりの赤ちゃんを見て、「なんとかわいい」という感情をつくって、それから五〇年たったところで、同じ人物を見て「老人になったなぁ。ぜんぜんかわいくありません。やっぱり無常だ」と思うのは、無常の観察ではありません。生まれた赤ちゃんがその瞬間から一瞬も止まることなく、変わっていったのです。しかし我々には、朝、見る赤ちゃんも、夜、見る赤ちゃんも、同じ赤ちゃんです。変わっていません。それが一〇年（一ヵ月でも一年でも同じですが）たつと、変わったのだと突然、感じるのです。遅過ぎです。

　瞬間たりとも止まることなく変わっていく現象の変化を、キャッチしなくてはなりません。それができるなら、心は煩悩をつくれない状態になっています。今ある花を見て、無常を忘れて「美しい」と感情を引き起こしたら、それは煩悩です。一週間たって、またそれを見て、「枯れたな、腐っている、捨てましょう、花は枯れるものだ、無常だ」と思ったそれも煩悩です。そのときは、美しい花が枯れて腐って、見苦しくなったことで、怒りの煩悩が生まれています。

　遅過ぎる状態で無常に気づいても、心が汚れてしまうことはそのままです。かえって「無常は世間に対する時間的なアプローチだ」と邪見までつくります。正しく無常を知る人は、瞬間、瞬間で絶えず起こる無常をキャッチするのです。

やってはいけない顛倒　②「楽なもの、善いもの」

「苦」に対して、「楽である（善いもの）」という、概念の顛倒・心の顛倒・見解の顛倒です。これは、やってはいけません。

　　

　お釈迦さまは、真理としておっしゃいます。「一切の現象は苦であり、空しいものであり、善いとするべき価値があるものではないのに、人々は価値がある善いものと思っているのです」と。

　この世のすべては「苦」です。生きることも、もちろん「苦」です。それが真理であるのに、人は「毎日は楽しい」「世界は素晴らしい」と思ってしまいます。そこも顛倒なのです。

自分の命にいちばん価値がある

　我々にとっては、世界は価値がある、家族は価値がある、学校は価値がある、自分が集めているがらくたは価値がある......。とにかく、なんでもかんでも価値があります。壊れている古いがらくたを集めて、わざわざどこかへ持って行って鑑定までしてもらったりします。

　これはたとえ話ですが、イチローが着ていたユニフォームだと、たとえば百万円の値段がついてもおかしく思わないでしょう？イチローの汗の匂いがついていたら、もっと高くて倍の値段がついたりします。汗臭かったら、ふつうは洗ってしまうのですが。

　記録に達したときのバットだったら、どれくらいの値段がつくでしょうかね？皆さん、数百万円でもおかしく思わないでしょう？でも、元の値段はいくらですか？まぁ、特注で作ったものですから、ふつうのバットの五〜六倍はするかもしれません。しかし、それでも一本一〇万円にはなりません。野球にぜんぜん興味のない人からすれば、「原価がそれぐらいのバットに、どうして数百万円も払うのか、気がしれません」ということでしょう。

　私たちは、なんにでも価値を感じて、それで苦しんだりしています。いちばん価値を高く設定しているものはなんだと思いますか？自分の命です。人生にすごい価値を感じています。実際は、人生では、すごく苦しみます。本当は、なんでも空しいのです。あまり、どうということはない、たいしたことはないのです。しかし、皆さんに、それはわからないのです。

やってはいけない顛倒　③「我はいる」

　一切の現象は、「実体のない」無我なのに、「我である」という、概念の顛倒・心の顛倒・見解の顛倒です。これは、やってはいけません。

　　

　一切の現象には、なにか種みたいな、芯みたいなものは、どこにもありません。ぜんぶ、つかみどころのない、なにかの幻、幻覚みたいなものです。しかし、人は「なにかあるんだ」と思っています。「実体があるんだ」と信じています。自分に、ものごとに、絶対に変わらない、霊魂・魂・自我・実体があると勘違いしています。

自分という実体はない

　たいていの人は、自分に霊魂があると思っていたりするでしょう？実体がある、魂があると思うでしょう？それは、かなりの捏造です。顛倒です。あべこべです。しかし、ぜんぶすっかりあべこべですから、自分にとってはあべこべに感じないことも、顛倒のポイントです。お釈迦さまが言うこと、真理のほうが変ではないかと思ってしまうのです。

　なぜなら、いつも「自分」という気持ちがあるでしょう？「あっ、ずっとありますよ」と思うでしょう？すると「自分という絶対に変わらないなにかはありませんよ、なんて、そんなわけがないでしょう。どうしてそんなに、自我はない、自我はないと興奮して力説するのですか」と思うわけです。

　日本の学者さんは「ブッダは非我と言ったけれど、無我だとは言っていない」と言い張って、「だから、我があるんだ」と頑張ります。もし「我」があるのだったら、どうして「非我」なんて言うでしょうか？まあ、とにかく「世の中に、自分に、なにか核となる実体がある」と思うのは、たいへんな顛倒です。芯のようなもの、実体はないというのが、真理です。

やってはいけない顛倒　④「肉体はきれい」

「不浄である」肉体に対して、「浄である」と思う、概念の顛倒・心の顛倒・見解の顛倒です。これは、やってはいけません。

　　

　最後の四種類目の顛倒は、肉体に対するアプローチです。

　人は、自分の肉体は、すごくきれいなものだと思っています。ですから我々は、きれい好きで、汚いものには触らないとか、あれは汚い、これは汚いとか、清潔にたいへんこだわります。しかし、自分自身はなに者か？といえば、本当は「汚い」という、そのものなのです。

　肉体は不浄そのものです。それなのに、あまりにも捏造してしまって、清潔感で病気になるところまで行ってしまいます。

肉体をきれいと思うと進歩がない

　なぜ、肉体をきれいだと思ってはいけないかというと、それによって心が前進しなくなるからです。自分や他人の肉体に執着して、「これこそ最善・最美・最浄だ」と思うと、今、置かれている状況から前進はできません。

　最善・最美・最浄という言葉は、強引に私がつくった単語です。最美と最浄は辞書になくて、ちょっとおかしいかもしれませんが、理解しやすくなるかと思って使うことにします。「最も美しい」「最も清らか」という意味です。

　私たちは、「私たちの身体が世界一美しくて、世界一清らかで、素晴らしいものだ」と思っていると、ずっと肉体に捕まえられたままになります。

　たとえば、他人のことを「あなたは世界一美人だ、なんて美しいのか」と思ってみてください。その人から逃げられなくなります。いても立ってもいられなくて、その人のことばかりが頭の中で回転します。写真を撮って家中に貼ったりして、たいへんな騒ぎです。病気になります。前進できません。

　お釈迦さまの時代に、そういう病気になった実例が記録されています。身体をきれいだと思ってしまうと、どんなことになるのかと、本当にあった話で説明します。少し長いお話ですが、当時の仏教の事情なども併せて紹介していきます。

絶世の美女に魅せられた出家僧

　これから紹介するエピソードは、この本と同じ出版社のサンガより発行されている『お釈迦さまが教えたこと5　人生は美しく清らかに』という本でも取り上げています。そこでは"妄想から抜け出すことの難しさ"の例として紹介していますが、今回はその妄想の域に入ることの中でも、とくに"肉体に捕らわれること"という観点からお話ししていきます。

　お釈迦さまがいらした時代、インドのコーサラ国にあまりにもすごい美人がいました。名は、シリマー（Sirimā ）といいました。人々は、一日彼女と生活するために、たとえば千両のような高額なお金を払わなくてはいけない決まりになっていました。しかし、そんなに払っても一日だけです。その一日で、彼女は料理を作ったり、歌ったり、踊ったり、なんでもしてくれます。ですから、たとえすごく高い料金でも、一日だけ絶世の美人と一緒に生活できることにすごく喜びを感じます。それほど美しかったのです。

　本当は、誰もが何度も高いお金を払って、一緒に過ごしてもらいたいと思うのですが、一般人にはとても払える値段ではありません。わざとそういう値段設定にしています。王様や億万長者やほんの数人にしか、彼女を呼んでお祭りをすることはできません。

　さて、そんなとき、ある人が出家しました。仏道の世界に入ったのです。仏道の世界に入ったばかりのときに「あの絶世の美女、シリマーは、ブッダの信者さんだ」と耳にしました。出家したばかりの人は、「あの有名人も仏教徒ですか」と、とてもびっくりしました。しかし、他のお坊さんたちにとっては、シリマーもただ一人の信者でしかありません。「あぁシリマーさんも、ときどき来ます」という程度です。シリマーさんも来るし、国王も来るし、偉い○○さんも来るしで、なにも特別なことはありません。国王が来てもお坊さんたちは、いきなり立ち上がってお辞儀をしたり、瞑想をやめたりなど、そんなことはぜんぜんありません。たとえ国王が来ても、国王のほうが修行の迷惑にならないように、皆を避けながらお釈迦さまのところに行きます。仏教の世界は、そういう世界です。人間は平等です。

　この人は出家したばかりで、まだまだ成長していないし、だらしない状態でした。そして、絶世の美女と名高いシリマーは、他のお金持ちの人と当番で、お釈迦さまと比丘たちにお布施をしていました。たくさんお坊さんたちがいますからね。彼女は毎日、何人かにお布施をしていたのです。

僧が行く家は、公平にくじで決める

　お坊さんが、いつも三百人や五百人ぐらいはいました。毎日、食事をお布施している人のところに、お坊さんが振り分けられます。この十人のお坊さんは、あちらに行きなさい、この五人のお坊さんは、あの家に行きなさい、この二十人のお坊さんは、あちらに行きなさいというように、認定されているお坊さんが行く先を決めます。そのお坊さんに逆らってはいけないと、これも戒律で決められています。そこに行ってもらうと決めたその人の判断は絶対で、逆らうことはできません。言われたところに行くしかありません。シリマーのところに行くお坊さんも、そのように決められます。

　判断するお坊さんも、平等にやらなくてはなりません。個人的な好みでやってはいけないし、人の顔色を見てやってはなりません。たとえば、お布施をする在家の人々に、「サーリプッタ尊者に来てほしいですか？」と聞けば、みんな一斉に「ぜひ家に来てください」と言うでしょう。サーリプッタ尊者は一人ですから、一日一ヵ所にしか行けません。「うちこそがサーリプッタ尊者にお布施したかった」などと、在家で文句ばかりがあがったり、争いが起こってはたまりません。ですから、不満を言うのは在家も禁止。まぁ、空いていれば要求は聞いてあげますが、空いていなかったら、どうしようもない、担当のお坊さんの指示に文句を言わずに従うことになっています。

　昔のお坊さんたちは、個人的に接待を受けるのは遠慮していました。リクエストがあっても「サンガ（僧団）に、注文を出してください」と答えます。「サンガが決めて、誰かが行くでしょう」という方法をとっていました。その方が、公平で安心です。

　お布施を受けるお坊さんは、毎日くじを引きます。毎日、お坊さんたちはそのくじを取り、そこに誰の家かと書いてありますから、そこに行きます。このくじの結果は、誰にも文句を言うことはできない決まりです。

発作が起こってもお布施をしたシリマー

　あるとき、例の出家したばかりのお坊さんに、シリマーの家に行くくじが当たりました。悪いことに、という感じです。まだ、修行をはじめたばかりでだらしないので、次の日から夜も「なんてツイていることか」などと考えてしまいます。ただで見られますからね。しかしそのときは、彼女はもうそんなに若くありませんでした。老人とまではいきませんが、若くて元気はつらつで、皆を喜ばせるほどの年齢ではありませんでした。

　シリマーは、仏教の信者になってからはお布施や瞑想で忙しく、かつて一日千両ももらっていたおかげでお金の不自由はありませんでした。ですからシリマー自身、美しさの見返りに誰かに一日千両をもらうなどということは、どうでもよくなっていました。

　さて、例のお坊さんがシリマーの家にお布施をいただきに行ったとき、彼女は発作を起こしました。発作を起こして、体がたいへんなことになりました。いきなり目が変なところに向いてしまうし、口が曲がる、よだれが出る、身体が震える、とてもひどい状態です。なんの美しさのかけらもありません。

　しかし、発作で倒れても彼女は、お布施だけはちゃんと召使いに頼んでやらせました。そして、お坊さんたちに挨拶をする時間になったら、彼女は「お礼をしに行かなくては」と必死です。毎日やっていることですから、発作で倒れたその日も「私をお坊さんたちのところに運びなさい」と、召使いに命じました。皆、ぼろぼろになっている彼女をなんとかして、お坊さんたちが托( たく ) 鉢( はつ ) に来たところに運んで行きました。すると、彼女はすごく両手が震えるのに、震えた手できちんと礼をしました。

　そのあとはさすがに家来が家の中に運んで、寝かしておきました。そして、その日、シリマーは亡くなったのです。ですから、ふつうのちょっとした痙( けい ) 攣( れん ) ではなくて、心臓発作か脳出血などを起こしたのでしょうね。

最悪の姿からも絶世の美女を捏造

　例のお坊さんが見たのは、もう、ぼろぼろで醜( みにく ) くなった瞬間のシリマーでした。シリマーの人生でいちばんかっこ悪い、最悪の姿です。それなのに、修行はまだまだで、捏造は達人ですから、いちばん醜い姿を見ても「この方は、相当美人だ」と思ったのです。「体格や肌の色など、元気なときは、さぞや素晴らしかったでしょう」と思ってしまいました。実際に見たのは、最悪最低、いちばん身体の形も悪いときです。なんの見栄の一かけらもない、醜い姿。それなのに「それでも美人だ」と思ったというのですから、すごい顛倒です。

　そのお坊さんは、本気になってしまいました。そのままお寺に帰って寝込んでしまいました。お坊さんたちが、いろいろなことを言っても、ご飯も食べません。お釈迦さまはその様子を知り、「この人は相当なショックを与えなければ治らない」と思いました。お釈迦さまは「放っておけ」とおっしゃって、国王に「葬式は中止にしてください」と、メッセージを送りました。

国葬を延期したお釈迦さま

　シリマーの死体は国の財産でしたから、国葬です。お釈迦さまに言われて国王は、「困ったなぁ」と思いながらも言われた通り、葬式をとりやめました。現代人とちがって、昔は国王であっても政府であっても、お釈迦さまに言われたらその通りに従うのが当たり前でした。お釈迦さまは「遺体は、そのまま放っておきない」とおっしゃったので、国王はその通りにしました。

　お釈迦さまから国王に、「ではそろそろ葬式を出してもいいでしょう」と連絡が来たのは、一週間もたってからのことでした。遺体はもちろん、腐っていました。膨らんだり、膿( うみ ) が出て、ものすごくひどい臭いが出はじめていました。

　お釈迦さまは「では、葬式をやりましょう。私も行きます」とおっしゃって、王様の命令で、皆を集めるように伝えました。町の人々や彼女を知っている人々を皆、集めたのです。

　お釈迦さまは、比丘たちと葬式に行く際に、アーナンダ尊者に「あの比丘にも一緒に行きましょうと言いなさい」と伝えました。そこでアーナンダ尊者が、例のお坊さんに「お釈迦さまが、お葬式にはあなたも一緒に行きなさいと言っています」と言うと、「あぁ、そうですか。シリマーさんのお葬式ですね。わかりました。行きます」と答えて、一週間もご飯を食べていなかったのになんのこともなく、さっと起き上がって出かけたのです。

　一緒に行くとはいっても、お釈迦さまはメッセージを送っただけで、その人はお坊さんのランクでは最下位ですから、後ろにいるだけです。おそらくお釈迦さまの顔を見ることも、見られることもなかったでしょう。

絶世の美女も、ただの腐る肉体

　お釈迦さまと国王が会話します。「彼女は若い時代には、一日どれくらいの料金をとっていたのですか？」とお釈迦さまが質問をし、王様は「一日、千両です」と教えます。それを聞いてお釈迦さまは「では王様、これを競( せ ) りに出してください」と、指示しました。「千両で買って持っていってくださいと、競りに出しましょう。前は一日だけでしたが、今は千両出せばずっと一緒にいられますから、たいへんお得です」とおっしゃいました。国王はお釈迦さまのおっしゃる通りに、職人さんに頼んで遺体を競りに出しました。

「彼女は、前は千両払ってもたった一日だけしか一緒にいられませんでしたが、今度は千両で権利ぜんぶを与えます。ずっと一緒にいられます」と、条件をはっきり、皆に示しました。その上で「千両で買いたい人は、どなたでしょうか？」と聞くと、その場にお金持ちはいっぱいいましたが、誰も手を上げません。お釈迦さまは顔色ひとつ変えずに、「どうやら売れないみたいだねぇ。じゃあ、値段を下げてみましょう」とおっしゃって、値段を下げて競りを続けさせました。

　値段を下げても、下げても、誰も手を上げませんでした。最後には、日本式に言うとたった一円になってしまいました。お釈迦さまは「あれまぁ、一円でも買わないんだね」とおっしゃって、「では、誰か欲しい人にでもあげなさい」と指示しました。そこで最後に、「では、ただでもらう人は、どなたかいるでしょうか？」と尋ねてみました。誰も手を上げません。

　そこでとうとう、お釈迦さまが「誰もいない？そう、誰もいないでしょうね」とおっしゃって、「そんなものですよ、この身体は。こんなものに執着するのは、愚か者以外、誰でもありません。肉体は不浄なものです。愚か者が、誰かにすごい価値があると思い込んだとてしても、しょせんは腐ってしまう肉体ですよ」と厳しく説法しました。そうして、お葬式に参加して帰ったのです。例のお坊さんは、その説法を聞いてすっかり病気が治って、悟りに達しました。

肉体は不浄という真理

第４章　　 あり得ない期待、保証されない願望

あり得ない期待・願望　①「生まれないようにとの期待」

　経典には、次のように記されています。

　　

「生まれる性質」の生命に、このような期待が生じます。「私は生まれる性質のものである。願わくば、私には『生( しょう ) 』ということは起こらないことを」と期待します。しかし、これは期待したからといって達成することはできません。これが「期待は叶わない」という苦です。

　　

　私たちは、「ご出産おめでとうございます」「お誕生日おめでとうございます」などと「生」を喜んでお祝いしています。皆さんが、「これはたいへんめでたいことなのに、なぜ『苦』だと言うのか、よくわからない」と思うのは仕方がないことです。「生は苦である」という話は単純な感想でもなく、気休めに言っていることでもありません。客観的で厳密な因果法則に則( のっと ) って現象の世界を知り尽くした、智慧の完成者であるお釈迦さまの言葉なのです。一般の人は「わからない」と思っても、理解できるように頑張るしかありません。

　では、「ご出産おめでとうございます」という場合、幸福を感じているのは誰でしょうか？生まれてきた赤ちゃんには、どうでもよいことでしょう。大人が自分勝手に喜んでいるだけではないでしょうか。生まれた本人にとっては、出産そのものが耐え切れないほどの苦です。生まれるときに赤ちゃんは、"生きるか死ぬか"というほどの危険に遭遇しています。生まれてきたことは、「おめでとう」と言われるようなことではなく、実際は、「なんとか命拾いしただけのこと」です。そして、そこから成長することも、苦の連続です。赤ちゃんの状況を無視して大人が大いに喜んでいるとしても、「生は苦」です。

　生まれた子のそばで大喜びしている大人は、大物を釣った釣り人のような感じです。釣り人は、大きい魚が釣れたらお祝いするでしょう？ですから、魚が簡単に釣れることを幸いなこととしています。しかし、魚の立場になってみれば、釣られることほど恐ろしいことはありません。赤ちゃんにとっても、生まれることほど恐ろしいことはありません。

生まれるという必然からは逃れられない

　生命にとって、生まれることは必然です。「私はなんて惨( みじ ) めな人間か、死んでも生まれるなんて」と嘆いて「もう、生まれたくありません。生まれないようにしてください」と願っても、それは叶いません。「二度と生まれたくはない」、「生まれない」、「今( こん ) 生( じょう ) で終わり」と思うのは間違いで、単なる気休めです。

　誕生は「生まれ」であるということは、誰でもすぐにわかるでしょう。しかし、「生」はこれで終わりません。実際には、ちゃんと誕生した瞬間から、もう人生は"絶えず生まれつつ"です。身体と環境になにも変化がなく止まっているなら、それに慣れてしまって、落ち着くことができます。しかし身体は瞬間、瞬間、変化していきますから、なかなか慣れません。成長すると同時に、今までやってきたことをやめて、新しいことをやらなければいけないのです。それができるかできないかも、わかりません。新しいことができても、苦しいです。もし新しいことができなくなったら、何倍も苦しみます。生きることさえ嫌になります。そのような身体の変化に加えて、我々の生きている環境もつねに変わっていきます。それにも慣れなくてはなりません。ですから、「生」といえば誕生のことのみだと思ってはなりません。つねに「生」があるのです。

　たとえば、家族の誰かが病気になったとしましょう。それまでは家の中で誰も病気ではありませんでした。つまり、新しい状況が「生まれた」ことになります。すると、それを家族は「嫌だ」と思ってしまいます。生まれた状況を「苦しい」と思います。もし、その病気が治ったなら、そのときにまた新しい状況が生まれたことになります。悪いことがなくなってよいことが生まれたから、今度は「楽しい」と思います。

　しかし人生の中では、次から次へとよいことが生まれるわけではありません。ほとんどの場合、生まれるのは、悪いことです。なにより、年をとるということも、絶えず新しいことが生まれるという意味です。「老いることは楽しい」と思う人はいません。しかし、「私は二度と生まれないように、悪い出来事が生まれないように」と期待しても、それは叶わない願望なのです。

変化とは、新しい状況が生まれること

　では、嫌なものだからといって、「あぁ、私には憂いや、嘆くことや、苦しみや、悲しみや、悩みがあってほしくない。どうかこれらのことが起こりませんように」と、お祈りしたところで、どうなるでしょう？いくら祈っても容赦なくひどい目にあいます。この期待も叶いません。

あり得ない期待・願望のまとめ

　この五つは、期待して、願望して、祈ったところで、叶えられるものではありません。生命の本質です。「生きもの」たる所以( ゆえん ) です。逃げることはできません。これらが「嫌だ」ということは、いわば魚が「泳ぎたくない」というようなものです。魚が「水は嫌いだ」と言っても、いい加減にしろというようなものでしょう？病気そのものでしょう？魚が「水は嫌い」と言ったら、じゃあどうなるでしょうか？いくら嫌いでも、水の中にいなくてはいけないでしょう？大嫌いでも、死ぬまでいなくてはいけないのです。

　生きているもの、人間にとって「生・老・病・死」「憂い・嘆き・苦しみ・悲しみ・悩み」というのは、ぜんぶセットです。それをセットにして生きています。「生きる」というのは、そういうことです。年をとることです。病気になることです。死ぬことです。憂いや嘆き、苦しみ、悲しみ、悩むことです。また、死ぬことがなかったら、生きてもいません。

　人間というのは、面白いことに、生きる本質ではないものを期待します。病気にならないようにというテーマで、なにか宗教でも立ち上げたら、爆発的に人気が出ることでしょう。「特別なお祈りがあります。それをすれば病気になりません」というような宣伝も、実際に世の中にあります。仏教の一部だと言っている宗派の中には、お釈迦さまが断言的に「禁止」とおっしゃった、生命の本質に逆らうことばかりをやっているところもあります。

　しかし、「病気になりません」という宣伝を掲げていても、病気にはなります。病気になるのが生命ですから、必ずなります。それなのに、病気にならないと言っている宗派に五十万人や百万人もの人が集まったりしているという広告などを見ると、「そんなにも脳細胞がない人がいるということか」と、驚きます。

「苦」という真理・本質はなくならない

　お話ししてきた五つのあり得ない期待・願望というのは、「苦」という真理です。「苦」は本質的にあるものですから、なくなることを期待しても叶いません。

　命というのは「苦」でできています。苦がなければ生きていられません。どうして呼吸するのかというと、やめてしまうともう、耐えがたい苦しみがあるからです。どうしてご飯を食べるのかというと、ご飯を食べなければ、耐えがたい苦しみで、死んでしまうからです。ですから苦が命をつくって、苦が命を管理して司( つかさ ) どって、苦のために生きています。

　苦に生まれて、苦のために生きていて、苦で死んでしまいます、という人生は、ふつうの人生です。その人生では、決まってものすごく悔しくなり、「そんなはずではない」という生き方で生きています。ですから際限ない悪行為、災( さい ) 禍( か ) を生み出すという真理もまた、生まれてしまいます。これを破るためには、智慧を開発しなくてはなりません。

人は叶わないことばかり期待する

　この章で説明した、誰もが思うような五つの期待・願望は叶いません。しかし、お釈迦さまは、現実的な、具体的な、合理的な期待まで叶わないとはおっしゃいません。この五つだけは「放っておきなさい」と教えます。「その五つは生命の本質だから、願ったり、期待しても無駄だよ」とおっしゃるのです。

　魚なら、「水の中」というのは生きる上でのセットです。その前提をなしにすることはできません。「そこを理解した上で、他の願望は叶うのだから、ほかのところで頑張ればよいのだよ」、ということです。

　生命の本質である五つ以外の願望は、どうということもなく、実にたやすく叶います。人間の期待や願望は、いくらでも叶えることができます。

　しかし、皆さんが期待するのは絶対に叶わないと定義されていることばかりです。病気にならないようにとか、年をとらないようにとか、死なないようにとか。それは話になりません。

現実的な期待・願望は叶う

　五つ以外のことは、条件さえそろえれば叶います。

　その条件とは、「希望が現実的であること」と「因果法則をわかっていること」です。

　因果法則というのは、この希望に達するためになにをそろえばよいのか、自分になにがないのか、という因縁関係のことです。それがわかった上で、目的を目指して達成に見合うだけの努力をするのです。そうすれば、期待や願望はことごとく叶います。念のために言いますが、「祈り」では決して叶いませんよ。

　たとえば「頭がよくなるように」「能力が向上するように」「平和でありますように」などは、叶えることができます。これらはぜんぶ現実的なことでしょう？「もう少し勉強ができるように」「もうちょっと点数が上がるように」と願う場合は、では、どうすれば叶うでしょうか？「今回の試験はこんな結果だったから、次の試験ではもう少し上の、これくらいの成績を目指そう」などと、目標を決めます。そして、「じゃあ、そのためには問題集を一日○ページやろう」「一日、○時間勉強しよう」など、具体的・現実的に努力を続ければ、叶います。

「平和であるように」と期待・願望したら、具体的・現実的な行動を起こして、実現できます。祈りではなく、行動によって、叶います。

仏教の道は、必勝の道

　仏教が最終的に目指す「解脱」自体は、世間の暮らしとは関係ないかもしれませんが、この世で生きている間でも、お釈迦さまの道を歩む人は成功者です。仏教は負けの道ではありません。大胆で、具体的な期待しかつくりません。勝ちの期待です。

「私は死なないように」、「病気にならないように」と頑張るのは、必ず負ける道です。必ず叶わない願望です。そうではなくて、「もう少し仕事がうまくいくように」というような、具体的で実際的な願望なら、負けません。

一生懸命、挑戦する仏教の人

　仕事はバリバリやりますが、別にクビになっても、すっぱりさようならという感じです。「この会社がだめでも次があるさ」、そんな感じで執着しないで、頼まれた仕事は誰にも文句を言われないようにしっかりとやります。

意外と俗的なローマ法王の選ばれ方

第５章　　 私は死なないという幻想を捨てる

いつでも、生きている実感しかない

　まず、私が皆さんに理解していただきたいと思うことは、「私は死なない」という幻想についてです。人は皆、「私は死ぬわけはない」という前提で生きています。そして、これこそ顛( てん ) 倒( どう ) の起こる大きな理由、問題になってしまうのです。ですので、このことをしっかり理解しましょう。

　世間一般で、ごくふつうに生活している私たちには、死の実感がありません。生きている我々は、つねに「今・ここにいる」と実感します。自分の外側のものについても、「今・ここにある」と実感します。目の前に花が見えれば、「花は、今、そこにある」と、目で認識します。頭で妄想するときも、その瞬間には、その妄想概念が頭の中に実在します。「生きている」「そこにある」というのが前提になります。

　つまり、人間は「生きている」「ある」と思うばかりで、ふだんはまったく「死ぬ」と思うことはありません。いつも「私は死ぬわけではない」という前提で生きています。ときどきは、頭の中で「そのうち死ぬでしょう」と思うこともありますが、死はいつも決まって将来に起こる出来事です。

　瞬間、瞬間の死の実感は、ありません。「その瞬間、その瞬間に『死ぬ』ということが起こるという考えは、あり得ない」として生きています。実際、私たちの心になにが起こるのかといえば、つねに「今・ここにいる」「今・ここにある」という実感だけです。

「ある」という実感しかない

　なにかを見ても、聞いても、味わっても、嗅いでも、その裏で「今・私はここにいる」という実感があります。ずーっと、ずーっと、絶え間なく、です。「私がいる」、「私がいる」、「私がいる」という実感のもとで生きています。

　認識はいつもあって、いつもはたらいています。人の話を聞いたり、音楽を聞くと、心はそれなりに変わります。しかし、いつでも、「今、人の話を聞いている私がいる」「今、音楽を聞いている私がいる」というように、ずーっと「私がいる」、「私がいる」というほうだけが「ある」のです。外のものについても、いつでも決まって「今・ここにある」という実感なのです。

　たとえば今、飲んでいるお茶のカップを見るとします、あるいは飲んでいるもののペットボトルを見るとします。すると「あ、あるじゃない、カップが」「あ、あるじゃない、ペットボトルが」と、認識します。いつでも、必ず「ない」という実感はなくて、「ある」という実感です。

妄想が危険なのは、それが自分にとって本物だから

　私たちにとって花が見えたということは、「そこに花がある」という意味になります。「花は今、そこにある」と目で認識します。音を聞いたということは、「そこに音がある」という意味に、私たちは解釈します。「ある」という言葉で情報を受け取っているのです。これこそ、顛倒が起こってしまうポイントです。

　目の前にものがなく、頭で妄想するときも、その瞬間、その妄想概念が頭の中に実在します。昨日のこと、一〇年前のこと、あるいはあり得ないことなど、いろいろな妄想をするでしょう？人間は必ず妄想します。そして、どんなに変な妄想をしているときでも、その瞬間の自分にとっては、それが本当のことになります。実在することになっています。「妄想が危ない」というのは、そういう理由からです。

「あ、これはペットボトルだ。見えるんだから、あるじゃない？」と思うだけでは、問題が起こるわけではありません。「見えるからある」と思うのは、ただ顛倒しただけのことです。しかし、妄想の場合はちがいます。なにか変なことを妄想していると、どんな妄想であっても、その瞬間で、自分にとって「それが本物」ということになります。

妄想の世界では、思い出さえも「今・ここ」に実在する

　たとえば、一〇年前に人になにか言われて、激しく怒ったことがあったとしましょう。そのことを突然、思い出したら、また怒りがこみあげてきます。とても頭にきたことだったら、一〇年前のことを思い出して、今でも怒れるでしょう。あのときの怒りを、たちまちよみがえらせることができるでしょう。

　もしかしたらその相手は、死ぬ間際になにか強烈に嫌なことを言い遺していったかもしれません。自分はかんかんに怒ったけれど、相手はもう死んでしまって、反論もできません。あまりにもひどいことを言われたので、死んで一〇年たってからでも、思い出してしまって、具合が悪くなるほどに怒りがこみあげます。

　しかし、激しく怒ったことは、もう一〇年も前のことなのですから、今、怒らなくてもよいでしょう？そのことは、もう過ぎ去って今はどこにも起こっていないことでしょう？ひどいことを言った人もこの世にはいないし、二度と同じことは起こりません。つまり、「もう実在していないのにどうして怒るのですか？」という話です。なぜ怒るかといえば、妄想が、その人にとっては実在しているからです。

「みんな、私のことは嫌いなんだ」「みんな、私のことを批判しているんだ」「私にはどうせ、なんにもできない」などと人は妄想します。「周りの人は皆、悪人だ。裁かなければならない」などと妄想します。「あの人は私のことを好きだ。私に夢中だ。私を愛しているんだ」などと妄想します。「私は特別な人間だ。特別に恵まれている。神様が私に悪人を裁くようにと頼まれているのだ」などと妄想します。いかなる妄想であっても、その妄想をしているとき、本人にとってそれは単なる妄想ではありません。実在する出来事です。ですから、見事なまでに妄想に忠実に、社会に対して危険なことを犯したりするのです。

存在の真理は推測できても実感はできない

「自分」というものは、ずーっと流れているプロセスです。たとえば、細胞がずーっと壊れていって、絶えず入れ替わっていますね。血液が流れて、どんどん身体が新しくなっていきます。同様に、音を聞くたびに自分は変わっていきます。ものを見るたびに自分が恐ろしいほどの勢いで変わっています。そのように、「自分」というのは、瞬間、瞬間で変わっていくプロセスです。

　しかし、今の説を聞いて「そうか、なるほどプロセスかもなぁ」と推測はできても、実感はありません。実感できるのは、「自分がいる」「自分がいる」という顛倒だけです。あべこべ思考のもとになる、あべこべ感覚の実感だけです。私が何度も、何度も「自分は瞬間で変わっていますよ。ものごとはすべて瞬間、瞬間で生滅していますよ」としつこく言うと、「あぁ、なるほどそうかもね」と思うことはできます。しかし、いつまで聞いていても、「あ、ほんとだ。瞬間の生滅がわかりましたよ。見えましたよ」などという実感は出てきません。

存在は、映画の仕組みのようなもの

ありがたいことも、ありがたくない

死なないという前提だと、悪行為しかしない

　自分の悪行為の生き方によって、なおさら不幸に、苦難に陥ります。だって、そうでしょう？人は皆、「生きることは正しい、私は死なないんだ」と思い込んで、「生きるためにはなんでもやってしまってオーケー」という生き方をしているでしょう？

　人は、快楽ばかりを求めています。しかし、思うような快楽は得られません。すると悔しくなって、なおさら快楽を求めます。あるいはもう攻撃する、戦う、怒る、憎む、あげくに戦争までする......。それが行為でしょう？その行為には、ちゃんと適した結果が出てきます。なに一つとして、よいことをしていないのです。快楽だけを求めるというのは、ものすごく悪いことです。それから、「とことん戦ってやる」と思うことも、ものすごく悪いことです。悪いことをした結果は、悪いことに決まっています。

「私は死なない」という前提は、人に悪行為しかさせません。したがってどんどん、どんどん不幸に、不幸に、不幸になることは、たとえ神様がついていても変えることはできません。

「満足しよう」、「幸福になろう」と思ってなにかを行っても、いつも期待が外れます。期待が外れると、さらに悔しくなります。そしてさらに同じ道で戦います。すると、どうなるでしょうか？幸福になるだろうと思ってやったことが、すべて反( ほ ) 故( ご ) になってしまいます。

あべこべ感覚の結果の不幸に、際限はない

　イラク戦争にしても、ブッシュ大統領は、はじめるときには「たとえ今、この戦争に反対している国も、結果的にイラクが自由になって万々歳、民主化してアメリカ主義になったら、アメリカのことを皆、ものすごく褒めてくれるだろう」と、思っていたことでしょう。しかし、期待通りにいきませんでした。大統領のリストの中で、最低、最悪の人間であるとさえ言われているぐらいです。

　あの俳優出身で、学歴的にそれほどでもなかったレーガン大統領でさえ、そんなに悪く言われませんでした。しかし、ブッシュ大統領は、一応、エリートの大学を出ているでしょう。ですから期待通りの結果にならないと、自分の優秀な頭脳が通用しないと、なおさら悔しくなるのです。そうすると同じ道をまた選択してさらに悪循環に陥ります。それが、あべこべ感覚の結果です。

　あべこべ感覚で「私は死なない」という考えで生きると、どんどん苦しみが膨張して拡張してひどくなって、ひどくなって続きます。終わりがありません。

「生・老・病・死」はぜんぶ同じこと

　第４章で詳しく説明した「生・老・病・死」というのは、四つではなくて、実は同じものです。たとえば、命を玉だと思ってください。玉なら側面は一つしかないでしょう？紙だったら裏と表がありますが、ボールだったら一つでしょうね？その球体に「生・老・病・死」と書いてあるのと同じことなのです。ボールは同じボールで、それを見る側面によって、単語が違ってくるだけです。

　瞬間、瞬間、我々は死んでいきます。それを成長していくとも言うし、年をとったとも言います。あるいは瞬間、瞬間、変化していることを病気になったとも言うし、その変化の一過程として、肉体の流れがストップしたところで死んだと言うだけです。それが事実です。

本当のことを知れば、幸福になる

　あべこべ感覚ゆえに、なにをやっても不幸にしかならない人間は、どうすればよいのでしょう？本当は幸福になりたいのですから、自分のあべこべ感覚に気づいて、それを止めればよいのです。

　あべこべ感覚で生きていた人間が真理を知るようになると、どうなるでしょうか？真理を知ると、幸福と安らぎがあります。「すべては変わる、無常です」。「生きるとは死につつあること」と理解すると、悪循環が破れます。

「私は死なない」ではなくて、「あ、本当は、生きるとは死につつあることだ」と知ります。いわゆる「生きる」とは、「死ぬ」ことです。「死ぬ」とは、「生きる」ことです。我々は瞬間、瞬間で死んでいきます。瞬間、瞬間で生まれています。仏教では、日本の道元禅師も同じことを言っています。「生( しょう ) と死というのは、同じ意味です」と。

「死につつある」と知ると、無理に挑戦しなくなる

「私は死なない」ではなく、「私は死につつある」という真理。これを理解すると「とにかく生きてやるぞ」「生きるためならなんでもやるぞ」「欲しい快楽のためなら、戦争でもなんでもやるぞ」という悪循環がなくなります。そして、「あぁ、まぁどうってことはない。どうせ死につつあるのですから、戦うのはやめましょう」と、なります。落ち着きます。気楽になります。無理に挑戦しなくなります。

「無理に挑戦しない」というのは、死なないために、年をとらないために、病気にならないために、そのような叶わないことのためには挑戦しないということです。それはもう、魚が泳がないために挑戦することと同じですから、意味がありません。

　魚は、泳がないように挑戦するよりは、身を守るために、皆と仲良くするために挑戦すればよいでしょう？意味のないことに、無理に挑戦する必要はありません。

死がわかれば悪を犯せず、幸福になる

「私は死につつある」とわかっただけで、人は、自然と悪いことをしなくなっていきます。満足しようとしている人、自然の流れに逆らってまで頑張る人、人を脅( おど ) したり、騙してまで商売をする人、戦っている人などを見ると「なぜそこまでやるのか」という気分になります。「なにもそこまでやらなくてもいいんじゃないかな」という気分になります。死を知っている人は、悪いことをしなくなります。

　仏教をしない人に「嘘をつくな」と言っても、期待できません。本人にとっては嘘をつかずに生きるのは、すごく苦しいです。しかしちゃんと道に入り、「私は死につつある」とわかっただけで、どうなるかというと「あら、気づいてみたら、悪いことはもうやってないや」という話なのです。そういうことなのです。

「悪いことをしなくなる」というのは、大酒飲みの人が、あるとき突然、理由もなくお酒をやめるような感じです。私が「あなた、どうしてお酒をやめたのですか」と聞いても、ただ「いえ、ただ飲みたくないだけです。自分でもよくわかりません」ということです。それが本物なのです。本当に自分が治ったということです。本物で、心がきれいになったということなのです。

　世間で生活している人々が悪いことをやめるのは、至難の業( わざ ) です。苦しいです。いくら悪いことと言われることでも、好きでやっていますからね。しかし、仏教の道を進めば、だんだんに悪を犯せなくなります。すべての悪を一発で犯せなくなるかと言ったら、それはちょっと成り立ちませんから、だんだんに、です。それは逆戻りしません。ずーっとその道で、進んでいきます。

　自然に行う善行為の結果で、幸福は流れ込みます。自分が偉いことをやっているわけではなくて、ただふつうに生きているだけです。しかし、悪を犯していないのです。犯せないのです。悪を犯さない人は、善行為をしています。それで、自然に幸福という流れが生まれてきます。幸福になり、安らぎを覚えます。

仏教の言う空しさとは

　仏教は、幸福になり、心の安らぎが得られる道です。しかし、そのことと仏教がよく言う「人生は空しい」という真理とは、一見、「あべこべ」に感じるかもしれませんね。しかし、これは矛盾しません。

「人生が空しい」という意味は、わかりやすく言うと「人生に価値がない」ということです。価値がないものごとが、回転していることです。そして、お釈迦さまが、仏教が言う幸福とは、悩み苦しみがないことです。しかし、悩みがなくても、価値がないものばかりが回転している、つまり空しいものは空しいです。最終的に「空しい」とわかると、心は解脱に達します。それを究極的な幸福というのです。

仏教の幸福、気楽さ

　仏教で言う幸福について、自分を例に説明してみます。

　私は、いいかげん年をとっていて、体調もそんなに良くはありません。このことを、「そんなことは、ある程度、年をとったら当たり前のこと」と思うのは幸福な考え方です。対して「まぁ、腰が痛くて立つこともたいへんなのに、二〜三時間も立って講演しなくちゃならないなんて、私はなんて不幸なんでしょう」と思うのは、不幸な考え方です。

　私は、病気にもなります。そのとき「どうして私が病気になったのだろう」とか「もっといい、魔法のような治療法はないのか」とか「病気なのに、十分に治療する時間もないのか」とか「専門医は予約しても三ヵ月とか半年も診てくれるまでに時間がかかって、どういうことだ」などと思うとしたら、そういうのが不幸です。

　予約が半年先にしか取れないのなら「あ、そう。お医者さんは半年待たなくてはならないの？じゃあ、半年待ってやるぞ」と思うだけです。「そう言っていて、あなた、診てもらう前に死んでしまったらどうするんです？」と聞かれても、「死んでしまったら、半年後にお医者さんに会う約束は叶わなくても大丈夫ですね」で、もう話は終わりでしょうね。そういう気楽さが幸福なのです。

　気楽に生きる人にとって、世の中でなにが起こっても、それは冗談や笑いの原因であって、悩みの種ではありません。我々にとってはもうなにが起ころうが、すぐ笑えます。「あ、たいへんだ、どうしてこんなことが」ということは、ないのです。それが幸福というものではないでしょうか。

なにを見ても笑いの原因になる

　私は、人間がやっていることは面白くてたまらないです。たとえば、電車に乗ってみると、座席がちょこっとでも空いていると、いちばん先に飛んできて座るのは小っちゃな子どもたちだったりします。子どもたちは逆立ちしても、つり革へぶら下がっていても、なにをやっていても大丈夫なほど元気なのに、ちょっと席が空いていたら、さっと座ります。それを見て「なぁに？あなた方は老人なの？」と思ったりして、楽しいです。

　人間がやっている屁理屈なことというのは、いくらでもありますね。笑えないものはなにもないですよ。そういうふうに「面白いなぁ」と発見しながら生きていると、たいへんな心の安らぎがあります。

　立っていて腰が痛くなったとするでしょう？なので、座ります。座った途端に、痛みがすーっとひいていきます。「へぇ、不思議だなぁ。座ると、このように変化するのだ」などと、発見していると面白いですよ。それなのに、痛いとたいへんと思って、「たいへんなことが起きた」「たいへんなことが起きた」と、もう顔が真っ赤になって、汗をたらしてあわてています。しかし、たいへんなことは起きません。変なことなら、もういくらでも四六時中、起きますけど。

たいへんなことはなにも起こらない

　人は「検査を受けてみたら、癌だと言われた」とあわてたり、怒ったりします。でも、それはたいへんなことではありません。「あなた、別に今日、癌になったわけじゃないでしょう。下手をすると五〜六年は癌だったかもしれないでしょう。ただ、今回、知っただけでしょう」ということです。「だから、もう癌だとわかったから、これからはたいへんではないんですよ。癌だと知らない間は、本当はたいへんなことが起きていたわけですけどね。そのときは知らなかったでしょう」ということです。私からみると、人はとても非論理的です。それがまた、とても面白く感じられます。なかなか相手には言えないのですが。

　私の体調不良は、実は、最初に医者に診てもらったとき「そんなの治りません」と言われたのです。そのとき、私がどう思ったかというと「この人は医者の割には頭がよいな。私はよい医者に出会ってよかった」と真面目に思いました。なぜかというと、医者というのは言葉を濁したり、嘘ばかりついて、患者さんをよく騙すでしょう。それなのにこの医者はなんのためらいもなく、きっぱりと「治りませんよ」と言うのですから。

　私ははっきりと「治りません」と言われたものだから、言われてからは病気のことはぜんぜん訴えようともしないし、自分が病気だとも思わなくなりました。「ずっと調子が悪いのが当たり前なんだ」と思ったのです。

　面白いもので「病気があっても当たり前だ」と思うと、どうってことなくなります。病気が治ったというわけでもないです。つまりは、苦しみが消えたわけではないのですが、どうということもなくなるという、不思議さです。「人生が空しいのに、幸福の道がある」ということは、これと同じです。

幸・不幸のまとめ

　悩み、落ち込み、やり切れない気分、悔しい気分など、そういうものを抱きつつ生きることは、不幸ではないでしょうかね？

　逆に、仕事が苦しくてもなんということもなく、へっちゃらでやってしまったり、試験勉強がもう本当に苦しいけれど、「まぁ、これはやらなくてはいけないから、やっちゃおう」というふうに、なんということなく行えれば、それはまぁ、幸福なのではないでしょうか。

死の理解から、人生がひろがる

　世の中、「私はいる」「私は死なない」という考え方に基づく、ありとあらゆる邪見でいっぱいです。しかし、お釈迦さまの時代というのは、もう二千五百年も前のことです。今は、人間も科学もはるかに優れています。智慧も知識も、もう徹底的に優れているものが備わっているわけですから、皆さんは真理の情報をちょっと聞いてみれば、わけもなく、超スピーディにお釈迦さまに近づけるはずです。

「私は死なない」という顛倒を直し、「私は死につつある」と真理を知れば、精神的な病気も無知も、なくなります。それまではすごく精神的な病気だったのですが、そうではなくなります。無知もなくなります。心は健全な状態になります。したがって、能力は向上します。

　心が健全になって、能力が向上すれば、努力は実ります。第４章で挙げた五つの「あり得ない期待・願望」は叶いませんが、他のことに関しては、努力によってなんでも実ります。心の汚れが消えます。輪廻を乗り越えて、解脱に達します。

　私たちが知るべきなのは、「我は死につつある」ということです。それを理解して、そのモットーで生きることです。すると、素晴らしい結果が出ます。

　解脱に達した人こそ、死を乗り越えたことになります。「死につつある」、と理解することが、死を乗り越える第一歩なのです。
