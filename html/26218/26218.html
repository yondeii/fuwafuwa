 Money . お金の正体。⑤  

 みなさんは、「お金」という言葉を聞いてどう思うだろうか。  

 「お金があれば、欲しいものが買えるのに」といった願感情や、「どうせ働いても大したお金はもらえないし、大金持ちになれるわけでもないから最小限のお金で十分」と思う人もいるかもしれない。  

 人それぞれ、「お金」に対する考えは違うかもしれない。  

 しかし、それはよく考えれば当たり前のことで、はっきるとした定義がないのが「お金」なのだ。  

 昔は一万円の価値を一万円の物質と交換していたが、現在はどうだろうか。  

 単なる紙切れを「紙幣」として取り扱い、その「紙幣」がたくさんあればあるほど「お金持ち」ということになる。  

 しかし、その「紙幣」を製造するコストはたか だ 1 0 円 ～ 2 0 円くらいだ。  

 そして、銀行の預金残高に「存在する」と思っているお金は、実は「存在しない」のだ。  

 これは、一体どういうことだろうか。  

 実は銀行は、私たちが銀行にお金を預けた瞬間に、顧客に対し「貸し付け」を行っているだ。  

 なので、自分の通帳には数字だけが記録され、銀行の金庫にそのお金がそのまま残るわけではない。  

 他にも、銀行にお金を預けてくれた人に対して利子の支払いをしたりしている。  

 他にも、高額のローンを組みたい人に対しては、「低金利でお金を貸します」といったフレーズでローンを組ませている。  

 これは、なぜ可能なのか。  

 その答えは、国から銀行への特別優遇制度が存在するからだ。  

 国からすれば、銀行は簡単に潰れてしまっては困るので、何かしら措置をとらなければならない。  

 それが、「実際のお金より多額のお金を貸し付けてもいい」住宅ローンに繋がるのだ。  

 銀行には、実際のお金 が 10 0 万円しか存在しないとする。  

 当然、家のローンは数千万単位なので、貸し付けることは不可能だ。  

 しかし 、 10 0 万円 で 100 0 万円までなら貸し付けても法律に触れないとしたらどうだろうか。  

 銀行側は、「少ないリスク」でお金を顧客へ貸すことが出来ることになる。  

 これは顧客には知られることなく行えるので、借りた側の人間は「自分はそれだけ返済能力がある」と勘違いしながら有り難く銀行からお金を借りることになる。これが、銀行がなかなか破綻しない理由だ。  

 これが、「お金の正体」と言えるだろう。  

    

    

    

    

    

    

    

 1 . お金を稼ぐということ。  

    

 今まで「お金の正体」というシリーズを書いてきたが、ここで今一度振り返ることにしたい。  

 「お金を稼ぐ」ということを、難しく考えてはいないだろうか。  

 もちろん、何もしないでお金が手に入るわけがないのだが、現代の日本人はあまりに真面目に働き過ぎではないだろうか。  

 国家が成熟した欧州の国々では、夏休みが平均して一ヶ月はあるという。  

 これにたいして、日本の平均取得日数は５日。  

 あまりに少なすぎるのではないだろうか。  

 周りの視線を気にして、なるべく取得しないようにする風潮が何年にも渡り引き継がれている。  

    

 これが、いわゆる「空気を読む」ということなのだろうが、果たしてそれが良い効果を生むのだろうか。  

 生まないのであれば、この国はなぜそれを継続し続けているのだろうか。  

 年間の自殺者数が年々増え続けており、毎年三万人もの人間が自ら命を経つ国はどう考えても異常だと感じるのは私だけではないはずだ。  

 国も表面上は「働き方改革」などと銘打ち、企業に働き方の改善を提案しているが、それに従えるのは余裕のある一部の大企業だけだろう。  

 何年も続いた習慣を変えるには、最低でも２０年はかかるのだから。  

 では、欧州の国々ではどのようにして生活に必要なお金を稼いでいるのだろうか。  

 例えばスペインでは、陽気な気候と過ごしやすさに惹かれ多くの観光客が毎年訪れているが、そこでは週末になると観光客相手に路上パフォーマンスを行う者がたくさんいる。  

 なぜなら、路上パフォーマンスを少しの間行えば生活に必要な「お金」がチップとして手に入るからだ。  

 食料自給率 が 10 0 パーセントを越えているスペインでは、投げ銭だけで暮らしていけるのだ。  

 これを聞いて、あなたはどう感じるだろうか。  

 「国も自給率も違うし、そんな生活出来るわけがない」と思うかもしれない。  

 しかし、本当にそうだろうか。  

 確かに、世界最大の産業は「観光産業」であるのは間違いない。  

 そして、その「観光産業」最大の売り上げを誇るのがスペインなことも間違いないだろう。  

 翻って、日本はどうだろうか。  

 四季折々の美しい景色と、世界に誇る日本料理の数々。  

 そして、それを発信するメディアも無料で使える時代に生きている。  

 自分で行動せずにただ「お金が欲しい」と願うだけでは、「お金」は手に入ることはないが、自分で行動すれば「世界に自分を売り込む」ことも可能だ。  

 インターネットは、国境を簡単に越えることが可能だ。  

 「自分」というメディアを、効率的に世界に向けて発信することが出来れば、夏休みがたった５日しかない働き方を我慢せずとも幸せに暮らすことが出来るかもしれない。  

 少しでも可能性のあることを、決して否定してはいけない。  

 何が世界を変えるのかは、誰にも分からない。  

 それこそが、唯一の「真実」なのだから。  

 製造するのにたった一 枚 2 5 円ほどしかかからない一万円札。  

 そんな紙切れに踊らされる日々を過ごしたいと思う人はいないはずだが、実際のところはどうだろうか。  

 ほとんどの人が、その紙切れの為に汗水を垂らしながら働いているのではないだろうか。  

 そうなってくると、もはや「お金の為に生きている」という状態に陥ってしまうかもしれないし、とても「健康的に生きている」とは言えないだろう。  

 そういった人々は、世界中に散らばるごく一部の支配層の人達から見れば「お金の奴隷」ということになる。  

 この「奴隷」という言葉は、アメリカ南北戦争の後、「消費者」という言葉に置き換えられることになるが、本質は変わることがない。  

 これは、昔みたいに「奴隷」として扱うより「商品」を買ってもらったほうが「２度おいしい」からに他ならないからであるが、まさに現代の社会に当てはまることになる。  

 そして、中国は「戦争は絶対に起こさせない」と宣言しているが、これも軍事需要を無理に引き上げるよりも自国民が自国の製品を買ってもらったほうが「おいしい」からに他ならない。  

 しかし、それを「面白くない」と感じる人間も中にはいるわけで、そうった人達からみればなんとかして中国に戦争を起こしてもらいたいと思っている。  

 その人達は、いわゆる既得権益を持つ支配層の人間だ。  

 武器を売って稼いでいる企業は、戦争が終わってしまえば路頭に迷うことになりかねない。  

 それを避ける為に、今も世界中で火種を生む事案を意図的に作っているのかもしれない。  

    

 そんな世界で生き抜いていくためには、世界から見た日本の視点を持つことに尽きるだろう。  

 日本だけで生活しているとどうしても「日本の常識は世界の常識」だと思い込んでしまいがちだからだ。  

 世界の動きを読む力をつければ、自分が暮らしている国が経済危機に陥る前に対策をとることが出来るかもしれない。  

 それが、大きなアドバンテージになることだけは間違いない。  

    

 2 . 日本のお金。  

 今、日本では「食料不足による恐怖」と「食べ過ぎによるダイエット」が混在している。  

 これを聞いて、疑問に思った人は正常と言える。  

 もし、何も感じなければ完全に感覚が麻痺していると言える。  

 なぜならば、食べ過ぎるということは「食糧が余っている」といいうことであり、正反対の意味になるからだ。  

 ダイエットが必要なほど食べ過ぎてしまうのであれば、幸せな国とも言えるだろう。  

 今、街には「コンビニ」と「スポーツジム」、さらには「ドラッグストア」が溢れている。  

 例えばコンビニでは「精製された小麦食品」が大量に販売されており、これを食べ続ければ誰でも完全に確実に中毒になってしまう。  

 もちろん、「中毒」がカラダにいいわけもなく、精製された不自然な「小麦製品」は、人間の脳をダメにしてしまう。  

 ダメになるというのは、具体的にいうと「脳の働きが落ちる」ということだ。  

 パフォーマンスが落ちれば、思考能力も落ちていく。  

 そうなると、国がおかしい政策を打ち出しても「おかしい」ということに気がつかない。  

 これでは国が良くなる方向に向かうわけがない。  

 江戸時代までは日本人は１日二食の生活を続けていたが、エジソンが発明した「トースター」を世界中に販売したいが為に、アメリカ政府の協力のもと「パンを食べる」食文化を根付かせた歴史もある。  

 しかし、こういった歴史を公に発表することもなく紺ん日に至っているということは、どれだけテレビの洗脳力が強いのかが伺い知ることが出来るだろう。  

 現代の日本人は、「食べ過ぎている」ことに気がついていない。  

 だから、１日三食しっかり食べても物足りずにおやつなどを食べてしまうのだ。  

 そして、カラダに何かしらの異常をきたし、病気になってしまう。  

 病気になってしまうと、それ相応の治療費がかってくるので、いくらお金を稼いでも「赤字」になってしまう可能性もある。  

 スポーツ選手がなるべく怪我をしないようにするのと同じで、一般的な仕事に従事している人でも「いかに病気にならないか」を意識しながら生活をしなければ、幸せに暮らすことは難しいのだ。  

 そして、「幸せに暮らしたい」という想いは万国共通のはずで、そう思わない人などいないということは、誰しもが「病気にならない」という目標に向かって努力するべきだろう。  

 そこにも、もちろんビジネスチャンスは存在するわけで、例えば誰しもがドラッグストアで目にする「サプリメント」などは、利益が毎年増え続けている分野でもある。  

 しかし、サプリメントほど「アヤしい」モノもないと感じる人も多いだろう。  

 実際はどうなのだろうか。  

 答えは、「分からない」だ。少なくとも、日本では。  

 なぜなら専門家が日本にいないからで、もしサプリメント専門家という職業が出来て、的確に病気の症状を直したらどうなるだろうか。  

 「病院がいらなくなる」のである。  

 そうなると、もちろん「都合が悪くなる人達」がたくさんいるわけで、そういった人達の圧力が強く未だに日本には「アヤしい」サプリメントで溢れかえっているのである。  

 病院がいらなくなるだけでなく、自分にあったサプリメントを取っていれば「病気にならなくなる」ので、ドラッグストアも潰れていくことになる。  

 現在の日本では、都会を中心にドラッグストアが増え続けているので、逆に病気になっている人が増えているのだろう。  

 そんな国の未来が、明るくなるとはとても思えないのだが、みなさんはいかがだろうか。  

 どんなトッププレーヤーでも、カラダを壊したら力を発することが出来ない。  

 そして、プレーヤーとは「全ての人」に当てはまるのではないか。  

 そうなると、今までは「病気になったら治療する」といったスタンスだったのが、「病気にならないように食生に気をつける」といったスタンスに変わってくるはずだ。  

 時代が変われば、人間の意識も変わる。  

 いくら、既得権益の人達が騒いでもそれを防ぐことは出来ないのだ。  

    

 もうひとつ、日本が世界と比べて「おかしい」と感じる点がある。  

 それは、 「 C D 販売」の実態についてだ。  

 もうすでにアメリカなどの先進国では「ダウンロード販売」が主流で、大きな都市に行っても滅多 に C D ショップを目にすることはない。  

 しかし、日本では今で も C D 販売が主流で、都市部にはタワーレコードが乱立しているのが現状だ。  

 ちなみに、アメリカのタワーレコードは倒産している。  

 C D をプレスするコスト、歌詞カードを付けるコスト、ショップまで運搬するコストなど、たくさんのコストがかかっているので、当然販売価格もそれ相応に高くなる。  

 C D アルバムの販売価格が一枚三千円近くするのは、その為だ。  

 しかし、それをおかしいと思わない日本人も感覚が麻痺してしまっているのだろう。  

 そして、この状況を面白がっている国も存在する。  

 それが、「中国」だ。  

 日本に訪れる中国人は、毎年増え続けていて、京都や秋葉原を訪れると一目瞭然だ。  

 その中国から日本を訪れた人達が何を購入するかと言えば、それは 「 C D 」だ。  

 これだけ聞くと日本人の我々は普通に聞こえるかもしれないが、中国から来た人達から見れば 「 C D 」が店舗で売っていること自体が「珍しい」のだ。  

 なぜなら、中国ではすでにダウンロード販売が主流で 、 C D をきれいに陳列して販売しているリアル店舗はほとんど存在しないからだ。  

 だから 、 C D を「おみやげ」として購入し、自分の国へ持って帰るのだ。  

 こういった現状は、自分の国だけで暮らしているうちは分からないだろう。  

 ダウンロードすれば一 曲 25 0 円なのに対し、リアルな店舗 の C D ジャケット販売で は 100 0 円を越えてくる。  

 ということは、その差は音楽とは「全く関係のない」コストということになる。  

 そのアーティストが好きでも、都内まで運ぶ移動コストにまでお金を払いたいと思う人はいないだろう。  

 このことに気づく人達が増えれば、日本も変わるかもしれないが、現状でみればまだま だ C D ジャケットというリアルなモノを丁寧に棚に飾る人がほとんどなので、変化が起きるまでの期間は長いだろう。  

 そして、日本では「音楽」の立ち位置自体にも変化が起きている。  

 今まで は C D の曲を聴いて楽しむといった形が当たり前だったが、いつのまにか「握手権」などのオマケを得る為のモノへと変わってきているのだ。  

 人気アーティストですら販売数を稼がないと店舗に置いてもらえなくなる為、なるべ く C D を購入してもらうように「リアルな触れあい」を提供するのだ。  

 もちろん、 「 C D 購入」という条件付きで。  

    

 人間の三大欲求である「食欲」についても、世界ではトレンドに変化が起きている。  

    

 先進国である欧州の国々では、国民は既に「質」を求める方向へとシフトしており、大量生産・大量消費のアメリカに追従している日本は、周回遅れを喰らっているのだ。  

 「フリーレンジ」と呼ばれる、放牧しながら飼育された鳥を料理に使用する欧州では、そうでない鳥を使用した料理を口にすることは滅多にない。  

 しかし、欧州から観光などで日本を訪れた外国人が日本の料理店に入った時に、メニューを見て一様に驚くという。  

 それは、とり肉のメニューに「フリーレンジ」の記載がないからだ。  

 「フリーレンジ」の料理しか食べないといった人も多く、事実フリーレンジのとり肉でないと「美味しくない」という。  

 ここで重要なのは、「美味しくない」ということだ。  

 これは当たり前と言えば当たり前なのだが、閉じ込められながら育ったのに「健康」なわけがないので、結果的に「美味しくなくなる」のだろう。  

 しかし、大量生産する為には閉じ込めておいたほうが管理コストが抑えられるので、目先の利益に捕らわれている日本では、まだまだ浸透する気配がない。  

 しかし、逆に今「フリーレンジ」の料理をウリにした料理店をオープンすれば、オーガニック主義の人達に確実にアプローチ出来るはずだ。  

 そして、今ならライバルも少ない。  

 これをチャンスと捉えられるかどうかで、「お金を稼げるか」が大きく変わってくるだろう。  

 この本のテーマは「お金」なので、「お金を稼ぐ方法」を考えていきたいと思うが、「株式投資」や「仮想通貨」などの「架空のお金」に執着したやり方は確実性がないので、あえて省かせていただきたい。  

 何せ、デスクに向かってパソコンをイジッているだけで手に入るお金は、もし手に入ってもすぐに使ってしまう。  

 それは、「自分で行動していない」ので、手に入れた実感がなく、ましてパソコンの画面だけのやりとりで数値が上下するだけだからだ。  

 そのようなことに人生の大切な「時間」を使うのではなく、自分のやりたいことで「お金を稼ぐ」方法を模索していったほうが、人生の質は確実に高まるはずで、人としての生き方もそちらのほうが正しいと個人的には感じる。  

 最近の仮想通貨ブームによって、貴重な時間を犠牲にして「仮想」のお金に執着する人が急増しているが、それらのお金はあくまでも「仮想」であり、存在するわけではない。  

 あたかも存在するようにみせかけ、人々に投資させるのはテレビを初めとするメディア側の人間だが、投資させても責任をとることは決してないことだけは覚えておいて欲しい。  

 「お金」の正体は「共同幻想」であって、人々が「価値がある」と思い込むことによって紙幣などの通貨が運用されているわけだが、一度この信用が無くなるとその国の通貨は「紙くず同然」となるわけだ。  

 過去にいくつかの国でそういったことが実際に起きているわけだが、その前には必ず兆候がある。  

 例えば、「バブル」のような。  

 最近の仮想通貨に対する日本の反応を見ていると、そう感じずにはいられない。  

    

 ３. 大切なもの。  

 生きていく上で、最も大切なモノとは何だろうか。  

 人間が生活していく為に必要なものとして、「衣食住」が挙げられるが、この中でも「無くてはならないモノ」といえば「食」で異論はないだろう。  

 なにせ、人間は食べないと「死んでしまう生き物」だからだ。  

 着るものがなくても、南国の暖かい地域であれば暮らしていけるだろうし、住む場所も雨風がしのげればなんとかなるかもしれない。  

 どうにもならないのが「食べ物」で、飢餓などの問題が世界中で起こっているのもこの為である。  

 しかし、日本では毎日何百万トンといった食料を廃棄しており、これが飢餓で苦しむ国の人々に行き届けば問題は解決すると言われている。  

 要するに、「ある」ところにはあるのだ。  

 アメリカは、一パーセントの「富裕層」 と 9 9 パーセントの「貧困層」に分かれていると言われているが、構造的には全く一緒だろう。  

 一部の既得権益者が、富を「独占」しているからに他ならない。  

 日本のコンビニからも大量のゴミが毎日排出されるが、これも「環境」よりも「マーケティング」を優先した結果だろう。  

 このゴミがなければ、日本の焼却炉の電力は減らすことが出来るかもしれない。  

 大きなことを成し遂げる為には、まず小さなことを積み重ねていくことだ。  

 政治家が口だけで「原発をゼロに」というより、よほど効果があるのではないかと感じてしまう。  

 世界で生産される作物を使えば、世界中で起きている飢餓は克服出来ると言われている。  

 この世界は、いつイノベーションが起こり、コンビニから食料ゴミが排出されない日がくるのだろうか。  

 現代のテクノロジーを駆使すれば、無理ではないはずだ。  

 「無理」にしているのは、紛れもなく「人間」なのだから。  

 世界中の先進国でオーガニック主義が浸透していく中で、今、方向転換を求められているのは間違いなく「日本」である。  

 では、なぜその方向に舵を切れないのか。  

 問題は、そこにある。  

 かつて戦争でアメリカに敗北した日本は、アメリカの食生活を取り入れざるを得ない状況になり、パンなどの小麦製品を大量に輸入することになる。  

 アメリカは今も昔も大量生産・大量消費の国なので、「質」よりも「量」を重視する製品がたくさん存在し、それらの製品のお得意様は紛れもなく日本だ。  

 この構造を変えない限り、日本はこの負のループを抜け出すことは難しいだろう。  

 しかし、今の政治家で「アメリカからの脱却を図ろう」と言い出す人はほとんどいないだろうから、日本が変わるとしたら「他国からの圧力」だと個人的には思っている。  

 今までの日本の歴史をみると、大きな変化は必ず「外圧」によって起こってきたからだ。  

 「黒船来日」や、 「 iPhon e 」の登場が良い例だろう。  

 もはや、日本は「変わる力」を失っているのかもしれないが。  

 では、変わる為には何が必要なのだろうか。  

 まず一歩目は、「ファストフードの広告」を無くすことなのかもしれない。  

 誰しもが「体に良くない」と認識しながらも、気がつくと食べてしまっているモノ。  

 それこそがファストフードであり、もちろんアメリカから輸入されてきた文化だ。  

 すでにヨーロッパの国々では、ファストフードの広告をテレビ の C M で流そうものならテレビ曲にクレームの電話が殺到するという。  

 それほど、「体にとってよくないモノ」という認識が根付いており、人々の食に対する「健康志向」は強い。  

 年間の医療費が年々増え続けている日本も、そこを目指すべきだと強く実感する。  

 一部の既得権益者によって、日本全体が「体によくないモノ」で埋め尽くされてしまっている現状を理解し、改善していかなくてはいけないのだが、変わる為には「個人」で変わっていくしかない。  

 一人一人の力が、大きな変化をもたらすのだ。  

 「いつかよくなるだろう」といった考えで、国の機関に頼っていても、国自体が疲弊し続けるだけで変化が起こることはないだろう。  

 日本のマスメディアについても、全く同じようなことが言える。  

 高額のスポンサー料に目がくらみ、著名なタレントもそれに甘んじ、人々の生活よりも「消費活動」を優先した結果、ファストフードの新商品が発売されるやいなや、テレビ で C M が放送されるのである。  

 しかし、自分のカラダよりも大切なモノが、この世界にあるだろうか。  

    

 一度、自分の心に聴いてみて欲しい。  

    

 ４. 個人の可能性。  

 時代は完全に「個人の時代」に突入したのだ。  

 大企業の役員よりも、ツイッターでフォロワーが三千人以上いる平社員のほうが、自分の意見を発信する力は強い。  

 世界へ向けて発信すれば、興味を持った人のアンテナにひっかかり、自分の新たな可能性に気づけるかもしれない。  

 それが、世界を変えるキッカケになることも、十分ありえるのだ。  

 個人の働き方にも、大きな変化が訪れていることも忘れてはならない。  

 今まで「副業禁止」だった大企業と呼ばれる会社で、「副 業 O K 」という制度が生まれてきている。  

 これは、規制が緩和されたという見方が一般的だが、本当にそうだろうか。  

 国がこの制度を認めたのであれば、いくら大企業といえど「定年まで雇う保証はありませよ」といったメッセージが含まれているのではないだろうか。  

 サラリーマンの場合、自分が勤めていた会社が万が一倒産したりしたら職を失うことになる。  

 経営者の場合も、同様に収入が途絶えるだろう。  

 しかし、それはあくまでも「今まで」の場合だ。  

 ふたつ以上の職業を持っていれば、ひとつの収入が途絶えたところで慌てることはなくなる。  

 だからこそ、自分の好きなことにチャレンジ出来て、結果的に人生の質が高まるのではないだろうか。  

 そして、そういった人間のもとに「お金」が集まることになる。  

 そうはいっても普段仕事に追われているなかで、複数の仕事を持つなど不可能なのではないかという意見もあるだろう。  

 確かに、「ちょっと前」までは不可能だっただろう。  

 しかしテクノロジーが年々進化した現在では、少しの「空き時間」があれば「もうひとつの収入源」を持つことは不可能ではなくなったのだ。  

 例えば、誰しもがひとつは持っている「スマートフォン」。  

 そのスマートフォンにワイヤレスのキーボードを接続すれば、どこでも文章をかけるようになり、家でパソコンを立ち上げる必要も無くなった。  

 文章を書くセンスを磨けば、電子書籍として自分の本を販売し、収入を得ることも出来る。  

 これも立派な「作家」という職業ではないだろうか。  

 そしてこれはどこの国にいても活動が出来る上に、印税 も 7 0 パーセントと、とても魅力的だ。  

 自分の住んでいる国が傾いても、焦らずに行動出来る。  

 これが現代人が目指す「働き方」だと個人的には感じる。  

 これからの激動の世界で「どこでもやっていける」力をつける為に、行動しよう。  

 今、この瞬間に人は変われるのだから。
