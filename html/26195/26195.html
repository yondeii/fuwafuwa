イギリスのビクトリア時代 ( １８３７ -- １９０１ ) には、多くの迷信がクリスマスプデｲングを取り巻いていました。クリスマスプデｲングの材料は１３でなければならないとか、プデｲングをかき混ぜるときには、東から西の方向へかき混ぜる ( 東方の賢者 ) とかいった迷信が、すべての人々のあいだに信じられていたのです。迷信でなくとも儀式めいた習慣もあり、ビクトリア時代にはクリスマスプデｲングを三つ用意する習慣がありました。クリスマスの日 ( １２月２５日 ) のために大きなプデｲングをひとつ、翌日のボクシングデイ ( １２月２６日 ) のために小さなプデｲングをひとつ、そして新年の日のために小さなプデｲングをひとつ、の合計三個のクリスマスプデｲングを作る習慣があったのです。その時代の貧しい人々がクリスマスの翌朝箱を抱えて、クリスマスの残り物を乞って歩いたところから、１２月２６日を " ボクシングデイ " と呼ぶようになりました。

ｂ．自然化をする球根植物

ところでクレマチスを育てようと思う人は、まず植物を支えることのできる格子 ( トレリス ) を買い求め、壁の上に安定して備え付けなければなりません。この格子はイギリスの家のレンガの壁に完全に固定できるようにと、小さなスクリﾕ - の類がすべてそろって売っています。幸いなことにイギリス人の男性はこうした仕事がとても得意なのです。家の壁に格子を固定するのが嫌ならば、庭に格子を建てることもできます。いずれにしても格子がなければクレマチスを咲かせることはできません。これは朝顔の場合と同じです。ところでクレマチスモンタナという五月に咲く花があり、これは淡いピンクの珍しくきれいな花なのですが、このクレマチスモンタナに限っては格子だけでは足りないことがしばしば起こります。このクレマチスは野生のツル植物のように育ちすぎてしまうのです。クレマチスモンタナは１９世紀初めに、インドからイギリスに紹介された花であるとされています。それ以来クレマチスモンタナは春から初夏にかけての優秀なクライマ - として、イギリスの風景に目を見張るような美を繰り広げてきました。それはこの花が母国インドで自由奔放に咲き、インド総督の注目の的となった習性が、小さな種子の中に失われずにそのまま包蔵されたまま、イギリスに移植されたからに違いありません。私の窓から見えるクレマチスモンタナの木は、成長をしすぎた結果壁を乗り越え、庭の外に立っている電信柱を上りつめ電線にからんでいます。しかしこの様子に苦情を述べるイギリス人は一人もおらず、クレマチスモンタナはツルが切られることもなく、電線に重くたれさがって繁茂しているのです。幸いなことに風雨の日でも事故にならず、空中に花を咲かせています。この光景はいかにもイギリス人らしい寛容さ、または怠惰を示すものです。クレマチスモンタナのツル枝は、ほかのクレマチスのツル枝よりも太くて強いので、このように高さを極めることができるのでしょう。

クレマチスビタルバはイギリスに野生の花です。野原を歩く人を楽しませてくれるので、 " 旅行者の喜び " と呼ばれています。

 

 

 

 

 

 

 

 

 

 

 

 

 

 

ｆ．さんざしの賛歌

 

私がなぜ " さんざし " を褒めそやすかということには、二つの理由があります。その一つは、さんざしの美しい白い花が、日本の桜の花を私に思い出させること。もう一つは、さんざしがイギリスのカントリ - サイドに重要な役割を果たしてきた歴史があるからです。イギリスを旅行したことのある人なら誰でも、この国のカントリ - サイドの自然が、 " つぎはぎ細工 " のように仕切られている事実に注目なされたでしょう。こうしたつぎはぎ細工式風景を可能にする境界線の一つが、樹木を植えた生け垣の列です。緑の生け垣は夏のカントリ - サイドの風景と非常によく調和しているので、あたかもそれが緑の牧草や農園のように、カントリ - サイドの自明な風景として想像されてしまうかも知れません。生け垣のもつ過去の歴史や、存在理由、生け垣の樹齢、などを考えることは、旅行者でなくとも一般の人々の念頭には浮かばないものであるかも知れません。

しかし生け垣の歴史は、古代の青銅時代にまでもさかのぼる古い歴史なのです。サクソン、中世以後、歴史のすすむにつれて、生け垣作りはイギリス全国に行われるようになりました。つぎはぎ細工のような風景の歴史的背景は、今から二世紀ほど前にはじめて、さんざし ( ホ - ソ - ン ) の生け垣が植えられたことに起源を持つといわれています。その当時農地を分割する必要から、そしてまた家畜の放牧の目的のために、これまでひと続きの野原であったオ - プンフｲ - ルドが、仕切り壁に隔てられるようにさんざしの生け垣で分割されていったのです。そして１９世紀にさんざしの生け垣の建設は、最高潮に達したといわれます。さんざしの木はイギリスの風土に合い、スコﾂトランド北部をのぞくイギリス全国に植えられています。牧草地、丘陵地、森林地域のどの土地にも育ち、生け垣とするには理想的なのです。けれどもさんざしの生け垣は、人家の生け垣にはならず、カントリ - サイイドの生け垣であることを再度ふれておきます。私がここで述べているさんざしは、 " コモンホ - ソ - ン " と呼ばれる木で、美しい白い花が五月に咲き、その花が散ったあと夏の終わりから秋に、小さな赤い実が枝に沢山実ります。さんざしはバラ科の落葉低木ですから、確かにさんざしの枝はバラのようにトゲがあり、その赤い実はバラが散ったあとに残るロ - ズヒﾂプにも似ています。ロ - ズヒﾂプといえば、イギリスでは戦争中に人々に摘まれたバラの実でした。さんざしの実も小鳥たちや小さな動物の貴重な食べ物になっているのです。

土地の境界線となった古代の生け垣から発展して、カントリ - サイドの生け垣は家畜の放牧の囲いの目的を持つようになりました。生け垣は家畜の行動範囲を制限すると同時に、キツネなどの動物から家畜を守る役割も果たしてきました。さんざしの枝には鋭いトゲがあるので、動物が通過しにくく、家畜の柵としては最適なのです。ところでさんざしの周辺の自然環境は、湿度、日当たり、風の強さなどが、生け垣に仕切られた両側の自然環境とは違っています。そのためにさんざしの生け垣には、オ - プンフｲ - ルドにあまり生育しないような種類の植物が、自ずと育つようになるのです。例えばツル植物のドﾂグロ - ズと呼ばれる野バラは、さんざしの木にからんで育ちます。 ( 英語では野生の植物にドﾂグという名前をつけることがあります。ドﾂグロ - ズ、ドﾂグバイオレﾂトなどがそれです ) 。ロﾂグロ - ズはイギリスの野バラの中で、最も一般的によく見られるバラで、白い花びらが仄かに " バラ色 " に染まった花を、六月から七月にかけて咲かせます。ドﾂグロ - ズのトゲは形がカ - ブしているので、さんざしの枝のトゲと複雑にもつれ合い、そのようにしてドﾂグロ - ズのツルは、さんざしにしっかりと支えられてゆくのです。野生の桜草やすみれ、カウパセリという植物、野イチゴなど、色々な植物がさんざしの生け垣の根もとに、いつのまにか生えるのです。このことはさんざしの生け垣が、管理を必要とすることを意味しています。もし生け垣を刈り込まずに放っておくと、幾年も経たないうちにブラﾂクベリ -( 鋭いトゲが無数にある ) まで根を下ろして、さんざしの生け垣はジﾔングルのように化してしまいます。このことを考えるならば、現存する生け垣の将来は、ひとえに農場経営者の管理に依存する " 未知の " 自然なのです。

さんざしの実は野鳥に摘まれてよその土地へ運ばれてゆき、実の中の種が地に落ち、そこからまたさんざしの木が育つケ - スもあるそうです。さんざしの樹齢は１００年を超えるといわれ、さんざしの生け垣は自然研究家にとっては、野生の動植物がひとつ箇所に生息する自然観察の舞台となってくれます。そして歴史家にとっては、過去の農業の歴史を研究する際に貴重な道標となってくれる、過去の証言としての自然であるともいえるのです。

ｇ . 　芝生の中に咲くひな菊

 

ひな菊は英語ではデイジ - といいます。それはとても小さな雑草のような花です。よく見ると愛らしいので粗末には扱いたくないのですが、生える場所が芝生の中で、タンポポやクロ - バ - と一緒に仲良く咲いているので、デイジ - には菊の花のもつ風格がありません。それは指先くらいの小さな花で、色は白とピンク。春から夏にかけて日が照ると花ひらき、夕暮れには花がとじ、これを繰り返していつしか芝生の随所にひろがってゆきます。要するにそれは ( 人の立場からすれば ) 雑草なのです。イギリスの公園の芝生は夏になるとデイジ - が咲き、注意して見ると公園の敷地全体にひろがって咲いています。ここまでくると公園はきれいな緑の芝地というよりも、デイジ - の野原です。芝生を植えかえるにも費用がかかり、植えかえてもまたデイジ - が生えてくるので、どうしようもありません。ジｴントルマンの球戯場の芝生だけは雑草の管理人がいるのですが .. 。それにしても芝生は、イギリス人の生活に絶対に必要とされる土地です。クリケﾂトは当然のことフﾂトボ - ルや陸上競技も、人々は芝生の上でしています。子供の遊び場も芝生で、どこの家にも ( 花はなくても ) 芝生はあるようです。ところでひな菊は踏んでも害がないので、デイジ - の野原でもスポ - ツや遊び場には適当だと思われるでしょうか。確かにそれは適当なのですが、芝生が生えているからこそ適当になるのです。ひな菊だけというのではありません。芝生がなければイギリスの地面は、踏みどころのない雑草の地面になってしまうでしょうから。そしてイギリスの雑草の中にはアザミのようにトゲある植物もあり、こうした植物は芝生に生えるとすぐに抜き取られてしまいます。ところがひな菊だけは芝生と ( ある意味で ) 平和に共存できるので、そのためにひな菊は掘り起こされることもなく、芝生の中に放任されたままでいるのです。デイジ - に親切な観察の眼を向けた詩人ワ - ズワ - スは、次のような詩を残しています。－
