これまでのあらすじ

妹尾( せのお ) 千鶴( ちづる ) は、桐旺会( とうおうかい ) 総合病院に務める新人女医。

ある宿直の日、突然訪れた老人の正体は、広域指定暴力団・関東帝釈( たいしゃく ) 会( かい ) の二次団体「広目組( こうもくぐみ ) 」の組長・東極( とうごく ) 狸( まみ ) 右衛門( えもん ) だった。

桐旺会総合病院はかねてより広目組との繋がりがあり、狸右衛門に気に入られた千鶴は、なかば強制的に広目組の担当医にさせられてしまう。

千鶴を迎えに来たのは広目組の若頭・護崎( もりさき ) 龍牙( りゅうが ) 。

強面( こわもて ) で寡黙な護崎に、はじめは警戒心の解けない千鶴だったが、接していくうちに次第にその実直な性格に惹かれていく。


護崎への想いが増す一方で、医者とヤクザという立場の違いに戸惑う千鶴。

護崎に連れて来られた会員制バーのオーナー・お蝶( ちょう ) と護崎の親密そうな関係にショックを受け、逃げるように夜の歓楽街へ走り出す。

そこで二人を待ち受けていたのは、関東帝釈会の宿敵、関西阿修羅( あしゅら ) 会( かい ) の半井( なからい ) という男だった――

「い、忙しくって......ちゃんと来月くらいには一度帰ろうと思ってたわ」


母：

『あ、今日はその話じゃないの。あなたの家、区内よね？』


千鶴：

「ええ」


母：

『暴力団よ、暴力団！』


――ドキッ。


千鶴：

「......え？」


私は思わず龍牙さんを見た。


母：

『今日テレビ見てたらね、区内で暴力団の抗争？　があるんだって。あなたぼーっとしてるところあるから、気がついたら巻き込まれてた！......なんてことないようにしてね。都会は怖いんだから！』


千鶴：

「き......気をつけるわ......」


その後二、三世間話をして通話を切って、私は知らずにため息をついた。


千鶴：

（心配をかけない為だけれど、やっぱりお母さんに隠し事は心が痛いな......）


龍牙：

「親御さんか？」


千鶴：

「......ええ、そう。まだ、私達のこと......話してないから」


龍牙：

「――そうか」


それ以上責めることはせず、その代わりなだめられるように頬を撫でられる。

それはとても心地いい。


龍牙：

「............ああ、そうだ千鶴」


千鶴：

「え？」


龍牙：

「明日の夜、空いているか？」


千鶴：

「明日は......ええ、確か大丈夫です」


龍牙：

「ちょっと一緒に行って欲しいところがある。ちと、断れねえところでな」


千鶴：

「？　私でよければ......」


龍牙：

「助かるよ――じゃあ、明日迎えに来る」


龍牙さんはそう言って、帰って行った――



――「シリーズ３」に続く
