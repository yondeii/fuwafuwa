 【現代語化文庫について】  

 　明治時代から昭和前期にかけ て 綺羅星きらぼし のごと く 陸続りくぞく と 上梓じょうし され た 不朽ふきゅう の名著群は、我々日本国民が共有する貴重な文化遺産として、現代 に 甦よみがえ らせて我々自身がしっかりと活用するとともに、後世の子々孫々にもきちんと継承していかなければならないものです。そのためには、まず、これらの名著 の 古風こふう な表記を、現代の日本人が読みやすく親しみやすい表記にあらためる必要があります。現代語化文庫では、この使命を果たすべく、原著におけ る 旧字きゅうじ 旧仮名きゅうかな を 新字しんじ 新仮名しんかな に直し、送 り 仮名がな をあらため、漢字 を 仮名書かなが きにし 、 句読法くとうほう をあらため、ルビを増やす等の方法により、一冊一 冊 叮嚀ていねい に現代語化を行い、新た な 息吹いぶき を吹き込んでいます。

 　本書は 、 文部省もんぶしょう 編 『 國體こくたい の 本義ほんぎ 』（昭和十二年刊）を現代語化したものです。現代語化にあたっては、原著におけ る 旧字きゅうじ 旧仮名きゅうかな を 新字しんじ 新仮名しんかな に直し、送 り 仮名がな をあらため、漢字 を 仮名書かなが きにし 、 句読法くとうほう をあらため、ルビを増やす等の改変を行いました。但し、「而して」については、「しこうして」・「しかして」の二通りの読み方があること に 鑑かんが み、仮名書きとすることは避けました。

 　我が国は、今 や 国運こくうん すこぶ る 盛さか んに、海外発展のいきお い 著いちじる しく 、 前途ぜんと いよい よ 多望たぼう な時 に 際会さいかい している。産業 は 隆盛りゅうせい に、国防 は 威力いりょく を 加くわ え、生活 は 豊富ほうふ となり、文化の発展は諸方面 に 著いちじる しいものがある。つとにシナ・インドに由来する東洋文化は、我が国に輸入せられて 、 惟神かんながら の国体 に 醇化じゅんか せられ、さら に 明治めいじ ・ 大正たいしょう 以来、欧米近代文化の輸入によって諸種 の 文物ぶんぶつ は 顕著けんちょ な発達 を 遂と げた。文物・制度の整備せる、学術の一大進歩をなせる、思想・文化 の 多彩たさい を 極きわ むる 、 万葉まんよう 歌人をして今日にあらしめば 、 再ふたた び 「 御民みたみ 吾われ 生け る 験しるし あ り 天地あめつち の 栄さか ゆる時にあえら く 念おも えば」 と 謳うた うであろう 。 明治維新めいじいしん の 鴻業こうぎょう により 、 旧来きゅうらい の 陋習ろうしゅう を破り 、 封建的束縛ほうけんてきそくばく を去って、国民はよくそ の 志こころざし を遂げ、そ の 分ぶん を 竭つ くし 、 爾来じらい 七十年、もって今日 の 盛事せいじ を見る に 至いた った。  

 　しかしながらこの盛事は、静かにこれ を 省かえり みるに 、 実じつ に 安穏平静あんのんへいせい のそれにあらずして 、 内うち に 外そと に 波瀾万丈はらんばんじょう 、発展の前途 に 幾多いくた の困難 を 蔵ぞう し、隆盛の内面に混乱をつつんでいる。すなわち国体の本義は、ややもすれ ば 透徹とうてつ せず、学問・教育・政治・経済その他国民生活の各方面に幾多 の 欠陥けっかん を有し、伸びんとする力と混乱の因と は 錯綜さくそう 表裏ひょうり し 、 燦然さんぜん たる文化は内 に 薫蕕くんゆう を 併あわ せつつみ、ここに種々の困難な問題を生じている。今や我が国は、一 大 躍進やくしん をなさんとするに際して、生彩と陰 影 相あい 共 に 現あらわ れた感がある。しかしながら、これあくまで発展 の 機き であり、進歩の時である 。 我等われら は、よ く 現下げんか 内外の真相 を 把握はあく し 、 拠よ って進むべ き 道みち を明らかにするとともに 、 奮起ふんき して難局の打開 に 任にん じ、いよいよ国運の伸展 に 貢献こうけん するところがなければならぬ。  

 　 現今げんこん 我が国の思想上・社会上 の 諸弊しょへい は、明治以降あまりに も 急激きゅうげき に多種多様な欧米の文物・制度・学術 を 輸入ゆにゅう したために、ややもすれば 、 本もと を忘れ て 末すえ に 趨はし り 、 厳正げんせい な批判 を 欠か き、徹底し た 醇化じゅんか をなし得なかった結果である。そもそも我が国 に 輸入ゆにゅう せられた西洋思想は 、 主しゅ として十八世紀以来 の 啓蒙思想けいもうしそう であり、あるいはその延長としての思想である。これらの思想 の 根柢こんてい をなす世界観・人生観は、歴史的考察 を 欠か い た 合理ごうり 主義であり 、 実証じっしょう 主義であり、一面において個人 に 至高しこう の価値 を 認みと め、個人 の 自由じゆう と 平等びょうどう とを主張するとともに、他面において国家 や 民族みんぞく を 超越ちょうえつ し た 抽象的ちゅうしょうてき な世界性 を 尊重そんちょう するものである。したがって、そこには歴史的全体よ り 孤立こりつ して、抽象化せられた個 々 独立どくりつ の人間とそ の 集合しゅうごう とが重視せられる。かかる世界観・人生観 を 基もと とする政治学説・社会学説・道徳学説・教育学説等が、一方において我が国の諸種 の 改革かいかく に貢献するとともに、他方において深く広くその影響を我が 国 本来ほんらい の思想・文化に与えた。

 　そもそも社会主義・無政府主義・共産主義等 の 詭激きげき なる思想は、究極においてはすべて西洋近代思想 の 根柢こんてい をなす個人主義に基づくものであって、その発現 の 種々相しゅじゅそう たるに過ぎない。個人主義 を 本もと とする欧米においても、共産主義に対しては、さすがにこれ を 容い れ 得え ずして、今やその本来の個人主義 を 棄す てんとして、全体主義・国民主義 の 勃興ぼっこう を見、ファッショ・ナチス の 擡頭たいとう ともなった。すなわち個人主義の行き詰まりは、欧米においても我が国においても 、 等ひと しく思想上・社会上の混乱と転換との時期 を 将来しょうらい しているということができる 。 久ひさ しく個人主義の下にその社会・国家を発達せしめた欧米が、今日の行き詰まりをいか に 打開だかい するかの問題はしばら く 措お き、我が国に関する限り 、 真しん に我が国独自 の 立場たちば に 還かえ り 、 万古不易ばんこふえき の 国体こくたい を 闡明せんめい し 、 一切いっさい の 追随ついずい を 排はい して、よく本来 の 姿すがた を 現前げんぜん せしめ、しかも固陋 を 棄す ててますます欧米文化 の 摂取せっしゅ 醇化じゅんか に 努つと め 、 本もと を立て て 末すえ を生かし 、 聡明そうめい にし て 宏量こうりょう なる新日本を建設すべきである。すなわち今日我が国民の思想 の 相剋そうこく 、生活 の 動揺どうよう 、文化の混乱は 、 我等われら 国民がよく西洋思想の本質 を 徹見てっけん するとともに 、 真しん に我 が 国体こくたい の 本義ほんぎ を 体得たいとく することによってのみ解決せられる。而してこのことは、ひとり我が国のためのみならず、今や個人主義の行き詰まりにおいてその打開 に 苦くる しむ世 界 人類じんるい のためでなければならぬ。ここ に 我等われら の重大なる世界史的使命がある。すなわち、『国体の本義』 を 編纂へんさん して 、 肇国ちょうこく の 由来ゆらい を 詳つまび らかにし、その大精神 を 闡明せんめい するとともに、国体 の 国史こくし に 顕現けんげん す る 姿すがた を明示し、進んでこれを今の世 に 説と き 及およ ぼし、もって国民の自覚と努力と を 促うなが す 所以ゆえん である。

 　人君じんくん 民を養やしな い、もって祖業そぎょう を続つ ぐ、臣民しんみん 君に忠に、もって父の志こころざし を継つ ぐ、君臣一体、忠孝一致は、唯た だ吾わ が国のみ然しか りとなす。  

 といっているのは、忠孝一本の道 を 極きわ め て 適切てきせつ に 述の べたものである。  

 　シナのごとき も 孝道こうどう を重んじて、「孝 は 百行ひゃっこう の 本もと 」といい、またインドにおいても父母の思い を 説と いているが、その孝道は、国に連なり、国 を 基もとい とするものではない 。 孝こう は東洋道徳の特色であるが、それがさら に 忠ちゅう と一つとなるところに、我が国の道徳の特色があり、世界にそ の 類例るいれい を見ないものとなっている。したがって、この根本の要点を失ったものは、我が国の孝道ではあ り 得え ない。武士 の 名乗なの りがその家の皇室 に 出い づることを名乗り、ま た 家憲かけん ・ 家訓かくん が皇室に対したてまつる関係をその遠 い 源みなもと としたごときは 、 全まった く同じ道理 に 出い づるものと見るべきである。  

 　 佐久良東雄さくらあずまお の  

 　すめろぎに　つかえまつれと　我われ を生みし　我が垂乳根たらちね は　尊とうと くありけり  

 という歌は、孝が忠に高められて、はじめてまことの孝となることを示すものである 。 乃木のぎ 大将夫妻がその子二人までも御国のため に 献ささ げて、しか も 家門かもん の名誉としたのも 、 家国一体かこくいったい ・ 忠孝一本ちゅうこういっぽん の心の現れである。かく忠孝一本の道によっ て 臣民しんみん が尽くす心は、天皇の 御 仁慈じんじ の 大御心おおみこころ と一となって君 民 相和そうわ の実が挙げられ、我が国の無限の発展の根本の力となる。  

 　まことに忠孝一本は、我が国体 の 精華せいか であって、国民道徳 の 要諦ようてい である。而して国体 は 独ひと り道徳のみならず、広く政治・経済・産業等のあらゆる部門 の 根柢こんてい をなしている。したがって、忠孝一本 の 大道だいどう は、これらの国家生活・国民生活のあらゆる実際的方面におい て 顕現けんげん しなければならぬ。我等国民はこ の 宏大こうだい にし て 無窮むきゅう なる国体の体現のために、いよい よ 忠ちゅう にいよい よ 孝こう に 努つと め 励はげ まなければならぬ。  

 


 

四、和わ と「まこと」

    

和

 　我 が 肇国ちょうこく の事実および歴史の発展 の 跡あと を 辿たど る時、常にそこ に 見出みいだ されるもの は 和わ の精神である。和は、我 が 肇国ちょうこく の 鴻業こうぎょう より出で、歴史生成の力であるとともに、日常離るべからざ る 人倫じんりん の道である。和の精神は、万物融合の上に成り立つ。人々があくまで自己を主とし 、 私わたくし を主張する場合には 、 矛盾むじゅん 対立のみあって和は生じない。個人主義においては、この矛盾対立を調 整 緩和かんわ するため の 協同きょうどう ・ 妥協だきょう ・ 犠牲ぎせい 等はあ り 得え ても、結 局 真しん の和 は 存そん しない。すなわち個人主義の社会 は 万人ばんにん の万人に対す る 闘争とうそう であり、歴史はすべ て 階級闘争かいきゅうとうそう の歴史ともなろう。かかる社会における社会形態・政治組織およびその理論的表現たる社会学説・政治学説・国家学説等は、和をもって根本の道とする我が国のそれとは本質的 に 相違そうい する。我が国の思想・学問が西洋諸国のそれと根本的に異な る 所以ゆえん は 、 実じつ にここ に 存そん する。  

 　我が国の和は、理性から出発し 、 互たが いに独立し た 平等びょうどう な個人の機械的 な 協調きょうちょう ではなく、全体の中 に 分ぶん をもって存在し、この分 に 応おう ず る 行おこな いを通じてよく一体 を 保たも つところ の 大和だいわ である。したがって、そこに は 相互そうご のものの間 に 敬愛随順けいあいずいじゅん ・ 愛撫掬育あいぶきくいく が 行ぎょう ぜられる。これ は 単たん なる機械的・同質的なもの の 妥協だきょう ・調和ではなく、おのおのそ の 特性とくせい をもち 、 互たが い に 相違そうい しながら、しかもその特性すなわち分を通じてよく本質 を 現げん じ、もっ て 一如いちにょ の世界 に 和わ するのである。すなわち我が国の和は、各自その特質を発揮し 、 葛藤かっとう と 切磋琢磨せっさたくま とを通じてよく一 に 帰き するところの大和である。特性あり、葛藤あるによって、この和はますま す 偉大いだい となり、その内容 は 豊富ほうふ となる。またこれによって個性はいよい よ 伸長しんちょう せられ、特質は美しき を 致いた し、しかも同時に全体の発 展 隆昌りゅうしょう をもたらすのである。実に我が国 の 和わ は 、 無為姑息むいこそく の和ではなく 、 潑剌はつらつ としてものの発展 に 即そく して現れる具体的 な 大和だいわ である。  

    

武ぶ の精神

 　而してこの和は、我が国 の 武ぶ の精神の上にも明らかに現れている。我が国 は 尚武しょうぶ の国であって、神社に は 荒魂あらみたま を 祀まつ る神殿のあるのもある 。 修理固成しゅりこせい の大命に は 天あま の 沼矛ぬぼこ がまず授けられ、皇孫降臨の場合にも、武神によって平和にそれが成就し 、 神武じんむ 天皇の御東征の場合にも武が用いられた。しかし、この武は決して武そのもののためではなく 、 和わ のための武であって、いわゆ る 神武しんぶ である。我が武の精神は、殺人を目的とせずし て 活人かつじん を眼目としている。その武は、万物 を 生い かさんとする武であって、破壊の武ではない。すなわち 、 根柢こんてい に和をもち生成発展を約束し た 葛藤かっとう であって、その葛藤を通じてものを生かすのである。ここに我が国の武の精神がある。戦争は、この意味において、決して他を破壊し、圧倒し、征服するためのものではなく、道 に 則のっと って創造の働きをなし 、 大和だいわ すなわ ち 平和へいわ を 現げん ぜんがためのものでなければならぬ。  

    

むすびと和

 　かくの如き和によって我が国の創造発展は実現せられる。「むすび」とは創造であるが、それはすなわち和の力の現れである 。 伊弉諾いざなぎ ノ 尊みこと ・ 伊弉冉いざなみ ノ 尊みこと 相あい 和して神々・国土を生みたもうた。これすなわち大いなるむすびである。むすびは「むす」から来たる 。 苔こけ むすというように、「むす」はものの生ずることである 。 露つゆ がむすぶというのは、露の生ずることをいう。ものが相和してそこにむすびがある。かくて君臣相和し 、 臣民しんみん 互い に 親和しんわ して国家の創造発展がなされる。現下の問題たる国家諸般 の 刷新さっしん 改善も、またこの和によるむすびでなければならぬ。それは、一に天皇 の 御稜威みいつ の下に国体に照らし て 誤あやま れる を 正ただ し 、 大和だいわ によって大いに新たなる成果を生み出すことでなければならぬ。  

    

神と人との和

 　さらに我が国においては、神と人との和が見られる。これを西洋諸国の神人関係と比較する時は、そこに大な る 差異さい を見出す。西洋の神話に現れた、神による追放、神による処罰 、 厳酷げんこく な る 制裁せいさい の如きは、我が国 の 語かた り 事ごと とは大いに相違するのであって、ここに我が国の神と人との関係と、西洋諸国のそれとの間に大なる差異のあることを知る。このことは我が国の祭祀 ・ 祝詞のりと 等の中にも明らかに見えているところであって、我が国においては、神 は 恐おそろ しきものではなく、常 に 冥助みょうじょ を 垂た れたまい 、 敬愛けいあい 感謝かんしゃ せられる神であって、神と人との間は極め て 親密しんみつ である。  

    

人と自然との和

 　またこの和は、人と自然との間の最 も 親した しい関係にも見られる。我が国は海に囲まれ、 山 秀ひい で 水 清きよ く、春夏秋冬の季節の変化もあって、他国には見られない美しい自然をなしている。この美しい自然は、神々ととも に 天あま ツ神の生みたもうたところのものであって、親しむべきものでこそあれ 、 恐おそ るべきものではない。そこに自然を愛する国民性が生まれ、人と自然との和が成り立つ。インドのごときは自然 に 威圧いあつ せられており、西洋においては人が自然 を 征服せいふく してい る 観かん があって、我が国のごとき人と自然との深い和は見られない。これに対して、我が国民は常に自然と相和している。文芸にもこの自然との和の心 を 謳うた った歌が多く、自然への深い愛は我 が 詩歌しいか の最も主な る 題材だいざい である。それ は 独ひと り文芸の世界に限らず、日常生活においても、よく自然と人生とが調和している。 『 公事根源くじこんげん 』等に見える季節々々によ る 年中行事ねんじゅうぎょうじ を見ても、古くから人生と自然との微妙な調和が現れている。年の始めの行事はいうに及ばず、三月 の 雛ひな の 節供せっく は自然の春にふさわしい行事であり 、 重陽ちょうよう の菊の節供も秋 を 迎むか えるにふさわしいものである。季節の推移 の 著いちじる しい我が国においては、この自然と人生との和 は 殊こと に美しく生きている。そ の 外ほか 、 家紋かもん には多く自然の動植物が用いられており、服装その他建築・庭園等もよく自然の美を生かしている。かかる自然と人との親しい一体の関係も、また人と自然と が 同胞はらから とし て 相あい 親しむ我が 国 本来ほんらい の思想から生まれたのである。  

    

国民相互の和

 　この和の精神は、広く国民生活の上にも実現せられる。我が国においては、特有の家族制度の下に親子・夫婦 が 相あい 倚よ り 相あい 扶たす けて生活を共にしている。「教育に関する勅語」には「夫 婦 相あい 和わ し」と仰せられてある。而してこの夫婦の和は、やがて 「 父母ふぼ に 孝こう に」と一体に融け合わねばならぬ。すなわち家は、親子関係によ る 縦たて の和と、夫 婦 兄弟けいてい によ る 横よこ の和 と 相あい 合したる 、 渾然こんぜん た る 一如一体いちにょいったい の和の栄えるところである。  

 　さらに進んで、この和は、いかなる集団生活の間にも実現せられねばならない。役所に勤めるもの、会社に働くもの、皆ともどもに和の道に従わねばならぬ。それぞれの集団には、上に立つものがおり、下に働くものがある。それらおのおのが分を守ることによって集団の和 は 得え られる。分を守ることは、それぞれの有する位置において、定まった職分を最も忠実につとめることであって、それによって上は下 に 扶たす けられ、下は上に愛せられ、また同業互い に 相あい 和して、そこに美しき和が現れ、創造が行われる。  

 　このことは、ま た 郷党きょうとう においても国家においても同様である。国の和が実現せられるためには、国民おのおのがそ の 分ぶん を 竭つ くし、分を発揚するより外はない。身分の高いもの、低いもの 、 富と んだもの、貧しいもの 、 朝野ちょうや ・公私その他農工商等、相互に自己 に 執著しゅうちゃく して対立をこととせず、一 に 和わ をもっ て 本もと とすべきである。  

 　要するに我が国においては、それぞれの立場による意見の対立、利害の相違も 、 大本おおもと を同じうするところより出づる特有 の 大和だいわ によってよく一となる。すべて葛藤が終局ではなく、和が終局であり、破壊をもって終わらず 、 成就じょうじゅ によって結ばれる。ここに我が国の大精神がある。而して我が国に現れるすべての進歩発展は、皆かくして成される 。 聖徳太子しょうとくたいし が憲法十七条に、  

 　和をもって貴しとなし、忤さか うることなきを宗むね と為な す。人皆党たむら 有り、また達さと れる者少し。ここをもってあるいは君父に順したが わずして、乍また 隣里さととなり に違う。しかれども上和やわら ぎ下睦むつ びて、事を論あげつら わんに諧かな いぬるときには、すなわち事理ことわり 自ずからに通ず。何事か成らざらん。  

 と示したもうたのも、我が国のこの和の大精神を説かせられたものである。  

    

君臣一体

 　我が国においては、君臣一体と古くよりいわれ、天皇を中心として億兆一心 ・ 協心戮力きょうしんりくりょく 、 世々よよ 厥そ の美 を 済な し 来き たった。天皇の聖徳と国民の臣節とは互いに融合して、美しい和をなしている 。 仁徳にんとく 天皇は、  

 　百姓貧しきは、すなわち朕が貧しきなり。百姓富めるは、すなわち朕が富めるなり。  

 と仰せられ、また 、 亀山かめやま 上皇は 、 蒙古もうこ 襲来しゅうらい の際 、 宸筆しんぴつ の 御 願文がんもん を 伊勢神宮いせじんぐう に 献ささ げて、  

 　朕が身をもって国難にかえん。  

 と御祈りあそばされ、また、今上天皇陛下御即位式の勅語に、  

 　皇祖皇宗こうそこうそう 、国を建て、民に臨のぞ むや、国をもって家となし、民を視み ること子の如ごと し。列聖相あい 承けて、仁恕じんじょ の化、下に洽あまね く、兆民相あい 率いて、敬忠の俗、上に奉じ、上下感孚かんぷ し、君民くんみん 体を一にす。これ我が国体の精華せいか にして、当まさ に天地と並び存すべき所なり。  

 と仰せられてある。ここに君民体を一にして、その苦楽をともにしたまう尊い和の純 粋 顕現けんげん を仰ぐことができる。また「君のため世のため何か惜しからん捨ててかいある命なりせば」という歌の心は 、 臣民しんみん が天皇に一身を捧げたてまつる和 の 極致きょくち を示したものである。  

 　かかる我が国の和の精神が世界に拡充せられ、それぞれの民族・国家がおのおのその分を守り、その特性を発揮する時、真の世界の平和とその進歩発展とが実現せられるであろう。  

    

まこと

 　「まこと」の心は、人の精神の最も純粋なものである。人はまことにおいて、その生命の本をもち、まことによって万物と一体となり、またよく万物を生かし、万物と和する。  

 　まことについては 、 賀茂真淵かものまぶち や 富士谷御杖ふじたにみつえ 等が特にこれを重んじて説いている 。 真言まこと すなわ ち 真事まこと である。言と事とはまことにおいて一致しているのであって、すなわち言われたことは必ず実現せられねばならぬ。この言となり、事とな る 根柢こんてい に、まことがある。御杖は心 を 偏心ひとえごころ ・ 一向心ひたぶるこころ ・ 真心まごころ というがごとくに分けている 。 偏心ひとえごころ とは主我的な心であり 、 一向心ひたぶるこころ と は 頑かたく なに行う心である。これらはいずれも完全な心とはいわれない。真心とは心の欲するところに従っ て 矩のり を 踰こ えざる心である。かかる心はすなわちわざであり、言であり、行であり、よく一事・一物に執せずし て 融通無礙ゆうづうむげ である。すなわ ち 私わたくし を離れた純粋の心、純粋 の 行ぎょう である。実にまことは万物を融合一体ならしめ 、 自由無礙じゆうむげ ならしめる。まことは芸術に現れて は 美び となり、道徳として は 善ぜん となり、知識において は 真しん となる。美と善と真とを生み出す根源にまことのあることを知るべきである。而してまことはまたいわゆ る 明あか き 浄きよ き 直なお き心、すなわ ち 清明心せいめいしん であり、それは我が国民精神 の 根柢こんてい となっている。

 　かくのごとくして諸藩 の 版籍奉還はんせきほうかん があり、さら に 廃藩置県はいはんちけん が行われて、大 政 全まった く朝廷に帰して王政の復古を仰ぎ、維新の大業 は 成就じょうじゅ した。国民 の 覚醒かくせい が常に天皇を中心として展開す る 姿すがた は、ここ に 遺憾いかん な く 顕現けんげん している。この偉業を翼賛したてまつっ た 先人せんじん の 功労こうろう 、志士 の 遺烈いれつ は深 く 欽仰きんぎょう すべきはもとより、慶喜がフランス公使より幕府を援助せんとする申し出に対して 、 断然だんぜん これを拒絶し、もって外 国 干渉かんしょう の 累るい を 断た った例のごときも、また見逃すことを得ない。

 　皇朕すめらわ れ謹つつし み畏かしこ み

 と 宣のたま わせられてある。すなわち我が典憲は 、 肇国ちょうこく の初めにあたって 、 日星じっせい のごと く 昭あき らかな る 大義たいぎ を時代の進運に適応し て 紹述しょうじゅつ あそばされ 、 丕基ひき を永遠 に 鞏固きょうこ にせられたものである。我 が 欽定憲法きんていけんぽう は、 「 朕ちん が 後嗣こうし およ び 臣民しんみん およ び 臣民しんみん の子孫たる者をして永遠 に 循行じゅんこう するところ を 知し らしむ」と仰せられ た 万古ばんこ 不磨ふま の大典であって 、 肇国ちょうこく の精神の一貫してここにいよい よ 鞏かた きを見る。さらに明治二十三年十月三十日には「教育に関する勅語」を下したまい、我が国の教育が一に国体 に 淵源えんげん すること を 昭示しょうじ あそばされた。  

 　以上我等は、我 が 国史こくし の展開が、天皇におかせられて は 皇祖皇宗こうそこうそう の遺訓の御紹述であり 、 臣民しんみん にあって は 私わたくし を去ってよ く 分ぶん を 全まっと うし 、 忠誠ちゅうせい もっ て 皇運こううん を 扶翼ふよく したてまつるにあることを見た。而してこ の 上下一如しょうかいちにょ の大精神は、すでに我 が 肇国ちょうこく において明らかに示されたものであって、この大精神 が 国史こくし を貫き 、 世々よよ 厥そ の美 を 済な して今日に至っている。ここに我等は 、 戊申詔書ぼしんしょうしょ に 「 炳へい とし て 日星じっせい のごとし」 と 仰おお せられ た 国史こくし の 輝かがや かし い 成跡せいせき を見るのである。  

 


 

二、国土と国民生活

    

国土

 　我が国土は 、 語かた り 事ごと によれ ば 伊弉諾いざなぎ ノ 尊みこと ・ 伊弉冉いざなみ ノ 尊みこと 二尊の生みたもうたものであって 、 我等われら と 同胞どうほう の関係にある。我等が国土・草木を愛するのは、かか る 同胞的親和どうほうてきしんわ の 念ねん からである。すなわち我が国民の国土愛は 、 神代かみよ よりの一体の関係 に 基もと づくものであって、国土は国民と生命 を 同おな じうし、我が国 の 道みち に 育はぐく まれてますま す 豊ゆた か に 万物ばんぶつ を 養やしな い、とも に 大君おおきみ に 仕つか え 奉まつ るのである。  

 　かくて国土は、国民の生命 を 育そだ て、国民の生活を維持進展せしめ、その精神 を 養やしな う上 に 欠か くべからざるものであって、国土・風土と国民と の 親した しく深き関係は、よく我 が 国柄くにがら を現しており 、 到いた るとこ ろ 国史こくし にそ の 跡あと を見ることができる。  

 　遠き祖先よりの語り伝えが、我が国性を示し、天皇御統治 の 大本おおもと を明らかにするものとして 、 撰録せんろく せられて 『 古事記こじき 』となり 、 編纂へんさん せられて 『 日本書紀にほんしょき 』となったが、これ に 伴ともな って 『 風土記ふどき 』 の 撰進せんしん を 命めい ぜられたのは、我が国体と国土との深い関係 を 物語ものがた るものである。ここに、 「 古事こじ 」と 「 風土ふうど 」と の 分わ かつことのできない深い関係を見る。我が国 の 語かた り 事ごと においては、国土と国民とが同胞であること が 物語ものがた られている。我が国民の国土 に 親した しみ、国土と一になる心は非常に強いのであって、農業に従う人々が、季節の変化に応和し 、 随順ずいじゅん する姿はよくこれを示している。それ は 祭祀さいし を中心とす る 年中行事ねんじゅうぎょうじ をはじめ 、 衣食住いしょくじゅう の生活様式の上にまで行きわたっている。

　山川も　よりてつかうる　神かん ながら　たぎつ河内こうち に　船出ふなで せすかも  

 とある。この歌 を 誦よ む者は、我が国民の国土・自然を見る心を知ることができるであろう。すなわち国民も国土も一になって天皇に仕えまつるのである。国民はかかる心をもって国土・自然と親しみ、その中に生活し、またそれによって産業 を 営いとな むのである。これもとよ り 神代かみよ におい て 天あま ツ 神かみ が我等と国土とを同胞として生みたもうたところか ら 出い づるのである。  

    

国民生活

 　こ の 本もと を一つにす る 親和しんわ ・ 合体がったい の心は、我が国民生活を常に一貫して流れている。この精神のあるところ、国民生活はいかなる場合にも対立的でなく、一体的なものとして現れて来る。  

 　我が国においては、政治上・社会上の制度 の 変遷へんせん にもかかわらず、いつの時代にも常にこの心が現れている。古く は 氏族しぞく が国民生活の基本をなし、経済生活の単位であって、それは天皇の下に同 一 血族けつぞく ・同一精神の団体をなしたのである。すなわち各人 は 氏うじ に 統合とうごう せられ、多く の 氏人うじびと の上 に 氏上うじのかみ があり、これ に 部曲かきべ の民 が 附随ふずい し、氏・部として の 分業ぶんぎょう 分掌ぶんしょう があり、職業によって、あらゆる人と物と が 相あい 倚よ り 相あい 扶たす けて、天皇を中心として国家をなした。而してそれぞれの氏族内においては 、 氏上うじのかみ が 氏神うじがみ を 祀まつ り 、 氏人うじびと もまた氏上と一体となって同一の祖先を祭るのである。而してこの祭祀を通じて、氏上と氏人とはただ一つとなって祖先 に 帰一きいつ する。そこに氏の政事もあり、教化もあり、またその職業もある。かくてこの一体たるものを氏上が率いて朝廷に奉仕した。  

 　かような親しい結合関係は 、 国史こくし を通じて常に存続していた。これ は 自我じが を主張する主我的な近代西洋社会のそれと全く異なるものであり、国初よ り 連綿れんめん として続く一体的精神と事実とに基づくものであって、我が国民生活はそ の 顕現けんげん である。そこには、一家・一郷・一国を通じて必ず融和一体の心 が 貫つらぬ いている。すなわち天皇の下に人と人、人と物とが一体となるところに我が国民生活の特質がある。これ 、 義ぎ は君臣にし て 情じょう は父子とい う 一国即一家いっこくそくいっか の道の存す る 所以ゆえん であり 、 君民一体くんみんいったい となり、親 子 相あい 和して、美し き 情緒じょうちょ が家庭生活・国民生活に流れてい る 所以ゆえん である。  

    

職業

 　氏族における職業の分掌は、やが て 家業かぎょう 尊重の精神を生んでおり、家業の尊重 は 家名かめい すなわち名を重んずることとなる。我が国の古代における名は、個人 の 氏名しめい の意味ではなく、氏の職業が名である。ここに我が国民の職業を重んじ、家名を尊重する精神を見ることができる。而してつとめの尊重は 、 宣命せんみょう をはじめとして多くの史実に見られる 。 天武てんむ 天皇の御制定になった冠位の名称にも勤務追進の文字が用いられている。この勤務尊重の精神は、生産・創造・発展のむすびの心であって、我が国産業の根本精神である。この精神は古 来 農事のうじ において最もよ く 培つちか われた。  

 　 豊葦原とよあしはら の 瑞穂みずほ の国という我が国名は、国初における国民生活の基本たる農事が尊重せられたことを示すものであり、年中恒例の祭祀が食事に関するものの多いのもこの精神の現れである 。 天照大神あまてらすおおみかみ を奉祀す る 内宮ないくう に並ん で 外宮げくう に 豊受大神とゆけのおおかみ を奉祀し、上、皇室をはじめたてまつり、国民が深厚な る 崇敬すうけい を 捧ささ げ来たっていることにも深く思いを致すべきであろう。  

 　国民の職業 が 農業のうぎょう の 外ほか に 、 商業しょうぎょう ・ 工業こうぎょう 等種々なる方面に分岐発展している今日においては、農業を尊重したまうと同じ御心は、これらのあらゆる産業についてもうかがうことができる 。 昭憲しょうけん 皇太后の御歌に、  

 　ひのもとの　くにとまさんと　あき人んど の　きそう心ぞ　たからなりける  

 と詠ませられて、商業の重んずべきことを示したもうている。我等はよくこの御精神を拝して、時勢の進運に伴い、おのおのその職業にいそしまねばならぬ。  

 


 

三、国民性

    

風土と国民性

 　 山鹿素行やまがそこう は、 『 中朝事実ちゅうちょうじじつ 』に「中国 の 水土すいど は 万邦ばんぽう に 卓爾たくじ し、人物 は 八紘はっこう に 清秀せいしゅう なり」と述べているが、まことに我が国 の 風土ふうど は 、 温和おんわ なる気候 、 秀麗しゅうれい な る 山川さんせん に 恵めぐ まれ、春花秋葉 、 四季折々しきおりおり の景色は変化 に 富と み 、 大八洲国おおやしまのくに は当初より日本人にとっ て 快こころよ い生活地帯であり、 「 浦安うらやす の国」と呼ばれていた。しかしながら時々起こる自然 の 災禍さいか は、国民生活 を 脅おびやか すがごと き 猛威もうい をふるうこともあるが、それによって国民が自然を恐れ、自然の前 に 威圧いあつ せられるがごときことはない。災禍は却っ て 不撓不屈ふとうふくつ の心 を 鍛錬たんれん する機会となり、更生の力 を 喚起かんき し、一層国土との親しみを増し、それと一体の念をいよいよ強くする。西洋神話に見られるごとき自然との闘争は、我が国 の 語かた り 事ごと には見られず、この国土は、日本人にとってはまことに生活 の 楽土らくど である。「やまと」が漢字で大和と書かれたことも、けだ し 偶然ぐうぜん ではない。  

 　 頼山陽らいさんよう の作とし て 人口じんこう に 膾炙かいしゃ せ る 今様いまよう に、  

 　花より明あ くる　み吉野よしの の　春の曙あけぼの 　見わたせば
　もろこし人びと も　高麗人こまびと も　大和心やまとごころ に　なりぬべし  

 とあるのは、我 が 美うつく しき風土 が 大和心やまとごころ を 育はぐく み 養やしな っていることを示したものである。ま た 本居宣長もとおりのりなが がこの 「 敷島しきしま の 大和心やまとごころ 」を歌って、「朝日 に 匂にお う 山桜花やまざくらばな 」といっているのを見ても、いかに日本的情操が日本の風土と結びついているかが知られよう。さら に 藤田東湖ふじたとうこ の 「 正気せいき の歌」には、  

 　天地てんち 正大せいだい の気、粋然すいぜん として神州しんしゅう に鍾あつ まる
　秀ひい でては不二ふじ の嶽たけ となり、巍々ぎぎ として千秋せんしゅう に聳そび え
　注そそ いでは大瀛だいえい の水となり、洋々ようよう として八州はっしゅう を環めぐ る
　発しては万朶ばんだ の桜となり、衆芳しゅうほう ともに儔たぐ いし難がた し  

 とあって、国土草木が我が精神とそ の 美び を競 う 有様ありさま が 詠よ まれている。  

    

清明心せいめいしん

 　かかる国土と、すでに述べたごとき君民和合の家族的国家生活とは 、 相俟あいま っ て 明浄正直めいじょうせいちょく の国民性を生んだ。すなわ ち 文武もんむ 天皇御即位 の 宣命せんみょう その他において、  

 　明あか き浄きよ き直なお き誠まこと の心
　清きよ き明あか き正ただ しき直なお き心  

 と繰り返されている。これはすでに、神道におけ る 禊みそぎ 祓はらえ の精神として語り事にもうかがわれるのであるが 、 天武てんむ 天皇の十四年に御制定になった冠位の名称には、勤務追進の上に明浄正直の文字が示され、いかにこの国民性が尊重せられたかがわかる。明浄正直は、精神の最も純な力強い正しい姿であって、いわゆ る 真心まごころ であり、まことである。このまことの外部的表現としての行為・態度が勤務追進である。すなわちこの冠位の名称は、明る い 爽さわ やかな国民性の表現であり、また国民の生活態度でもあった。而してまことを本質とする明浄正直の心は、単なる情操的方面に止まらず、明治天皇の御製に、

 と仰せられてあるごとく、よ く 義勇ぎゆう 奉公ほうこう の精神として発現する。万葉集には「海行かば 　 水潰みづ くかばね　山行かば　草むすかばね 　 大君おおきみ の 　 辺へ にこそ死なめ　かえりみはせじ」と歌われ 、 蒙古もうこ 襲来以後は 、 神国思想しんこくしそう が 顕著けんちょ なる発達 を 遂と げて 、 大和魂やまとだましい として自覚せられた。まことに大和魂は 「 国祚こくそ の永命を祈り 、 紫極しきょく の靖鎮 を 護まも り」来たったのであって、近く は 日清にっしん ・ 日露にちろ の 戦役せんえき において力強 く 覚醒かくせい せられ 、 且か つ 具現ぐげん せられた。

 　そもそも没我の精神は、単なる自己の否定ではなく 、 小しょう なる自己を否定することによって 、 大だい なる真の自己に生きることである 。 元来がんらい 個人は国家より孤立したものではなく、国家 の 分ぶん としておのお の 分担ぶんたん するところをもつ個人である 。 分ぶん なるがゆえに常に国家 に 帰一きいつ するをその本質とし、ここに没我の心を生ずる。而してこれと同時に 、 分ぶん なるがゆえにその特性を重んじ、特性を通じて国家に奉仕する。この特質が没我の精神 と 合がっ して他を同化する力を生ずる。没我・献身というも、外国におけるがごとき、国家と個人とを相対的に見て、国家に対して個人を否定することではない。また包容・同化は他の特質 を 奪うば い、その個性 を 失うしな わしむることではなく、よくそ の 短たん を 棄す て て 長ちょう を 生い かし、特性を特性として 、 採と ってもって我を豊富ならしめることである。ここに我が国 の 大おお いなる力と、我が思想・文化の深さと広さとを見出すことができる。

武士道ぶしどう

 　我が国民道徳の上に顕著なる特色を示すものとして 、 武士道ぶしどう を挙げることができる。武士の社会には 、 古いにしえ の氏族における我が国特有の全体的な組織および精神がよく継承せられていた。ゆえに、主とし て 儒教じゅきょう や 仏教ぶっきょう に学びながら 、 遂つい によくそれを超えるに至った。すなわ ち 主従しゅじゅう の間は恩義をもって結ばれながら、それが恩義を超えた没我の精神となり、死 を 視み ること帰するがごときに至った。そこでは死を軽んじたというよりは、深く死に徹して真の意味においてこれを重んじた。すなわち死によって真の生命 を 全まっと うせんとした 。 個こ に 執しゅう し 個こ を立て て 全ぜん を失うよりも 、 全ぜん を 全まっと う し 全ぜん を生かすために個を殺さんとするのである。生死は根本において一であり、生死を超え て 一如いちにょ のまことが存する。生もこれにより、死もまたこれによる。しかるに生死を対立せしめ、死 を 厭いと うて生を求むることは 、 私わたくし に 執著しゅうちゃく することであって武士 の 恥はじ とするところである。生死一如の中に、よく忠の道 を 全まっと うするのが我が武士道である。

 　我が国の文化は 、 肇国ちょうこく 以来の大精神の顕現である。これを豊富にし発展せしめるために外来文化 を 摂取せっしゅ 醇化じゅんか して来た。シナ の 明みん 時代に著された 『 五雑俎ござっそ 』に 、 経書けいしょ のうち 『 孟子もうし 』 を 携たずさ えて日本へ往く者があれば、その船は必 ず 覆溺ふくでき するという伝説を掲げているごときは 、 凡およ そ革命思想が我が国体と根本的 に 相容あいい れないこと を 物語ものがた るものであり、我が不動の精神とこれに基づく厳正な批判との存することを意味している 。 菅原道真すがわらのみちざね の語といわれる 「 和魂漢才わこんかんさい 」なる言葉が一般に行われたのも、かような意味においてである。

 　かくて我が国の文化は 、 一貫いっかん せる精神をもつとともに、歴史の各時代においておのお の 異こと なる特色 を 現あらわ している。而して創造は常に回顧と一となり 、 復古ふっこ は常 に 維新いしん の原動力となる。すなわち今 と 古いにしえ とは一となり、そこに新時代の創造 が 営いとな まれる。我が国の歴史 を 辿たど るものは、到るところにこの事実の明瞭に現れているのを見るであろう。したがって、我が国においては、復古なき創造は真の意味における創造ではない。それと同時に創造なき復古は真の復古ではない。た だ 肇国ちょうこく 以来一貫せる精神に基づく「むすび」こそ、我が国のまことの発展の姿でなければならぬ。  

    

学問

 　 元来がんらい 我が国の学問は、歴代の天皇の 御 奨励しょうれい によって発達し、今日あるを得たのである。すなわちつとに儒教・仏教ならびにこれ に 随伴ずいはん した大陸の文化を摂取し、これを保護奨励したもうたのである 。 遣隋使けんずいし ・ 遣唐使けんとうし にそえて多数 の 留学生るがくしょう 、学問僧を遣わされて広く外国文化の粋を採りたもうたことや、万葉集の撰集に次いで、古今和歌集以下いわゆる二十一代集等 の 勅撰ちょくせん 、あるいは勅版の印行等、学問を御奨励あそばされたこと は 枚挙まいきょ に 遑いとま がない。これは近く明治維新以来、西洋の学問・技術の摂 取 普及ふきゅう に関する明治天皇の 御 軫念しんねん にも拝することができる。かく学問を保護奨励したまうことは、一に皇 祖 肇国ちょうこく の御精神 を 恢弘かいこう し、国運 の 隆昌りゅうしょう 、民福の増進 に 大御心おおみこころ を注がせたまうがため に 外ほか ならぬ。

 　 国史こくし については聖徳太子はつとに『天皇記』・『国記』等を作りたまい、次い で 天武てんむ 天皇の聖旨に基づき 、 元明げんめい 天皇は『古事記』三巻を撰録せしめたまい 、 元正げんしょう 天皇は勅して『日本書紀』三十巻 を 編纂へんさん せしめたもうた。而して『日本書紀』が撰進せられた翌年から 、 宮中きゅうちゅう においてこれ が 講筵こうえん を 設もう けさせられ 、 臣民しんみん をして我が国のまことの姿を明らか に 覚さと らしめたまうところがあった。勅命による修史の事業は 、 醍醐だいご 天皇 の 御代みよ に至るま で 相継あいつ ぎ、いわゆ る 六国史りっこくし の成立を見るに至ったが、後 世 民間みんかん にも、 『 大日本史だいにほんし 』のごとき修史事業 が 企くわだ てられたのである。また、江戸時代 に 勃興ぼっこう し た 国学こくがく は、古典の研究に発した復古の学であり 、 国史こくし とともによく国体を明らかにし、国民精神の宣揚に大いに貢献するところがあった。  

 　我が国のあらゆる学問は、その究極を国体に見出すとともに、皇運 の 扶翼ふよく をもってその任務とする。江戸時代に西洋の医学 ・ 砲術ほうじゅつ その他が伝来した時、非常な困難を排してその研究に当ったのも、また、明治維新後、西洋の学術百般の採用に専念し、努力したのも、皆これ皇運を扶翼したてまつ る 臣民しんみん の道に立ってのことであった。しかしながら非常の勢をもって外来文化を輸入し、諸方面に向かって大いに発展しつつある今日の学問においては 、 知し ら ず 識し らず の 間あいだ にこの中心を見失うおそれなしとしない。明治天皇 は 五箇条ごかじょう の 御誓文ごせいもん の中に、  

 　智識ちしき を世界に求め、大おおい に皇基こうき を振起しんき すべし

教育

 　我が国の教育も、また一に国体に基づき、国体の顕現を中心とし 、 肇国ちょうこく 以来の道にそ の 淵源えんげん を 有ゆう すべきことは、学問の場合 と 全まった く同様である。我が教育は、古 く 氏上うじのかみ が 氏人うじびと を 率ひき い て 朝廷ちょうてい に 奉仕ほうし した時代においては、そ の 氏々うじうじ における祖先以来 の 奉仕ほうし の歴史 の 伝承でんしょう が教育の内容をなした。例えば高 橋 氏文うじぶみ において、高橋氏の 祖 磐鹿六鴈いわかむつかり ノ 命みこと が 景行けいこう 天皇に奉仕し て 忠勤ちゅうきん を 擢ぬき んでてより 、 代々だいだい 家職 を 襲つ ぎ 、 朝廷ちょうてい の 内膳職うちのかしわでのつかさ に奉仕す る 由来ゆらい を述べて、その子孫を教訓し、もっ て 奉公ほうこう の 念ねん を 厚あつ うせしめた如き、古来諸家 の 氏文うじぶみ は 皆みな この類である。後世武士の教育についても、この伝統による家庭教育を重んじ、家門の名を守るべきことを常 に 訓おし えたのである 。 吉野朝よしのちょう の忠 臣 菊池きくち 氏の家訓た る 菊池武茂きくちたけもち 起請文きしょうもん に、

 とあるはその例である。

 　そもそも「おしえ」は 「 愛お し」の語が示すよう に 慈いつく し み 育そだ てる意味であり、人間自然 の 慈愛じあい を基とし て 道みち に 従したが って人を育てることである。「みちびく」 は 子弟してい をし て 道みち に至らしめる意味である。我が国の教育は、明治天皇が「教育に関する勅語」 に 訓おし えたもうたごとく、一に我が国体 に 則のっと り 、 肇国ちょうこく の御精神を奉体して、皇運 を 扶翼ふよく するをその精神とする。したがって、個人主義教育学の唱え る 自我じが の実現、人格の完成というがごとき、単なる個人の発展完成のみを目的とするものとは、全くその本質を異にする。すなわち国家を離れた単なる個人 的 心意しんい ・性能の開発ではなく、我が国 の 道みち を体現するところの国民の育成である。個人の創造性 の 涵養かんよう 、個性の開発等 を 事こと とする教育は、ややもすれば個人 に 偏へん し個人 の 恣意しい に流れ 、 延ひ いては自由放任の教育 に 陥おちい り、我が国教育の本質 に 適かな わざるものとな り 易やす い。  

 　教育は知識と実行とを一にするものでなければならぬ。知識のみ の 偏重へんちょう に陥り、国民として の 実践じっせん に 欠か くる教育は、我が国教育 の 本旨ほんし に 悖もと る。すなわ ち 知行合一ちこうごういつ してよ く 肇国ちょうこく の道を行ずるところに、我が国教育の本旨の存することを知るべきである。諸々の知識 の 体系たいけい は実践によって初めて具体的のものとなり、そ の 処ところ を 得う るのであって、理論的知識 の 根柢こんてい には、常に国体 に 連つら なる深い信念とこれによる実践とがなければならぬ。而して国民的信念および実践は理論的知識によってますます正確にせられ、発展せしめられるのであるから、我が国教育においても、理論的・科学的知識はいよい よ 尊重そんちょう 奨励しょうれい せられねばならぬが、同時にそれを国民的信念および実践と離れしめずして、もって我が国文化 の 真しん の発達 に 資し するところがなければならぬ。すなわち一面諸科学 の 分化ぶんか 発展を図るとともに、他面そ の 綜合そうごう に留意し、実行に高め、もってかかる知識をしておのおのそ の 処ところ を 得え しめ、そ の 本領ほんりょう を 発揮はっき せしむべきである。  

 　 畏かしこ くも明治天皇は、明治十二年 、 教学大旨きょうがくたいし に、  

 　教学の要、仁義じんぎ 忠孝ちゅうこう を明らかにして知識ちしき 才芸さいげい を究きわ め、もって人道じんどう を尽つ くすは我祖訓そくん 国典こくてん の大旨たいし 、上下しょうか 一般の教おしえ とするところなり。しかるに輓近ばんきん 、もっぱら智識ちしき 才芸のみを尚とう とび、文明開化ぶんめいかいか の末に馳は せ、品行ひんこう を破り、風俗ふうぞく を傷そこな う者少すくな からず。しかる所以ゆえん の者は、維新いしん の始首として陋習ろうしゅう を破り、知識を世界に広むるの卓見たっけん をもって、一時西洋の所長しょちょう を取り、日新にっしん の効こう を奏そう すといえども、その流弊りゅうへい 仁義忠孝を後にし、徒いたずら に洋風これ競きそ うにおいては、将来の恐おそ るるところ、終つい に君臣くんしん 父子ふし の大義を知らざるに至らんも、測はか るべからず。これ我が邦くに 教学の本意ほんい にあらざるなり。  

 と 仰おお せられている。まことに今 の 時世じせい に照らして深く思いを致さなければならぬところである。

 　我が国 の 道みち は、古来の諸芸に も 顕著けんちょ に現れている 。 詩歌しいか ・ 管絃かんげん ・ 書画しょが ・ 聞香もんこう ・ 茶ちゃ の 湯ゆ ・ 生華いけばな ・ 建築けんちく ・ 彫刻ちょうこく ・ 工芸こうげい ・ 演劇えんげき 等、皆その究極において は 道みち に 入い り、また道よ り 出い でている。道の現れは、一面において伝統尊重の精神となり、他面において創造発展 の 行ぎょう となる。したがって、中世以来我が国 の 芸道げいどう は、ま ず 型かた に入っ て 修練しゅうれん し、至っ て 後のち に型を出るという修養方法を重んじた。それは個人 の 恣意しい を排し、まず伝統に生 き 型かた に 従したが うことによって 、 自おの ずから道 を 得え 、而し て 後のち これ を 個性こせい に従って実現すべきことを教えたものである。これ我が国芸道修業 の 特色とくしょく である。

六、政治・経済・軍事

    

祭政一致

 　我が国 は 万世一系ばんせいいっけい の天皇御統治の下、祭祀・政治はその根本を一にする。大化の改新におい て 唐制とうせい を採用するに際し 、 孝徳こうとく 天皇が悦もって民を使うの道を問いたまえるに対し 、 蘇我石川麻呂そがのいしかわのまろ は「まずもっ て 神祇じんぎ を 祭いわ い 鎮しず めて、しかして後 に 応まさ に政事 を 議はか るべし」 と 奏上そうじょう している。我が 国 古いにしえ の 成文法せいぶんほう は 近江令おうみりょう よ り 養老令ようろうりょう に至って完成せられたが、そ の 職員令しきいんりょう の初めにま ず 神祇官じんぎかん を置き、また特 に 神祇令じんぎりょう を設けてある。明治天皇は 「 神祇じんぎ を 崇あが め 祭祀さいし を 重おも んずる は 皇国こうこく の大典政教の基本なり」 と 詔みことのり せられている。すなわち、祭祀の精神 は 肇国ちょうこく 以来政事 の 本もと となったのであって 、 宮中きゅうちゅう におかせられては 、 畏かしこく く も 三殿さんでん の御祭祀をいと も 厳粛げんしゅく に執り行わせられる。これ皇 祖 肇国ちょうこく の御精神を体し 、 神かん ながら御世しろしめしたま う 大御心おおみこころ よ り 出い づるものと拝察したてまつるのである。実 に 敬神けいしん と 愛民あいみん とは歴代の天皇 の 有難ありがた き 大御心おおみこころ である。  

    

欽定憲法きんていけんぽう

 　明治天皇は 、 皇祖皇宗こうそこうそう の 御 遺訓いくん 、御歴 代 統治とうち の 洪範こうはん を 紹述しょうじゅつ したまい、明治二十二年二月十一日をもっ て 皇室典範こうしつてんぱん を御制定になり 、 大日本帝国憲法だいにっぽんていこくけんぽう を発布あそばされた。  

 　外国におけ る 成文憲法せいぶんけんぽう は、大体において既存の統治権者 を 放逐ほうちく し、また は 掣肘せいちゅう することから生まれた。前の場合はいわゆ る 民約憲法みんやくけんぽう と 称しょう せられるけれども、そ の 実じつ は平等な人民が自由の立場において交互に契約したものではなくして 、 権力争奪けんりょくそうだつ における勝利者によって決定せられたもの に 過す ぎない。後の場合はいわゆ る 君民協約憲法くんみんきょうやくけんぽう と 称しょう せられるものであって、これは伝統的の権力者たる君主が新興勢力 に 強要きょうよう せられて相互の勢力圏を協定したもの に 外ほか ならぬ。なおこ の 外ほか に 欽定憲法きんていけんぽう の名 を 冠かん するものがあっても、それは程度の差こそあれ、実質においては、やはりこの種の協約憲法以外のものではない。  

 　しかるに帝国憲法は 、 万世一系ばんせいいっけい の天皇が「祖宗 に 承う くるの大権」をもっ て 大御心おおみこころ のままに制定あそばされた欽定憲法であって、皇室典範とともに全く「みことのり」に外ならぬ。  

 　而してこの欽定せられた憲法の内容は、外国におけるがごとき、制定当時の権力関係を永久に固定せんがために規範化したものでもなく、あるいは民主主義・法治主義・立憲主義・共産主義・独裁主義等の抽象的理論または実践的要求を制度化したものでもない。また外国の制度を移植 し 模倣もほう したものでもなく 、 皇祖皇宗こうそこうそう の御遺訓を顕彰せられた統治の洪範 に 外ほか ならぬ。これは、典憲欽定に際し て 皇祖皇宗こうそこうそう の神霊 に 誥つ げたもうた御告文に、  

 　皇祖
　皇宗の遺訓を明徴めいちょう にし、典憲を成立し、条章を昭示し  

 と仰せられ、また、  

 　皇祖
　皇宗の後裔に貽のこ したまえる統治の洪範を紹述するに外ならず  

 と 宣のたま わせられたことによって も 昭あき らかである。  

 　かくのごと き 皇祖皇宗こうそこうそう の御遺訓を紹述せんと の 大御心おおみこころ は、ひとり典憲欽定に際してのみならず、明治 の 御代みよ を一貫し て 渝かわ らせられぬものであったことは、  

 　世はいかに　開ひら けゆくとも　いにしえの　国のおきては　たがえざらなん
　かみつよの　御代みよ のおきてを　たがえじと　思うぞおのが　ねがいなりける
　さだめたる　国のおきては　いにしえの　聖ひじり の君の　みこえなりけり  

 の 御製ぎょせい によって も 拝はい せられる。しかも、かくのごと き 叡慮えいりょ は明治 の 御代みよ に限られたことではなく、御歴 代 一貫いっかん の 大御心おおみこころ である 。 皇祖皇宗こうそこうそう の 御 遺訓いくん は歴代天皇によっ て 紹述しょうじゅつ せられるのであって、こ こ 万世一系ばんせいいっけい の 皇統こうとう は自然の御一系たらせられるのみではなく、同時に御自覚の御一系たらせたま う 有難ありがた き事実が拝せられる。ゆえに 、 欽定きんてい あそばされ た 典憲てんけん は 、 皇祖皇宗こうそこうそう の 後裔こうえい に 貽のこ したまえる 御 統治とうち の 洪範こうはん の 紹述しょうじゅつ として、これ を 奉戴ほうたい し、またひとえにかくのごときものとし て 謹解きんかい し 、 循行じゅんこう するを要する。  

 　而してこ の 連綿不断れんめんふだん の御統治の洪範を新たに典憲として紹述あそばされたのは 、 御告文おつげぶみ に、  

 　顧かえり みるに、世局せいきょく の進運に膺あた り、人文じんぶん の発達に随したが い、宜よろ しく
　皇祖
　皇宗の遺訓いくん を明徴めいちょう にし、典憲を成立し、条章を昭示しょうじ し、内うち はもって子孫の率由そつゆう するところとなし、外そと はもって臣民しんみん 翼賛よくさん の道みち を広め、永遠に遵行せしめ、ますます国家の丕基ひき を鞏固きょうこ にし、八洲はっしゅう 民生みんせい の慶福を増進すべし。  

 と仰せられてあるところにうかがわれる。国運 の 隆昌りゅうしょう 、 臣民しんみん の 懿徳いとく 良能の発揚、慶福の増進を念じさせたまうことは、 「 天壌無窮てんじょうむきゅう の 宏謨こうぼ 」 に 循したが わせたまい、「祖宗の遺業を永久 に 鞏固きょうこ ならし」めたま う 所以ゆえん である。而して憲法欽定の特殊なる御目的は、君臣 の 遵守規範じゅんしゅきはん を 明徴めいちょう にし、ま た 臣民しんみん 翼賛の道を広めたまうところにあることが拝せられる。而して世局の進運 、 人文じんぶん の発達が、この憲法御制定 の 機縁きえん となっている。このこともまた「そ れ 大人ひじり の 制のり を立つる 、 義ことわり 必ず時 に 随したが う」との 御 祖訓そくん に 随したが わせたもうたのである。かくのごと き 立憲りっけん の御精神を拝して外国における憲法制定 の 由来ゆらい に思い を 及およ ぼす時、よ く 彼我ひが の憲法の本質的差異を知ることができる。  

 　我が憲法 に 祖述そじゅつ せられてあ る 皇祖皇宗こうそこうそう の御遺訓中、最も基礎的なものは 、 天壌無窮てんじょうむきゅう の 神勅しんちょく である。こ の 神勅しんちょく は 、 万世一系ばんせいいっけい の天皇 の 大御心おおみこころ であり 、 八百万やおよろず ノ神の念願であるとともに、一切の国民の願いである。したがって、知ると知らざるとにかかわらず、現実に存在し規律す る 命法めいほう である。それはひとり将来に向かって の 規範きはん たるのみならず 、 肇国ちょうこく 以来の一 大 事実じじつ である。憲法第一条に「大日本帝国は 、 万世一系ばんせいいっけい の天皇これを統治す」とあるのは、これを昭示したもうたものであり、第二条 は 皇位継承こういけいしょう の資格ならびに順位を昭らかにしたまい、第四条前半 は 元首げんしゅ ・ 統治権とうちけん 等、明治維新以 来 採択さいたく せられ た 新あたら しき概念をもって、第一条をさら に 紹述しょうじゅつ したもうたものである。天皇は統治権 の 主体しゅたい であらせられるのであって、かの統治権の主体は国家であり、天皇はそ の 機関きかん に過ぎないとい う 説せつ のごときは、西洋国家学説の無批判的 の 踏襲とうしゅう という以外に は 何等なんら の根拠はない。天皇は、外国のいわゆる元首・君主・主権者・統治権者たる に 止とど まらせられる御方ではなく 、 現御神あきつみかみ とし て 肇国ちょうこく 以来の大義 に 随したが って、この国をしろしめしたまうのであって、第三条に「天皇は、神聖にし て 侵おか すべからず」とあるのは、これを昭示せられたものである。外国において見られるこれ と 類似るいじ の規定は 、 勿論もちろん かかる深い意義に基づくものではなくして、元首の地位を法規によって確保せんとするものに過ぎない。  

    

天皇御親政

 　なお、帝国憲法の他の規定は、すべてかくのごとき御本質を有せられる天皇御統治の準則である 。 就中なかんずく 、そ の 政体法せいたいほう の根本原則は、中世以降のごとき 御 委任いにん の政治ではなく、あるいはまた英国流の 「 君臨くんりん すれど も 統治とうち せず」でもなく、または君民共治でもなく 、 三権分立さんけんぶんりゅう 主義で も 法治ほうち 主義でもなくして、一に天皇の 御 親政しんせい である。これは 、 肇国ちょうこく 以 来 万世一系ばんせいいっけい の天皇 の 大御心おおみこころ においては一貫せる御統治 の 洪範こうはん でありながら、中世以 降 絶た え て 久ひさ しく政体法上制度化せられなかったが、明治維新におい て 復古ふっこ せられ、憲法にこれを明示したもうたのである。  

 　帝国憲法の政体法 の 一切いっさい は、この御親政の原則の拡充紹述 に 外ほか ならぬ。例え ば 臣民しんみん 権利義務の規定のごときも、西洋諸国におけ る 自由権じゆうけん の制度が 、 主権者しゅけんしゃ に対し て 人民じんみん の 天賦てんぷ の 権利けんり を 擁護ようご せんとするのとは異なり、天皇 の 恵撫けいぶ 慈養じよう の御精神と、国民 に 隔へだ てなき翼賛の機会 を 均ひと しうせしめたまわんと の 大御心おおみこころ よ り 打出うちい づるのである。政府・裁判所・議会 の 鼎立ていりつ のごときも、外国における三権分立のごとくに、統治者の権力 を 掣肘せいちゅう せんがために、その統治権者より司法権と立法権とを奪い、行政権のみを容認し、これを掣肘せんとするものとは異なって、我が国においては 、 分立ぶんりゅう は統治権の分立ではなくして 、 親政輔翼機関しんせいほよくきかん の分立に過ぎず、これによって天皇の御親政の翼賛をいよいよ確実ならしめんとするものである。議会のごときも、いわゆる民主国においては、名義上の主権者たる人民の代表機関であり、ま た 君民共治くんみんきょうち のいわゆる君主国においては、君主の専横を抑制し、君民共治するための人民の代表機関である。我が帝国議会は 、 全まった くこれと異なって、天皇の御親政を、国民をして特殊の事項につき特殊の方法をもって、翼賛せしめたまわんがために設けられたもの に 外ほか ならぬ。  

    

我が国の法ほう

 　我が国 の 法ほう は、すべてこ の 典憲てんけん を基礎として成立する。個々 の 法典ほうてん 法規ほうき としては、直接 御 親裁しんさい によって定まるものもあれば、天皇の 御 委任いにん によって制定せられるものもある。しかし、いずれも天皇 の 御稜威みいつ に 淵源えんげん せざるものはないのである。その内容についても、これを具体化す る 分野ぶんや およびその程度には、種々の品位階次の相違はあるが、結局においては、 御 祖訓そくん 紹述しょうじゅつ のみことのりたる典憲の具体化ならぬはない。したがって 、 万法ばんぽう は天皇 の 御稜威みいつ に帰する。それゆえに、我が国の法は、すべて我が国体 の 表現ひょうげん である。  

 　かくて我が国の法は 、 御稜威みいつ の 下もと に 、 臣民しんみん 各自が皇運扶翼のために、まことを尽くし 、 恪循かくじゅん す る 道みち を示されたものである。され ば 臣民しんみん が 国憲こっけん を重んじ 、 国法こくほう に 遵したが うは、国民 が 真しん に 忠良ちゅうりょう な る 臣民しんみん として生き る 所以ゆえん である。  

    

経済

 　経済は、物資に関する国家生活の内容をなすものであって、物資は、ただに国民の生活 を 保たも つがために必要なるのみならず 、 皇威こうい を発揚するがための不可欠なる条件をなすものである。したがって、国の経済力 の 培養ばいよう は 、 皇国こうこく 発展の一つの重要なる基礎である。  

 　されば 、 畏かしこ く も 肇国ちょうこく の当初において、皇祖 が 親した し く 生業せいぎょう をさずけたまい、経済すなわち産業が国の大業に属することを御示しあそばされた 。 神武じんむ 天皇は「いやしくも民 に 利くぼさ あらば、何 ぞ 聖造ひじりのわざ に 妨たが わん」 と 宣のたま い、さら に 崇神すじん 天皇は「農は天下 の 大本おおもと なり、民 の 恃たの みてもって生くるところなり」と仰せられ、歴代の天皇は常に億 兆 臣民しんみん の生業を 御 軫念しんねん あそばされた。しかるに、久しきにわたる封建時代において、職業は漸次固定し、経済 は 著いちじる しく硬化したために、産業の発達には見るべきものが少なかった。江戸時代の末においては、これを打開せんがた め 幾多いくた の経済学者および経済生活の指導者が現れた 。 就中なかんずく 、 二宮尊徳にのみやそんとく のごときはそ の 著いちじる しいものである。尊徳において は 一円融合いちえんゆうごう の理 、 報徳ほうとく の道を説き 、 勤労きんろう ・ 分度ぶんど ・ 推譲すいじょう を主張し、これを天地の大法に合致す る 大道だいどう とし 、 皇国こうこく 本源の道 を 示現じげん するものとし て 説と いた。  

 　我が国が明治維新によって世 界 列強れっきょう の間 に 伍ご するや、従来の農業生産のみをもってしては、経済力の発展 を 図はか ること の 困難こんなん なること が 痛感つうかん せられた。ここにおいて明治以来しばし ば 聖諭せいゆ を下したまい、近代西洋の生産技術を採用し、また勤倹の重んずべき を 訓誡くんかい あそばされ、また実業教育を整え、産業を奨励し、もっ て 国富こくふ の増進 、 臣民しんみん の慶福のため に 大御心おおみこころ を注がせたもうた 。 臣民しんみん もまたよく天皇 の 大御心おおみこころ を体し、官民協力、勤倹よ く 産さん を 治おさ めて、今日見るがごと き 国力こくりょく の充実を見るに至ったのであり、その急速なる発達は、世界 の 驚異きょうい とするところである。  

 　我が国民経済は 、 皇国こうこく 無窮の発展のため の 大御心おおみこころ に基づく大業であり、民の慶福 の 倚よ るところのものであって、西洋経済学の説くがごとき個人の物質的欲望を充足するための活動 の 聯関れんかん 総和ではない。それは、国民 を 挙あ げて「むすび」 の 道みち に 参さん じ、各人そ の 分ぶん に従い、おのおのそのつとめを尽くすところのものである。我が国に早くより発達し た 農事のうじ は、地物そのものの生成を人の力によって育成することであり、人と土と が 和合わごう して生産 を 営いとな むことである。これ我が国産業の根本精神である。近代 に 勃興ぼっこう した商工業といえども、もとよりこれと同一の精神によっ て 営いとな まるべきはいうまでもない。  

 　我が国近代の経済活動 の 根柢こんてい には、西洋思想 の 著いちじる しい浸潤があるにもかかわらず、常にかか る 肇国ちょうこく 以来の産業精神が流れていたと見るべきである。もとより我が国民 の 悉ことごと くが、その経済活動において常にかかる精神を意識していたというのではなく、また我が国民が、生産活動のあらゆる場合に 、 営利えいり の観念を離脱していたというのでもない。しかし我が国の産業に従事する者の多くが、単に自己の物質的欲望の充足に導かれるというよりは、むしろ何よりもまずおのおの の 職分しょくぶん を守り、つとめを尽くすという精神によって和合の中にその業務にいそしんで来たことは 、 見逃みのが し難い事実である。さればこそ、最近に見るがごとき我が産業界の世界的躍進をもたら し 得え たのである。  

 　「むすび」の精神 を 本もと とし 、 公おおやけ を先に し 私わたくし を後にし 、 分ぶん を守りつとめを尽くし 、 和わ をもって旨とする心こそ、我が国固有の産業精神であって、それは産業界に強き力を生ぜしめ、創意 を 奨すす め、協力をもたらし 、 著いちじる しくその能率を高め、産業全体 の 隆昌りゅうしょう を来たし、やがて国富を増進す る 所以ゆえん となる。将来我が国民の経済活動においては、この特有の産業精神が十分に自覚せられ、これに基づいていよいよその発展 が 図はか られねばならぬ。かくて、経済は道徳と一致し、利欲の産業にあらずして 、 道みち に基づく産業となり、よく国体 の 精華せいか を経済において発揚 し 得う ることとなるであろう。  

    

軍事

 　我が国体の再現は、軍事について も 全まった く同様である。古来我が国においては、神の御魂 を 和魂にぎみたま ・ 荒魂あらみたま に 分わか っている。この両面の働き の 相協あいかな うところ、万物はおのおのそのところ に 安やす んずるとともに、いよいよ生成発展する。而し て 荒魂あらみたま は 、 和魂にぎみたま と離れずして一体の働きをなすものである。この働きによって天皇 の 御稜威みいつ にまつろわぬものを「ことむけやわす」ところに皇軍の使命があり、いわゆ る 神武しんぶ とも称すべ き 尊とうと き 武ぶ の 道みち がある。明治天皇 の 詔みことのり には「祖宗以 来 尚武しょうぶ の国体」 と 仰おお せられてある。天皇は明治六 年 徴兵令ちょうへいれい を 布し かせられ 、 国民皆兵こくみんかいへい の 実じつ を 挙あ げさせたまい、同十五年一月四日には、陸海軍軍人 に 勅諭ちょくゆ を 賜たま わって、  

 　我が国の軍隊は、世々天皇の統率したまうところにぞある。  

 と 仰おお せ 出いだ され、また、  

 　朕は汝等なんじら 軍人の大元帥だいげんすい なるぞ。されば、朕は汝等を股肱ここう と頼たの み、汝等は朕を頭首とうしゅ と仰ぎてぞ、その親しみは特こと に深かるべき。朕が国家を保護して上天の恵みに応じ祖宗の恩に報むく いまいらすることを得う るも得え ざるも、汝等軍人がその職を尽くすと尽くさざるとに由よ るぞかし。我が国の稜威いつ 振るわざることあらば、汝等能よ く朕とその憂いを共にせよ。我が武ぶ これ揚あが りてその栄えい を耀かがや かさば、朕汝等とその誉ほまれ を偕とも にすべし。汝等皆その職を守り朕と一心ひとつこころ になりて力を国家の保護に尽くさば、我が国の蒼生そうせい は永く太平の福さいわい を受け、我が国の威烈いれつ は大いに世界の光華こうか ともなりぬべし。  

 と 諭さと したもうた。この勅諭は 、 畏かしこ くも天威 に 咫尺しせき したてまつるがごと く 尊とうと く拝誦せられる。まことに皇軍の使命は 、 御稜威みいつ をかしこみ 、 大御心おおみこころ のまにまによ く 皇国こうこく を保全し、国威を発揚するにある。我が皇軍は、この精神によって日清・日露の戦 を 経へ て、世界大戦に参加し、大いに国威 を 中外ちゅうがい に輝かし、世界列強の中に立ってよく東洋の平和を維持し、また広く人類の福祉を維持増進するの責任ある地位に立つに至った。  

 　ここにおいて、我等国民は、 「 文武ぶんぶ 互いにその職分 に 恪循かくじゅん し、衆庶おのおのその業務 に 淬励さいれい し」と仰せられた聖旨を奉体し 、 協心戮力きょうしんりくりょく ・ 至誠奉公しせいほうこう 、もっ て 天壌無窮てんじょうむきゅう の皇運を扶翼したてまつり 、 臣民しんみん たる の 本分ほんぶん を 竭つ くさねばならぬ。  

 


 

結語

    

 　我等は、以上我が国体の本義とそ の 国史こくし に顕現す る 姿すがた とを考察して来た。今や我 等 皇国こうこく 臣民しんみん は、現下の諸問題に対していかなる覚悟と態度とをもつべきであろうか 。 惟おも うに、ま ず 努つと むべきは、国体の本義に基づいて諸問題の起因をなす外来文化 を 醇化じゅんか し、新日本文化を創造するの事業である。  

 　我が国に輸入せられた各種の外来思想は、シナ・インド・欧米の民族性や歴史性に由来する点において、それらの国々においては当然のものであったにしても、特殊な国体をもつ我が国においては、それが我が国体に適する か 否いな かがまず厳正に批判検討せられねばならぬ。すなわちこの自覚とそれに伴う醇化とによって、はじめて我が国として特色ある新文化の創造が期し得られる。  

    

西洋思想の特質

 　そもそも西洋思想は、そ の 源みなもと をギリシャ思想に発している。ギリシャ思想は 、 主知的精神しゅちてきせいしん を基調とするものであり 、 合理的ごうりてき ・ 客観的きゃっかんてき ・ 観想的かんそうてき なることを特徴とする。そこには、都市を中心として文化が創造せられ、人類史 上 稀まれ に見る哲学・芸術等 を 遺のこ したのであるが、末期に至ってはその思想および生活において 、 漸次ぜんじ に個人主義的傾向を生じた。而してローマは、このギリシャ思想を法律・政治その他の実際的方面に継承し発展せしめると同時に、超国家的なキリスト教を採用した。欧米諸国の近世思想は、一面にはギリシャ思想を復活し、中世期の宗教的圧迫 と 封建的専制ほうけんてきせんせい とに反抗し、個人の解放、その自由の獲得を主張し、天国を地上 に 将来しょうらい せんとす る 意図いと に 発足ほっそく したものであり、他面には、中世期の超国家的 な 普遍性ふへんせい と 真理性しんりせい とを尊重する思想を継承し、しかもこれを地上 の 実証じっしょう に求めんとするところから出発した。これがため自然科学を発達せしめるとともに、教育・学問・政治・経済等の各方面において、個人主義・自由主義・合理主義を主流として、そこに世界史的に特色ある近代文化 の 著いちじる しい発展をもたらした。  

 　そもそも人間は現実的の存在であるとともに永遠なるもの に 連つら なる歴史的存在である。また、我であると同時 に 同胞どうほう たる存在である。すなわち国民精神により歴史に基づいてその存在が規定せられる。これが人間存在の根本性格である。この具体的な国民としての存在を失わず、そのまま個人として存在するところに深い意義が見出される。しかるに、個人主義的な人間解釈は、個人たる一面のみを抽象して、その国民性と歴史性とを無視する。したがって、全体性・具体性を失い、人間存立の真実 を 逸脱いつだつ し、その理論は現実よ り 遊離ゆうり して、種々 の 誤あやま った傾向 に 趨はし る。ここに個人主義・自由主義ないしその発展たる種々の思想の根本的な る 過誤かご がある。今や西洋諸国においては、こ の 誤謬ごびゅう を自覚し、而してこれ を 超克ちょうこく するために種々の思想や運動が起こった。しかしながら、これら も 畢竟ひっきょう 個人の単なる集合をもって団体あるい は 階級かいきゅう とするか、ないしは抽象的の国家 を 観念かんねん する に 終おわ るのであって、かくのごときは誤謬 に 代か うるに誤謬をもってするに止まり、決して真実の打開解決ではない。  

    

東洋思想の特質

 　我が国に輸入せられたシナ思想は、主とし て 儒教じゅきょう と 老荘思想ろうそうしそう とであった。儒教は実践的な道として優れた内容をもち、すこぶる価値ある教えである。而し て 孝こう をもって教えの根本としているが、それはシナにおいて家族を中心とし て 道みち が立てられているからである。この孝は実行的な特色をもっているが、我が国のごと く 忠孝一本ちゅうこういっぽん の国家的道徳として完成せられていない。家族的道徳をもって国家的道徳の基礎とし、忠臣は孝子の門より出づるともいっているが、シナに は 易姓革命えきせいかくめい ・ 禅譲ぜんじょう 放伐ほうばつ が行われているから、その忠孝は歴史的・具体的な永遠の国家の道徳とはなり得ない。老荘は 、 人為じんい を捨てて自然に帰り 、 無為むい をもって化す る 境涯きょうがい を理想とし、結局その道は文化 を 否定ひてい する抽象的のものとなり、具体的な歴史的基礎の上に立たずして個人主義 に 陥おちい った。そ の 末流まつりゅう はいわゆ る 竹林ちくりん の 七賢しちけん のごとく、世間を離れて孤独を守ろうとする傾向を示し 、 清談せいだん 独善どくぜん の 徒と となった。要するに儒教も老荘思想も、歴史的に発展する具体的国家の基礎をもたざる点において、個人主義的傾向に陥るものといえる。しかしながら、それらが我が国 に 摂取せっしゅ せられるに及んでは、個人主義的・革命的要素は脱落し 、 殊こと に儒教は我が国体に醇化せられて日本儒教の建設となり、我が国民道徳の発達 に 寄与きよ すること が 大だい であった。  

 　インドにおける仏教は、行的・直観的な方面もあるが 、 観想的かんそうてき ・非現実的な民族性から創造せられたものであって 、 冥想的めいそうてき ・非歴史的・超国家的なものである。しかるに我が国 に 摂取せっしゅ せられる に 及およ んでは、国民精神 に 醇化じゅんか せられ、現実的・具体的な性格 を 得え て 、 国本こくほん 培養ばいよう に 貢献こうけん するところが多かったのである。  

    

新日本文化の創造

 　これを要するに、西洋の学問や思想の長所 が 分析的ぶんせきてき ・ 知的ちてき であるに対して、東洋の学問・思想は 、 直観的ちょっかんてき ・行的なることを特色とする。それは民族と歴史との相違から起こる必然的傾向であるが、これを我が国の精神・思想ならびに生活と比較する時は、なおそこに大なる根本的の差異を認めざるを得ない。我が国は、従来シナ思想・インド思想等を輸入し、よくこれを摂取醇化して皇道 の 羽翼うよく とし、国体に基づく独自の文化を建設し得たのである。明治維新以来、西洋文化 は 滔々とうとう とし て 流入りゅうにゅう し 、 著いちじる しく我が国運 の 隆昌りゅうしょう に貢献するところがあったが、その個人主義的性格は、我が国民生活の各方面にわたって種々 の 弊害へいがい を 醸かも し、思想の動揺 を 生しょう ずるに至った。しかしながら、今やこの西洋思想を我が国体に基づい て 醇化じゅんか し、もっ て 宏大こうだい なる新日本文化を建設し、これ を 契機けいき として国家的大発展をなすべき時 に 際会さいかい している。  

 　西洋文化の摂取醇化にあたっては、まず西洋の文物・思想の本質 を 究明きゅうめい することを必要とする。これなくしては、国体の明徴は現実を離れた抽象的のものとなるであろう。西洋近代文化 の 顕著けんちょ なる特色は、実証性を基とする自然科学およびその結果たる物質文化 の 華はな やかな発達にある。さらに精神科学の方面においても、その精密性と論理的組織性とが見られ、特色ある文化を形成している。我が国はますますこれらの諸学を輸入して、文化の向上、国家の発展を期せねばならぬ。しかしながらこれらの学的体系・方法および技術は、西洋における民族・歴史・風土の特性より来たる西洋独自の人生観・世界観によっ て 裏附うらづ けられている。それゆえに、我が国にこれを輸入するに際しては、十分この点に留意し、深くその本質 を 徹見てっけん し 、 透徹とうてつ し た 見識けんしき の下によくその長所を採用し短所を捨てなければならぬ。  

    

諸般の刷新

 　明治以来の我が国の傾向を見るに、あるいは伝統精神 を 棄す て て 全まった く西洋思想に没入したものがあり、あるいは歴史的な信念を維持しながら、しかも西洋の学術理論に関して十分な批判を加えず、そのままこれ を 踏襲とうしゅう し て 二元的にげんてき な思想 に 陥おちい り、しかもこれを意識せざるものがある。また著しく西洋思想の影響を受けた知識階級と、一般のものとは相当な思想 的 懸隔けんかく を来たしている。かくて、かかる情態から種々の困難な問題が発生した。かつて流行した共産主義運動、あるいは最近における天皇機関説の問題のごときが、往々にして一部の学者・知識階級の問題であったごときは、よくこの間の消息を物語っている。今や共産主義は衰頽し、機関説が打破せられたように見えても、それはまだ決して根本的に解決せられてはいない。各方面における西洋思想の本質の究明とその国体による醇化とが、今一段の進展を見ざる限り、真の成果を挙げる事は困難であろう。  

 　 惟おも うに西洋の思想・学問について、一般に極端なるもの、例えば共産主義・無政府主義のごときは 、 何人なんぴと も容易に我が国体 と 相容あいい れぬものであることに気づくのであるが、極端ならざるもの、例えば民主主義・自由主義等については、果してそれが我が国体と合致するや否やについては多くの注意を払わない。そもそもいかにして近代西洋思想が民主主義・社会主義・共産主義・無政府主義等を生んだかを考察するに、先に述べたごとく、そこにはすべての思想の基礎となっている歴史的背景があり、しかもそ の 根柢こんてい には個人主義的人生観があることを知るのである。西洋近代文化の根本性格は、個人をもって絶対独立自存の存在とし、一切の文化はこの個人の充実に存し、個人が一切価値の創造者・決定者であるとするところにある。したがって、個人の主観的思考を重んじ、個人 の 脳裡のうり に描くところの観念によってのみ国家を考え 、 諸般しょはん の制度を企画し、理論を構成せんとする。かくして作られた西洋の国家学説・政治思想は、多くは、国家をもって、個人を生み、個人を超えた主体的な存在とせず、個人の利益保護、幸福増進の手段と考え、自由・平等・独立の個人を中心とする生活原理の表現となった。したがって 、 恣ほしいまま な自由解放のみを求め、奉仕という道徳的自由を忘れ た 謬あやま れる自由主義や民主主義が発生した。而してこの個人主義とこれに伴う抽象的思想の発展するところ、必然に具体的・歴史的な国家生活は抽象的論理 の 蔭かげ に見失われ、いずれの国家も国民も一様に国家一般ないし人間一般として考えられ、具体的な各国家およびその特性よりも、むしろ世界一体の国際社会、世界全体に通ずる普遍的理論のごときものが重んぜられ 、 遂つい に は 国際法こくさいほう が 国法こくほう よりも高次の規範であり、高き価値をもち、国法はむしろこれに従属するものとするがごとき誤った考えすら発生するに至るのである。  

 　個人の自由なる営利活動の結果に対して、国家の繁栄を期待するところに、西洋における近代自由主義経済 の 濫觴らんしょう がある。西洋に発達した近代の産業組織が我が国に輸入せられた場合も、国利民福という精神が強く人心を支配していた間は、個人 の 溌剌はつらつ たる自由活動は著しく国富の増進に寄与し得たのであるけれども、その後、個人主義・自由主義思想の普及とともに 、 漸ようや く経済運営におい て 利己主義りこしゅぎ が公然正当化せられるがごとき傾向 を 馴致じゅんち するに至った。この傾向は貧富 の 懸隔けんかく の問題を発生せしめ、遂に階級的対立闘争の思想を生ぜしめる原因となったが、さらに共産主義の侵入するや、経済をもって政治・道徳その他百般の文化の根本と見るとともに、階級闘争を通じてのみ理想的社会を実現し得ると考うるがごと き 妄想もうそう を生ぜしめた。利己主義や階級闘争が我が国体に反することは説くまでもない。皇運扶翼の精神の下に、国民おのおのが進んで生業に競い励み、各人の活動が統一せられ、秩序づけられるところにおいてこそ、国利と民福と は 一如いちにょ となって、健全なる国民経済が進展し得るのである。  

 　教育についてもまた同様である。明治維新以後、我が国は進歩した欧米諸国の教育 を 参酌さんしゃく して、教育制度・教授内容等の整備に努め、また自然科学はもとより精神諸科学の方面においても大いに西洋の学術を輸入し、もって我 が 国学こくがく 問の進歩と国民教育の普及とを図って来た 。 五箇条ごかじょう の 御誓文ごせいもん を奉体して旧来 の 陋習ろうしゅう を破り、智識を世界に求め た 進取しんしゅ の精神は、この方面にもまた長足の進歩を促し、その成果は極め て 大だい なるものがあった。しかしそれと同時に個人主義思想の浸潤によって、学問も教育もややもすれば普遍的真理というがごとき、抽象的なもののみを目標として 、 理智りち のみの世界、歴史と具体的生活とを離れた世界 に 趨はし らんとし 、 智育ちいく も 徳育とくいく も知らず識らず抽象化せられた人間の自由、個人の完成を目的とする傾向を生ずるに至った。それと同時にまたそれらの学問・教育が、分化し専門化し て 漸ようや く綜合統一を欠き、具体性を失うに至った。この傾向を是正するには、我が国教育の淵源たる国体の真義を明らかにし、個人主義思想と抽象的思考との清算に努力する の 外ほか はない。  

 　かくのごとく、教育・学問・政治・経済等の諸分野にわたって浸潤している西洋近代思想 の 帰き するところは、結局個人主義である。而して個人主義文化が個人の価値を自覚せしめ、個人能力の発揚を促したことは、その功績といわねばならぬ。しかしながら西洋の現実が示すごとく、個人主義は 、 畢竟ひっきょう 個人と個人、ないしは階級間の対立 を 惹起じゃっき せしめ、国家生活・社会生活の中 に 幾多いくた の問題と動揺と を 醸成じょうせい せしめる。今や西洋においても、個人主義 を 是正ぜせい するため幾多の運動が現れている。いわゆる市民的個人主義に対する階級的個人主義たる社会主義・共産主義もこれであり、また国家主義・民族主義たる最近のいわゆるファッショ・ナチス等の思想・運動もこれである。  

 　しかし、我が国において真に個人主義のもたらし た 欠陥けっかん を是正し、その行き詰まりを打開するには、西洋の社会主義ないし抽象的全体主義等をそのまま輸入して、その思想・企画等 を 模倣もほう せんとしたり、あるいは機械的に西洋文化を排除することをもってして は 全まった く不可能である。  

    

我等の使命

 　今や我が国民の使命は、国体を基として西洋文化を摂取醇化し、もって新しき日本文化を創造し、進んで世界文化の進展に貢献するにある。我が国はつとにシナ・インドの文化を輸入し、しかもよく独自な創造と発展とをなし遂げた。これ正に我が国体の深遠宏大の致すところであって、これを承け継ぐ国民の歴史的使命はまことに重大である 。 現下げんか 国体明徴の声は極めて高いのであるが、それは必ず西洋の思想・文化 の 醇化じゅんか を 契機けいき としてなさるべきであって、これなくしては国体の明徴は現実 と 遊離ゆうり する抽象的のものとなり易い。すなわち西洋思想の摂取醇化と国体の明徴と は 相あい 離るべからざる関係にある。  

 　世界文化に対する過去の日本人の態度は、自主的にしてしかも包容的であった。我等が世界に貢献することは、ただ日本人たるの道をいよいよ発揮することによってのみなされる。国民は、国家 の 大本おおもと として の 不易ふえき な国体と 、 古今ここん に一貫し中外に施し て 悖もと らざ る 皇国こうこく の道とによって 、 維こ れ 新あら たなる日本をますます生成発展せしめ、もっていよい よ 天壌無窮てんじょうむきゅう の皇運を扶翼したてまつらねばならぬ。これ、我等国民の使命である。
