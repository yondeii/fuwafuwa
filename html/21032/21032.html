　　一章　 連環れんかん の 計けい



　　　　　１


　もの 悲がな しい 鳴な き 声ごえ が、 夕ゆう 焼や けの空にひびいている。 黒くろ い 羽は ばたきが、天をおおう。

　 都みやこ の 長安ちょうあん に、カラスが 増ふ えた。

　お目当ては、 宮殿きゅうでん から出る 大量たいりょう の 残飯ざんぱん と、 広場ひろば にさらされる 罪人ざいにん のしかばねである。

　四百年つづいた 漢かん 王朝おうちょう に、 滅ほろ びの 予感よかん がただよっていた。

　 最初さいしょ に漢の 政治せいじ を 乱みだ したのは、 宦官かんがん たちである。 彼かれ らは、 皇帝こうてい の 側近そっきん という 立場たちば を 利用りよう して 私腹しふく をこやし、 民たみ を 苦くる しめていた。

　 地方ちほう では、 黄巾こうきん の 乱らん という 反乱はんらん が 起お こった。反乱といっても、 実態じったい は 山賊さんぞく と 変か わりがない。

　この 討伐とうばつ に 活躍かつやく した 若わか い 将軍しょうぐん たちが、宦官たちから 権力けんりょく をとりもどそうとこころみた。 袁紹えんしょう と 曹操そうそう がその 中心ちゅうしん である。

　ところが、 混乱こんらん の 末すえ に、都の 新あたら しいあるじとなったのは、 董卓とうたく という人の 皮かわ をかぶった 獣けもの であった。 悪行あくぎょう のかぎりをつくす董卓に 対たい し、袁紹と曹操は 諸侯しょこう をあつめて 対抗たいこう する。

　袁紹ひきいる 連合軍れんごうぐん が 激闘げきとう の 果は てに 虎牢関ころうかん を 落お とすと、董卓は都の 洛陽らくよう を焼きはらい、長安に 逃亡とうぼう した。

　しかし、諸侯の 不和ふわ から、連合軍はここで 解散かいさん し、 各地かくち に 散ち った。袁紹、曹操、 孫堅そんけん 、 袁術えんじゅつ 、 公孫瓚こうそんさん といった諸侯は、 本拠地ほんきょち をさだめて 独立どくりつ をはたしている。

　そのような地方の 情勢じょうせい などおかまいなく、董卓は長安で 我わ が 世よ の 春はる を 楽たの しんでいた。

　長安には 宮殿きゅうでん がふたつある。

　 古ふる くから 残のこ るほうには、漢の皇帝である 献帝けんてい がとじこめられていた。

　献帝につきしたがっているのは、 皇后こうごう と 身み の 回まわ りの 世話せわ をする 者もの が 数人すうにん だけで、その何百 倍ばい もの董卓の手下が、 見張みは りについている。 衣服いふく や 食事しょくじ は、 庶民しょみん と変わらない。

　もうひとつの宮殿は、新しく 造つく らせた董卓の 居城きょじょう である。

　 柱はしら は 朱あか くぬられ、 屋根やね には 金箔きんぱく がはられ、 壁かべ には 真珠しんじゅ や 宝玉ほうぎょく がきらめいていて、 豪華ごうか なことこのうえない。この宮殿には、都からさらってきた八百人の 美女びじょ が 暮く らしていた。

　董卓は 毎日まいにち のように 宴会えんかい をひらき、ぜいたくをつづけている。

　少しでもさからうと 殺ころ されてしまうため、 誰だれ も 文句もんく は言わない。董卓のそばには、中国一の 猛将もうしょう である 呂布りょふ がひかえているので、 反抗はんこう できないのだ。

　あるとき、董卓は 宴うたげ の 席せき に都につとめる役人をまねいた。

　 参加さんか した百人あまりが、董卓の 顔色かおいろ をうかがいながら 酒さけ を 飲の んでいると、一人の兵士がやってきて、呂布に 何事なにごと か 報告ほうこく した。

「 承知しょうち した。ご 苦労くろう 」

　呂布は席を立って董卓に 近ちか づき、耳もとでささやく。

「何と、そのようなことが。では、こうするがよい」

　董卓の 指示しじ を 受う けた呂布は、 宴会場えんかいじょう をずんずんと歩いて、一人の 大臣だいじん の前でとまった。 張温ちょうおん という男である。

「ど、どうしたのですか」

　張温は 真ま っ 青さお になって、ふるえだした。

「ついてこい」

　呂布は 短みじか く 命めい じたが、張温が 腰こし をぬかしてしまったので、その 首くび 根ね っこをつかんで 持も ちあげた。

　ほかの役人たちが 固唾かたず をのんで見守るなか、呂布は張温を 連つ れて外に出る。

　しばらくすると、呂布は一人でもどってきた。ふたのついた 盆ぼん を 両手りょうて でささげもっている。呂布は董卓の前でふたをあけた。

　役人たちの 悲鳴ひめい がひびきわたる。

　そこにあったのは、張温の 生首なまくび であった。うらめしそうな顔で、 酒気しゅき のただよう宴会場を見つめている。

「ご苦労だったな」

　董卓は呂布に 杯さかずき をあたえてさがらせた。

「張温は袁術に 謀反むほん をすすめられておったので、こうして 始末しまつ させた。まあ、 当然とうぜん のむくいだな。おまえたちもわしにさからったらどうなるか、よくわかっただろう」

　董卓は張温の首をわきにおいたまま、 平気へいき な顔で酒を飲みつづけた。

「さあ、 遠慮えんりょ するな。どんどん食べてくれ」

　あつめられた役人たちはみな 黙だま りこくり、 必死ひっし の思いで 料理りょうり を口に入れた。色とりどりの 贅沢ぜいたく な料理はしかし、 灰はい のような 味あじ がするのだった。


　 王允おういん は宴の 様子ようす を思いだして、 深ふか いため 息いき をついた。

　董卓に 仕つか えるのはもうこりごりであった。 悪事あくじ の 片棒かたぼう をかついでいると思うと、 吐は き 気け がしてくる。それに、いつ、何のきっかけで殺されるかわからない。

　世のため人のため、そして何より自分のために、董卓を 討う ちたい気持ちがある。だが、呂布が守っている 以上いじょう 、手出しはできない。王允は五十 代だい のなかばで、 政治家せいじか としての 力量りきりょう は 評価ひょうか されていたが、 腕力わんりょく はからっきしだった。

　かといって、諸侯と 通つう じて 陰謀いんぼう をめぐらせれば、張温のような目にあう。長安を 脱出だっしゅつ するにしても、 一族いちぞく を 連つ れてどこまで逃げられるかわからない。 八方はっぽう ふさがりで、出口はないように思われる。

「何か 方法ほうほう がないものか」

　王允が 何度目なんどめ かのため息をついたとき、 背後はいご から声がかけられた。

「どうなさいましたの、お 父様とうさま 」

　 鈴すず を 転ころ がしたような 美うつく しい声である。王允はゆっくりとふりむいた。

「何でもないよ、 貂蝉ちょうせん 。少し 仕事しごと のことで 悩なや んでいただけだ」

　貂蝉は王允の友人の子どもで、 今年ことし 十六 歳さい になる。友人は洛陽で董卓に殺されたので、王允が 養女ようじょ としてこっそりと 育そだ てていた。

「もしかして、董卓のことではございませんか」

　王允はまじまじと貂蝉を見つめた。

　 長なが いまつげが、 濡ぬ れたような黒い 瞳ひとみ をふちどっている。 鼻はな すじはすっきりと 通とお って、ふっくらとした 紅あか い 唇くちびる のあいだから、白い歯がのぞいていた。はかなげな 細ほそ い腰に、たおやかな 仕草しぐさ 、あらためて見ると、貂蝉は男の目をひきつけて 離はな さぬ美女である。うれいを 秘ひ めた 表情ひょうじょう がよく 似に あっていた。

　王允が 無言むごん でいると、貂蝉は思いつめたように言った。

「 私わたし にできることは何でもおっしゃってください。お父様はみなしごの私を我が子のようにかわいがってくださいました。ご 恩おん 返がえ しがしとうございます」

　王允は貂蝉に 歌うた や 舞まい をならわせており、貂蝉はそのどちらにもすばらしい 才能さいのう をしめしていた。 平和へいわ な世であれば、 将来性しょうらいせい のある将軍か大臣に 嫁とつ いで、 幸しあわ せな 生活せいかつ を 送おく ることができただろう。

「呂布という男が 部下ぶか にいるから、董卓は思うがままにふるまっているのでしょう？　二人を 仲なか 違たが いさせればよろしいではありませんか」

「連環の計か......」

　王允は 娘むすめ が言わんとすることを 察さっ した。

　貂蝉は美しいだけではない。かしこく、そして 勇気ゆうき ある娘だ。

「私ならできる。お父様もそうお思いでしょう？」

「いや、しかし......」

　王允はためらった。

　貂蝉は自分の 魅力みりょく で董卓と呂布をとりこにし、二人を 争あらそ わせようというのである。

「かわいい娘に、そんな 危険きけん なまねはさせられぬ」

「私は董卓が 憎にく いのです。あの男に 復讐ふくしゅう できるのならば、この 身み がどうなろうと 悔く いはありません」

　 強つよ い 調子ちょうし で、貂蝉は言い切った。

　その美しい姿を見て、王允は大きく息を吐きだした。

「わかった。そこまで言うのであれば、 計略けいりゃく をさずけよう」

「はい、お父様」

　貂蝉はあでやかにほほえんだ。

　 一変いっぺん したその表情は、王允に計略の 成功せいこう を 信しん じさせ、さらにつらい気持ちにさせたのだった。



　　　　　２


　王允は心を 鬼おに にして、「連環の計」にとりかかった。

　まず、 大金たいきん をはたいて 豪華ごうか な 冠かんむり をつくらせ、呂布に 贈おく った。呂布が 好す きな 金色きんいろ で、大きな 真珠しんじゅ をはめこんだ 逸品いっぴん である。

　呂布は 喜よろこ び、お 礼れい を言うために王允の 屋敷やしき にやってきた。 待ま ちかまえていた王允は、とっておきの 美酒びしゅ と料理でもてなす。

「 将軍しょうぐん のようなお 忙いそが しい 方かた に、わざわざ足を 運はこ んでいただいて 申もう し 訳わけ ございません。お口に合うかわかりませんが、どうか 召め し上がってください」

「これはかたじけない。 先日せんじつ の贈り物もすばらしかった。 朝廷ちょうてい の大臣だけあって、さすがに王允 殿どの は気が 利き くな」

「私が今の 地位ちい にあるのも、すべて董卓様と呂布様のおかげでございます。ご 恩おん 返がえ しをするのは 当然とうぜん です」

　王允は呂布にお 世辞せじ を言ってもちあげ、しきりと酒をすすめた。おだてられた呂布は 機嫌きげん 良よ く笑いながら、 杯さかずき をからにしていく。


　呂布がほろ 酔よ い 気分きぶん になったところを 見計みはか らって、王允は貂蝉を呼んだ。

「おまえもこちらに来て、お 酌しゃく をしなさい」

「はい、お父様」

　しずしずとあらわれた貂蝉が、呂布のとなりにつく。

「ん、こちらは......」

　ゆっくりと顔をあげた貂蝉を目にして、呂布は言葉を 失うしな った。 着き 飾かざ った貂蝉の美しさに、思わず見とれてしまったのである。

「養女の貂蝉でございます」

　貂蝉が酒をついでも、呂布はまだぼうっとしている。

「さ、いかがですか」

　貂蝉がささやきかけると、呂布はびくっとして 我われ に返り、あわてて酒を飲み 干ほ した。

「すまん。しかし、王允殿にこれほど美しい娘がいるとは知らなかった」

「もったいないお言葉でございます。では、 余興よきょう をお見せしましょう」

　王允は貂蝉に舞を 披露ひろう するよう命じた。貂蝉は 着替きが えのためにいったん 席せき をはずす。その 後うし ろ 姿すがた を 追お って、呂布の 視線しせん が動いた。

「この娘の 両親りょうしん はすでに 亡な くなってましてな。私が引き取って 育そだ てているのです。よい 縁談えんだん を 世話せわ してやれれば 安心あんしん なのですが」

「ああ、そうか」

　呂布は 上うわ の 空そら で答えてから、はっとして王允に顔を向けた。

「行き先は決まってないのか」

　呂布は 愛人あいじん をたくさんかかえているが、まだ 妻つま はいない。王允はもったいぶってうなずいた。呂布の 反応はんのう は 計画けいかく 通どお りである。

「ええ、なかなかふさわしい方が見つかりません」

　貂蝉がもどってきたので、 会話かいわ は 中断ちゅうだん された。 笛ふえ の 音ね にあわせて、 優美ゆうび に舞いおどる貂蝉に、呂布の目は 釘くぎ 付づ けになる。

　 曲きょく が 終お わると、貂蝉は一度だけ、呂布と目を合わせて 微笑ほほえ んだ。どんな男もひきつける 魔性ましょう の笑みだった。

　貂蝉が 去さ ると、呂布はがばっと王允に 向む きなおった。思わず王允がのけぞるほどの 勢いきお いである。

「王允殿、 頼たの みがある」

「何でございましょう」

「貂蝉を妻にしたい」

　 単刀直入たんとうちょくにゅう に、呂布は言った。王允はおどろいたふりをしながら、 内心ないしん でよしよしとつぶやいた。 頭あたま の中で五つ数えてから、 返事へんじ をする。

「ありがとうございます。将軍に 見初みそ めていただいて、こんなにうれしいことはありません。ただ、いろいろと 準備じゅんび がございますので、 一月ひとつき 後ご にまたお会いして、 具体的ぐたいてき な話をするということでいかがでしょうか」

「あ、ああ、そうだな。 承知しょうち した」

　呂布は顔を赤くしてうなずいた。 急きゅう に 恥は ずかしくなったようだ。

「 突然とつぜん 、言われても 困こま るよな。すまなかった。では、一月後を楽しみにしている」

　そそくさと王允の屋敷を後にする呂布であった。


　それからというもの、王允は 積極的せっきょくてき に貂蝉を 人前ひとまえ に出した。三日に一度は宴会をひらいて、 同僚どうりょう の前で貂蝉に舞わせ、何か 行事ぎょうじ があれば貂蝉を 同行どうこう させた。

　貂蝉の美しさはすぐに、 街中まちじゅう の 評判ひょうばん となった。いくつかの縁談も舞いこんできたが、すでに呂布の申し込みを受けていると 断ことわ る。そうすると、相手は 無念むねん そうにあきらめる。ただ一人をのぞいて。

　二十日ほど 経た ったある日、王允に董卓の宮殿から呼び出しがかかった。

「いよいよ来たか」

　 暗くら い表情の王允に、貂蝉が笑いかける。

「計画はうまく行っているのです。もっと喜んでくださいな」

「あ、ああ」

　王允はこれから董卓に仕えねばならない貂蝉のつらさを思うと、とても笑うことはできない。だが、今さら引き返すこともできないのである。

　宮殿をたずねた王允に、董卓はせかせかと話しかけた。

　劉備と関羽はその日のうちに準備をして、曹操の陣へと旅立った。

　そのころ、曹操軍はすでに兗州を出発し、東へ向かっていた。 総勢そうぜい 十万の大軍で、夏侯惇が 先鋒せんぽう をつとめ、夏侯淵、曹洪、典韋らの 武将ぶしょう が 本隊ほんたい の曹操にしたがっている。陳留の守りは、曹仁と荀彧がかためていた。

　曹操は、配下の将軍のうち、 一族いちぞく の夏侯惇と曹仁に深い信頼を寄せている。それゆえ、独立した部隊を任せるのである。逆に、夏侯淵と曹洪は、 手元てもと において 指示しじ を出してこそ、力を 発揮はっき できると考えている。もちろん、 今後こんご は 成長せいちょう して、 一軍いちぐん を任せられるようになってくれるだろう。

　徐州の 境さかい を目前にしたある日、曹操の陣に劉備からの手紙が 届とど いた。陶謙からの使者として交渉におとずれたのだという。いきなりたずねて 攻撃こうげき されるのをおそれ、あらかじめ手紙で 意図いと を知らせたのだった。

「ほほう、あの劉玄徳か」

　曹操は手紙を読みながら、数年前の 記憶きおく をたどった。劉備たち義兄弟の活躍はよくおぼえている。関羽と張飛の 武勇ぶゆう はもちろん、劉備という男の 独特どくとく の 雰囲気ふんいき が印象にのこっていた。

　高い地位も多い兵力もないのに、黄巾の乱でも虎牢関の戦いでも、人の 耳目じもく を集める 働はたら きを見せた。いずれ、 乱世らんせ の 雄ゆう として 台頭たいとう してくるかもしれない。早めにたたきつぶすか、味方にするか考えなければならないだろう。

「そういうお考えでしたら、私に 策さく があります」

　郭嘉は曹操に近づき、なにやら耳打ちした。曹操は大きくうなずき、夏侯淵を呼びだした。本隊の進軍をとめ、劉備をむかえる準備をするよう命じる。

「郭嘉の 智謀ちぼう は私の上をいくな。おぬしが味方でよかった。今後とも 助言じょげん してくれるとありがたい」

「もったいないお言葉でございます。では、ひとつお願いをしてもよろしいでしょうか」

　郭嘉が 姿勢しせい をととのえる。曹操は 手て 振ぶ りで先をうながした。

「曹操様は徐州の者は兵も民も殺すよう命令を出されました。これは 撤回てっかい していただきたいと思います」

「復讐ではなく、 領土りょうど 拡大かくだい が目的だからか」

「おっしゃるとおりです」

　 無人むじん の地を領土にしても仕方がない。 将来しょうらい の 領民りょうみん に憎まれるような 行為こうい はつつしむべきだと、郭嘉は言う。 正論せいろん だ。

「考えておこう」

「それでは間に合いません。すぐに先鋒部隊にご命令ください」

「な......」

　郭嘉は 絶句ぜっく した。曹操の視線が、 刃やいば のごとく 鋭するど い。

「もちろん、非をみとめて許しを 請こ う者は 別べつ だ。だが、私に 敵対てきたい したらどうなるか、 世間せけん に知らしめる 必要ひつよう があるのでな」

「 我わ が行くは 覇道はどう 。 出自しゅつじ や 血統けっとう にたよらず、知恵と力で切りひらく道だ。 流血りゅうけつ をためらっていては、前に進むことはできぬ。おぬしも 覚悟かくご しておいてもらいたい」

　郭嘉はうつむいて何度かくちびるをふるわせていたが、やがてすっと顔をあげた。

「承知しました。曹操様について行きます」


「これからの働きも期待しているぞ」

　曹操は満足そうに言った。

　郭嘉は曹操の考えにすべて 賛同さんどう したわけではない。だが、その 器量きりょう と自信と、 気宇きう の 壮大そうだい さに 衝撃しょうげき を受けた。新しい時代を 創つく るのは、曹操のような人物にちがいない。曹操に才をみとめられ、そのもとで力を発揮するのは、 何物なにもの にも 替か えがたい喜びだと思えるのであった。


　曹操は 陣中じんちゅう とは思えない 豪華ごうか な料理と酒で、劉備と関羽をもてなした。ただ、曹操本人は顔を出さず、郭嘉が代わりに応対した。

「申し訳ございません。曹操様は途中で病気にかかって、 到着とうちゃく が 遅おく れております。三日のうちには 着つ くはずですので、それまでゆるりとお待ちください」

　劉備は顔をしかめた。

「ゆっくりしているひまはない。早く曹操殿にお会いして、戦をとめてもらわねばならぬのだ」

「そう言われましても、私たちも 主君しゅくん の指示なしでは動けないものですから、どうしようもありません」

　劉備は仕方なく、曹操の到着を待った。きっちり三日後、ようやく曹操が顔を見せた。

「待たせてすまなかった。それにしても久しぶりだな。虎牢関でともに戦ったときは、このようなかたちで再会するとは思わなかった」

　劉備は 世間せけん 話ばなし や思い出話に 興きょう じるつもりはなかった。

「このたびはお願いがあって参りました」

　曹操が 苦笑くしょう する。

「 内容ないよう はわかっている。だが、かなえてやるわけにはいかぬ」

「 筋すじ のとおらぬ 恨うら みでみだりに戦をしかけるのは、 武ぶ を 汚けが す行為です。つみあげた 名声めいせい が地に落ちますぞ」

　劉備は正面から斬りこんだが、曹操の心は動かない。

「親を殺されて手をこまねいていては、 不孝ふこう のそしりを受けるであろう。 仇あだ 討う ちは子のつとめである」

「仇討ちであれば、犯人を 探さが してつかまえればいいでしょう」

「それは陶謙がすべきことではないかな。犯人は陶謙の部下で、陶謙が父の護衛に 差さ し向けた男だ。犯人を差しだすことができなければ、陶謙の 責任せきにん を 問と うのが当然だと思わないか」

　劉備は言葉につまった。曹操の父をおそった犯人の行方は知れない。袁紹か袁術のもとに逃げたのではないかと思われているが、探している時間はなかった。

「陶謙が犯人をかくまっているかもしれぬ。 否定ひてい する 証拠しょうこ はあるか」

「陶謙様はそのような方ではありません」

　劉備は言ったが、それでは証拠にならない。別な方向から説得する必要を感じた。

「戦になれば民が苦しみます。英雄たる者は 下々しもじも の幸せを考えるべきではありませんか」

「英雄とは 光栄こうえい だな」

　曹操は微笑した。

「私も民の幸せを願っている。だが、私にとって民とは私にしたがう者たちのことだ。武器をとって 抵抗ていこう するのであれば、民も兵も変わらぬ。私の進む道に立ちふさがるのなら、 排除はいじょ するだけのことだ」

　劉備は曹操をにらんだ。視線でその考えを否定するかのように。

「では、どうあっても兵を 収おさ めていただけないのですね」

　曹操は 余裕よゆう たっぷりにうなずいた。

「 残念ざんねん ながらな。 貴公きこう と話ができて楽しかった。次は戦場でお会いしよう」

　軽くあしらわれたかたちとなった劉備は、 肩かた をふるわせながら退出した。 決けっ して 危害きがい をくわえないよう、曹操が配下の者に命じる。

　郭嘉が曹操にたずねた。

「曹操様は劉備たちを味方にしたいのではないか、と思っておりました」

「その気持ちはある」

　曹操は劉備との会話を思い起こしながら答えた。

「だが、今の時点で 誘さそ ってもよい返事は 得え られないだろう。いずれ機会があるはずだ」

　一方、劉備は 暗くら い表情で馬にゆられていた。交渉がうまくいかなかったせいもあるが、それ以上に落ちこんでいた。関羽が声をかける。

「戦って追い返すしかありませんな。翼徳は喜ぶでしょう」

「ああ、仕方ない。だが、曹操は恐ろしい男だ。董卓などよりずっと、漢王朝にとって害になるのではないか」

　劉備はつぶやいた。

　このときはまだ、曹操と郭嘉にしてやられたことに気づいていなかった。



　　　　　４


　下邳の 城下じょうか は、曹操軍の旗でうめつくされていた。夏侯惇ひきいる先鋒部隊が、城を攻めたてている。

　陶謙は城壁の上に立って、みずから 防衛ぼうえい 戦せん の指揮をとっていた。 弓兵ゆみへい をならべ、 一斉いっせい 射撃しゃげき で敵を遠ざけている。

「しかし、こんなに早く敵がくるとは思わなかったな」

　陶謙のつぶやきに、糜竺が答える。

「申し訳ございません。私も予測できませんでした」

　曹操は本隊をあえてゆっくりすすめて陶謙を 油断ゆだん させ、その間に、先鋒部隊が下邳を 急襲きゅうしゅう する 作戦さくせん をとっていた。

　先鋒部隊は前線基地の小沛を通らず、遠回りして徐州に攻めこんだ。抵抗しようとした街の人々を 皆みな 殺ごろ しにしたため、徐州の民はふるえあがり、次々と曹操軍に 降伏こうふく する。そして、 妨害ぼうがい がなくなったところで、 一直線いっちょくせん に下邳をめざし、おどろくべき 速はや さで城下に 達たっ したのだ。

　陶謙は幽州の公孫瓚に 援軍えんぐん をたのんでいたが、まだ返事はきていない。小沛には二万の兵がいたが、もはや 連絡れんらく はとれなかった。

「劉玄徳は 無事ぶじ だろうか」

「いまごろ、小沛の兵をひきいてこちらに向かっているでしょう。あと三日ももちこたえれば、きっと援軍がきます」

　 根拠こんきょ はなかったが、糜竺は 断言だんげん した。 咳せ きこみながら指揮をとる陶謙のすがたを見ていると、 希望きぼう をうしなうようなことは言えない。

　敵軍はおよそ三万。城内の 守備しゅび 兵へい も同じ数だけいる。糜竺は戦に明るくないが、兵力からすれば 充分じゅうぶん に守りきれるはずだ。

　しかし、城内には 兵糧ひょうろう のそなえはあったが、矢や 投石とうせき 用よう の石が不足していた。おまけに、敵はこれから本隊がやってくるのだ。援軍がこなければ、かなり 不利ふり な戦になるだろう。

　陶謙がまたはげしく 咳せき をした。体を 折お り 曲ま げて、苦しそうにあえぐ。

「陶謙様。ここは部下にまかせて、中に入りましょう。お体にさわります」

　糜竺は陶謙をささえ、城壁の階段をおりようとする。

　そのとき、背後で歓声があがった。

「味方だ。援軍がやってきたぞ！」

　おどろいてふりむくと、西からやってきた 軍勢ぐんぜい が、曹操軍におそいかかるのが見えた。劉と書かれた旗が遠くでひるがえっている。

「まさか、劉玄徳殿がもう来てくれたのか」

　糜竺はやつれた顔に 笑え みを浮かべた。これで、何とか戦えそうだ。

　実はこのとき、劉備と関羽はまだ到着していなかった。張飛が 単独たんどく で小沛の兵をひきいてきたのだ。

　曹操軍の先鋒部隊が徐州に攻め入ったと聞いて、留守番をしていた張飛はなやんだ。勝手に兵を動かしたら、劉備に 怒おこ られるかもしれない。

　だが、それもつかの間のこと。

「味方が攻められているというのに、この張飛様が 黙だま っているわけにはいかん」

　張飛は小沛の兵をすべて連れて下邳へ 急行きゅうこう した。それで、 危機きき に間に合ったのである。

「誰か、張飛様の相手になるやつはいないか！」

　張飛は 叫さけ びながら、敵陣に突っこんだ。

　曹操軍からは、夏侯惇の 副将ふくしょう をつとめる 于禁うきん が前に出てきた。 銀色ぎんいろ に光る 槍やり をしごいて、突きかかってくる。

　三合目に、張飛の蛇矛が于禁の 脇腹わきばら をとらえた。穂先がよろいにひっかかり、于禁は馬からひっこぬかれて 宙ちゅう づりになった。

「うわっ。おろせ、この 野郎やろう 」

　于禁があばれると、よろいのひもが切れて地に落ちた。 腰こし を打ってうめき声をあげる于禁を部下たちが助けに走る。

　それを 尻目しりめ に、張飛は 突撃とつげき していた。蛇矛をふるって敵兵をけちらし、曹操軍の 奥おく 深ふか くへ攻めいる。その後から、小沛の軍団が槍をならべてつづく。

「大将はどこだ !? 」

　張飛はわめくが、夏侯惇は姿を見せない。張飛は 軍旗ぐんき がたくさん立っているところを本陣と見定め、そこをめざして馬をすすめる。

「おらおら、張飛様のお通りだぞ！」

　 調子ちょうし よく攻めこんでいた張飛だが、ふと 妙みょう なことに気づいた。いつまで立っても旗に近づけず、前方には敵がたくさんいる。それどころか、四方八方どこを見回しても、敵だらけなのである。

　張飛の部隊は、いつのまにかすっかり 包囲ほうい されていた。

　曹操軍は小沛の部隊が助けにくることを予想しており、あらかじめ作戦をたてていたのだ。張飛が突撃してきたら、陣形を 変化へんか させて 包囲ほうい 網もう に誘いこむてはずだった。

　作戦を考えたのは郭嘉だが、 完璧かんぺき に 実行じっこう した夏侯惇の 手腕しゅわん も 見事みごと だ。

「敵は 袋ふくろ のねずみだ。 一気いっき におしつぶせ！」

　夏侯惇の 激励げきれい にこたえて、曹操軍が外から攻めたてる。

　張飛は血に 染そ まった蛇矛をふりまわして戦ったが、 単独たんどく では状況を変えられない。包囲された兵は、一人また一人と倒れていく。

　その様子は、城壁の上からもよく見えていた。陶謙も糜竺も、助けに行かなければならないことはわかっている。だが、城門を開けて兵を出せば、敵が城内に 侵入しんにゅう してくるかもしれない。そう思うと、 判断はんだん がつかない。

「見殺しにするわけにはいきません。兵を出しましょう」

　糜竺が言って、陶謙がうなずいたときである。

　 北方ほっぽう から 一筋ひとすじ の光が走った。

　 白馬はくば ぞろいの 騎兵きへい 隊たい があらわれ、曹操軍におそいかかったのだ。白馬部隊は、公孫瓚軍の 主力しゅりょく として 名高なだか い。

「おお、公孫瓚殿からの援軍か」

　陶謙はほっと息をついて、その拍子にまた咳きこんだ。

「我こそは 常山じょうざん の 趙雲ちょううん なり。いざ、 参まい る！」

　 白銀はくぎん のよろいに身をかためた若者が叫んだ。これが公孫瓚配下の 勇将ゆうしょう で、常山出身の趙雲である。

　趙雲は夏侯惇めがけて一直線に馬を 馳は せ、槍で突きかかった。

「む !? 」

　夏侯惇は顔をのけぞらして槍をかわした。偃月刀をふるって、走り去る趙雲の背中をねらう。趙雲は体を低くして、偃月刀に 空くう を切らせた。 弧こ を 描えが いて駆けもどり、ふたたび槍を突きだす。

　 神速しんそく の突きは、寸前で偃月刀にはばまれた。お返しとばかりに、夏侯惇が手首をひねると、偃月刀が生き物のように動いて、趙雲の 頭上ずじょう をおそう。趙雲は馬ごと体をかたむけ、これをかわした。

　趙雲の 速度そくど と夏侯惇の 技術ぎじゅつ 、どちらも 群ぐん を抜いている。 互角ごかく の戦いがつづくうちに、白い騎兵隊は外から包囲網を突きやぶった。

「ありがたい。天の助けか」

　張飛は生き残っている兵を集め、この穴から 脱出だっしゅつ を 果は たした。趙雲と夏侯惇が戦っているのを見て、 加勢かせい しようと馬を寄せる。

「今日はここまでだな」

　夏侯惇に二人を相手にする気はなかった。兵をまとめて 素早すばや く 退却たいきゃく させる。趙雲に 追撃ついげき する気はなく、張飛には気持ちはあっても 余力よりょく がなかった。

　張飛と趙雲は下邳の城内に入り、曹操軍は城からはなれて陣をしきなおした。

「みな、よく戦ってくれた」

　夏侯惇は 負傷ふしょう した于禁をなぐさめ、配下の兵士たちの 頑張がんば りをほめたたえた。ここで勝負を決める必要はない。曹操の本隊が着くのを待ってから、あらためて攻撃すればよいのである。

　張飛と趙雲の活躍で、下邳は 落城らくじょう をまぬがれたが、それも一時のことと思われた。



　　　　　５

　夏侯淵は舌打ちし、弓を手にとったが、どうやら矢が切れていたらしい。陳宮はずっと身をちぢめていたが、矢が飛んでくることはなかった。

　陳宮と張遼は何とか追っ手を振り切った。濮陽で呂布と合流する。

　呂布は敗戦に腹を立て、 雪辱せつじょく に燃えていたが、兵力の半分近くを失っている。しばらくは大人しくしているほかなかった。



　　　　　２


　人々のなげきが 雲くも となって、雨を 降ふ らせた。 霧雨きりさめ の 底そこ に、 哀かな しみにつつまれた下邳の城があった。

　徐州の 主あるじ である 陶謙とうけん が死んだのだ。 小沛しょうはい に 急使きゅうし が送られて、 劉備りゅうび が駆けつけた。

「治める者がいなくては、徐州は 賊徒ぞくと に 荒あ らされてしまいます。民の平和と安全のためにも、どうか陶謙様の後をついでください」

　 糜竺びじく をはじめとする数十人の 臣しん たちが、涙もかわかぬまま 要請ようせい する。しかし、劉備はなおもためらっていた。

「朝廷の 許可きょか も得ず、勝手に後をついで独立するなど、賊と変わらぬではないか」

「いつまでぐずぐずされているのですか」

　 関羽かんう がしびれをきらして迫る。

「これだけの者が兄者に助けをもとめているのです。その 期待きたい にこたえることこそ、 義ぎ の道でありましょう」

「そうそう、まさか 怖お じ 気け づいているわけじゃないよな」

　劉備はそれでも迷っていたが、ようやく承知した。

「民のためだ。徐州を一時あずかることとしよう」

　誰も反対する者はいなかった。劉備は徐州の支配者となり、政治をおこなうことになった。最初の課題は、曹操と呂布の勢力を警戒して、守りをかためることである。


　曹操は呂布を濮陽の城に追いつめていた。攻める準備はできているが、 強引ごういん に突っこめばどれだけの 犠牲ぎせい がでるかわからない。城の 四方しほう をかためて様子を見ている。

　そこへ劉備が陶謙の後をついだという情報が届いた。

「私が苦労して手に入れられなかった徐州を、劉備は 一滴いってき の血も流さずに得たか。呂布より先に、あの男を倒すべきかもしれんな」

　曹操が言うと、荀彧が反対した。

「陶謙が死んだばかりで 喪中もちゅう の軍を攻めるのも大人げありません。それより、今は足もとをかためるべきです。 予州よしゅう には強大な勢力がいませんから、まずはここを攻めとり、土地と兵と 食糧しょくりょう を 確保かくほ しましょう」

　曹操はその意見を 採用さいよう した。予州 一帯いったい は都に近くて 交通こうつう の 便べん がよく、 田畑でんぱた も 豊富ほうふ だ。今は 黄巾賊こうきんぞく の 残党ざんとう が 暴あば れまわっていて荒れ果てているが、 復興ふっこう させれば、 生産力せいさんりょく の高い土地になるだろう。

　曹操は夏侯惇と曹仁に呂布を警戒させ、残りの武将をひきいて、周囲を 攻略こうりゃく してまわった。

　黄巾賊の残党など、もはや曹操の敵ではない。曹操軍は 連勝れんしょう をつづけたが、あるとりでを攻めたときは勝手がちがった。とりでといっても、村人が山にこもって 自衛じえい しているだけなのだが、その 指導者しどうしゃ が妙に強いという。

　こてんぱんにやられた 曹洪そうこう から 報告ほうこく を聞いて、曹操は 興味きょうみ をいだいた。 典韋てんい とわずかな護衛の兵をひきいて、とりでにおもむく。

　村の指導者は典韋とそっくりの 体型たいけい をした巨漢であった。曹操は典韋を一人で送り出し、 力ちから 比くら べをさせた。

　 遠目とおめ に 眺なが めていると、思ったとおり二人は互角で、とっくみあったまま動かない。と、どちらかが 足あし 払ばら いをかけて二人は 転ころ がり、上になったり下になったりしながら 殴なぐ り合いをはじめた。

「もういいだろう」

　曹操が 鉦かね を 鳴な らして退却させると、二人は連れだって戻ってきた。典韋が言うには、戦っている 最中さいちゅう に 意気いき 投合とうごう したから、曹操のもとで働かせたいという。指導者の男は 許褚きょちょ と名乗った。

「村のみんなを大事にしてくれるなら、おれは曹操様に仕えたいと思う」

　曹操が期待したとおりの 展開てんかい である。もちろん 拒否きょひ する理由などない。

「おまえたち二人なら、呂布にも勝てるかもしれないな」

　曹操は村人に新しい 農地のうち をあたえ、許褚を配下の武将にくわえた。

　こうして、兗州と予州をほぼかためた曹操は、あらためて濮陽攻略にとりかかった。濮陽は四方に 濠ほり をめぐらせた 堅城けんじょう である。城内の兵は一万ほどしかいないと思われ、兵糧もとぼしくなっているはずだが、相手が呂布だけに 油断ゆだん はできない。

　策を 練ね っていると、向こうから動きがあった。

　濮陽に住む商人たちが、呂布の支配を嫌って、曹操に助けをもとめてきたのだ。

「お気をつけください。わなの可能性があります」

　郭嘉が進言したが、曹操の心はすでに決まっていた。

「たとえ見えすいたわなであっても、私を頼ってくる者は受け入れる。見捨ててしまえば、次から私を頼る者はいなくなるからだ。ただ、危険にそなえて、 最大限さいだいげん の 配慮はいりょ はしよう」

　商人たちは、夜になったら、合図とともに城門をひらくという。

　曹操は典韋と許褚をつれてみずから 突入とつにゅう すると 定さだ めた。夏侯惇と夏侯淵の部隊を左右において、いつでも救援に行けるよう 待機たいき させる。

　その夜、城壁の上に、白地に義と書かれた旗がひるがえった。内部からの合図である。

　同時に、城門がひらき、 跳は ね 橋ばし がおろされた。何事かと、 警備けいび の兵がさわぎだす。

「突入せよ！」

　曹操は剣をかざして命じた。

　曹操軍が吊り橋をわたって、城内へ 殺到さっとう する。 真ま っ 暗くら な 街路がいろ には守備兵はいない。兵士たちは喜んだが、曹操は逆に嫌な予感をおぼえた。

「あせるな。各隊の 長ちょう は 慎重しんちょう に左右を確認しろ」

　そのときである。

　辺りが急に明るくなった。赤い光が、 薄闇うすやみ を左右に走る。ぱちぱちという音がひびき、 熱風ねっぷう が 押お し寄せた。 火矢ひや だ。

「 裏うら 切ぎ り 者もの は 始末しまつ したぞ。みなの者、かかれ！」

　呂布が義の旗に火をつけ、曹操軍めがけて放り投げる。

　商人たちの裏切りは事前にもれていたのだ。待ちかまえていた呂布軍は、左右から曹操軍におそいかかった。燃えさかる火の海に曹操軍を追いつめ、槍や矛を突き刺す。

　方天画戟が一つ弧を描くと、三つの首がはねとんだ。炎を背景に画戟をふるう呂布は、死へいざなう 鬼神きじん であった。

　曹操軍は呂布を見ると戦う気持ちをうしない、悲鳴をあげて逃げまどう。だが、 踊おど り 狂くる う炎が行く手をはばみ、城門へたどりつけない。

　城外からは救援部隊が突入をこころみたが、 寸前すんぜん で跳ね橋があげられてしまった。城門も 閉と じられようとしている。そこへ、許褚が 鉄鞭てつべん をふりまわして突進した。鉄鞭は 節ふし のついた重い鉄の棒だ。許褚はこの 兇器きょうき で守備兵を打ちすえ、門を 確保かくほ する。

「許褚よ、曹操様を頼む」

　言いおいて、典韋は城壁への 階段かいだん を駆けのぼった。守備兵を追いちらし、跳ね橋に近づく。そして、戟をふりかぶると、跳ね橋をまきあげる 鎖くさり を一撃で 断た ち切った。

　がくんと音がして、勢いよく跳ね橋がおりる。典韋は 助走じょそう をつけて城壁を飛びこえ、跳ね橋の 端はし に乗った。そのまま、橋と 一緒いっしょ に濠をこえて、向こう側へ 降お りたつ。

「曹操様、こっちだ」

　曹操は炎にまかれながら、城を飛びだした。必死で馬を走らせて、典韋が手を振る方へ駆ける。その後に許褚がつづき、呂布と張遼が追いすがる。

　曹操が駆け抜けると、夏侯惇が槍隊をならべて、呂布の行く手をはばんだ。曹操は何とか逃げ切ったが、多くの兵を失って、包囲をつづけられなくなってしまった。



　　　　　３


　曹操は手足に 火傷やけど を 負お っていたが、 症状しょうじょう は重くなかった。

「一度や二度の負けを 悔く やむ必要はない。 最終的さいしゅうてき に勝てばいいのだ」

　 医師いし に 包帯ほうたい をまいてもらいながら、曹操は自分に言い聞かせるようにつぶやいていた。

「曹操様、お耳に入れたいことが」

　郭嘉の報告を聞いた曹操は、軽く目をみはった。曹操は火傷の 具合ぐあい が重く、生死の境をさまよっているといううわさが流れているらしい。

「それは使えるな」

　曹操が言うと、郭嘉も微笑してうなずいた。

「私も同じことを考えました」

　二人は声をひそめて、作戦をこまかく 検討けんとう した。

　夜が明けると、濮陽の城に 急報きゅうほう が届けられた。

「曹操が 怪我けが をして死にかけているそうです。この 機き を 逃のが さずに攻めれば、曹操軍はばらばらになるでしょう」

　呂布が勢いよく立ちあがる。

「昨日の戦いは 中途ちゅうと 半端はんぱ で終わったからな。今度こそ、曹操の息の 根ね をとめてやる」

　軍師の陳宮があわててひきとめた。

「危険です。これはわなにちがいありません」

　呂布は 口角こうかく をあげて 不敵ふてき に笑った。

「そんなことはおれでもわかる。だが、城にこもるのはもう 飽あ き飽きだ。わなもそれと知っていれば 怖こわ くない。食い破ってくれるわ」

「私も出陣させてください」

　張遼が 志願しがん すると、呂布は当然だとばかりにうなずいた。

「おまえも少しはできるようになったからな。曹操軍の武将なら、誰と戦っても負けぬであろう」

「ありがたきお言葉。この張遼、 必かなら ずや期待にこたえてみせましょう」

　陳宮は留守番を命じられた。呂布と張遼は、兵をかき集めて出陣する。

　曹操軍が退却した後を追っていくと、こんもりした林に行きついた。 丈たけ の 低ひく い茂みがあって、いかにも伏兵が 隠かく れていそうだ。

「あやしいな」

「すぐに調べさせましょう」

　張遼が偵察隊を送ったところ、茂みの奥に旗が見え隠れしており、ときおりざわざわと音がするという。

「ふっ。そんな見えすいたわなにひっかかるとでも思ったか」

「迂回いたしますか」

　張遼がたずねると、呂布は首をふった。

「いや、後ろでうろちょろされると 面倒めんどう だ。伏兵を 始末しまつ してしまおう」

　呂布は全軍に林への突撃を命じた。

　呂布が駆る 赤兎馬せきとば を先頭に、一万あまりの軍勢が林に攻めかかる。

「むむっ？」

　やぶの中に馬を乗り入れた呂布は、 異変いへん に気づいて眉をひそめた。敵の 気配けはい がないのだ。方天画戟をふりまわしても、 枝えだ が落ちてくるだけで、人を斬る手応えはない。

　周囲では怒りととまどいの声があがっている。木の間に 綱あみ がめぐらせてあったり、足もとの草が 結むす んであったりして、移動が 妨害ぼうがい されているのだ。

「いかん。戻れ！」

　呂布は叫んだが、わずかに遅かった。

　四方からときの声があがったと思うと、次々と矢が 射い かけられたのである。 枝葉えだは のすきまから、矢の雨が降りそそぎ、呂布軍をおそう。

　曹操軍は呂布軍を林に閉じこめて攻めたてていた。伏兵と見せかけて動きにくい林に誘いこむ作戦だったのだ。

　上を見て矢を 避さ ければ、下のわなにかかる。林から飛び出せば、待ちかまえていた敵兵に槍で突き殺される。落とし穴に落ちる者もいれば、味方同士でぶつかる者もいる。呂布軍はすっかり混乱し、悲鳴をあげて逃げまどった。

「 卑怯ひきょう な手ばかり使いおって！」

　呂布がうなりながら林から出てきた。 黄金おうごん のよろいには矢が何本も突き立っている。まるで手負いの 猛獣もうじゅう のようだ。

　そのすがたを見て曹操軍の兵が逃げちるなか、二人の巨漢が立ち向かった。典韋と許褚である。二人は呂布を前後にはさんで、 戟げき と 鉄鞭てつべん をふるう。

　呂布は方天画戟をかかげて戟を受けとめ、体をひねって鉄鞭をよけた。赤兎馬が主人の意図をくんで左に 跳は ね、 挟はさ み 撃う ちを 回避かいひ する。

　方天画戟が 鋭角えいかく 的てき な 軌跡きせき を描いて、典韋をおそった。かろうじてかわした典韋があやうく落馬しかかるところを、許褚が割りこんで救う。画戟が鉄鞭にぶつかるたびに激しく火花が散って、許褚はだんだんと押されていった。

　そこへ、今度は夏侯惇が加勢した。三人で同時に斬りかかると、さすがの呂布も 額ひたい に汗の玉を浮かべた。

「おれはまだ、やられるわけにはいかぬのだ」

　呂布が画戟を返して逃げだす。夏侯惇が追いすがって偃月刀をふりかぶる。

　と、矢が 飛来ひらい して、夏侯惇の気をそらした。夏侯惇は迫りくる矢をたたき落としたが、その隙に呂布は遠くはなれていた。

「ご無事でようございました」

　弓矢で退却を 援護えんご したのは張遼である。

「ふん、よけいなことをしやがって」

　呂布は強がったが、表情には疲労の色が 濃こ い。

「濮陽に戻って復讐戦を 挑いど むぞ」

　呂布はなお戦意を失っていなかったが、この敗戦によって、一万の手勢はほとんどが戦死するか降伏するかしていた。濮陽を 保たも つことはできず、呂布は陳宮や張遼といった腹心と貂蝉を連れて、東へおちのびたのだった。



　　　　　４


　兗州から追い出された呂布は、 冀州きしゅう の袁紹を頼るつもりであった。以前、助けてくれたから、もう一度受け入れてくれると思ったのだ。しかし、陳宮が調べたところ、袁紹は曹操からの知らせを受けて、呂布をとらえようと待ち受けているという。

「では、どこへ向かえばいいのだ。またそのあたりの城をうばおうか」

「徐州を劉備がついだそうです。彼はお 人ひと 好よ しとのうわさですから、きっと助けてくれるでしょう」

　陳宮が 提案ていあん すると、呂布は 素直すなお にしたがった。

「劉備という男はよく知らないが、この 際さい だから誰でもかまわない。とにかく、貂蝉を安全なところに連れて行かないとな」

「承知しました。 下邳かひ に使者を送って、迎えに来させましょう」

　呂布が落ち 延の びてきたという情報はすぐに下邳に伝えられ、劉備たちをおどろかせた。

「何と、あの呂布が私を頼ってくるというのか」

「今まで何人もの主君を裏切ってきた男です。追い返したほうがよろしいかと」

「そうだ。あいつだけは信用しちゃいけねえ」

　関羽と張飛が 真剣しんけん な表情で進言したが、劉備は聞き入れない。

「あのとき、呂布が兗州を攻めなければ、徐州は曹操の手に落ちていた。私たちは彼に恩を返さねばならない」

　糜竺も劉備に反論する。

「お言葉ですが、呂布が兗州を攻めたのは自分の 利益りえき を考えてのことです。私たちを助けてくれたわけではありません」

「それはわかっている。だが、 一国いっこく をあずかる者として、困っている者には手を差しのべるべきだ」

　劉備をのぞく三人は顔を見合わせてため息をついた。劉備は意外と 頑固がんこ で、とくに人助けとなると、周りの言うことをまったく聞かない。

　こうして、劉備と呂布は下邳で 対面たいめん することになったのである。

　まず、陳宮が 口上こうじょう を述べた。

「このたびは会見の機会をくださいまして、ありがとうございます。呂布様は董卓を倒す功をあげながら都を追われ、さらには袁紹や曹操にも憎まれて、よるべのない身となりました。もし受け入れてくださるのであれば、われら一同、劉備様のために力をふるうつもりでおります。何なりとお申しつけください」

　劉備が答える。

「 当代とうだい 一いち の勇者を迎えることができて、うれしく思います。皇帝陛下が 悪臣あくしん の手から救われれば、きっと呂布殿にもお声がかかるでしょう。それまではぜひ、客人として滞在していただきたい」

　呂布は劉備を見て、ふと首をかしげた。どこか 見み 覚おぼ えがある。劉備の後ろに立つ関羽と張飛に目をうつすと、 記憶きおく がよみがえってきた。虎牢関で戦った者たちではないか。

　だが、呂布はおとなしくしていた。呂布のほうには何もわだかまりはないし、陳宮に黙っているよう言われていたからである。

　わだかまりがあるのは張飛で、大きな目をむいて呂布をずっとにらみつけていた。

　その夜は歓迎の宴がひらかれた。

　劉備は 普段ふだん ぜいたくを 好この まないが、大事な客を迎えるとあって、 山海さんかい の 珍味ちんみ をふるまった。 鯉こい の 蒸む し 物もの や 豚ぶた の 丸まる 焼や きをならべて、呂布をもてなす。

「おいこら、その 無礼ぶれい な口の 利き き方は何だ。兄貴は皇帝陛下の一族なんだぞ。しかも、曹操に負けて行くところもないおまえたちを助けてやったんだ。 土下座どげざ して礼を言うべきだろう」

　劉備がたしなめたが、すでに酔っ払っている張飛はとまらない。

「表へ出ろ。おれが力ずくで礼儀を教えてやる」

　そこまで言われては、呂布も黙ってはいない。すっと立ちあがって、張飛を見下ろした。

「できるものならやってみるがよい。今度は何人でかかってくるのだ？」

「あのときは腹が痛かったんだ。今なら一人で充分だ」

　張飛は強がったが、酒のせいで足もとがふらついている。

「いい 加減かげん にしないか」

　関羽が割って入って、張飛を押さえつけた。反対側では、張遼が呂布をなだめてすわらせる。

　宴会は気まずい雰囲気のまま、おひらきとなった。

　劉備は張飛を 叱しか りつけたが、また同じことが起きないとはかぎらない。呂布を下邳にとどめるのは危険だ。

　そこで劉備は、かつて自分たちがそうしていたように、小沛に住んで徐州を守ってもらうことにした。呂布は 快こころよ く 承諾しょうだく し、小沛にむけて旅だった。

　徐州にはいまだ、 嵐あらし の雰囲気がただよっているようであった。

　呂布を破った曹操は、兗州から予州にかけての 地域ちいき をほぼ 手中しゅちゅう におさめた。そこで、交通の便のよい予州の 許昌きょしょう に移動して、本拠地と定めた。

　どの諸侯も兵糧の確保に頭を 悩なやま ませていた。それは十万を 超こ える軍勢をそろえる曹操も同様である。

　この政策によって、曹操の 領内りょうない では、生産力が一気に高まることになる。

　国の 基礎きそ をかためた曹操は、皇帝を救出する計画に 着手ちゃくしゅ した。天下に 号令ごうれい するには、皇帝の名前を借りるのが手っ取り早い。

　そのころ、ときの皇帝である献帝は、長安の宮殿に閉じこめられていた。董卓の旧部下たちは、献帝と皇后に肉抜きの 粥かゆ と野菜くずの食事しかあたえず、 囚人しゅうじん のようにあつかっていた。

　献帝のまわりにも 忠義ちゅうぎ の人物はいる。彼らが献帝を救いだし、もとの都の 洛陽らくよう に連れてきた。

　しかし、洛陽は董卓に焼きはらわれて、すっかり荒れ野原になっている。献帝たちは 廃屋はいおく を 改造かいぞう して 仮かり の宮殿としたが、ろくに食べるものもない。しかも、長安から追っ手が出ている。

　困った献帝は、近くで名声を高めていた曹操を頼ることにした。

　曹操はちょうど救出部隊を送ろうとしていたところである。 渡わた りに 船ふね とばかりに、夏侯惇と曹洪が指揮する部隊を洛陽に行かせた。

　夏侯惇は長安からの追っ手を軽くひねって、献帝の安全を確保した。そうしておいて、曹操を呼ぶ。

　曹操は大軍をひきいて洛陽におもむき、献帝と対面した。皇帝に対する正式な礼をして、 言上ごんじょう する。

「私は漢王朝の恩を受けて久しく、常にご恩返しをしたいと思っておりました。今、その機会を与えられまして、これ以上の喜びはありません。私の指揮下にある兵は百万を数えますが、これは陛下のために働く正義の軍でございます。いかなる 逆賊ぎゃくぞく も打ち負かしてご覧に入れましょう」

「実にたのもしい。そなたのような 忠臣ちゅうしん に出会えて、 苦労くろう がむくわれた気分だ」

　仮の 玉座ぎょくざ にすわった献帝は 感激かんげき の 面おも 持も ちで、曹操を見下ろしている。

「これからは何の心配もいりません。 国家こっか のことはすべて私が引き受けますので、どうぞ心 穏おだや やかにおくつろぎください」

　曹操の細い目がぎらりと光った。

　献帝はふと、落ちつかない気持ちになった。曹操の軍勢に宮殿をかこまれていることが、あらためて思い起こされた。

「う、うむ。よろしく頼む」

　献帝としては、そう言うしかない。

　曹操はまず、献帝を洛陽から許昌に移動させた。許昌に新しい宮殿を 造つく って、献帝を 住す まわせる。

　董卓によって皇帝にされてからはじめて、献帝はやわらかな寝台で休み、腹一杯食事をとることができるようになった。身の回りの世話をする 宦官かんがん や 女官にょかん も大勢あてがわれた。曹操の保護の下で、献帝はぜいたくな生活を手に入れたのである。

　だが、曹操は政治的な権利は何も与えなかった。皇帝の名前で出される命令はすべて、曹操の手によるものである。献帝の 意思いし が 反映はんえい されることはなかった。

　献帝は曹操のあやつり 人形にんぎょう になった。しかし、それがわかっていても、漢王朝に 忠誠ちゅうせい をちかう者は、皇帝の命令を 無視むし できない。

　漢の皇帝を手中にしたことで、曹操の権力は 飛躍ひやく 的てき に高まった。

　 建安けんあん 元年がんねん （西暦一九六年）の出来事である。

　　　　　１


　 曹操そうそう の領土は、中国の中央部に 帯おび のように横に 伸の びている。その東の先にあるのが、 劉備りゅうび の治める 徐州じょしゅう である。

　まず、劉備に使者を送り、徐州攻撃の件を 謝罪しゃざい して、友好的な関係をむすんだ。そして、劉備を正式に徐州の長官に 任命にんめい する。皇帝の名が入った任命状をもらって、劉備は 非常ひじょう に喜んだ。

「陛下の背後には曹操がいます。これはおそらく、曹操の意思でしょう」

「しかし、陛下のご命令ということであれば、したがわざるをえないでしょう」

「ちょうどいい。おれが殺しに行ってやろう。命令があるんだったら、兄貴が 非難ひなん されることもないだろう」

　関羽と 張飛ちょうひ は、呂布をまったく信用していないのである。だが、劉備はちがった。

「いや、一度助けたのだから、きちんと 面倒めんどう をみるべきだ。呂布殿と話し合い、何とか許してもらう道を考えよう」

　劉備は呂布に代わって事情を説明し、決して朝廷にそむいてはいないという内容の手紙を書いた。

　曹操は劉備からの手紙を荀彧に見せた。

「どうやら計略は失敗に終わったようだ。やはり武力を使わねばならないか」

「 知恵ちえ の 泉いずみ はまだかれておりません。次は、 駆虎くこ 呑狼どんろう の計を用いましょう」

「それはどのような策なのだ」

「呂布をそそのかして、劉備を攻撃させます。勝ったほうを私たちが討てばよいでしょう。もっとも、劉備は漢王朝の忠臣ですから、 共存きょうぞん できるかもしれません」


　荀彧が計略を説明すると、曹操は薄く笑った。

　荀彧は袁術の住む寿春に 密偵みってい を送り、あるうわさを流させた。徐州の劉備が攻めてくるという内容である。

　同時に、曹操は勅命すなわち皇帝の命令として、劉備に袁術を攻めるよう伝えた。

　 指令しれい の手紙を読んだ劉備は、すぐに戦の準備をはじめる。

「これもまた曹操の勝手な命令に決まっています。お考え直しください」

　進言する糜竺に、劉備は困った顔を向けた。

「たとえそうであっても、 勅命ちょくめい （皇帝の命令）にはしたがわないといけない。でないと、私ばかりかおまえたちも朝廷の敵になってしまう。それに、袁術はもともと漢王朝をうやまう気持ちがなく、民に対しても冷たいという。たとえ曹操の命令であっても、 討伐とうばつ に 値あたい する人物だ」

「しかし、徐州の守りはいかがしましょうか。 下邳かひ を 空から にすれば呂布が......」

　糜竺はあっと叫んでひざをうった。

「そうです。いっそのこと、呂布に袁術を攻めさせればいいのではありませんか」

　だが、劉備は首をふった。

「呂布殿は私の部下ではなく、客人だ。そのようなことを命令するわけにはいかない」

　糜竺はしゅんとなって、うつむいた。

「でしたら、誰かに下邳を守っていただかねばなりません。私は戦のことはわかりませんから」

「では、それがしが 居い 残のこ ろう」

　関羽が言ったが、劉備はみとめなかった。

「 雲長うんちょう には私のそばにいて助言してもらわないと困る」

「じゃあおれだな」

　張飛が進みでた。普段なら留守番は嫌がるところだが、関羽も張飛も、呂布が攻めてくるにちがいないと思っている。それで志願したのだ。

　劉備は眉をひそめた。

「 翼徳よくとく に任せるのは、少し心配だ」

「大丈夫だって。この前もうまくやっただろう」

　張飛は胸をはったが、劉備はまだ迷っている。張飛の 気性きしょう からすると、こちらから呂布を攻めたりしかねない。とくに酔っ払ったら何をしでかすかわからなかった。

「では、私たちが帰ってくるまで酒を飲まないと 約束やくそく できるか」

　張飛はうっと言葉につまった。太い腕を組んで、しばらく考えこむ。

　劉備と関羽がなかばあきれながら見守っていると、張飛はおおげさにため息をついて言った。

「わかった。約束しよう」

　そこで、劉備と関羽は兵をひきいて、袁術討伐に向かった。

　袁術は荀彧が流したうわさを信じて警戒しており、 国境こっきょう 付近ふきん で両軍は激しくぶつかることになる。



　　　　　２


　劉備が袁術討伐に出かけたという情報は、すぐに 小沛しょうはい の呂布のもとにもたらされた。

「劉備という男は、本当にお人好しだな。これで徐州がおれのものになる」

　呂布が戦の準備を命じると、 張遼ちょうりょう が反対した。

「劉備殿はわれらの恩人です。その留守をおそうのは信義に反するのではありますまいか」

　その言葉を聞いた呂布は、浮かせかけた腰をおろして、ちらりと迷いを見せた。

「 陳宮ちんきゅう はどう思うか」

　陳宮の心はとうに決まっている。 念頭ねんとう にあるのは劉備ではなく、その背後にいる存在だ。

「劉備は曹操の手下になりはてました。もはや、頼りにはなりません。この機に徐州を手に入れ、ふたたび曹操と戦うべきだと思います」

「軍師殿！」

　張遼がきびしい視線を向けると、陳宮は目をそらした。二人のやりとりをながめていた呂布は、迷ったまま立ちあがった。

「どうすべきかは、後で指示する」

　呂布が向かったのは 寝室しんしつ である。

「 貂蝉ちょうせん よ、徐州がほしいか」

　美しい妻にむかって、呂布はぶっきらぼうにたずねた。

「私は呂布さまの 隣となり にいられれば、それで幸せなのです。他には何もいりません」

　貂蝉はそう言って、長いまつげを伏せた。呂布の帰りを待つ日々に、貂蝉は疲れていた。 質素しっそ でもいいから、落ちついて 暮く らしたいと思っている。

「おれはおまえに、もっといい生活をさせてやりたいのだ。こんな 田舎いなか にいるなら、まだ 董卓とうたく の部下でいたほうがましだった。おれはもう一度、おまえを都に連れて行きたい」

「呂布さま、私は今のままで充分です。充分ですから」

　呂布は貂蝉の言葉を最後まで聞かず、ただ抱きしめた。

「まずは徐州を手に入れる。その後、曹操に勝って、都にもどるんだ」

　すれちがいに、呂布は気づかない。たくましい腕のなかで、貂蝉はひっそりと涙するのであった。

「こんなことなら、兄貴についていけばよかった。とりあえず袁術が相手でも我慢したのになあ」

「うーん、お 祝いわ いの席だからな。おれだけ飲まないというのも失礼だろう。一杯くらいなら、兄貴も許してくれるさ」

「後で行くから、守りをかためておけ」

　張飛は命じたが、指揮官がいなくては呂布と戦えるわけもない。あっというまに城門を突破され、城内へ 侵入しんにゅう されてしまった。

　糜竺に言われて、張飛はふらふらと後につづいた。馬に乗って敵兵のただ中を通り抜け、城外に出てから気づいた。

「失敗してしまった。兄貴に何て言えばいいんだ」

　張飛は糜竺の言葉にしたがって、のろのろと動きだした。徐州を追われた劉備軍は、まるで 葬式そうしき の列のように肩を落として、南へ向かった。

　そこへ、うなだれた張飛がやってきた。報告を聞いて、劉備は目をおおった。

「まさか、呂布が裏切るとは......」

　 絶句ぜっく した劉備の前に、張飛は泣きながらひざまずいた。

「全部おれが悪いんだ。兄貴たちの言いつけを守らなかったから。こうなったら、死んでお 詫わ びするしかない」

　張飛は短剣を抜いて自分の首に突き刺そうとする。劉備はあわてて引きとめた。

「早まるな。呂布を信じた私も悪かった。失敗は成功で取りかえせばよい」

　張飛は涙をふいて立ちあがった。

「じゃあ、これから徐州に引き返して、呂布をやっつけよう」

　あまりに早い変わり身にあきれて、劉備は関羽と顔を見合わせた。

「そんな簡単にはいかない。兵力は少ないし、兵糧はこころもとない。おまけに、帰る家がなくなったとあっては、兵が 動揺どうよう するだろう」

「それより、この状態で、呂布に背後をおそわれたら、われわれは 挟はさ み撃ちにあってしまいます」

「 退ひ かねばならぬか」

　劉備たちは話し合った結果、許昌へ向かうことに決めた。

　劉備は漢王朝の臣であり、徐州をうばわれた罪を皇帝に詫びなければならない。曹操を頼ると思うと気持ちは 複雑ふくざつ だが、他に行く当てはなかった。



　　　　　３


　荀彧が提案した 駆虎くこ 呑狼どんろう の計は、見事に成功した。劉備と呂布の間を引き 裂さ くことができたのである。あとは、劉備を使って呂布を倒せばよい。劉備は呂布よりもはるかに 扱あつか いやすいはずだ。

　曹操は荀彧にたずねた。

「劉備ははたして、私の思いどおりに動いてくれるだろうか」

「何をさせるかによるでしょう。使いようによっては役立つと思います」

　 楽観らっかん 的てき な荀彧に対して、 郭嘉かくか が反論を述べた。

「私は不安です。はっきりした 根拠こんきょ はありませんが、彼を生かしておくと、将来のわざわいとなるような気がします」

　曹操はしばらく二人に 議論ぎろん させていたが、やがて心を決めた。

「今はあせらず、ゆっくり手なずけることにしよう。劉備が 駄目だめ でも、関羽は配下にしたいものだ」

　劉備が許昌に着くと、曹操はみずから門まで出迎えた。劉備がまず罪をあやまる。

「私は徐州の長官に任命されながら、その役割を果たすことができませんでした。皇帝陛下には何とお詫びしてよいやらわかりません」

　曹操は微笑をたたえて、劉備をなぐさめた。

「勝敗は時の 運うん だ。陛下も気になさらないだろう。まずはゆっくりと休んで疲れをいやし、その後で呂布を倒す計画をたてようではないか」

「できれば、陛下にお目にかかって、直接お詫び申しあげたいと思います」

「うむ。では、すぐに機会をつくろう」

　曹操は先に立って歩き、劉備を真新しい宮殿に案内した。

　よろしいのですか、と郭嘉が 小声こごえ でたずねる。曹操は献帝をめったに 人前ひとまえ に出さない。親しくなって、反乱の相談などされると困るからだ。

　しかし、曹操は劉備に聞こえるようにわざと大きな声でこたえた。

「何の 不都合ふつごう があろうか。 劉りゅう 玄徳げんとく 殿は皇族の一員であられる。陛下に 危害きがい をくわえることなどあろうはずがない」

　劉備はそれを聞いて、信用されていると感じた。

　許昌の宮殿は長安や洛陽には遠く及ばないが、 風格ふうかく のある 建築けんちく で、皇帝の 在所ざいしょ にふさわしい。劉備は、いちいち感心しながら奥へすすんだ。

　そして、夢にまでみた皇帝との対面である。

　献帝はまだ十代の少年であったが、劉備の目には 神々こうごう しくさえ映った。

「一度の敗戦を詫びる必要はない」

　献帝は静かに言った。感情のこもらぬ声だ。

「今後も曹操を助けて、漢王朝のためにつくしてくれ」

　劉備は口の中がからからにかわいて、うまく話すことができなかった。ただ、 平伏へいふく するのみである。

　見かねて、曹操が言い 添そ えた。

「劉玄徳殿は陛下の遠いご 親戚しんせき にあたるそうです」

「そうか！」

　はじめて献帝の声が明るくなった。

「後で 系図けいず を調べてみよう」

　 謁見えっけん はそれで終わりだったが、劉備は 感激かんげき した。

　系図を調べた結果、劉備の 先祖せんぞ は三百五十年ほど前の皇帝の息子だとわかった。百人以上の子どもをもうけた人で、その遠い子孫が劉備である。献帝は 感心かんしん し、劉備を 叔父おじ と呼んで 親した しむようになった。

　曹操は呂布討伐の準備をする一方で、劉備を連れて市場や農村をまわった。その 活気かっき をみれば、曹操が 内政ないせい に力をいれていることがわかる。

「私は曹操殿のことを 誤解ごかい していたのかもしれない。彼は民のことを考えて政治をおこなっている」

　劉備は関羽と張飛に、そう話すようになった。

「兄者がそう言うなら、それがしはしたがいます」

「呂布を倒すためなら、曹操の力を借りてもいいな」

　それが二人の 反応はんのう である。

　ともあれ、呂布との対決の日は 刻一刻こくいっこく と近づいていた。



　　　　　４


　しのつく雨が降りつづいていた。

　城の周りの 平野へいや は 泥沼どろぬま と 化か し、小さな 水路すいろ が 網あみ の目のように 延の びている。 雨音あまおと は耳にきざみこまれ、体の一部となったように思える。

　日に日に、寒さがこたえるようになっていた。 秋あき の 長雨ながあめ はもう、十日もつづいている。

　水の 幕まく に 包つつ まれたように見える城は、下邳であった。呂布軍が城にこもって、すでに三ヶ月が 経た っている。

　曹操は 万全ばんぜん の準備をととのえ、十万を 超こ える大軍を 動員どういん して、この戦にのぞんでいた。 確実かくじつ に呂布の 息いき の 根ね をとめるという覚悟があらわれている。

　 陣容じんよう も 豪華ごうか だった。夏侯惇、曹洪らの武将と郭嘉、荀彧といった軍師が 参戦さんせん しており、劉備たちも一部隊を任されている。

　曹操軍は三方からいっせいに徐州に攻め入り、あっというまにいくつかの城を落として、下邳の 城下じょうか に 達たっ した。徐州の民は抵抗しようとしなかった。彼らは曹操を 憎にく み、 恐おそ れていたが、劉備がいることで曹操軍を歓迎したのである。

　また、曹操軍は 整然せいぜん と列を組んですすみ、田畑にはいっさい立ち入らなかったし、民に 危害きがい を加えることもなかった。曹操がきびしい 規則きそく をつくって、それを守らせていたのである。

　行軍の途中、曹操の馬が鳥におどろいて跳ね、畑の 作物さくもつ を踏んでしまったことがあった。畑に立ち入るのは、むち打ちの 刑けい に 値あたい する。どんな事情があっても、また誰であっても、規則は守らねばならない。曹操は自分をむちで打つように命じた。

　その態度をみて、全軍が 襟えり をただした。それで、規則を破る者はいなくなったのである。

　呂布は曹操軍が攻めてきたと聞くと、袁術と同盟をむすんで対抗しようとした。袁術が南から軍を出せば、曹操軍を 挟はさ み撃ちにしたり、予州の留守をおそったりできる。だが、曹操軍の 快進撃かいしんげき はそれを許さなかった。

　袁術が迷っているあいだに、曹操軍有利の情報が次々と届く。その結果、 臆病おくびょう な袁術は兵を出す気がなくなったのだ。

　曹操は大軍で下邳を囲むと、長期戦のかまえをとった。敵の兵糧がなくなり、内部からくずれるのを待つ作戦である。

「城内の民を 飢う えさせるおつもりですか」

　劉備が不満を述べると、曹操は苦笑した。

「力攻めをすれば、犠牲はもっと増えるだろう。苦しければ、城内で反乱を起こせばよいのだ。そうなれば、われらが攻めこんで救ってやれる」

「そうですか......」

　劉備は納得できなかったが、いい考えも浮かばない。引き下がるしかなかった。

　呂布は何度か、軍をひきいて挑戦してきた。しかし、曹操は応じない。呂布が攻撃してくれば、そのぶん退却する。城からある程度はなれると、他の部隊が城内に突入しようとするので、呂布は引き返さざるをえない。

　また、一騎打ちは絶対にするな、という命令も出ていた。立ち向かうなら、三人以上でと 釘くぎ を刺されている。曹操軍の名だたる武将たちが三人でかかれば、呂布をおさえられるのだ。

　関羽と張飛は戦えないのが不満であったが、命令にそむけば、劉備の立場が悪くなる。そう思って、仕方なく 我慢がまん していた。

　ところが、下邳は意外によく持ちこたえた。呂布の 功績こうせき ではない。劉備が治めていたときに、 倹約けんやく をして兵糧をたくわえたり、城壁を整備したりしていたからである。

　曹操も一時は弱気になった。このままでは、兵糧がたりなくなるおそれがある。だが、荀彧も郭嘉も退却には反対だった。

「脱走してきた兵士によりますと、呂布は絶望して酒びたりの毎日を送っているそうです。勝利は目の前、ここであきらめては、今までの苦労が水の 泡あわ になってしまいます」

　荀彧が言うと、郭嘉もつづいた。

「城攻めについては、私に策があります。これから雨の多い季節になりますから、それを利用したいと思います」

　郭嘉が地図を出して計略を説明する。

「水攻めか。おもしろい」

　曹操は二、三の修正をくわえて、ただちに実行を命じた。

　残る問題は兵糧である。 追加ついか の 輸送ゆそう を指示したが、届くまでに時間がかかる。

「非常時だ。仕方あるまい」

　曹操は食料の 配給はいきゅう 係がかり を呼んで、一人一人の割り当てを少なくするよう命じた。しかし、十日もすると、兵士たちも食事が減ったことに気づき、不満が出てくる。雨が降りつづくなか、ただ城を包囲しているのだ。 唯一ゆいいつ の楽しみである食事についての不満は 放置ほうち できない。

　曹操はもう一度食料係を呼びだした。

「兵士の不満をおさえるために、おまえの協力が必要なのだ。成功すれば、おまえはこの戦いの一番の 功労者こうろうしゃ になるだろう」

　食料係は、期待と不安が 混ま ざった表情でたずねた。

「私にできることでしょうか」

「おまえにしかできないことだ」

　曹操はわずかな間をおいて言った。

「その首をくれ」

　食料係は一瞬かたまってから、ひっと悲鳴をあげた。

「わ、私に何の 罪つみ があるというのですか。曹操様のご命令にしたがっただけではありませんか」

　いつのまにか、曹操は抜き身の剣を手にしている。

「わかっている。だから、おまえの家族には一生苦労はさせない」

「そ、そんな......」

　食料係は後ずさりしたが、曹操の踏み込みは速かった。剣が 水平すいへい に走り、食料係の首が 胴どう をはなれて転がる。悲鳴をあげる 暇いとま もなかった。

「 恨うら むなら恨むがよい。おまえの死には、私が責任をもつ」

　曹操はつぶやくと、食料係の胴体をこっそり手厚くほうむるよう命じた。

　首の方は、陣中にさらされた。

「食料の割り当てが減ったのは、係がごまかしていたからだ。 罪人ざいにん はすでに 罰ばっ した。今日からは元通りの量を配らせよう」

　曹操が説明すると、兵士たちの不満は 嘘うそ のようにおさまった。本拠地からの輸送隊も無事に到着し、全軍が士気をとりもどす。

　郭嘉の計略も準備がほぼ終わり、後は曹操の攻撃命令を待つのみとなった。下邳の落城が近づいている。

　食料係の死についての 真相しんそう は 秘密ひみつ にされ、劉備たちが知ることはなかった。もし知っていたら、曹操に協力することはなかったかもしれなかった。



　　　　　５


　呂布は 荒あ れていた。

　戦おうとすると曹操軍が逃げるので、戦にならない。一騎打ちに応じる武将もいない。かといって、深追いすれば、下邳の城が攻められてしまう。下邳などどうでもよいが、貂蝉をうばわれるわけにはいかない。

　呂布は身動きがとれなくなっていた。同盟を組んだはずの袁術からも、まったく 音沙汰おとさた がない。援軍が姿だけでも見せてくれれば、いくらでもやりようがあるのだが。

「何で何もかもうまくいかないのだ」

　雨がつづいて屋敷に閉じこめられると、呂布は酒を飲んでうさを晴らすようになった。

　すると、城内で反乱が起こった。陳宮と張遼がすぐに兵をひきいて 鎮圧ちんあつ したが、士気の低下は疑いようもなかった。呂布軍はみな、ほとんどあきらめている。兵糧もこころもとなくなってきた。

「もうだめかもしれない」

　呂布は酒臭い息を吐きだすと、貂蝉を抱きしめて言った。

「おれと一緒に脱出しよう」

　だが、貂蝉は呂布の手からするりと抜けだすと、冷たいまなざしを 注そそ いだ。

「あなたはいつから、目の前の敵に背中を向けるような男になったのですか。お酒に逃げて、救いがあるのですか」

　呂布は思わず手をあげ、貂蝉をなぐろうとする。

「な、何 !? 　おれはいつだって......」

　しかし、ふりあげた手は力なくおろされた。呂布はきびすを返し、荒々しい足取りで部屋を出て行く。

　呂布を見送る貂蝉の瞳に、冷たさはすでにない。気持ちがはなれたわけではなかった。まだ愛しているから心が痛い。

「私だってわかっているわ。もうだめだって。でも最後くらい......」

　貂蝉はつぶやいて、顔をおおった。細く白い指の間から、銀色の涙があふれてとまらなかった。


　雲あいから、太陽がわずかに顔をのぞかせた。薄い光のすじが 水みず 浸びた しになった大地を走る。長くつづいた雨がやみ、黒雲がゆっくりと遠ざかっていく。

　呂布は冷水で顔を洗い、さっぱりとした体で、部下たちの前にあらわれた。両眼には 覇気はき の炎が燃えている。貂蝉の言葉がきいて、気力をとりもどしたのだ。

　陳宮は顔をかがやかせて、呂布の前に進みでた。追いつめられた状況だが、呂布さえやる気になれば、まだ逆転の可能性はある。

「物見より、敵軍にあやしい動きがあるとの報告です。 堤つつみ をつくって川の水をためており、水攻めをする気かと思われます」

「ただちに兵を送って計略を 阻止そし せよ」

　呂布は張遼を指名した。

「かしこまりました」

　張遼がさっそうと部屋を出て行く。

「水攻めの成功失敗にかかわらず、敵は総攻撃をかけてくるだろう。そこに 勝機しょうき が見いだせるはずだ」

　呂布は力強く言った。曹操軍も、攻撃をしかけておいて逃げることはないだろう。城門で迎えうち、返り討ちにしてやる。

　呂布は全軍に出陣の準備を命じた。

　そのころ、曹操は武将たちを一堂にあつめて 軍議ぐんぎ を行い、まさに計略の実行を指示しようとしていた。そこへ、伝令が駆けこんでくる。

「水門に敵襲！　敵の勢いはすさまじく、このままではもちこたえられません」

　曹操はぴくりと眉をあげた。

「呂布か？」

「いえ、張遼という若い将のようです」

　その名前に、関羽が反応した。下邳の宴席で、議論を戦わせたのを思いだしたのだ。

「それがしに行かせてもらえませんか」

　曹操は 興味きょうみ 深ぶか げに関羽を見やった。

「もちろんだ。関雲長が救援に行くなら、これほど頼もしいことはあるまい」

　城に攻めかかるのに比べれば、水門を守るのは 地味じみ な 任務にんむ である。しかも、相手は呂布ではない。曹操軍の武将たちは志願しようとしなかった。

「他の者たちは持ち場にもどって、攻撃準備にかかれ。水攻めの成功を待って、総攻撃に移る」

「御意」

　一同はいっせいに返事をして、勢いよく立ちあがった。それぞれの持ち場に散っていく。

　劉備と張飛は南へつづく街道の守りを任されていた。袁術からの援軍を警戒しての配置である。

「関の兄貴はいいよなあ。おれも前線に出たいなあ」

　張飛がぐちるのはいつものことだ。劉備は微笑を浮かべて言った。

「ここは私一人でもいいだろう。呂布があらわれたら、翼徳も戦いに行けばよい。曹操殿の許可はすでに得ている」

「本当か !? 」

「ああ、今度ばかりは呂布から逃げるわけにはいかない。策は考えているそうだが、互角に 渡わた りあえる者がいないと 駄目だめ らしくてな。時期がきたら、指示があるだろう」

　張飛はとたんに機嫌をなおした。蛇矛をみがきながら、出番を待つ。



　　　　　６


　 黄土おうど 色いろ の水が、堤にたまってゆれている。いまにもあふれそうな大量の水が、 木製もくせい の門でとめられている。この水門を開けば、下邳の城下に水が流れこんで、 濠ほり をくずし、城門を押しあけるだろう。

　その堤の前で、 激戦げきせん の火花が散っていた。

　張遼がひきいる呂布軍は、水門を確保して計略を防ごうと、決死の突撃をかけた。曹操軍の 方陣ほうじん を張遼の方天戟が突きくずし、後につづく兵士たちが 穴あな を広げる。

　曹操軍の陣形はみだれ、何十人もが行き場をなくして堤に飛びこんだ。 派手はで な水音とともに、泥まじりのしぶきが高くあがる。ややあって、いくつもの頭が、水面に浮かびあがってきた。 懸命けんめい に泳いで 岸きし をめざす。

　張遼は 雑草ざっそう をかるように曹操軍をなぎ倒していたが、その前進は水門を目前にしてとまった。

「それがしがお相手しよう」

　黒々としたひげを風にそよがせて、 堂々どうどう たる 偉丈夫いじょうぶ があらわれたのだ。 大柄おおがら な 栗毛くりげ の馬にまたがり、青龍偃月刀を肩にかついでいる。音に聞こえた関羽その人である。

「一度、お手合わせ願いたいと思っていた」

　曹操もうなずいた。

「そうだな。この男は猛獣だ。人に 御ぎょ せるはずがない」

「待て。今度こそ忠誠をちかう。だから考え直してくれ」

　呂布がわめく。

「あなたほどの男が 命いのち 乞ご いとは情けない。最後くらい、いさぎよくしたらどうですか」

　声をかけたのは、隣でしばられている張遼である。だが、呂布の心には届かなかった。

「うるさい。貂蝉のために、おれは生き延びねばならんのだ」

　呂布は立ちあがろうとしたが、足までしばられているので転んでしまう。

「貂蝉！」

　呂布ははいつくばって、愛する人の名を叫んだ。びりびりと音がして、綱が 弾はじ けとんだ。両手が自由になる。呂布はさらに足の綱をひきちぎろうとする。

　人の輪がじりじりと下がるなか、曹操は腕組みをして、呂布を見下ろしていた。

「呂布よ。おまえの女は、城から身を投げて死んだぞ」

「な !? 」

　呂布の動きがとまった。

「やれ」

　曹操の指示で、夏侯淵と曹洪が駆け寄った。左右から、剣を突き立てる。それが何度かくりかえされて、呂布は息絶えた。

　天を 見み 据す えたその瞳は、かすかにぬれていた。

　最後に、曹操は張遼の前に立った。

「先ほどの言葉は、おぬし自身にもあてはまるのであろうな」

　張遼は決意をたたえて顔をあげた。

「もちろんです。 未熟みじゅく さゆえに 捕虜ほりょ となったからには、すでに命はないものと考えております」

「ならば、その命、私にあずけてくれ」

　張遼は一瞬、返事につまった。周囲をうかがい、関羽の姿を見つけて、目で 問と う。

　生きろ、と関羽のくちびるが動いた。

「わかりました。お仕えいたします」

　張遼はいましめをとかれ、曹操軍の一員となった。

　呂布軍の兵士たちで生き残った者は、張遼と同じ道を歩んだ。

　曹操は自軍の兵士たちに 土木どぼく 工事こうじ を命じて水攻めの 後あと 始末しまつ を行い、住民に食糧をあたえて戦の傷をいやした。

　そして、 臨時りんじ の長官と守備する軍を徐州に残して、許昌へ帰る。そのとき、曹操の眉をひそめさせる出来事があった。

　徐州の住民たちが、劉備を長官にしてくれと望んだのである。

「それは陛下がお決めになることだ。都にもどってからおうかがいをたててみよう」

　曹操はそう答えたが、心中はおだやかではなかった。劉備の人気をあらためて思い知ったのである。

「あの男をどうすべきか。そろそろ決断せねばなるまいな」

　細い目で宙をにらむ曹操であった。



　　　　　９


　 人中じんちゅう で無敵をほこった呂布も倒された。

　曹操は 丞相じょうしょう と大将軍をかねることになり、漢朝の臣として文武の 頂点ちょうてん をきわめた。名声と権力は増すばかりである。

　勝利を祝って、曹操は 巻まき 狩が りを 企画きかく した。許昌につとめる武将や文官を呼びあつめ、 献帝けんてい も 招待しょうたい した大規模なものである。

　巻狩りというのは、多くの人を使ってあらかじめ 獲物えもの を追いたて、一カ所に集めて行う狩りだ。皇帝など身分の高い人の 娯楽ごらく や、神にささげる 行事ぎょうじ としてもよおされる。

　心やさしい献帝は、楽しみのために動物を殺すことは好まなかった。だが、曹操の命令にはさからえない。皇帝 専用せんよう の特別の弓矢をもって参加した。

　狩り場につくと、曹操は献帝の後について馬を走らせた。 臣下しんか が皇帝と馬をならべるのは 不敬ふけい だが、曹操は馬の頭半分しか 遅おく れていない。その距離が、今の曹操の権力をよくあらわしている。

　この日は劉備たち義兄弟も参加を許されていた。劉備は献帝の 御前ごぜん で、見事に一羽のうさぎをしとめた。

「劉備殿は弓も 達者たっしゃ なのだな」

　劉備が 恐縮きょうしゅく していると、曹操が言った。

「陛下、あちらに 鹿しか がおります。ぜひ、腕前をご 披露ひろう いただきたい」

　腕前といっても、献帝は馬に乗るのがせいいっぱいで、矢を射ても当たるわけがない。三度失敗すると、あきらめて弓をおろした。

「丞相に任せよう」

「かしこまりました」

　曹操が献帝にむかって手を差しだす。弓を寄こせということだ。

　献帝は一瞬ためらってから、皇帝専用の弓を渡した。

　それを使って、曹操が矢を射る。

　狙いあやたまず、矢は鹿をとらえた。鹿はどさりと地面に倒れる。

「さすがだな」

「すべて陛下のご 威光いこう のおかげです」

　曹操は献帝の 称賛しょうさん に答えると、皇帝の弓を自分の背の袋にしまった。

　このやりとりを見ていた関羽が、腰の剣に手をかけた。曹操が皇帝にまったく 敬意けいい を払わず、失礼を働いているのに怒ったのだ。

　劉備も同じ気持ちだった。

　だが、行動はちがった。

　手を伸ばして関羽をひきとめたのである。

「なぜとめるのですか。やつは漢朝にわざわいをもたらす 悪党あくとう ですぞ」

「陛下の御前で剣を抜いてはならない。それに、周りは曹操の部下ばかりだ。雲長の命もないぞ」

　二人は小声で言い合った。

「 不忠ふちゅう を見逃すくらいなら、それがしは死んでもかまいません」

「いいから、今はよせ」

　劉備が強くたしなめると、関羽はしぶしぶ手をおろした。

「兄者のご命令であればしたがいます。ですが......」

「ああ、わかっている。このままにはしておかない」

　劉備にも、曹操が皇帝をないがしろにしていることはわかった。曹操は宦官たちや董卓と同じ穴のムジナなのだ。

　自分は曹操ではなく、献帝の、漢朝の臣である。漢朝に忠誠をつくすには何をすべきか、劉備は心に問いかけていた。
