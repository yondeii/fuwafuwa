電子書籍はページの概念がないので、本書はページ項がありません。
短い内容ですが、途中から読まれる方は見出しを参考にして下さい。
本書が英語学習やリスニング試験で役立ったり、洋楽が楽しく聞けたり
英会話習得の第一歩になれば幸いです。

日本人は英語の発音が上手くなる可能性がある？
－ － － － － － － － － － － － － － － － － － － － － －

具体的な方法の前に、ここでは日本人が持つイメージや先入観のある内容を
ご紹介したいと思います。書店の英語書籍やネットで見られる英語の情報を
見ますと「日本人は英語の発音が苦手」が前提になっている場合がほとんど
ではないでしょうか。
おそらくは「日本人はＬ，Ｒの発音が苦手」な事実からだと思いますが、
ではなぜ「発音が苦手」なのでしょうか。
これは単に、英語の「本当の発音」が聴き取れていないからです。
聴き取れない発音を真似しても上手くいく理由はありません。しかし見方を
変えると「聴き取りができれば、英語の発音は上手くなる」ことになりますが、
日本語のある特徴から「英語が聴き取れる＝発音が上手くなる」可能性がある
ことに気が付きました。

　○　英会話の習得さえ苦労しているのに何言ってるの？

と思われるかもしれませんが、本書で英語の聴き取り能力が上がった方なら
理解できない内容ではありません。
その理由は「発音の強弱が少ない（アクセントがそれほど重要でない）」
日本語にあります。
たとえば「東京」を（標準語で）発音する時は、ほとんど強弱がありません。
ところが英語では、ほとんどの単語にアクセントの強い箇所があるので、
「ｔｏｋｙｏ」はどこかに強いアクセントを付けます。
つまり英語は、アクセントも含めて言葉の意味を判断する言語です。
その他にも、シンプルな文法や品詞が分かりやすい特徴もありますが、
ここに（アクセントが重要、必要な）英語と日本語の違いがあります。

そして英会話の学習番組を見ますと、ネイティブの方が、生徒さんに
対して「ここの発音を、もっと強くすると上手になる」と言う場面を
見かけます。これは発音（の強弱）が重要な表れではないでしょうか。
このような状態を見ますと、日本人が英語を話す（発音する）ときは
「日本語のアクセントは弱いので、母国語のクセが出にくい」ことに
なります。
これが「日本人は英語の発音が上手になる可能性がある」理由です。

英会話の番組では、英語を習得した（英語が母国語でない）方が出演
しています。おそらくは世界中で使われる「実際の英語」を紹介する
意図があると思いますが、なかには母国語のアクセントが強く出ている
英語もありました。
しかし日本語はアクセントが弱いので、英語を話すまでの流れは以下に
なるのではないでしょうか。

　○　「英語の発音が聴き取れる（耳の）状態になる」
　　　　↓
　　　「聴き取った本当の発音を（そのまま）まねて発音する」
　　　　↓
　　　「日本語のクセが出にくいので、発音が上手くなる」

英語の発音が上手くなるには、先にリスニング力が必要です。
そして聴き取りができると、あとは「正確な発音」を真似するだけです。
リスニング力をつける前に発音記号や舌の位置がどうとか、唇を軽く噛む
とかで覚えようとすると遠回りになってしまうのではないでしょうか。
それでは具体的な内容に入っていきたいと思います。

（２つの習慣で）英語、英会話のリスニング力が上がる方法
－ － － － － － － － － － － － － － － － － － － － － － － － － － －

日本人が英会話を聞くと「会話スピードが、日本語より相当に速く聞こえる」
印象ではないでしょうか。不思議なことに、英語が母国語の方は
「相手は英語が苦手そう」と分かっても（？）ゆっくり話してくれません。
それどころか「英語が苦手なの？（話せないの？）」と続くのは分かりますが、
ここでも何故かゆっくり話してくれないようです。これは言語よりも
「自分の意思や気持ちを、はっきり言わなければいけない」文化や生活習慣が
影響しているのではないでしょうか。

すると自身の「聞き取りが難しい」状態を伝えることが重要になりますが、
会話のたびに「聞き取りが難しい／ゆっくり話して」では、リスニングの
向上につながりません。それに本格的に英会話を習得したい方なら、
普通の英会話の「速さ」で話したい（聴き取れるようになりたい）のでは
ないでしょうか。

そこで本書は（２つの習慣で）英語、英会話のリスニング力が上がる方法を
ご紹介したいと思います。

　① 　英会話のスピードが速い映像ソフトや英語の教材を用意して
「イヤホンを使用して」「やや大きめの音量で」視聴します。
　　　※音楽ソフトの音声は不可です。

　② 　① とは別の機会に、普通の英会話スピードの音声を視聴します。
　　　※普通の英会話を聴き取れる状態が目的なので、英会話教材の
　　　　聞き取りやすい（意図的に速度を遅くした）音声は不可です。

① の行為は、音楽ソフトを使わないで下さい。
「やや大きめの音量」は、耳に負担がかからない程度の音量で構いません。
常に① ② をセットにする必要はありませんが、① だけは、１日１回でも
よいので毎日行なって下さい。１回の視聴は、５分程度でも構いません。
聞く時間の「長さ」よりも、毎日の習慣にすることが重要です。

また同じ内容を毎日視聴しても問題ありません。
この場合は聞き取れる英単語は増えますが、飽きやすいので継続が難しく
なってしまいます。普段から英会話や洋楽を聞く方は、② の状態に当たる
ため、① の行為だけで構いません。
そして① ② の行為は、集中して聴き取る意識を持たなくても構いません。
普通に聞いている感じで十分ですが、これは「聴き取らなくてはいけない」
ストレスを防ぐためです。

何故このような練習方法か
－ － － － － － － － － － － －
　
　○　普通の英会話さえ聴き取れないのに、何故それよりも速い音声を
　　　聞く必要があるのか。聴き取りが難しくなるだけではないか。

ご紹介した方法を読まれた方は、このように思われるのではないでしょうか。
実際に、その通りの状態になるので、普通の英会話以上に聴き取ることが
できません。しかし① の行為は、速い英会話スピードに「耳を慣らす」目的
なので、この音声は聴き取れなくても構いません。
普通の英会話を聞いたときに、会話スピードが「遅い」と感じることで
結果的にリスニング力が上がる方法になっています。

そして二段階の英会話スピードによって、最後には（② の音声である）
普通の英会話に耳を対応させてしまいます。
また英会話の発音細部を聞き漏らさないように、やや大きめの音量と
イヤホンを利用します。

－ － － － － － － － － － － － － －
日本人は英語が聞き取れない？
－ － － － － － － － － － － － － －

　○　英語で使われる周波数は日本語よりも高いので、日本人は
　　　英語が聞き取れない。

英会話教材の広告や英語書籍には、上記のような内容がありました。
グラフでの詳しい説明もあったと思いますが、英語圏の人も日本人も
同じ人間です。
日本人が聞き取れないのは、何らかの理由があるのではないでしょうか。
以前に読んだオーディオ雑誌には「人間の耳は２０～２０キロヘルツの
音域を聴き取れる」とありましたが、国によって（耳の）能力に違いが
あるとは聞いたことがありません。
それに留学のように一日中英語の環境にならなくても、英会話を習得
される方はたくさんいらっしゃいます。

すると英語が聞き取れない（理解できない）日本人でも「音として」英語が
聴き取れる状態になるのは可能と言えるのではないでしょうか。
本書はこのような「耳の状態」にすることで、リスニング力を上げる内容に
なっています。

－ － － － － － － － － － － － － － － － －
「徐々に慣れる」発想を、逆転させる
－ － － － － － － － － － － － － － － － －

学校の授業や一般的なリスニング練習は、ゆっくりの英会話を聞いて
スピードに慣れる内容も多いのではないでしょうか。
これは運動のように「初めから急に、無理をするのは良くない」
「順を追って慣れるのが、自然な行為」という先入観があるからです。
突然に負担の大きい運動を始めると筋肉痛になるのは、誰でも想像
できます。しかし－

　○　ご紹介している方法は運動でも筋トレでもありません。

突然に速い英会話を聞いても後日に耳は痛くなりませんし、そのような
ソフトがあれば無理な行為でもありません。
そこで（一見すると）意味のなさそうな、聴き取りが難しい① の音声を
聞いて、普通の英会話を聞くときにリスニングの慣れを作ってしまいます。
つまり一般的な「徐々に慣れる」方法を逆転させた内容になっています。

－ － － － － － － － － － － － － － － －
二歩は進めなくても、一歩は進める
－ － － － － － － － － － － － － － － －

すでに書きましたが、① の行為を行なっても、ほとんどの内容は聴き取る
ことができません。
しかし１日１回、短時間でも継続すると、耳が「聴き取れない」音声に
慣れよう（対応しよう）とするので、脳の「音声を処理する」能力が上がり
ます。そして① の速い英会話に慣れると（① ではなく）② の普通の英会話が
聴き取れる「耳の状態」になっています。

　○　これが「英語、英会話のリスニング力が上がった状態」です。

まるで「突然に二歩進もうとしたけど出来なくて、でも一歩は進めた」状態
になって、英語のリスニング力は上がります。
ちなみに耳の状態といっても音声情報は「脳で」処理されるので、正確には
脳が「ある感覚」を掴んだ（覚えた）ような状態になります。

－ － － － － － － － － － － － － － －
右脳を使うと、英語が聴き取れる
－ － － － － － － － － － － － － － －

このような簡単な方法で、なぜリスニング力は上がるのでしょうか。
これは今まで聞き取れなかった音域が聴き取れる「聴力が上がった」
状態と「脳の情報処理が速くなった」状態が影響しているようです。
この時の脳内は、具体的にどのようになっているのでしょうか。

人間の発達した大脳は、主に左脳を使って言語を扱います。
そして英会話の聴き取りは「言語に関する行為」なので、左脳を使って
入力情報の処理を行ないます。
しかし① の行為は、英会話（言語）のスピードが速いので、ほとんどの方は
聴き取れない状態になってしまいます。
すると脳は「この言語情報を処理するには、右脳も使う必要がある」と
判断して、通常の言語を扱うとき以上に活動的になります。
その結果、左脳だけでなく右脳も使い始めて脳の情報処理が速くなるので
（言語を理解しようとするので）英語のリスニング力は上がります。

① のときに「音楽を使うのは不可」としました。
これは洋楽などを使うと、主に右脳を使ってしまうからです。
趣味で洋楽を（長年）聞いたとしても一向に英語が聞き取れないのは、
ここに理由があるのではないでしょうか。

－ － － － － － － － － － － － － － － － － － － －
左脳を限界以上に使ってから、右脳を活性化
－ － － － － － － － － － － － － － － － － － － －

また「言語は、主に左脳で扱われる」ことから、日本の教育では右脳を使う
機会が少ないと言われています。そして普段の生活でも言語が重要なため、
ここでも右脳はそれほど使われない事になります。
つまり日本の生活環境は右脳を使う機会が少ないので、実用、能力開発の
書籍は右脳を「直接的に」訓練する内容が見られます。

　○　ご紹介した方法は、これらの本とは逆の順序になります。

最初は「左脳」に対して、能力以上の情報処理を働きかけます。
そして「言語を扱う左脳だけでは、情報処理ができない」と判断させて
しまいます。ここから左脳をフルに使うと同時に、右脳も使うことで
脳全体を活性化させています。
ご紹介した方法はリスニング力が上がる成果がでますが、能力開発でも
「逆の順序」のほうが確実に脳を活性化できるのではないでしょうか。

ＦＭ放送を聞いているときでした。
ラジオから流れる洋楽が、今までとは違った感じに聞こえていました。
不思議に思いながらボリュームを上げると、英語のアクセントは聴き取り
やすく、音符の位置も分かるようになっていました。そして男性よりも
（女性や子供の）高い声の方が、英語を聴き取りやすい発見がありました。

ちなみにネイティブの方が、ラップ＆ ヒップホップ系の（速い）歌詞を

英語のリスニング力が上がったときは、このような状態でした。

英語のリスニング力が上がった「耳の状態」は
－ － － － － － － － － － － － － － － － － － － － －

　○　英語を聞いたときの「耳の状態」が変わります。
　　（ある感覚が掴めます）

本書ではこのように書いてますが、普段の生活では何故か「耳の変化」は
感じられません。
しかし英語を聞く時だけは、アクセントの位置や細かい発音が聴き取れるので、
耳が良くなったような状態になります。特に洋楽を聞いている時は、音符の
付いている個所が分かりやすいのですが、これは歌詞で同じような単語が多く
使われるからかもしれません。
そして左脳で言葉（歌詞）を聞くと同時に、右脳で演奏（音）を聞くので、
両脳をバランス良く使っている状態になるのではないでしょうか。

－ － － － － － － － － － － －
脳から見た「耳の状態」は
－ － － － － － － － － － － －

「右脳を使うことで脳の情報処理が速くなり、英語が聴き取れる」
内容は、すでにご紹介しました。
ここでは、もう一つの理由である「聴力が上がった状態」について
見ていきたいと思います。
こちらは脳内で、具体的にどのようになっているのでしょうか。

まず英会話を聞いたときに、耳から入った音声情報は、脳内部の
「音に関する個所」へと向かいます。そして音に関する箇所は、
音声の高さ（周波数）によって複数の「聞く個所」に分けられます。
（「進化しすぎた脳」より）
それによると主に日本語で使われる（１５００Ｈｚ以下の）低い周波数と、
英語の高い（２０００Ｈｚ以上の）周波数は別の個所になっています。
つまり日本語の環境では、英語を聞くときの個所がほとんど使われないと
言えるのではないでしょうか。日本人は英語のリスニングが苦手なのは、
ここに理由があると思います。

ご紹介した方法は、イヤホンと大きめの音量を使って、英語を聞く箇所を
「強制的に」使う状態にしてしまいます。
そしてこの方法を継続すると脳内の「聞く箇所」が活性化されるので、
英語のリスニング力は上がります。今まで聞こえなかった（英語の）発音が
聴き取れるので、聴力が上がった状態とも言えるのではないでしょうか。

－ － － － － － － － － － － － － － － － － － － － －
英語のリスニング力が上がるまでの練習期間は
－ － － － － － － － － － － － － － － － － － － － －

ご紹介した方法を１日１回、継続して、英語のリスニング力が上がるのは、
およそ数ヵ月後になります。ただし個人的には、音楽を聞いているときに
「耳の変化」に気付いたので、実際は、もう少し早い時期かもしれません。
一般的には、世代が上がるほど高音域の聞き取りは難しくなります。
（英語を聞く箇所よりも、ずっと上の音域だとは思いますが）
そのため効果が表れる時期は、人によって差がありますが、若い人や
言語能力の高い女性（の脳）は、短い練習期間になるのではないでしょうか。

－ － － － － － － － － － － － － － － － －
音としてのリスニング力が上がったら
－ － － － － － － － － － － － － － － － －

英語のリスニング力が上がると、今まで聞こえなかった（本当の）発音が
聴き取れます。すると英会話の学習は「次へ」行けますが、具体的には
どのような内容がよいのでしょうか。
簡単に言うと、できるだけ英会話を聞く機会を増やすのがよいと思います。

ところで生まれたばかりの赤ちゃんは、どのような過程で話せるように
なるのでしょうか。
最初は、簡単で限られた言葉を沢山かけられて、意味を理解できるように
なります。そして（覚える意識なく）少しずつ話せるようになりますが、
この環境は、復習や反復の要素があります。すると復習を重視した内容は
自然な「言葉の覚え方」に近いので、英会話の学習で使えるのではないかと
思いました。
そこで復習要素があって簡単な解説もあって、集中できる（短い）時間で、
英会話を聞く機会もあって費用は安くて・・欲を言えばきりがありません。
ですがこの条件は、テレビの英会話番組を使うことで可能になります。

－ － － － － － － － －
英会話番組の使い方
－ － － － － － － － －

テレビの英会話番組を見ますと「１日１フレーズ」の内容もあります。
そして普通に視聴すると「その日に、その時だけの内容」になるので、
これを「同じ内容を、何度も繰り返す」学習に変えてしまいます。
具体的には、深夜に放送されている再放送（１０分番組×４日分）を
録画します。１週間分の内容になりますが、計４０分しかありません。
この番組を１週間で１日１回視聴します。

「同じ内容で１日４０分の視聴」は、長く思われたり飽きやすいと
思われるかもしれません。ですが余分な個所はスキップや早送りで
短かくなりますし、視聴を忘れた日があっても何度か復習できます。
そして次の週も「１週間、同じ内容の視聴」と続きますが、先週の内容の
確認は行ないません。
英会話は、テストや試験のように「この時までに、覚えなければいけない」
わけではないので「どれくらい覚えたか」の基準は意味がありません。
たとえ「これだけ覚えた」と確認しても、その時だけのことです。

「視聴しても覚えていなければ、意味がないのではないか」と思われる
かもしれません。ですが「覚えたまでとはいかない」状態は、記憶できる
（覚えられる）までの潜在的な予備知識になりますし、英語学習の継続に
よって強固な記憶の土台になります。
そして視聴を継続すると、昨日は聴き取れなかった箇所が理解できたり、
少しずつ内容が分かるようになります。
この「前回よりも分かる」状態が、継続のモチベーションを作ってくれます。
復習を重視した学習は、英会話の内容から、単語や熟語も自然に覚えられる
のではないでしょうか。

－ － － － － － －
英会話の学び方
－ － － － － － －

英会話のビジネスを見ていると、不思議に思うことがあります。
それは「○○部売れた！／受講した！／利用した！」のＰＲがあるだけで
肝心な「○○の人が英会話を話せるようになった！」が少ないことです。
そして利用者のレビューを見ますと「話せるようになった」内容は少なく、
それでいて評価は高いという不思議な状態もありました。
ネット上で「話せるようになった」とＰＲしているのは、脳が柔軟な時期に
行なわれる子供向けの教室くらいではないでしょうか。
「それなら英会話の先生や著者さんは間違ったことを教えているのか」
というと、そうではありません。実際に話せるようになった方もいる
でしょうし、もしかしたら以下だと思っているのではないでしょうか。

　○　自分がこの方法で話せるようになったから「誰でも」
　　　同じ方法で英会話ができるようになる。

これはブログなどでＰＲしている方や、何かを利用して話せるようになった
方も同様です。しかし現状は、そうではありません。もしそうなら既に
それ以外の英会話ビジネスや書籍は淘汰されていることになるからです。
そして「英会話が習得できる」が売り物にも関わらず、誰にでも効果がない
現状や結果に対して触れている内容を見たことがありません。
ここで推測になりますが、教材の先生や著者、何かで効果のあった方は、
元から言語能力が高い「少数の」方なので、比較的、短期間の留学や
年齢に関係なく英会話を習得されたのだと思います。そのため同じ方法を
行なっても「多数である」言語能力が普通の方は話せるようになりません。

　○　どの英会話ビジネスでも効果があるのは、言語能力の高い
　　　一部の人だけ。

本書でご紹介した方法でリスニング力が上がったときは、あまりに簡単なため
「誰にでも」効果があると思っていました。その後は、レビューから間違いに
気が付きましたが、それでも必要な方法だと思っています。その理由は、
意味は分からなくても「音として」正確に聴き取れないと、発音の練習が
非効率になるからです。また語彙力が上がれば上がるほど「音として」から
「あの英単語」と理解できるようになります。

　○　「聞く」はそれでいいかもしれないけど「話す」はどうなのか。

番外編でも書いた気がしますが、おそらくは「後から」でも構いません。
英会話を聞いて（大雑把でも）意味が分かるようになれば、手持ちの語彙で
伝えることが、発音の練習や「話す」英会話になるからです。
残念ながら「誰にでも英会話が習得できる」方法は存在しません。
そして多数の方に必要なのは、本書の「耳の状態」と、聞く機会や覚える機会
などの「英会話に必要な語彙力の取り込み」ではないでしょうか。
英会話番組を使った方法もご紹介しましたが「誰にでも」ではなく一例です。
語彙力の取り込みは、自身に合った（必要に応じた）内容が一番だと思います。

リスニング力を上げるには、特別な教材や道具は必要ありません。
ご紹介した方法は、ほとんどの方に効果的な内容になっています。
