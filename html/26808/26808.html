第 １ 章 何 故 オ ト コ は パ ー ト ナ ー を 欲 し が る の か ？

　 ち ょ っ と 真 面 目 な お 話 に な り ま す が 、 男 で あ ろ う と 女 で あ ろ う と パ ー ト ナ ー は 本 来 求 め て 当 た り 前 な の で す 。

人 と い う の は 、 子 孫 を 繁 栄 さ せ る た め だ け に 生 ま れ て き た と 言 っ て も 過 言 で は あ り ま せ ん 。

も ち ろ ん 人 生 を 謳 歌 す る た め に も 色 々 な 異 性 と 知 り 合 う こ と は と っ て も 大 切 で す ね 。

し か も そ の 相 手 が 一 生 の 伴 侶 と し て 付 き 合 っ て 行 け る の で あ れ ば 最 高 で す 。 野 生 の 動 物 で も 昆 虫 で も 深 海 の 生 物 で も 草 や 木 や 命 あ る す べ て の 生 物 は 必 ず 性 交 を 通 し て 子 孫 を 作 ろ う と し ま す 。

セ ッ ク ス が 気 持 ち い い と 感 じ る の は 、 神 様 が 子 孫 を 残 し て ほ し い た め に 作 ら れ た 感 情 で す 。 気 持 ち よ く な け れ ば 誰 も パ ー ト ナ ー 探 し に 力 を 入 れ る こ と も な い で し ょ う 。

な の で 、 自 然 な 行 動 で あ り 決 し て 異 常 で は な い と い う こ と を 理 解 し て く だ さ い 。

特 に 、 男 性 は 多 く の 精 子 に よ っ て 日 々 気 持 ち が 子 孫 繁 栄 に 掻 き 動 か さ れ て い ま す 。 そ れ に 対 し 、 女 性 は 基 本 １ ヶ 月 に 一 つ の 卵 子 を 排 出 し て 精 子 を ひ た す ら 待 ち 続 け て い ま す 。

男 性 が 異 性 を 求 め た が る 頻 度 と 、 女 性 の そ れ を 比 較 す る と 圧 倒 的 に 男 性 の 方 が 性 欲 が 強 く 現 れ る の は そ う い っ た 身 体 的 な ゆ え ん な の で し ょ う 。

第 ２ 章 パ ー ト ナ ー の 存 在 っ て 一 体 な に ？

　 そ れ で は 、 男 性 に と っ て パ ー ト ナ ー の 存 在 と は ど の よ う な も の な の で し ょ う か 。

こ れ を 単 純 明 快 に 答 え る と 、 「 セ ッ ク ス 」 に 到 達 し ま す 。 当 た り 前 で す ね 、 第 １ 章 で 述 べ た と お り で す 。

し か し 、 で は ど の よ う な 女 性 を 好 む の か と い う こ と に つ い て は 個 々 の 気 持 ち や 人 間 性 、 価 値 観 、 理 想 と い っ た 外 的 要 因 と 内 的 要 因 の 重 な っ た 部 分 で お 互 い に 引 き 合 う も の で 結 ば れ て い く の で は な い か と 考 え ら れ ま す 。

人 は そ れ ぞ れ 生 ま れ た 環 境 や 育 っ た 環 境 で 人 の 数 だ け 違 っ た 人 格 が あ り

ま す ね 。 も し 、 双 生 児 が 同 じ 環 境 で 育 っ た と し て も 、 次 第 に 個 々 の 考 え 方 や も の の 見 方 に 違 い が 生 じ て 、 別 人 格 と し て 人 間 形 成 が で き て い く と い う の は 全 く 当 然 の こ と と 言 え ま す 。

ど の よ う な 異 性 と 出 逢 い 、 ど の よ う に 理 解 し あ い 、 ど こ ま で 連 れ 添 っ て い け る か は そ の 人 の 生 き 方 次 第 で す が 、 そ ん な 異 性 を う ま く 探 し 当 て ら れ る か は 、 は っ き り 言 っ て 縁 と し か 申 し 上 げ る こ と が で き ま せ ん 。

　 し か し な が ら 、 人 は わ ず か １ ０ ０ 年 前 後 の 間 に 子 孫 を 残 し 次 の 世 代 に 繋 げ て い く 事 を 神 さ ま か ら 使 命 と し て 受 け 取 っ て 生 ま れ て き て い ま す の で 、 男 性 に は 精 子 を 作 る 機 能 が あ り 、 女 性 に は 卵 子 を 作 る 機 能 が 身 体 に 備 わ っ て い る の で す 。

あ と は 、 そ の 人 の 「 努 力 」 で す ね 。

不 幸 に し て 一 生 理 想 の パ ー ト ナ ー に 出 会 え な い こ と も あ る で し ょ う 。 病 気 や 身 体 の 生 ま れ つ き の 障 害 で 子 孫 を 残 す 機 会 に 恵 ま れ な い 人 も い る で し ょ う 。

ま た 、 不 幸 な 事 故 や 災 害 で 心 が 疲 れ て し ま っ て 性 欲 が な く な る こ と も あ り ま す 。

そ れ で も 男 性 は 女 性 を 、 女 性 は 男 性 を ま た は 同 性 ど う し で も 、 パ ー ト ナ ー と し て 側 に い て ほ し い と 思 う の は 何 故 で し ょ う か 。

　 こ れ は 、 「 凹 凸 の 安 定 感 」 を 持 ち 、 人 間 性 を 高 め た り 他 人 に や さ し く 接 す る 心 を 養 っ た り 、 自 分 の 欠 け て い る と こ ろ を 気 づ か せ て く れ た り 又 は 相 手 の 欠 点 を 補 う な ど 、 人 の 一 生 で 経 験 を 豊 か に 幸 せ に す る こ と で 生 き 甲 斐 を 持 て る こ と に 感 謝 で き る 人 間 を 作 る た め の 「 手 段 」 と し て 違 っ

た 性 が あ る の だ と 、 筆 者 は 感 じ て い ま す 。

そ れ ほ ど 凹 凸 は 人 間 生 活 に と っ て 大 切 な ん だ な 、 と 感 じ る の は 私 だ け で し ょ う か 。

人 に 愛 情 を か け る こ と は 、 そ う し よ う と 思 っ て で き る も の で は あ り ま せ ん 。 理 屈 な く 異 性 に 魅 か れ 、 理 屈 な く 愛 情 が 生 ま れ 、 そ し て 無 理 な く 結 ば れ る と い う こ と こ そ 縁 で は な い か と 思 う の で す 。

第 ３ 章 ど ん な 方 法 で 出 会 い た い ？

　 で は 、 そ の よ う な 縁 を 作 る た め に ど ん な 努 力 を す れ ば い い で し ょ う か 。

ま ず は 、 女 性 （ パ ー ト ナ ー ） を 探 す た め の 流 れ を 考 え て み ま し ょ う 。

彼 女 候 補 の 女 性 と 何 ら か の 接 触 → そ の 女 性 と の コ ミ ュ ニ ケ ー シ ョ ン を 図 る → そ の 女 性 と 友 達 に な っ て 会 う → 友 達 以 上 に 親 密 に な る → 交 際 ス タ ー ト ！

簡 単 に 言 う と こ ん な と こ ろ で し ょ う か 。 で は 次 に も う 少 し 詳 し く ご 説 明 し ま し ょ う 。

１ ． 女 性 に 出 会 う こ と

ま ず 大 好 き と 思 え る 女 性 と 出 会 う 必 要 が あ り ま す 。 こ れ が 彼 女 の 作 り 方 の 第 一 歩 で す 。

な に も ア ク シ ョ ン を 起 こ さ ず に 「 女 性 と の 出 会 い が な い な ぁ 」 と 嘆 く の で な く 、 女 性 と 出 会 う 機 会 を 自 分 か ら 作 り 出 し ま す 。

今 の 自 分 の 環 境 で 周 り に 気 に な る 女 性 が い な い な ら 、 婚 活 サ イ ト か ら の 出 会 い や 街 コ ン な ど の パ ー テ ィ ー に 参 加 し て も い い で し ょ う 。 F a c e b o o k や ほ か の S N S で も よ い の で す 。 き っ か け に 良 し 悪 し は な い と 思 っ て く だ さ い 。

女 性 と 出 会 う 手 段 は 何 だ っ て か ま い ま せ ん 。 大 切 な の は ど う い う 手 段 で 知 り 合 っ た の か で は な く 、 つ き 合 っ て お 互 い が 幸 せ に な る こ と で す 。

２ ． 笑 顔 笑 顔 で 明 る い 自 分 を ア ピ ー ル ！

人 と の 出 会 い は 、 第 一 印 象 が 大 事 で す 。 実 に 最 初 の ６ 秒 で 決 ま る 、 と も 言 わ れ て い ま す 。

と い う の も 、 出 会 っ て 間 も な い 頃 は 互 い に 「 相 手 が ど う い う 人 物 な の

か ？ 」 が 分 か ら な い 状 態 な の で 、

警 戒 心 も あ る 以 上 、 会 話 を し て も 硬 く な り が ち で す 。 直 接 会 っ て い る と す れ ば 、 あ な た の 立 ち 居 振 る 舞 い が 全 て 第 一 印 象 に 繋 が っ て い ま す 。 も ち ろ ん 、 服 装 の 趣 味 も 判 定 基 準 に な り ま す か ら 、 あ ま り む さ っ 苦 し い フ ァ ッ シ ョ ン は 、 好 み で あ っ て も 控 え て お き ま し ょ う 。

お 互 い に 良 い 関 係 を 作 る に は 、 早 い 段 階 で 緊 張 を 解 い て お く 必 要 が あ り 、 そ れ に 役 立 つ の が 「 笑 顔 笑 顔 」 で す ！

お わ か り の 通 り 、 笑 顔 は 見 て い る だ け で リ ラ ッ ク ス し た 気 分 に さ せ て く れ て 、 自 然 と 相 手 の 気 持 ち も 楽 し く さ せ る こ と が で き ま す 。

３ ． 相 手 と の 距 離 を 縮 め る メ ー ル や L I N E の 秘 訣

最 近 で は 、 メ ー ル や L I N E は 好 き な 女 性 と 距 離 を 縮 め る た め の 最 適 な 手 段 で す ！

女 性 が 喜 ぶ メ ー ル を 送 る こ と で 、 距 離 が 縮 ま り や す く な る で し ょ う 。

女 性 か ら の 返 事 を 貰 い や す く す る た め に は 、 感 情 が 伝 わ る よ う な 素 直 な メ ー ル を 送 る こ と が 大 切 で す 。

　 ① 絵 文 字 を 適 度 に 使 う 。 使 い す ぎ に は 注 意 ！ 大 事 な こ と は き っ ち り 文 章 で 伝 え る 。

　 ② 喜 怒 哀 楽 を 表 現 す る 。 「 怒 」 は 必 要 あ り ま せ ん が 、 「 喜 」 の 表 現 を ひ た す ら 多 く す る こ と が 大 事 で す 。

「 あ り が と う ・ ・ ・ 。 」 「 あ り が と う ！ 」 「 あ り が と う ( ^ ^ ) 」 な ど 、 同 じ 言 葉 で も 終 わ り に 一 工 夫 を す る と い い で す ね 。

４ ． 女 性 に 好 意 を 持 た せ る に は

女 性 心 理 と し て は 、 話 し か け て く れ る ＝ 私 に 興 味 が あ る 、 私 の こ と を 嫌 い で は な い 、 と 感 じ る の で す 。

女 性 に 声 を か け る 心 構 え と し て は す っ か り 心 を 開 い て い る よ 、 と い う 気 持 ち で 話 し か け る こ と が 大 切 。

お ど お ど 遠 慮 を し た り 、 気 を 遣 っ て い て は い け ま せ ん 、 む し ろ 、 も う 打

ち 解 け た ！ と い う 気 持 ち で 明 る く 接 し ま し ょ う 。

そ れ で も 慣 れ な い よ う な ら 、 女 性 を 女 性 と 思 わ ず に 同 性 と 思 っ て 話 し か け て み ま し ょ う 。

女 性 は 意 外 に も 同 じ テ ン シ ョ ン で 言 葉 を 返 し や す く な る も の な の で す ！

　 グ ラ ス の 露 の し た た り を お し ぼ り で ふ き 取 っ て く れ た り 、 吸 い 殻 で い っ ぱ い の 灰 皿 を 取 り 換 え て く れ た り 、 グ ラ ス に つ い た 口 紅 の 跡 を 指 で ふ き 取 る し ぐ さ っ て 、 ド キ ッ と し ま せ ん か ？

そ ん な と き こ そ 、 彼 女 を 短 い 言 葉 で 褒 め て み て く だ さ い 。 効 果 は そ の あ と す ぐ に 現 れ ま す 。

５ ． 更 に 距 離 を 縮 め る 作 戦

あ な た が 社 会 人 な ら ば 、 同 じ サ ー ク ル に 入 る 、 一 緒 に バ ー ベ キ ュ ー を 企 画 す る 、 共 通 の ス ポ ー ツ を す る 、 一 緒 に ボ ラ ン テ ィ ア 活 動 に 参 加 す る 、 な ど で す 。

　 こ の よ う に 一 見 す る と 恋 愛 と は 全 く 関 係 な い 取 り 組 み を お 目 当 て の 女

性 と 行 う の で す 。

そ う す る と 彼 女 は 、 あ な た に 対 し て 警 戒 心 が ほ ぐ れ 軽 い 気 持 ち で 接 し て く れ る よ う に な る の で す 。

６ ． 今 度 は あ な た 自 身 に 好 意 を 抱 い て も ら う 作 戦

　 女 性 に 恋 愛 対 象 と 見 て も ら え な け れ ば 、 い つ ま で た っ て も 彼 女 は で き ま せ ん 。 た と え 何 百 人 の 女 性 と 出 会 え た と し て も 、 で す 。
