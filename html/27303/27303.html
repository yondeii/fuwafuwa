はじめに

 

私は、これまで、若い日本語教師の悩みをたくさん聞いてきました。

中国のある大学で授業をしていたときの話です。ある教師が「生徒が授業に興味も関心も示さないんですよ！」と相談に来ました。もちろん私にも、多少そのような経験があります。最初は、あまり意識しなかったのですが、その教師の授業をよく観察してみると、学生たちは私たちが中学や高校でよくやった「内職」 をしていました。志望校の受験科目にあまり関係ないと思ったとき、 受験の教科である 英語の単語を机の下で覚えたり、さしあたり、今 、 行われているの授業よりも、優先度が高いと考える科目の勉強を したりする 、教師の視線を気にしながら 行う 、いわゆる「内職」 です。

「なにやってんだ！？」 と 、 教師から一喝されたことも一度ならずあったはずです。 しかし、面白くない授業に対する学生たちの反応は、どこの国でも同じです。

中国の大学で日本語を学んでいる学生の多くは 、卒業後、日系の会社で通訳や翻訳の仕事 に就くことを 希望しています。そして、日本語能力試験 一 級に合格することは、 そのような 企業に就職するための登竜門なのです。ですから、彼らは授業中、面白くない授業はほとんど聞いていなくて、日本語能力試験一級の文法編や文章読解の問題を解きまくっているのです。その光景に出くわした私は決心しました。

これからは、 「学生との真剣勝負！」 だと。 『やるか、やられるか』 です。

『行列のできる授業をするぞ！ 』 と決意しました。

『 学生が、身を乗り出して聞くような授業です ( ・・・・・・・・・・・・・・・・・・・・ ) 』

もともと、大学の授業 で は、教師が学生のモチベーションを高める ことを意識した 授業はあまり見かけません。しかし、日本では、カリスマ講師と呼ばれる教師が予備校にはいますよね。『日本語学習者』に行う授業に限っては、そのような魅力ある学習が必要だと 私は 思います。

『体験的にそう思います』

彼らの当面の目標である、日本語能力試験の教材開発を研究しようと思ったのは 、 そのような動機からです。私の手作り教材は、新聞、雑誌からの記事は一切使用していません。著作権の問題もあるのですが、自分の思想、 大げさ に言えば哲学でしょうか。常日頃、 目にする 政治・経済・文化・人間関係や自然との触れ合いの中で自分が感じてきたことを文章にしてみました。

 

【一文の文章でも授業が一コマになることがあります。】

たとえば、授業で、日本料理について文章問題があったとします。 読者の 皆さんも、ご存知のように、日本料理は味だけでなく、『見た目』も重んじます。

そのことから、『日本の食文化』 についての話になります。普通、「今夜、 赤提灯 ( あかちょうちん ) にいくか！？」　 という文章があったら、ほとんどの留学生は 理解できません 。大人の日本人だったら、すぐに、「居酒屋で飲もうよ！」という 『 ものの 喩 ( たと ) え 』 としての 比喩 ( ひゆ ) 表現は分かりますよね。 また 、私は日本 人の サラリーマン を例にとり、 アフターファイヴの説明から、授業に出席している留学生のお国の事情を聞いたり し ます。

それが一つの「意見交流の場」 になります。

この 日本語能力試験問題集は、日本の文化や日本人の考え方を、日本語教師が日本語学習者と対話 しながら学習 できるような内容を考え、教材作成しました。そして、実際の日本語能力試験よりも難しめに作成しています。それは、日本語教師が日本語能力試験を指導する時に、教師自身も学生とともに考える面白い授業ができるように、との思いがあったからです。

実際、私が大学で留学生の日本語能力試験の指導をしていて、面白い授業 とは何かということを常に考えてきました。私の教材は、そのような授業を日本語教師にしてもらいたいと願って作成した教科書です。

 

本書の試験問題は、「言語知識・読解」分野が実際のＮ２試験と同様の形式と問題数でつくられています。なお、本書では 聴 ( ちょう ) 解 ( かい ) は 取 ( と ) り 扱 ( あつか ) っていません。

 

2014 年 1 月

　　　　著者　中村一男

問題９　文章の文法

【文章の流れに合った文かどうかを判断することができるかを問う問題】

 

 

問題　次の文章を読んで、【　 50 　】から【　 54 　】の中に入る最もよい ものを 、 1 ・ 2 ・ 3 ・ 4 か ら 一つ選びなさい。

 

ある人が人づきあいで悩んでいると言っていた。人づきあいは確かに難しいと思う。私も若い頃には相当 悩 ( なや ) んでいた時期があった。「相手が自分のことをどう思っているのか」などは一番 【 　 50 　 】 になることである。

1.         昔の大学と違い、偏差値の低い大学に入学した学生の学力不足を 補 ( おぎな ) うために、中学や高校レベルの特別補講を実施しなければならないこと

2.         昔の大学生は学力が高くて特別補講などはまったく必要とせず、大学側は、もしそのような学生がいたら退学させていたこと

3.         大学は学力の低いすべての高校生を入学させた結果として、その責任を果たす時代になったこと

4.         近年の学生の学力不足問題は、日本が初めて直面する深刻な問題であること

 

70 　 A と B は 、学生の学力問題をについて、どう思っているか。

1.         A も B も大学生の学力は、昔とそんなに変わりはないと考えている。

2.         A も B も学生の質においては、大学間の差はあまりないと考えている。

3.         A は大学間における学生の質の格差を指摘して、 B は一流大学に合格した、いわゆる「勝ち組」の企業の評価を指摘している。

4.         A は難関大学に合格した学生は就職試験において有利であると指摘し、 B はそのような指摘はしていない。


A

近年、日本の大学は少子化の影響を受けてさまざまな問題に直面しています。中でも大学全入時代と言われているように、学生が高望みさえしなければ、全員大学に入れるようになっているようです。しかし、数字上はそのようになりますが、一流大学といわれる大学、すなわち難関大学に入学するには、激しい競争に勝ち抜いて合格しなければならないのは、どこの国でも同じだと思います。

このため、日本の大学では難関大学と比較的楽に入学できる大学との間に二極化現象が起きています。そして難関大学に入学できた学生と簡単に入学できた偏差値 （注） の低い大学の学生との学力差が問題になっているのです。

現在、大学当局は学力の低い大学生を対象に、高校や中学のレベルで補講授業をしなければならないという昔では考えられない現象 が起きているのです。

（教育評論家　Ａ氏）

（注）偏差値：学力などの検査結果が集団の平均からどの程度ずれているかを示す数値。

 

B

近年、少子化時代の影響で大学の「定員割れ」が問題となっている。一部の有名大学を除いて、入学試験の答案用紙に名前を書くだけで大学に入れる時代がやってきた。誰でも大学に入れるような時代になり、分数の計算さえもまともにできない大学生が珍しくなくなった。そのような大学生が大学卒業後にまともな就職があるとは考えにくい。

他方、 T 大や K 大といった超難関国立大学や W 大や J 大といった超難関私立大といった一流大学に合格した「受験勝ち組」の価値評価は、今後ますます高まって行くだろうと私は考える。

相対的に大学の質が低下しているこのような状況の中で、極めて近い将来、企業側はどのような学生を採用しようとするのだろうか。まず間違いなく「受験勝ち組」を採用するだろう。なぜなら、「受験勝ち組」であれば質の低下の幅も最小限だろうと企業は判断するのではないかと考えるからである。

（大学受験評論家　Ｂ氏）

問題　次の文章を読んで、後の問いに対する答えとして、最もよいものを 1 ・ 2 ・ 3 ・ 4 から一つ選びなさい。

私は留学生に対して日本文化についての講義をしています。以下は、日本文化事情の講義録の一部です。

 

さて、みなさんは「 アフター５」という言葉を聞いたことがありますか。聞いたことのある人は手を挙げてみてください。（一人も手を挙げる者はない）

そうですか、それではこのサラリーマンの「アフター５」という言葉と、 居酒屋 ( いざかや ) の「酒」と「焼き鳥」という奇妙な取り合わせで説明していきます。焼き鳥とか居酒屋という 概念 ( がいねん ) 、つまりその言葉の持つ意味、背景を知ることで日本人の行動様式を知ることができるのではないかと思います。そして、将来、みなさんが日本の企業に就職した場合に、今からお話しすることは日本文化、日本社会、日本人を知る上でとても役に立つと思います。

まず、日本人の毎日の労働時間は 8 時間で、朝 9 時から 5 時というのが標準勤務時間と考えてもいいと思います。もちろん企業、会社によりかなりの違いはあります。たとえば、深夜から朝方まで働く人、夕方から深夜にかけて働く人、つまり三勤交代制という勤務体系を採用している会社も日本にはたくさんあります。しかし、今回の講義の中ではそこまでは触れません。


話を本題に戻します。なぜ、「アフター５」というのかを話してみたいと思います。日本人のサラリーマンの多くの仕事が 5 時に終わります。そのまま真っすぐ自宅に帰る人、帰らずに職場の同僚や上司と食事に行く人、様々ですが、特に東京、大阪といった都会で働く多くのサラリーマンは居酒屋とよばれる日本風の 食事処 ( しょくじところ ) にいきます。そこで酒を飲みながら、酒の 肴 ( さかな ) の焼き鳥あるいは 刺身 ( さしみ ) を食べたりして、その場を楽しむのです。

居酒屋では、 勤 ( つと ) め 人 ( にん ) たちは世間話とか職場での話題などを話しながら、お 互 ( たが ) いのコミュニケーションを 図 ( はか ) ります。最近では、日本特有の年功序列制度、終身雇用制度という雇用制度に変化がみられ、アメリカ型の能力主義や、より良い職場を求めて転職する会社員もいますが、基本的には現在においても日本の伝統的な雇用制度は広く日本の企業の社会的慣習として、根付いています。

また、雇用システムがどのように変化したとしても、人々が会社に勤めることには変わりはありません。そこには仕事以外の人間関係が必ず生まれます。会社には 平社員 ( ひらしゃいん ) から 社長、会長までいます。しかし、同じ人間には変わりありません。仕事中には言えないようなことでも、お酒を飲みながら、「酒の肴」をつつきながらであれば、話すこともでき、時として、よい人間関係を築く上で、「憩いの場」となるのです。

「アフター５」が日本人の「憩いの場」として長い歴史があるのもよくわかります。
