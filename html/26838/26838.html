 FORGET ME NOT BLUE  

    

 　言葉によって人を傷つけ、遠ざける事ができるという事を初めて強く自覚した時、私はそのあまりの理不尽さに悲しくなって、ひねもす泣いて過ごしてしまった。傷つけるという行為が理不尽なのではない。傷つけるために使われる言葉そのものが理不尽だと感じた。  

 　一人っ子同士で結婚した両親の間に生まれた、これまたたった一人の子として、父母にも祖父母にも生まれた時から溺愛され、まだ幼かった当時の私にとって、言葉というのは巧みに駆使して周囲の人間の愛を勝ち取り、その精神を自分の腕にからめとる時に使う魔法のようなものであり、同時に私の幼い利己心を充たすために必要な、お菓子のようなものであったから。  

 　私にとって言葉とはいつで も 豪奢ごうしゃ な 虚飾きょしょく に満ちていて、なおかつ常に彼らを自分の元にひざまずかせ奉仕させるために計算され洗練されたものでなければならなかった。  

 「ねえ、私のこと、好き？」  

 　首をかしげて上目遣いで彼らを見ながら話し掛ける。途端に彼らは狂喜し私の頬に口付けし、私にも同様のことをするように望んだ。｢パパのこと好き？｣「ママのこと好き？」そ う 訊き いてくる彼らの目の奥のさらに向こうの、こと幼子に対しては可愛いという感情のみを礎にして接し、思考力が退化してかろうじて本能の領域ばかりが活動するようになった脳の中には、私の返事 を 渇望かつぼう している親という名の動物がいた。  

 　そんな時、「好きだよ」でもなく「うん好き」でもなく、私はいつもこう答えていた。  

 「当たり前でしょう？　嫌いなわけないよ」  

 　生まれながらにして高貴な身分を保証された人間が、臣下に対してそうするように、私は彼らに私の愛を自分のほうから言葉を通してかりそめのものであっても与えることで、それを半ば意識的に求める彼らと、そして私自身を満足させた。裏切られる心配がないという最低限のルールの下での他愛もない遊び。私は五つになる頃まで、その、今となっては私の中で遺骸となった遊びに夢中になり、そしてより成長して広い世界に触れるにつれて、空しさという感情を自発的に認識できるようになり、やめた。  

 　その後しばらくはなりをひそめて平々凡々と暮らしていた私の自意識だったが、時が過ぎ、思春期を迎える頃、私は自分が女性しか愛せない体に生れてきたことを嫌でも直視せざるをえなくなり、再び、言葉で自分を飾り立てることを始めた。しかし幼かった頃のように自分を周りの人間よりも高みに立たせるために使うのではない。自分の中で人を愛する心 の 萌芽ほうが を感じた時より今にいたるまで、私はそれを自らを守るための貞操帯として用いた。言葉を使って、体ではなく、心を守った。いや、結局は守ると言い聞かせて自分を傷つけていたりしたのだから、昔西洋で拷問に使われたという、鉄の処女に自ら志願して入るようなものだった。鋼鉄の彫像の中に入れば外から攻撃を受けずに済む。内側にある無数の鉄 の 棘とげ に突き刺される覚悟があればの話だったが、少なくとも不特定多数の人間から攻撃されるよりは、自ら の 知悉ちしつ できる範囲で傷を負う方が、痛みに対する処し方も想定の範囲内で済むので、楽だった。  

 　十六の夏、初めて本気で人を愛した時、それが小学校からずっと親友として付き合いつづけた女の子だったことに、自分で も 愕然がくぜん とした。恋の弓矢はしばしば軌道を逸れて飛ぶ。私相手の少女のことを、知らぬ間に曲線的に好きになった。気が付いたら彼女を中心にしたサークルが描かれていて、私はその中から出られなくなった、そんな、感じ。すべてが彼女を中心に回り、私は彼女の意思を自分の意思 と 錯誤さくご するほど彼女に夢中になった。きっかけは忘れた。好きであること自体が、とても大きな問題だったから。中学の時も、高校の時も、本当は狂おしいくらいに彼女のことを想っていたのに、その彼女から朝の登校時に「早苗は好きな人いる？」などと無邪気 に 訊き かれても「あなたよ」とは答えずに「別に」などとその場しのぎのことを言い、そして、夜には眠れないほど後悔した。  

 　「あたし、本当はあなたが好きなの」  

 　一度でいいからそう言ってみたかった。  

 　言ってみたいとはと思ったけれど、私という存在を黙殺されるのが恐くて、できなかった。  

 「彼とのこと占ってよ」  

 　彼女― ― 佐織さおり は、いつもそう言って私にタロット を 繰く らせた。  

 　彼女の恋を占うのは、本当は嫌だった。鈍い女。頼まれるたびに、心底そう思った。悔しくて、悲しくて、まともな精神状態ではなかったが、断るわけにもいかず、当時愛用していた不思議の国のアリスのデッキをつかって、私は歯噛みしながら彼女と、彼女が愛しているとはにかみながら私に告げた男性のことを占った。いっそ悪魔や死神やソードのスートのバッドカードが出ればいいなどと、強く念じながら、クロスの上でカードを並べた。元来が占いが好きで、やれ血液型だ星座だと、その手の話題には飛びついて饒舌にまくし立てていた佐織は、余程都合が良かったのか、私に何度も占いをねだった。彼女の頼みを断りきれないまま、私は学校で占い、彼女の部屋で占い、喫茶店で占い、時には試験の合間に電話での相談にものった。彼女は私の占いを信用していたのか利用していたのか、卑屈に両手を合わせたり、ケーキをおごってくれたり、「やっぱり早苗の占いは頼りになるね」などとおだてたりして、私に占わせつづけた。私は、佐織が誰か他の男と付き合うことなど絶対許せなかったのだけれど、それでも自分が彼女と同じ性別であるということで負い目を感じていたし、結ばれることはないのだとはっきりとわかっていたので、それを続けざるを得なかった。続けていればネガティブなカードが出るかもしれない。そうなったら彼女にはっきり駄目よと告げよう。そんなことも考えながら私はいつも彼女の求めに応じ、カード を 手繰たぐ った。ヘキサグラム、ケルティッククロス、プリドヴェル の 錨いかり 、ヘブンズゲート、ホロスコープ、フィフティーンカード。その頃しばしば用いていた、さまざまな形式のスプレッドをつかって佐織と彼との相性を見たり運命を見たりしたけれど、どんな時でも決定的に黒いカードは出なかった。私の願いとは裏腹に。  

 「やっぱりあたし、彼とうまくやれるんだね」  

 　私の占いに勢いづいた佐織は、私に初めて占いを頼んでから三ヶ月後に彼に告白し、そして、彼の心を、我が物とした。  

 「ありがとう」  

 　佐織は言った。  

 「おめでとう」  

 　心の中 で 鳴咽おえつ しながらも、それを表に出さないようにして、私は彼女を形ばかりに祝福した。  

 　誕生日を迎えたばかりの十七の秋を失意にまみれて過ごし、私はその後大学受験のために勉強に打ちこんだ。何かを必死に行うことでしか自分の傷をごまかすことはできないと思っていたし、運動が苦手な私は部活動にも入っていなかったから、勉強くらいしかやることがなかったのだ。  

 　高校を出たあと、佐織は周囲も驚くほどのすばやさで、彼との結婚を決めた。彼女自身、勉強が得意ではなかったし、彼の方も、大学にはいかずに家業の漁業を継ぐことになっていたので、親や親戚の思惑などどこ吹く風と、同級生の誰よりもはやい新婚旅行に、卒業式を終えてまもなく、旅立っていった。  

 　私の方は一浪してからどうにかこうにか国立大の文学部に合格し、故郷を離れ、東京で学生として平凡な毎日を過ごしていた。  

 　私は、今でも佐織のことが好きなのだと思う。  

 　だから佐織の頼みはつい聞いてしまう。  

 「ねえ早苗」  

 　久しぶりに私の所に電話をかけてきた佐織の声は涙ぐんでいた。  

 「占って欲しいんだけど」  

 　私は「旦那と何かあったの？」などと心配そう に 訊き く振りをしつつ、不謹慎にも、内心では嬉しく思っていた。形はどうあれ、彼女は、私を、頼っている。それが、たまらなく誇らしかった。  

 「うちの人が浮気しているかどうか、見て欲しいの」  

 　うちの人が、ねえ――私は手近にある筆立ての中の何十もの色鉛筆の中から一本を適当に選んでから、黒い地のメモ紙に彼女の言葉を記した。途中、その偶然手にした淡いブルーの色鉛筆で、まったく話に関係のない相合傘を書いたり、ハートマークを何度も何度もなぞったりなどと、子ども染みた真似をしながら。  

 「最近私のことを避けているみたいなの」  

 「携帯に電話しても電源が切られているの」  

 「子どもを寝かせて夜遅くまで起きて待っているのに、あの人ったら、毎晩飲み歩いているの」  

 　彼女は私に対し て 堰せき を切ったように愚痴を続けた。  

 　私は、彼女の言葉 を 反芻はんすう し、彼女があまりにも私と違いすぎる人生を歩んでいることをまざまざと見せ付けられ、苦々しく思った。佐織の人生に踏み込むことは出来そうにもないと感じた。いまだ彼女のサークルに囚われたままの私。必死になって彼女を囲む世界のすべてを把握しようとするけれど、その中 に 円周率パイ のようにどうあがいても割り切れない因子があり、私は断念する。  

 　私は、手近にあったメジャーアルカナだけのシンプルなカードを手にして、彼女のために占いを始めた。  

 　クロスをしかずに直接テーブルにカードを広げる。佐織のことを考えながらシャッフルし、気が乗らないまま、ランダムに並べる。悪魔の逆位置、星の逆位置、世界の正位置、力の正位置、法皇の正位置、魔術師の正位置、正義の逆位置、復活の正位置。  

 　仕方がないかと思いながら彼女に占断を告げる。  

 「彼が誘惑に負けて自分勝手な行動をとっているのは、今の家庭の状況が自分の理想と違うからね。もともと自分の思い通りに振る舞えなければ我慢の出来ない人みたいだし。雑事を押し付けられるのが嫌いみたいだから、深刻な顔で彼に家庭のことについて相談するのはやめといたほうがいいわね。今は彼の中では遊びたいという部分が突出しているから自分でも行動を抑制することが出来ないようだけれど、そのうち大人にならなければならないことに気が付くだろうから、しばらく辛抱すれば、事態は好転するはずよ。その時は彼もかなり自分のしてきたことを反省するはずだから、早まって別れようとしないで我慢すれば、またもとの仲のいい二人に戻れるわよ」  

 　まるで扱いの難しいダイアトニックのコンサーティーナで、器用にジンタ を 奏かな でる芸達者なピエロのよう。元来が道化として生きる宿命を負った存在であるせいで、技術が余計に浮き彫りになり、当人の意思とは無関係に、哀れさをかもしだす。  

 「早苗、ありがとう。何か、楽になった。うん」  

 「そう。あたし、役に立てた？」  

 「いつもありがとう。早苗に相談すると、気が楽になるんだ」  

 　佐織が少しだけ元気を取り戻した声で言った。  

 　私はほぼ予定調和の展開を人生 の 諧謔かいぎゃく として受け入れるよりなかった。  

 「早苗は好きな人、出来た？」  

 「ううん、まだ」  

 　私は答える。いつものように、言葉を探して、自分で自分を別の自分に飾り立てようと試みるが、切なさと虚しさがないまぜとなり、適切な装いが出来ない。試み は 頓挫とんざ する。  

 　佐織は言う。  

 「何かあったら、私に、相談してね」  

 　私の心に土足で踏み入る彼女の言葉。心がめりめりと音を立ててひび割れながら、たわむ。相談なんて、できない。そうはっきり言えればいいのに、私はまたも上辺 を 繕つくろ い、そして中身を自 ら 綻ほころ ばせる。  

 「わかってる。好きな人が出来たら、真っ先に教えるよ」  

 　砂漠を旅し、食料がつき、喉が渇いて死にそうな時に見つけたオアシスの泉の水を嬉々として手 で 掬すく って一気に飲んだら毒気混じりの塩水だった。そんな感じ。言葉にすればするほど、私は、塩分の作用で飢える。更なる毒気に、喉の猛りをかきむしる。  

 　それからしばらく高校時代の思い出話やら大学のことやら同級生のことやらを話してから「またね」の言葉だけを残して、佐織は電話を切った。  

 　私は、一人だけ次元の異なる場所に取り残されたような気がして、途端に寂しくなって、色鉛筆の頭をかじった。そして、淡いブルーの色鉛筆に書かれた色名を選んだ私自身の無意識のあざとさに、空しく、笑った。  

 　 忘れな草の青フォーゲット・ミー・ノット・ブルー ――花言葉は、「私を、忘れないで」。  

 　頬杖を突いてしばらく無為に時間を過ごした後、とりあえず熱いシャワーを浴びようと私は立ち上がり、バスルームに向かった。私は、私の周りのすべてのものを、体から洗い流してしまいたかった。  

    

 


 

 CHARCOALBLACK  ( 旧題 　 IF I NEVER KNEW YOU)  

    

 　こうして離れて見ているだけでも、あなたの笑顔は、実に収まり良く、私 の 動悸どうき に、重なる。廊下の向こうで、そして、教室の隅で、高校最後の学年の、一つ一つの瞬間を惜しむ様にして、友達とはしゃぐ快活なあなたの日焼けした顔を見ると、空気が透明であることに、感謝せずには、いられない。私の目線の先には、たいていあなたの笑顔があり、私はそれを、理性 で 濾過ろか することもせず、密かに楽しみ、そしてあなたは、その視線に気付かずに、無自覚に、私を、悦ばす。女が女を好きになっても、仕方がないとは思うけれど、それ以上に、私がビアンであるということ自体が、私の人生において、仕方ない。仕方のないことは、どんなに周囲に干渉されても、受入れる以外に道はなく、私も、あなたを好きだということを、ここ数ヶ月、真摯に、受け止めている。いくらかの、葛藤と、ともに。  

 　テニス部の女は日焼けしていて色気がない。はばからずに言いきる男子も、たくさんいるけれど、私は、あなたのその浅黒い肌を、とても素適だと思う。舌の上で転がして融かす上質なチョコレートのように、あなたは、私の瞳に、甘 い 余韻よいん を、残す。確かに紫外線は有害。肌に負担をかける。いつか嫌な染みを残す。だけれど、日焼けしたそ の 痕あと が、そうやってあなたを汚しても、大丈夫。抱きしめて、私が舐めて拭ってあげる。あなたの肌を優しく撫でて、指紋で綺麗に削ってあげる。それほど私は、あなたが、好き。あなたが、受入れてくれなくても。  

 　一度も話したことがないのに、なぜなのだろうと、自分でも、不思議に思う。未知の魅力が、私をあなた に 惹ひ きつける。ノンケに恋して何をする。そう言いきり、きっぱりとあきらめ、みずから蓋をできるだけの潔さをもてず、不安と希望がないまぜになった、私の、幼すぎる、想い。  

 　私の恋は、池に落した鉄の斧。水面の波紋が合図になって、女神が出てきて、私に問う。   

 　あなたが落したのは、この、金の斧ですか。それとも、こちらの、銀の斧ですか。  

 　私は答える。  

 　私が落したのは、鉄の斧です。でも、その金と銀の斧も、両方、欲しい。  

 　女神がその申し出を断り、鉄の斧だけを渡したら、私は 、 反駁はんばく する。  

 　鉄で出来た斧は、放っておいても、いずれ錆びます。銀より も 峻烈しゅんれつ に。あなたの池に落ちたから、すぐに酸化が始まるでしょう。そうしたら、使い物にならなくなる。だから、新しい斧をください。あなたの水にさらされて、腐食するだけの、これの代わりに。  

 　同じように、私は、自分の心を手ずから落して、誰かがよりよい選択肢を与えてくれるのを、待っている。言葉にせずに見詰めつづけるという、消極的な、告白。私の心臓の鼓動で振動する空気の波に、あなたが気付いてくれたなら、その時こそ私は私の欲する物が何であるかを告げようと思いながら、あなたを、ただ、見ている。  

 　私は、あなたに、酸化する。その時点で、私の体は、男には、使い物に、ならない。そして、ひるがえれば、あなたにも。それがために、私は、あなたが私のほうを見ると同時に、視線を、外す。とても、同じ目線では、いられない。私の視野は仰角 。 憧あこが れてあなたを見上げる。他の女を見る時 は 俯瞰ふかん 。まとめて下に見据える。私は天上のあなたと下界の凡婦との間で空に漂い、足元を固めることも出来ずに錆びていく。体をむしばむ反応を、自分でも止めることは出来ず、意識も、応じて、もろくなる。あなたのふとし た 仕種しぐさ で、傷ついたり悦んだり、感情の針が、面白い様に、振り切れる。私という人間が、錆びきる前に気付いてと、私は、あなたに、無言で、訴える。行き着く所まで行けば、私は、時間に、崩れ去る。あなただけを見ている時は、幸せ。怖さも、臆病さも、夢に、少しだけ、還元される。だけど、その後自分の姿を鏡に映して見詰めると、たまらなく、虚しい。私の肉体は女。隠し立ての出来ない現実。黙殺も出来ない。水面に浮かぶ顔が、あざとすぎれば、女神は、多分、呆れる。  

 　友人 の 昭仁あきひと は言う。  

 「ゲイもビアンも、因果なもんだね。幻とわかっていても 、 潤うるお うために、オアシスを求めずには、いられない」  

 　僕ら一生濡れることなんてないんじゃないの？  

 　笑いながら彼は続ける。  

 　私は、彼が口にはしなかった本音を、見抜く。  

 『多分、涙以外では』  

 　濡れなくてもいいけどさ――  

 　私は、あなたを見詰めたままで、答える。  

 「せめて気付いて欲しいよね。こんなに、想っているんだから」  

 「駄目、本田は図太くて鈍いから 、 小夜子さよこ の気持ちには気付けないって」  

 　昭仁の言葉に多少の苛立ちを覚えながらも、あなたが色恋に疎い女であることは否めない。  

 「確かに鋭かったら、私もこんなに苦しまなくて済んだと思うけど」

 　ようやく口に出せたのは、虚しいばかりの、優越。  

 　私は、これが最後の未練と、あなたの手跡を財布から取り出し、そっと唇に当てた後、その紙を、破いて、捨てた。足元に散らばる、ありふれ た 顛末てんまつ に、目に見えないバクテリアが、群がる。  

 　私は 、 滑稽こっけい なくらい錆びきった自分がこの世界に崩れていくのをまざまざと感じ、身を滅ぼしても何か未来に残るものがあるはずだと、自分を偽り 、 叱咤しった しながら、何食わぬ顔で昭仁の前に出られるように、水道で、顔を、洗った。
