第 三 章 　 正 し い ア プ ロ ー チ の 方 法

連 絡 先 の 聞 き 方

　 「 ま た ご 飯 で も い こ う よ 、 ア ド レ ス 教 え て 」 と ス ト レ ー ト に 連 絡 先 を 聞 く こ と っ て な か な か で き ま せ ん よ ね 。 こ う い う の は 慣 れ て い な い と な か な か で き ま せ ん 。 飲 み 会 な ど で 複 数 い た 場 合 「 ま た み ん な で ご 飯 い こ う よ 」 と 「 み ん な で 」 を 強 調 す る の は あ り で す 。 こ れ な ら あ な た も 女 性 も 構 え ず に す み ま す 。 そ れ に 複 数 な 集 ま る 場 で は 、 あ な た が 言 い 出 さ な く て も 「 ま た 飲 み に い き ま し ょ う ！ 」 と い う 話 に な り ま す 。 そ の 時 あ な た が 手 を あ げ て 「 じ ゃ あ 俺 幹 事 す る よ 」 と 言 え ば 、 自 然 に 連 絡 先 を 聞 く こ と が で き ま す 。

　 み ん な で 写 真 を 撮 る の も あ り で す 。 写 真 を 撮 っ た 後 、 「 あ と で み ん な に 送 る よ ！ 」 と 言 え ば こ れ も ス ム ー ズ に 連 絡 先 が 聞 け ま す 。 よ っ て 連 絡 先 を 聞 く 理 由 ＝ 必 要 性 を 作 っ た ほ う が ア ク シ ョ ン を 起 こ し や す い で す 。 ま た 相 手 の 興 味 の あ る 情 報 を 教 え る の も い い で す 。 「 こ の サ イ ト に 詳 し い こ と 書 い て る よ U R L 送 ろ う か 」 や 「 ○ ○ 好 き な ん だ ね 。 そ れ じ ゃ あ 」 と 言 っ て 動 画 や 写 真 を 送 る の も い い で す 。

　 そ の た め に も 相 手 の 興 味 の あ る こ と を 引 き 出 す こ と が 大 切 で す 。

相 手 の 趣 味 や 興 味 を 引 き 出 し て 、 キ ッ カ ケ を 作 っ て く だ さ い 。 そ の 際 「 ○ ○ を 送 ろ う か 」 と 携 帯 を 手 に 持 ち な が ら い う と 、 流 れ で 教 え て も ら い や す い で す 。

距 離 の 縮 め 方

　 異 性 と 距 離 を 縮 め る に は 「 モ ノ の 貸 し 借 り 」 が オ ス ス メ で す 。

　 ま ず は 会 話 や Ｌ Ｉ Ｎ Ｅ で 相 手 の 興 味 を 探 り ま し ょ う 。 そ れ で 本 や 漫 画 や 映 画 、 音 楽 な ど 相 手 の 好 み が わ か れ ば 、 「 あ 、 ○ ○ が 好 き な ん だ 」 「 他 に は ど ん な 作 品 が オ ス ス メ ？ 」 と い う 流 れ か ら 「 だ っ た ら コ レ と か 好 き な ん じ ゃ な い ？ 」 と オ ス ス メ の 作 品 を 貸 し た り で き ま す 。

　 そ う す れ ば 、 本 ・ 漫 画 ・ D V D ・ C D ・ 等 を 貸 す の を キ ッ カ ケ に ア ポ を と れ ま す 。 そ の 後 も 感 想 を Ｌ Ｉ Ｎ Ｅ で 送 り あ っ た り 、 貸 し た も の を 返 し て も ら う た め に 会 っ た り 、 そ の 時 に ま た 続 き の も の を 貸 し た り と 、 接 点 を 増 や し 続 け る こ と が で き ま す 。

　 男 女 が 距 離 を 縮 め る の に 、 モ ノ を 貸 し 借 り は き わ め て 有 効 。 そ の 際 は 相 手 が ハ マ り そ う な 作 品 を 予 測 す る の が ポ イ ン ト 。 同 じ 作 品 に ハ マ る と 共 通 の ホ ッ ト な 話 題 が で き て 、 会 話 や Ｌ Ｉ Ｎ Ｅ も 盛 り 上 が る か ら で す 。

　 た だ し 趣 味 の 押 付 け で 熱 く な ら な い よ う に 注 意 し て く だ さ い 。 あ な た が 全 力 で オ ス ス メ の 作 品 で も 相 手 が 好 み で は な か っ た ら 相 手 は 困 る だ け で す 。 相 手 が ハ マ ッ て も 一 方 的 な ト ー ク を し な い よ う に し ま し ょ う 。 会 話 の キ ャ ッ チ ボ ー ル が 大 切 で す 。

Ｌ Ｉ Ｎ Ｅ や メ ー ル で 距 離 を 縮 め る

　 連 絡 す る 時 は 「 や や ツ ン デ レ 」 が ベ ス ト で す 。 「 基 本 は 相 手 に 合 わ せ な が ら 、 相 手 よ り や や 控 え め 」 を 心 掛 け て く だ さ い 。

　 片 思 い で は 相 手 と 自 分 の 間 に 温 度 差 が あ り 、 そ の 段 階 で 好 意 を 見 せ す ぎ る と 、 相 手 は 重 く 感 じ ま す 。 理 想 は 「 こ の 人 は 私 の こ と 好 き か な ？ 」 と 相 手 に 思 わ せ る の が ベ ス ト で す 。 明 ら か に 好 き だ 、 と 確 信 は さ せ な い 方 が い い 。

　 連 絡 は 「 や や 」 控 え め に を 心 掛 け な い と つ い や り す ぎ て し ま い ま す 。

　 連 絡 は 「 相 手 か ら き た ら 返 信 す る 」 と い う 双 方 向 の や り 取 り が 基 本 で す 。 相 手 か ら ど ん ど ん 返 信 が く れ ば 、 何 回 で も 送 っ て Ｏ Ｋ で す 。 で も 自 分 か ら 連 絡 す る 場 合 は 、 一 日 一 回 ま で に し ま し ょ う 。 返 信 が こ な け れ ば 、 そ れ 以 上 送 る の は や め ま し ょ う 。

　 皆 さ ん も 、 し ょ っ ち ゅ う 送 っ て く る 人 よ り 、 た ま に 送 っ て く れ る 人 の 方 の が 貴 重 に 感 じ ま せ ん か ? 連 絡 は 控 え め に し た ほ う が 、 逆 に 存 在 感 が 増 す の で す 。

　 文 章 の 長 さ や テ ン シ ョ ン や 絵 文 字 の 量 や 改 行 な ど は 、 相 手 に 似 せ て 書 き ま し ょ う 。 す る と 「 セ ン ス が 合 い そ う 」 と 思 っ て も ら え ま す 。 相 手 が く だ け た 口 調 に な っ て き た ら 、 こ っ ち も く だ け

　 と い う の も 、 メ ー ル や Ｌ Ｉ Ｎ Ｅ だ と 相 手 の 状 況 や 反 応 が 見 え な い た め 、 意 図 が 伝 わ ら ず 誤 解 さ れ る リ ス ク が 高 い か ら 。 お ま け に メ ー ル や Ｌ Ｉ Ｎ Ｅ は 残 る た め 、 深 く 印 象 が つ き ま す 。 相 手 が 不 機 嫌 な と き に 変 に く だ け た 口 調 の メ ー ル を 送 る と 、 イ ラ ッ と さ れ た り と か ・ ・ ・ 。 だ か ら こ そ 、 シ ン プ ル が 一 番 で す 。

　 下 手 に ア ピ ー ル を 狙 っ た り 、 長 文 を 書 い た り せ ず 、 伝 え た い 伝 え た い こ と を シ ン プ ル に に 伝 え ま し ょ う 。 飾 ら な い 、 自 然 体 の 言 葉 で 。

　 最 後 に 、 「 最 近 寒 い か ら 風 邪 気 を つ け て ね 」 「 お 疲 れ 様 ！ ま た お い し い 焼 肉 屋 い こ う ！ 」 と こ れ く ら い シ ン プ ル で い い で す 。 そ の ほ う が 相 手 も サ ク ッ と 返 し や す い で す 。

　 会 っ て る と き に 楽 し そ う だ と 「 こ の 人 は 、 私 と 会 え て 嬉 し い ん だ 、 私 の こ と 好 き な の か な ？ 」 と 相 手 は 思 い ま す 。 で も そ の 後 、 あ ま り メ ー ル や Ｌ Ｉ Ｎ Ｅ は し な い こ と 。 す る と 「 あ れ 、 好 き じ ゃ な か っ た の か な ？ 」 と 気 に な っ て 、 ま た 会 っ て 確 認 し よ う と し ま す 。

　 メ ー ル や Ｌ Ｉ Ｎ Ｅ は 難 し い し 、 振 り 回 さ れ る の は 辛 い で す よ ね 。 返 事 が こ な い と 一 日 中 気 に な り ま す し 、 着 信 が な っ た ら め っ ち ゃ 早 い 速 度 で 反 応 し た り 。 ハ ッ キ リ い う と 、 時 間 の 無 駄 で

す 。 携 帯 を ベ ッ ド に で も 放 り 投 げ て オ ナ ニ ー で も し ま し ょ う 。

第 四 章 　 デ ー ト に 行 く

　 男 性 目 線 の 恋 愛 本 に は ＜ 練 り に 練 っ た デ ー ト プ ラ ン を 成 功 さ せ れ ば 付 き 合 え る ＞ と 書 か れ て い ま し た が 、 一 番 優 先 す べ き は 「 デ ー ト に Ｏ Ｋ を も ら う こ と 。 」 女 性 か ら す れ ば 「 さ く っ と 食 事 で も 」 と 誘 わ れ た ほ う が 、 さ く っ と Ｏ Ｋ し や す い で す 。

　 女 性 も そ こ ま で 親 し く な い 異 性 と 長 時 間 過 ご す の は 緊 張 す る し 、 会 話 が 続 く か 不 安 に な り ま す よ ね 。 一 方 １ ， ２ 時 間 ほ ど ご 飯 を 食 べ る だ け だ と 気 軽 に Ｏ Ｋ も ら い や す い で す 。 お 酒 が 飲 め る 女 性 だ と 、 緊 張 が ほ ぐ れ る と い う メ リ ッ ト も あ る 。

た だ し 、 共 通 の 趣 味 が あ る の で あ れ ば 、 「 今 度 一 緒 に イ ベ ン ト い こ う よ 」 「 新 し い 映 画 お も し ろ そ う だ け ど い か な い ？ 」 と 誘 え ま す 。 そ の よ う な 共 通 項 が な い 場 合 は 「 さ く っ と 食 事 」 誘 う の が ベ ス ト で す 。

　 よ く あ る 聞 く 話 し で す が 、 恋 愛 慣 れ し て な く て 、 誘 っ た 時 に 「 断 ら れ た ら ど う し よ う 。 嫌 わ れ た ら ど う し ょ う 。 」 と マ イ ナ ス 思 考 に 落 ち が ち で す 。

　 一 方 恋 愛 に 慣 れ て い る 人 は 「 と り あ え ず 誘 っ て み る か 」 と 気 軽 に 誘 う の で す 。 そ し て 気 軽 に 誘 わ れ て 方 は 気 軽 に Ｏ Ｋ し や す い の で す 。 「 た か が １ ， ２ 時 間 ほ ど 食 事 す る だ け だ ！ 」 と 自 分 に 言 い 聞 か せ 、 「 と り あ え ず 食 事 に 行 っ て み て 、 も っ と 仲 良 く な れ ば い い な 」 く ら い の テ ン シ ョ ン で 誘 い ま し ょ う 、

キ ッ カ ケ を 作 る

　 食 事 デ ー ト の 場 合 、 食 べ 物 の 話 題 か ら 誘 う の が ス ム ー ズ で す 。

　 メ ー ル や 会 話 の 中 で 、 「 こ の 前 、 ○ ○ で お い し い パ ス タ を 食 べ た 」 的 な 話 を 振 れ ば 「 い い で す ね ー 。 ど こ に あ る ん で す か ？ 」 と 相 手 は 返 し て き ま す 。 そ こ で 「 ピ ザ が 最 高 だ っ た よ 。 よ か っ た ら 一 緒 に 行 こ う 。 」 と 誘 え ま す 。

　 ま た は 「 最 近 、 ど ん な と こ ろ で ご 飯 食 べ て る の ？ 」 「 料 理 で な に が 好 き ？ 」 と 話 を ふ っ て 「 ○ ○ に 旨 い イ タ リ ア ン あ る ん だ け ど 、 よ か っ た ら 今 度 ど う ？ 」 と 誘 っ て み る 。

　 モ ノ の 貸 し 借 り を キ ッ カ ケ に す る の も い い 手 で し ょ う 。 「 今 度 、 ○ ○ 貸 す よ 。 」 と い う 流 れ か ら 「 じ ゃ あ ご 飯 で も 食 べ よ っ か 」 と デ ー ト に つ な げ ま し ょ う 。 相 手 が 仕 事 や レ ポ ー ト な ど の 課 題 を 抱 え て い た ら 「 そ れ 終 わ っ た ら 、 お 疲 れ 様 会 で 、 旨 い ご 飯 で も い く か ！ 」 と 誘 う の も い い

で す ね 。

　 些 細 な 頼 み ご と す る の も 手 で す 。 資 料 を メ ー ル し て も ら う と か 、 テ ス ト 範 囲 を 教 え て も ら う と か 、 相 手 の 負 担 に な ら な い 頼 み ご と を ひ ね り 出 し て 「 お 礼 に 食 事 で も 」 と 誘 え ま す 。

第 ６ 章 　 告 白 の 場 所 、 シ チ ュ エ ー シ ョ ン

　 手 を 繋 い だ 時 点 で 好 意 は 伝 わ り ま す が 、 告 白 は 必 ず し ま し ょ う 。

　 こ の 人 は 本 当 に 私 の こ と が 好 き な の か ？ 単 に 彼 女 が 欲 し い だ け じ ゃ な く て ？ ヤ リ た い だ け で は な い よ ね ？ 勢 い と か で は 言 っ て な い よ ね ？ ち ゃ ん と 私 の 中 身 分 か っ て る ？ な ど 。
