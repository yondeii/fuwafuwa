「 そ こ は 、 っ て 、 こ こ を ど う に か し な い と ダ メ で し ょ ？ 」

　 白 い 下 着 の 真 ん 中 に は も う シ ミ が 出 来 て い て 、 そ の 上 を ゆ る ゆ る と 俺 は な ぞ っ た 。 隙 間 か ら 手 を 入 れ る と 、 ぬ ち 、 と 糸 を 引 く 愛 液 が 手 に 絡 み つ く 。

「 ほ ら 、 も う こ ん な に 」

　 先 輩 の 目 の 前 で 指 を 広 げ て 、 糸 を 引 く 様 を 見 せ つ け る と 、 一 層 先 輩 は 恥 ず か し そ う に 唇 を 震 わ せ た 。

「 指 、 入 れ ま す ね 」

「 う 、 う ん ... ... 」

　 差 し 入 れ た 初 め て の 女 性 器 は あ た た か い 。 グ ニ グ ニ と 中 で 指 を 踊 ら せ て い る と 、 先 輩 が あ る と こ ろ で ピ ク リ と 体 を 強 張 ら せ て 、 甘 い 声 を 漏 ら し た 。

「 こ こ ？ 」

「 は っ 、 あ ♡ だ め 、 だ よ ♡ ん 、 ふ ... ... ♡ 」

「 体 は ダ メ っ て 言 っ て な い で す よ ？ 」

言 っ た 。 も う 、 そ ろ そ ろ い い か な 。 ナ カ は ほ ぐ れ て 柔 ら か い 。

「 入 れ ま す ね 」

　 先 輩 の 耳 元 で 低 く 囁 く と 、 コ ク ン と 頷 く 気 配 が あ っ た 。 俺 が ジ ッ パ ー の 前 を く つ ろ げ て 、 ボ ロ ン と 自 身 の 性 器 を 取 り 出 す と 、 そ の 大 き さ に 驚 い た の か 、 先 輩 は 息 を 飲 ん だ 。 き っ と 怖 い と 思 っ て い る に 違 い な い 、 初 め て な ん だ か ら 。

　 先 輩 に 対 し て こ ん な こ と を し た ら 失 礼 か も し れ な い け れ ど 、 俺 は 優 し く 先 輩 の 頭 を 撫 で た 。

「 大 丈 夫 で す よ 」

「 痛 く な い 、 か な ... ... ？ 」

「 痛 く な い よ う に 、 し ま す 」

　 痛 く な い 、 な ん て 言 い 切 れ な か っ た け れ ど 。 俺 は そ っ と 彼 女 の 入 り 口 に ペ ニ ス を あ て が っ て 、 に ゅ ぷ ... ... と 腰 を 進 め た 。

「 ん ♡ は っ 、 あ ♡ は い 、 っ た ね ... ... 」

「 は い 、 入 り ま し た 」

　 途 中 、 少 し 痛 か っ た み た い で 先 輩 は ち ょ っ と 涙 ぐ ん で い た け れ ど 、 頑 張 っ て く れ た ら し い 。 少 し だ け 血 が 滲 ん で い て 。 あ あ 、 正 真 正 銘 、 俺 は 先 輩 と セ ッ ク ス し て ... ... ま し て や 、 そ の 憧 れ の 、 あ の 、 ゆ か り 先 輩 の 処 女 を 貰 っ て し ま っ た の だ と 実 感 し た 。

「 動 き ま す よ 」

　 少 し で も 痛 み が 和 ら ぐ よ う に 、 胸 を 揉 ん で 、 乳 首 を 刺 激 し な が ら 腰 を 振 る 。

　 先 輩 は 無 言 で 教 室 の 机 の 上 に 上 体 を 倒 し て 、 俺 が や り や す い よ う に 姿 勢 を 整 え て く れ た 。 長 い 黒 髪 が 絹 の よ う に 流 れ て 散 っ た 。

「 は っ ... ... 」

「 あ ♡ ん っ ♡ ふ ♡ 」

「 ど う 、 で す か ？ 」

「 ん 、 き も ち 、 い い よ ... ... ？ ♡ 」

　 こ こ に き て や っ と 、 先 輩 は ふ わ り と 花 が 開 く よ う な 笑 顔 で 嬉 し そ う に 笑 っ て く れ た 。 そ の 眦 か ら ひ と し ず く 涙 が も れ る 。

「 嬉 し い ... ... 最 後 に 、 っ て 思 っ て た か ら 」

　 腰 を 振 り な が ら た ず ね る 。

「 だ っ て 」

　 ど う い う こ と だ 、 な ん て 思 う 暇 も な か っ た 。 ス ピ ー ド を 上 げ て 抽 送 を 繰 り 返 す と 快 感 が 頭 の 中 で ス パ ー ク し て 、 先 輩 の い や ら し い 声 が シ ャ ワ ー の よ う に 降 っ て き て 俺 の 脳 み そ が 溶 け て 行 く 。

第 四 章 　 君 と 僕 の 踏 切

　 そ の 日 は 、 上 京 す る 先 輩 を 見 送 っ た 。 あ の 日 の こ と な ど 何 も な か っ た よ う に 、 行 き の 列 車 の 時 刻 を 、 俺 と み ど り に 教 え て く れ て 、 そ れ で 俺 は み ど り と 二 人 、 ホ ー ム を 去 っ て 行 く 電 車 に 向 か っ て 手 を 降 っ て い る 。

　 電 車 が 消 え て 見 え な く な る ま で 手 を 降 り 続 け た 。 大 き く 。

　 ど う し て 先 輩 が 「 一 度 だ け 」 と 言 っ た の か は よ く 分 か ら な い 。 好 き ... ... で い て く れ る な ら 、 遠 距 離 だ っ て あ る の に 、 と 不 純 な 俺 は 思 っ て い た 。 実 際 、 距 離 的 に は 不 可 能 で も 。

　 な ん と も 言 え な い 虚 無 感 に 包 ま れ て 俺 は 突 っ 立 っ て い た 。

　 蝉 の 鳴 き 声 が シ ャ ワ ー の よ う に 降 り 注 ぐ 。 太 陽 か ら の 熱 で ホ ー ム の コ ン ク リ が 揺 ら め い て い た 。

　 カ ン カ ン カ ン 。

　 踏 切 の 閉 ま る 音 。

　 そ ん な 俺 の 後 ろ か ら 、 み ど り が 突 然 抱 き つ い た 。 お い 、 と 声 を か け て 振 り 返 ろ う と し て 、 俺 は 止 ま っ た 。 泣 い て 、 い る ... ... ？ 　 そ ん な に 先 輩 と の 別 れ が さ み し か っ た の だ ろ う か 。

　 だ が そ ん な 俺 の 胸 中 と は 違 っ て 。 み ど り は 、 恐 る 恐 る 。 見 た こ と も な い 、 可 愛 ら し い 女 の 表 情 を し て 俺 を 見 上 げ る 。

「 好 き 、 だ よ ... ... 」

　 好 き 、 と 繰 り 返 さ れ る 声 に 俺 は 言 い よ う の な い 感 情 を 抱 い た 。

　 踏 切 が 閉 ま っ て 、 開 い て 。 人 が 線 路 を 渡 っ て 行 く 。

　 夢 の よ う な 思 い 出 と ― ― 俺 は 、 こ の 可 愛 い 幼 馴 染 の 、 こ の 献 身 深 い 幼 馴 染 の 、 気 持 ち を ず っ と ず っ と 踏 み に じ っ て い た の だ と ― ― 目 眩 が す る よ う な 、 真 夏 の ひ と 時 だ っ た 。
