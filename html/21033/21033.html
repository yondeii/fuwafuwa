　　一章　舌戦



　　　　　１


　 建安けんあん 十三年（ 西暦せいれき 二〇八年）冬。

　二十年ほどつづいた 戦乱せんらん が 終お わりに 近ちか づいている。そう 考かんが える 者もの は多い。

　 中国ちゅうごく の北半分を 平定へいてい した 曹操そうそう は、 満まん を 持じ して 南征なんせい の 号令ごうれい を 発はっ した。 総そう 兵力へいりょく は五十万をこえる 大軍たいぐん だ。

　曹操の 武器ぶき は兵力だけではない。その 権力けんりょく の 源みなもと は、 都みやこ の 宮殿きゅうでん に 閉と じこめられている 漢かん の 皇帝こうてい であった。曹操は漢の 丞相じょうしょう の 地位ちい にあり、皇帝の名で 命令めいれい を出したり、 諸侯しょこう に 官位かんい をあたえたりすることができる。曹操にさからう者は 朝廷ちょうてい の 敵てき として、 処罰しょばつ の 対象たいしょう となった。

　南征の 最初さいしょ の 標的ひょうてき は、 劉表りゅうひょう の 治おさ める 荊州けいしゅう であった。劉表は曹操の大軍を前に 病死びょうし し、後をついだ 劉琮りゅうそう は、 降伏こうふく の 道みち を 選えら んだ。

　 困こま ったのは、 客将きゃくしょう として荊州にいた 劉備りゅうび である。 追お いつめられた劉備は 民たみ とともに南へ走り、 諸葛亮しょかつりょう の 知略ちりゃく と、 張飛ちょうひ 、 趙雲ちょううん の 武勇ぶゆう によって、 難なん を 逃のが れた。劉備 一党いっとう は 江夏こうか に 落お ちついて、 孫権そんけん との 同盟どうめい に 活路かつろ を見いだそうとしている。

　 当初とうしょ 、劉備は 江陵こうりょう に向かうと言っていた。曹操はその 情報じょうほう を 信しん じて江陵に 先さき 回まわ りし、出しぬかれたわけだが、くやしがったのは 一瞬いっしゅん だった。江陵は多くの 軍船ぐんせん と 物資ぶっし が 集あつ められている 基地きち であり、ここをおさえることには大きな 意味いみ があったのだ。

「劉備にふりまわされて、 大局たいきょく を 見失みうしな ってはならぬ。 遠征えんせい の 目的もくてき は 孫権そんけん を降伏させることだ。劉備はその 後あと で、ゆっくりと 料理りょうり すればよい」

　曹操は劉備に 裏切うらぎ られたことを 忘わす れてはいないが、 怒いか りに 任まか せて 暴走ぼうそう することはなかった。江東の孫権に 使者ししゃ を 送おく って、降伏をうながす。

　 無理むり に 攻せ めようとしないのは、 若わか くして死んだ 軍師ぐんし 、 郭嘉かくか の 遺言ゆいごん があったからだ。 地ち の 利り がある江東を攻めれば、大きな 被害ひがい をこうむる 恐おそ れがある。 理り をといて降伏させるべし、と郭嘉は 主張しゅちょう していた。

　とはいえ、曹操は 戦いくさ の 準備じゅんび も 進すす めている。曹操軍は 水軍すいぐん が 弱点じゃくてん であったが、降伏した劉表軍を 吸収きゅうしゅう したことで、それを 克服こくふく しつつある。

　孫権はどちらかといえば 慎重しんちょう な 性格せいかく で、あえて 不利ふり な戦にのぞむことはあるまい。孫権が降伏すれば、この中国 大陸たいりく に、もはや曹操の 相手あいて になる 勢力せいりょく はない。曹操による 統一とういつ は、 目前もくぜん に 近ちか づいていると 思おも われた。

　しかし、 時代じだい の 流なが れにあらがう 者もの もいる。

　言わずとしれた劉備の 一党いっとう だ。劉備たち 義兄弟ぎきょうだい 三人は、 黄巾賊こうきんぞく の 討伐とうばつ に 参加さんか して 以来いらい 、 民たみ の 味方みかた をつらぬいてきた。

　曹操を 倒たお し、献帝を 救出きゅうしゅつ して 漢かん 王朝おうちょう を 復興ふっこう させる。そして、民のための 政治せいじ を行う。その 夢ゆめ を 実現じつげん させるまで、劉備はあきらめるつもりはなかった。

　もっとも、 現実げんじつ の 問題もんだい として、劉備たちだけの力では、曹操に 対抗たいこう のしようがない。何としても、孫権を戦いの 場ば にひきずりこまなくてはならなかった。

　 交渉こうしょう の 任にん をになうのは、 伏竜ふくりゅう こと諸葛亮である。だが、劉備の 信頼しんらい を 一身いっしん に集める軍師は、なかなか 腰こし をあげようとしなかった。

「こちらから先に出向いてはなりません。もうすぐ、相手の使者がやってきます。それを 待ま ちましょう」

　諸葛亮の言うとおり、しばらくして孫権から使者がやってきた。劉表が死んだことに 対たい するおくやみだという。劉表の 長男ちょうなん の 劉琦りゅうき が 応対おうたい することになった。

　 出迎でむか えの 用意ようい をする劉琦に、諸葛亮がたずねる。

「今まで、孫権との 間あいだ で、おくやみやお 見舞みま いの使者を 交換こうかん したことはありますか」

「いえ、はじめてです。 孫家そんけ とは 仇かたき 同士どうし でしたから」

　孫権の父である 孫堅そんけん は、劉表との戦いで 戦死せんし した。以来、孫家の者たちは劉表にうらみをいだきつづけている。孫権の 代だい には、 国境こっきょう 付近ふきん で 何度なんど か 争あらそ いがあり、孫権軍が 優位ゆうい に戦いを進めていた。

「では、おくやみは 口実こうじつ ですね。使者には、 他ほか に目的があるのです。曹操軍のことを聞かれても、 決けっ して 答こた えないでください。しつこい場合には、諸葛亮に聞け、と言ってもらえれば、私が対応します」

　諸葛亮は同じことを劉備にも言いきかせ、自分は 裏うら で使者の 様子ようす をうかがった。

　孫権の使者は 魯粛ろしゅく という、 堂々どうどう とした中年の男だ。関羽ほどではないが、豊かなひげの 持も ち 主ぬし である。

「ほう、孫権もかなりの 人物じんぶつ を送りこんできましたね」

　諸葛亮がつぶやいた。

　魯粛は孫権の 側近そっきん の一人である。 裕福ゆうふく な家に生まれたが、 貧まず しい者にほどこしをしすぎて、たくわえを使いはたしてしまったという。その 気前きまえ の 良よ さは、 孫策そんさく と 周瑜しゅうゆ が兵をあげたとき、 倉くら のひとつをまるごと 軍ぐん 資金しきん として差しだしたほどだ。孫権が兄のあとをついだとき、周瑜が 頼たの みこんで、魯粛は孫家の 臣しん となった。以来、 的確てきかく な 助言じょげん で、孫権の信頼を得ている。

　魯粛は劉琦に 丁寧ていねい な 礼れい をし、孫権からの 手紙てがみ をわたして、みずからもおくやみの 言葉ことば を 述の べた。 儀式ぎしき のようなやりとりが終わると、 宴会えんかい に 移うつ る。

　劉備と劉琦を相手に 酒さけ をくみかわした魯粛は、やはり曹操軍のことを聞きたがった。

「私は 必死ひっし に逃げていただけなので、よくわからないのです」

「しかし、諸葛亮 殿どの の 計略けいりゃく で、 曹仁そうじん や 曹洪そうこう を 破やぶ ったというではありませんか」

「それでしたら、 孔明こうめい に聞いたほうが早いでしょう」

　劉備がかわすと、魯粛は身を 乗の りだしてきた。

「では、諸葛亮殿を 紹介しょうかい してくださいませんか。一度、お目にかかりたいと思っていたのです」

　どうやら魯粛は、最初から諸葛亮と話すのが目的だったようだ。それなら、と劉備は諸葛亮を 呼よ び 寄よ せた。

　魯粛と諸葛亮は、 初対面しょたいめん のあいさつがすむと、ひざをつきあわせて 会談かいだん した。二、三のやりとりで 腹はら のさぐりあいをした後、魯粛が 意い を決したように言った。

「 実じつ は、私と 一緒いっしょ に 柴桑さいそう にいらしていただきたいのです」

　柴桑は孫権が 拠点きょてん としている 街まち である。長江の 要所ようしょ にあって、 軍事的ぐんじてき な 重要性じゅうようせい が高い。諸葛亮は 微笑びしょう をたたえたまま、目の 動うご きで 理由りゆう をたずねた。

「すでにお 察さっ しかもしれませんが、私は曹操との戦いを主張しております」

　柴桑では、曹操の 侵略しんりゃく にどう対応するか、 激はげ しい 議論ぎろん がくりひろげられている。降伏を主張する 文官ぶんかん たちに対して、 武将ぶしょう たちは戦いをのぞんでおり、魯粛はそちらについているという。

「 和わ を 重おも んじる孫権 様さま は、いまのところ、 戦火せんか をさける 方向ほうこう へかたむいているご様子。そこで、諸葛亮殿に 説得せっとく の 手助てだす けをお 願ねが いしたいのです」

「手助け、と言いますと？」

「曹操の兵力は 彼かれ らが 自慢じまん するほどではなく、戦っても 負ま けることはない。そう 説明せつめい してください。 高名こうめい な 臥竜がりゅう 先生せんせい の言葉であれば、孫権様も耳をかたむけると思うのです」

　諸葛亮はかすかに 眉根まゆね をよせた。

「うそをつくのは、気が進みませんね」

「私たちが曹操に降伏すれば、 困こま るのはあなたがたでしょう」

「 必かなら ずしもそうではありません。曹操を倒すための 手段しゅだん はいくつもあります。ですが、魯粛殿が力を 貸か してほしいというのであれば、 協力きょうりょく いたします。もちろん、説得だけでなく、その後の 実戦じっせん についても」

「ありがとうございます。では、 打う ち 合あ わせは 船ふね の中ですることにして、さっそく 出発しゅっぱつ いたしましょう」

　まず、諸葛亮と魯粛の間に、 同盟どうめい 関係かんけい が 成立せいりつ したのであった。

「孫権軍を 利用りよう して曹操を 討う つ。その 策さく を進めてまいります」

　諸葛亮は劉備にそう言って、柴桑へ向かった。



　　　　　２


　柴桑についた諸葛亮を迎えたのは、 張昭ちょうしょう であった。 内政ないせい にひいでた 初老しょろう の 政治家せいじか で、孫家を 支ささ える 功臣こうしん だが、年をとってますます 頑固がんこ になっている。江東を守るためには、強き者にしたがわねばならない、というのが、張昭の主張だ。

　張昭はあいさつを終えると、文官たちが待つ 部屋へや に、諸葛亮をさそった。魯粛が 眉まゆ をひそめて引き止める。

「諸葛亮殿は孫権様に 面会めんかい するためにいらしたのです。 長旅ながたび でお 疲つか れですから、まずは 休やす んでいただくべきでしょう」

「何を言うか。 知恵者ちえもの にとっては、議論は 最高さいこう のもてなしである。 荊州けいしゅう 一の 賢者けんじゃ と 評判ひょうばん の先生には 物足ものた りないかもしれんが、話を聞きたがっている者が 大勢おおぜい 待っている。少しの間、お相手願ってもかまうまい」

　つまり張昭は、諸葛亮にどの 程度ていど の 知識ちしき と 教養きょうよう があるか 試ため してやる、と言っているのである。魯粛はしぶい 表情ひょうじょう になったが、諸葛亮はおだやかに 微笑ほほえ んで受けいれた。

　 案内あんない されて部屋に入ると、ほうっという 感心かんしん の声があがった。白い 上衣じょうい に身を 包つつ んだ諸葛亮の 自信じしん にあふれた 姿すがた に、 一同いちどう は思わずため 息いき をついたのである。

　部屋には、孫権に 仕つか える文官が 二列にれつ にならんですわっていた。その 反対はんたい 側がわ に、諸葛亮の 席せき がある。まるで、 罪人ざいにん を 問と いつめるようだ。

　諸葛亮がすわると同時に、一人が立ちあがって、 質問しつもん を 投な げかけてくる。諸葛亮はよどみなく答え、聞き返した。相手が言葉につまり、顔を赤くして 着席ちゃくせき する。

　諸葛亮の答えに、黄蓋は 満足まんぞく そうにうなずいた。

「どうして約束を守ってくれなかったのですか。あれでは、降伏しろと言っているようなものです。孫権様の怒りはすぐにおさまるでしょうが、戦う気になられるかどうか......」

「わかりました。周瑜殿は水軍の 訓練くんれん で、しばらく 留守るす にしていましたが、今日あたりもどってくるはずです。面会の 段取だんど りをつけましょう」

　魯粛はため息をついて立ちあがった。

「降伏すべきであると思う」

　諸葛亮はかすかに眉を動かした。

「私もそう考えます。戦には 犠牲ぎせい がつきもので、勝ったとしても多くの兵が命を失い、 戦場せんじょう となった地は 傷きず つきます。 将しょう から民まで、みなの命を 保たも つには、降伏するよりほかにありません」

「まことに、そのとおり。 意地いじ や 誇ほこ りのために、 仲間なかま たちを 危険きけん にさらすわけにはいかない」

　二人とも、 本音ほんね で語っているわけではなかった。そして、 互たが いにそれはわかっている。

「降伏にあたっては、二人の人物を 差さ しだせば、曹操も 素直すなお に兵を引くでしょう」

「ちがいます」

「曹操は女好きです。江東一の美女を 贈おく れば、孫権殿の 地位ちい も 保証ほしょう されるでしょう」

　周瑜は怒りをおさえ、口に手をあててかわいた笑いをひびかせた。

「何と！」

　諸葛亮はゆっくりと頭をさげた。

「これは失礼しました。そこまでは知らなかったものですから」

　明らかにうそだったが、周瑜はその点は 追及ついきゅう しなかった。 背筋せすじ にふるえが走っている。諸葛亮は 予想よそう 以上の 陰謀家いんぼうか であった。まさか、 個人的こじんてき な弱点までさぐりだして、 挑発ちょうはつ してくるとは思ってもみなかった。

　諸葛亮が頭を下げたまま言う。

「しかし、そうなると、降伏はむずかしいでしょうか」

　一瞬、周瑜はここで諸葛亮を 斬き ってしまおうかと考えた。手もとに 剣けん があれば、そうしていたかもしれない。

　だが、今は諸葛亮の策に乗っておこう。現時点で強大なのは曹操のほうで、味方は 増ふ やしておくべきだ。

　周瑜は 一足いっそく 飛と びにたずねた。

「劉備軍は役に立つか」

「お役に立ってみせます。私たちが 加くわ われば、十に八の勝ち目が、十に十となるでしょう」

　顔をあげた諸葛亮は、 涼すず しげな表情をしていた。 静しず かな自信は、おだやかな口調にこめられている。

「上手くいくといいがな」

　いずれ、かならず倒す。

　そう思いながら、周瑜は 将来しょうらい の 敵てき と手をむすぶことを決意した。

「 役目やくめ ははたした、と、 江夏こうか で待つ者に伝えるがよい」

　諸葛亮が 無言むごん で 一礼いちれい する。

　わきで見守っていた魯粛は、大きく息を 吐は きだした。秋も 深ふか いというのに、 服ふく が 汗あせ でぐっしょりとぬれていた。それほどに、 緊張きんちょう する 会談かいだん だったのである。


　 翌日よくじつ 、孫権の御前で会議がひらかれた。

　孫権の 左側ひだりがわ には、張昭を 筆頭ひっとう とする文官たちがならんでいる。 右側みぎがわ は武官である。 先頭せんとう に周瑜、それから 程普ていふ 、黄蓋、 周泰しゅうたい 、 太史慈たいしじ 、さらに、 呂蒙りょもう 、 甘寧かんねい といった将がつづく。

　甘寧はもともと 長江ちょうこう や 付近ふきん の 湖みずうみ で 水賊すいぞく をしていた男だ。 弱よわ きを 助たす け、強きをくじく 好漢こうかん で、民には人気があった。 一時期いちじき 、劉表の 家臣かしん である 黄祖こうそ に仕えていたが、いくら 戦功せんこう をあげてもみとめられなかったので、孫権の 陣営じんえい に 寝返ねがえ った。 先年せんねん 、この甘寧の 活躍かつやく で、孫権は黄祖を 破やぶ り、父孫堅の 仇かたき を 討う つことに 成功せいこう している。

　孫権は、 自慢じまん の文官と武官の顔をひとりずつ 確認かくにん して言った。

「あらためて 問と う。曹操に降伏すべきか、 否いな か」

　最初に 発言はつげん したのは、 年長ねんちょう の張昭である。それまでの主張をくりかえした後、張昭はつけくわえた。

「諸葛亮はわれわれの兵力を利用して、みずからの目的を 達成たっせい しようとしています。かの者の 口車くちぐるま に乗ってはいけません」

　これに対抗して、黄蓋が主戦論を 述の べた。

　孫権は 双方そうほう の話をうなずきながら聞き、周瑜に視線を向けた。周瑜がどちらにつくのか、 一同いちどう に緊張が走る。

「 先代せんだい はおっしゃいました。国を守ることにかけては、孫権様のほうがすぐれていると。その言葉を、 証明しょうめい するときではないかと考えます」

「戦うべきだ、というのか。敵は百万をこえる大軍なのだぞ」

「ですが、曹操軍に 不利ふり な点は五つもございます。その点をつけば、必ず勝利をつかむことができるでしょう」

「不利な点とは？」

「まず、北西に 馬騰ばとう という敵をかかえながら、 長期ちょうき の南征に出ていること。次に、 経験けいけん のない水戦にのぞもうとしていること。第三に、 兵糧ひょうろう やまぐさが 足た りないこと。第四に、兵士たちがなれない 気候きこう で 体調たいちょう をくずしていること。第五に、袁紹軍や劉表軍にいた兵が多く、 寄よ せ集めであること」

　周瑜は 両手りょうて を広げて、一同を見わたした。

「 逆ぎゃく に言えば、私たちはそのすべてを 裏返うらがえ した 利点りてん を持っています。 背後はいご に敵はなく、兵糧は 豊富ほうふ で、兵士たちは 戦意せんい にあふれています。長江の流れを 盾たて に、 得意とくい とする水戦に持ちこんで、 団結力だんけつりょく を 武器ぶき に戦えば、勝利は 約束やくそく されたようなもの。 迷まよ いはもう 捨す ててください。 父祖ふそ の地を守るために、いざ戦おうではありませんか」

　周瑜の 音楽的おんがくてき な 美声びせい がひびいて、みなを 興奮こうふん にみちびいた。

「おれはやるぞ。 先陣せんじん をきらせてくれ」

　太史慈が 叫さけ ぶと、黄蓋が負けじと声をはりあげた。

「若い者に任せておれるか。年の 功こう を見せてやる」

　武官たちは口々に、自分を使うよう 訴うった える。文官のなかにも、こぶしをかためて気合いを入れる者がいた。張昭は 苦にが い表情で、孫権を見つめている。

　その孫権は目を 閉と じて考えこんでいたが、やおら立ちあがった。

「曹操は漢を乗っとって、自分の手で天下を治めようとしている。かの 逆賊ぎゃくぞく に、父と兄が切りひらいた江東の地をわたすわけにはいかぬ。天は曹操に対抗するため、孫家に周瑜をつかわしてくれた。私は周瑜を信じて、曹操と戦うぞ！」

　孫権は 腰こし の剣を抜き、 天井てんじょう に向かってかかげると、 一気いっき にふりおろした。目の前の 机つくえ の 角かど が 欠か けて、 木片もくへん が飛びちる。

「今後、降伏を口にする者は、この机と同じ目にあうと 心得こころえ よ！」

　一瞬の沈黙のあと、 議場ぎじょう は 熱狂ねっきょう につつまれた。

「この手に勝利を！　孫家に 栄さか えあれ！」

　黄蓋のかけ声に、みなが声をあわせる。

　孫権は周瑜に先ほどの剣をさずけ、全軍の 大だい 都督ととく すなわち総司令官に 任命にんめい した。五万の兵をあたえて、曹操軍を打ち破るよう命じる。

「わが 全身ぜんしん 全霊ぜんれい をかけて、 使命しめい をはたしてまいります」

　周瑜は力強く 宣言せんげん した。

　必勝の 誓ちか いを胸に、孫権軍は動き出す。


　周瑜の本隊が 出陣しゅつじん した後、柴桑では小さなもめ 事ごと が起こっていた。

　孫権の妹の 孫尚香そんしょうこう が、自分も戦う、と言って兄を困らせていたのである。

　孫尚香はこの年二十六歳、 黙だま っていればそれなりに美しい、というのが孫権の 評ひょう で、男 勝まさ りの性格は変わっていない。花よりも 弓ゆみ を 愛あい し、腰にはつねに弓をたずさえている。長く 伸の びた 髪かみ は 無造作むぞうさ にたばね、後ろに流していた。

　孫権が 当主とうしゅ となって以来、戦に 参加さんか することは 許ゆる されていないため、 侍女じじょ の 一団いちだん とともに 狩か りをしたり、剣や弓の 訓練くんれん をしたりしている。こっそりと 山賊さんぞく 退治たいじ に出かけたこともあった。 見事みごと に 戦果せんか をあげたのだが、孫権からはひどく怒られ、一ヶ月もの間、部屋に閉じこめられた。

　それからは、戦の練習をするのがせいぜいだ。おかげで、二十人を数える侍女たちはみな武器が使えるようになり、孫尚香自身は「 弓腰姫きゅうようき 」と 呼よ ばれるようになった。孫家に仕える者たちからは 半なか ばあきれられており、まだ 結婚けっこん はしていない。

「国が生き残れるかどうかの大事な戦なんでしょ。戦える者は一人でも多いほうがいいに決まっているわ」

　つめよる孫尚香に、孫権は眉をひそめた。

「大事な戦だから、足手まといになる者は連れていけないのだ」

「何でよ。私は剣も弓も使える。悪いやつをやっつけたこともある。足手まといになるはずなんかないわ」

　孫尚香は 軽かる くあごをあげて、胸をはった。孫権は頭をかかえたい気持ちだったが、何とかこらえて説得する。

「おまえが 怪我けが をすれば、母上が 悲かな しむだろう」

「だれが怪我をしても、悲しむ人はいるわ。私はみんなの役に立ちたいの」

「だったら、ここで毎日、出陣した者の無事を 祈いの っていろ。私もそうするつもりだ」

「そうよ」

　ふいに孫尚香は 矛先ほこさき を変えた。

「何でお兄様は出陣しないの？　戦のときは先頭に立つのが孫家の当主じゃなくって？　亡くなったお父様も、 策さく 兄様にいさま もそうだったわ」

　だから早死にしたのだ、とは孫権は言わなかった。それは言ってはならぬことで、孫権が今まで 陣頭じんとう に立たなかった 理由りゆう でもある。三十までは無理はしない、孫権はそう誓いを立てている。

　黙っていると、孫尚香はたたみかけてきた。

「戦がこわいのね。お兄様の 臆病者おくびょうもの ！」

　孫権はため息をついた。

「私は父や兄のように強くはない。戦場に私がいれば、私を守るために兵をさかねばならず、都督に 神経しんけい を使わせることにもなる。作戦を決めるときにも、いちいち 許可きょか をもとめることになろう。私がいないほうが、都督は戦に集中できるのだ」

「ちがう。お兄様が強いこと、私は知ってる。当主がいれば、 士気しき があがるわ。守るべき人がいれば、力を出せるもの」

「それもひとつの考えだ。しかし私は、 将兵しょうへい が 安心あんしん して戦える 環境かんきょう をつくり、帰ってくる場所をしっかりと守ることも、大切な仕事だと思っている。それは、私がすべき仕事だ。都督が戦の指揮をすべきであるようにな」

　教えさとすように言うと、孫尚香は目を 伏ふ せた。さらに孫権はつづける。

「人にはそれぞれ、はたすべき役割があるのだ」

　孫尚香はうつむいたままで言った。

「......私の 役割やくわり って何だろう」

　孫権は 即答そくとう できなかった。孫尚香に役割をあたえなかったのは、当主たる自分自身である。曹操の一族との 縁えん 組ぐ みを考えたこともあったが、本人の反対もあって、 実現じつげん はしなかった。

「今は、不安を感じている人々を 勇気ゆうき づけるのが、おまえの役割だ」

　とってつけたような答えになったが、孫尚香はうなずいた。

「わかったわ」

　孫権はほっとしたが、去っていく妹のつぶやきを聞きとって、 憮然ぶぜん とした。

「城を抜けだして、 義賊ぎぞく にでもなろうかなあ」

　孫家の 姫ひめ は、そう言って、ため息をついていたのである。

　周瑜ひきいる孫権軍は、 夏口かこう に 布陣ふじん した。江陵の曹操軍の 進路しんろ をふさぎ、 対決たいけつ の 姿勢しせい をしめす。

　諸葛亮は相談役として、孫権軍に 同行どうこう していた。江夏の劉備は、諸葛亮がなかなか帰ってこないので、気が気ではない。

「そろそろ、劉備からの使者が来るころだな」

　周瑜はつぶやくと、諸葛亮を呼んで、ひとつの頼み事をした。

「曹操軍の 輸送路ゆそうろ をおそえ、と言うのですね」

「ああ、正面から戦うのは、われわれが引き受ける。劉備殿の軍には、 側面そくめん からの 援護えんご をお願いしたい。まずは輸送路の 偵察ていさつ に行ってもらえないだろうか」

「かしこまりました」

　諸葛亮が出て行くと、魯粛が周瑜に 意図いと をたずねた。周瑜が声をひそめて答える。

「今から劉備を呼び出すので、少し 留守るす にしてもらおうと思ってな。 妙みょう なたくらみをされては困るから、 可能かのう なかぎり、 両者りょうしゃ を引きはなしておきたいのだ」

「劉備軍に輸送路を 絶た つことができますでしょうか」

「無理だな。曹操は 抜ぬ け 目め のない男だ。 官渡かんと の戦いでは、敵の 兵糧ひょうろう 庫こ を 焼や き 討う ちにして、勝利を決めている。 自軍じぐん の輸送路は、 前線ぜんせん 以上に 警戒けいかい しているだろう」

　もし、曹操軍が諸葛亮を見つけて殺してくれたら、将来の 悩なや みの 種たね がへる。周瑜はそう言ったものの、その可能性はかぎりなく低いと考えていた。諸葛亮はそんな 失策しっさく は 犯おか さないだろう。

　ちょうどそのとき、劉備の使者として 糜竺びじく がやってきた。兵糧を差し入れにきたというが、様子をさぐりに来たに決まっている。周瑜はにこやかに応対して、劉備本人の 訪問ほうもん をもとめた。

「協力して曹操に当たるのですから、一度ごあいさつしておきたいのです。本来であれば、私が江夏にうかがうところですが、本陣をはなれるわけにはいきません。劉備殿に来ていただけないでしょうか」

　糜竺は 承知しょうち したが、諸葛亮を連れて帰りたいと願った。周瑜はさも 残念ざんねん そうに言った。

「 申もう し 訳わけ ございません。諸葛亮殿には私と一緒に策を立ててもらっています。今は敵陣を偵察に出かけておりますので、ここにはおりません」

　糜竺は仕方なく、いったん江夏に戻ることにした。

　その帰りの船に、 小舟こぶね が近づいた。乗っているのはただ一人、諸葛亮である。

　諸葛亮は周瑜の策を読んで、途中で引き返してきたのだった。

「私はしばらく 戻もど れません。劉備様に伝えてください。周瑜と会うときは、必ず関羽殿か趙雲殿を 護衛ごえい に連れて行くようにと」

　 川面かわも をわたる風にのって、諸葛亮の声は糜竺にとどいた。糜竺が一礼すると、諸葛亮は自分で船をこいで去っていった。

　二日後、劉備は夏口の陣をおとずれた。ただ 一隻いっせき の船で劉備があらわれたと聞き、周瑜は喜んだが、その背後に立つ人物を見て、喜びは 霧きり と 散ち った。

　 青龍せいりゅう 偃月刀えんげつとう を背負い、美しいあごひげをなびかせた 長身ちょうしん の男は、世に知られた関羽その人である。背筋を伸ばしてすっくと立ち、 辺あた りに 油断ゆだん なく目を 配くば っている。

「さすが、 顔良がんりょう 、 文醜ぶんしゅう を斬り、曹操にもとめられた男だ。どこにもすきがない」

　周瑜は会談の場に兵を伏せ、劉備を殺すかとらえるかしようと思っていたのだが、関羽を見てあきらめた。劉備とはなごやかに話し、同盟の誓いをむすんで別れる。

「孔明のおかげで、孫権との協力 態勢たいせい がととのったようだな。会えなかったのは残念だが、曹操に勝って 再会さいかい すればよいか」

　何も知らない劉備は、 上機嫌じょうきげん で船にゆられている。そこへ、またしても小舟が近づいた。舟をつけて、はしごをのぼってきたのは諸葛亮である。

「劉備様、無事でようございました」

　諸葛亮が周瑜の陰謀を説明すると、劉備はおどろいて言った。

「そんな 危険きけん なことがあったとは。周瑜は信用できないのだな。では、孔明も早く帰ってくるがよい」

「いえ、私は 大丈夫だいじょうぶ です。周瑜との 知恵ちえ 比くら べはつづきますが、曹操を倒すまでは、互いに無理はいたしません。それより、劉備様には曹操をとらえるための準備をしていただきたいと思います」

　いつ戦になってもいいように、水軍と 騎馬きば 隊たい をきたえておくこと。十一月 下旬げじゅん になったら、趙雲を足の速い船に乗せて、長江の 南岸なんがん に 待機たいき させておくこと。その二点を、諸葛亮は頼んだ。

「もちろんかまわないが、どういう意味があるのだ？」

「いずれわかるでしょう」

　諸葛亮は微笑を浮かべた。

「私は 東南とうなん の風が 吹ふ いたら帰ります」

　そう言い残すと、また小舟に乗って、夏口に戻っていく。

「東南の風......」

　劉備はつぶやいた。それはまるで何かの 呪文じゅもん のように聞こえた。

　だが、諸葛亮は必ず勝って帰ってくる。信じる気持ちには、一点の 曇くも りもないのだった。


　夏口に着いた諸葛亮は、周瑜の前にあらわれて、曹操軍の様子を見てきたかのように語った。

「曹操軍の警戒はきびしく、輸送隊をおそうのは本陣を破るよりむずかしいくらいです。まずは水戦で相手の 出鼻でばな をくじき、そのあとで 計略けいりゃく を考えてはいかがでしょう」

「そうだな、 貴公きこう の言うとおりにしよう。疲れただろうから、ゆっくりと 見物けんぶつ していてもらいたい」

　周瑜はにこやかに 労ろう をねぎらったが、内心では 荒あ れる思いをかろうじておさえていた。 監視かんし 役やく からの報告で、諸葛亮が 実際じっさい には偵察に行かず、糜竺に会っていたことを知ったばかりだ。諸葛亮はこちらの意図をすべてお見通しだったわけだ。

「やつのことばかり考えているわけにはいかない。 当面とうめん の敵に集中しなければ」

　周瑜は自分に言い聞かせた。

　そこにちょうど、曹操軍の 先発せんぱつ 隊たい が近づいているとの報が入った。周瑜は気をまぎらわせるため、みずから出陣することにした。先鋒には甘寧を 指名しめい する。

　長江は 大河たいが である。 河口かこう から 遠とお くはなれたこの地でも、水はとうとうと流れており、 大だい 船団せんだん 同士どうし がすれちがえるほどの川はばがある。

　曹操軍が見えてくると、周瑜は甘寧に 突撃とつげき を命じた。同時に、周泰と太史慈に左右からまわりこんで 攻撃こうげき をかけるよう指示する。

　甘寧は 戦船いくさぶね の 舳先へさき に立って弓をかまえると、敵船団にねらいをつけて、ひょうと 放はな った。 矢や はうなりをあげてまっすぐに飛び、敵将の胸に 突つ き 刺さ さる。

　ばたりと 仰向あおむ けに倒れたのは、蔡瑁の弟の 蔡勲さいくん であった。

「よし、 手本てほん は見せてやったぞ。 乱射らんしゃ しながら突きすすめ！」

　甘寧の 号令ごうれい にあわせて、船が 速度そくど をあげる。 弩いしゆみ が放たれて、曹操軍におそいかかった。

　曹操軍の兵士たちは、 北方ほっぽう 出身しゅっしん の者が多く、まだ 水上すいじょう の戦いになれていない。矢をさけるどころか、立っていることもできないのだ。

　そこへ、正面から甘寧、右から周泰、左から太史慈の指揮する船隊が 突進とっしん した。はなれている間は弓矢で攻撃し、近づいたら船をぶつけて乗り 移うつ る。

「まわりは 一寸いっすん 先さき も見えない 深ふか い霧です。曹操もこんなときに 出撃しゅつげき してはこないでしょう。わなの可能性を考えますから。私たちは朝日が 昇のぼ るまで、酒でも飲んでゆっくり待ちましょう」

　魯粛は 半信はんしん 半疑はんぎ である。

「 伏兵ふくへい の危険がある。霧が 晴は れるまで、軽はずみな行動はさけよ。ただ、陣に突っこまれてはならぬ。陣の前に 弓隊ゆみたい をならべ、船の灯りを目標に矢を 射い るのだ」

「お 約束やくそく の矢を調達してまいりました。十万本はくだらないはずです。どうぞお受け取りください」

「あ、ああ。ご苦労だった」

　諸葛亮はだれも思いつかない計略を 用もち いて、曹操軍から矢を手に入れたのだ。

「くっ、諸葛亮め。わが 智謀ちぼう の上をいくというのか」

　周瑜はこぶしをにぎりしめた。 爪つめ が手のひらに食いこんで、血がにじんでいた。



　　　　　３


　その朝、水軍指揮官の于禁は、 胃い をおさえながら曹操の部屋に向かっていた。諸葛亮の計略にまんまと引っかかり、たくさんの矢を敵に 贈おく ってしまったのだ。曹操は怒っているにちがいない。

「仕方あるまい。こちらの 判断はんだん に 誤あやま りはなかった。敵が 一枚いちまい 上手うわて だっただけのことだ。そもそも、子どものいたずらのような計略をしかけてくるのは、敵が戦力の 差さ をみとめている 証拠しょうこ だ。あわてずに訓練を積んで、決戦にそなえよ」

　于禁は 感心かんしん して引き下がった。やはり曹操は 器うつわ が大きい。

　しかし、曹操も内心では、やられっぱなしでいるのを 不快ふかい に思っていた。そこへ、軍師の 賈詡かく がやってきて、策を 提案ていあん した。

　処刑された蔡瑁の 一族いちぞく に、 蔡中さいちゅう という男がいる。彼を孫権軍に送りこんで、情報をさぐらせてはどうかというのだ。

「蔡瑁が殺されたから 寝返ねがえ った、と言わせれば、周瑜も信用するでしょう」

「なるほど。 試ため してみる 価値かち はあるな」

　曹操はすぐに蔡中を呼んだ。

　蔡中は 太ふと り 気味ぎみ の中年の男で、蔡瑁ほどの指揮 能力のうりょく も 欲望よくぼう もない 凡人ぼんじん であった。はじめて曹操の前に出て、その 迫力はくりょく にふるえている。

「わ、私は何をすればいいのですか」

「五百人ほどの兵士を連れて、孫権軍に 偽にせ の降伏をしてもらう。何か 妙みょう な動きがあったら人を送って報告せよ。決して裏切ってはならんぞ」

「裏切るだなんてとんでもない。荊州には家族がおります。何でも曹操様の言うとおりにいたしますから、どうかひどいめにあわせたりしないでください」

　両手をすりあわせて頭をさげる姿は、曹操をあきれさせた。これほど 情なさ けない男が役に立つだろうかと思うが、そのほうがかえって周瑜も 油断ゆだん するかもしれない。それに、 敵方てきがた にもぐりこむ 任務にんむ は 危険きけん であり、名のある武将を使いたくはなかった。

　かくして、蔡中がひきいる五百の兵は、十隻の船に乗って、曹操の陣を出た。夜の闇にまぎれて 航行こうこう し、夏口に 達たっ する。

　そのとき、周瑜は諸葛亮と作戦を話しあっていた。

「あれほど見事な計略を成功させた貴公だ。すでに決戦で用いる策を考えているだろう。魯粛には、必勝の策があると言ったそうではないか」

「私の策は、 珍めずら しいものではありません。おそらく、周瑜殿の胸のうちにあるものと同じではないかと」

　二人の天才はしばし見つめあった。互いを 値踏ねぶ みし、内心をうかがおうとする視線には、 敬意けいい と 敵意てきい が 等ひと しく 含ふく まれている。

「では、私の策を聞いてもらおうか」

「お待ちください」

　諸葛亮が余裕の笑みを浮かべる。

「それぞれのてのひらに策を書いて、同時に見せあうのはいかがでしょう」

　ややあって、周瑜は諸葛亮のそれを 鏡かがみ に 映うつ したような笑みを返した。

「おもしろい」

　周瑜は 筆ふで とすずりをとり、自分のてのひらに文字を書きつけた。そして、それをかくし、諸葛亮に筆をわたす。諸葛亮も、同じように書きしるした。

「では」

　周瑜と諸葛亮は互いにむかって、左のてのひらを突きだした。同時に、二人が笑い出す。

「火」

　二人のてのひらには、まったく同じ一字が書かれていたのだ。

「これで決まった。後は策の 効果こうか が 最大限さいだいげん に 発揮はっき されるよう、 細工さいく を進めるとするか」

「ええ、私のほうでもいくつか 布石ふせき を打っております。必ずや、曹操をとらえてみせましょう」

　ふと、周瑜は笑いをおさめた。

「とらえる、と言うと、 追撃ついげき は劉備軍が行ってくれるのかな？」

「そのつもりでおります。 江夏こうか は背後をふさぐのに 都合つごう がいい 位置いち ですので」

「わかった。期待しよう」

　周瑜は 長なが めにまばたきして、瞳にひらめいた危険な光を 消け した。追撃の途中で諸葛亮をおそう策を 検討けんとう したのだ。諸葛亮は危険な 存在そんざい であり、いずれ 害がい になる前に、 始末しまつ せねばならない。協力するのは、曹操軍を破るまでだ。

　諸葛亮が帰るのと入れかわりに、蔡中が降伏してきたとの報告が入った。周瑜が 招まね き入れると、蔡中はおどおどしながら 口上こうじょう を述べた。

「蔡瑁は罪なくして殺されました。曹操はわが一族の 仇かたき であります。必ずお役に立ちますので、どうか孫権様の陣営に加えてください」

　周瑜は笑みを浮かべて、蔡中の手をとった。

「承知した。今夜はゆっくり休むがよい。明日にでも、曹操軍の情報を話してもらおう。その後、 甘寧かんねい の下で 先鋒せんぽう 隊を任せる。曹操の首をとってこい」

「かしこまりました」

　蔡中がほっとした様子で 退出たいしゅつ する。

　するどい視線でそれを見送った周瑜は、甘寧を呼びだした。

「あの蔡中という男、こちらの様子をさぐるために、曹操が送りこんできた 密偵みってい だろう」

「では、斬ろうか」

　先走る甘寧を、周瑜は苦笑して 制せい した。

「あせることはない。しばらく 泳およ がせて、できればうその情報を曹操に伝えさせたい」

「では、おれはやつが妙なまねをしないよう、見張っていればいいんだな」

　甘寧は考えさせれば、なかなか頭の 回転かいてん が 速はや い。周瑜は 満足まんぞく げにうなずいた。

「ああ、頼む。害になるようだったら、斬ってもかまわない」

「わかった」

　甘寧が去ったとき、もう 夜半やはん を 過す ぎていた。周瑜はいつも 遅おそ くまで 書類しょるい を 処理しょり したり、作戦を考えたりしている。それでいて、朝も早い。

　翌日、蔡中から曹操軍の 装備そうび について情報を得たあと、周瑜は干した魚とかゆで朝食をとっていた。そこへ、あわてた様子の魯粛がやってきた。

「周瑜殿、蔡中を信用してはなりませんぞ。あれは曹操の計略にちがいありません」

「どうしてそう思うのだ」

「本当に降伏するなら、一族を連れてくるはずです。蔡瑁の処刑が理由というなら、なおさらでしょう」

　周瑜はうなずき、魯粛に策を説明した。魯粛がむっとして口をとがらせる。

「そういうことでしたら、早めに言ってくだされば安心できましたのに」

「敵をあざむくには、まず味方からだ。 他ほか の者には言わないでくれ」

「もちろん言いませんが、きっと諸葛亮殿は気づいてますよ」

　魯粛のささやかな 逆襲ぎゃくしゅう に、周瑜は一瞬、眉をひそめた。が、 何事なにごと もなかったかのように 話題わだい を進める。

「実はこちらからも、だれかをにせの降伏で曹操軍に送りこみたいと考えている。 火計かけい をより効果的に実行するためだ。だから、蔡中のような臆病者では意味がない」

「ですが、 勇敢ゆうかん な武将がいきなり降伏しても信じないでしょう」

「そう、そこだ。曹操に信じさせるには、どうすればよいか。家族を連れて行けというのはむずかしい。ばれたら、まちがいなく殺されるからな」

「重要なのは降伏の理由でしょうね」

　周瑜は腕を組んで、 天井てんじょう を見上げた。

「苦肉の計しかないか......」

　周瑜の口調があまりにもしずんでいたので、魯粛はおそるおそるたずねた。

「どういう計略でしょう」

「何か理由をつけて、実行者を 半殺はんごろ しになるまで 痛いた めつけるのだ。そこで逃げだして降伏すれば、敵も信用するだろう」

　魯粛が眉をひそめる。

「それはつらいですね。実行者だけでなく、痛めつけるほうも」

「そちらは私がやるからよい」

　周瑜はきっぱりと言った。自分で苦しみを引き受けない者に、人に痛みをおしつける 資格しかく はない。 覚悟かくご を決めた横顔は、いつにもまして美しかった。

「はたして、だれに任せるべきか......」

　周瑜は目を閉じて考えこんだ。魯粛がそっと退出するのにも、気づかないほど 悩なや みは深かった。



　　　　　４


「 邪魔じゃま するぞ」

　太い声に、周瑜は目をあげた。ろうそくの光が消えかかっている。見入っていた作戦図をしまい、ろうそくをとりかえて、周瑜は 来客らいきゃく に応対した。

「こんな夜ふけにいかがされましたか、黄蓋殿」

「なに、都督殿が困っていると聞いてな」

　だれか、よけいなことを言った者がいるらしい。周瑜は思わずひたいに手をやった。

　黄蓋は気軽な調子で志願した。

「計略の実行者をさがしているそうじゃないか。おれにやらせてくれ」

「曹操を信用させるには、 肉体にくたい 的てき にも 精神せいしん 的てき にも傷つけねばなりません。それでも、ばれて殺される可能性は少なくないのですよ」

「心配はいらん。おれは痛みには強いんだ。今までさんざん 鈍感どんかん だと言われてきたからな」

　確かに、黄蓋の 丈夫じょうぶ さと 我慢がまん 強さは飛び抜けている。 防御ぼうぎょ がへたなので、戦のたびに 生傷なまきず をつくるのだが、まったく痛がらないし、すぐに 治なお ってしまう。

　それでも、周瑜がうなずかないので、黄蓋は表情をあらためた。めずらしく 真剣しんけん な顔つきで語り出す。

「おれは 先々代せんせんだい から孫家に仕え、山よりも高く海よりも深い 恩おん を受けてきた。もちろん、それなりの働きをしてきたという 自負じふ はあるが、まだまだ恩返しには足りぬ。孫家が 危機きき にある今こそ、おれみたいな 年寄としよ りの 出番でばん だ。 孫堅そんけん 様と 孫策そんさく 様が道半ばでたおれ、 次代じだい に引きついだ 夢ゆめ を、おれがいしずえとなって、 未来みらい へゆずりわたしたい」

　周瑜はため息をついた。

「だれにも 真相しんそう は話せません。つらいですよ」

「のぞむところだ」

　黄蓋の決心がゆらがないのを見てとって、周瑜は 静しず かに頭をさげた。

「お願いします」

「おお、まかせておけ」

　それから二人は、 念入ねんい りに打ち合わせを行った。味方をもだます 大芝居おおしばい である。 細こま かい検討は朝までつづいた。

　少しだけ 眠ねむ って、周瑜は 昼前ひるまえ に 軍議ぐんぎ をひらいた。諸将を前にして状況を説明する。

「曹操軍は大軍で、 陣列じんれつ は 長大ちょうだい だ。こちらから攻めこんでも、 簡単かんたん には破れまい。しかし、蔡中殿の情報によれば、兵糧が少ないそうだ。そこで、われわれは夏口の陣を守って、 持久戦じきゅうせん に持ちこもうと思う。こちらの兵糧は三カ月はもつ。それだけ持ちこたえれば、敵もあきらめるだろう」

　諸将は顔を見合わせた。 太史慈たいしじ が 異論いろん をとなえる。

「それは少し 消極的しょうきょくてき すぎませんか」

　そこで、黄蓋が太史慈をおしのけて進み出た。

「たかだか三カ月で敵が帰るなら、最初から 苦労くろう はせんわ。おれたちが集まったのは、敵を倒すためで、陣にひきこもるためじゃない。 今月中こんげつちゅう に敵陣を落として、曹操の首をとれないようでは、降伏したほうがました」

　周瑜が落ちついて言い返す。

「気合いだけでは戦には勝てません。敵味方の戦力差を見すえ、 的確てきかく な作戦をたててはじめて、強大な敵を打ち破ることができるのです」

「 若造わかぞう が 生意気なまいき なことを言うな。おれは孫堅様の時代から、戦場を行き来してきたのだ。本ばかり読んでいたような 軟弱者なんじゃくもの とは経験がちがう。そのおれが言うのだ。戦は攻めなければ勝てん！」

　歩みよる黄蓋を、周瑜は 冷笑れいしょう で迎えた。

「経験からは何も学ばなかったようですね」

「何を !? 」

　黄蓋はいきなり周瑜になぐりかかった。右のこぶしがまともに周瑜のほおをとらえる。周瑜は後ろに 吹ふ っとばされて、 尻餅しりもち をついた。

「おやめください」

　太史慈と 周泰しゅうたい が黄蓋に駆けよって、体をおさえつける。

「こら、はなせ。 一発いっぱつ じゃ、とても気がすまん」

　周瑜は赤く 腫は れあがったほおをおさえながら、立ちあがった。くちびるから血がしたたっている。

「黄蓋よ。軍議の席で、 上官じょうかん たる都督になぐりかかるとは、 宿将しゅくしょう にふさわしからぬふるまいだな。作戦に口を出す 資格しかく などないことを、みずから 証明しょうめい したか」

　周瑜の冷たい口調に、一同は 凍こお りついた。ただ黄蓋だけが、怒りをぶちまける。

「若くして 出世しゅっせ したからといって、 調子ちょうし に乗りおって。おれがはじめて戦場に出たとき、おまえなどまだ、かけらも存在していなかっただろう」

「これ以上、しゃべらせては士気にかかわるな。黄蓋は都督に対する 反逆はんぎゃく の罪により、処刑する」

　周瑜は剣を抜いて、黄蓋に近づいた。

「お待ちください」

　周瑜の前に立ちふさがったのは、 程普ていふ である。黄蓋とずっと一緒に戦ってきた 相棒あいぼう だ。

「黄蓋は確かに罪を犯しました。しかし、それまでに多くの 手柄てがら を立てているのも 事実じじつ です。私がよく言い聞かせますから、どうか命だけはお助けください」

「それではしめしがつかぬ」

　周瑜が程普をおしのけようとすると、見守っていた諸将がいっせいに声をあげた。

「なにとぞ、お許しください」

「今は 仲間なかま 割わ れをしている場合ではありません」

　太史慈と周泰が、むりやり黄蓋の頭を下げさせる。

　周瑜は許しをこう諸将を見回して、軽くため息をついた。

　 不眠ふみん 不休ふきゅう で作業を急いだので、全体が三日ほどで 完成かんせい した。数千の船が 一体化いったいか されて、まるで河に浮かぶ巨大な 城しろ のようである。

　曹操は満足げに、陸と河につらなる陣を 視察しさつ した。

　風はほとんどなく、波はおだやかである。 合体がったい した船団は安定しており、馬が歩くとかわいた音を立てるが、ゆれはまったくない。目を閉じると、船の上であることを 忘わす れてしまいそうだ。兵士たちも陸上と同じように戦えるだろう。

　各船には 弩いしゆみ がそなえつけられ、弓兵も配置されている。矢は一部敵に奪われてしまったが、それでも充分にあった。

　日が 暮く れると月が出て、 銀色ぎんいろ の光で夜を 照て らしだした。長江の 水面みなも は 絹きぬ のようになめらかで、 盆ぼん のような月を映している。

　曹操は気持ちの高ぶりをおさえられなかった。

「 戦勝せんしょう の 前祝まえいわ いだ。今夜は宴にしよう。兵士たちにも酒を 配くば れ」

　かくて、月をながめながら宴会がひらかれた。

　曹操が武将たちを集めて 演説えんぜつ する。

「私は 黄巾賊こうきんぞく 討伐とうばつ に参加してこの方、漢朝のために賊を退治し、悪人を 打倒だとう して、天下統一のために戦ってきた。残る強敵は江東の 孫権そんけん だけだ。この戦に勝てば、ほかの 小物こもの たちは先を争って降伏するだろう。百万の兵士たちと、 知勇ちゆう かねそなえた将軍を 有ゆう するわれらに、敗北の二文字はない。いざ、統一のために、最後の戦にのぞもうではないか」

　いっせいに拍手が鳴りひびき、ときの声があがった。

　その夜は、曹操も武将たちも勝利を 確信かくしん して、酒を楽しんだのであった。

　翌日、曹操は出陣を命じた。船団は長江の北岸に 沿そ って、東へ向かう。

　水軍をひきいるのは、 于禁うきん 、 張郃ちょうこう 、 夏侯淵かこうえん といった武将たち、陸軍は曹操の本隊で、 許褚きょちょ と 張遼ちょうりょう が護衛につく。 夏侯惇かこうとん と 曹洪そうこう は、 遊撃ゆうげき 部隊をひきいて、水陸の 連絡れんらく にあたった。最強の布陣で、曹操は孫権軍を 叩たた くつもりであった。


「そうか、やっと動き出したか」

　 周瑜しゅうゆ は 優美ゆうび な仕草で立ちあがった。 矢継やつ ぎ 早ばや に命令を送り、出陣の方針を伝える。

　武将たちが集まってくると、周瑜は一同を見わたして口を開いた。

「持久戦と言ったのは、敵をあざむくための 方便ほうべん だ。やつらが出てくる以上、全力で迎え撃つ。長江はわれらの河、地の利がどちらにあるかは明らかだ。 慣な れ 親した しんだ水上の戦いで、われらの力を見せてやろうぞ。そして、 父祖ふそ の地を守るのだ」

　落ちついた口調だが、力がこもっていた。

　武将たちは必勝を誓って、持ち場に走る。よく訓練された孫権軍の船団はすぐに動き出した。隊列を組んで、南西へ向かう。

　周瑜の乗る 指揮船しきせん には、諸葛亮も同乗している。二人は三階建ての船の一番上で、冷たい風に吹かれていた。

「貴公は、十度戦って十度勝てると言ったな。今もそれは変わらぬか」

「ええ。計略はすべて成功しています。負ける 要素ようそ はありません」

「大した自信だ」

　周瑜は 前方ぜんぽう を指さした。

「戦場はおそらく、 赤壁せきへき の辺りになる」

　諸葛亮が微笑して応じる。

「その地名は、きっと 歴史れきし に残るでしょう」

　そのとき、曹操軍の船が鎖でつながれているという 報ほう が入ってきた。

「曹操も意外に間が抜けているな。火計の 威力いりょく が高まるだけではないか」

　周瑜はつぶやいた後、はっとして諸葛亮を見た。

「これも貴公の策か？」

「いえ、私ではありません。古くからの友人が、 一肌ひとはだ 脱ぬ いでくれたようです」

　これは龐統のことである。周瑜は知らなかったが、あえてたずねはしなかった。

　ついで、もう一つ情報が入ってきた。曹操軍は、北岸に沿うようにして、移動しているという。

　これを聞くと、周瑜は空を見上げて顔をしかめた。

　にらんでいるのは風である。冬の風は、北か西から吹く。曹操はそれを計算に入れて、 風上かざかみ に立つよう船団を進めているのだろう。この配置だと、先頭の船に火をつけても回りが遅く、火計の効果は弱くなる。風がやんだときをねらうか、それとも敵船団の中心に火を放つか。

「風のことですか？」

　周瑜の心を読んだかのように、諸葛亮がたずねた。周瑜は風の 行方ゆくえ をながめながら答える。

「そうだ。少々、よけいな 手間てま がかかりそうだな」

「ご安心ください。私が東南の風を吹かせて差し上げましょう」

　周瑜は諸葛亮をふりかえった。

「東南の風を吹かせるだと？」

「ええ、戦場に着いたら、陸地に簡単な 祭壇さいだん を立ててください。そこで、天に祈ります。ほどなくして、風向きが変わるでしょう」

「貴公が 祈祷師きとうし だったとはな」

　周瑜は諸葛亮に疑いのまなざしを向けた。祈ったくらいで 天候てんこう を変えることができたら、世の中から 飢饉ききん はなくなるだろう。

　だが、諸葛亮は相変わらず自信にあふれている。

「風のことは私に任せ、周瑜殿は指揮に集中してください」

「......わかった」

　周瑜は承知したが、諸葛亮を信じたわけではなかった。諸葛亮が失敗したら、自分が取り返す。むしろその 展開てんかい を願っていた。



　　　　　２


　敵を発見したのは、どちらが先だっただろうか。

　両軍の偵察部隊が、 幅はば 広い長江の 中央部ちゅうおうぶ で出会った。ともに二十隻ほどの小舟を 連つら ねており、曹操軍は 張南ちょうなん が、孫権軍は 周泰しゅうたい が指揮をとっている。

　張南はもと 袁紹えんしょう 軍にいた武将だが、泳ぎが 得意とくい で、水戦にも自信を持っていた。

「偵察だけでは 出世しゅっせ はできない。敵将の首一つくらいは持って帰って、曹操様にみとめてもらうのだ」

　張南は舟を横にならべ、配下の兵に弓での攻撃を命じた。孫権軍は槍の 穂先ほさき のような陣形をとって、突撃してくる。

　矢を放ったときには、すでに孫権軍は 間近まぢか にせまっていた。

「おわっ！　か、 回避かいひ しろ」

　張南は叫んだが、すでに遅かった。

　先頭に立つ周泰は、矢を盾で受けとめると、ためらいなく舟をぶつけた。両者の舟が大きくゆれて、波をかぶる。周泰は曲刀をふるいながら、ゆれる舟を 蹴け って、張南の舟に乗り移った。

　血の雨が舟をたたき、あっというまに、五人の兵が河に飛びこんだ。うち三人は、周泰に斬られて 致命傷ちめいしょう を 負お っている。残る二人は足元がおぼつかず、自分から落ちていったものだった。

　周泰を前にして、張南はふるえながら剣をかまえる。

「身のほど知らずだな」

　周泰はつぶやくと、曲刀をひとふりした。張南の首が飛んで、水中に 没ぼっ する。体のほうも、すぐに後を追った。

　周泰は動きをとめず、 別べつ の舟に飛び移っている。敵兵がへっぴり 腰ごし で突きだす槍をかわし、目にもとまらぬ速さで斬りつける。まばたきするまもなく、舟は 制圧せいあつ された。

「敵兵はすべて片付けました」

　部下の報告に、周泰は軽くうなずいた。河に飛びこんだ兵までは追っていないが、舟はすべて 確保かくほ している。

　退却を命じる 鐘かね が鳴っていた。 深追ふかお いはするな、ということだろう。周泰ひきいる部隊は敵の舟をひいて、 意気いき 揚々ようよう と 帰還きかん した。

　曹操は北岸に陣を張って、孫権軍の様子をうかがっている。敗北の報告を聞いても、眉ひとつ動かさない。

「やはり、小舟の上での 戦闘せんとう は敵のほうに利があるな。龐統の策を聞いておいてよかった。陸と同じ 条件じょうけん で戦えば、こちらが 有利ゆうり だからな」

　軍師の 荀彧じゅんいく が、心配そうに進言した。

「火計の心配はありませんでしょうか。船同士がむすばれていると、火の回りが早くなりますし、逃げるのがむずかしくなります」

　曹操は余裕たっぷりに笑った。

「それは私も考えた。だから、北岸から 離はな れなかったのだ。見よ、風は北から吹いている。風上にいれば、火はこわくない」

「なるほど、そこまでお考えでしたか」

「 念ねん のため、 各船かくせん には、消火用の水を用意させておこう。 火矢ひや であれば、それで対応できる。 乱戦らんせん になれば、火をつけられても、燃え広がるのは南のほう、つまり敵の船だ。こちらから火をつけてもいいくらいだな」

「手配しておきましょう」

　荀彧は感心して引き下がった。

　曹操はその後ろ姿を見送りながら、ふと思いついて張遼にたずねた。

「 蔡中さいちゅう からの報告はまだか？」

「まだです。 黄蓋こうがい の船も確認できていないそうです」

「遅いな」

　敵陣に 忍しの びこんでいる蔡中、寝返りを 約束やくそく している黄蓋、二人の連絡を待ってから、曹操は作戦をさだめるつもりだった。敵が先にしかけてくれれば、その方がいい。鎖でつながれた大船団を動かすのは、 若干じゃっかん の不安があった。

　曹操は 寒気さむけ を感じて、上衣を 重かさ ねた。北風が強くなっている。それは、曹操軍にとって 恵めぐ みの風となるはずだった。


　周瑜は眉をひそめて、 曇くも り 空ぞら をながめている。

　 建安けんあん 十三年（ 西暦せいれき 二〇八年）十一月二十日、この日は、朝から身を切るような北風が吹いていた。太陽はなかなか顔を見せない。

　雲が晴れてきたのは昼過ぎで、このころ偵察隊同士の戦闘が起こった。周泰をはじめ、 太史慈たいしじ 、 程普ていふ らの諸将は、勢いに乗って攻撃すべし、と主張したが、周瑜はみとめない。

「夜までは風を待つ」

　周瑜は南岸の祭壇を指さした。

　しなやかな指の先では、諸葛亮が天に祈りをささげていた。祭壇は 正方形せいほうけい で、 四辺しへん にはそれぞれ白、黒、赤、黄の 旗はた がなびいている。 壇上だんじょう の中央にはかがり火が燃えており、 香こう がたかれている。

　諸葛亮は白い 道士どうし 服ふく に身をつつんでおり、髪はまっすぐにたらし、足は 素足すあし であった。手に持った 若木わかぎ の 杖つえ を天に向け、何やら呪文をとなえている。まるで 仙人せんにん か道士のようだ。

　祈祷は長くはつづかない。諸葛亮はひとしきり祈ると、食事をとり、ゆっくりと休んでから、ふたたび壇上にあがった。日がかたむくと、今度は 仮眠かみん に入る。

「はたして真剣にやっているのか？」

　孫権軍のだれもが疑いの目を向けた。諸葛亮を信頼している 魯粛ろしゅく は、心配になって様子を見に行った。

「諸葛亮殿、この 儀式ぎしき で本当に風が吹くのですか？」

　たずねると、諸葛亮はいつものように微笑した。

「ええ、次の祈りで、風が吹きます」

　諸葛亮はそう言うと、着がえて 水辺みずべ に向かった。水で身を 清きよ め、最後の祈祷に 臨のぞ もうというのである。

　魯粛の報告を聞いた周瑜は、右手を 宙ちゅう にかざして、風を確かめた。

「ほぼ止まっているな。これなら、やつが失敗しても、作戦は決行できる」

　黄蓋はすでに、準備を終えて待機していた。黄蓋の乗る戦船は、油をふりかけたわらや 薪まき をたっぷりと積んでおり、 船首せんしゅ には合図の青い旗がたっている。ほかに、二十隻ほど、 同様どうよう の船が用意されていた。

　 甘寧かんねい は蔡中とその部下をひとつの船に集めていた。戦勝の前祝いと 称しょう して酒を飲ませ、妙な動きをしないように見張っている。

「いつ、行動を起こすのですか？」

　危険に気づいたのか、蔡中が不安そうにたずねた。

「さあな、黄蓋しだい、いや風しだいかな」

　その口調に 恐怖きょうふ を感じて、蔡中は気分が悪いとうったえた。船を下りたいと望むが、甘寧は許さない。

　いつのまにか、蔡中と部下たちのまわりを、孫権軍の兵士が囲んでいた。

「これはどういうことですか」

「もうすぐわかるさ」

　甘寧はすごみのある微笑を浮かべて、蔡中を見やった。その手にはすでに、刀がにぎられている。

「軍師殿の合図はまだかな」

　甘寧はちらりと指揮船を見た。合図の火は、まだ 灯とも っていない。どうやら周瑜は、ぎりぎりまで風を待つつもりのようだ。諸葛亮は 無視むし して、自分たちの力だけで戦えばいいと思うのだが、そこは天才同士、分かり合っているところもあるのだろう。甘寧としては、火計の成功を願うのみだ。

　火計の 実行者じっこうしゃ たる黄蓋は、船の上にどっかりと腰をおろしていた。燃えさかる 両眼りょうがん は 遠とお く敵陣をにらみつけている。

　上半身裸の背中には、 幾筋いくすじ もの傷が走っている。まだふさがりきっておらず、赤くはれて痛々しいが、黄蓋はむしろ 誇ほこ らしげにさらしていた。

「痛いということは生きているということ。生きているかぎり、おれは孫家のために働く。 孫堅そんけん 様、 孫策そんさく 様、この黄蓋の 生い き 様ざま をお見守りください」

　岩よりもかたい忠誠心を武器に、黄蓋は強大な敵にいどもうとしている。

　日はすっかり 暮く れて、長江は 闇やみ にしずんだ。各船では、かがり火が燃えている。 今夜中こんやじゅう の出撃が伝えられているため、眠る者はいない。

　空では、雲の切れ間で星がまたたいていた。月は見えない。風は止まっていて、寒さは遠のいている。

　周瑜は長江と祭壇を 交互こうご にながめながら、じっと考えこんでいた。

　諸葛亮はなぜ、あれほど自信を持って風が吹くと言えるのか。本当に風をあやつる 能力のうりょく があるとは信じられない。周瑜は今まで、雨を 降ふ らせるとか、 洪水こうずい をおさえるとか言う道士を何人も見てきたが、実際に成功した者は一人もいなかった。

　風をあやつることができないなら、答えはひとつ。諸葛亮は、今日、東南の風が吹くことを知っているのだ。なれた土地なら、たとえば雲の動きや動物の様子から、天候を 予測よそく することはできる。儀式は 単たん なるはったりではないか。

　だとすると、諸葛亮自身はもう必要ない。

　周瑜は 丁奉ていほう という若い将を呼んで命じた。

「諸葛亮をとらえろ。兵を二百ほど連れて行け」

「諸葛亮殿をとらえるのですか」

　丁奉はその場に突っ立ったまま、命令を 繰く りかえした。意図がわからぬようだ。周瑜は思わず、声を 荒あら らげた。

「早く行け。逃げられてしまう」

　そのとき、 生なま 暖あたた かい風がほおをなでた。周瑜ははっとして、旗に目をやった。

　 赤地あかじ に黒で孫と書かれた旗は、力強くひるがえっている。

　方角は......東南の風だ。

「黄蓋に合図を出せ！」

　周瑜はするどく命じると、丁奉の背中を押して送りだした。

　合図のかがり火が灯され、左右にふられる。

　黄蓋の船がすべるように動きだした。

　赤壁の戦いが、はじまろうとしている。



　　　　　３


　丁奉が祭壇に駆けつけたとき、諸葛亮の姿はすでになかった。壇上の 炎ほのお は水をかけて消されたようで、まだ 湯気ゆげ をあげている。

「ついさっきまでここにいたはずだ。陸か、河か、どちらに逃げたのだ？」

　丁奉は長江に目をやって、一隻の船が出発したのを見た。孫権軍の船はすべて 船首せんしゅ に 灯火とうか をかかげているはずだが、それが見えない。

「あれだ、追え！」

　丁奉はすぐに船を用意させた。 帆ほ の多い 快速船かいそくせん である。

　諸葛亮の船は戦場をはなれ、 夏口かこう や 江夏こうか のある下流のほうに向かっている。丁奉は帆をいっぱいに張り、風に乗って追いかけた。徐々に差がつまって、声が届く 距離きょり にまで近づいた。

「諸葛亮殿、どこに行かれるのです。都督殿がお呼びですから、急いで本陣にお戻りください」

　丁奉の声に気づいて、諸葛亮が 船尾せんび に姿をあらわした。

「私の役割は終わりましたので、主君のもとに帰ろうと思います。みなさまのご 武運ぶうん をお祈りいたします」

「困ります。急用ですので、せめて止まって話を聞いてください」

　丁奉は言いながら、さらに船の速度をあげさせる。

「その顔を見ると、話を聞くだけではすみそうにありませんね」

　諸葛亮は 奥おく に下がり、かわって一人の武将が前に出た。 白銀はくぎん のよろいが、かがり火に 反射はんしゃ して光った。武将が声を張りあげる。

「われこそは 常山じょうざん の 趙雲ちょううん なり！」

　丁奉はびくっとした。趙雲といえば、 長坂ちょうはん で大活躍した 勇将ゆうしょう ではないか。 槍やり の 名手めいしゅ と聞いているが、今は弓をかまえてこちらにねらいをつけている。

　趙雲はぎりぎりと弓をひきしぼった。

「わが 腕前うでまえ 、とくとご 覧らん あれ！」

　　四章　 義ぎ の 道みち



　　　　　１


　 諸葛亮しょかつりょう は 趙雲ちょううん に守られて、 劉備りゅうび の待つ 江夏こうか に到着した。劉備が 満面まんめん に笑みを浮かべて出迎える。

「よく無事に帰ってきてくれた。東南の風が吹いたから、そろそろ帰ってくるころだろうと思っていたのだ。その様子だと、計略は成功したようだが、戦は勝ったのか？　 曹操そうそう はどうした？」

　たずねてから、劉備ははっと気づいた。

「いや、報告は後でいい。疲れているようだから、先に休んでくれ」

「ありがたいお言葉ですが、われわれにとっては、ここからが 本番ほんばん です。まだ休むわけにはいきません。出陣の用意はできていますか」

「ああ、それでは、みなを呼ぼう」

　 関羽かんう 、 張飛ちょうひ ら武将たちが集まると、諸葛亮は事の 次第しだい を説明した。

「すると、水戦は 孫権そんけん 軍の勝利に終わりそうなのだな」

　劉備が確認すると、諸葛亮は 断言だんげん した。

「ええ、そうなります」

「じゃあ、追撃戦だな。百万の大軍を追いかけるんだ。これはおもしろいぞ」

　張飛がほおひげをふるわせて笑った。諸葛亮は淡々とつづける。

「曹操を討ちとる機会は今をおいて他にありません。予想される 逃走とうそう 路ろ に兵を送って、曹操軍を 奇襲きしゅう します。ただ、敵は 敗やぶ れたといえど、いまだに兵力はこちらを大きく上回ります。曹操だけをねらい、 雑兵ぞうひょう は深追いせぬようお願いします」

「それで、どこに行けばいいんだ？」

　今にも駆け出しそうな様子で、張飛がたずねる。

　諸葛亮は、 関平かんぺい と 劉封りゅうほう に頼んで地図を広げさせた。

「曹操は 襄陽じょうよう と 江陵こうりょう という、 荊州けいしゅう の 主要しゅよう 都市とし をおさえています。ふたつのうち、どちらに逃げこむかですが、江陵とみてまちがいないでしょう。曹操は江陵の 留守るす をまもっている 曹仁そうじん を信頼していますから、彼と合流するはずです」

　諸葛亮は筆をとり、地図の二点に 印しるし をつけた。

「この場所で曹操軍を待ち伏せしてもらいます。 一刻いっこく を争いますので、すぐに出発してください」

　命令を受けたのは、張飛と趙雲であった。二人がただちに 出立しゅったつ すると、もう一人の男が太い眉をひそめた。

「それがしの任務は？」

　諸葛亮はすました顔で答える。

「関羽殿には、今回は江夏の留守をまもっていただきます」

「敵が攻めてくるのか？」

「いえ、その可能性はほとんどありません」

　関羽はむっとして、諸葛亮をにらんだ。

「それがしは前線に出たほうが役に立てると思うのだが」

「しかし......」

　言いにくそうにする諸葛亮を見て、劉備も首をかしげた。

「どうしたのだ、 孔明こうめい ？　 雲長うんちょう を戦わせない理由があるのか」

「関羽殿はかつて、曹操から多大な 恩おん を受けております。いくら任務とはいえ、その曹操を討つことができるでしょうか」

　諸葛亮が 指摘してき すると、劉備も無言になった。義を 重おも んじる関羽の性格は、劉備が一番よくわかっている。

　だが、当の関羽は顔を赤くして怒っていた。

「それがしをばかにするつもりか。兄者と軍師殿の命令に、個人的な気持ちを優先させたりはしない。もし、任務を果たせなかったら、いかなる罰も受けよう」

　劉備と諸葛亮は顔を見合わせた。

「さあ、命令を出してくれ。地図を見るかぎりでは、江陵の手前にあるこの谷の辺りが、最後の 埋伏まいふく 地点ちてん になるのではないか」

「おっしゃるとおりです」

　諸葛亮はじっと関羽を見つめて言った。

「では、お願いしましょう。もし失敗したときは、 追放ついほう 処分しょぶん とします。曹操を見逃したら、劉備様のもとには帰れないものと思ってください」

「承知した」

　関羽はさらに細かい指示を聞くと、大またで部屋を出て行った。関平が後につづく。

　劉備は 義弟ぎてい の後ろ姿を、不安そうに見送った。

「少し言いすぎだったのではないか？　今回は関羽のかわりに私が行ってもよかった」

「しかし、曹操は 夏侯惇かこうとん や 張遼ちょうりょう といった 強者つわもの に守られています。関羽殿でなければ、撃退されてしまうでしょう」

　しばし、沈黙がつづいた。諸葛亮は劉備の問いを待っている。それを察して、劉備は重い口を開いた。

「この戦いで、曹操を討てなかったら、どうなるのだ？」

　諸葛亮は静かに微笑んだ。

「ご心配なく。私はそのつもりで計画を立てております」

「そうか」

　劉備がほっと息をつく。

「では、私は次の策にかかります。じっと 吉報きっぽう を待っているわけにはいきません。 新あら たな敵との戦いもはじまっていますから」

　うなずいて、劉備は立ちあがった。ついてきた民の落ちつき場所もそろそろ決めねばならない。早く生活を安定させてやりたいと思う。そのこともまた、諸葛亮は考えているにちがいなかった。


　曹操は 夜通よどお し、江陵への道を駆けていた。前を張遼、後ろを 許褚きょちょ に守られての退却行である。

　火計が決まった瞬間に、負けることはわかった。だが、曹操には敗戦の 規模きぼ を小さくし、大勢の兵士たちを無事に連れ帰る 義務ぎむ があった。退却と反撃の指示を出し、船から逃げてくる味方の 援護えんご を夏侯惇らにゆだねて、曹操はようやく逃亡に移った。

　計略にかかったことは、 後悔こうかい していない。 信念しんねん をつらぬいた結果だからだ。

　ただ、敗北に 直面ちょくめん して、思い出したことがある。亡き 郭嘉かくか は、孫権と戦ってはならないと言っていたのではなかったか。

「郭嘉が生きていれば、負けることはなかっただろう」

　かの若き 俊才しゅんさい ならば、無理な戦をとめ、計略を見抜いて、最終的には勝利をもたらしてくれただろう。諸葛亮や 周瑜しゅうゆ など、相手にならなかったはずだ。

　そこまで考えて、曹操は今一人の人物を思い浮かべた。あの、 不愉快ふゆかい な雰囲気の男、知恵はまわるが、いかにもあつかいにくそうだった。名前は 司馬懿しばい といった。

　だが、逃げる曹操に、先のことをゆっくり考える 暇ひま はない。

　最初に立ちはだかったのは、周瑜が送りこんだ 太史慈たいしじ の部隊であった。すでに 一番いちばん 鶏どり が鳴いて 久ひさ しく、辺りはうっすらと明るくなっている。

「曹操よ、死して荊州の土となれ！」

　太史慈は 南方なんぽう の丘の上から、一直線に駆けおりてきた。 選え りすぐりの 騎兵きへい 隊が二千騎、後につづいている。

　曹操軍の隊列は 縦たて に長く伸びている。その 横腹よこはら に攻撃を受けてはたまらない。太史慈のすばやい槍さばきに、たちまちに死体の山ができた。孫権軍の騎兵隊は 縦横じゅうおう にかけめぐり、逃げまどう歩兵を突き殺す。

　 曹洪そうこう と 夏侯淵かこうえん が後方から駆けつけるまで、曹操軍は 散々さんざん に討ち果たされた。曹洪は 復讐ふくしゅう とばかりに、配下の騎兵隊をならべて突撃しようとしたが、太史慈の姿はすでにない。ある程度戦うと、風のごとく逃げ去ってしまったのだ。決して無理をするな、という周瑜の指示を 忠実ちゅうじつ に守った結果だった。

　曹操は列の先頭近くを駆けているが、敵の 妨害ぼうがい もあって、なかなか味方が追いついてこない。

「曹操様、かたまっていては 標的ひょうてき になってしまいますし、速度もあがりません。軍を分けましょう。江陵か襄陽で合流することを定めて、曹操様は先に行くべきです」

　張遼の進言に、曹操はしぶしぶうなずいた。騎兵のみ一万ほどの部隊で、曹仁の待つ江陵へ急ぐことにする。歩兵部隊は夏侯惇にひきいさせ、襄陽をめざすよう命じた。

　ここからが、本当の 苦難くなん であった。

　曹操の選ぶ道には、必ず伏兵があらわれた。それも、ここに敵が来たら 厄介やっかい だという場所に、あらわれるのだ。

「曹操よ、覚悟しろ！　 常山じょうざん の趙雲が、劉備様の命により、貴様を討つ！」

　白銀のよろいが光を放った。槍をたずさえた趙雲が、曹操軍におそいかかる。

　 長坂ちょうはん で見せた 超人的ちょうじんてき な武勇を知らぬ者はいない。趙雲の名を聞いただけで、曹操軍の将兵はすくみあがった。

　趙雲は千騎の兵をしたがえ、槍を頭上でふりまわして突撃した。あっというまに十人以上の護衛兵を 落馬らくば させ、曹操にせまる。

　 神速しんそく の突きを、曹操はかろうじてかわした。いったんすれちがった趙雲が弧を描いて戻ろうとするところに、張遼が立ちはだかった。

「ここはお任せください。曹操様は先へ！」

　張遼の真剣な目を見て、曹操は馬に 鞭むち をいれた。

「趙雲を倒そうなどと考えるな。必ず生きて帰れよ」

　言いおいて、全速力で馬を走らせる。

「逃がしはしない！」

　趙雲も追いかけたが、曹操の乗る名馬は影も 踏ふ ませずに駆ける。さらに張遼が妨害してくるので、またたくまに距離があいてしまった。

　趙雲は 追跡ついせき をあきらめて、張遼に向き直った。つづけざまに突きを放ったが、張遼は 方天戟ほうてげき をあやつって、左右に受け流す。反撃はせずに、防御に集中するかまえだ。

　一騎打ちは長くはつづかなかった。

　逃走中の曹操軍に対して、 雪辱せつじょく に燃える劉備軍の士気は高いし、 疲労ひろう もたまっていない。騎兵同士のぶつかり合いは劉備軍の 圧勝あっしょう に終わり、張遼が囲まれそうになったのである。

「曹操様の命令ゆえ、ここは 退ひ かせてもらう」

　張遼は趙雲との間に敵兵の壁をつくり、何とか 脱出だっしゅつ に成功した。

　その背中をねらおうと弓をかまえる兵がいたが、趙雲がするどい声を発してやめさせた。趙雲は戦場をふりかえって、 負傷ふしょう した敵兵の降伏を受け入れ、死者はほうむるよう命じた。

「曹操を逃がしたのは残念だが、一番手柄は張飛殿にゆずることにしよう」

　その張飛は、 街道かいどう の真ん中に立って、曹操を待っていた。 蛇矛だぼう を大地に突き刺し、大きな 瞳ひとみ で東のほうをにらんでいる。

　やがて、 土煙つちけむり があがって、曹操軍が近づいてきた。その数は、五千に 満み たない。

「ほう、あいつの言ったとおりじゃないか」

　張飛はにこりと笑って、蛇矛を引き抜いた。

「む、あれは張飛か」

　曹操は眉をくもらせた。劉備軍の将たちは、まるで 未来みらい を見通しているかのように、行く先々にあらわれる。これも諸葛亮の計だというのだろうか。

「曹操様、おれがやつと戦います」

　許褚の巨体が、曹操の前に影をつくった。

「許褚よ、 典韋てんい につづいておまえまでいなくなったら、私を守る者がいなくなる。決して無理はするなよ」

「わかりました」

　許褚は馬を下りて戟をかまえ、張飛と向かいあった。そのわきを、曹操が駆け抜ける。

「あ、おい、こら待て」

　追いかけようとした張飛の肩口を、許褚の戟がおそった。張飛は飛びすさってよけると、左にまわりこみながら、はげしく突いた。許褚が戟ではじく。

「強いやつは、みんなで倒せばいい」

　許褚の号令で、曹操軍の兵士たちが距離をとって張飛を囲んだ。及び腰だが、すきがあれば、いっせいにおそいかかる体勢だ。

　だが、張飛はまったくひるまないどころか、大声で笑い出した。

「ははは、ひっかかったな。張飛様の伏兵の計だ！」

　笑い声を合図に、両側の林から劉備軍があらわれた。曹操軍の背後から、いっせいに突撃する。

　張飛隊は曹操軍に大打撃をあたえた。逃げることができたのは、許褚をはじめとする数百人だけである。

　だが、曹操はすでに戦場を 脱だっ している。その運命は、最後の一人にゆだねられた。



　　　　　２


　曹操にしたがう兵は、千人を切っていた。張遼が追いついてきたのが、 唯一ゆいいつ の 救すく いである。これ以上伏兵がいたら、逃げ切れないかもしれない。曹操は思ったが、口にも 態度たいど にも出していない。

　道は二つに分かれていた。山 越ご えの近道と、平野を行く遠回りの道。曹操は兵を休ませ、 斥候せっこう を出した。

　すると、山越えの道のほうで 煙けむり があがっており、人の 気配けはい があるという。

「では、平野の道を行きましょうか」

　張遼が言ったが、曹操はすぐには動き出さなかった。馬にまたがったままで、考えこんでいる。やがて出した 結論けつろん は、張遼の進言とは 逆ぎゃく であった。

「これは平野のほうに 誘導ゆうどう するための工作だろう。あえて山道を行くべきだ」

　考えすぎではないか、と張遼は思った。考えれば考えるほど、諸葛亮の 術中じゅっちゅう にはまる気がするのだ。だが、曹操はもう山道に馬を進めている。張遼は仕方なく後を追った。

　 難路なんろ であった。道は細く、くねくねと曲がりながら、 険けわ しい山を 登のぼ っていく。馬に乗って進むのは危険であり、下りて歩かねばならなかった。

　その上、冷たい雨が降ってきて、 下着したぎ までずぶぬれになった。足取りは重くなり、 骨ほね まで冷えきって、体力がどんどんけずられていく。降りつづく雨は 足跡あしあと にたまり、あふれて流れだした。気を抜くと、足をとられそうになる。

　曹操はかぶとを失い、上衣は 泥どろ だらけになっていたが、決して下を向かなかった。自信に満ちた瞳で空をにらみ、 天命てんめい にあらがうようにして歩いている。

　その姿を目にして、兵士たちも力をふりしぼった。最後尾を守る張遼も、兵をはげましながら、足を動かしつづけた。

　これだけ歩きにくい道なのだから、敵はいないだろう。だが、その期待も裏切られる。 山頂さんちょう 付近ふきん にさしかかって、道が 平たい らになったときだった。

　 唐突とうとつ に雨がやんで、雲の切れ間から太陽が顔を出した。その光に照らされて、堂々たる体格の男の姿が浮かびあがる。またがる馬は、 真ま っ 赤か な全身から 水滴すいてき をしたたらせており、まるで血の汗を流しているように見えた。

　曹操軍の兵士たちは、 放心ほうしん してすわりこんだ。わずかに残っていた気力は、男の姿を目にしたとたんに消え失せていた。

「こうなるような気がしていたのだ」

　曹操はつぶやくと、足をひきずるようにして、男の前に立った。

「久しいな、関羽よ」

　曹操が軽く頭をさげて礼をすると、関羽も馬上で礼を返した。

「あのときは、お世話になりました」

　関羽が 青龍せいりゅう 偃月刀えんげつとう を右手にかまえる。

「本日はわが主の命により、曹操殿をお待ちしておりました。覚悟はよろしいですかな」

「おぬしに討たれるなら 本望ほんもう だが、まだやり残したことがあるのでな。できれば、見逃してほしいところだ」

「その、やり残したことを 阻はば むために、それがしたちは戦っているのです」

　曹操は馬上の関羽を見あげているが、そのするどい視線に 卑屈ひくつ さはみじんもない。

「私は今でも、おぬしを配下に迎えたいと思っている」

「お気持ちはありがたく思います」

　関羽は偃月刀を 握にぎ りなおした。だが、まだ動けない。

「関羽殿！」

　張遼が駆けてきて、曹操の 斜なな め後ろに立った。

「おお、張遼殿か」

　関羽はわずかに目を細めたが、声ははずまなかった。

「関羽殿は、曹操様に対する 恩義おんぎ をいかにお考えですか」

　関羽は沈黙している。 徐州じょしゅう で命を救われたにもかかわらず、曹操のもとを離れた。それは約束を守ってもらっただけだからよい。だが、関所を突破して何人かの将を殺したことを、曹操はとがめなかった。張遼を 派遣はけん して 夏侯惇かこうとん をとめてくれた。 許昌きょしょう にいる間、 甘かん 夫人ふじん に 危害きがい が及ぶこともなかった。 赤兎馬せきとば も、曹操にもらったものだ。

　恩は返さねばならない。 義理ぎり は 果は たさねばならない。

　だが、主君の命令もまた、 拒こば めなかった。

　 相反あいはん する思いに、関羽の心は引き 裂さ かれた。

　曹操はただ、関羽の決断を待っているようだった。張遼がふたたび口を開いた。

「関羽殿の主君は、恩を返すために命令にそむいた部下を、許さぬような人なのですか？」

　関羽は曹操軍の兵士たちを見た。みなずぶぬれで、 精せい も 根こん も 尽つ き果てたといった様子で、へたりこんでいる。

　関羽はわずかに目を閉じ、そして開いた。配下の兵に、 散ち らばるように指示する。

　道がひらけた。

　ひらけた道を、曹操はゆっくりと歩いた。

「礼を言うぞ」

「それがしは義の道を歩むのみ。次に会ったときは、全力で戦わせていただく」

　ふらふらになった曹操軍の兵士たちが通り過ぎるのを、関羽はじっと見守っていた。


　曹操軍が山を下りたころには、辺りはすっかり 暗くら くなっていた。遠くに江陵の明かりが見えている。

　周瑜の傷は深くなかったが、矢には毒がぬられていた。孫策の例を思いだして、周泰や程普は青くなったが、周瑜の 意識いしき ははっきりしていた。

「私は 重病じゅうびょう で死にかかっていることにする。敵があなどって攻めてきたときが 好機こうき だ。それまでは、 堅かた く陣を守って、絶対に外に出ないように」

　それだけ言うと、周瑜は 程普ていふ に指揮を任せて眠りについた。熱は高かったが、うなされるほどではなく、命に 別状べつじょう はないように思われた。



　　　　　４


　孫権軍の陣は静まりかえって、三日の間、 物音ものおと ひとつしなかった。周瑜が死んだといううわさは、江陵の城にも届いている。曹仁は逃げだした孫権軍の兵をとらえて、うわさが本当かどうか確かめようとした。

「都督はまだ死んでいませんが、意識はないようです。将軍たちはだれが指揮をとるかでずっともめています。それで、おれたちは不安になって逃げだしたのです」

　兵の 証言しょうげん は、曹洪を満足させた。

「今なら簡単に勝てる。総攻撃だ」

「わなかもしれないのだぞ」

　曹仁が 指摘してき したが、曹洪は引かない。結局、 夜襲やしゅう をかけることで話がまとまった。

　 夜半やはん 過ぎ、曹操軍はそろそろと孫権軍の陣に近づいた。かがり火はたかれているが、見張りは多くない。物見台も矢倉も 無人むじん である。やはり、 規律きりつ が乱れているようだ。

　曹操軍の兵士たちは、はやる気持ちをおさえながら、息をひそめて進んだ。はりめぐらされた柵と、河の水をひいた 濠ほり を越える。 陣門じんもん に達したところで、先頭の兵が叫んだ。

「敵がいません！」

　悲鳴のような声だった。まるでそれを合図にしたかのように、左右から孫権軍があらわれる。いつの間にか、すっかり囲まれていたのだ。

　孫権軍は矢を射かけた後、甘寧と周泰を中心にはげしく攻めたてた。わなにかかって気落ちした曹操軍は、ろくな 抵抗ていこう もできずに討たれていく。夜明け前、曹洪と曹仁は何とか囲みを破って逃げだしたが、追撃を受けたため江陵には戻れず、襄陽めざして落ちていった。

　 完勝かんしょう をおさめた孫権軍だが、周瑜はまだ熱が下がらず、起きあがれるほどには回復していない。周瑜をのぞく各将が、江陵に乗りこむことになった。

　ところが、曹操軍が捨てていったはずの江陵の城門は堅く閉じられており、城壁の上には旗がひるがえっているではないか。

「劉、と書いてありますな」

　呂蒙の言葉に、一同は顔を見合わせた。

「まさか、劉備軍が？」

　その声が聞こえたかのように、城壁の上に趙雲が姿をあらわした。

「江陵は私たちが手に入れました。孫家のみなさまには兵を 退ひ いていただきたい」

「何だと !? 」

　怒りくるう諸将をおさえて、程普が言った。

「約束がちがうではないか」

「いえ、私たちは約束どおり、あなたがたが負けたのを見て、兵を進めたのです」

「確かに、一度は負けて後退したが、こうして最後には勝ったではないか。曹操軍を追い払ったのはおれたちだ」

「そう言われましても、あなたがたが負けたのは事実ですし、 現げん に江陵をおさえているのは私たちです。明けわたすことはできません」

　城壁の上には弓隊がならび、孫権軍の諸将にねらいを定めている。

「あいつら、絶対に許さない。態勢をととのえなおして、必ず攻め落とす」

　呂蒙がこぶしを握って言うと、甘寧がつづいた。

「先に襄陽を落としてはどうだ。曹操軍を完全に追い出してから、江陵をゆっくり包囲すればいい」

　そこへ伝令が走ってきて、急報を告げた。

　何と、襄陽も劉備軍に奪いとられたというのだ。 偽にせ の使者を送って江陵への救援をもとめ、夏侯惇が城を出たすきに、関羽と張飛の部隊が襄陽をおそったという。襄陽の民が劉備軍に味方したため、ほとんど 戦闘せんとう にはならなかったそうだ。

「まさか......」

　だれも、とっさには言葉が出てこない。

　諸将はいったん陣に戻って対策を話し合った。 普段ふだん なら周瑜の判断にゆだねるのだが、 高熱こうねつ で 寝込ねこ んでいる状態のところに、これ以上の 重荷おもに を背負わせるのはためらわれた。

　呂蒙や周泰は攻撃を主張したが、程普は首をたてにはふらない。曹操軍との 激闘げきとう で、兵士たちは疲れており、また犠牲も少なくはなかった。戦うにしても、今は無理だ。そこで、魯粛が申し出た。

「私が 交渉こうしょう に行ってまいります。劉備殿は 仁義じんぎ をわきまえた方ですので、このような非道なまねをして平気でいられるとは思えません。 正論せいろん を 説と けば、わかってくれるでしょう」

　これがみとめられて、魯粛は江陵に乗りこむことになった。

　劉備と諸葛亮を前に 熱弁ねつべん をふるう。

「私たちは、曹操に荊州を追われたあなたがたを助け、多大な犠牲を 払はら って、曹操軍を打ち破りました。諸葛亮殿の知恵は 借か りましたが、実際に血と汗を流したのは私たちです。それなのに、だまし討ちのようなやり方で荊州を奪うとは、人の道に外れた行いではありませんか。曹操を倒した私たちに、荊州を返していただきたい」

　答えたのは諸葛亮である。

「返す、とはおかしな表現ですね。荊州はもともと劉表殿の 領地りょうち でした。それを、曹操が 卑劣ひれつ な手で奪ったのです。本来、荊州は劉表殿の 長男ちょうなん である 劉琦りゅうき 殿が治めるべきです。あなたがたに返せと言う 権利けんり はないはずですよ」

「だからといって、劉備殿が荊州を奪っていいということにはならないでしょう」

「奪ったわけではありません。劉琦殿のために取り返したまでのこと。劉の旗をご 覧らん になったでしょう」

　その言葉とともに、劉琦が姿をあらわす。魯粛はおどろいて、 二に の 句く が 継つ げなかった。劉の旗の劉は、劉琦の劉だというのだ。

「これからは私が父の後を継いで、荊州を治めていきます」

　劉琦の言葉は、魯粛にとって、交渉の失敗を意味していた。

「今日のところは、これでお引き取り願いましょう」

　うながされて、魯粛は 退出たいしゅつ した。今後の策を話し合うため、孫権軍は 柴桑さいそう まで引き返すことになった。

「これでよかったのか」

　劉備の問いに、諸葛亮はうなずいたが、表情はきびしい。

「ええ、いまのところは。ですが、曹操や孫権が同時に攻めてくるようなことになれば、荊州はひとたまりもありません。 早急そうきゅう に次の手を打つ必要があります」

　劉琦は魯粛が去ってすぐ、 床とこ に戻った。もともと体が弱かったところに、父の死や曹操の 侵攻しんこう などがあって、重い 病やまい にかかっていたのである。治る可能性は高くない。

　劉備たちは孫権軍の力を利用して曹操を撃退し、荊州という 待望たいぼう の 根拠地こんきょち を手に入れた。だが、 前途ぜんと にはまだ 困難こんなん が 連つら なっている。 志こころざし を果たすための戦いは、これからはじまるのであった。
