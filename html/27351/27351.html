「 桐 島 先 生 、 ご 無 沙 汰 し て い ま す 。 私 の こ と 覚 え て ら っ し ゃ い ま す か ？ 」

新 郎 の 友 人 と し て 参 加 し て い た す み れ が 、 ビ ー ル 片 手 に 伊 織 の テ ー ブ ル に 回 っ て き た の は 、 宴 も た け な わ を 迎 え た 頃 だ っ た 。

「 如 月 す み れ さ ん ？ だ っ た か な ... ？ 」

久 し ぶ り に 再 会 す る 恩 師 が 、 自 分 の 名 前 を す ん な り 言 い 当 て て く れ た こ と に 、 す み れ は ぱ っ と 相 好 を 崩 す 。 手 の ひ ら の ビ ー ル グ ラ ス を す み れ に 差 し 出 し な が ら 、 伊 織 は 語 り か け る 。

「 そ う か 、 君 も 新 郎 と 同 い 年 だ っ た の か 。 卒 業 し て 以 来 だ な 。 ず い ぶ ん ご 無 沙 汰 し て い る ね 。 」

す み れ が 注 い だ ビ ー ル を 一 口 飲 み 干 す と 、 頬 を 赤 ら め た 伊 織 が 、 居 住 ま い を 正 し な が ら さ ら に 尋 ね る 。

「 と こ ろ で 、 如 月 君 は も う ご 結 婚 し て い る の か な ？ 」

伊 織 は 円 卓 に 乗 せ て あ る 未 使 用 の グ ラ ス を 手 に と っ て 、 返 杯 を 教 え 子 に 勧 め る 。 そ の ビ ー ル を な み な み と 注 が れ な が ら 、 す み れ は に っ こ り 微 笑 み 返 す 。

「 い え 、 ま だ 独 身 な ん で す 。 で も 、 こ の 秋 に は 式 だ け で も 、 彼 と 挙 げ よ う か と い う 話 に は な っ て い ま す 。 」

「 そ れ は 良 か っ た 、 じ ゃ グ ッ と 行 き な さ い 」 そ う 勧 め る 恩 師 の 言 葉 に 従 う よ う に 、 如 月 す み れ は 小 グ ラ ス の ア ル コ ー ル を 一 息 に 飲 み 干 し た 。

今 は 丁 度 春 ３ 月 だ か ら 、 す み れ が 結 婚 す る ま で に は 半 年 以 上 も 時 間 が 残 っ て い る 。 そ の 間 に 、 結 婚 の 準 備 を 進 め て い こ う と い う 算 段 な の だ ろ う 。

「 あ の ォ ー 。 良 か っ た ら 、 あ と で こ こ に お 電 話 頂 け ま せ ん か ？ ぜ ひ 先 生 に も 、 私 の 結 婚 式 に 、 参 加 し て 頂 き た い の で 。 」

差 し 出 さ れ た 名 刺 に は 、 す み れ が 現 在 勤 め て い る 保 育 士 と し て の 勤 務 先 と 、 裏 側 に は 手 書 き で 携 帯 電 話 の 番 号 が 記 入 し て あ る 。

「 え え ー ？ 僕 な ん か を 招 待 し て い い の ？ 学 園 の マ ド ン ナ だ っ た 君 な ら 、 い く ら で も 参 加 し て く れ る 先 生 方 が い る は ず で し ょ う ？ 」

驚 き と と も に 、 小 さ な カ ー ド を 受 け 取 っ た 桐 島 は 、 ま じ ま じ と す み れ の 瞳 を 覗 き 込 む 。

「 桐 島 先 生 に だ け 来 て 欲 し い ん で す 。 あ と は 、 身 内 し か 呼 び ま せ ん か ら 、 ぜ ひ ご 参 加 下 さ い ま せ ん か ？ 」

ふ ー ん と 、 し ば し 信 じ ら れ な い 様 子 で 伊 織 は 、 名 刺 と す み れ の 屈 託 無 い 笑 顔 を 見 比 べ る 。

高 校 を 卒 業 し て か ら 早 ９ 年 。 そ の 間 に 一 人 前 の 社 会 人 と し て 生 活 し て き た 如 月 す み れ は 、 よ り 洗 練 さ れ た 美 し さ を 漂 わ せ て い る 。 普 段 は 保 育 所 の 若 き 主 任 と し て 、 生 き 生 き と 仕 事 に 打 ち 込 ん で い る 様 子 が 、 輝 く 笑 顔 か ら も 推 察 で き た 。

「 そ れ じ ゃ 、 先 生 。 お 電 話 お 待 ち し て い ま す ね 。 」

に こ や か に 告 げ る と 、 き ら び や か な 出 で 立 ち の す み れ は 、 も と 居 た 自 分 の テ ー ブ ル に 戻 っ て い く 。 そ の 後 ろ 姿 を 見 つ め な が ら 、 遠 い 日 の 過 去 の 記 憶 を 、 つ い 昨 日 の 事 の よ う に 伊 織 は 思 い 返 し て い た 。

（ す み れ は 、 あ の 事 を も う 忘 れ て し ま っ た み た い だ な 。 こ れ で も う 、 心 配 な い な 。 ）

甘 酸 っ ぱ い 記 憶 が 、 桐 島 の 脳 裏 を か す め る 。 遠 い 日 の あ の 出 来 事 は 、 今 で も 二 人 だ け の 秘 密 な の だ か ら 。

２ ． シ テ ィ ホ テ ル の 再 会

伊 織 と す み れ が 別 の 場 所 で 再 び 待 ち 合 わ せ し た の は 、 披 露 宴 の 翌 週 の 夕 方 の 事 で あ る 。 軽 く 食 事 で も し な が ら 、 近 況 に つ い て 話 し 合 お う と い う こ と に な っ た 。 待 ち 合 わ せ に 選 ん だ の は 、 都 心 に 近 い シ テ ィ ホ テ ル の ビ ュ ッ フ ェ で あ る 。

「 待 た せ ち ゃ っ た か な ？ 」

約 束 の 時 間 よ り も １ ０ 分 程 度 遅 れ る こ と は 、 す で に 携 帯 で 連 絡 済 で あ っ た 。 文 庫 本 に 見 入 っ て い た す み れ が 、 満 面 の 笑 み を 浮 か べ て 伊 織 を 迎 え 入 れ る 。

「 い え 、 私 も さ っ き 到 着 し た ば か り で す 。 ま ず は 食 事 に し ま し ょ う か ？ 」

待 ち 合 わ せ の 相 方 が 到 着 し た こ と に 、 初 老 の 支 配 人 が 気 付 い て メ ニ ュ ー を 持 参 す る 。

「 桐 島 先 生 、 こ こ の パ ス タ は と っ て も 評 判 な ん で す よ ！ 白 ワ イ ン に と っ て も 合 う ん で す 。 」

笑 顔 の 教 え 子 が メ ニ ュ ー を 指 さ し 、 い く つ か お す す め の 料 理 の オ ー ダ ー を 済 ま せ る 。 程 な く し て フ ル ボ ト ル の 白 ワ イ ン が 運 ば れ 、 二 人 は 改 め て 再 会 の 祝 杯 を 上 げ た 。

「 二 人 の 再 会 と 、 如 月 君 の 結 婚 を 祝 し て 、 カ ン パ ー イ 」

透 明 な 液 体 が 注 が れ た 二 つ の グ ラ ス を 、 小 さ く カ チ ン と 鳴 ら す 。 そ れ か ら ２ 時 間 、 二 人 は 互 い の 近 況 や と り と め の な い 世 間 話 を 楽 し む 。 フ ル ボ ト ル の ワ イ ン が あ ら か た 空 に な っ た 頃 、 頬 を 赤 く 染 め た す み れ が 伊 織 に 提 案 す る 。

「 先 生 、 最 上 階 の ラ ウ ン ジ で 飲 み 直 し ま せ ん ？ ま だ 、 い い で し ょ ？ そ れ と も 教 え 子 と 二 人 だ と 奥 様 に し か ら れ ち ゃ う か な ？ 」

う ふ ふ 、 と 意 味 深 に 笑 う す み れ は 、 酔 い が 回 っ て き た せ い な の か 、 少 し 饒 舌 に な っ て き て い

る 。 独 り 暮 ら し を し て い る せ い で 、 す み れ に は 門 限 が な い の は 分 か っ て い る 。 さ て 、 ど う し た も の か と 、 伊 織 が 思 い 悩 ん で い る う ち に 、 す み れ が ふ い に 伊 織 を 真 っ 直 ぐ 見 つ め て 語 り 出 す 。

「 先 生 は 、 あ の 頃 の 出 来 事 を 、 も し か し ら お 忘 れ に な っ た わ け じ ゃ あ り ま せ ん よ ね ？ 」

― あ の 頃 の 出 来 事 ― 、 す み れ の そ の 一 言 で 、 伊 織 は 、 忘 れ 去 っ て い た 、 い や 忘 れ 去 ろ う と し て い た ほ ろ 酸 っ ぱ く も 苦 い 記 憶 を 思 い 返 し た 。

（ や は り 、 如 月 も 、 あ の 日 々 の 事 は ず っ と 覚 え て い た ん だ ... ）

そ の 紛 れ も な い 事 実 に 、 桐 島 は 先 日 の 披 露 宴 で ど う し て も 会 い た い と 口 に し た す み れ の 真 意 を 初 め て 理 解 し た 。

「 怒 っ て い る わ け じ ゃ な い ん で す 。 で も 、 結 婚 す る 前 に ど う し て も 先 生 と 、 も う 一 度 会 っ て 、 あ の 頃 の 事 を 話 し て み た か っ た ん で す 。 」

ぽ う っ と 頬 を 赤 ら め て い る の は 、 ワ イ ン の せ い だ け で は な い ら し い 。 フ ル ボ ト ル を あ え て 注 文 し た の も す み れ の 方 だ 。 初 め か ら そ の 事 を 告 げ た く て 、 わ ざ と 多 め の ワ イ ン を 口 に し た の か も し れ な い 。

「 桐 島 先 生 。 さ あ 、 行 き ま し ょ 。 夜 景 が と っ て も き れ い で す よ ！ 」

請 求 書 の ボ ー ド を 持 ち 上 げ な が ら 、 し ゃ な り と 席 を 立 と う と す る 教 え 子 を 慌 て て 追 い か け る 。 上 着 の ポ ケ ッ ト か ら 財 布 を 取 り 出 し 、 桐 島 は 会 計 を 済 ま せ た 。

３ ． 乙 女 の 願 い

桐 島 伊 織 が 如 月 す み れ の 通 っ て い る 女 子 校 に 赴 任 し た の は 、 彼 が ２ ７ 才 の 時 の こ と で あ る 。 結 婚 と 同 時 に 新 た な 赴 任 先 に 転 任 し て き た 音 楽 教 師 は 、 女 生 徒 の 憧 れ で も あ っ た 。 特 に 、 ブ ラ ス バ ン ド の 顧 問 に な っ て か ら は 、 入 部 希 望 者 が あ と を 絶 た な い 状 況 が 続 い て い る 。

け れ ど 、 そ れ ま で 副 主 将 と し て 、 ブ ラ バ ン を リ ー ド し て い た 如 月 す み れ に と っ て 、 こ の 若 い 顧 問 を た や す く 受 け 入 れ る こ と に は 抵 抗 が あ っ た 。 そ れ も あ っ て 、 桐 島 目 当 て に 入 部 し て く る 親 衛 隊 も ど き の 女 生 徒 に は 、 厳 し く 当 た る こ と が し ば し ば 目 撃 さ れ た 。

「 如 月 さ ん 、 そ ん な に 厳 し く 言 わ な く て も い い じ ゃ な い ？ 」

軽 く た し な め る よ う な 伊 織 の 物 言 い も 、 す み れ に は か え っ て 癪 に 障 る 。

「 桐 島 先 生 、 ブ ラ バ ン は 先 生 の 私 設 応 援 団 で は あ り ま せ ん か ら 、 誤 解 無 く ！ 」

学 年 一 番 と 噂 さ れ る 美 貌 の 持 ち 主 で あ る 如 月 す み れ 。 成 績 も 常 に ト ッ プ ク ラ ス で 、 正 義 感 が 強 く 、 女 子 高 の 下 級 生 な ら ず と も 同 級 生 や 先 輩 か ら も 憧 れ の 存 在 だ っ た 。 ス リ ム な ス タ イ ル に 、 シ ョ ー ト カ ッ ト の 凜 々 し い 顔 立 ち が 際 立 っ て い る 。

そ の 自 信 の 高 さ が 、 若 い 顧 問 を 容 易 に 受 け 入 れ る こ と が 出 来 な か っ た の だ 。 練 習 メ ニ ュ ー の 内 容 や 遠 征 先 の 候 補 地 な ど 、 こ と ご と く 桐 島 に 食 っ て か か り 、 ア ル ト サ ッ ク ス の ソ リ ス ト と し て 有 名 な 伊 織 も 正 直 途 方 に 暮 れ て い た 。

夏 休 み に 入 っ た そ ん な あ る 夕 暮 れ の こ と で あ っ た 。 予 定 し て い た 練 習 メ ニ ュ ー も あ ら か た 終 え て 、 ブ ラ バ ン の 部 員 も ほ ぼ 全 員 帰 宅 し 終 え た 時 刻 。 顧 問 で あ る 伊 織 は 、 部 室 の 施 錠 と 見 回 り を 兼 ね て 、 音 楽 棟 の あ る 建 物 を １ 階 か ら 順 に 見 回 っ て い た 。

深 夜 遅 く ま で 活 動 し て い る そ の 建 物 だ け は 、 警 備 員 が 巡 回 し な く て も 済 む よ う 機 械 警 備 が 導 入 さ れ て い る 。 夕 闇 が 迫 り 、 ま だ ７ 時 前 だ と い う の に 、 明 か り の 灯 っ て い な い 音 楽 教 室 は 静 ま り か え っ て い た 。

伊 織 は ふ と 、 一 部 屋 だ け 明 か り が 漏 れ て い る 練 習 ル ー ム に 気 付 く 。 そ こ は 管 楽 器 専 用 の 練 習

部 屋 で あ っ た 。 誰 が 残 っ て 練 習 し て い る の だ ろ う と 、 伊 織 が 中 を 覗 く と 、 こ の 夏 休 み か ら 部 長 を 任 さ れ て い る 如 月 す み れ 本 人 で あ る 。

す み れ の パ ー ト は 、 桐 島 が 専 門 と す る ア ル ト サ ッ ク ス 。 顧 問 と し て の 伊 織 に は 猛 烈 に 反 発 し て い る も の の 、 ソ リ ス ト と し て 活 躍 し て い る プ ロ 級 の 腕 前 に だ け は 、 す み れ も 歯 が 立 た な い 。 そ の ギ ャ ッ プ か ら 、 部 長 と し て も 桐 島 に は 素 直 に 接 す る こ と が 出 来 な か っ た の だ 。

窓 越 し に 見 つ め る す み れ に は 、 ど う し て も う ま く 吹 け な い フ レ ー ズ が あ っ た 。 繰 り 返 し 同 じ 旋 律 を 吹 こ う と し て も 音 が 裏 返 っ て し ま う 。 深 い た め 息 を 付 き な が ら も 、 懸 命 に サ ッ ク ス を 響 か せ よ う と す る そ の 姿 に 、 伊 織 も つ い ほ だ さ れ て 、 ド ア を 小 さ く ノ ッ ク す る 。

「 そ こ の パ ー ト は 指 を こ う し て 動 か し な が ら 、 タ ン ギ ン グ す れ ば い い ん だ 。 」

ふ い に 声 を か け ら れ 、 ビ ク ッ と し て イ ス か ら す み れ は 立 ち 上 が る 。 周 り が 薄 暗 く な っ た こ と さ え 気 付 か ぬ ま ま 、 一 心 不 乱 に 練 習 に 取 り 組 ん で い た こ と に 、 す み れ は 初 め て 気 付 く 。

「 桐 島 先 生 、 余 計 な こ と 言 わ な い で 下 さ い 。 私 な り に 頑 張 り ま す か ら 。 」

あ の 当 時 、 桐 島 は 新 婚 で 、 妻 と の 夫 婦 生 活 も ほ ぼ 毎 晩 楽 し ん で い る よ う な ラ ブ ラ ブ の 時 期 だ っ た 。 妻 の 明 里 も 次 第 に 女 の 悦 び に 目 覚 め 始 め 、 夫 婦 の ナ イ ト ラ イ フ は 至 っ て 好 調 で あ っ た 。

「 い つ だ っ た か 、 先 生 と 部 活 で 遠 征 し た 晩 に 、 一 杯 し ち ゃ っ た こ と が あ っ た で し ょ う ？ あ の 時 な ん て 、 完 全 に 私 の 排 卵 日 だ っ た の よ 。 」

ど き っ と す る 告 白 の 連 続 に 、 伊 織 は 胸 が 今 更 な が ら バ ク バ ク し 始 め る 。 け れ ど 、 至 っ て 快 活 な す み れ は 、 ケ ラ ケ ラ と 笑 い な が ら 、 伊 織 の 反 応 を 楽 し ん で い る 。

「 で も 、 ど ん な 危 険 日 に 先 生 と 交 わ っ て も 、 私 に は 赤 ち ゃ ん が 出 来 な か っ た 。 先 生 は 私 と セ ッ ク ス し た 回 数 覚 え て い ま す か ？ 全 部 中 に 出 し た ん で す よ 。 下 着 が 精 液 の ニ オ イ で 一 杯 に な っ て 、 い つ 母 親 に バ レ る か と 、 ド キ ド キ も の だ っ た ん で す 。 」

あ る 時 な ど は 、 射 精 す る 直 前 に 、 す み れ が 無 理 矢 理 避 妊 具 を 取 り 外 し て し ま い 、 そ の ま ま 間 に 合 わ ず 、 大 量 に 射 精 し て し ま う こ と も あ っ た 。 そ れ も 回 数 が 増 え れ ば 、 無 邪 気 な い た ず ら で は 済 ま せ ら れ な い 。 何 度 か す み れ を 諫 め た こ と を 思 い 出 す 。

「 桐 島 先 生 と 一 緒 に 過 ご し た 高 校 生 活 の 一 番 の 思 い 出 は 、 先 生 と 数 え 切 れ な い 位 エ ッ チ を し

ま く っ た こ と か な ？ 一 生 分 の セ ッ ク ス を 楽 し ん だ っ て 感 じ た も の 。 」

確 か に 如 月 の 言 う 通 り か も し れ な い 。 正 月 や お 盆 を 含 め て 、 一 年 中 活 動 し て い る ブ ラ バ ン で 、 顧 問 の 桐 島 と 主 将 の す み れ は 、 帰 宅 が 毎 夜 深 夜 に 及 ん で い た 。 建 物 の 管 理 と 戸 締 ま り 一 切 を 任 さ れ て い た 桐 島 で あ る 。 そ の 桐 島 と 一 緒 に 帰 宅 す る す み れ の 目 的 は た だ ひ と つ 。 顧 問 専 用 に あ て が わ れ て い た 畳 敷 き の 宿 直 室 で 、 存 分 に 交 わ る こ と だ け で あ っ た 。

「 先 生 か ら 、 私 は あ ら ゆ る テ ク ニ ッ ク を 教 え 込 ま れ ま し た 。 中 で イ ク っ て い う 喜 び を 知 っ た の は ひ と 月 も し な い う ち だ っ た わ 。 そ れ も 、 ま だ 高 ２ の 夏 で す よ ！ そ れ に 先 生 は 、 私 の 制 服 姿 に 異 様 に 興 奮 す る か ら 、 制 服 で 交 わ っ て ば か り い ま し た よ ね ？ お か げ で 、 新 し い 制 服 、 高 ３ に な っ て 新 調 し た ん で す よ ！ ひ ど い と 思 い ま せ ん ？ 」

す み れ の 言 う 通 り 、 美 少 女 の す み れ が 相 手 と は 言 え 、 何 度 も 体 を 交 わ ら せ て い れ ば 、 新 た な 興 奮 が 欲 し く な る も の で あ る 。 練 習 中 、 他 の 部 員 が 残 っ て い る 間 に も 、 顧 問 の 部 屋 に す み れ を 呼 び 込 ん で は 、 数 え 切 れ な い ほ ど バ ッ ク か ら 犯 し た こ と を 思 い 出 す 。

壁 ひ と つ 向 こ う で 、 何 も 知 ら ず に 練 習 に 励 ん で い る 他 の 女 生 徒 に バ レ な い よ う 、 シ ョ ー ツ だ け 外 し て 背 後 か ら 交 わ る ス リ ル を 二 人 で 楽 し ん だ 。 し か も 、 フ ィ ニ ッ シ ュ は す べ て 中 出 し で あ る 。 ウ ブ な 女 生 徒 ば か り だ か ら バ レ ず に 済 ん だ が 、 生 臭 い 性 臭 は 、 音 楽 棟 の 中 に 確 か に 漂 っ て い た は ず で あ る 。

「 そ ん な 関 係 も 、 私 が 卒 業 し た 途 端 、 一 方 的 に 拒 否 す る な ん て 、 ひ ど い と 思 い ま せ ん ？ た と え 、 先 生 の 奥 様 が 妊 娠 し た か ら っ て 、 そ れ を 理 由 に 私 と の 関 係 を 断 ち 切 る な ん て 、 あ ん ま り だ わ ！ 」

す み れ が 不 満 を 漏 ら す よ う に 、 二 人 の 爛 れ た 肉 体 関 係 に 終 止 符 を 打 っ た の は 桐 島 本 人 で あ る 。 そ れ と い う の も 、 生 理 期 間 中 を 除 け ば 、 ほ ぼ 毎 日 密 壺 の 奥 深 く に 精 液 を 飲 み 込 ん で い る 教 え 子 が 、 い つ 妊 娠 す る か 冷 や 冷 や も の だ っ た か ら で あ る 。

そ れ と 同 時 に 、 め っ き り 妻 と の 性 生 活 は 回 数 が 減 っ て し ま い 、 妻 の 不 満 が 蓄 積 し 始 め る 。 ど ん な に 精 力 剤 を 服 用 し て も 、 妻 の 膣 奥 に 注 ぎ 込 め る 精 液 の 絶 対 量 は 減 る 一 方 。

「 あ な た 、 も し か し ら 浮 気 し て い る ？ 」 そ う 妻 が い ぶ か っ た の も 当 然 で あ る 。

も と も と 体 力 ・ 精 力 と も に 自 信 を 持 っ て い た 桐 島 が 、 極 端 に 痩 せ て き た の は こ の 頃 の こ と で あ っ た 。

「 桐 島 先 生 、 ダ イ エ ッ ト が 成 功 し ま し た ね ！ 」 同 僚 か ら 褒 め ら れ て も 、 言 い 返 す 言 葉 が 見 つ か ら な い 。 そ ん な 中 、 す み れ た ち 高 校 ３ 年 生 が 卒 業 し 、 同 時 に 二 人 の 乱 れ た 肉 体 関 係 は い っ た ん 終 止 符 を 打 つ こ と が 出 来 た 。 そ し て 、 伊 織 の 妻 も そ の 頃 、 無 事 妊 娠 す る 。

「 あ れ だ け 、 私 の 体 に こ れ を 刻 み 付 け た の に 、 さ っ さ と 関 係 を 清 算 し ち ゃ う な ん て 、 ひ ど す ぎ ま す 。 」

周 り に は ば か る こ と な く 、 す み れ は カ ウ ン タ ー の 下 か ら 伊 織 の 股 間 を 的 確 に ま さ ぐ り 出 す 。 た と え ９ 年 経 っ た と し て も 、 あ の 頃 の 関 係 を す み れ は 忘 れ て い な い の だ 。 巧 み な 指 さ ば き で 、 伊 織 の 先 端 を 弄 び な が ら 、 す み れ は 伊 織 を 見 つ め 続 け る 。

「 あ の 頃 、 先 生 は 、 ア ル ト サ ッ ク ス の 吹 き 方 を 真 似 て 、 私 に こ れ の し ゃ ぶ り 方 を た っ ぷ り 教 え て く れ ま し た よ ね ？ 私 は も っ と 先 生 と の 関 係 を 楽 し み た か っ た の に 、 臆 病 な 先 生 は 私 か ら 逃 げ 出 し ま し た 。 私 の 気 持 ち な ん て 、 こ れ っ ぽ っ ち も 理 解 し て く れ よ う と し な か っ た で し ょ う ？ 」

周 り に 他 の お 客 が 居 な い こ と を 好 都 合 に 、 す み れ は フ ァ ス ナ ー か ら 懐 か し い 伊 織 の 男 根 を 直 に 取 り 出 す 。 す で に ガ チ ガ チ に 勃 起 し 、 鈴 口 か ら は 透 明 な 粘 液 が お び た だ し く 漏 れ て い る 。 そ の 粘 液 を 亀 頭 か ら 肉 幹 全 体 に ま ぶ し な が ら 、 す み れ は 夢 中 で し ご き 出 す 。

「 う わ ぁ ー 、 久 し ぶ り ー 、 こ の 感 触 ！ こ の ペ ニ ス が 私 の 青 春 の 全 て だ っ た の に 。 先 生 っ た ら ひ ど い 人 。 だ か ら 今 夜 は た っ ぷ り 意 地 悪 し て あ げ る ん だ か ら ！ 」

バ ッ ク ミ ュ ー ジ ッ ク が 流 れ て 無 け れ ば 、 す み れ の 手 元 で ク チ ャ ク チ ャ 音 を 立 て る 肉 径 の 卑 猥

な 音 は 丸 聞 こ え だ ろ う 。 す み れ が こ ん な 公 衆 の 面 前 で 、 伊 織 を 射 精 さ せ よ う と し て い る こ と は 見 え 見 え で あ る 。

「 私 は あ と 半 年 後 に 結 婚 す る の 。 で も 、 そ の 前 に ど う し て も や り 残 し て い る こ と が あ る わ 。 」

い よ い よ す み れ の ス ラ イ ド は ス ピ ー ド を 増 し て い く 。 精 嚢 が び く び く し 始 め 、 射 精 が 間 近 い こ と を 伊 織 に 感 じ さ せ た 。

「 先 生 、 ダ メ で す よ 。 こ ん な 所 で イ ッ ち ゃ っ た ら 許 さ な い か ら 。 下 の 階 に 部 屋 を 取 っ て あ る か ら 、 今 夜 は そ の 部 屋 で 、 私 と 散 々 し て 下 さ い 。 そ れ ま で 我 慢 し な い と 、 奥 さ ん に み ん な バ ラ し ち ゃ い ま す よ ！ 」

喜 々 と し て 微 笑 む ２ ７ 才 の 教 え 子 は 、 妖 艶 な 目 つ き で 、 伊 織 と 手 の 中 の 肉 棒 を 交 互 に 見 比 べ る 。

こ ん な 場 所 に の こ の こ 姿 を 現 し た 自 分 が 浅 は か だ っ た の か も し れ な い 、 そ ん な 後 悔 が 脳 裏 に 浮 か ん で く る 。 よ う や く 断 ち 切 っ た は ず の 教 え 子 と の 禁 断 の 関 係 が 、 よ も や こ う や っ て 復 活 す る と は 思 っ て も み な か っ た 。

そ ん な 桐 島 に 、 す み れ は 肉 棒 の ス ラ イ ド を 強 め 、 畳 み か け る 。

「 そ れ と ね 、 先 生 、 こ れ か ら 結 婚 ま で の 半 年 間 ... え ー と 、 大 体 １ ０ ０ 日 位 か し ら 。 昔 二 人 で 約 束 し て お い た 、 ア ル ト サ ッ ク ス の 練 習 に お 付 き 合 い し て 頂 き ま す か ら ね 。 も ち ろ ん 楽 器 は 、 今 私 の 手 の ひ ら の 中 に あ る こ の ペ ニ ス で す け ど 。 」

シ レ っ と 言 い 放 ち な が ら も 、 保 育 士 如 月 す み れ は 妖 し い 微 笑 み を 口 元 に 浮 か べ 、 先 端 か ら こ ぼ れ だ し て い る 粘 液 を 肉 棒 に ま ぶ し 続 け る 。 射 精 す る こ と も 許 さ れ ず 、 生 殺 し の ま ま 下 半 身 を 弄 ば れ 続 け る 桐 島 伊 織 は 、 こ れ か ら 繰 り 広 げ ら れ る １ ０ ０ 日 間 の 個 別 レ ッ ス ン を 悲 壮 な 思 い で 考 え 始 め て い た 。
