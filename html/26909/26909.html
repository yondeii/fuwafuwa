「 Weekend love story 」

～ bloom in one's heart ～


私は恋をしていた。

彼は週末になると向かいのカフェのオープンテラスでいつも本を読んで いた。


始めは何となく店のガラス越しに外を眺めていたら、

週末になると見かける事に気が付いた。

知らない間に彼を意識するようになっていた。

これが恋だと理解するのに時間はかからなかった。

私はガラス越しの週末が楽しみになった。


週末がくる度に彼を見ていると、

本を読みに来る事が目的じゃない事に気が付いた。

彼も恋をしていた。

相手は、カフェの店員だった。


叶わぬ恋と分かっていた私は、彼を応援する事にした。

ガラス越しで声が届くわけは無いのに ...


ある日、彼が店にやってきた。

彼「すいません。この『花』を下さい。」

彼は私を指差して言った。

花屋「プレゼントですか？」

彼「はい。大切な人に贈ろうと思って・・・」

花屋「彼女ですか？」

彼「そうなれば良いなと思ってます ( 笑 ) 」

花屋「そうなんですか ( 笑 )   　頑張って下さい。」

彼「はい。ありがとうございます。」


私を握りしめ、彼はいつものカフェのいつもの席へ向かい座った。

カフェの店員がやって来た。

店員「いらっしゃいませ。いつものでいいですか ? 」

彼「はい。あのう ... 」

店員「はい？ あら、綺麗な花ですね。」

彼「あのう ... 」

店員「？ ... 」

私 ( 声なき声 ) 「頑張って ... 」

彼「これを受け取って下さい。」

店員「私にですか？」

彼「はい。ずっとアナタが好きでした。」

店員は花を受け取った。


私は、店員の笑顔を見て恥ずかしそうに笑う彼の笑顔が嬉しかった。

ガラスケースを出た私は、彼と店員の恋の行方を見届ける事ができず、

消えゆく意識の中で「せめて、彼と一言でも話しをしたかった。」

っと声なき声で願うことしか出来なかった。

私の「週末の恋」が終わりをつげた。




















～ epilogue ～

どれくらい眠っていたのか、起きていたのか分からない時間を過ごしたのだろう。


目覚めると、目の前に彼とあのカフェの店員の顔があった。

私を見て 2 人とも微笑んでいた。

私は 2 人の子供として産まれていた。


月日が立ち、

物心がつき、思ったように喋れる様になった頃・・・

不意に昔の事を思い出した私は、思い切ってお父さんに話してみた。

私「お父さん、あのね ... 」

「 .................................... 」

お父さん「あ～、あの時 声をかけてくれたのは、○○だったのか。」


お父さんは、私の話しを信じて優しく頬笑んでくれた。


お父さん「普通は誰も信じ無いと思うが、お父さんはお母さんに

告白する時に確かに誰かの声を聞いた気がするんだ。」

「ありがとう。◯◯。これから、いっぱいお話しできるよ。」

私は嬉しくて泣きながら抱きついた。

お父さんは優しく背中を撫でてくれた。

そして、私に静かに優しく語りかけてきた



お父さん「大切な人に出会うと、その人を失わない様に願う事があるんだ」

「そして、出会わせてくれて、ありがとう ...

そう祈る事もある ... 」

「誰かを大切に思う事って理屈じゃない ...

時に見た目や身体つきで、その人を好きになる事もあるかも知れない ... 」

「理屈じゃないって、そう言う理由が無いから ...

言葉で表せない無いんだよ ... 」


私は、お父さんの胸の中で声を上げずに泣いていた。


お父さん「お前も、お父さん以外で見つけることができたら ...

勇気を持って声をかけてごらん ... 」

「待ってるだけじゃ、幸せになれないよ ( 笑 ) 」

「まぁ、お父さんとお酒を一緒に飲める人が良いけどな～♬」


私は、この人以上に好きになる人に出会えるのか少しだけ

不安になった ...





































THE END






















『 Life of Music 』

～ Love Song ～


彼女「今なにしてた？」


僕「晩御飯が終わってベランダから町をながめてた ... 」


彼女「また、 BGM は尾崎豊でしょ ? 」

そんなたわいの無い会話をするのが僕らのいつもの日課だった。

僕らは大学のサークルで出会い付き合い始めた。

大学を卒業後、僕は地元の企業に就職して、彼女は都会の企業に就職をした。



就職してから仕事を覚えるのに忙しくなり、お互いの休みの日に会うって付き合い方をしていた。


お互いに忙しくなってくると普段はメールがほとんどで、

電話をする時はいつもこんな感じだ。


その日の出来事や電話が出来なかった時の出来事、テレビのドラマの感想 ...

いつも、ありきたりの会話だった。


電話をする時に僕はいつも今の気分で音楽をかけていた。


彼女はいつも『誰のファンなのって？』聞いてくるけど、その時の気分で聴いているので特に決まっていなかった。


彼女からすると、明るい感じや暗い感じとか聴いてる曲で僕の気分が分かるらしい。


ただ、コロコロと変わるのでどこかフワフワしてつかみ所の無いと言うのが印象として強いらしい。


僕からすると、そんな事を考えた事も無いから良くわからなかった。

彼女は僕の事を誰よりも理解してくれていたと思う・・・


彼女「ねぇ、もし私がアナタ以外の誰かに一緒にいて欲しいって言われたらどうする？」

僕「え？急にどうして？」

彼女「もしもの話しだよ（笑）」

僕「もしもでも、いい気分にはなれないよ ,,, 」

彼女「そう ... 」

独り言のようにつぶやいて、彼女はその話題を辞めた ...

僕はこの時、彼女からのシグナルを感じる事が出来なかった。


週末になると僕は彼女の住む市内に会いに行った。

彼女の買い物に付き合い、行ってみたかった店に一緒に行き、

その時によって彼女の家に泊まったりして、僕はこの時間が永遠のに続くもの勘違いしていた。

恋の終わりは、突然やって来た ...





いつもの様にカフェで話していると、彼女は話しを切り出してきた。

彼女「ねぇ、キミの夢って何なの？」

僕「う～ん ... 急に言われると困るけど、強いて言えば君と幸せな家庭を築きたいかな（笑）」

彼女「え～ ... それってプロポーズ？」

僕「そうとってもらってもかまわないけど、君はどうしたいの？」

彼女「私は幸せになりたい ...

アナタはいい人だけど、私はこの街に来て変わってしまった ... 」

「私は昔の私じゃなくて、もっと違う生活がしたいの ...

ごめんなさい、アナタにはもっと相応しい人がいるわ ...

私みたいな女じゃなくて ... 」

そういって彼女は席を立ち、出口に向かって行った。

僕は呆気にとられて、引き止める事も出来ずに席に座りっぱなしになっていた。


それ以降、彼女からの連絡がいっさい無くなった ...

連絡を取ってもいつも、留守番電話になった。


僕は悲しいと言うより、彼女のサインに気付かない自分を

責めていた。

  事情を聞いた男友達は、飲みに行くたびに言葉をかけてくれた。

『元気出せよもっといい女は他にいるって・・・』

飲み仲間はこぞってそういった。

『見返してやれよ。』

『普通の田舎娘を捕まえろ。』

色んな意見を聞かされた。

僕にはどれも違うとしか思えなかった。


僕は自分の事が分からなくなった ...


僕はいいヤツなのだろうか？

好きな人を満足させてやれるのだろうか？


いつもその事ばかり考えていた ...


その度に音楽が僕を救ってくれた。


時には、僕が言いたい事を叫び ...

『あ～、そうだったのかと』答えを教えてくれたような気がした。


その音楽を作っているのは、同じ人間なんだと身体で理解したのは、

彼女の言葉に僕は愕然とした。

君は真面目すぎるんだよ（笑）

意味の無い歌詞なんて、当たり前で聞き手がどう受け取るかだよ（笑）」

流す事も出来るのに不器用に受け止めてしまっていいネタ元だよ（笑）」

彼女「も～ ...

～ epilogue ～

彼女との交際はすぐに週刊誌に載ったが、彼女は

「私はアイドルじゃありませんので ( 笑 ) 」とキッパリと交際宣言をして世間を賑わせていた。


そして、数年後には結婚をして 2 人の子供にも恵まれたが彼女は歌手を続けいた。

2 人目の子供の事が分かった時、何故か会社の方から専業主夫になる事を勧められて、今は子育てに奮闘中だ。

イクメンって奴の仲間入りをしていた。


ある時、長女が質問をしてきた。

長女「ねぇ～、パパ？」

僕「なんだい？」

長女「どうしてウチはママだけが働いているの？」

ママ「もう、パパも働いているわよ ( 笑 ) 」


彼女は焦った感じで訂正してきた。

長女「でも、パパってトモちゃんのママと同じで家の事しかやってないよ～」

僕はポリポリと頭をかいた。

確かに世間のママと行動が同じようになっている。


彼女は、長女が分かるように言葉を選んでいた。

ママ「パパはね、ママの為にいつも詩を書いてくれてるの。

だから、ママが歌えるのはパパがいるからよ。」

長女「ふ～ん ... パパって偉いんだ～

ゴメンね、パパ ( 笑 ) 」


長女がペコりと頭を下げて謝ってきたので、『気にしてないよ』って言うしかなかった。


ママ「本当にこの子はアナタの子よね～

天然な所がパパそっくり ( 笑 ) 」

彼女は、あの時と変わらない笑顔で笑っていた。


僕と娘はお互いに目をパチクリさせながら笑うママを見ていた。


そんな彼女の曲は、今日も TV の CM で流れていた。
