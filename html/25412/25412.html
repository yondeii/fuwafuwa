　 あ が り 症 は 、 話 す 前 も 話 し て い る と き 、 話 し 終 わ っ た 後 も 纏 わ り つ い て く る の で 、 で き れ ば あ が り 症 を 克 服 し た い と 思 う 方 は 多 い の で は な い で し ょ う か 。

　 本 書 で は 完 璧 と は ま で は い か な く と も 、 そ ん な あ が り 症 を 少 し で も 減 ら せ る 考 え 方 を お 伝 え で き れ ば と 思 い ま す 。

第 一 章 　 人 前 で 話 す と き の 考 え 方

　 ま ず 、 人 前 で 話 す こ と が 苦 手 で 辛 い と 思 っ て い る 人 の が ど の く ら い の 割 合 で い る か を お 伝 え い た し ま す 。 実 は 九 割 強 の 人 が 話 す こ と が 苦 手 と い う 割 合 に な っ て い ま す 。 つ ま り 人 前 で 話 す こ と が 苦 手 と お っ し ゃ っ て い た あ な た は そ の 九 割 に い る と い う こ と に な り ま す 。 ほ と ん ど の 人 が そ の 悩 み で 悩 ん で い る の で す 。 逆 に 人 前 で 話 す こ と が 得 意 と 思 っ て い る 人 の 割 合 の 方 が 圧 倒 的 に 少 な い こ と に な り ま す 。

　 で す の で 、 あ が り 症 で 話 す こ と が 苦 手 と 思 っ て い る こ と は 何 も お か し い こ と で は な い の で す 。 ま た 人 前 で 話 す こ と が 得 意 と 答 え た 一 割 の 人 が 聞 き 手 に 満 足 の い く お 話 を で き る と は 限 り ま せ ん 。

　 こ の 割 合 は 上 手 く 話 せ る 人 と そ う で な い 人 の 割 合 で は な く 自 分 が

緊 張 す る か し な い か の 違 い だ け な の で す 。 さ ら に 言 う の で あ れ ば 緊 張 と い う の は 自 分 が 決 め た こ と で は あ り ま す が 決 し て 他 人 が 決 め た こ と で は あ り ま せ ん 。

　 少 し 回 り く ど い 言 い 方 を し ま し た が 結 論 を 申 し 上 げ ま す と 、 あ が り 症 で 緊 張 し て し ま う こ と は 普 通 の こ と な の で あ が っ て し ま う こ と に 対 し て 深 く 考 え な い こ と が あ が り 症 を 克 服 す る た め に 必 要 な 第 一 歩 に な り ま す 。

第 二 章 　 あ が り 症 に な ら な い た め の 自 身 の つ け 方

　 第 一 章 で は 人 前 で 話 す 際 の あ が り 症 に 対 す る 考 え 方 を お 伝 え し て き ま し た 。 こ の 章 で は い か に 緊 張 を な く す か を お 伝 え い た し ま す 。

　 ど の 様 な こ と を し た ら 良 い か 、 最 初 に 結 論 か ら 申 し 上 げ ま す と 「 自

分 の 話 し て い る と き の 声 、 内 容 を 録 音 す る 」 と い う こ と で す 。 最 初 は 自 信 が な く 聞 き た く な い と 思 う か も し れ ま せ ん が 、 こ の 方 法 が あ が り 症 を 克 服 す る た め に 最 も 重 要 に な り ま す 。

　 自 分 の 声 と い う の は 自 分 で 聞 い て い る も の と 他 人 が 聞 い て い る も の に 差 が あ り ま す 。 で す の で 、 自 分 の 声 が 他 人 に ど の よ う に 聞 か れ て い る か 把 握 す る こ と で 自 分 の 話 の 弱 点 が 見 え て き ま す 。

　 最 初 は 自 分 の 声 な ん て 聞 き た く な い と 思 う か も し れ ま せ ん が 実 践 し て み る こ と を お 勧 め し ま す 。 少 し 厳 し い 言 い 方 を し ま す が 、 あ な た が 話 し て い る と き は 、 あ な た が 聞 き た く な い と 思 っ て い る 声 を み ん な が 聞 い て い ま す 。 自 分 が 納 得 で き る よ う な 話 し 方 を 研 究 し て 相 手 に 話 す よ う に し ま し ょ う 。

　 そ し て 自 分 の 話 し て い る と き の 声 を 録 音 、 弱 点 を 修 正 し て い く う ち に 自 分 で も 自 信 が 持 て る よ う な 話 す 力 が 身 に 付 き ま す 。 こ う い っ た 事 前 準 備 こ そ が 自 信 に 繋 が り 、 あ が り 症 の 克 服 に も 繋 が り ま す 。

終 わ り に

　 最 後 ま で お 読 み く だ さ り 誠 に あ り が と う ご ざ い ま し た 。 人 前 で 話 す と き の 考 え 方 や 事 前 準 備 の 大 切 さ を 書 か せ て い た だ き ま し た が 、 や は り 私 の 経 験 上 、 準 備 を す れ ば す る ほ ど 緊 張 は し な く な り ま す 。 も ち ろ ん 緊 張 を ゼ ロ に す る と い う の は 難 し い こ と で は あ り ま す が 、 自 信 を つ け る こ と で 自 分 は あ が り 症 と 思 う こ と も 少 な く な り ま す 。

　 短 く 書 か せ て い た だ き ま し た が 、 本 書 が き っ か け で 人 前 で 話 す こ と に 対 し て の 苦 手 意 識 が 少 し で も な く な れ ば 良 い と 思 っ て い ま す 。
