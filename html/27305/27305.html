はじめに

 

私は、これまで、若い日本語教師の悩みをたくさん聞いてきました。

中国のある大学で授業をしていたときの話です。ある教師が「生徒が授業に興味も関心も示さないんですよ！」と相談に来ました。もちろん私にも、多少そのような経験があります。最初は、あまり意識しなかったのですが、その教師の授業をよく観察してみると、学生たちは私たちが中学や高校でよくやった「内職」 をしていました。志望校の受験科目にあまり関係ないと思ったとき、 受験の教科である 英語の単語を机の下で覚えたり、さしあたり、今 、 行われているの授業よりも、優先度が高いと考える科目の勉強を したりする 、教師の視線を気にしながら 行う 、いわゆる「内職」 です。

「なにやってんだ！？」 と 、 教師から一喝されたことも一度ならずあったはずです。 しかし、面白くない授業に対する学生たちの反応は、どこの国でも同じです。

中国の大学で日本語を学んでいる学生の多くは 、卒業後、日系の会社で通訳や翻訳の仕事 に就くことを 希望しています。そして、日本語能力試験 一 級に合格することは、 そのような 企業に就職するための登竜門なのです。ですから、彼らは授業中、面白くない授業はほとんど聞いていなくて、日本語能力試験一級の文法編や文章読解の問題を解きまくっているのです。その光景に出くわした私は決心しました。

これからは、 「学生との真剣勝負！」 だと。 『やるか、やられるか』 です。

『行列のできる授業をするぞ！ 』 と決意しました。

『 学生が、身を乗り出して聞くような授業です ( ・・・・・・・・・・・・・・・・・・・・ ) 』

もともと、大学の授業 で は、教師が学生のモチベーションを高める ことを意識した 授業はあまり見かけません。しかし、日本では、カリスマ講師と呼ばれる教師が予備校にはいますよね。『日本語学習者』に行う授業に限っては、そのような魅力ある学習が必要だと 私は 思います。

『体験的にそう思います』

彼らの当面の目標である、日本語能力試験の教材開発を研究しようと思ったのは 、 そのような動機からです。私の手作り教材は、新聞、雑誌からの記事は一切使用していません。著作権の問題もあるのですが、自分の思想、 大げさ に言えば哲学でしょうか。常日頃、 目にする 政治・経済・文化・人間関係や自然との触れ合いの中で自分が感じてきたことを文章にしてみました。

 

【一文の文章でも授業が一コマになることがあります。】

たとえば、授業で、日本料理について文章問題があったとします。 読者の 皆さんも、ご存知のように、日本料理は味だけでなく、『見た目』も重んじます。

そのことから、『日本の食文化』 についての話になります。普通、「今夜、 赤提灯 ( あかちょうちん ) にいくか！？」　 という文章があったら、ほとんどの留学生は 理解できません 。大人の日本人だったら、すぐに、「居酒屋で飲もうよ！」という 『 ものの 喩 ( たと ) え 』 としての 比喩 ( ひゆ ) 表現は分かりますよね。 また 、私は日本 人の サラリーマン を例にとり、 アフターファイヴの説明から、授業に出席している留学生のお国の事情を聞いたり し ます。

それが一つの「意見交流の場」 になります。

この 日本語能力試験問題集は、日本の文化や日本人の考え方を、日本語教師が日本語学習者と対話 しながら学習 できるような内容を考え、教材作成しました。そして、実際の日本語能力試験よりも難しめに作成しています。それは、日本語教師が日本語能力試験を指導する時に、教師自身も学生とともに考える面白い授業ができるように、との思いがあったからです。

実際、私が大学で留学生の日本語能力試験の指導をしていて、面白い授業 とは何かということを常に考えてきました。私の教材は、そのような授業を日本語教師にしてもらいたいと願って作成した教科書です。

 

本書の試験問題は、「言語知識・読解」分野が実際のＮ２試験と同様の形式と問題数でつくられています。なお、本書では 聴 ( ちょう ) 解 ( かい ) は 取 ( と ) り 扱 ( あつか ) っていません。

 

2014 年 1 月

　　　　著者　中村一男

問題９　文章の文法

【文章の流れに合った文かどうかを判断することができるかを問う問題】

 

 

問題　次の文章を読んで、【 　 50 　】から【 　 54 　】の中に入る最もよいものを、 1 ・ 2 ・ 3 ・ 4 から一つ選びなさい。

 

（問題は次のページにあります）

 


 

 

『文章読解の問題』で四択問題を作成してみると、教材作成のおもしろさに気づきます。自分で教材を作成しないで、市販の教材で授業をすると、学生に答えさえ教えればいい、ヒントさえ与えればいいという気持ちになりがちです。（私の場合）

教材を教師自らが作成しないと、このような 【 　 50 　】 が身に付き、内容の浅い授業に 陥 ( おちい ) ることが多い、というのが私の体験から得た結論です。このような授業スタイルは、留学生の顔を見るとよくわかります。途中で自分自身が息切れを 【 　 51 　】 しまい、ステレオタイプの授業しかしていないな、ということに気づいてくるのです。

『教材を自ら作成しよう』これが私の日本語教育の基本的理念なのです。それでは授業に役立つ教材開発とは何か。たとえば、教材の中に次のような文章があったとします。

『私は次の三点を" 頭に入れて ( ・・・・・ ) "文章を創作し、問題を作成しています』

問題　 A と B はそれぞれ、ある雑誌の読者欄に投稿された意見である。 A と B の二つ文章を読んで、次の問いに対する答えとして、最もよいものを 1 ・ 2 ・ 3 ・ 4 から一つ選びなさい。

 

69 　 英語教育は早ければ早いほどいい というのは、なぜか。

1.         早く英語を勉強すれば学校の英語の成績がよくなるから。

2.         英語は早く勉強すれば外国人のような 発音 ( はつおん ) ができるようになるから。

3.         社会人になっても英語の必要性を感じるので、できたら早くから英語の勉強をしておいた方がいいと感じるから。

4.         英語は世界の共通言語であり、 他 ( ほか ) の外国語より大切なのでできるだけ早く始めたほうがいいと思うから。

 

70 　 A と B は英語教育について、どう思っているか。

1.         A も B も英語教育は大切であると思っているが、そんなに早く始める必要はないと考えている。

2.         A も B も早期の英語教育には賛成であるが、日本人の母語である国語の勉強を優先させることが大切であると考えている。

3.         A は早期の英語教育の必要性を感じているが、 B は早期の英語教育よりも、むしろ国語の学習を優先することが大切であると思っている。

4.         A は自分の経験から早期の英語教育に賛成しているが、 B はそれほど英語ができなかったという思いは抱いていない。


A

近年、英語の早期教育について、多くの専門家の間で論争が続いている。英語は国際化時代において世界の共通言語であると言われている。今日、日本の近隣諸国であるアジア各国においても、英語教育に力を入れている国も多いと聞く。そのような状況の中で、日本でも早期教育として、英語は多くの小学校でも教えられている。

学生時代、私自身は英語が得意ではなかった。しかし、社会人になって、仕事で英語を使う機会が増えた。このため、週末の土曜や日曜日など、暇な時間を利用しては、自宅で英語を勉強している。私の経験から言わせてもらえば、やはり英語教育は早ければ早いほどいい と思っている。そうすれば、私のように社会人になってこんなに苦労しなくてもいいのではないかと思うのである。

私は英語の早期教育に携わる先生にエールをおくりたい。

　　（４５歳・会社員）

 

B

小学校の英語教育について、専門家の間で意見が分かれています。確かに、国際化の時代に英語は必要です。私は英語が得意ではありません。私の友達で英語がペラペラの人がいます。少し、うらやましい気がします。

しかし、私は小学校で英語を教えることには反対です。なぜかというと、日本人は、もっと国語を勉強した方がいいと思うからです。国語はすべての教科の基礎になると考えます。英語であろうと数学であろうと 論理的 ( ろんりてき ) 思考 ( しこう ) が求められます。物事を論理的に考える力は、国語を通してだと思います。

ですから、小学校では英語よりも国語の勉強に多くの時間を使うべきだと思います。英語は今までのように中学校からでも十分だと思います。

（２１歳・学生）

　


 

 

問題１３　主張理解（長文）

【論理展開が比較的明快な評論など、 900 字程度のテキストを読んで、全体として伝えようとしている主張や意見がつかめるかを問う問題】

 

 

問題　次の文章を読んで、後の問いに対する答えとして、最もよいものを 1 ・ 2 ・ 3 ・ 4 から一つ選びなさい。

 

私には中学 1 年生と 3 年生の、 2 人の女の子がいます。主人は普通の会社員です。子供たちはとくべつ学校の勉強ができるというわけではありません。でも子供たちは私にも主人に対しても優しく、家での手伝いもよくしてくれます。

ただ子供たちに対して心配することが何も ないというわけではありません ( ・・・・・・・・・・・・・・ ) 。 (1) 少し気 になること は、ときどき学校から帰ってきて 機嫌 ( きげん ) が悪い時があるのです。思春期の子供たちだからといって、何も心配することはないとは、私は考えてはいません。 2 人の子供には共通した「思い 癖 ( くせ ) 」があるのです。何か嫌なことがあると、 クヨクヨ ( ・・・・ ) する癖です。たぶん心の中に、いつかどこかで体験した過去のマイナスイメージが頭にインプットされているのでしょう。機嫌が悪い時には、たぶん一度入力されたその過去のマイナスイメージが「思 い癖」として脳内から出力されるのでしょう。でも、数日もすると子供たちは何事もなかったかのように元気を取り戻します。どうやら時間が経てば、この「思い癖」も (2) 脳から 削除 ( さくじょ ) される ようです。やはり、思春期の女の子に必要なものは (3) 「時間」だったのです 。

私たち大人であっても同じことがいえると思います。人間はそれぞれ、何か特定の出来事に対して、 敏感 ( びんかん ) に反応するようになっているのだと思います。例えば、何かの試験を受けることになった時、自分の過去の経験という 枠組 ( わくぐ ) みで考えます。過去に受験を失敗した経験があれば、必ず今回の試験も失敗するのではないかという「思い癖」が頭をよぎります。でも、過去は過去、今は今、という前向きな気持ちに切り替えて、過去の失敗と距離を置くのが「思い癖」を取り除くいちばんの秘訣ではないかと思います。


しかし、それでもなかなかうまくいかないこともあります。そんな時は、少し他のことに集中してみるといいと思います。やがて、クヨクヨしていたことが、気にならなくなると思うのです。やはり、人間には、 時の過ぎるのを待つ ( ・・・・・・・・・ ) 「知恵」も必要だと思います。結局、子供にとっても大人にとっても、何かを解決してくれる 処方箋 ( しょほうせん ) は「時間」であることだけは間違いなさそうです。

 

 

71 　 (1) 少し気になること とは、ここでは何か。最も適当なものを選びなさい。

1.         子供たちの学校の成績

2.         子供たちのすぐに怒る癖

3.         親から見た子供たちのくよくよする癖

4.         親から見た子供に対する不満

 

72 　 (2) 脳から削除される とは、 具体的にはどういうことか。

1.         脳の働きが鈍感になること

2.         気になっていたことを忘れ去ること

3.         脳の中で何か大切なことを忘れること

4.         脳の働きが弱くなること

 

73 　(3) 時間だったのです とあるが、ここで作者が最も言いたいことは何か。

1.         子供たちの成長には、勉強だけでなく健康な体を作る時間が必要である。

2.         子供たちが母親の愛情を理解できるようになるためには、時間が必要である。

3.         子供たちの悩みを解決するには、ある一定の時間が過ぎ去ることを待つ必要がある。

4.         子供たちの悩みを解決するためには、忍耐強く勉強して待つ時間が必要である。
