はじめに

 

私は、これまで、若い日本語教師の悩みをたくさん聞いてきました。

中国のある大学で授業をしていたときの話です。ある教師が「生徒が授業に興味も関心も示さないんですよ！」と相談に来ました。もちろん私にも、多少そのような経験があります。最初は、あまり意識しなかったのですが、その教師の授業をよく観察してみると、学生たちは私たちが中学や高校でよくやった「内職」 をしていました。志望校の受験科目にあまり関係ないと思ったとき、 受験の教科である 英語の単語を机の下で覚えたり、さしあたり、今 、 行われているの授業よりも、優先度が高いと考える科目の勉強を したりする 、教師の視線を気にしながら 行う 、いわゆる「内職」 です。

「なにやってんだ！？」 と 、 教師から一喝されたことも一度ならずあったはずです。 しかし、面白くない授業に対する学生たちの反応は、どこの国でも同じです。

中国の大学で日本語を学んでいる学生の多くは 、卒業後、日系の会社で通訳や翻訳の仕事 に就くことを 希望しています。そして、日本語能力試験 一 級に合格することは、 そのような 企業に就職するための登竜門なのです。ですから、彼らは授業中、面白くない授業はほとんど聞いていなくて、日本語能力試験一級の文法編や文章読解の問題を解きまくっているのです。その光景に出くわした私は決心しました。

これからは、 「学生との真剣勝負！」 だと。 『やるか、やられるか』 です。

『行列のできる授業をするぞ！ 』 と決意しました。

『 学生が、身を乗り出して聞くような授業です ( ・・・・・・・・・・・・・・・・・・・・ ) 』

もともと、大学の授業 で は、教師が学生のモチベーションを高める ことを意識した 授業はあまり見かけません。しかし、日本では、カリスマ講師と呼ばれる教師が予備校にはいますよね。『日本語学習者』に行う授業に限っては、そのような魅力ある学習が必要だと 私は 思います。

『体験的にそう思います』

彼らの当面の目標である、日本語能力試験の教材開発を研究しようと思ったのは 、 そのような動機からです。私の手作り教材は、新聞、雑誌からの記事は一切使用していません。著作権の問題もあるのですが、自分の思想、 大げさ に言えば哲学でしょうか。常日頃、 目にする 政治・経済・文化・人間関係や自然との触れ合いの中で自分が感じてきたことを文章にしてみました。

 

【一文の文章でも授業が一コマになることがあります。】

たとえば、授業で、日本料理について文章問題があったとします。 読者の 皆さんも、ご存知のように、日本料理は味だけでなく、『見た目』も重んじます。

そのことから、『日本の食文化』 についての話になります。普通、「今夜、 赤提灯 ( あかちょうちん ) にいくか！？」　 という文章があったら、ほとんどの留学生は 理解できません 。大人の日本人だったら、すぐに、「居酒屋で飲もうよ！」という 『 ものの 喩 ( たと ) え 』 としての 比喩 ( ひゆ ) 表現は分かりますよね。 また 、私は日本 人の サラリーマン を例にとり、 アフターファイヴの説明から、授業に出席している留学生のお国の事情を聞いたり し ます。

それが一つの「意見交流の場」 になります。

この 日本語能力試験問題集は、日本の文化や日本人の考え方を、日本語教師が日本語学習者と対話 しながら学習 できるような内容を考え、教材作成しました。そして、実際の日本語能力試験よりも難しめに作成しています。それは、日本語教師が日本語能力試験を指導する時に、教師自身も学生とともに考える面白い授業ができるように、との思いがあったからです。

実際、私が大学で留学生の日本語能力試験の指導をしていて、面白い授業 とは何かということを常に考えてきました。私の教材は、そのような授業を日本語教師にしてもらいたいと願って作成した教科書です。

 

本書の試験問題は、「言語知識・読解」分野が実際のＮ２試験と同様の形式と問題数でつくられています。なお、本書では 聴 ( ちょう ) 解 ( かい ) は 取 ( と ) り 扱 ( あつか ) っていません。

 

2014 年 1 月

　　　　著者　中村一男

問題９　文章の文法

【文章の流れに合った文かどうかを判断することができるかを問う問題】

 

 

問題　次の文章を読んで、【　 50 　】から【　 54 　】の 中に入る最もよいものを、 1 ・ 2 ・ 3 ・ 4 から一つ選びなさい。

もう一つ説明の難しかった文を紹介します。「日本人は 現象に流されやすい ( ・・・・・・・・・ ) 」という抽象的な表現です。抽象的な表現でも、母語話者であれば自動的に理解できます。しかし、日本語学習者がこの表現を理解するためには、日本人の考え方を理解する必要があります。西洋的な思考法は、物事を"論理的"視点から捉える傾向があるのに対して、日本人の思考法は、物事を"情緒的"に捉える傾向があります。 水の上に浮かんだ枯れ葉 ( ・・・・・・・・・・・ ) が、自分では進むことができず、水の流れに任せるしかないように、他人の意見やその場の雰囲気をそのまま受け入れてしまう様子を「 流される ( ・・・・ ) 」と表現しているのです。このような日本人の性格を表す慣用句は、他にも「 寄 ( よ ) らば大樹の 陰 ( かげ ) （注 1 ） 」「長いものには 巻 ( ま ) かれろ （注 2 ） 」などがあります。

私はまだまだ未熟な日本語教師です。留学生に分かるように説明するには" ヘトヘト ( ・・・・ ) " （注 3 ） になります。授業で使う教材は「ちゃんと予習をしておく」ということが大切であるとわかりました。

（注 1 ） 寄 ( よ ) らば大樹の 陰 ( かげ ) ：頼る相手は、力のある者を選ぶのがよいということ。

（注 2 ）長いものには 巻 ( ま ) かれろ：権力のある人や勢いのある人には、素直に従った方が得であるという意味。

（注 3 ）ヘトヘト：ひどく疲れていて、体力や気力がなくなっている様子。

問題　 A と B はそれぞれ教育雑誌に寄せられた意見である。 A と B の両方を読んで、次の問いに対する答えとして、最もよいものを 1 ・ 2 ・ 3 ・ 4 か ら一つ選びなさい。

 

69 　 早急に調査しなければならなくなった というのはなぜか。

1.         家庭での読書が読書習慣を身に付けるための重要な手段だから、もっと家庭内の読書に関するデータが必要と考えたから。

2.         家庭での読書に関するアンケートを、これまで以上に多くの質問事項を増やさなければならなくなったから。

3.         読書に関するアンケート調査によると、ほとんどの生徒が家庭での読書をしていないことが分かり、その原因について早急に調査する必要があると考えたから。

4.         この調査結果は、携帯メールや家庭用ゲームとの関連があると思われるので、早急に調査しなければならないと考えたから。

 

70 　 A と B は、読書習慣を身に付けることについて、どう思っているか。

1.         A も B も、家庭での読書は、読書習慣を身につけるうえで、大切であると考えている。

2.         A も B も、読書習慣を身に付けるためには、学校の図書館を利用することが一番有効であると考えている。

3.         A は学校の図書館を利用する生徒が増加したが、そのことが家庭での読書習慣に必ずしも結びついていないのではという疑問を抱き始めている。

4.         B は家庭での読書は、子供が読書習慣を身に付けるうえで大切であるとは考えていなくて、学校の図書館を利用すべきであると考えている。


A

私は小学校の教師である。今、学校では生徒に読書の習慣を身につけさせるために、昼休みの図書館利用の推進を企画している。そのためか、昼休みに図書館で本を読む生徒が多くなった。私たち教師はこのような図書館を利用する生徒の増加は、国語力アップにつながるものと確信している。同時に本を読む習慣を小学校のうちに身につければ、中学校や高校に入っても本を読む習慣は続くであろうとも思っている。

しかし、私たちの試みに不安がないわけではない。先日、五年生と六年生に実施した読書アンケートの調査結果に驚くような結果が出たのである。アンケートの質問の一つとして「家で読書をしますか」という問いに対し、実に 95 ％の生徒が家では読書をしていないという調査結果が出たのである。 私たちは、生徒が家でなぜ読書をしないかということについて、早急に調査しなければならなくなった のである。

（４１歳・学校教師）

 

B

私は小学生の母親です。先日、子供が学校からの「おたより」を持ってきました。それによれば、家で親と子供が一緒に本を読み、読書習慣を身につけようという 趣旨 ( しゅし ) のものでした。

確かに家で子供が読書をすることはよいことだと思います。

しかし、短い時間とはいえ、親が子供と一緒に毎日読書をしなければならないなんて、聞いたこともないし、第一、親は食事の後かたづけや雑用があり、とても子供と読書をする時間がとれません。親と一緒に読書したからといって、子供の読書習慣が身につくものではないと考えるのは私だけでしょうか。

何事も強制されたものは、長続きするものではありません。読書は子供自身が興味を持つ本を読むことから始めることが大切で、推薦された本や与えられた本を読んでも、長い目で見たら読書嫌いの人間にしかならないのでないか、とすら考えるのです。

（３１歳・主婦）


 

 

問題１３　主張理解（長文）

【論理展開が比較的明快な評論など、 900 字程度のテキストを読んで、全体として伝えようとしている主張や意見がつかめるかを問う問題】

 

 

問題　次の文章を読んで、後の問いに対する答えとして、最もよいものを 1 ・ 2 ・ 3 ・ 4 から一つ選びなさい。

 

大企業や官公庁に雇用されることが、大学に行く大きな目的となっていたのが、私たちの時代でした。これは、日本人の「 寄 ( よ ) らば 大樹 ( たいじゅ ) （注 1 ） 」の意識ではなかったかと思います。しかしながら、近年のグローバル化した社会では、私たちの時代と異なり、絶対的な安定した企業がなくなりつつあります。「今日は一流でも、明日は倒産」ということも、 珍 ( めずら ) しくなくなりました。もう、 (1) 国や企業をあてにする時代ではなくなった のです。

大学を卒業しても、仕事がない。――大学を出ても就職が困難な状況は、一般的に、景気等のマクロ経済的要因と、職を求める求職者と採用する企業側とのニーズのミスマッチが、主な原因です。景気の良かった時代には、多くの大学生が 5 社～ 10 社の内定 （注 2 ） をもらっていました。その時々の経済の動向が主な要因で、大学生の就職が大きく左右されているのは、この事からも 一目瞭然 ( いちもくりょうぜん ) です。

しかし、このような状況を 嘆 ( なげ ) くだけでは、問題を解決することはできません。" 発想の転換 ( ・・・・・ ) "を私は若者に期待したいのです。

「就職がなければ、自分で起業をする」――これが就職難に苦しんでいる若者への私からのエールです。誰でも、何か 得意なこと ( ・・・・・ ) はあるのです。その得意なものを、「 売り ( ・・ ) 」にして起業するのです。一人でも、あるいは友達とでもいいと思うのです。「 ゼロ ( ・・ ) 」から出発するのです。


"若者は失う物はありません"。いや、むしろ、若いうちの前向きな冒険は、 (2) 毒 ( ・ ) に ( ・ ) な ( ・ ) る ( ・ ) ど ( ・ ) こ ( ・ ) ろ ( ・ ) か ( ・ ) 薬 ( ・ ) に ( ・ ) な ( ・ ) る ( ・ ) のです。「人生の遠回り」とも思えるかもしれませんが、若いうちの遠回りは、自分の「 引き出し ( ・・・・ ) 」がたくさんできるのです。若者の野性味 溢 ( あふ ) れる生命力は『思いもよらないこと』を実現するのです。

私の友達で、長年務めた銀行を中途退職し、世界に飛び出した男がいます。今、彼は、海外旅行を企画し、多くの日本人を海外に送り出すビジネスをしています。最初はそれこそ「ゼロ」からの出発だったそうです。

自分で起業し、社長になり、理想の会社作りをする。――全国で何万人にもの若者が企業を起こし、 2 ～ 3 人の友達と一緒に汗を流す。そうすれば、全国で、何百万人の雇用が創出できるのです。たとえ、 (3) 就職難の時代でも発想の転換一つで、現実の 厳しい社会の壁 ( ・・・・・・・・ ) を克服することができる のです。いつの時代でも、『若者が新しい時代の幕を開けてきた』のです。それは、歴史が証明しています。

（注 1 ）寄らば大樹（の陰）： 権力を持ったものには、従った方がよいということ。

（注 2 ）内定：正式発表の前に、内々に採用を知らせる行為。

 


 

71 　 (1) 国や企業をあてにする時代ではなくなった とあるが、その理由は何か。

1.         国だけでなく、企業も労働者のことを真剣に考えなくなった時代になったから。
