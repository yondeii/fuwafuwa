 Money . お金の正体。      　  

 仮想通 貨 ver.  

 国民に「仮想通貨とはどのようなものか？」とアンケートをとると、「ネットのお金でしょ」「なんよく分からないけど怪しい」といった意見がほとんどではないだろうか。  

 そこで今回は、みんな興味はあるけどよく分からない「仮想通貨」について話していきたいと思う。  

 これを読めば、「仮想通貨」というキーワードを聞いたときに、詐欺などにひっかかることがなくなるはずだ。  

 よく分からないのに首を突っ込むから、リスクを判断出来ずに誤った判断をしてしまう。  

 みなさんのまわりにもいないだろうか。  

 なんとなく仮想通貨を購入してみたが、結局損をしている人達が。  

 それもそのはず。  

 ビットコイン以外を「アルトコイン」と呼ぶが、その「アルトコイン」 の 9 0 パーセント以上は「詐欺」と言われている現状があるからだ。  

 そういったことを知らないと、痛い目を見ることになる。  

 何事も、本質を見抜くことが大切なのだ。  

    

 １、　ビットコイン。  

 仮想通貨は、「ビットコイン」と「アルトコイン」という２種類に分類することが出来る。  

 「ビットコイン」は、この仮想通貨の世界に初めて登場した通貨だ。  

 世界初なので、テクノロジーがブレイクスルーして生まれた経緯がある。  

 それ以外を、「アルトコイン」と呼ぶ。  

 いわゆる巨人軍が「ビットコイン」で、他の球団が「アルトコイン」という考え方で大丈夫だ。  

 この「アルト」という言葉には、「代替」という意味がある。  

 馴染みのない言葉なので、余計に分かりづらいかもしれないが、実態は至ってシンプルだ。　  

 そして、このアルトコインは世界中に存在しており、なん と 200 0 種類に及ぶと言われている。  

 この数になると、把握するのも難しくなってくるのだが、こうしている間にも種類は増え続けているのが現状だ。  

 そしてそ の 9 0 パーセント以上のコインは「詐欺」と言われており、このアルトコインをしっかりと理解していないと「仮想通貨」のリスクにはまってしまうことになる。  

 自分の周りでも、「詐欺にあった」と嘆いている人を見たことがあるかもしれない。  

 しかし、仮想通貨に限らず、よく仕組みを理解しないまま投資に手を出すと、利益を出せる確率はガクッと下がる。  

 では、なぜこれほどまでに「詐欺」が多いのか。  

 簡単に言うと「セキュリティに問題がある」からだ。  

 しかし、これだけ聞いても本質は見えてこないだろう。  

 しっかりと把握する為には、それぞれの仮想通貨を詳しく知っていく必要がある。  

 そして、ビットコインに比べなぜアルトコインがこれほどまでに多いのか。  

 これは、 「 IC O 」という言葉がキーワードになってくる。  

 これも、馴染みがない人がほとんどだろう。  

 「 IC O 」とは、 「 Intia l　 Coi n　 Offerin g 」の頭文字をとった略称である。  

 資金調達したい企業が、独自の仮想通貨を発行したえ投資家に販売し、資金調達を行うことをいう。  

 最近ようやく浸透してきたクラウドファウンティングに近いが、先にコインを発行する点で違う。  

 そして、各企業が独自仕様の仮想通貨を発行したため、アルトコインの種類が激増したのだ。  

 ビットコイン・・ ・ 201 4 年に、日本でデビューしたコイン。  

 「信用できる」ものだが、「金」よりも価格変動性が激しい。  

 その証拠に、ニュースで「流出した」とか「暴落した」という言葉をよく耳にするだろう。  

 その度にニュースやウェブで取り上げられ、仮想通貨を保有している人たちを焦らせているのだ。  

 それに加え、「暗号通貨」という言葉の「暗号」という言葉が「怪しい空気」を醸し出している。  

 一言で表すと、ビットコインは「お金」ではなく決済システムである。  

 suic a などの電子マネーと、使い方はあまり変わらない。  

 目の前にそれっぽいコインを置かれて、「これがビットコインですよ」と言われて、信じてしまう人もいるのではないだろうか。  

 アルトコインを購入する場合は、ビットコインで購入する形が多い。  

 こう聞いても、まだ分からない人は多いと思う。  

 では、アルトコインとは一体どのようなものか。  

 実態に迫っていこう。  

 多くのアルトコインにビットコインの特徴が引き継がれている。  

 銀行システムの場合、開発を請け負う業者が徹底的に不具合を精査して、しらみつぶす。  

 ビットコインの場合、世界中の有志であるエンジニア達がそれをしらみつぶす。  

 ビットコインは全てが公開されているから、世界中の誰にでも精査や監査ができる。  

 いわゆる、常に「監視下」に置かれている状態だ。  

 ビットコインは、電子マネーとして使える特徴を持っている。  

 ビットコインを利用して商品を購入する場合には、まず「円」という通貨をビットコインに両替しなくてはならない。  

 私達のお金を、ドルなどの外貨に両替するイメージだと分かりやすい。  

 その後の流れは、一般的な電子マネーと一緒。  

 一般的な通貨で送金を行う際には、銀行などを仲介する必要がある。  

 しかし、ビットコインの場合には、個人間が直接支払うことが出来る。  

 そして、基本的には、手数料が無料。  

 これは、大きなポイントだろう。  

 また、国によって単位が異なることもない。  

 ビットコインには、「ウォレッ ト( 財 布) 」という概念がある。  

 円にたとえると、銀行口座に当たるもの。  

 すべての人が、ウォレットを持つことになる。  

 一般的には 、 Q R コードに変換して使う。  

 では、円やドルとの決定的な違いは何だろうか。  

 それは、通貨を管理する「中央銀行」が存在しないことだ。  

 つまり、発行を司る組織が存在しない。  

 その代わりに、コンピュータのネットワークを利用して通貨を管理する仕組みになっている。  

 すべての取引記録が記載された一つの大きな取引台帳が存在する。  

 そして、ビットコインの発行総量は、事前に決められている。  

 もちろん、マイニングによって発行される量も調整されることになる。  

    

 では、ビットコインを説明する上で欠かせないキーワードの「ブロックチェーン」とは、一体何であろうか。  

 それを一言で表すと、「共有された公開台帳」と言うことが出来る。  

 形は、まさにブロックがチェーン状に連なったもの。  

 ブロックには、まとまったビットコインの取引履歴が記載されている。  

 もちろんそれは暗号化され、他人が読み取ることは出来ないようになっている。  

 暗号化には「ハッシュ関数」と呼ばれるものが使用され、暗号化されたデータから元の取引履歴を解読することは出来ない。  

 ここで大切になってくるのが、その「安全性」だ。  

 もちろん、安全性を高める為に様々な仕組みがあるが、ざっくりと説明するとこうなる。  

 取引データがネットワークに接続されてい大量のコンピュータに保管されている為、犯罪者がデータの改ざんを行う為には全てのコンピュータに保管されているデータを改ざんしなくてはいけないことになる。  

 この作業は、非常に困難というより不可能に近い。  

 では、「ブロック」とは、どのようなものか。  

 それは、マイニングに成功したマイナーと呼ば れt る人たちが生成するモノだ。  

 その権利を手に入れる為に、高性能なコンピュータを大量に使って膨大な計算を行い、ブロックの生成に必要な「ハッシュ値」を探す。  

 ハッシュ値を探す時に使われるのが 、 [Numbe r　 Use d　 Onc e 」を略して出来た単語、「ナンス」。  

 ナンスは 、 3 2 ビットの数字から成り立っている。  

 日本語に訳すと、「一度だけ使用される使い捨ての数字」を意味する。  

 「ナンス」が他のマイナーから承認されると、新しいブロックがつなげられ、取引データが記録される。  

 もうひとつ。  

 ブロックチェーンの「分散台帳」を見たり、管理・監視する人の端末のことを「ノード」と呼ぶ。  

 そしてノード全体でブロックチェーンを監視するシステムのことを、 「 P2P(Pee r　 T o　 Peer ) 」と呼ぶ。  

 このシステムは、非常に透明性があるのが特徴だ。  

 なぜなら、取引履歴は誰でも見ることができるからである。  

 最近は現金を持ち歩くリスクの高さに気付き、電子マネー派が増えてきていると聞く。  

 さらに安全性が高まったのが仮想通貨という認識で間違いない。  

 クレジットカードだと、使用した際にカード情報を抜き取られる心配があるが、ビットコインには実態がないのでそのリスクも存在しない。  

 仮想通貨をなんとなく怖いから使わないのではなく、徹底的に調べあげて使ってみることがこれからの時代に必要なスキルだと感じる。  

    

 ２、アルトコインの種類。  

 アルトコインとは、実現困難なことを可能にするために開発されたコイン。  

 170 0 種類以上のアルトコインが存在する。  

 こう聞くと、その数の多さに驚くかもしれない。  

 ここからは、主なアルトコインを紹介していこうと思う。  

 リップ ル (XRP ) ・・・時価総額３位の、有名なコイン。  

 企業の目標は、あらゆる場所に「価 値 ( = お 金) 」を届けること。  

 なぜなら、世界の約半数の人が、口座を持っていないというデータがあるから。  

 リップルが使用しているのは、 「 ILP ( インターレジャープロトコ ル) 」と呼ばれるもの。  

 「通 貨( 決済手 段) 」と通 貨( 決済手 段) を繋げるシステム」のことを指す。  

 これは、通貨の違いによる不便を解消しようとする狙いがある。  

 リップルでは、自動で最短・最安の方法を検出してくれる。  

 リップルは、分散台帳のグローバルなリーダーとして注目を集めている。  

 「低コスト、安全、スピーディーな次世代国際送金システム」として、世界中の金融機関が注目している。  

 201 2 年に、サンフランシスコで設立され、欧州の大手銀行であるサンデールやスタンダートチャータードなどが主要株主として名を連ねていることからも、信用の厚さが伺い知ることが出来る。  

 では、セキュリティの面ではどうだろうか。  

 リップルは他の仮想通貨とは異なり、ブロックチェーンと呼ばれる今まで利用してきた情報を都度組み込んでいく仕組みをとっていない。  

 その代わりに、管理者であるリップルが選定した 「 Validater( = 承認 者) 」と呼ばれる企業の一定数の承認があって初めて情報が書き込める仕組みを取っている。  

 それを、「コンセンス」と呼ぶ。  

 8 0 ％以上のバリデーダーの承認により、取引が完了する。  

    

 ライトコイ ン (LTC ) ・・・ビットコインに次いで２番目に世に出た仮想通貨として知られている。  

 ビットコインのソースコードを利用。  

 マイニングの仕組みがあるのも、ビットコインと同じ。  

 ビットコインの取引が格納されるブロックが生成されるのは 約 1 0 分間隔だが、ライトコインの場合は 約 2. 5 分間隔でブロックが生成される。  

 しかし、その分ビットコインよりも安全性が劣るということも覚えておきたい。  

    

 発行総量は、ビットコイン の4 倍にもおよび 、 840 0 万になる点も注目だ。  

 日本ではあまり知名度が高くないが、海外では比較的人気がある。  

 では、なぜ決済を早くすることが出来るのだろうか。  

 それは、 「 Segwit ( セグウィッ ト) 」というデータ圧縮技術を実装しているからである。  

 セグウィットは、簡単に言えば「ブロック内の特定のデータを別の場所に格納して、取引データを軽くする」技術のことをいう。  

 これを別の領域で保存しておけば、処理すべきデータ量が削減でき、処理時間の短縮にも貢献する。  

 つまり、「ライトコインはビットコインの欠点になりそうな部分を補うことを目指している」といえる。  

    

 イーサリア ム (ETH ) ・・・正式には、仮想通貨の名称ではなく、プロジェクトの名称。  

 ビットコインがブロックに'書き込めるのは、仮想通貨取引データに限られるが、イーサリアムはサブプログラムを書き込むことが出来る。  

 これを、スマートコントラクトと呼ぶ。  

 自動的に通貨発行などを行えるので、応用範囲が広いのも特徴。  

 イーサリアムにおける取引手数料報酬は、ビットコインにおける手数料と異なる。  

 イーサリアム は (GAS ) と呼ばれる、取引の複雑さに応じた手数料として燃料が必要になる。  

 ビットコインとの違いをもうひとつ上げていこう。  

 それは、「開発」の報酬に対する違いだろう。  

 どちらも「マイニング」と呼ばれる、取引が正しいかどうかを計算する作業を行う必要がある。  

 その作業を行う人を通称「マイナー」というが、彼らの支払いにもそれぞれの通貨が使われている。  

 ビットコインは将来的な支払いに対する報酬は決まっていて、４年で半減していくように設計されている。  

 その一方、イーサリアムは毎年一定の額が報酬として支払われるようになっているため、将来的にはイーサリアムの開発が進んでいくと考えられている。  

    

 ビットコインキャッシ ュ (BCH ) ・・・ビットコインからのハードフォークによって誕生した。  

 ハードフォークは、一定の時期以降に関して仮想通貨システムの仕様変更を行うことで生じる現象のことをいう。  

 主に、ビットコインの抱える「スケーラビリティ問 題( 容量の不 足) 」を解決することを目的としたアップデートにより誕生した通貨である。  

 ちなみに、「スケーラビリティ問題」とは、近年急速に取引データが増加した為、ブロックの生成が間に合わずに取引が承認されないなどの問題を指す。  

 ブロックサイズが大きいのも特徴だ。  

 サイズが小さいと処理速度が低下していくことも覚えておきたい。  

 それでは、今後、仮想通貨の中でどのような存在になっていくのだろうか。  

 実は、様々な要素から、これからの将来性が期待されている通貨が、このコインだ。  

 例えば、ビットコインキャッシュを基軸通貨に採用するイギリスの取引所が誕生したり、ビットコインキャッシュを推奨する金融大 手 SB I グループの仮想通貨取引所が設立されたりといった動きが出てきている。  

 また、ビットコインと比べて手数料が安く送金スピードが速いといったメリットがあることから、今後ますます需要が高まる可能性もある。  

    

 ネ ム (XEM ) ・・・正式名称は、ゼ ム (XEM ) という。  

 「ネム」は、プロジェクトの名称だということは、あまり知られていない。  

 ハーベスティングを採用しているが、これは「収穫」を意味している。  

 一定のネムを保有しているなどの条件を満たすと取引検証に参加できる権利が与えられる。  

    

 NE M とは、 「 Ne w　 Econom y　 Movemen t 」の頭文字をとったもので、「新しい経済活動」という経済に新しい革新をもたらすことを理念としたプロジェクトを指している。  

 ちなみに、開発に日本人が携わったこともあり、日本人投資家にも人気が高い仮想通貨といえる。  

 NE M のブロック生成にかかる時間は約１分間で、ビットコイン の 1 0 分間に比べると、 約 1 0 倍の短さでブロック生成が行われる。  

 ビットコインで言うマイニングを 、 NE M では「ハーベスティング」と言う。  

 ハーベスティングを行うには、アカウントに１ 万 XE M 持っていることが必要で、それ以外に条件はない。  

 ということは、ビットコインとは違い、誰でも参加できるということになる。  

 「ネム」と言えば 、 201 8年1月 2 6 日にコインチェック社から 約5億 230 0万 XE M が流出した事件が記憶に新しいだろう。  

 流出前に、外部からコインチェック社の複数の社員宛にメールが届き、業務パソコンがマルウェ ア( 悪意のあるソフトウェ ア) に感染したとみられている。  

 しかし、これは「ネム」自体のミスではないことを憶えておきたい。  

 ここで、もう少 し NE M の仕組みについてみていこう。  

 ネムでは 、 Proo f　 O f　 Importance(Pol ) という独自の仕組みを採用。  

 これは、ネム専用のウォレット 「 Nan o　 Walle t 」にある、ネムの「保有量」「保有期間」などから得られるスコアによって、ネットワーク運営にどれだけ貢献したかを評価する仕組みだ。  

 事前異生まれた恵みをそのままいただくといった意味合いから、ハーベスティングという名称がついたのだ。  

 ハーベスティングには、２種類が存在する。  

 ローカルハーべスティング・・・自分のパソコンの計算処理能力を使ってハーベスティングを行う方法。  

    

 デリゲー ド( 委 任) ハーベスティング・・・ハーベストをスーパーノードという第三者に委託する方法。  

 ちなみに「スーパーノード」とは、重要度が高く評価された参加 者( ノー ド) のことを言う。  

 スーパーノードになるには 、 3000000XE M 以上を保有するなど厳しい条件が設けられている。  

    

 リス ク (LSK ) ・・・イーサリアムと同じスマートコントラクトが実装されている。  

 マイクロソフトがパートナーシップを組んでいることでも注目されている。  

 リスクは、分散型アプリケーションプラットフォームを使用している。  

 そこで使われている仮想通貨が、「リスク」ということになる。  

 機能的な面で柔軟性が高く、開発の自由度が大きいということが、支持を集める要因となっている。  

 リスクでは、スマートコントラクトを記述する言語として 、 JavaScrip t を採用。  

 その言語はすでにウェブの世界ではメジャーな言語であり、扱えるエンジニアは世界中にとっては開発しやすい環境といえるだろう。  

 リスクでは、メインとなるブロックチェーンの他に、サイドチェーンと呼ばれる「枝を伸ばすことが出来る」ようになると言われている。  

 そうなると、アプリケーションの処理能力やトラブルからの復帰に役立つ。  

 さらに、新たな機能を実装したり、改善を施したりする際に、メインチェーンに手をつけずに行えるというメリットがある。  

    

 ３、仮想通貨の正体。  

    

 ここで、仮想通貨の正体を探っていこう。  

 ビットコインにおいては、いわゆる「ブーム」が起きている。  

 ユーザー数も増加の一途をたどっているし、時価総額も上がり続けている。  

 伸び率も目を見張るものがあるだろう。  

 しかし、実態を理解している人があまりに少ないことも確かだ。  

 まずは、ビットコインを入手しないことには始まらないのだが、そのやり方すらも知らない人は多い。  

 そして、これは実は簡単に手に入れることが出来る。  

 もちろん、不正を防ぐ為に本人確認書類の提出は必要だが、よほどのことがない限りは審査は通る。  

 審査が通ると、アプリを通じて自分の「ウォレット」を手にいれることが出来る。  

 ここで、そのウォレットに入るお金の正体を考えてみよう。  

 例えば、馴染みのある「一万円札」で考えてみる。  

 一万円札は自分の財布に入れることは出来るが、特定の一万円札を「自分のもの」と名乗ることは出来ない。  

 なぜなら、お札に所有者の名前が書いてあるわけではないからだ。  

 だから、万が一スリなどの被害にあい、財布を奪われてしまったらその瞬間にそのお金は盗んだ人の物になってしまう。  

 それに比べて、ビットコインの場合はどうだろうか。  

 ビットコインに「名前がない」ところまでは一緒だ。  

 しかし、「電子署名」という技術が存在し、現在の所有者に無断で送金出来ないようになっている点で、大きく異なっている。  

 必要があれば、遡って全ての履歴を確認することが出来るので、マネーロンダリングのような不正は起こりにくい。  

 もちろん、ビットコインを使用して買い物やレストランでの食事も出来る。  

 日本ではまだ普及していないが、サンフランシスコなどの地域では、ビットコインを使える店舗が増えてきている。  

 では、ビットコイン決済とクレジット決済の違いは、一体どのようなところにあるだろうか。  

 １番の違いは、「手数料」だろう。  

 お店でクレジットカードを提示すると、レジ端末などでカードを読み取り、取引された記録がクレジットカード会社に送られることになる。  

 締め日になると、カード利用者に代わりお店に代金を支払い、後日カード利用者から代金を回収する仕組みだが、ビットコインの場合はどうだろうか。  

 ちなみにクレジットカード会社では、２ ～ 1 0 ％程度の手数料を受けとることになる。  

 しかし、ビットコインによる支払いでは、お店側に直接ビットコインを送ることが出来る。  

 決済代行会社に頼んでも 、1 ％程度しかかからない。  

 これは、かなり割安と言えるだろう。  

 しかも、決済システムの導入コストは、限りなくゼロに近い。  

 取引所からビットコインが流出してしまうリスクは、ゼロではない。  

 しかし、財布の中身のお札みたいに抜き取られる心配がない。  

 これでも、まだビットコインは「危ないモノ」と感じるだろうか。  

 ４．ビットコインにおける支払い  

 ビットコインの「支払い方法」について、もう少し深く掘っていこう。  

 まずは、仮想通貨を取り扱っている業者から、ビットコインを購入する。  

 その後、ビットコインをモバイルウォレットに移すか、アプリをスマートフォンにダウンロードする。  

 お店の端末 の Q R コードをアプリに読み込ませ、支払いをする。  

 一見すると難しいように感じるかもしれないが、実際にやってみると思いの外簡単なことに気づくはずだ。  

 もうひとつ、関心があるのが「メリット」と「デメリット」だろう。  

 まずは、「メリット」から。  

 第一に、「現金を持ち歩く必要がない」ことが挙げられる。  

 これはクレジットカードにも同じことが言えるのだが、スリなどの被害にあう可能性が無くなる。  

 そして、海外で買い物をする際にも、手数料を安く抑えることが出来る。  

 もちろん、為替レートを気にすることなく決済することが出来るので、現金よりも優れていると言えるだろう。  

    

 ビットコイン決済を導入すると、店舗側にもメリットが生じることになる。  

 クレジットカードに比べ、手数料が安いので、その分の利益が増えるからだ。  

 それに加え、ビットコインの価格が上がれば、利益がその分上乗せされる。  

    

 次に、「デメリット」はどのようなものがあるだろうか。  

 それは、「価格の変動」がまず第一に挙げられる。  

 1BT Cが 10 0 万円の時もあれば 、 6 0 万円まで下がってしまうこともあり得る。  

 これでは店舗側は損を被る形になってしまう。  

 この、「変動するリスク」こそが導入が遅れている最も大きな原因と言えるだろうし、この幅が小さくなればなるほど、日本でもビットコインによる決済が普及していくはずだ。  

 もうひとつ挙げるならば、それは「決済スピードの遅さ」だろう。  

 ビットコインの仕組みでは、支払いが行われると、世界中に存在するパソコンによって検証作業が行われ、最 低 1 0 分程度の時間を要してしまう。  

 しかし、これもいずれ短縮されていくはずだ。  

 なぜなら、誰もが想像出来なかったスピードでテクノロジーが進歩しているからだ。  

 日本だけがこの流れに乗り遅れている感が否めないが、ここからの挽回に期待したいものだ。
