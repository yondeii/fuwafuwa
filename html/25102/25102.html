( 1 ) セ フ レ 女 性 の 候 補 は 友 達 か ら 見 つ け る よ う に し よ う

ま ず 初 め に 、 セ フ レ を 見 つ け る に は 相 手 が 必 要 で す 。 「 S N S ？ ？ 」 「 ナ ン パ ？ ？ 」 パ ッ と 頭 に 思 い つ く の な ん て そ ん な に あ り ま せ ん 。 さ ら に は セ フ レ に す る 人 で し た ら あ る 程 度 知 っ て い る 人 で な い と 心 配 と い う 点 も あ り ま す 。 私 が オ ス ス メ す る セ フ レ 女 性 は 「 友 達 o r 面 識 が あ る 人 」 か ら 選 ぶ こ と が お す す め で す 。 で は 、 な ぜ 身

近 な 距 離 に い る 友 達 や 面 識 が あ る 人 を セ フ レ 候 補 と す る の が 良 い の か ？ ？ も う 少 し 深 堀 り し て い き ま し ょ う 。

・ セ フ レ は 長 期 的 な 付 き 合 い が で き る 人 を 前 提 で 考 え る

今 回 、 セ フ レ を 作 り た い と い う こ と で 「 今 す ぐ に で も ヤ レ る 女 性 を 見 つ け た い 」 と い う 方 は ナ ン パ や 合 コ ン を お す す め し ま す 。 ま ず セ フ レ を 作 る 上 で 大 前 提 で 考 え て ほ し い の が 、 「 長 期 的 な 付 き 合 い を 視 野 に 入 れ て 考 え る 」 こ と で す 。 期 間 は 人 そ れ ぞ れ で す が 、 目 安 と し て 3 ヶ 月 〜 1 年 く ら い と 考 え て お き ま し ょ う 。 セ フ レ は ヤ ッ て 終 わ り で は な く 、 継 続 的 に お 互 い に セ ッ ク ス を 楽 し む パ ー ト ナ ー だ と 考 え て く だ さ い 。 た と え セ ッ ク ス だ け の 関 係 だ と し て も 長 期 的 な 付 き 合 い が で き る 人 を セ フ レ に す る の が ベ ス ト で す 。 互 い の 価 値 観 や 恋 愛 に 発 展 し て い か な く て も セ ッ ク ス を 楽 し め る 関 係 で い た い と 思 う の な ら 長 期 的 な 付 き 合 い と コ ミ ュ ニ ケ ー シ ョ ン は 大 切 で す 。

・ セ フ レ は あ く ま で セ フ レ 。 割 り 切 る 関 係 が で き る 人 を 選 ぼ う

恋 人 で は な い け ど セ ッ ク ス を す る 友 達 セ フ レ 。 中 に は 恋 愛 感 情 を 持 っ て し ま う 方 も い ま す が 、 必 ず セ フ レ と は 恋 愛 感 情 を 抱 か な い よ う に し て く だ さ い 。 セ フ レ を 作 る 際 に お 互 い が フ リ ー な 存 在 で あ れ ば 特 に 問 題 が あ っ た り 、 傷 を 作 る よ う な こ と は 少 な い で す が 、 例 え ば ど ち ら か に 奥 さ ん o r 恋 人 が い る 場 合 。 も し 、 あ な た の 彼 女 に セ フ レ が い た ら ど う し ま す か ？ ？ た だ で は す ま な い ぞ と 、 怒 り の 感 情 が 湧 い て き ま す よ ね 。 つ ま り 、 そ う い う こ と で す 。 セ フ レ を 作 る 際 に 相 手 の パ ー ト ナ ー が い る か の 確 認 も 必 須 で す 。 思 い も よ ら な い ト ラ ブ ル に な る 可 能 性 も あ り ま す 。 そ し て 、 そ の ど ち ら か が 恋 愛 感 情 を 抱 け ば も っ と や や こ し く な り ま す 。 セ フ レ に す る 女 性 は あ く ま で セ フ レ で す 。 恋 愛 感 情 を 抱 か な い よ う に 気 を つ け ま し ょ う 。

・ お 互 い に 信 頼 で き る 人 間 で あ る か も 大 切

そ し て 、 セ フ レ に す る 女 性 で 最 も 大 事 な こ と が 「 信 頼 で き る 関 係 」 な 人 か で す 。 そ れ は 、 い か な る 状 況 で あ っ て も 互 い に 信 頼 で き る か が 鍵 を 握 っ て い ま す 。 近 い 存 在 で 言 う な ら ば 、 セ フ レ は 親 友 と の 友 情 関 係 の よ う な 存 在 を 目 指 し て く だ さ い 。 そ し て 、 相 手 の こ と を し っ か り と 気 遣 っ て く だ さ い 。 セ フ レ に 対 す る 男 女 の イ メ ー

ジ は 大 き く 違 い ま す の で 注 意 し ま し ょ う 。 例 え ば 、 男 性 側 か ら す れ ば セ フ レ が で き た と な れ ば 、 友 達 に 自 慢 し た く な る よ う な 話 で す 。 で す が 、 女 性 側 か ら す る と 「 セ フ レ が い る 」 と い う 事 実 は あ ま り 公 に 公 開 さ れ た く な い 事 実 な の で す 。 女 性 の 場 合 は 「 セ フ レ ＝ ふ し だ ら な 女 性 」 だ と い う イ メ ー ジ が ど う し て も つ い て し ま い ま す の で 、 男 性 側 も セ フ レ が い る と い う こ と は 公 に せ ず 相 手 の プ ラ イ バ シ ー を 尊 重 し 、 互 い に 秘 密 の 関 係 で い る と い う こ と を 肝 に 命 じ て く だ さ い 。

次 に 重 要 な の が 「 ど ん な 性 格 ・ 中 身 の 女 性 を セ フ レ に す る か 」 で す 。 上 記 で 「 友 達 や 面 識 が あ る 人 か ら 選 ぼ う 」 の さ ら に 次 の ス テ ッ プ の 選 び 方 で あ る と 考 え て く だ さ い 。 セ フ レ に 求 め る 要 素 を 上 で 簡 単 に 説 明 し ま し た が 、 何 よ り も 一 番 大 事 な の は 「 体 の 相 性 が 合 う か 」 が 一 番 重 要 で す 。 そ し て 、 何 よ り 純 粋 に セ ッ ク ス を 楽 し め る 人 で あ る か も 大 事 で す 。 男 性 の 中 で 「 セ ッ ク ス が あ ま り 好 き で は な い 」 と い う 人 は 少 な い か も し れ ま せ ん が 、 女 性 は 意 外 に も 「 セ ッ ク ス が 好 き じ ゃ な い 」 と い う 女 性 が 男 性 に 比 べ る と 多 数 存 在 し ま す 。 で す の で 、 ま ず は 「 セ ッ ク ス が 好

き 」 と い う 女 性 が 好 ま し い で す 。 そ し て 、 次 に 重 要 な の が 「 セ ッ ク ス を 楽 し め る 人 」 で す 。 「 セ ッ ク ス が 好 き 」 な の と 少 し 似 て き ま す が 、 セ ッ ク ス に 対 し て あ ま り 深 く 考 え ず に 「 ス ポ ー ツ を す る 」 く ら い の 感 覚 を 持 っ て い る 女 性 で す と 望 ま し い で す 。 セ ッ ク ス を し た 後 に 「 今 日 も 楽 し か っ た ね ！ 」 と 言 え る よ う な 関 係 が 理 想 で す ね 。

( 3 ) セ フ レ に し て は い け な い 女 性 の 特 徴

次 に 「 セ フ レ に し て は い け な い 女 性 の 特 徴 」 を 紹 介 し ま す 。 前 提 と し て 1 回 ヤ ッ た 後 に 「 あ の 子 と ま た ヤ り た い な 〜 。 セ フ レ に で き な い か な ？ 」 と 考 え る こ と が 男 性 な ら あ る と 思 い ま す 。 で す が 、 こ こ が 「 セ フ レ の 女 性 を 選 ぶ o r 見 つ け る 」 上 で 最 も 重 要 で す 。 そ れ で は 最 初 に セ フ レ に し て は い け な い 女 性 の 特 徴 を 紹 介 し て い き ま し ょ う 。

・ 性 格 が メ ン ヘ ラ な 女 性 は ヤ り や す い が 気 を つ け ろ

ワ ン ナ イ ト ラ ブ や ナ ン パ で ヤ レ る 女 性 に 多 い の が 「 メ ン ヘ ラ な 女 性 」 で す 。 メ ン ヘ ラ 女 性 は 、 寂 し い ・ 人 肌 恋 し い と 男 性 と の セ ッ ク ス で 心 を 癒 す パ タ ー ン の 人 が 多 い で す 。 こ れ が 、 一 度 き り の 関 係 で し た ら い い の で す が 、 セ フ レ と し て 長 期 的 に 付 き 合 う 場 合 「 メ ン ヘ ラ 」 な 女 性 は 完 全 に や め て お い た 方 が い い で す 。 私 も 過 去 に メ ン ヘ ラ な 女 性 と セ フ レ の 関 係 に な っ た こ と が あ る の で す が 、 何 か こ と あ る ご と に L I N E で 「 疲 れ た 」 「 も う 生 き て て も つ ま ら な い 」 「 セ ッ ク ス し か 生 き が い が な い 」 「 今 す ぐ 会 い た い 」 な ど 連 絡 が き て 何 故 か こ ち ら 側 ま で メ ン タ ル が や ら れ て し ま っ た 過 去 が あ り ま す 。 私 が 遭 遇 し た メ ン ヘ ラ セ フ レ 女 性 は 表 で は 明 る い の で す が 、 1 人 に な る と 急 に メ ン ヘ ラ に な っ て し ま う 女 性 で し た 。 メ ン ヘ ラ な 女 性 は 男 性 に 話 し 相 手 に な っ て ほ し い 。 そ の た め に 話 し 相 手 の 延 長 上 と し て 体 の 関 係 に な る こ と が 多 い の で す 。 根 本 的 に 彼 女 た ち は 寂 し が り や で 人 の ぬ く も り が 欲 し い 人 種 で す 。 で す が 、 メ ン ヘ ラ の 対 応 を し て い る と 、 何 か あ っ た 際 に は 真 っ 先 に 疑 わ れ ま す し 、 な に よ り 事 あ る ご と に メ ン ヘ ラ 発 言 を し て 男 性 側 の 心 を 蝕 ん で き ま す 。 メ ン ヘ ラ 女 性 の セ フ レ は 作 ら な い よ う に し て く だ さ い 。

・ 彼 氏 持 ち o r 既 婚 者 は 基 本 N G

( 4 ) セ フ レ に し や す い 女 性 の 特 徴

続 い て 紹 介 す る の は 「 セ フ レ に し や す い 女 性 の 特 徴 」 で す 。 セ フ レ に し や す い 女 性 と い う の は 幾 つ か 共 通 事 項 が あ り ま す 。 そ の 共 通 事 項 に あ て は ま る 女 性 を 探 し 出 し 、 セ ッ ク ス を し て セ フ レ を 打 診 す れ ば 言 い 訳 で す の で 、 し っ か り そ の 特 徴 を こ こ で お さ え て お き ま し ょ う 。
