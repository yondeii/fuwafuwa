 2 . 質問を聞くときにはポイントを抑えよう  

 ・写真に写っていない物や、人物がしていない動作が描写されている場合は、すぐに消去しましょう。  

 ・〇か×か判断できない選択肢があった場合でも落ち着いてほかの選択肢を吟味し、他に〇だと思うものがあればそれを選びましょう。  

 ・他がすべて×なら、判断できなかったものを選びましょう。

 ＊頻出設問例＊  

 写真の種類と描写のタイプ  

 [ 人物の写 真 ( 1人 / 2人/ 大 勢 )]  

 　人物の写真の場合は、人物の動作に関する描写が基本となります。また 、2 人の写真の場合には、お互いの位置関係や場所の描写、大勢の写真の場合には、全体の状況や場所などが描写されます。  

 [ 物＋風景の写 真 ]  

 　物＋風景の写真の場合には、写っているものの位置関係や状態、場所などが描写されます。

 使われる文法  

 [ 現在進行 形 ]  

 A man is running on the ground.   

 ( 男性は広場を走っています 。 )  

 　基本的に、人物の動作を表す際に使われます。動詞に注意して聞き取ることで、確実に正解を選べるようにしましょう。  

    

  目次に戻る↑  

    

    

 


 

 [ 現在 形 ]  

 The cup is filled with water.  

 ( そのコップは水で満たされている 。 )  

 Cars are parked in a row.  

 ( 車は一列に停められている 。 )  

 　物の状態は、現在形の受動 態( ～され る) という形で表されます。  

    

  目次に戻る↑  

    

 


 

 [ 現在完了 形 ]  

 Wet clothes have been dried in the veranda.  

 ( 濡れた服がベランダに干されている 。 )  

    

 この現在完了形の形では、「動作が過去に行われた状態のままであること」を意味します。

 [ 現在進行形の受動 態 ]

 ( その家は今、ペンキで塗られています 。 )  

 このように物を主語にして、動作を受けていることを示す描写も多く出題されます 。 The ma nや Peopl e といった人物を示す単語が出ませんが、基本的には「人物が動作を行っている最中」ということに注意しておいてください。

 1 . 質問を聞く  

 　基本的に質問の前半部分の動詞までを聞くことで、質問内容を理解することが出来ます。全部聞き取ることが難しい方は、前半部分を特に集中して聞きましょう。  

 　また、質問文で使われている単語を選択肢にも混ぜてくる場合があるので、注意しよう。  

 [ 質問 文]　 Did you send the E-mail already?  

 ( 誤答 例 1)Send it by the end of the week.  

 ( 誤答 例 2)The E-mail is from my customer.  

 ( 誤答 例 3)No, he hasn't sent it yet.  

    

  目次に戻る↑  

    

 


 

 ＊頻出設問例＊  

  W H 疑問文  

 　 What / When / Where / Which / Who / Why / How / How often / How lon g など、疑問詞から始まる疑問文の場合、疑問詞を絶対に聞き逃さないようにしましょう。  

 　また 、 Wha tと Wh y は「動詞＋α」まで聞かないと答えを選べません。  

 [ 設問 例 ] ( 通 常 )  

 When will you leave for your business trip?  

 → In two days.  

 [ 設問 例 ] (難 )  

 Why don't you hand out these resume?  

 → OK. I'll do it today.  

    

  目次に戻る↑  

    

 


 

  Yes/N o 疑問文  

 Are you~ ?や Do you~ ? のよう に Yes/N o で答えられる疑問文も出題されます。ただし、その回答が、必ずし も Yes/N o が使われているわけではないので注意が必要です。  

 [ 設問 例 ] ( 通 常 )  

 Did you commute by train?  

 → Yes, recently.  

 [ 設問 例 ] (難 )  

 Isn't she a new employee?  

 → She has worked for 2 years in another department.  

    

  目次に戻る↑  

 


 

    

 依頼・提案・申し出  

 冒頭の表現で依頼か提案かというのが分かります。応答は「受け入れるか」「断るか」です。受け入れる場合は、決まり文句が使われることが多いです。また、断る場合は、必ず受け入れられない理由が伝えられます。  

 [ 設問 例 ] ( 通 常 )  

 Could you review this document?  

 → Sure.  

 [ 設問 例 ] (難 )  

 Could you review this document?  

 → How many pages are there?  

 冒頭だけの聞き取りで答えられるものも多くありますが、ひねった問題にも正解できるように、動詞まで聞き取って、意味を理解しましょう。  

    

  目次に戻る↑  

    

 


 

 選択疑問文  

 「午前か午後か」のよう に2 つのうちどちらかを選ばせる問題です。どんなに長い質問でも、「北か南か」「バスか車か」のように同列のもののどちらかを聞く問題のため、最後まで集中して聞き取ることが求められます。  

 [ 設問 例 ]  

 Do you often commute by car or by train?  

 → I often take a train.  

    

  目次に戻る↑  

    

 


 

 ステートメント  

 質問ではなく、報告の形を取る問題もよく出題されます。この問題の場合は、選択肢との意味の関連付けが必要なため、「動詞＋α」の理解が求められます。  

 [ 設問 例 ]  

 She will come here to order office supplies this afternoon.  

 → OK. So I'll prepare for it until that time.  

 [ 設問 例 ]  (難 )  

 This is the last dish in this full course meal, isn't it?  

 → It's the second from the last.  

 このように 、 ~, isn't it ?や ~, don't you ? のような付加疑問文もステートメントと同様と考えて答えてください。

 　 Part 3 の会話問題は、基本的に、「問題発生→解決」の順に会話が進むので、その流れを覚えておくことが、正答率アップのポイントです。

 1 . 設問を読む  

 　会話が始まる前に設問を読んでおくことで、求められている情報をあらかじめ確認しておくことが出来る。時間があれば、選択肢にも目を通しておこう。  

 (例 )  

 What are the speakers discussing?  

 ( 2 人は何について話していますか 。 )  

 Who most likely is the man?  

 ( 男性は誰だと思われますか 。 )  

 Where most likely are the speakers?  

 ( 2 人はどこにいると考えられますか 。 )  

    

  目次に戻る↑  

 


 

 2 . 会話の冒頭を聞く  

 　冒頭を聞くことで、「どこで」「誰と誰が」「何について」話しているかを理解することが出来ます。冒 頭 ( 1 文目もしく は2 文 目) が基本的 に1 問目に対応しています。  

    

  目次に戻る↑  

 


 

 3 . 会話の詳細を聞く  

 　2 問目 と3 問目は、会話の内容が完全に聞き取れなくても、話し手のピンポイントの会話が答えになることが多いため、求められている情報を頭に入れて、待ち伏せして、聞きましょう。  

 　また、基本的に、「問題発生→解決」の順に会話が進むことにも注意しましょう。  

    

  目次に戻る↑  

 


 

 ＊頻出設問例＊  

 概要を問うもの  

 　基本的 に1 問目に登場します。これらは定型問題のため、暗記しておくことで設問を一瞬で読むことが出来るようになります。  

 [ 設問 例 ]  

 ・ What are the speakers discussing?  

 ( 2 人は何について話していますか 。 )  

 ・ What is the topic of the conversation?  

 ( 会話のトピックは何ですか 。 )  

 ・ Where is the conversation most likely taking place?  

 ( 会話はどこで行われていると考えられますか 。 )  

 ・ Where most likely are the speakers?  

 ( 2 人はどこにいると考えられますか 。 )  

 ・ Who most likely are the speakers?  

 ( 2 人は誰だと考えられますか 。 )  

    

  目次に戻る↑  

 


 

 ・ Who most likely is the man?  

 ( 男性は誰だと思われますか 。 )  

 ・ What problem does the man mention?  

 ( 男性はどんな問題を述べていますか 。 )  

 ・ What is the man's problem?  

 ( 男性の問題は何ですか 。 )  

 ・ What is the problem?  

 ( 問題は何ですか 。 )  

    

  目次に戻る↑  

    

 


 

 詳細を問うもの  

 　定型問題とそれぞれの会話特有のものがあります。  

 [ 設問 例 ] ( 定型問 題 )  

 ・ What does the man ask the woman to do?  

 ( 男性は女性に何をするように頼んでいますか 。 )  

 ・ What does the woman suggest the man do?  

 ( 女性は男性に何をするように勧めていますか 。 )  

 ・ What does the woman suggest?  

 ( 女性は何を進めていますか 。 )  

 ・ What does the man ask for?  

 ( 男性は何を求めていますか 。 )  

 ・ What does the woman ask about?  

 ( 女性は何についてたずねていますか 。 )  

    

  目次に戻る↑  

    

 


 

 ・ What is the man worried about?  

 ( 男性は何を心配していますか 。 )  

 ・ What is the woman concerned about?  

 ( 女性は何を心配していますか 。 )  

 ・ What will the man probably do next?  

 ( 男性は次に何をすると考えられますか 。 )  

 ・ What does the man say he will do?  

 ( 男性は何をすると言っていますか 。 )  

 ・ What does the woman say she has done?  

 ( 女性は何をしたと言っていますか 。 )  

    

  目次に戻る↑  

    

 


 

 [ 設問 例 ] ( 特有の問 題 )  

 ・ What time will the speakers probably meet?  

 ( 2 人は何時に会うと考えられますか 。 )  

 ・ Why did the man go to Seattle?  

 ( 男性はなぜシアトルに行ったのですか 。 )  

 ・ How did the man receive the product?  

 ( 男性はどのようにして商品を受け取りましたか 。 )

 ( 男性は何をしようとしていますか 。 )
