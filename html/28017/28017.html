　 【 風の音】

 

　 美音 ( みおん ) 　著

 

 

○　はじめに

 

　こんにちは、著者の美音です。今作『【風の音】』では、今年の夏から秋にかけて感じたことを詩にしています。読んでいただけたら、幸いです。

 

 

　風の音

 

風の音がきこえる

サラサラ　サラサラ

サラサラ　サラサラ

私の心にささやくように

私をいだき

私を抱擁す

風の音がきこえる

サラサラ　サラサラ　

幸せの 刻 ( とき )

 

　淡い秋の空

 

プラチナ色の雲

ホワイトゴールドの雲

淡い淡い秋の青の空

その下に電線の波が通っている

それぞれの屋根の下に

それぞれの生活

目に見えぬ何かが　目に見えぬ生活が

動いている

時を一刻一刻きざんでいる

時の流れは、さりげなく移る

今あるこの一瞬は......

不思議だなぁ

あたりまえの事が

不思議だなぁ

 

　長月のあじさい

 

秋の冷たい雨に打たれ

花はもうすでに落ちて

葉もかなり枯れている

それでも雨を受け

どんと、そこに存在している

植物の強さ

受け身ではあるが力強く

そこに在る

凛としたそのすがた

 

　秋のコスモス

 

秋の風に吹かれて

コスモスがゆれる

風の道にゆれている

夏の空気と違い

乾いた秋の、ひんやりとした風の中で

コスモスの花のあざやかさが

ことさらに引き立つ

秋の風、秋の空気

風の造る、コスモスの花の、波の道

 

　おぼろげな三日月

 

神無月の、おぼろげな三日月

まわりは夜のとばり

雲にかすれる

おぼろげな三日月

雨上がりの湿っぽい空気が流れて

歩く人の影がゆれる

かすかにかすかに、雨のにおい

おぼろげな三日月

かすかに映る

 

　心の宇宙

 

刻 ( とき ) にどよめき、刻に安らぎ

刻にせかされ　刻に心に

深く突く思い　いてもたってもいられない

右に左に前へ後ろへ、ゆり動かされ

心が混だくする

心は宇宙の様だと思う

形はないけれども　いつも何かを考え

何かを思い、悩んだり、喜んだり

悲しんだりする

人の心の思いの不思議さと付き合って

いくのも、いいものかもしれない

人の心の広さと深さは

宇宙の様なものに感じる

 

　時空の 刻 ( とき )

 

夢にみる

夢にみる　昔日の日々

やさしく、悲しく、切なく

心の足跡を残す

時の流れの事実と

今の時の、時空の刻

妙に切なく複雑で

身の置き場のない様な時もある

それでも、外では、秋の虫が鳴いていて

秋の夜はふけてゆく

さりげなく、深く、夜はふけてゆく

 

　雨の音

 

雨の音がきこえる

ポツポツ　ザアザア

リズムを刻む　雨の音

人の心と、木々にひびく　

雨の音

自然の中の摂理の音と、時間

何かを潤す天からの雨

悩める人の心に、安らぎを

苦しむ人に、平安を

悲しむ人に、いたわりを

悦ぶ人に、ほほえみを

乾いた心の中に、潤いを

雨よ降れ　やさしく降れ

 

 

　道

 

山、山、山をこえ

谷、谷、谷をこえ

ぬかるんだ道に足をとられ

いばらの木が、前に立つ

しんどくて、しんどくて、へとへとになる

それでも道は続いていて

気の滅入る時がある

 

人それぞれの道があるんだな

人には分からない苦労があるんだな

たまには心の荷物を置いて

周りの景色や草木を見よう

周りの人に話をしよう

安らぐ気持ちを与えてくれる

 

一休み　一休み

少しづつ　少しづつ

一歩づつ歩めばいいのだな

心身の傷を少しづつ癒しながら

一歩づつ　一歩づつ

 

 

　秋の川

 

秋の川は少し水の量が減り

川べりの道は枯れ草が目立つ

秋を迎え　冬支度

空気は、少し乾き

空はグレーの様な　淡きブルーの様な

 

神無月の午後の 一時 ( ひととき )

散歩する人も夏にくらべて

軽やかに感じる

白さぎや鳥たちが川によりそい

秋を満喫している

空気感も

秋、独特の趣き

コスモスの花が色を添える

 

 

　秋の風音

 

トントントン

秋の音が、桜の葉の上に

コスモスの花の上にきこえる

実りの秋の音

 

命をささえる実りの秋の

豊かさとありがたさ

心の実りとは何か

むずかしく考えないで

今まで自分らしく生きてきたこと自体が

実りそのもの

たとえ自分らしくないこともあったかもしれない

迷い、動揺して、葛藤して

自分自身と向き合っている

それでも、必ず光は見える

足元を見つめて一歩づつ

空には秋の風の音が

きこえる

 

　人とは

 

人とは何だろう

日々のいつもの営み

移ろいながら、時間が過ぎていく

何げなく時が過ぎ

ちょっと良い事

ちょっといやな事

一日の時間の中にある

人とは何だろう

いつも考えているわけではないけれど

ふっと自分の中に問う

人とは何か

悩む自分、苦しむ自分、

喜ぶ自分、楽しむ自分、

他人の事を思う自分、

自分らしさとは何かと

自分に問いかける

人とは何か

永遠のテーマなんだろうなぁ

 

　散歩道

 

さまざまな人が散歩する桜並木

ウォーキングする人

ランニングする人

犬と散歩する人

人それぞれの思いで散歩する

それぞれの思いを受け止める、散歩道

桜並木は春夏秋冬

それぞれのたたずまいで

散歩する人を受け止める

深く深く、深呼吸する

ただ、それだけで良い

人の思いを受け止めてくれる、散歩道

 

　空

 

夢、夢、夢　空、空、空

なぜに哀しい

青い空、白い雲

何も言わない空

夢、夢、愛、愛

哀しく、美しい

深く青い心の空

 

　心の泉

 

甘くささやく心の声は

たどりついた心の泉

密のような甘い匂いに

心が満たされる

霧の中に浮かぶ、球体

まあるく心を包む

手に触れるとふわりと

飛んで、心の中を駆けまわる

甘くささやく心の声は

幸せの泉へたどりつく

 

　雨上がりの秋の午後

 

雨上がりの秋の午後

湿った空気と湿った大地が

秋の匂いを醸し出す

アース（地球）色に染まった大地と

冬支度をした木々たちが

秋の憩いの時を送っている

深く静かに、秋の雨の匂いを放ち

木々は色づいて行く

さりげなく、秋の 一時 ( ひととき ) は過ぎて行く

秋の午後の、憩いの 刻 ( とき )

 

 

　年季の入ったトラック

 

どれ位の荷物を積んで

今までどれ位、走ってきたのだろう

人の生活を支えるために

どれだけの物を運び続けたのだろう

人も同じ

年季の入った人に味が出る

その人、独特の味わいのある

人柄になる

トラックの 錆 ( さび ) のほころびが

それを象徴している

人も同じ

そのほころびもその人の象徴

愛着もひとしおになる

今日も、時は一刻一刻

流れて行く

味わいも、さらに深くなる

 

　さくら

 

さくら、さくら　なぜに咲く

人に幸せ与えるため

そのやわらかな色に

人は安らぐ

さくら、さくら　はらはらと散る

夏の桜の葉　たくましく

木もれ日やわらかい

いつの間にか葉は紅葉し

少しずつ葉は舞い落ちる

たしかに秋のおとずれは近づき

やがて葉はすべて落ちる

葉のない桜の木の枝も

グレーピンクの色合いの

存在感があって希望が持てる

寒い冬をのりこえ

寒い風に耐えて

春に花の咲く日を待っている

 

　桜の枝葉

 

桜の枝葉が垂れ下がる

垂れ下がる

冬、春、夏、秋の想いをのせて

垂れ下がる

愛、哀しみ、喜び、慈しみ

それぞれの想いが

想いの色を紅葉にして

垂れ下がる　垂れ下がる

それぞれの想いを払うように

一葉づつ、ひらりひらりと

舞い落ちる

そして道は落葉の色に変わり

冬を迎える　

寒さに耐える　葉の落ちた桜は

ベージュの中にほのかに宿る

淡い淡いさくら色

人の日常は春夏秋冬

移り変わり　心模様も

移り変わる

その時に見る桜の木は

その時の気持ちを反映する

四季は巡りくる、巡りくる

人生の航路の様に

 

　心の風

 

心に

さまざまな想いがつみかさなる

つみかさなって重くなる

そのような時は

人との会話の、風

自然がくれる心の、風

時間の流れが風となって

心が軽くなる

今日も良き日でありますように！
