●はじめに

 

こんにちは。

ソクラテスです。

私 は 201 3年6 月現在、アマゾン で 1 5 冊の電子書籍を出版しています。

 

 

本書では、電子書籍の出し方 、 Amazo nの KD P （キンドル・ダイレクト・パブリッシングの略） の登録の仕方など、具体的にご説明しています。本書を読んで出版してみよう！という方や、電子書籍って、こうやって出版するんだ、という将来的に出版を考えておられる方向けに書いています。

 

 

本書では、アマゾンの電子書籍に焦点を当ててご説明しています。

他社でも電子書籍はありますが、私自身が他社で出版していないため、

ご説明に値しないからです。

 

 

では早速、いきましょう！

 

 

●電子書籍ってなに？

 

電子書籍とは、今までの紙の本をアマゾンのキンドル や iPhon e 、タブレット 、 P C などのデジタル機器で読む事ができるようにアレンジしたものと、最初からそういうデジタル機器向けに出版するものとがあります。

 

 

そして 、電子書籍は既存の紙の本がデジタル機器で読む事ができたり、紙の本がなくて最初からデジタル機器向けに書かれたもの の2 種類があります。

 

 

マンガのコミック や過去の小説なんかも次々と電子書籍化が進んでいますね。

今後、ますます電子書籍を利用する方の人口が増えてくると言われています。

 

 

●電子書籍のメリット

 

電子書籍のメリットとしては、無料で出版できる事です。

しかも、出版した後の管理料なども無料です。

 

 

そして、一度出版すれば、アマゾ ン KD P がある限り、

「 永遠に」出版し続ける事ができます。しかも自分自身は何もしなくても、です。

 

 

さらに、 出版した電子書籍の内容を適宜修正する事もできます。

ただ、 最初に買っていただいたお客さまと、修正後に買っていただいたお客さまで内容が違う事になるので、なるべくなら出版当時のままが良いでしょう。

 

 

紙の本と決定的に違うところは、リンクやメールアドレスを載せる事ができる点です。紙の本なら、お客さまは手入力でスマホ や P C を操作しなければならないのに対し、電子書籍なら最初からデジタル機器を持って読んでいただいているので、リンクやメールはワンタップで完結しますね。

 

 

リンク先に自分のブログやフェイスブック、ツイッターなどを紹介する事もできます。

 

 

●販売価格

 

販売価格は自分で決めます。

完全に自由に設定して大丈夫です。

私 は 201 3年6 月現在 、 29 9 円で統一しています。

これは報酬のところでも登場しますが 、7 割の報酬をいただくためにはこれぐらいの価格設定にしてくださいというような事が書いてあったのでそうしているだけで 、 29 9 円にこだわっている訳ではありません。

 

 

価格設定について は1人1 人考え方があると思います。

紙の本のよう に 1,28 0 円と か 98 0 円という設定でも良いでしょうし、逆 に 10 0 円 や 9 8 円でも良いと思います。

 

 

その内容に見合った価格で設定してあげるのが良いと思います。

そういう意味では、私 の F X 系の電子書籍は明らかに安すぎますが...。

時期をみて値上げしようと思っています。

 

 

●報酬

 

こちらの報酬は、販売価格 の 7 0 ％ か 3 0 ％を選択します。

例えば販売価格 が 29 9 円なら、

7 0 ％ の 20 9 円が報酬ですが、ダウンロード手数料 が1 円だけかかりますの で 20 8 円がこちらの報酬になります。

 

 

●ダウンロード手数料

 

ダウンロード手数料は、アマゾンが 自社サーバーに自分の電子書籍を保管してくれていて、保管に関する手数料は無料なのですが、お客さまが購入してくれたときに初めて手数料をいただくというシステムになっています。なので、売れない限りは手数料が完全にゼロ円です （ 201 3年6 月現在）。

 

 

電子書籍の容量によってその手数料は決まってくる のですが、容量 が1 メガにつ き1 円です。私は今ま で1 メガを超えた事がありません。

私の 今まで最も多い文字数のメルマガの過去投稿分①で も 5 5 キロバイトなので 、 1,00 0 キロバイト の1 メガに到達しようと思えば、その 約 2 0 倍の文字数となり、かなりたくさんの文字数が必要であると思いますので、ダウンロード手数料はほ ぼ1 円固定という認識で大丈夫です。

 

 

もちろん、途中に画像などを貼り付ければその分、容量が大きくなりますが 、1 メガにつ き1 円なので、仮 に2 メガになって も2 円であり、しかも売れない限りはダウンロード手数料が発生しないので安心ですね。

 

 

●ジャンル

 

電子書籍では、どんなジャンルの本を書けば良いのでしょう？

とよく聞かれますが、回答はとしては、「完全に自由」です。

私 は F P であり 、 F X トレーダーなので、その関係の本なら書ける！と思っただけです。

 

 

これは、それぞれの生い立ちや経験などに基 づく方がより書きやすいと思います。

例えば 、登山が好きな方なら登山について書けば良いです。

その人にしかない体験が詰まっているのですから。

学校の先生なら、教育について書いても良いと思いますし、こんな児童がいてこんな生徒やこんな学生がいたな～と個人情報をさらさない範囲で書けば、それもその人にしか書けない内容です。

 

 

ラーメンが好きでよく食べ歩いている方なら、それも書く材料になりますし、劇団員の方なら今までの体験を書くだけでも貴重ですね。

 

 

全員に共通して書けるとすれば、半生です。この世に生まれてから今までの人生を綴った物語です。幼稚園時代から小学生、中学生など、その時にどんな事をしていたのか、どんな事が好きだったのか、家族の事、友達の事、ちょっとした事件の事、楽しかった事、先生の事、当時のテレビの事、時事で思った事など、何でも良いのです。

 

 

それらは、本人にしか書けません。誰にも真似できません。文章力が...なんて言われますが、後から読み返して意味が伝わる程度に書けば良いですし、書いている内にだんだんスラスラと書けるようになるものです。

 

 

何も小説家になれと言っている訳でもありませんから、あれこれ考えずに、まずは書いてみる事です。今はパソコンがありますから文字をペンで書くよりも数倍の速さパソコンで書く事ができます。

以上 がジャンルになります 。 KD P では、カテゴリという言い方をしています。

まぁどちらにしても ジャンルもカテゴリも同じですね。

 

 

そしてさらに、細分化したカテゴリ分けをしています。

例えば「ビジネス、経済」のカテゴリなら、

一般、会計、広告・販促、会計監査、銀行・銀行業、簿記、予算編成、商業倫理、ビジネスマナー、商法、経営数学、ビジネス文書、商業、消費者行動、企業金融、ファシリティマネジメント、外国為替、フランチャイズなど、

かなり細かく分けられています。

 

 

全てのカテゴリが、こんな感じでかなり細かいので、きっとどこかには当てはまるでしょう。

 

 

●キーワード

 

検索キーワードとして、最 大7 つまで登録できます。

例え ば F X の電子書籍なら、 「 F X 、外国為替、メルマガ、自動売買、勝つ、負ける 、 Fore x 」 の7 つとかですね。

 

 

お客さまが 電子書籍を探すときに、その検索に引っ掛かりやすいようにあらかじめ登録しておきます。

 

 

●表紙

 

本には表紙がありますね。電子書籍でも同様にありますが、ない場合は私の電子書籍のように、アマゾン社が後日、表紙がない著者のためのちょっとした 定型の画像を付けてくれます。

 

表紙がある場合 は .jpe g 画像にしてアップロードします。

 

 

●タイトル

 

タイトルは重要ですよね。

何が書いてある本なのかが一目で分かりますから。

まれに、タイトルと内容違うな～という本もありますが、その本は早急にタイ トルを変えるべきだと思います。それぐらい、タイトルは重要だと思います。

 

 

登山の本なら、「富士山に登って気付いた事」とか、「富士山はやっぱり最高」とかですね。

 

 

私の知り合いは、借金の発生から完済までの実話を本にしていました。実話だけに途中とても重かったですが、ハッピーエンドでホっとしたのを覚えています。

 

 

●サブタイトル

 

サブタイトルがあると、どういう方に向けて書いてあるかが分かるので、私は好んで使っています。

 

 

～富士山を登ってみたい方へ～とか、～富士山には何があるのか気になる方へ～で、特定の客層に絞り込みます。

 

 

ですが、これは好みだと思います。

サブタイトルを付けない作家さんもいらっしゃいますしね。

 

 

この電子書籍のサブタイトルは、～試しでも良いから出版してみたい方へ～

ですね。これは、このサブタイトルの通りで、あなたはこの電子書籍の内容が気になったから買ってくれたという事ですね。とても感謝しています。ありがとうございます。

 

 

●内容紹介文

 

電子書籍が書き終わってからになると思いますが、その内容について、お客さんが見たときに読んでいただく説明文を作ります。最大 で 4,00 0 文字も書けますが、内容のちょっとした宣伝文句で良いでしょう。

 

 

● KDP

 

KD P 、キンドル・ダイレクト・パブリッシングとは 、 Amazo n 社が電子書籍のために立ち上げた組織です。我々は、ここにユーザー登録をして、ここに電子書籍を提出します。

 

 

アマゾンに登録している方なら、同 じ I D ・パスでそのまま登録が完結します。

 

 

氏名、住所、電話番号、生年月日、印税受入口座情報など、画面の指示に従って入力します。

 

 

●電子書籍の書式

 

KD P に提出するファイルは 、 Wor d ファイル か epu b ファイルですが 、 epu b で書こうと思えば専用のアプリ を P C にダウンロードした上に、操作がかなり分かりにくかったので、私は専 ら Wor d で書いています。現在の総文字数も下に常に表示されますし、日本語がおかしかったら赤の波線で示唆してくれます。

 

 

電子書籍でも縦書きが主流なようですが、私は横書きしか出していません。というのは、パソコンやスマホ、タブレットで読むホームページやブログ、フェイスブック、ツイッターなど、その全てが横書きで、みなさんは デジタル機器と言えば横書きに慣れているはずで、それをわざわざ縦書きにするのはどうかと思っています。

 

 

いずれ、横書きが主流になると思っています。特に私の場合 、 F X の本なんかでは数字が非常にたくさん登場しますので、これが縦書きだと違和感があるのです 。 F X に限らずとも、全てのジャンルで数字は出てきますね。西暦何年とかも。それで縦書きは、ね～。

 

 

もちろん、縦書きでも構いませんのでどちらでもご自由に書いてくださいね。

 

 

●文字数

 

電子書籍の場合 、 5,00 0～ 10,00 0 文字が適当、とされているそうです。これは、実際に電子書籍を買った方々へのアンケートで、それぐらいの文字数の電子書籍を好む傾向が強いのだそうです。

 

 

紙の本なら 、1 万文字なんかでは終わりませんよね 。 1 0 万文字超えもざらにあります。それをそんな著しく少ない文字数で良いのかな？と思うかも知れませんが、そういう統計が出ているので、今はそれに従うのが無難でしょうね。

 

 

私の考えでは 、 iPhon e などで読む場合は、普通の紙の本のような長い内容だと疲れてしまうのかなと思っています。それを 、1 万文字以内だとサラっと読めるので、好まれているのだと思います。

 

 

●ペンネーム

 

私はソクラテスとし て F X をしているので、これをペンネームにしていますが、別に本名でも何でも良いでしょう。私は、巻末に本名をさらしています。

 

 

● KD P に登録

 

ここで、アカウントを登録します。もちろん無料です。念のため、パソコンからする方が良いと思います。

https://kdp.amazon.co.jp/self-publishing/signin

アカウントを取得したら、

https://kdp.amazon.co.jp/self-publishing/help?topicId=A37Z49E2DDQPP3

に出版までの流れが全て書いています。

 

 

●電子書籍を出版

 

そしていよいよ、書い た Wor d ファイル を KD P にアップロードです。

 

・ KD P にログインしたら、「新しいタイトルを追加」のボタンを押します。

 

・ 「 KD P セレクトにこの本を登録する」にチェックを入れます。

これは詳しくは分かりませんが、アマゾンが販促をしてくれるそうです。

ここは素直にチェックを入れておきましょう。

 

・タイトルを入力します。フリガナを入力します。ローマ字を入力します。

 

・内容紹介を書きます。

 

・「著者等を追加」ボタンを押して、著作者を入力します。

 

・ 1 8 禁の表現内容（文章、シーン、画像など）が含まれていますか？のところで「はい」「いいえ」のいずれかを選択します。

 

・出版する権利の確認で、「これはパブリックドメインの作品ではなく、私は必要な出版する権利を保有しています。」にチェックを入れます。

 

・「カテゴリーを追加」 で2 つまで選択します。

 

・検索キーワードを最 大7 つまで登録します。

 

・表紙 の jpe g 画像があればアップロードします。なければスルーします。

 

・「デジタル著作権管理を適用する」を選択します。

 

・電子書籍が横書きか縦書きかを選択します。

 

・「本をプレビュー」のボタンを押すと、実際の画面が見られます。スルーしても登録はできますが、できればここ で Wor d ファイルがどんな感じでお客さんの端末で表示されているかをチェックしておくと良いと思います。

 

・「保存して続行」ボタンを押して次のページへ。

 

・「全世界の権利―すべての地域―」を選択します。もちろん、特定の地域を選択しても構いません。

 

・ロイヤリティの選択で 、 3 5 ％ か 7 0 ％かを選択します。私 は 7 0 ％にチェックを入れています。そして、アメリカの 「 Amazon.co m 」に例え ば 2.9 9 と入力して、以下のチェックボックスにチェックを入れると、アメリカ の 2.9 9 ドルに合わせた価格設定を自動で計算してくれます。そして 「 Amazon.co.j p 」で、何円で販売するのかを入力して、□のチェックボックスはスルーします。チェックを入れると、アメリカの価格を元に日本円で計算されますので少しややこしいですよ。

 

・ Kindl e 本のレンタルで、「この本をレンタル可能にする」にチェックを入れます。というかここ、最初からチェックが入っていますね...。

 

・ KD P 利用規約に同意する場合は、チェックを入れます。

 

・「保存して出版」ボタンを押します。

 

・ポップアップが出て、「本棚に戻る」のボタンを押すと完了です。

 

ここで初めて 、 Amazo n 社がその電子書籍の審査に入ります。英語の場合 は 2 4 時間以内、その他の言語の場合 は 4 8 時間以内に出版が完了するそうですが、私の場合は 、5～ 1 0 時間ぐらいで出版が完了していまして、出版が完了すれば、登録しているメールアドレスに連絡が入ります。私は今ま で N G を出した事がないので 、 N G の場合がどんな様子なのかが説明できません。

 

 

●リアルタイム販売数

 

「レポート」ボタンを押して、「今月の販売数」を押すと、今月の現時点での販売数がタントルごとに表示されています。必要に応じ て CS V ファイルをダウンロードする事もできます。自分の電子書籍が売れているという事実は、とても嬉しいですよ！

 

 

●支払い

 

これについては正直言いまして、詳しい事は分かりません。ただ、 事実として私 は 201 2年 1 2 月に初めて出版して 、 201 3年5月 3 0 日に登録銀行口座に初めて入金がありました。

 

 

金額的に、どうや ら2 ヶ月分でした。という事は 、 201 3年2 月末までの売上 の 7 0 ％とダウンロード手数料を引いた金額 が5月 3 0 日にあったという事になります。まぁこれに関して は KD P に問い合わせるとかしてみると良いと思います。

 

 

私はアマゾンを信じているので、印税の支払いはいつ？などと問い合わせる気はありません。ただ 、5月 2 0 日に英語で支払い通知のメールが来ていまして、その通知に対して私は何かアクションを起こさないといけないのかの質問はしました。分からなかったもので...。

 

 

翌日、回答が来まして、通知だけですので特に何もなさらなくても良い、という事でした。放置していた ら5月 3 0 日に入金された、という流れです。

 

 

●最後に

 

以上が電子書籍の書き方から出版、入金までです。だいたいお分かりになったでしょうか？電子書籍の最大の魅力は、完全無料で出版し続けられる事です。

 

 

紙の本だと、最初 に 20 0～ 30 0 万円が必要で、印税 は3～7 ％だそうです。最も、紙の本ならではの名誉があるので、私もいずれは紙の本を出版しようと思っています。印税が目的ではなく、私はこの時代にいたという軌跡としてと名誉のためです。

 

 

Amazo nの KD P なら、最初にかかる費用 は0 円で、出版管理料など も0 円です。つまり 、 Wor d ファイルに書けさえすれば、幼稚園児でも老人の方でも国籍も言語も関係なく、インターネット環境さえあれば電子書籍が出版できます。しかも一度出版した電子書籍は 、 KD P がある限り、永遠 に0 円で出版し続ける事ができます

 

 

これはものすごい革命だと私は思っています。今までの出版の常識を根本から覆 したからです ！ KD P のおかげで私は電子書籍を出版する事ができましたし、こうやってあなたがデジタル機器で読んでくれています。これが紙の時代のままなら、あなたと私が繋がるのはもっと先になっていたでしょうし、なかったかも知れませんね。
