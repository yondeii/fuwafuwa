A 子 は こ れ ま で 僕 の 話 を 聞 く 側 だ っ た の が 、 急 に 自 分 が 聞 く 側 と な り 少 し お ど け た よ う な 表 情 を 見 せ て い た が そ の 心 の 中 は ど こ か 嬉 し そ う な 気 持 ち に な っ て い た 。

帰 り 道 い つ も の と こ ろ で 「 じ ゃ あ ね ！ 」 と 振 り 返 る と A 子 か ら 「 今 日 は 金 曜 日 だ か ら も う 少 し だ け 私 の 家 で 飲 ん で 行 か な い ？ 」 と A 子 か ら 打 診 さ れ た 。

大 学 時 代 か ら 合 わ せ る と 、 こ れ ま で 7 年 く ら い の 付 き 合 い で は あ っ た が さ す が に 家 に 行 く こ と は な か っ た 。 心 の 中 で は こ れ ま で の 余 裕 な 気 持 ち で い た が 、 ど こ か 何 か 起 き る こ と を 期 待 し て い る 自 分 も い た 。

「 1 時 間 だ け な ら い い よ ！ 」 と 僕 は A 子 に 告 げ て 二 人 で コ ン ビ ニ で お 酒 を 買 い 、 家 に 向 か う こ と に し た 。

家 に 入 る な り 、 A 子 は 僕 に 甘 え て き て 、 一 気 に こ れ ま で 7 年 間 心 の 中 で 抱 き 続 け て い た 恋 心 が 爆 発 し た 。

A 子 と の セ ッ ク ス が 終 わ っ た 後 に 、 ベ ッ ト で 彼 女 の 頭 を な で な が ら 「 〇 〇 と の セ ッ ク ス が 人 生 で 一 番 気 持 ち か っ た 」 と 言 わ れ た 。
