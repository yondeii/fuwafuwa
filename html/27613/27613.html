第一章 　　　

エルネスト・チェ・ゲバラと

キューバ革命 （ 195 2 年 〜 195 9 年）

 


 

①　モーターサイクル・ダイアリーズ

監督：ヴァルテル・サレス

出演：ガエル・ガルシア・ベルナル、

ロドリゴ・デ・ラ・セルナ

Diarios de Motocicleta (2004) de Walter Salles

    

 　モーターサイクル・ダイアリーズ は 2 3 歳だったエルネスト・チェ・ゲバラの初めてのラテンアメリカへの旅の物語である。ゲバラはアルゼンチンの中部にあるコルドバ州で育てられた。彼は幼い頃から喘息の発作に苦しんでいたため、両親は喘息の治療により良い環境を求めて、コルドバ州のアルタグラシア市に引っ越した。その後、高等学校を卒業してから、両親や兄弟とともにアルゼンチンの首都ブエノスアイレスに暮らした。  

 　ゲバラは裕福な社会環境に育てられた。先住民や貧困層から離れた生活だったが、その存在はラテンアメリカ社会と文化の歴史に大きな役割を果たしたことを知っていた。彼と親友の兄だったアルベルト・グラナドの旅は、冒険の旅とラテンアメリカの発見への旅であった。私たちも、ゲバラと同じように、本書を通じて、そし て 1 5 本の映画を通じて、ラテンアメリカの社会、文化や政治経済の歴史と現状を把握するために、ラテンアメリカの多くの人々に会う旅へ出発する。ラテンアメリカという空間だけではなく、時代の流れに関してもゲバラ の 2 3 歳の旅は本書の出発点でもある。  

 　ゲバラとグラナドの出発 は 195 2年1 月だった。グラナドはゲバラの親友だった中学校の同級生の兄で、ゲバラ の6 歳年上だったが 、 194 0 年代前半から２人は近い友人だった。グラナドは一時的にゲバラが参加していたラグビー・チームのコーチだった 。 194 3 年に、グラナドは反政府デモに参加して、他の学生とともに逮捕された。ゲバラとグラナドの弟は釈放されるまでほとんど毎日刑務所に足を運んだ。２人の政治に関する意識を深化させた出来事であった。  

 　出発の時ゲバラはブエノスアイレスで医学を勉強していた。卒業まで一学期しか残らなかったが、休学して旅に出た。彼はラグビーをしていた。アルゼンチンではサッカーが国民的なスポーツだが、ラグビーをする人は、特にその時代では、たいてい裕福な人だった。母親との愛情関係は深く、旅の道中彼女に書いた手紙が多かった。ゲバラとグラナドにとって、旅先の未知の世界は、大学、友人、家族、恋人より魅力的だった。ゲバラの心のなかでは、自分の目で現実を確かめることの楽しさと、現実を体験するべきという倫理的な要素が合流して、一つの強力的な動機になっていただろう。  

 　映画で は2 人がブエノスアイレスから出発するが、実際の旅ではグラナドの住まいだったコルドバ州から出発し、ブエノスアイレスのゲバラの家族に会いに行く。ブエノスアイレスから出発すると、最初の目的地はゲバラの恋人チチナが家族と夏休みを過ごしていたブエノスアイレス州の大西洋海岸にあるミラマルだった。チチナの家族はコルドバの上流階級の代表的な家族だった。ゲバラはチチナから離れることは愛情関係の側面でも、物質的な側面でも、その後のチェ・ゲバラという革命家の誕生において、象徴的な出来事だった。ゲバラがミラマルから出発した時、チチナとの愛情関係は続いていたが、その日からそれぞれの日々の体験による以前より離れていく。２人とも裕福で、名字は知られている家族で育ったが、ゲバラの両親はリベラルで物質的な豊かさを大切にしなかった。教育の面でも、息子や娘に自由を与え、自力で自分の思想を形成することを優先した。反対に、チチナの家族フェレイラ家では教育が保守的だった。ゲバラとチチナの共通点は読書を通じて社会を知ることと普遍的な正義の必要性を認めることだったが、ゲバラだけはラテンアメリカの人々に会いに出て、やがて正義を現実にするために政治的な活動を行った。  

 　ゲバラとグラナドはアルゼンチンの南部からアンデス山脈を越えてチリに入る。真夏なのに、寒い雪の中でグラナドの中古モーターサイクルは走り続ける。アルゼンチン、チリ、ペルーで出会う人々に対する２人の行動は異なった精神と倫理を見せてくれる。ゲバラは率直であり、自分を有利にするために嘘をつけられない。自分の欲望を満足させることか、他人に対する思いやりを優先させることか、という選択肢が現れる場面では、２人の行動が異なる。しかし、互いに不信感がなく、旅は友情への賛歌でもあった。  

 　旅ではチリ北部やペルー南部に進むにつれ先住民が多くなる。ゲバラたちはアナコンダ銅鉱業社というアメリカ多国籍企業を所有するチュキカマタ鉱山（当時世界最大銅鉱山）で自分の土地を失ったという夫婦に出会う。夫は日雇い労働者として鉱山で働こうとしていた。鉱山の仕事は ３K の仕事（きつい、危険、汚い）で、特に高い危険性で知られている。この夫婦やペルー南部の先住民との会話では土地所有の問題が現れている。ラテンアメリカの政治経済では土地所有構造が根本的な問題である。大土地所有と零細土地所有は共存する。多くの先住民は零細土地を持つか、あるいは土地を持っていない。映画では貧困層の農民が自分の土地から追い出されたという話が繰り返される。土地なし農民の増加は植民地時代に始まり 、 2 1 世紀にもアグロビジネスの拡大とともに続いている。  

 　ゲバラとグラナドはマチュピチュやクスコに行く。彼らは、インカ系の女性たちと話すと、彼女たちの経済状況、教育機会、スペイン語とケチュア語（インカの言語）をマスターする度合いと高原で仕事できるようにコカ葉をかむこと等に関する話題がでてくる。子供はゲバラとグラナドを案内する時、先住民からみたスペイン人（植民地時代の征服者）というテーマにも触れる。先住民の衣装、町の建築と会話の内容は独特な世界を描く。これらのシーンを通じてスペイン人がやってくる以前のラテンアメリカから生き残った姿が描かれる。ゲバラがマチュピチュで日記に書く文章が彼の２つの世界の間の存在を表している。先住民はゲバラの血から遠い。しかし、ゲバラは先住民の土地だったラテンアメリカに生まれた。そのことは土地所有の不公平な構造と重なり、２人のラテンアメリカの理解では正義の関する意識が深まる。  

 　「モーターサイクル・ダイアリーズ」はゲバラが旅中書いた日記とグラナドがこの旅について書いた本と彼とのインタービュに基づいている。映画の製作中、キューバに住んでいたグラナドは旅の幾つかのロケーションに行き、その場で監督などの映画作成担当者を案内した。ゲバラの日記は日本語で様々な版で刊行されている。ラテンアメリカを知るためにだけではなく、人間主義的（ヒュマニスト）な世界観について考えさせられる本である。現代社会において、他人との競争を優先される人間の生き方に対して、連帯を強調する見方である。  

 　映画に含まれる一部の人々との出会い（チリの市場の店員、ペルーの農家、クスコで案内する子供と先住民の女性たちなど）は偶然で、それらのシーンはインプロビゼーションによって仕上げられた。  

 　出発から予定されていたが、２人はペルー北部のアマゾン地域にあるハンセン病患者の収容所でインターンとして３週間を過ごす。収容所では川が患者と従業員（医者、看護婦、シスターなど）を分裂する。ゲバラはこの状況を社会階級間の分裂の表れとして感じる。その時、グラナドはベネズエラで就職先が見つかったという知らせが届く。安定的な生活は望ましいかどうかに関する会話が出てくる。その会話の内容はゲバラの革命に至る長い旅が始まったばかりということを感じさせる。  

 　ゲバラの誕生日パーティーと２人の収容所を去るさよならパーティーが重なった。ゲバラはスピーチを要求され、彼がそのスピーチではラテンアメリカの統合について語る。ボリバル（現在のベネズエラから）やサンマルティン（現在のアルゼンチンから）という南米独立運動の指導者が目指した、南米の地域統合がそれぞれの地域の有力者の利害関係によって現実にならなかった。ゲバラは南米の人間は一つだと強調する。その時、メスチゾ （ Mestizo,   先住民とスペイン人の混血）の存在について語る。自分がヨーロッパ系であり、旅中先住民の生活に接近したゲバラにとっては、メスチゾがラテンアメリカ人として強くなったアイデンティティを表す 。 2 1 世紀に入ってから南米諸国で誕生した中途左派政権は改めて南米の政治的な地域統合の方向に向かっている 。 200 7 年 の UNASU R の誕生はボリバル、サンマルティンとゲバラの望みを少なくとも萌芽的にかなえようとしている。しかし 、 201 0 年代後半にアルゼンチン、ブラジルなどに誕生した保守的な政権は統合運動 や UNASU R に反対して、アメリカとの関係を強めた。  

 　「モーターサイクル・ダイアリーズ」の監督はヴァルテル・サレス（ブラジル人）、ゲバラの役を演じる俳優はガエル・ガルシア・ベルナル（メキシコ人）、グラナドの役を演じる俳優はロドリゴ・デ・ラ・セルナ（アルゼンチン人）、脚本を書いたホセ・リベラ（プエルト・リコ出身）という組み合わせは、偶然でも、ラテンアメリカの統一について物語る。デ・ラ・セルナはゲバラの遠い親戚である。監督と俳優たちはとても長い準備期間を要した。デ・ラ・セルナはグラナドの体系に近づくため に 1 0 キロ以上太った。ガルシア・ベルナルはアルゼンチンのスペイン語を自然に話せるようになった。３人とも、この旅の日記を始めとして、ゲバラが書いた本、あるいは彼に関する多くの書物を読んだ。ゲバラの日記は長い間サレス監督のお気に入りの本だった。  

 


 

②　チェ 　 2 8 歳の革命

監督：スチーブン・ソダーバーグ

出演：ベニシオ・デル・トロ

Che, el Argentino (2008) de Steven Soderbergh

    

 　ゲバラ は 195 2 年の旅からアルゼンチンに戻り、医学の勉強に集中した 。 195 3 年４月には医学の卒業証書を授与した。その年の７月に再びラテンアメリカの旅に出発した。この時は、子供の頃の友人カルロス・フェレルと一緒に旅に出た。最初の予定はカラカスまで行って、アルベルト・グラナドに会うことだった。バイクではなく、電車でボリビアに行った。その時、ボリビア政府 は 195 2 年に鉱山労働者が推進するクーデターによって誕生した民族革命運動党政権下で、鉱山などの国有化、農地改革、義務教育導入、軍部の解体などを行おうとしていた。ゲバラたちは数週間ボリビアに滞在してからペルーやエクアドルに移る。その内、アルゼンチン人中心の知り合いが増え、エクアドルのグアヤキル市のペンション で6 人はともに暮らした。その時ゲバラはカラカスに行く予定を後にして、グアテマラに行くことを決意する。  

 　ゲバラ は 195 3 年末から８ヶ月間グアテマラで過ごす。グアテマラも、ボリビアと同じようにラテンアメリカの最も貧しい国の一つで 、 195 1 年の総選挙で誕生したハコボ・アルベンス政権の下で 、 195 2 年から農地改革を行っていた。アルベンスはアメリカに依存していた経済から脱却し、独立的な経済発展を目指して、グアテマラを資本主義経済の下で前近代社会から近代社会へ変貌させたかった。その目的を達成するためには、大土地所有と零細土地所有が共存していた土地所有構造を、農地改革を通じて変えることと、発電所や道路などのインフラ整備が主な課題であった。  

 　しかし当時、アメリカのユナイテッド・フルーツ社は、グアテマラで一番多くの土地を所有していた地主だった。ユナイテッド・フルーツ社は以前から中米で安い土地を買い占めていた。その目的は競争相手を存在させないことだった。そうするとバナナの生産量の増加を避けて、バナナの価格は下がらないことになる。それと同時に、バナナの栽培とアメリカへの輸出を独占することができた。コストを押さえるために労働条件を最低限に押さえていた。そのために、中米諸国の政府と密接な関係を確保していた。  

 　アルベンス政権が目指した経済発展は、ユナイテッド・フルーツ社にとって理想的な経済や社会政策とは正反対だった。アルベンス政権の農地改革の内容は、大地主が所有する利用されない土地を、ローカルの農家に引き渡して土地をよみがえらせることだった。このように利用されない多くの土地は、ユナイテッド・フルーツ社が所有していたものだった。アメリカのアイゼンハワー政権 、 CI A とユナイテッド・フルーツ社の間には密接な関係があった。例えば 、 CI A の長官アレン・ダレスはユナイテッド・フルーツ社の弁護士だった。アメリカ政府はアルベンスが共産主義者だと宣言した 。 2 0 世紀の間、ラテンアメリカ諸国の経済や社会政策がアメリカ多国籍企業の利益に不利な場合、アメリカ政府はその政権を倒すことに努めた。  

 　 195 4 年にグアテマラでは、ラテンアメリカの経済的な独立を指示していた多くのラテンアメリカの活動家が集まっていた。ゲバラはグアテマラに滞在の間、やがて彼の最初の妻になるイルダ・ガデアに出会った。ガデアはペルー人経済学者や政治活動家であり、政治亡命者としてグアテマラに移住した。７月３日にクーデターは成功し、親米独裁政権が始まった。アルベンス政権と協力していたガデアは逮捕されたが、８月に釈放された。ゲバラはアルゼンチン大使館に避難した。  

 　グアテマラで民主的に選ばれた政権が、アメリカが推進するクーデターによって倒されたことで、ゲバラの心の中で「革命を現実にするためには暴力しかない」という意識が強まっただろう。そのような意識の萌芽はティーンエイジャー時代から幾つかのエピソードで記録されている。ゲバラ が 1 5 歳になっ た 194 3 年にグラナド（当 時 2 1 歳）はデモに参加していた学生とともに逮捕された。その時、彼らの釈放を要求するためのデモが行われた。グラナドはゲバラの友人であったのに、デモに参加しなかったことを友人に問われると、「デモに参加することは警察の暴力を受けるしかない。銃を持たないと意味がない」と言う。「モーターサイクル・ダイアリーズ」ではマチュピチュを訪ねた時、グラナドの提案で、先住民党を組織して選挙を通じて革命を行おうと言ったシーンでは、ゲバラは「弾丸なしで革命は無理だ」と答えた。  

 　やがて、メキシコでゲバラはフィデル・カストロに出会い、ゲリラ戦を通じてキューバ革命に成功する。映画「チ ェ 2 8 歳の革命」は２人の出会いと革命の成功の３年半 （ 195 5 年７月 〜 195 9年1 月）の間の出来事、特にキューバのゲリラ戦（後半の２年間）を描く。そして映画ではこの期間の出来事が 、 196 4年 1 2 月にゲバラが受けたインタビューの内容と絡み合わせられている。その時ゲバラは国連で演説するためにニューヨークに滞在していた。実際のインタビューの内容と映画で紹介される内容はかなり異なる。ゲリラ戦についてはほとんど触れなかった。インタビューはゲリラ戦、革命成功後のキューバなどに関するゲバラの考え方を紹介するために、監督がとった手法だと思われる。  

 　 195 4年9 月の半ば頃ゲバラは一人でメキシコに入った。当時、フィデル・カストロはキューバで逮捕されていた 。 195 3年7月 2 6 日に、フィデルは率いた小隊はキューバのモンカダ兵営を攻撃した。攻撃の結果として、フィデルの小隊に死者は多く出て、彼は逮捕された。成功として考えられる唯一な要素はバティスタ独裁政権の反対運動が広く知られるようになったことだろう。ゲバラ は 195 5年6 月にメキシコでフィデル・カストロの弟ラウルに出会う。その直後、フィデルは釈放され、メキシコに渡る 。 195 5 年７月にゲバラはフィデルに初めて会った。映画「チ ェ 2 8 歳の革命」のストーリーは２人が出会った夕食会からスタートする。その夜、数時間続いた２人の意見交換の後、ゲバラはフィデルが率いる 「7月 2 6 日運動」に加わることを決意した。フィデルはゲバラに、メキシコに滞在するグループの医者という任務を与えた。  

 　ゲバラは８月に妊娠していたイルダ・ガデアと結婚した。翌年の２月に娘イルダは生まれた。  

 　同 じ 195 6 年２月にメキシコ市の郊外にあるチャルコ市でゲバラを含む 約 2 0 人はゲリラ戦の訓練を始める。６月に、フィデル、ラウルやゲバラを含む、グループの大半はメキシコの警察に逮捕される。釈放されるまで困難だったが 、 1 1月 2 5 日 に 7 8 人はグランマ号というヨットでメキシコのベラクルス州トゥクスパン港を出発する。ガデアは娘とメキシコに残った。  

 　キューバ で 195 2 年３月には、民主政権に対するクーデターを通じて、フルヘンシオ・バティスタが独裁政権を誕生させた。冷戦中バティスタは主な目的として共産主義の脅迫に反対すると宣言した。しかし、カストロを含むバティスタと戦った主な組織は、愛国心主義的で、キューバの経済的な独立とキューバ人の貧困からの脱却を要求していた運動だった。これらの組織に対してバティスタは暴力を利用し、人権損害は膨大だった。  

 　悪天候によってグランマ号の到着は予定より も2 日遅れた。キューバで待っていた 「7月 2 6 日運動」のメンバーはすでにいなかった。その上、グランマ号は座礁したため、食料、薬品や武器の大半は船に残されたままだった。３日後、キューバ軍隊の攻撃を受け、多くの仲間が殺され逮捕され、生き残った人数は 約 2 0 人と思われている。生き残ったグループの内、フィデル、ラウルとゲバラは含まれていた。しかし、バティスタ政権 は3 人とも死者のリストに含めた。生き残ったグループは集合するまで３週間を要した。「チ ェ 2 8 歳の革命」はこれらの出来事を含んでいない。映画で描かれるゲリラ戦のシーン は 195 7月3 月から始まる。最初の舞台はマエストラ山脈だった。  

 　徐々にバティスタ独裁政権の反対運動は全国に広がった。イデオロギーや戦略的に異なる「平原派」と「山脈派」が存在していた。前者では非暴力的な手段（スト、デモ、政党との同盟、学生運動との連携など）が革命の成功のために決定的だと思われていた。また、イデオロギー的には代表制民主主義や反共産主義が支配的な思想だった。一方、「山脈派」はゲリラ戦による革命を起こすことが決定的な戦略だった。彼らによると、「平原派」の活動は 「7月 2 6 日運動」の全国的な支持基盤を拡大するための手段だけであった。イデオロギー的に愛国心主義的な運動でありながら、マルクス主義的な思想が比較的に強かった。  

 　映画で描かれ る 195 8 年５月３日の両派の全国のリーダーたちの会議では、「山脈派」とフィデル・カストロのリーダーシップが確定された。４月のゼネストの失敗はその主な背景だと思われる。会議の直後 、5月6 日に、バティスタ政権は軍隊の大規模な攻撃を行った。最初の段階では成功するようにみえたが、７月後半にゲリラの反撃は始まった。  

 　ゲバラは９月にアレイダ・マルチと出会う。彼女は翌年ゲバラの２人目の妻になる。アレイダはサンタクララ市出身だ。革命運動の成功ために、サンタクララの戦闘は決定的であった。サンタクララの戦闘 は 1 2月 2 8 日に始まり 、3 日間に及んだ。ハバナへの道は切り開かれた。車でハバナに向かう最後のシーンでは、「モーターサイクル・ダイアリーズ」と同じように、ゲバラの倫理的な思想や行動が強調されている。  

 


 

    

    

    

    

    

第2 章

トマス・グティレス・アレア監督と

キューバ革命直後の

社会変貌 （ 195 9 年 〜 196 2 年）

 


 

③　低開発の記憶

監督：トマス・グティエレス・アレア

出演：セルヒオ・コリエリ、

デイジー・グラナドス

Memorias del Subdesarrollo (1968) de  

Tomás Gutiérrez Alea

    

 　 195 9年1 月にキューバ革命政権は誕生した。新政府のメンバーの思想は不均質的だった。「チ ェ 2 8 歳の革命」で引用した「平原派」を代表するメンバーが多く、共産主義よりも愛国心主義的な革命だった。しかし 、 195 2 年のグアテマラと同様に、ラテンアメリカ諸国のナショナリズムの敵は、アメリカの経済的な帝国主義だった。そして、冷戦中のアメリカ政府にとっては、反米の立場を示す政権は共産主義政権ではないかと疑われた。実際、革命政府の中では、ゲバラをはじめとする、マルクス主義的な思想を持っていた人がいた。  

 　アメリカとの対立の中で 、 195 9 年 〜 196 0 年の革命政権の内、「マルクス主義派」が強くなり、資本主義の諸制度のもとで愛国心的（反米的）な経済政策を行おうとする派は弱くなった。その内、農地改革や外資系企業や大地主の収用が進み、上流階級と専門職の一部の人はアメリカに移住した。  

 　「低開発の記憶」では 、 196 1年4 月のアメリ カ CI A による組織や、海外に滞在する訓練されたキューバ反革命派によるキューバ侵略（ピッグス湾侵略）から 、 196 2年 1 0 月のアメリカによるキューバでのソ連のミサイル発見から勃発した「ミサイル危機」までのキューバを描く。しかし、歴史を述べる映画ではなく、上流階級に属する主人公セルヒオの意識変化を通じて、革命の意義を多面的に検討し、個人と社会間の関係に関する普遍的な価値に注意を払う映画だ。  

 　セルヒオは低開発の意味について物語る。彼によるとキューバ人の意識はその瞬間に集中し、長期的な知識や能力の蓄積が無理だと言う。それは低開発の主な原因であり、低開発の心理的、場合によっては民族的な説明である。グティエレス・アレア監督はこの説を越え、映画の多様な場面で人々の革命以前の価値観の存続を強調する。アメリカに渡るセルヒオの妻や友人パブロのような人だけではなく、エレナ、彼女の家族、あるいは文学のシンポジウム形式などで、革命を否定する価値観が生き生きと表れている。「チ ェ 2 8 歳の革命」では、勝利直後に兵士がゲバラに「革命は出来あがった」と言うが、ゲバラは「違う、革命はこれから始まる」と答えた。ゲバラはその時に「新しい人間」 （ Hombre Nuev o ）を生み出す時代が始まったと思っていただろう。だが「低開発の記憶」では、グティエレス・アレア監督はその道のりが長く、目的地はほど遠いと気づかせる。  

 セルヒオの妻の場合や友人パブロの場合にも、革命以前に自分のまわりにいた人の出発を、その時代の生活の終焉として嬉しく受け取る 。2 年間以上に及んだ革命政権のもとでは、妻やパブロの表面的な世界観がセルヒオの知的好奇心と合わないことは以前より明らかであった。セルヒオはキューバを後にしない。それと同時に、セルヒオは一般市民の世界観から遠い。階級社会で育てられ、搾取されていた人々は彼にとって遠い存在だった。実際、カストロを含む革命運動に参加した人の内、裕福な家計の息子や娘、若い専門家（弁護士、医者、教員、大学生など）は少なくなかった。  

 革命以前、セルヒオは父のビジネスを継ぐ道を選んだ。革命後、過去の生活は自分の周囲からも、キューバからも消えていく。新しい社会（新しい人間）は見えてこない。そしてセルヒオは、これから新たな社会制度を形成する人たちとの関係が無い。妻と両親はアメリカへ移住し、セルヒオは飼っていた小鳥（自分？）を籠から放す。セルヒオは孤独の中、社会の変貌を考えてみる。  

 　セルヒオは散歩しながら人々を観察し、女性を通じて自分の階級ではない人と関係を持つ。その女性エレナは、セルヒオが遠くからみている社会への窓になる。遊びから始まった関係は、エレナとその家族の反応を通じて、古くから残る価値観と直面させる。エレナは女優になりた い 1 6 歳の女性だ。セルヒオはエレナの行動や意識を考慮し、その不安定性を強調する。彼女に期待していたが、セルヒオにとっては彼女が低開発の証明そのものであった。  

 　セルヒオとエレナの会話の中で象徴的では、興味深い場面がある。エレナはセルヒオに革命を指示するかどうかを尋ねる。セルヒオは「君はどう思う」と尋ねると、エレナは「あなたは革命派ではない、反革命派でもない。あなたは何者でもない」と答える。スペイン語 の Tú no eres nad a という表現は話題の範囲を越えて、英語 の You are nothin g にあたる。つまり、相手の存在を否定する表現で、セルヒオの状況を表している。自分の過去を嬉しく捨てるが、その代わりになにもない。同じ会話では、セルヒオがエレナに、前妻のドレスが似合ったら君にあげるという。その時に流れている音楽の歌詞は 、 No sientas lástima de m í 、訳すと、「私のことをかわいそうに思わないでください」という意味だ。映画の中で、官能的な場面でありながら、他階級関係も表している。二人の関係で、セルヒオは無意識的に彼女を遠い存在として扱う。そして、エレナと別れたい時、彼女と向き合うことはなく話さずに逃げる。セルヒオの行為は低開発の代表的な行為だと言えるだろう。  

 　セルヒオは自分の過去を振り返ってみる。キリスト教の学校では不公平的に扱われた時、正義と権力の関係を学んだと言う 。 1 3 歳には売春宿に連れて行かれ、その時から毎週通った。多様な思い出のうち、旧恋人ハンナとの関係は特別な意味を持つ。ハンナの家族はヒトラー政権のドイツから逃げ、キューバで暮らしたのち、ニューヨークに移住した。ハンナとの関係は理想的な関係として紹介される。セルヒオは彼女とニューヨークへ行き、小説家になる夢を実現するために二人で努力しようという話があったものの、セルヒオはお父さんの家具屋を継いでキューバに残ることになった。ある程度のお金を稼いだ後、ニューヨークに行こうとしたが、気がついた ら2 年が過ぎていて、二人の関係は終わった。セルヒオは安定と安全を選び、世界観が自分とそれほど合わないラウラと結婚し、パブロのような友人と付き合う。彼の現在の状況は、いろんな人との関係から、そして自分の多様な選択からなったことが明らかになる。  

 　セルヒオはエレナと一緒にアーネスト・ヘミングウェイの家を訪ねる。ヘミングウェイは長い間キューバに住んでいた。セルヒオは、ヘミングウェイとキューバやキューバ人との関係、狩猟や闘牛に対しての情熱などについて、皮肉たっぷりに話す。二人は部屋の中のアメリカの雰囲気について話していた時、セルヒオがエレナにソ連とアメリカのどちらの方を好むと尋ねたら、エレナは自分の考え方を伝えない。セルヒオは自分のマンションで同じような質問（革命派か反革命派か）を受けた時と同じ態度だった。社会変貌の真最中、二人とも自分の立場を伝えない、あるいはその立場がない。しかし、彼らはその社会に結ばれていて、皆必ず社会の行方の影響を受ける。これは映画の主なテーマだ。消極的に社会の影響を受けるか、積極的に社会に影響を与えようとするか、という選択肢がある。  

 　ヘミングウェイは「誰がために鐘は鳴る」と言う名作を書いた。舞台 は 193 7 年の内戦中のスペインだ。激しい社会変化の時代だった。小説のタイトルはイギリスの詩人ジョン・ダン （ 157 2 年 〜 163 1 年）の作品に基づいている。その詩ではダンが「誰でも完全にできた島ではない、人間一人一人は大陸（地球）の一部だ 、 ... ... 誰の死であっても私を小さくする、なぜなら私は人類に結ばれている、だから誰がために鐘は鳴ると尋ねないで：鐘は君のことで鳴っている」と書いた。個人と社会の間には弁証法的な関係があり、お互いに影響を与え受ける。  

 　現代社会では社会変遷に関することなら、「私には関係ない」との答えがよく聞かれる。個人の望みを反映する表現かもしれないが、現実からは遠い。自分は消極的な立場を取る時にも社会に影響を与えている。好みの問題ではなく、自分自身は必ず社会の行方の一部である。この簡単なメッセージはこの映画の革命的なメッセージでもある。  

 　映画の前半には、逮捕されたピッグス湾の侵略者の供述を通じて、個人とグループの関係が紹介され、組織内の役割分担とブルジョア社会の役割分担が比較される。映画の最後のシーンでは、キューバ社会が歴史上最も緊張した瞬間に近づく 。 196 2年 1 0 月のミサイル危機だ。その時、セルヒオは一人で自分のマンションにいる。社会から完全に離れていると感じさせる。しかし、それは外観に過ぎない。これらの出来事の影響はセルヒオやマイアミに逃げた人たちだけではなく、本書でこれから述べるように、ラテンアメリカの人々の将来と密接な関係があった。  

 　映画はエドムンド・デスノエスの小説に基づいている。この小説は日本語に翻訳されている。デスノエスは文学のシンポジウムのシーンに役者出演する。ナレーターはデスノエスの革命体制中の高いステータスについて皮肉的に語る。シンポジウムのパネラーはデスノエス、アルゼンチンの小説家ダビッド・ビニャス、ハイチの詩人ルネー・デュペストレとイタリア人小説家ジャンニ・トッティだ。聴衆の中にいたアメリカ人は、革命政権が社会秩序を全面的に変えたいのに、なぜ未だに出席者は議論にあまり参加できないシンポジウム形式にこだわるのかを尋ねる。出席者の間にほくそえみがみられる。そのアメリカ人はデスノエスの小説を英語に翻訳したジャック・ゲルベルだ。このシーンで、グティエレス・アレア監督は、革命の知識人の行動を通じて、「新しい人間」を形成するまで長い道があることを伝える。監督は革命政権を建設的に批判し、革命の中の革命は必要とされることを強調する。このことは、グティエレス・アレア監督の最後の作品、「苺とチョコレート」では重要なテーマになる。  

 　グティエレス・アレア監督 は ICAI C （キューバ映画芸術や映画産業協会）本部にて革命以前の映画でカットされていたシーンについてセルヒオと会話するシーンに出演している。  

 


 

④ 　 1 2 の椅子

監督：トマス・グティレス・アレア

出演：エンリケ・サンティエステバン、

レイナルド・ミラバジェス

Las Doce Sillas (1962) de Tomás Gutiérrez Alea

    

 　 「 1 2 の椅子」 （ 196 2 年）は「低開発の記憶」 （ 196 8 年）の６年前に作成されたが、両方の作品は革 命2〜3 年後の社会の急速変化を描く。「低開発の記憶」はキューバ社会の内省分析でありながら、 「 1 2 の椅子」はコメディー映画で、パロディーを通じて社会の急変の特徴を紹介する。主人公である旧大地主イポリトや彼の旧使用人オスカルは革命思想と関係なく、二人とも現状から最大の利益を得られるように行動する。その内、社会制度の変化による二人の間の新たな関係が徐々に表れてくる。原作は旧ソ連 で 192 8 年に出版された同名の小説である。  

 　イポリトの母親は死亡する直前、神父から終油の秘跡を受けていた時、 「 1 2 の椅子の内の１つに宝石コレクションを隠した」ということを伝える。その後、イポリトに同じことを言う。神父とイポリトは別々 に 1 2 の椅子を探す 。 1 2 の椅子は革命政府に収用されたものであった。  

 　イポリトの家は老人の保護施設になっていた。１つの椅子はある老女の部屋に残っていた。神父は強制的にその椅子を老女から奪い取る。その時老女はマルクスの資本論を読んでいた。その後イポリトは、椅子を持って逃げる神父を偶然に見かけ二人は戦うが、結局その椅子に宝石はなかった。  

 　大半の椅子は財産回収省にあった 。 195 9 年 と 196 1 年の間、キューバに「財産回収省」が存在した。正式に 「 Ministerio de Recuperación de Bienes Malversado s 」、つまり「横領された財産回収省」と名付けられた。革命政府の理解では労働者の搾取を通じて蓄積された財産は労働者に返すべきものである。  

 　「財産回収省」は椅子をオークションにかける予定だった。それを知ったイポリトには十分なお金がない。オスカルのアイディアで、二人の知り合いであるヘルトルディスを通じて反革命会議を開き、資金をカンパしようとする。その言い訳は、反革命暫定政府を立ち上げることだった。ヘルトルディスは革命後の自分の仕事を振り返ってみる。裕福層のために多様なサービスを提供していたが、顧客は次々とアメリカに渡る、新たなキューバ社会では彼女のサービスは必要とされていない。ヘルトルディスを含む、オスカルに資金を渡す人たちは自分の利益しか考えていないということは明らかだ。他方、神父はカトリック教会の経済的な協力を得て椅子を探し続ける。  

 　オークションで椅子を手に入れることが出来なかったため、二人は色んな手段を使って探し続ける。椅子は一客ずつ売られたので、色々な所へ足を運ぶ：献血センター、チャイナタウン、サトウキビの収穫、サーカス、鉄道員の労働組合。どこでも、イポリトとオスカルの態度は利己的であり、連帯を優先する社会的な雰囲気と大きな差を見せる。しかし、オスカルは以前から労働者であり、どこでも一般市民との関係は自然的で、最初からうまく行く。イポリトの場合、誰とも付き合うことができない。家主と地主として自分の「場」はなくなったにも関わらず、彼はそれを受け入れようとしない。一般市民と平等な立場で行動することができず、アメリカ政府の援助を待つしかない。他方でオスカルは利己主義者であっても、新たな社会的関係を楽しめる。  

 　イポリトとオスカルの間の関係は徐々に変わって行く。ある時、一変したかのように見える。どのように利益を分け合うかに関して、オスカルが有利になっていく。その内、オスカルは旧家主イポリトに「貴方は働いたことがない」と言う。イポリトは「働く必要がなかった」と自信満々答える。ラテンアメリカの上流階級の観点から考えると、働くと言うことは一般市民あるいは貧しい人々だけがすることである。このような意識は植民地時代から存在する。大地主の間には起業家精神が極めて低く、蓄積した資本は主に贅沢消費のために利用されるようになった。労働者の購買力は低く、国内市場は広がりにくい。裕福層は贅沢な輸入品を消費していた。キューバ革命はこの状況を終わらせようとした。  

 　イポリトの自慢はもはや時代遅れだ。オスカルはイポリトに足りない資金を乞食に成りすませて集めさせる。その後、二人は偶然にサトウキビの収穫に参加する。トラックの荷台に多くの人と一緒に乗り、革命歌を歌う 。 196 0 年 〜 196 1 年に多くの人は初めて収穫に参加していただろう。皆ひとつの目標に向かって協力し合う喜びを感じさせる。イポリトは出来るだけ早くこの状況から抜け出したい。二人の社会秩序に対する意識は離れていく。最後 の 1 2 個目の椅子は、鉄道員の労働組合にある椅子で、あそこに宝石コレクションがあることは、ほぼ確実だった。椅子を手に入れたイポリトはオスカルを裏切るのだが、結局宝石コレクションは、鉄道員の労働組合がもっと前に見つけていたのであった。労働組合はその財産を活用して、組合員のために立派な建物を建築した。オスカルはイポリトに「貴方はニューヨークに行かない、キューバで死ぬ、そして貴方のお葬式に誰も行かない」と言う。イポリトは一人で逃げて行く。オスカルはその場で若者たちが行っていた野球の試合に加わる。彼はやっと雇い主の本質を意識することで、新たな社会の意義を理解したのだろう。  

    

 　 200 1 年 に NH K が放送した「３７年ぶりの帰郷　～亡命キューバ人が見た祖国～」というアメリカで制作されたドキュメンタリーでは、キューバの大地主の娘シルビア・モリニとその息子と、生まれ育ったキューバの家を見に行く。シルビアの父親はサトウキビを栽培する最大の大地主だった。シルビアと彼女の息子 は 196 0 年にアメリカに渡った 。 199 7 年には、彼女が生まれ育った家は金融機関になっていた。自分の財産は「財産回収省」によって父親の労働者に渡された。隣の家には未だにモリニ家の旧使用人たちが住んでいた。  

 　一人の旧モリニ家の労働者は革命以前のサトウキビ畑においての労働条件について語る。農業休み季節には生き延びることが精いっぱいだったと言う。シルビアの家と使用人との関係は映画 「 1 2 の椅子」を思い出させる。イポリトは政府に全てとられたと言う。オスカルは何もとられなかったという話になると、イポリトは「もちろん、君は何も持っていなかったから」と言う。オスカルはイポリトの労働者だったから何も持っていなかったと言えるだろう。しかし、ドキュメンタリーの内容を考慮する限り、モリニ家の旧労働者は利己主義者ではなく、協力や連帯意識を持つ。  

 　シルビアもイポリトと異なる。彼女はキューバの生産社会的関係の変貌を徐々に認めるようになる。社会変貌の特徴を認識する過程は一般市民と付き合いながら発展して行く。彼女の心には昔の世界観が残っても、人間らしさが目立つ。しかし、時々昔の家主として「上から」話すこともある。イポリトとの大きな違いは彼が一般市民と会話することに興味がないことだ。彼は全てもとに戻るという期待が強い 。 196 0 年には、アメリカに渡った多くのキューバ人は革命政権がすぐに終わると思っていた。しかし、「低開発の記憶」で取り上げ た 196 2年 1 0 月のミサイル危機の妥協による、ソ連はキューバからミサイルを撤廃する代わりに、アメリカはキューバに軍事攻撃を行わないと約束した 。 3 7 年後、シルビアは全てもとに戻るという期待がない。シルビアは 、 3 7 年間という長い時間の間隔を徐々に埋めていく。アメリカに戻ると、経済封鎖の終焉を要求するようになる。政府関係者に無視されるとうつ病に陥る。  

    

 　トマス・グティエレス・アレア監督は 「 1 2 の椅子」 の4 年後 、 196 6 年には「ある官僚の死」を発表する。この映画もコメディーであり、チャールズ・チャップリン、ローレル＆ハーデ ィ, バスター・キートンやハロルド・ロイドへのオマージュでもある。コメディーでありながら、映画は革命政権の官僚主義や生活の標準化への強い批判である。 「 1 2 の椅子」のサトウキビ収穫の自発的な喜びではなく、他人の悩みに対する無関心がみられる。ある労働者は彫像を大量生産するための機械を発明する。彼は全ての市民の家にキューバ独立運動の英雄や詩人であるホセ・マルティの像があることを目指す。彼は「模範労働者」と任命されるが、チャップリンの「モダン・タイムス」のシーンと同じように、機械に飲み込まれ死に至る。彼の労働組合の証明書は彼の遺体とともに墓に埋葬される。そうなると、手続きの関係で未亡人に年金が支払われない。遺体を掘りおこすための手続きは永遠に終わらない。フランツ・カフカの小説「城」を思い出させる。  

 　多くの街角にあったマルティの像は新たな宗教の誕生のように考えられる。拡大していた官僚主義も革命の「固定化」の危険性を物語っていた。映画を制作する知識人の任務はこれらの偏向を明らかにし、革命のダイナミズムを確保することであると思われていた。知識人の役割に関する議論 は 196 0 年代の半ばのキューバで活発だった。「ある官僚の死」はその議論の１つの成果として考えられる。  

 　 「 1 2 の椅子」と「ある官僚の死」の最初の部分にはアニメのシーンがある。裕福な婦人が椅子に宝石を隠すシーンと、労働者が機械に飲み込まれるシーンはアニメである。両方の映画は東京で開催された「キューバ映画 祭 200 9 」で上映され 、 DV D 化された。キューバの芸術、キューバ社会や革命の動向の理解に関して、貴重な貢献である。

⑤　チェ 　 3 9 際別れの手紙

監督：スチーブン・ソダーバーグ

出演：ベニシオ・デル・トロ

Che, Guerrilla (2008) de Steven Soderbergh

    

 　エルネスト・チェ・ゲバラはキューバ革命運動が成功しても、また、キューバ革命政権で中心な役割を果たしても、ゲリラ戦を通じて革命を世界に広げるという目的を決してやめなかった。彼はラテンアメリカ、アジアとアフリカの低開発諸国を経済的な帝国主義から脱却させることを目指し、そのための唯一な手段はゲリラ戦であると確信していた。ゲバラの思想はソ連が促進していた資本主義諸国との「平和共存」戦略と対照的であった。  

 　ゲバラは革命政権の最初の 約6 年の間、キューバの農地改革、製造業の発展、ソ連や中国の国々との経済・政治的な関係の深化や拡大の促進に関する中心な役割を果たしながら、ラテンアメリカ諸国ではゲリラ戦を推進した。ゲバラが推進していた運動と、ラテンアメリカ諸国におけるソ連型共産党との間には対立が生じた。以下で考慮するように、その対立は「チ ェ 3 9 歳の別れの手紙」と「クアトロ・ディアス」の両方の映画で異なったかたちで表れている。  

 　ゲバラは、祖国アルゼンチンでもゲリラ戦を切り開きたかった 。 196 3 年にゲバラは、キューバで長い訓練を受けたグループをアルゼンチンに送った。「国民ゲリラ群」として知られるようになったそのグループはアルゼンチン北部、ボリビアと国境線があるサルタ州で活動した。ゲバラは過去に暮らしたコルドバ州やブエノスアイレス市から支えられ、ボリビアの活動家とも関係を築いていた。翌 年 196 4年4 月にこのグループのゲリラ戦は失敗に終わった。メンバーは戦中に亡くなり、あるいは逮捕された。グループのリーダーであったホルヘ・マセッティの行方は未だに不明である。マセッティ は 195 8 年からキューバで活動したアルゼンチン人記者であった。ゲバラはサルタでこのグループに加わる予定だったが、グループの敗戦でその予定が変更された。  

 　 196 4 年の年末にゲバラは海外でゲリラ戦を指導するために、キューバ政府における自分の任務を辞職した。彼が最初に向かった地域はラテンアメリカではなく、アフリカのコンゴ共和国であった。アフリカ大陸の中央にあり 、9 カ国と国境を持つコンゴ共和国から全アフリカ大陸にゲリラ戦を拡大するつもりであった。コンゴ共和国 は 196 0 年にベルギーの支配から独立し、独立運動のリーダーであったパトリス・ルムンバが最初の首相になった。しかし、ルムンバは首相とし て3 ヶ月しか続かない （ 196 0年6 月 〜9 月） 。 196 0年 1 2 月のクーデターによって軍事独裁政権が誕生する 。 196 1年1 月には多国籍企業のビジネス活動を守る目的と、冷戦中に資本主義諸国のアフリカ大陸の支配を強めるために、アメリ カ CI A の指導に従って、ルムンバは暗殺される。ルムンバは共産主義者ではなく、愛国心主義者であり、コンゴの独立的な経済発展を目指していた。ルムンバの暗殺はキューバ革命 の1 年後に起きた。以上記述したコンゴの出来事 は 195 4 年にゲバラがグアテマラで体験したアルベンス政権に対するクーデター（ 第2 章参照）とよく似ている 。 196 4 年コンゴでは、タンザニアから支えられていた反独裁政権ゲリラ組織が活動していた。  

 　ゲバラはコンゴのゲリラ運動に加わることを決意し、映画のタイトルでもある「別れの手紙」を書いた 。 196 5年 1 0 月に行われたキューバ共産党の 第1 大会ではフィデル・カストロがその手紙を読んだ。映画 「 3 9 歳別れの手紙」はこのシーンから始まり、次に映画の場面は 「1 年後」つま り 196 6年 1 0 月頃に移る。映画ではゲバラのコンゴにおけるゲリラ活動は取り上げられない。  

 　ゲバラ は 196 5年4 月にタンザニアの首都ダルエスサラームに到着し 、 1 1 月まで派遣され た 12 0 人のキューバ人とコンゴのゲリラ組織とともに戦うが、多くのトラブルがあり、ゲバラの戦略は大失敗に終わる。失敗の要因はコンゴのゲリラ組織のリーダーたちとの関係、コンゴのゲリラ兵士やリーダーたちの無規律、ゲリラ組織の内多様な派閥の存在、言語や文化の問題、そして最終的にタンザニア政府のサポートの取りやめが挙げられる。その後、ゲバラはダルエスサラームのキューバ大使館で数週間を過ごし、その間コンゴの経験について書いた。ダルエスサラームから旧チェコスロバキアに移り、首都プラハ で5 ヶ月過ごす。その間、ゲバラはボリビアでゲリラ戦の源を切り開くことを決意した。準備を行うため に 196 6年7 月に秘かにキューバに戻る。  

 　人相を変えたゲバラは自分のアイデンティティを隠しなが ら 1 1月2 日に自宅で自分の子供たちに会う。このシーンとフィデル・カストロとの会話は映画で紹介されている。ゲバラ は 1 1月3 日にボリビアに到着し 、7 日にボリビア共産党リーダーであったマリオ・モンヘが所有す る 150 0 ヘクターの広い農場に 、 2 4 人のゲリラ兵士とともに身を落ち着ける。 「 3 9 歳別れの手紙」はゲバラ暗殺までの、ボリビアのゲリラ活動を物語る。ゲリラ組織は「ボリビア民族解放運動」と名づけられた。  

 　ゲバラ の 195 3 年の 第2 ラテンアメリカ旅での最初の目的地はボリビアであった（ 第2 章参照）。彼はボリビア民族革命運動党のパス・エステンソロ政権の行方を現地で確かめたかった。パス・エステンソロ の2 期目の政権は軍事クーデターによって倒された。その軍事クーデターを率いたレネ・バリエントスはゲバラ が 196 6年 1 1 月にボリビアに到着した時にボリビアの大統領であった。ボリビア最大の労働組合組織であったボリビア労働者中央本部は、バリエントス政権を倒すために熱狂的に戦っていた。最大の加盟組合はボリビア炭鉱労働者同盟であった。  

 　ボリビア共産党の立場を代表するマリオ・モンへはゲバラ達が活動する農場を提供したが、ゲリラ戦に対して批判的な見方を持っていた。特に、一般市民から離れた場所で、革命運動の源としてゲリラ組織をスタートさせることに対して批判的であった。その上、ソ連の「平和共存」戦略があり、ソ連はラテンアメリカ諸国の共産党にゲリラ戦を拒否するように指導していた。そして、共産党のリーダーとボリビア人としてモンへは、革命運動のリーダーに自分を任命するよう要求した。しかし、ゲバラは自分が運動のリーダーであることを明らかにした。映画ではボリビア共産党とゲバラ率いるゲリラ組織間の決裂が取り上げられる。モンへとゲバラの会談 は 196 6年 1 2月 3 1 日に行われた。時間が経つにつれて、モンへと共産党の立場はゲバラの活動に対してより批判的になる。モンへは過去に他のボリビア共産党員とともにキューバでゲリラ戦のための訓練を受けたことがあったにも関わらず 、 196 7 年にそれらの共産党員にゲバラのゲリラ組織に参加しないように指導し言い聞かせた。これらの出来事はゲバラの組織を孤立させた要因の一つである。  

 　一般市民との関係がうまくいかなかったもう一つの要因は、ゲバラの組織が活動していた地方に暮らしていた農家の態度である。「ボリビア民族解放運動」の農場（キャンプ）はサンタクルス市か ら 25 0 キロ離れて、ニャンカウアス川に横断されていた。その地方では農家が少なく、外から来る人に対して不信感が強い。その上 、 196 7年3 月という早い段階でボリビア政府はゲバラの組織の存在を知ってしまった。「ボリビア民族解放運動」は農家との協力基盤がまだできていない時点ですでに、ボリビア軍は農家に協力を求めていた。農家の協力基盤はゲリラ戦の「源」を拡大させるために重大な要素である。キューバのマエストラ山脈ではメキシコから渡っ た 2 0 人以内の運動家しか残っていなかったが、都市からの指示と彼らのメッセージに対する農家の感受性が運動の拡大を可能にした主な要因であった。  

 　ゲバラのキューバ離れとボリビア戦略に関しては疑問が残っている。ゲバラは多くの国でゲリラ戦の「源 」 (foco)   の形成を通じて革命を世界にもたらしたかったことは確実だが、それを実現するためにキューバからの出発したタイミングに関して以下の説がある。ゲバラはキューバ革命の方向性に不満を持っていた。彼は、最初の段階からキューバで製造業の発展は不可欠であると思っていた。そのために、最初に重工業、特に鉄鋼業を発展させるべきだと思って 、 196 0 年から「経済計画中央委員会」の設立を通じてその課題に勤めた。その後、ソ連を始めとして、共産主義諸国を何度か訪ねた。  

 　その内、フィデル・カストロと他の革命政府の幹部はキューバ革命が生き残るためにソ連に近づくことは不可欠であると思うようになった。ソ連は共産主義諸国の間、国際分業関係を促進していた。キューバの場合はサトウキビを提供すると、資本主義諸国の市場より利益の高い交換相場で製造業の生産物を手にいれることが確実だった。そうなると、キューバの産業化を諦めるしかなく、ゲバラの経済発展に関する考え方は後回しにされた。  

 　ゲバラは「平和共存」戦略に対しても批判的であった。ゲバラによると、協力や連帯を基本にする社会的関係を広げるために、できるだけ早くアメリカ帝国主義を無くす必要があった。ゲバラは直ちに市場経済の代わりに労働に対して連帯に基づいた姿勢を実現したかった。ソ連の共産党はそれを長期的な目標として考えていた。ゲバラにとっては、ソ連よりも、中国の毛沢東やアメリカと戦争中の北ベトナムの政治的な姿勢のほうが好ましかった。  

 　 196 5年2 月にアルジェリアで行われた「アフリカとアジア間の連帯に関する経済セミナー」において、ゲバラは自分のソ連に対する批判的な見方を明らかにした。ゲバラはキューバに戻ると、フィデル・カストロと対談し 、2 週間後ゲバラの行方は不明になる 。4 月に秘かにコンゴ共和国へ出発する。  

 


 

⑥　クアトロ・ディアス

監督：ブルーノ・バレット

出演：アラン・アーキン、ペドロ・カルドーゾ、フェルナンダ・トーレス、

フェルナンダ・モンテネグロ

    　 O Que é Isso, Companheiro? (1997)  

de Bruno Barreto

    

 　ブラジルの面積と人口はラテンアメリカの最大である。ブラジルはチリとエクアドル以外、南米 の9 カ国と国境を持つ。冷戦中アメリカ政府にとってはブラジル政治の行方が大切であった。ブラジルは比較的に豊かな国であるものの、地方により、そして社会階層や階級により極めて大きな貧富の格差が存在する。  

 　 195 0 年代後半にブラジルの大統領はジュセリーノ・クビチェック （ 195 6 年 〜 196 1 年）であった。彼の主な目的はブラジルの経済発展の基盤を拡大し、多国籍企業の直接投資を増やし、ブラジルの工業化（重工業、自動車産業など）を促進することであった。そのための主な手段は道路、発電所などのインフラ整備やアメリカとの関係の強化であった。彼のスローガンは 「 5 0 年の進歩 を5 年で」であった 。 195 6 年からブラジリア市は建設され 、 196 0 年にはブラジルの首都になった。国土の中部にあり、ブラジルの経済統合を象徴する町であるが、経済発展は以前と同じようにサンパウロを中心として南部に集中した。大勢の人は北東部や北部から南部に移住することになった。  

 　クビチェック政権は多くの市民の指示を得て、ブラジルの黄金時代のようであった。最低賃金は以前より高く、一般労働者の生活水準が向上された。中流階級は拡大し、その中から民衆文化の新たな運動が誕生した 。 195 8 年にリオデジャネイロでボサノバという新たな音楽ジャンルが誕生し、数年後ラテンアメリカ、アメリカやヨーロッパでもよく知られるようになる 。 195 8 年サッカ ーW 杯のスウェーデン大会ではブラジル代表が初めて優勝した。映画「クアトロ・ディアス」の冒頭で観られるリオデジャネイロ の 196 0 年代始めのシーンは当時の楽観主義を思わせる。砂浜、ボサノバ、サッカー、カーニバルなど、世界に広がったブラジルのイメージ は 195 0 年代の経済発展とともに誕生した。  

 　 196 1年1月 3 1 日に初めて新首都ブラジリアで新大統領は就任される。国民民衆同盟の候補者として政権を握ったジャニオ・クアドロス大統領 （ 196 1年1 月 〜8 月）は、ソ連と革命政権のキューバと外交を確立した。クアドロス大統領はエルネスト・チェ・ゲバラにブラジルの最も高い「南十字星勲章」を与えた。クアドロス大統領は共産主義者ではなく、彼の考え方は冷戦に対して中立主義的な立場を取り、両サイドと外交を持つことだった。クアドロス大統領は民族自決を求めて、その観点か ら 196 1年4 月にアメリカ の CI A が指導したキューバへのピッグス湾侵略を批判した。言うまでもなく、クアドロスの外交政策はアメリカ政府や多国籍企業、ブラジルの軍隊とブラジルの保守的な政党の好みではなかった。クアドロス大統領はクビチェック政権下の汚職を強く批判し、大統領選挙のキャンペーンで多くの資金を使うことも批判した。クアドロス大統領の出身は裕福な家庭ではなく、大企業や権力者との関係は薄く 、 196 1年8月 2 5 日に、自分に対して多様な圧力があることを訴え、大統領を辞任した。  

 　そのためジョアン・ゴウラール副大統領が大統領 （ 196 1 年 〜 196 4 年）になった。ゴウラール大統領はブラジル最南であるリオグランデ・ド・スル州出身で、影響力があった大地主の長男であった。クアドロスが政権を握った総選挙で、ゴウラールはブラジル労働党 （ PT B ）や社会民衆党 （ PS D ）の指示の下で、副大統領になった。当時のブラジルでは大統領と副大統領は別々に選ばれ、それぞれの席に異なる政党の候補者が選ばれる可能性があった。ゴウラール大統領は社会民衆党、労働党、社会党や共産党の幹部と関係があったため、大地主や軍隊の一部を含むブラジルの保守派、アメリカ多国籍企業とアメリカ政府は彼が大統領になることに関して不満を感じていた。  

 クビチェック政権でも参加した経済学者セルソ・フルタドはゴウラールの経済計画大臣になり 、3 カ年計画を提案した。ゴウラール政権 は3 カ年計画を通じて進歩的な経済改革を行おうとしたが、議会はそれを承認しなかった。改革の内容は農地改革、教育改革、税制改革、読み書きできない人が投票できるような選挙法改正、金融機関法改正と電力、石油や薬品会社の国有化を含んでいた 。3 カ年計画は実行されないまま、経済危機に陥り、大地主、軍隊、右派、多国籍企業やアメリカ政府の不安は高まる 。 196 4年3 月にゴウラール大統領は政令を通じて経済改革を始めようとした 。3月 3 1 日に軍事クーデターが勃発した。  

 　 200 4 年に機密扱い資料を解除され、ブラジルのアメリカ大使リンカーン・ゴルドンはアメリカ政府の関係者 や CI A 幹部に、ブラジルで軍事クーデターを指示するように要求したことが知られるようになった。ゴルドン大使によると、ゴウラール大統領やブラジル共産党は共産主義独裁政権を確立するためにクーデターの準備を始めていた。ゴルドン大使はアメリカ政府にカステロ・ブランコの軍事クーデターを成功させるために、武器、石油 や CI A の参加を求めた。  

 　軍事独裁政権 は 196 4年3月 3 1 日のクーデターにより誕生し 、 198 5 年まで続いた。軍事政権はクビチェック、クアドロス、ゴウラールの元大統 領3 人とセルソ・フルタドを含む多くの知識人や政治家の政治活動を禁止した。  

 　以上記述した政治経済に関する出来事と同時に 、 195 0 年代後半 や 196 0 年代前半には、映画製作においてブラジルのシネマ・ノヴォ（新しい映画）と言う映画運動が誕生し、発展した。イタリアのネオレアリズムやフランスのヌーヴェルヴァーグの影響を受け、国民に自分の生活を見せた。自分の現実を見ることを通じて人々の間にはその現実を変える必要があると言う意識を広げるようにした運動である。運動のスローガンは「手に一台のカメラ、頭に一つのアイディア」である。ネルソン・ペレイラ・ドス・サントス監督の「リ オ 4 0 度」 （ 195 5 年）はシネマ・ノヴォの最初の映画として知られている 。 196 4 年にはグラウベル・ローシャ監督の「黒い神と白い悪魔」が上映される。ペレイラ・ドス・サントスとゴラウベル・ローシャはシネマ・ノヴォ運動の指導者とみなされている。映画「クアトロ・ディアス」の冒頭で 、3 人の若者がマンションの一室で話している時、壁には「黒い神と白い悪魔」のポスターが貼ってある。「クアトロ・ディアス」 は 196 9 年の出来事を記録する。「黒い神と白い悪魔」は当時の学生運動の運動家にとって欠かせない作品であった。それとともに、ブルーノ・バレット監督からグラウベル・ローシャ監督へのオマージュでもある。  

 　 196 8 年に大規模なデモが全国の大都市に広がり、リオデジャネイロやサンパウロの学生運動は最も活発であった。映画「クアトロ・ディアス」の冒頭ではその時のデモのシーンが記録されている 。 196 8 年に多くの国では学生運動が活発になっていた。例としてはフランス の5 月革命、メキシコのトラテロルコ広場事件と日本の学生運動が挙げられる。学生運動は反ベトナム戦争運動と重なり、反アメリカ帝国主義の側面が強かった。その上ブラジルでは、アメリカの軍事独裁政権の成立や発展への支援が加わる。  

 196 8 年の学生運動の激しさの背景にはもう一つの出来事がある。それは映画 「 3 9 歳別れの手紙」で取り上げられるボリビアにおいて、エルネスト・チェ・ゲバラ の 196 7年 1 0 月の暗殺である。ゲバラの作戦の失敗はラテンアメリカの反米運動を弱体化せず、むしろゲリラ戦は唯一な道であるという考え方が全域に広がった。軍事独裁政権の誕生とゲリラ戦の拡大は、この時代のラテンアメリカ社会や政治経済を特徴づけている。  

 映画「クアトロ・ディアス」の冒頭 で3 人の若者はテレビを見ている時、壁にはゲバラのポスターは貼ってある。そのポスターにはゲバラの言葉が引用され、訳すると「硬くなるべきだが、優しさを無くしてはいけない 」 (hay que endurecerse pero sin perder la ternura jamás ) ということである。この言葉はその時点でゲリラ戦の道を選ぶ２人の若者にとって相応しい。一人はカトリック教会の神父になるために勉強している。もう一人、主人公フェルナンド（ゲリラ戦名パウロ）は大学生である。彼らは武器を使ったことがなく、純粋な心を持ちながらも独裁政権と戦うことに義務感を持つ。彼らの複雑な気持ちは映画を通じて伝わってくる。三人目の若者はゲリラ戦の道に入ることは自殺的な行為であり、政治的にも誤った選択であると考え、ゲリラ戦に参加しないと決意する。ゲリラの道を選ぶ若者たちが形成する組織名は 「 1 0月8 日革命運動」であり、 「 MR 8 」として知られている 。 1 0月8 日はエルネスト・ゲバラが逮捕された日である。  

 「クアトロ・ディアス」の主人公である若者たちがゲリラ戦の道を選んだ要因はもう一つある 。 196 8年 1 2月 1 3 日に軍事独裁政権が発表し た AI 5 として知られている５つの政令により以前より厳しく政治活動を取り締まり、テレビ、ラジオ、新聞、雑誌などのメディアが検閲されるようになる。政治的な組織は自分の立場を表現できる場が無くなる。この措置 は 196 8 年の学生運動に対する政府の対応である 。 A I ５の発表はクーデターの中のクーデターであると知られていて 、 A I ５を促進したのは軍事政権の中の右派である。映画でフェルナンドはこの政令が平和的な立場を不可能にしたと言い、ゲリラ戦の道しかないと言う。しかし、演劇の俳優でもある三人目の若者は平和的な政治活動の可能性はまだあると言う。多くの場合この選択肢は生死に関わることになる。  

 ゲリラ戦を選んだ若者達は地下組織に参加することになり、政府にとっては見つかりにくくなる。軍隊とゲリラ、両方の組織にとっては敵に関する情報を得ることが主な目的になる。その意味で「クアトロ・ディアス」ではブラジル社会や政治経済に対する軍隊やゲリラの思想についてほとんど何も述べられていない。記録されるのは情報を得るための戦いであり、社会派の映画でありながらスリラーでもある。映画の主人公フェルナンド（ゲリラ戦名パウロ）の正式の名前はフェルナンド・ガベイラである 。 194 1 年に生まれたガベイラ氏は記者と政治家として活動を続いた 。 199 5 年 と 201 1 年の間、リオデジャネイロ州を代表する衆議院議員だった。映画「クアトロ・ディアス」は、ガベイラ著 の 197 9 年に出版された本に基づいている。フェルナンド・ガベイラは冷戦時代の多くの政治活動家と同じように、逮捕、拷問と亡命を経験した。

 　サルバドル・アジェンデ は 197 0年 1 1 月か ら 197 3年9 月の間、チリの大統領であった。アジェンデ は 190 8 年にバルパライソの裕福な家庭に生まれ 、 193 3 年にチリ社会党の創立に携わり、享 年 6 5 歳まで社会党の一員として活動した。アジェンデ政権の特徴は、資本主義経済と代表民主制の下で誕生した政権でありながら、社会主義経済の創立を目指した政権である。キューバ革命に至ったゲリラ戦ではなく、総選挙で政権を握り、憲法を尊重しながら次第に社会主義経済の様相を導入しようとした。この出来事は冷戦真最中のアメリカが、ラテンアメリカで「もう一つのキューバを許さない」と言う立場を取ってから起きたことである 。 197 0 年には南米で最も経済力を持っていたブラジルとアルゼンチンが軍事独裁政権下にあった。  

 　チリのクーデター は 197 3年9月 1 1 日に勃発した。軍隊は大統領宮殿を攻撃して、アジェンデは国民に最後のラジオ演説をしたのち自殺した。映画「マチュカ」で主人公ゴンサロは、クーデターのシーンとピノチェット独裁者による最初の演説をテレビで見る 。 200 9 年には当時のアジェンデ政権に関するホワイトハウス機密解除文書が公の場に出て話題になった。それ以前、クーデターにおいてアメリカ政府 と CI A の役割は知られているが、その段階では決定的とは言えなかった。アメリカ政府 と CI A が、アジェンデ政権 の2年 1 0 ヶ月の間に、政権の政治的・経済的な不安定要素を作り出す役割を果たしたことは、これらの文書を通じて再認識できるようになった。冷戦中、平和的に社会主義経済へ移行する唯一の試みは成功することがなかった。その原因はアメリカの影響力だけではなく、多様な要素と絡んで複雑であるので、アジェンデ政権の主要な特徴を振り返ってみよう。  

 　 197 0年9月4 日の総選挙の投票率は、アジェン デ 36. 3 ％、アレッサンドリ（保守派 ） 34. 9 ％、トミック・ロメロ（キリスト教民主党 ） 27. 9 ％という結果だった。どの候補者も大半の投票数を得られない場合、議会が大統領を選ぶことになっていた。アメリカのニクソン大統領の命令に従っ て CI A はアジェンデが大統領にならないように戦略をたてたが失敗に終わった 。 CI A は反共産主義者が多いチリの軍隊と接触しクーデターを示唆するが、陸軍司令官レネ・シュネイダーは憲法を尊重する立場を取る 。 CI A は、軍隊の反シュネイダー派を通じ て 197 0年 1 0月 2 2 日にシュネイダーを暗殺する。その結果、軍隊とそれぞれの政党では憲法を守る意思が益々強まり、議会 は 1 0月 2 4 日にアジェンデを選んだ（アジェン デ 15 3 票、アレッサンド リ 3 5 票） 。 1 1月4 日にアジェンデ大統領は就任された。  

 　アジェンデ政権の経済大臣ペドロ・ブスコビックは資本主義から社会主義への移行経済計画を推進した。その主要な柱は、銅鉱山を始めとするチリ経済の中心的な産業や企業の国有化、農地改革、労働者の購買力を増加させるために物価の凍結と賃上げをすることであった。アメリカ多国籍企業のアナコンダ銅鉱業社と、ケンネコット社が所有する銅鉱山は国有化された。映画「モーターサイクル・ダイアリーズ」（ 第1 章参照）では 、 195 2 年にゲバラとグラナドがアナコンダ銅鉱業社の所有するチュキカマタ鉱山（当時世界最大銅鉱山）を訪ねるシーンがある。そこでゲバラは職長に労働者の不適切な扱いかたを訴えたところ、職長はここが観光地ではなくアナコンダ社の私的所有地であると答えた。ゲバラは職長が運転していたトラックに石を投げた。これらの銅鉱山の採掘から多国籍企業が多くの利益を得ていたことは南米でよく知られていた。  

 　経済改革の決定的な失敗は物価の凍結であった。資本主義経済では何をどれくらい生産する、そしてそれらの商品の価格を決めるのは個々の資本家（企業）である。経済活動の結果は無数の経営者の決断の結果である。社会主義の移行経済と言っても 、 197 0 年か ら 197 3 年のチリ経済は資本主義経済制度に基づいていた。基本的に民間企業の経営者は利潤率を減らしたくない。経営者は物価の凍結と賃上げの組み合わせの利潤率への影響を避けるために、生産されたものをやみ市で売ることにした。政府はやみ市場を無くしたかったが、成功の有無はやみ市の規模によることになる。映画「マチュカ」の舞台であ る 197 3 年のチリでは、物不足とやみ市の姿が描かれている。やみ市と物不足 は 197 1 後半から存在していた。  

 　 197 1 年後半にすでにチリ社会は二極化していた。「マチュカ」で描かれるアジェンデ政権を指示する社会党、共産党、労働者、貧困層などのデモと、政権を反対する右翼の政党や裕福な市民のデモが続いていた。中流階級は分かれていて、同じ家庭内でも若者はアジェンデ政権を指示し、両親は保守派を指示することは少なくなく、政権後半になると政府の反対派を指示する人が多くなる。もちろん、現実はより複雑であり、上流階級の若者は左派組織に参加し、貧困層の内アジェンデ政権に反対する人もいた。社会全体の政治意識はより一層高まり、特に裕福層の内普段には政治に無関心な人の一部がデモに参加することもあり、映画「マチュカ」ではゴンサロの母親がその一例である。  

 ゴンサロの姉の恋人は 、 197 1 年か ら 197 3 年の間に存在した「祖国と自由」という組織のメンバーである（反政府デモでは彼のヘルメットに「祖国と自由」の印が描かれている）。彼とマチュカが初めて会うシーンでは、彼は貧困層に対して侮辱の思いをあらわにする。「祖国と自由」は反政府右翼組織であり、デモだけではなく、都市ゲリラ活動も行っていた。「祖国と自由」は、海軍総司令官アルトゥロ・アラジャや左翼組織のメンバーを暗殺し、ラジオ局、左派政党本部、石油パイプ、キューバとソ連の大使館などを攻撃した。  

 　 197 1 年の年末にフィデル・カストロはチリ に3 週間滞在した。カストロはチリの細長い地形のほとんどの地方を訪ねた。カストロにとってはキューバ革命以来、初めて社会主義革命の道を歩んでいたチリは非常に重要な存在だった。なぜならその試みの結果が、ラテンアメリカにおける冷戦の行方を左右することを知っていた。カストロは首都サンティアゴの郊外で行われていた反政府デモを見に行った。デモでは主に裕福層の女性達が参加していた。  

 キューバ革命はゲリラ戦の結果であったため、ゲリラ戦が成功してから裕福層を守る軍隊はすでに存在しなかった。チリのアジェンデ政権はキューバと異なり、世界で初めて代表民主制の下で社会主義革命を行おうとする政権であった。反政府の権力者はマスメディア、大地主、大企業や多国籍企業を中心として製造業、運送業、軍隊などあらゆる経済社会部門に存在していた。カストロは、平和的に社会主義への移行は成功できないと判断した。アジェンデは全てのチリ人が最終的に代表民主主義を守ると思っていた。この異なった考え方は「モーターサイクル・ダイアリーズ」においてマチュピチュのシーンで描かれている。グラナドは先住民の政党を成立し、総選挙を通じて政権を握り、革命を行おうと提案をした時 、 2 3 歳のゲバラは武器なしの革命は無理だと答えた。アジェンデ政権の行方はラテンアメリカにおける冷戦の行方だけではなく、ラテンアメリカを越えて、資本主義諸国では平和的に社会主義へ移行できるかどうかに関して主要な試みであった。  

    

 　映画「マチュカ」では 、 197 3年3 月の学年の始まりから同年 の9 月のクーデターと、その直後の軍隊弾圧の始まりまでを物語る。映画は裕福層と貧困層が最も激しい対立を見せていた時期で、裕福層の子供ゴンサロ・インファンテと、貧困層の子供ペドロ・マチュカとの出会いと友情を描き、それぞれの家族と関係者の生活、両サイドのデモなどを見せる。子供達が友人になるきっかけは学校の校長先生であった神父の教育プログラムである。学校と教育プログラムを運営した校長先生は別名で当時サンティアゴ市東南郊外にあるペニャロレン市に存在していた。エンディングで記録されているように映画「マチュカ」はその学校の校長先生へのオマージュである。実際に存在する学校名はセイント・ジョルジュであり、校長先生はヘラルド・ウエラン神父であった。  

 ウエラン神父 は 192 7 年にアメリカのデトロイト市に生まれ、カナダで司祭に叙階され た 195 5 年から享 年 200 3 年の間に、チリで教育に務めた 。 195 5 年からエリート学校であるセイント・ジョルジュ学校に所属し 、 196 5 年に他の教員や文部省の職員とともに、「実験的プログラム 」 (programa experimental ) を設計した。ウエラン神父は子供達が人間らしく、また、キリスト教信者やチリ人としてより良い社会に貢献できるように柔軟性のある教育プログラムを目差した。生徒はチリの現実を知るべきだと思い、そのためには知識だけではなく、生徒の価値観や性格の育成も主要であると考えていた。  

 アジェンデ政権が誕生す る6 年前 、 196 4 年にセイント・ジョルジュ学校で奨学金制度が導入された。この制度は、学校から近いマポチョ川の向こう側に広がるスラム街に暮らす子供達に対して、セイント・ジョルジュ学校に入学できように奨学金を給付することだった。裕福層と貧困層の子供達は学校が所有する農場で働いて、そこから得た収入はスラム街の子供の奨学金を償うために使われていた。教室や農場の活動を通じて異なる階層の子供はお互いに知ることになり、チリ社会の現実を自分の目で確認することになる。そのことは子供の価値観や性格を育成し、将来的に責任のある立場にいる時に役に立つと神父は思っていた。そして最も重要なことは、互いに知り尊重し合うことによって偏見を無くすことだった。プログラムを促進していた先生達は階級や階層間の偏見の主な原因は交流が足りないことであると考えていた。他階級の子供が尊重し合って、社会統合の萌芽を育てようとした。  

 ウエラン神父 は 196 9 年にセイント・ジョルジュ学校の校長先生になった。彼は各クラスにスラム街から来る子供が少なくと も3 人はいるようにした。映画「マチュカ」で紹介されるように裕福層の子供達の両親の一部はウエラン神父の教授法に反対した。  

 　「マチュカ」の監督アンドレス・ウッドはセイント・ジョルジュ学校で勉強した。「生まれるための三年間 」 (Tres años para nacer ) と言う本を書いた著者アマンテ・パラゲスも、またセイント・ジョルジュ学校で勉強した。パラゲスはその本で、子供の観点か ら 197 3 年のクーデターの体験を記述した。彼こそが、ウッド監督の映画で描かれているマチュカなのである。  

    

 


 

⑧　ミッシング

監督：コスタ＝ガヴラス

出演：ジャック・レモン、シシー・スペイセク、ジョン・シーア

Missing (1982) de Costa-Gavras

    

 　映画「マチュカ」 は 197 3年9月 1 1 日に勃発したクーデター直後のスラム街市民抑圧で終わる。映画「ミッシング」はクーデター直前か ら 1 0 月半ば頃までに至る。「ミッシング」 は 3 0 歳アメリカ人記者であったチャールズ・ホーマンの誘拐と、妻と父親によるチャールズの捜索についての記録である。映画はチャールズの活動と彼が行方不明者になってからの、親戚による彼の捜索を通して、チリで起きたクーデターの準備と実行に関わるアメリカ政府の密接な関係を物語る。「ミッシング」 は 197 8 年に出版されたトーマス・ハウザー著「チャールズ・ホーマンの処刑」に基づいていて、メキシコで撮影された。  

 　チャールズはアメリカ情報源 （ FI N ）と言うニュース・エージェンシーの共同設立者であり、「チリ映画」社 （ Chile Film s ）のために脚本を書いていた 。 FI N はアメリカに関する情報をスペイン語でチリ人に紹介し、ベトナム戦争中だったアメリカに関する反戦運動など、アメリカの主なマスメディアが十分に取り上げていなかった情報を提供していた 。 FI N は、アジェンデ政権下のチリ人に、アメリカでも進歩的な考えを持ち冷戦時代で反政府運動を行っている人々の活動を紹介し、チリとアメリカの関係に関するニュースも取り上げていた。チリに渡る以前チャールズはハーバード大学を卒業して、ザ・ネーションとクリスチャン・サイエンス・モニターのために記事を書いていた。  

 　チャールズは「アメリカス大通り」と言 う 8 5 分のドキュメンタリー映画の脚本を書いた。これは彼の最後の仕事であり、チャールズの処刑か ら2 年 後 197 5 年にはペルー人であるホルヘ・レジェス監督によって完成作品が上映された。ドキュメンタリーはアジェンデ大統領が使った表現を引用して、チリを「静かなベトナム」として紹介した。「静かなベトナム」の意味はアメリカ政府がチリに対して暴力よりも「経済的な戦争」を行っていたということである。チャールズが知らない内にアメリカ政府の情報局は、彼がアメリカとチリで行った活動を「国家を転覆させる活動」として扱っていた。  

 　映画「ミッシング」の後半にあるシーンでは、クーデター直後のビニャ・デル・マルでアメリカ軍大佐がチャールズと友人テリーを無線機がある家に連れて行く。そこで大佐は「チリの軍隊はうまくやっているが、チリの本当のヒーローはトラック・オーナーである」と言う。トラック・オーナー は 197 2年 1 0 月にロックアウトを起こして、全国の流通を麻痺状態にさせた。経済の他のセクターはロックアウトに加え、翌年アジェンデ政権を倒したクーデターの成功を可能にするための決定的な出来事だった。アメリカ政府 は CI A を通じてこのロックアウトを経済的に支えた。これは「静かなベトナム」の最も目立った事例である。  

 　チャールズとテリーはクーデターが勃発した日に偶然に観光地であるビニャ・デル・マルに滞在していた。太平洋海岸にあるビニャ・デル・マルや隣にあるバルパライソ市にはクーデターを起こしたチリ海軍やアメリカのミルグループ （ Milgroup=Military Grou p ）の本部があった。チャールズとテリーはアメリカ軍幹部が滞在していたミラマル・ホテル に9月 1 0 日か ら 1 5 日まで宿泊した。　  

 　以上引用した会話では大佐が同 年7 月に海軍大将と一緒にアメリカでチリ軍のための武器を購入したと言う。チャールズはアメリカ人であったから、そしておそらくクーデターは順調だったため、大佐はチャールズに主要な情報を伝える。チャールズはその情報を手帳に記入する。大佐が無線機のある家に車で向かっていたシーンでは 、 196 1年4 月 に CI A が計画したキューバ侵略（ピッグス湾侵略）（ 第2 章「低開発の記憶」参照）で自分も参加したことをチャールズに伝える。彼によると、ケネディ大統領がより協力的だった場合、キューバ革命政権を倒すことが成功しただろうと言う 。 196 1 年にピッグス湾侵略が成功していたら、チリで何の問題も無かっただろうと言う。この断言ではアメリカの右翼の思想はまとめられている。承知のよう に 196 3 年にケネディ大統領は暗殺された。  

 　映画のオープニング・シーンではタワー主将はチャールズとテリーをビニャ・デル・マルから首都サンティアゴに連れて行く。主将はチャールズの家まで連れて行こうとしたが、チールズは住んでいる場所を知らせたくなかった。映画でタワー主将として紹介される人の本名はライ・デイビッドであり、アメリカ海軍の主将であった。彼はアメリカ軍とクーデターを起こしたチリ軍の間の連絡員長であった 。 201 1年 1 1 月にチリのセペダ大臣は、上訴裁判所にデイビッドはチャールズとフランク・テルジの処刑の責任者であると伝え、最高裁判所にデイビッドの身柄をアメリカから引き渡すよう要求した （ La Segund a、 201 1年 1 1月 2 9 日）。  

 　クーデターを通じてアメリカ軍とチリ軍の戦略は、「静かなベトナム戦略」からベトナムのような「消滅戦略」に変わった。代表民主主義下で社会主義経済に移行しようとしたアジェンデ政権は最後まで暴力を避けた。チリの試みはキューバ革命と対照的であり、チリ社会党の共同設立者であるアジェンデは命を落とす最後まで民主制度と国民の意思を信じた。しかし、映画を観ると分かるように、軍隊の暴力は想像力を超えるもので、まさに消滅戦略であった。それに対して、いくらかの国民の意思があっても、恐怖は支配的になり、この戦略は「国家テロ」として知られている。現実は映画で観察できるものより遥かに残酷的であった。  

 　ピノチェト独裁者は、地方の軍部により逮捕されていたアジェンデ政権の幹部に対する「甘い扱い」に不満があった。直ちに、ヘリコプターで全国を渡るためのグループを形成し 、9月 3 0 日に出発した。このグループは「死のキャラバン」として知られている。全国の軍事基地でそれぞれの基地の司令官の意見を無視しながらおよ そ 10 0 人を抹殺した。当時アントファガスタ陸軍基地の司令官であったホアキン・ラゴスは（エル・パイス 、 200 1年1月 2 8 日参照 ） 200 1 年にチリのテレビで「逮捕された人々の残酷的な殺し方とピノチェトの責任」について話した。彼の供述はピノチェトの裁判につながった。  

 　ピノチェト独裁政権の抑圧は海外まで至った。カルロス・プラツとオルランド・レテリエルの暗殺は代表的である。カルロス・プラツと彼の妻であったソフィア・カスバートは 、 197 4年9月 3 0 日にアルゼンチンの首都ブエノスアイレスで暗殺された。映画「マチュカ」の解説で記述したように、アジェンデは大統領にならないために 、 197 0年 1 0 月 に CI A の作戦によって陸軍最高司令官レネ・シュネイダーが暗殺された。当時のフレイ大統領は、シュネイダーのポストに民主主義を守る思想があったカルロス・プラツを任命する。アジェンデが大統領になってからも陸軍最高司令官としてプラツを再任する。プラツはクーデターが実行されないように政治的な解決を探し求めた。クーデターが勃発すると、彼の命が危険にさらされたため 、9月 1 5 日にブエノスアイレスに亡命した。ブエノスアイレスでもチリの秘密情報 局 DIN A とアメリカ の CI A はプラツの生活に関する細かい情報を持っていた。暗殺され た 197 4 年に、プラツはチンコッタ社のパブリック・リレーションズ部長として働いていた。  

 オルランド・レテリエル は 197 6年9月 2 1 日に、ワシント ン D.C . でチリの秘密情報局によって暗殺された。レテリエル は 197 1年1 月から駐米チリ大使であり 、 197 3 年には続けてアジェンデ政権の外務大臣、内務大臣と国防大臣であった。レテリエルは経済学者であり政治家であった 。 197 5 年からワシント ン D.C . にある政策研究所 （ Institute for Policy Studie s ）の研究者として勤めていた。プラツとレテリエルの暗殺を実行した人物 は CI Aと DIN A のエージェントである。  

    

 2 0 世紀の最も洗練された詩人の一人であるパブロ・ネルダの人生の最後三年間はアジェンデ政権と深く関わっている。アジェンデは裕福な家庭で生まれたが、ネルダは労働者階級の家庭で生まれた 。 190 8 年生まれアジェンデ は 193 3 年の設立から社会党員であり 、 190 4 年生まれネルダ は 194 5 年からチリ共産党員であった。大学生で教育学を勉強していたネルダ は 2 0 歳の若さで彼の有名な作品「二〇の愛の時と一つの切望の歌」を発表した。ネルダの存在は知られるようになり 、 2 3 歳から外交官としてのキャリアが始まる 。 194 8 年にチリ大統領はチリ共産党を不法化し、ネルダの政治活動は地下組織の活動に変わり、やがて翌年に亡命する 。 195 0 年にはメキシコでラテンアメリカ文化、地理と歴史や世界の階級闘争について論じる「おおいなる歌」を発表する。  

 　 196 9 年にチリ共産党はネルダを大統領候補として選ぶが、ネルダは大統領候補を辞任し、社会党のアジェンデ候補を指示する。社会党、共産党などは「国民統合」 （ Unidad Popula r ）という同盟を形成し、アジェンデが大統領になるとネルダは駐仏大使に任命される。アジェンデ政権の一年目が終わろうとしていた時 、 197 1 年にネルダは文学ノーベル賞を受賞する。その時点までチリ経済は順調であり、政権を代表していた詩人が文学ノーベル賞を受賞することは、アジェンデとネルダを指示していた人々にとって最高の贈り物になった 。 197 3年2 月に病気でネルダは駐仏大使を辞任し、その時からチリの黒島 （ Isla Negr a ）に暮らす。クーデター直後に首都サンティアゴにある彼のもう一つの家は軍人によって略奪され彼の本は燃やされた 。9月 1 9 日にネルダは緊急にサンティアゴの病院に運ばれたが 、 2 3 日に死亡する。未だに、病気で死亡したか、殺されたかの議論は続いている。  

 　チリ文学 と 197 3 年のクーデターにもう一つ重なる事例がある。ロベルト・ボラニョ は 195 3 年にサンティアゴに生まれ 、 196 8 年に彼の家族はメキシコに移住する 。 197 3 年 に 2 0 歳になったボラニョはバス、電車、船やヒッチハイクを利用して、アジェンデ政権の役に立ちたいという目的でチリに向かう。しかし、チリに着いたらクーデターが勃発し、アジェンデとネルダは死亡する。ボラニョは逮捕されたが、彼と他の人々を監視していた警察官の内、ボラニョの同級生がいたため、彼の仲介によって釈放された。その後、エル・サルバドルとメキシコに暮らしてからボラニョは母親が暮らしていたスペインのカタルーニャ地方に移住する 。 199 0 年代にボラニョは彼の代表作を執筆する。その内、短編小説集「通話」と小説「野生の探偵たち」は日本語に訳されている。小説 「 266 6 」がほぼ完成した頃に 、 200 3 年にバルセロナで死亡した。翌年に 「 266 6 」は出版され、ボラニョは世界中で知られるようになる。

アルゼンチンの軍事独裁政権と亡命

（ 197 0 年代後半）

    

    

    

    

 


 

⑨　瞳は静かに

監督：ダニエル・ブスタマンテ

出演：ノルマ・アレアンドロ、

コンラッド・バレンスエラ

Andrés no quiere dormir la siesta (2009)  

de Daniel Bustamante

    

 　アルゼンチン の 2 0 世紀の政治史において、フアン・ドミンゴ・ペロンは一番大きな影響力をもった歴史上の人物である。 第2 次大戦直後、資本主義諸国の大半では、政治的な民主化と労働運動の制度化の拡大が目立った。日本とアルゼンチンでは、歴史的な背景がずいぶん異なるにも関わらず、その点では同じような事実を確認できる。しかし、日本とアルゼンチンの間では対照的な出来事があった。日本では農地改革が実行されたが、アルゼンチンでは軍部を通して支配していた大地主の権力がそのまま続いた。  

 　アルゼンチンでは、総選挙で選ばれた民主政権に対しての最初の軍事クーデターが 、 193 0年9 月に起こった 。 3 5 歳の陸軍軍人であったペロンは、積極的にこのクーデターに協力した 。 188 0 年代か ら 192 0 年代の間、ヨーロッパからの移住者は非常に多く、豊かな国だったアルゼンチンで初めて中流階級を代表する政権が誕生したが、大地主、対外貿易や金融機関を代表する上流階級はその状況を許さず、歴史上初めてのクーデターが勃発した。その後クーデターは繰り返され、最後であり、また、一番血を流したクーデターは 、 197 6年3月 2 4 日に勃発した。映画「タンゴ　ガルデルの亡命」は、このクーデターの影響で、亡命する以外に選択肢がなかった人々の内の何人かの、パリで過ごす亡命生活について物語る。映画はドキュメンタリー映画ではなく、幾つかのジャンルを含んだ、主にミュージカル的な映画である。映画の内容について論じる前に 、 197 6 年のクーデターを導いた主な出来事を記述したい。  

 　ペロン は 194 6 年 と 195 5 年の間で 、 194 6 年 と 195 2 年の二つの総選挙によって選ばれたアルゼンチンの大統領であった 。 195 5 年の軍事クーデターによって、ペロン政権は倒され、ペロンはフランコ軍事独裁者が支配するスペインに亡命する。ペロン政権は強い労働組合、所得の再分配と労働組合を通じて、労働者のための福祉制度の形成という特徴で知られている。しかし、政府は労働組合を通じて労働者の行動をコントロールしていて、政府が認めた労働組合以外の組織の中で、労働者の不満が生じると認めようとしない 。 195 0 年以降のストライキに対する政府の権威主義的な対度は、その立場を明確に表している。ペロンはこのような国家コーポラティズムを通じて輸入代替工業化（輸入されていた製造業の製品を国内で生産すること）を実現しようとした。  

 　ペロンがスペインに亡命してからのアルゼンチン国内では、禁止されていた労働組合を中心に社会運動の輪が徐々に広がった。この運動の目的はペロンが帰国を許され、大統領候補になる権利を与えるようにすることだった。ペロンを支持する社会運動や政治家の内、最初の段階から右翼と左翼は共存していたが、ペロンの近くにいた人物には、右翼の代表が多かった。それと同時に、ペロンが帰国できるように、左派の社会運動を利用した 。 197 0 年になると左派のペロニスタ社会運動の一部はキューバ革命とゲバラの影響を得て、ゲリラ活動を開始し、ペロンはこのゲリラ活動を支持した。これ以外にも、マルクス系のゲリラ組織が存在していた 。 197 3年3 月に行われた総選挙では、ペロン運動の左派エクトル・カンポラが政権を握る。カンポラはすぐにペロンが大統領になれるように再び総選挙を呼びかける。やがて 、 197 3年 1 0 月にペロンは再びアルゼンチンの大統領になる。  

 　翌年 の7月1 日にペロンは死亡した。その時のペロン政権では、ペロニスタ運動の右派が権力を握っていた。ペロン政権の社会福祉大臣ホセ・ロペス・レガ は 197 3 年に「アルゼンチン反共産同盟」 （ AA A＝ Alianza Anticomunista Argentin a ）という地下組織を形成した。この右翼の組織 は 197 3 年か ら 197 6 年の間、多くのインテリ、芸術家、大学生、労働者などであった左派の活動家を暗殺した 。 AA A の活動は冷戦において 第2 章、 第4 章と 第5 章で取り上げたキューバ、ボリビア、ブラジルとチリの出来事と重なる共通点が多い。しかし、アルゼンチンでペロニスタ運動の歴史は長く、一般市民への影響力も膨大だった。ペロンスタ運動に参加した一般市民のほとんどは運動の左派を支持していた。政府は最初 に AA A を通じて、そし て 197 6年3月 2 4 日から軍事独裁政権の地下組織を通じて、左派運動に対してチリと同じような「消滅戦略」を実行し、その規模はチリを遥かに上回る。  

 　軍事独裁政権の下で軍隊の地下組織は、全国で毎晩のように左翼の活動家や彼らの関係者を誘拐し、拷問を通じて関係者の行方を探り、その情報を利用して関係者たちをも誘拐する手法をとっていた。チリのピノチェト政権と比べると抑圧過程は時間的に長く、組織的にはより秘密的で分散化したものであった。誘拐された人々の大半は行方不明者になった。正確なデーターは存在しないが、アルゼンチン で 197 4 年か ら 197 9 年の間に、暗殺された人々と行方不明者になった人々を合わせると、約３万人と推測されている。  

 　映画「瞳は静かに」（スペイン語タイトル ： Andrés no quiere dormir la siesta   アンドレスは昼寝をしたくない）の舞台 は 197 7 年のサンタフェ市（当時の人口はおよ そ 3 0 万人）である。当時８歳だったアンドレスのお母さんは道路を横断する時に事故で死亡する。しかし、事故は計画されたもので、その原因はお母さんと左翼の活動家の関係にあった。アンドレスは、お父さんと共に、お祖母さん（父の母）の家に引越す。お父さんは神経質な人で、アンドレスに対して権威主義的な態度を示す。お祖母さんは家族と近所のすべての出来事を把握していると自慢し、保守的な考え方を持っている。  

 　アンドレスの新しい住まいのすぐ近くには空き地があり、子供達はよくそこで遊んでいた。空き地の横にはアルゼンチン軍隊の地下組織が「秘密逮捕センター」として利用している家があった。毎晩、誘拐された左翼の活動家や彼らの関係者はその家で拷問を受けていた。近所の人たちは地下組織の活動を知っていたが、犠牲者を助けようとしない。国家テロに対する 恐怖 と暴力の支配のもとで強化された 個人主義 が合わせたかたちで、特に中産階級のうち、不法な逮捕と拷問に対する見知らぬふりをしていた人が圧倒的に多かった。  

 　昼の間、「秘密逮捕センター」で逮捕された人に拷問を行う国家準軍隊組織は子供たちに近づく。その準軍隊組織のひとりのセバスティアンは、アンドレスと関わるようになる。セバスティアンはアンドレスの亡くなったお母さんの恋人のアルフレッドに関する情報を求める。映画ではノラ（アンドレスのお母さん）とアルフレッドの関係が描かれている。アルフレッドはデモで使うチラシを一時的にノラの家で隠してほしいとお願いする。映画の中で、これらのチラシは恐怖の象徴になっている。お母さんが死亡した後 、 197 7 年の間に、アンドレスとお父さんやお祖母さんとの関係、学校の出来事、セバスティアンとの関係、友達との関係等を通じて、アンドレスの性格が徐々に変わっていく様子が描かれている。性格の変化の中で一番目立つことは、残刻な行動に対する無関心と無感動である。やがて、その無感動はお祖母さんの死に対してもあらわれる。  

 　映画「瞳は静かに」の主な貢献は、恐怖が支配的になっていたアルゼンチン の 197 7 年で、多くの市民、特に中産階級の人々の行動を明らかにすることである。日常生活の多様な場面で個人主義は支配的になる。多くの人々の考え方と行動が非人間的になる。これはゲバラと当時アルゼンチンの左翼の活動家が目指していた「新しい人間」とは正反対の方向である。社会は以前より二極化に陥る。自分の行動が連帯や思いやりに基づく人々が少数派になる。  

 


 

⑩　タンゴ　ガルデルの亡命

監督：フェルナンド・ソラナス

出演：マリー・ラフォレ、

フィリップ・レオタール、

ミゲル・アンヘル・ソラ、マリナ・ヴラディ

El Exilio de Gardel, Tangos (1985)  

de Fernando Solanas

    

 　チリでも、アルゼンチンでも 、 197 0 年代の軍事独裁政権の下で危険にさらされていた人々は可能な限り海外に亡命した。多くの亡命者はメキシコと西ヨーロッパ諸国に渡った。「タンゴ　ガルデルの亡命」はパリに亡命したアルゼンチン人何人かの生活を見せている。この映画の監督であるフェルナンド・ソラナスは当時パリで暮らしていた亡命者であった。  

 　映画はストーリーとして亡命者の生活を見せるよりも、亡命生活の精神的な諸側面をコラージュのように紹介して行く。話の筋道になるのが彼らの作曲しているタンゴのダンス・ミュージカルであり、その名は「タンゲディア」である。「タンゲディア」 （ tanguedi a ）はタンゴ （ tang o ）＋悲劇 （ tragedi a ）、あるいはタンゴ＋コメディ （ comedi a ）を意味する。このように「タンゴ　ガルデルの亡命」は、悲劇とコメディがうまく絡み合っている。タンゴはアルゼンチンの首都ブエノスアイレスを代表する音楽であり、そのノスタルジックや情熱的な特徴は、亡命者の精神状態を表すために最適な音楽であるだろう。  

 　タンゴ は 1 9 世紀後半のブエノスアイレスの下町で誕生し、ヨーロッパからの移住者がブエノスアイレス人口の半分以上を占めてい た 192 0 年代に、カルロス・ガルデルが作曲したものや、彼が歌った曲とともに世界で知られるようになった。ガルデルはフランスのトゥールーズ市、もしくはウルグアイのタクアレンボ市のどちらで生まれたかは未だに明らかになっていない。しかし、少なくと も 1 0 歳頃からブエノスアイレスに暮らしていたことは確かである。彼の母親はヨーロッパからの移住者の一人である。スペイン語を話せない移住者が多かったブエノスアイレスの下町では、ルンファルドと言うイタリア語の影響が強い俗語が誕生した。タンゴの歌詞ではルンファルドの単語がよく使われる。  

 　タンゴの歴史は三つの時代で分けられる。「タンゴ　ガルデルの亡命」では、三つの時代とも引用されている。 第1 は 、 192 0 年代 〜 193 0 年代前半のガルデルの時代であり、タンゴを演奏するための主な楽器はギターであった。 第2 は 、 194 0 年代 〜 195 0 年代のタンゴの黄金時代であり、バンドネオンが主な楽器になり、タンゴはオーケストラによって演奏される。「タンゴ　ガルデルの亡命」では 、 198 0 年の正月パーティーで、ピアニストと作曲家オスバルド・プグリエセのオーケストラがこの時代のタンゴを演奏する。 第3 は 、 196 0 年代の新タンゴと呼ばれるもので、この時代の代表はアストル・ピアソラである。彼は「タンゴ　ガルデルの亡命」のサウンドトラックを作曲した。  

 　黄金時代のタンゴとともに育った世代は、ピアソラの音楽がタンゴではないと訴えたため、ピアソラは自分の音楽を「ブエノスアイレス市の現代ポピュラー音楽」と呼んでいた。ピアソラはアメリカでクラシック音楽の教育を受け、タンゴ を 196 0 年代のブエノスアイレスを反映するアヴァンギャルドな音楽に変貌させた 。 197 0 年代に亡命したアルゼンチンの若者達はピアソラの音楽と誕生したばかりアルゼンチン・ロックとともに青春時代を過ごした。ピアソラは作曲家とバンドネオン演奏家であったが 、 199 2 年に死亡した。現在、作曲家とバンドネオン演奏家としてアルゼンチンの現代音楽を代表する最も洗練されたアーティストはディノ・サルツィである。彼の音楽には、アルゼンチンタンゴとフォルクローレ、クラシック音楽と現代ジャズの影響がある。  

 アルゼンチンではカルロス・ガルデルが国民的の英雄であり、彼の歌声は、人々のアルゼンチン人としてのアイデンティティを、より深いものにさせる。ガルデルは自分のキャリアのピークともいえ る 193 5 年、飛行機の事故で死亡した。ガルデルの存在は伝説的になり、「タンゴ　ガルデルの亡命」は、亡命者の故郷に対する懐かしさの象徴ともいえる。この映画ではガルデルが「パリに繋がれて」 （ Anclado en Parí s ）と「帰る」 （ Volve r ）を歌うが、両方とも亡命者の状況と気持ちをよく表した歌詞である。映画ではフアン・ドスとマリアナがいる公衆電話ボックスに近づいてきて、その後ガルデルを連れ去る人物がいるが、それはエンリケ・サントス・ディセポロ （ 190 1 年 〜 195 1 年）である。彼は最も有名なタンゴ作曲家の一人である。ディセポロは、多くのアルゼンチン人が最近まで暗記で覚えていた、「ウノ」と「カンバラーチェ」というタンゴの歌詞を書いた。それらのタンゴでは美しい詩を通じて 、 192 0 年代のブエノスアイレス下町の人々の苦しみを伝えている。だが、これらは二曲とも 、 197 6 に誕生した軍事独裁政権によって禁止される。  

 「タンゴ　ガルデルの亡命」で は 198 0 年の正月パーティーが描かれる。そのパーティーで、最初にタンゴを演奏するのは、オスバルド・プグリエセ （ 190 5 年 〜 199 5 年）率いるオーケストラである。プグリエセはピアニスタとマエストロであり 、 193 6 年に自分のオーケストラを形成した。プグリエセは政治的な意識が強く 、 193 5 年に創立された音楽家の労働組合に加入し、翌年からはアルゼンチン共産党員になる。彼のイデオロギーと政治活動によって、ペロン政権下でも 、 195 5 年に誕生した軍事独裁政権下でも、逮捕されたことがある。「タンゴ　ガルデルの亡命」でソラナス監督 は 8 0 歳に近づいていたマエストロ・プグリエセに敬意を表した。この映画が初演され た 198 5 年には、プグリエセ は 8 0 歳になり、誕生日祝いとして彼のオーケストラはコロン劇場で演奏した。  

 　「タンゴ　ガルデルの亡命」では、アルゼンチンとチリとペルーの独立運動のリーダーと、三カ国の独立の英雄であるホセ・デ・サン・マルティン （ 177 8 年 〜 185 0 年）も紹介される。ヘラルドはマリアナと一緒にブーローニュ・シュル・メールに向かい、サン・マルティンが人生の最後 の2 年間半を過ごした家を訪ねる。サン・マルティンは独立運動で成功したが、政治的な争いの結果としてヨーロッパに移住し、フランスを中心にヨーロッパの地で最後 の 2 6 年間を過ごす。アルゼンチンで自分の命が危険にさらされていたので、サン・マルティンも政治亡命者であったといえる。映画のエンディングが近づくと、サン・マルティンはガルデルとともにヘラルドの部屋に現れる。  

 　「タンゴ　ガルデルの亡命」は、音楽と歴史を通じてアルゼンチン人のアイデンティティというテーマを取り上げる。亡命者は外国に自分の歴史とアイデンティティを持って行く。その中では行方不明者が大きな役割を担う。亡命者の親戚、友人、同僚の内に、誰かしらの行方不明者が頻繁にいた。亡命生活の中では行方不明者の存在が遍在する。「タンゴ　ガルデルの亡命」では、行方不明者の精神的な存在が、あちこちに出てくる人形によって表現されている。映画ではパリで行われたラテンアメリカ諸国の軍事独裁政権の下で、行方不明者になった人々に関する情報を要求するためのデモが上映され、人形たちは陸軍のトラックに乗っている。「行方不明者に関する国家委員会報告書」によると行方不明者の内 、 3 5 歳未満の学生と労働者が圧倒的に多い。  

 　「タンゴ　ガルデルの亡命」では「タンゲディア」の構成がコラージュのようなものであり、フアン・ドスとピエールによるコラージュのように芸術に関する思想と、一般的な世界観を現す興味深い発言が用いられている。幾つかの例を引用しよう：「自分になると言う決断はいつもリスクを伴う」、「リスクがないと、創造は無理である」、「知的な思索ではなく、想像的に生きることのファンタジーを要求しよう」、「快適さと虚栄心に対して注意すべき」、「美しさに利達するために、醜さを危険にさらすべき」「美学のルールを窓から投げましょう、ジャンルを混合しよう」、「徹底したものは死んでいる、未完成万歳、生きること万歳」、「年を取ることはやりがいのないと言うことである」。最後に亡命者の苦しみと故郷の状況を描く肯定：「我々が後にした国はすでに存在しない」。  

 　 198 3 年からアルゼンチンの民主化とともに多くの亡命者は帰国し、ソラナス監督もその一人であり、アルゼンチンで「スール／その先 は ... ... 愛」 （ Sur, 198 8 年）を製作する。この映画の主人公フロレアルは軍事独裁政権が崩壊するまで逮捕されていた。フロレアルは釈放されると、妻と息子の家に行こうとしたが、夜中に家に近づくと入らなくて、ブエノスアイレスを歩き回る。その夜にまるで夢の中にいるかのように、逮捕されている間の友人、家族と国民の苦しみが表れてくる。ピアソラの音楽とロベルト・ゴジェネチェが歌う黄金時代のタンゴが、ブエノスアイレス市南部の風景と、フロレアルのストーリーが美しく絡み合う。「スール」は一番知られているタンゴ局の一つのタイトルであり、ブエノスアイレス市南部を指す。その地区は工場が多い下町であり、タンゴが花開いた地区でもある。「スール」はスペイン語で南を意味しながら、ブエノスアイレス市南部だけではなく、南米の南部にあるアルゼンチンと世界の「南部」として発展途上国の意味も含む。  

 　ルイス・プエンソ監督 の 198 5 年の作品である「オフィシャル・ストーリー」は 、 197 6 年か ら 198 3 年の軍事独裁政権に関するもうひとつの有名な映画である。この映画 は 198 6 年にアカデミー外国語映画賞を受賞した。残念ながら日本で は DV D 化されていない。オフィシャル・ストーリーの意味は歴史の教科書で紹介される歴史であり、場合によって現実から外れて権力者の好みで作られたものであることを示唆する。主人公は母と娘であり、母親は高等学校のアルゼンチン史の教員であり、父親は軍人である。娘は養子に迎い入れた子供であるが、映画が進むにつれ、徐々に行方不明者の娘であることが明らかになっていく。軍事独裁政権の下で、行方不明者の中には妊娠の女性が多くいた。彼女達は子供を産んだ後暗殺された。それらの新生児は子供を欲しかった軍人や、警察官やその関係者に渡された。行方不明者の母親達の組織「五月広場の祖母達」 （ Abuelas de Plaza de May o ）はその子供達を今も探している 。 198 0 年か ら 201 4年8 月現在まで、行方不明者の息子・娘 達 11 5 人の身元が明らかになり、本当の家族の存在を知ることができた。  

 


 

    

    

    

    

    

第6 章

中米諸国における内戦：

エルサルバドルの事例

（ 198 0 年代前半）

 


 

⑪　サルバドル

監督：オリバー・ストーン

出演：ジェームズ・ウッズ、ジム・ベルーシ、

ジョン・サベージ

Salvador (1986) de Oliver Stone

    

⑫　イノセント・ボイス

監督：ルイス・マンドキ

出演：カルロス・パディジャ、

レオノル・ヴァレラ

Voces Inocentes (2004) de Luis Mandoki

    

 　映画「サルバドル」は、中米エルサルバドル で 198 0 年か ら 198 1 年の間に 、 CI A の協力を得たエルサルバドル政府による市民の抑圧を描く。映画の主人公はアメリカ人記者リチャード・ボイルであり、脚本はボイルが内戦中の経験に関して書いた本に基づいている。映画で描かれる出来事は中米で起きた他の出来事と関連しているので、以下グアテマラ、ニカラグアとエルサルバドルで冷戦時代に起きたことを簡単にまとめる。  

 　本書の 第2 章で「チェ 　 2 8 歳の革命」を取り上げた時、ゲバラ が 195 4 年にグアテマラで過ごした数ヶ月について記述した。その時ゲバラは 、 CI A が率い た PBSUCCES S 作戦によるアルベンス政権の転覆を目の当たりにした。独裁政権下のグアテマラで は 196 0 年から反政府ゲリラ活動が始まり、グアテマラの陸軍と右翼の準軍隊組織との戦いは内戦にまで至った。グアテマラの内戦 は 196 0 年か ら 199 6 年ま で 3 6 年間続き、国連の「歴史解明委員会」によると、内戦によっ て 2 0 万人を超える人々が命を失い 、4万5 千人以上行方不明者が出ている。犠牲者の大多数は、内戦に参加していなかったマヤ系先住民である。人権侵害が一番ひどかった時期は、アメリカのロナルド・レーガン政権から経済的な支援や軍事的な指導を得た 、 198 2年3 月か ら 198 3年8 月のリオス・モンツ政権であった。「歴史解明委員会」の報告書によると、内戦中に起きた人権侵害 の 9 3 ％はグアテマラ政府によるものであった。  

 　リゴベルタ・メンチュウ は 198 2 年にグアテマラ人先住民として、自分の経験をベネズエラ人人類学者エリザベス・ブルゴスに物語った。翌年メンチュウの話の録音に基づいて書かれたブルゴス著「私の名はリゴベルタ・メンチュウ、私の政治意識はこのように誕生した」は、スペイン語とフランス語で出版され、日本語版は「私の名はリゴベルタ・メンチュウ―マヤ＝キチェ族インディオ女性の記録」と言うタイトル で 198 7 年に出版された。メンチュウは子供の頃から彼女の生活環境を述べながら自分の政治意識の誕生と政治活動について論じる。メンチュウ は 199 2 年にノーベル平和賞を受賞した。  

 　 199 2 年はコロンブスがアメリカ大陸に到着してか ら 50 0 年になった年である。スペインとポルトガルは現在の中南米を植民地にしたことで、先住民は自分の文化や社会秩序、蓄積した知識を失った。植民地建設者はマヤ、アステカ、インカなどの先住民文化を意識的に破壊し、先住民にキリスト教を中心にヨーロッパの文化を押しつけた。スペイン人とポルトガル人がヨーロッパから持ち込んだ病気や、金銀の鉱山などで先住民を非人間的な労働条件で働かせた結果、多くの先住民は若い内に死亡した。スペインとポルトガルを中心として、先住民が足りなかった時は、アフリカから奴隷として連れてきた労働者を使って働かせた。先住民の立場からみると 、 199 2 年 の 50 0 周年記念は祝うべきことではなかった。その時、メンチュウの社会正義のために、そして先住民の権利が尊重される上で他民族や他文化間の和解のための貢献が認められ、メンチュウは平和ノーベル賞を受賞した。  

 　グアテマラとエルサルバドルで起きた 、 198 0 年代前半の先住民を中心とした市民の大量虐殺の二つの要因は 、 197 9 年にニカラグアで起きたサンディニスタ革命の成功と翌年アメリカのレーガン政権の誕生にある。ニカラグアのサンディニスタ運動の歴史 は 192 0 年代にさかのぼる 。 192 6 年にアメリカ軍 は2 回目のニカラグア侵略を行う。アウグスト・セサル・サンディノはアメリカ軍と戦うためにゲリラ組織を形成する 。 193 2 年にアメリカ軍はニカラグアを去るが 、 193 0 年に組織した「国民警備隊」がゲリラを倒す課題を継ぐ。アメリカは「国民警備隊」を率いるためにアナスタシオ・ソモサを選ぶ。ソモサの組織 は 193 4 年にサンディノを暗殺する 。 193 6 年か ら 197 9 年までソモサ家は、軍事独裁政権を通じてニカラグアを支配する。暗殺されたサンディノは国民英雄になり 、 196 1 年にサンディニスタ民族解放運動 （ FSL N ）が設立される。  

 　 196 0 年代に は FSL N の模範であったサンディノの思想と行動に、キューバ革命の影響が加わる 。 197 0 年代半ば頃には新聞ラプレンサの社長ペドロ・チャモロや、他の経営者、キリスト教幹部の一部が、ソモサ独裁政権を民主化に向けて、これ以上の人権侵害をくい止めるよう、政治活動の自由化を要求した 。 197 8年1 月にソモサ独裁政権は、ペドロ・チャモロを暗殺する。その時から中流階級と経営者の間に、独裁政権への不満が一層高まる。アメリカ大統領ジミー・カーター （ 197 7 年 〜 198 1 年）はこの民主化運動を支持し、その 内 FSL N を支持する市民は都市部でも農村でも多くなる 。 197 8年2 月に は FSL N はニカラグア議会を占拠し、その結果として議会にいた政治家の釈放の代わりに逮捕されてい た FSL N のメンバーらが釈放される。ゲリラ戦は全国に広がり 、 197 9年7 月 に FSL N のリーダーと兵士は首都マナグアに入る。キューバ革命以来、ラテンアメリカでゲリラ活動による 第2 の革命「サンディニスタ革命」が誕生する。  

 　サンディニスタ革命はアメリカの中米政策に大きなダメージを与え、アメリカ政府はその裏にはソ連の背景があると判断した。ニカラグアでサンディニスタ革命が成功してから、エルサルバドルとグアテマラでは、人権上の市民の安全性が悪化した。映画「サルバドル」で描かれる出来事はサンディニスタ革命の翌 年 198 0 年に始まる。エルサルバドルの内戦 は 198 0 年か ら 199 2 年まで続いた。その間ニカラグア革命政権は、レーガン政権 と CI A 率いた作戦により、日常的にホンジュラスからゲリラ戦の攻撃を受けるようになる 。 197 0 年代にグアテマラ、エルサルバドルとニカラグアでは①政治的な自由の欠如、②極めて激しい貧富の差、③冷戦による資本主義諸国と共産主義諸国間の緊張の高まりが重なった 。 198 0 年代はレーガン政権 （ 198 1 年 〜 198 9 年）の年代であり、グアテマラ、エルサルバドルとニカラグアの人々にとって恐怖の時代であった。  

 　 197 0 年代にエルサルバドルでは左翼の政党とゲリラ組織が次々に誕生した 。 198 0年 1 0 月に左翼のゲリラ組 織5 つが統合し、「ファラブンド・マルティ民族解放運動」 （ FML N ）を形成する。ファラブンド・マルティ は 192 5 年に中米共産党の設立に参加し 、 193 0 年に設立されたエルサルバドル共産党の幹部であった。一時的にニカラグア国民英雄であるアウグスト・サンディノの秘書でもあった 。 193 1 年か ら 194 4 年までエルサルバドルはマクシミリアノ・エルナンデスの軍事独裁政権の下にあった 。 193 2 年 に 3 8 歳だったマルティはエルサルバドル軍に処刑された。その時からマルティはエルサルバドルの左翼組織によって軍事独裁政権に抵抗した模範的な人物として扱われ 、 198 0 年には多様な左翼組織の統合によって誕生し た FML N は、ニカラグア の FSL N と同じように 、 193 0 年代の独裁政権の犠牲者になったリーダーの名を組織名に付けた。  

 　 FML N が設立された時点で、すでにエルサルバドル政治は二極化していて、その状況が確立するのに大きな影響を与えた出来事は 、 198 0年3月 2 4 日のオスカル・アルヌルフォ・ロメロ大司教の暗殺である。ロメロ大司教を暗殺する者が極右派軍部の会議で決定され、教会でミサの最中に大司教が暗殺される場面は、映画「サルバドル」で描かれている。ロメロ大司教は人権侵害を証言し、貧困層の人々の立場から政治的な暴力を訴えていた。大司教は貧困層の英雄だが、彼の暗殺は暴力の悪循環の拡大に大きく貢献した。ロメロ大司教の像はロンドンのウェストミンスター寺院（イギリスの王様と女王の戴冠式が行われる寺院）にある 「 2 0 世紀の殉教 者 1 0 人の像」の内の１つである。ロメロ大司教は「解放の神学」というカトリック教会の運動に参加していた。この運動によると神父達の課題は人間と神様を結ぶだけではなく、この世で人間同士の助け合いによるより良い社会を作ることである。「解放の神学」にとって平和的に社会正義の建設は、カトリック教会のもう一つの使命である。  

 　映画「サルバドル」では、ロメロ大司教の暗殺が決定された軍部の会議を率いり、手元に弾を持っていた実際の人物は、ロベルト・ダブイソンである。ダブイソンは当時、パナマのアメリカ領で運河ゾーンにあった「アメリカ陸軍米州学校」 （ SO A ）で勉強 し 197 2 年に卒業した。市民の拷問と暗殺に関わっている多くのラテンアメリカ諸国の軍人は、この学校で教育を受けたので 、 SO A（ School of the America s ）皮肉的に暗殺者学校 （ School of Assassin s ）と呼ばれることもある 。 200 1 年にアメリカのジョージア州に移転し、機関名は「西洋球安全保障協力研究所」に変わった。  

 　ダブイソン は 197 0 年代末頃、陸軍少佐たった時エルサルバドルの「暗殺隊」 （ Escuadrones de la Muert e ）を組織し、それ以降暗殺隊を率いた。暗殺隊は軍人、警察官と一般市民で構成される極右派の準軍隊組織である。暗殺隊はある程度独立的に意思決定と活動ができ、多くの細胞（グループ）で構成されていた。暗殺隊は軍人と警察官以外に、経済的に豊かで右翼のイデオロギーを持つ政治活動家と、教育水準が低いサディストな市民などが共存していた。映画「サルバドル」の幾つかのシーンでは、リチャードを脅迫する暗殺隊の一般市民が描かれている。暗殺隊は地下組織であり、その活動はマスコミから人々に伝わることがない。暗殺隊の目的は、貧困層を中心に一般市民が恐怖を感じることで、左翼ゲリラ組織と協力させないことである。権力を持つ軍部が率いり、夜中に活動する地下組織のため、誰が誘拐し、拷問と暗殺を行ったかを解明するのは困難である。  

 　映画「サルバドル」では、リチャード・ボイルと宣教師だった本名ジャン・ドノバンの友情関係が描かれている。映画の後半では 、3 人のアメリカ人シスターとジャンがレイプされ、暗殺されるシーンがある。これ は 198 0 年の出来事である 。4 人の女性聖職者は精神的・医学的に、内戦で怪我をした貧困層の犠牲者を助けていた。この暗殺を実行し た5 人の「国民警備隊」兵士は逮捕され、主な責任者であった、当時「国民警備隊」の司令官カルロス・ビデス・カサノバ は 201 2年2 月現在、アメリカのフロリダ州に暮らしているが、暗殺か ら 3 1 年後にフロリダ州の裁判官は、ようやく彼をエルサルバドルに引き渡すための道を切り開いた （ The Guardian Weekl y、 201 2年3月2 日） 。4 人の女性聖職者はロメロ大司教とともに「解放の進学」の考え方を支持していた 。5 人ともゲリラの暴力についても批判的であった。しかし、アメリカ政府とエルサルバドル軍にとっては貧しい農民は潜在的にゲリラに協力する人間であり、シスター達のように貧しい農民を助ける人は共産主義者とみなしている。エルサルバドル軍によっ て5 人は計画的に暗殺されるが、このケースはごく一部で、数万人にのぼる無罪の犠牲者が出ている。  

 　映画「イノセント・ボイス」は、農村に暮らす一般市民の観点からエルサルバドルの内戦を描き、映画の主人公は「チャバ」とい う 1 1 歳の男の子である。映画は現実にあったストーリーに基づいていて、その舞台 は 198 0 年か ら 198 1 年の間の、クスカタンシンゴ農村である。当時この村は左翼ゲリラとエルサルバドル軍と、準軍隊組織の両戦線の間に置かれていた。映画はチャバと彼の家族、小学校と近所の友人達、教会の神父と村に暮らす人々の日常を物語る。当時エルサルバドル軍は 、 1 2 歳以上の子供や少年を強制的に兵士にさせて、ゲリラと戦わせていた。チャバの伯父さんと近所の知り合いはゲリラ組織に参加し、チャバ は 1 2 歳になることに対して恐怖を感じていた。「イノセント・ボイス」はチャバとお母さんと兄弟の関係、同級生との初恋など、少年の成長を描く。映画の監督はメキシコ人ルイス・マンドキであり、メキシコで撮影され て 200 4 年に公開された。  

 　「イノセント・ボイス」のエンディングでは、内戦の犠牲者人数とともにこの内戦による約百万人のエルサルバドル人が亡命したとあり、アメリカ政府はエルサルバドル軍を訓練と指導するための人員と 、 1 0 億ドル以上の軍事支援を与えたと書かれている。エルサルバドル軍の訓練と指導に関しては、アルゼンチン軍事独裁政権の下で 、 197 6 年か ら 197 9 年の間に行方不明者になった人々を誘拐、拷問と暗殺を担当した準軍地下組織を率いるアルゼンチン人軍人が中米に送られ、エルサルバドル軍、グアテマラ軍とホンジュラス軍に訓練と指導をした。これ は CI A の作戦であり、チャーリー作戦と言う名をつけられた。  

 　暗殺隊を率いるダブイソンは 、 198 1年9 月に民族共和同盟 （ AREN A ）という右翼の政党を設立した 。 199 2 年にメキシコでチャプルテペック平和協定は結ばれ、内戦は終わった。エルサルバドルは民主化したが、その時から総選挙で二大政党は争い、それらは内戦の敵同士であっ た FML Nと AREN A である。ダブイソン は 199 2 年に病気で死亡した 。 200 9 年 に FML N は初めて政権を握った。  

 


 

    

    

    

    

    

第7 章

リオデジャネイロのファヴェラ

（ 195 0 年代 〜 199 0 年代）

 


 

⑬　黒いオルフェ

監督：マルセル・カミュ

出演：ブレノ・メロ、マルペッサ・ドーン、

レア・ガルシア

Orfeu Negro (1959) de Marcel Camus

    

⑭　オルフェ

監督：カルロス・ ヂエゲス

出演：トニ・ガヒード、パトリシア・フランサ、

ムリロ・ベニーシオ

Orfeu (1999) de Carlos Diegues

    

 　「黒いオルフェ」 は 195 9 年に公開されたフランス人マルセル・カミュ監督の映画であり、カンヌ国際映画祭ではパルム・ドールを受賞し、アカデミー賞では外国語映画賞を受賞した作品である。「黒いオルフェ」はヴィニシウス・ヂ・モライスによ る 195 6 年の戯曲「オルフェウ・ダ・コンセイサウン」に基づいている。戯曲も映画も古典ギリシア神話を物語るオルペウスとエウリュディケを登場人物に起用し、リオデジャネイロのカーニバルを舞台にして、主人公は黒人である。「黒いオルフェ」は、ファヴェラと呼ばれるリオデジャネイロのスラム街に暮らす人々のカーニバルの過ごし方を描く。ファヴェラは丘の上にあり、場所によってはリオの美しい景色は楽しめられる。人々の数日間の喜びと古典ギリシアの悲劇はポストカードのような風景と重なり、時によりファヴェラの生活は天国に近いものとして現れている。サンバと誕生したばかりボサノヴァという官能的な音楽は、享楽的な雰囲気をかもし出している。映画のサウンドトラックではアントニオ・カルロス・ジョビン、ルイス・ボンファ、ヴィニシウス・ヂ・モライスとアントニオ・マリアの曲が含まれている。ジョビン（音楽）とヂ・モライス（歌詞）の二人は、「イパネマの娘」のように世界中で知られているボサノヴァの代表曲を共作した。  

 　「黒いオルフェ」 は 2 3 ページで取り上げた、クビチェック政権 （ 195 6 年 〜 196 1 年）の時に製作された。クビチェック政権は多くの市民の指示を得て、 「 5 0 年の進歩 を5 年で」というスローガンで、経済成長を通じてブラジルの黄金時代を築き上げようとした。比較的に楽観主義の傾向が目立った時代であり、ブラジル北部と北東部からは、多くの人がサンパウロとリオデジャネイロに移住し、仕事を見つけて生活が安定するまでの一時的な住居として、ファヴェラに住むようになった。このような国内移住 は 193 0 年代のジェツリオ・ヴァルガス政権の産業化過程に始まったが、その時の住宅不足がファヴェラを拡大させた。クビチェック政権の下でファヴェラの拡大は続いたが、そこに暮らす人々は現在と比べると少なく、ファヴェラで過ごした年数は比較的に浅かった。初期の頃のファヴェラは「黒いオルフェ」で感じとれるように、故郷を彷彿とさせる活気と、心の純粋さを可能にしただろう。  

 　しかし、「黒いオルフェ」は当時のファヴェラにおいて生活の現実を反映していないということで、製作された当初、ブラジルの多くの知識人によって批判された 。 2 5 ページで記述したよう に 195 0 年代後半か ら 196 0 年代前半には、映画製作においてブラジルのシネマ・ノヴォ（新しい映画）という運動が誕生し、発展した。イタリアのネオレアリズムや、フランスのヌーヴェルヴァーグの影響を受け、国民に自分の生活を見せた。自分の現実を見ることを通じて人々の間には、その現実を変える必要があるという意識を広げるようにした運動である。ネルソン・ペレイラ・ドス・サントス監督の「リ オ 4 0 度」 （ 195 5 年）はシネマ・ノヴォの最初の映画として知られている。当 時 2 6 歳だったドス・サントスは、「リ オ 4 0 度」で初めてリオのファヴェラに暮らす人々の苦労を描くことにした。  

 　当時ドス・サントスはブラジル共産党員だったが、ブラジル政府と摩擦を生み出したくなかった共産党はこの映画の政策を拒否し、ドス・サントスは「革命が成功してからこのような映画を製作したら良い」と言われたそうだ。映画プロデューサーがドス・サントスのプロジェクトに興味を示さなかったため、彼は協同組合を設立し、投資家に映画の「株」を売った。映画のコストを最低限にしたドス・サントスは、リオのワーキング・プアとイタリアのネオレアリズムに対する彼の二つのコミットメントを合流させた。俳優と女優のほとんどはプロではない。  

 　「リ オ 4 0 度」のオープニング・シーンでは、リオデジャネイロの美しい航空風景が紹介される。ポン・ヂ・アスーカル、コパカバナ、イパネマと他の南部のビーチ、隣接する裕福な地区が紹介されてから、ファヴェラがクローズアップされる。「リ オ 4 0 度」では、ある日曜日の朝 に5 人の黒人少年がファヴェラを出発し、コパカバナ・ビーチ、コルコバド丘のキリスト像、パン・ヂ・アスーカル、動物園、マラカナン競技場などリオの主な観光スポットにピーナッツを売りに行く。それらの場所では観光客とブラジルの中上流階級に出会う。「黒いオルフェ」と異なり「リ オ 4 0 度」では、階級間の関係を通じて貧富の差と人種差別という社会問題が、日常生活の一部として紹介される。  

 　「リ オ 4 0 度」の撮影 は 195 4年3 月に行われたが、その後映画は禁止された。ファヴェラ住民の生活をそのまま見せる最初の映画であり、政府にとってはリオの好ましいイメージではなかった。しかし、クビチェック政権の誕生が確実になった時点 で 195 5年 1 2 月に上映許可を得て、翌年に全国で上映された。その後、ドス・サントス監督 は 195 7 年に「リオ北部」を製作し、この映画もリオのファヴェラに暮らす人々の問題を取り上げている。ファヴェラとリオデジャネイロのイメージに関しては、「黒いオルフェ」と「リ オ 4 0 度」は対照的である。「黒いオルフェ」では、リオで目立つ社会階級の二極化が紹介されない。「黒いオルフェ」は世界で上映され、リオのポストカードのような美しい風景と熱帯の国の人々の幸せという二つのイメージ、そしてボサノヴァの普及に大きく貢献した。本章ではリオのファヴェラの現実を論じるので 、 195 0 年代の状況を紹介するためには「リ オ 4 0 度」と「リオ北部」は相応しい映画だったが、日本で は DV D 化されておらず、多くの場合海外でも見つかりにくい作品である。「黒いオルフェ」は、「リ オ 4 0 度」 の4 年後に、「リオ北部」 の2 年後に製作され、支配層の観点からすると「やっとファヴェラの好ましいイメージが映画化された」と言えるだろう。ドス・サントス監督はその後数十年もの間、興味深い映画を製作し続けた。  

 　シネマ・ノヴォの代表でもあるカルロス・ヂエゲス監督は 、 199 9 年に「黒いオルフェ」のリメイク映画である「オルフェ」を製作した。「オルフェ」は「黒いオルフェ」と同じように古典ギリシア神話物語とヴィニシウス・ヂ・モライスの戯曲に基づいていて、舞台はカーニバル時期のリオのファヴェラである。「黒いオルフェ」 の 4 0 年後に製作された「オルフェ」は 、 199 0 年代にファヴェラの人々が直面する現実を物語る。ヂエゲス監督は他のシネマ・ノヴォ映画運動関係者と同じように「黒いオルフェ」に対する不満を持ち、いつかリメイク映画を製作したいと思っていた。「黒いオルフェ」と「オルフェ」の間 に 196 4 年か ら 198 5 年まで続いた軍事独裁政権があり、軍事政権の存在はリメイクの製作に悪影響を与えただろう。ヂエゲス は 198 0 年にヴィニシウスに「オルフェ」のプロジェクトを紹介した。ヴィニシウスは喜んで脚本を自分自身が書くことを提案した。しかし、ヴィニシウスは同 年7 月に死亡し、ヂエゲスはプロジェクトを休止した 。 199 0 年代に入り、ようやく新しい脚本に基づいた「オルフェ」が製作された。  

 　ヂエゲス監督によると、ファヴェラの歴史は三つの段階で分けることができると言う。最初の段階 は 195 0 年代まで続き、その間にファヴェラの人口は比較的に少なく、長閑で文化的に純粋な場所であり、これは「オルフェウ・ダ・コンセイサウン」、「リ オ 4 0 度」、「黒いオルフェ」や、当時多くのサンバ曲の歌詞と共通した点である。丘の上にあるファヴェラは「天国に近い」場所として描かれた。二つ目の段階 は 196 0 年代か ら 198 0 年代まで続く。この時期は軍事独裁政権と重なる。ファヴェラの人口は増え、苦労をしてファヴェラを後にするという期待は徐々になくなっていく。絶望に満ちた時代であり、挫折は暴力を生み出し犯罪はどんどん増える。この第二段階は、細かくいえば、また三つの時期で分けることができる。本章で取り上げる「シティ・オブ・ゴッド」は、ファヴェラの第二段階とその 内 196 0 年代 、 197 0 年代 、 198 0 年代前半のそれぞれの特徴を物語る 。 198 0 年頃になるとファヴェラが「地獄に近い」場所として描かれる。  

 　ファヴェラの三つ目の段階 は 199 0 年代からの時代であり、「オルフェ」はこの段階を描いている。ファヴェラの住民は数十年前からファヴェラに暮らしている。すでにファヴェラの独特な文化は形成し、住民の間にはファヴェラで暮らすプライドが目立つ。主に麻薬製造と取引の形で犯罪は増え続ける。貧困からの脱却は音楽、他の芸術、サッカーあるいはビジネスになったカーニバルと関連する仕事意外に考えにくい。三つ目の段階では不利な条件が多く存在するにも関わらず、ファヴェラ住民間のプライドが広がっていく。映画「オルフェ」では主人公オルフェが詩人と作曲家として成功し、金銭的にも不自由がないのに、ファヴェラから離れようとしない。  

 　最後に、「黒いオルフェ」と「オルフェ」の原作である戯曲を書き、ボサノヴァの創始者の一人であるヴィニシウス・ヂ・モライスについてだが、彼 は 191 3 年にリオデジャネイロに生まれ 、 1 0 代から詩と歌詞を書き始めた。大学で法律を勉強したヴィニシウス は 3 0 代に外交官になり、ロサンゼルスとパリに派遣された 。 195 4 年に駐仏ブラジル大使館で勤めていた時、戯曲「オルフェウ・ダ・コンセイサウン」がサンパウ ロ 40 0 周年記念賞を受賞した 。 195 8 年に発行されたエリゼチ・カルドゾという女性歌手のレコードはアントニオ・カルロス・ジョビンとヴィニシウス・ヂ・モライスが作曲した曲で構成されている。このレコードに含まれていたシェガ・ヂ・サウダジはボサノヴァの最初の曲であると思われている。その時点からジョビンとヴィニシウスは、ジョアン・ジルベルトとともにボサノヴァというブラジルの新音楽ジャンルの創始者として知られるようになる。翌年にはジョビンとヴィニシウスの音楽を含む「黒いオルフェ」が製作される。ヴィニシウス は 40 0 以上もの曲を作曲し、同時に詩人であった彼は、主にボサノヴァの歌詞を書いた。  

 　 200 5 年にはヴィニシウスの人生を物語る「ヴィニシウス―愛とボサノヴァの日々」という映画が製作された。この映画では、ヴィニシウス、ジョビンとジョアン・ジルベルトだけではなく、数十年の間にブラジル音楽において最も有名な作曲家、歌手とギタリストなどがヴィニシウスについて思い思いに語る。その内「オルフェ」の音楽を担当した、優れた作曲家で有名な歌手として知られているカエタノ・ヴェロゾや、多くの歴史に残った曲 と 197 0 年代の民主化運動の祝歌となった「君にも関わらず」 （ A pesar de Voc e ）を作曲したシコ・ブアルキが含まれている。

Cidade de Deus (2002)  

de Fernando Meirelles y Katia Lund

    

 　「シティ・オブ・ゴッド」（ポルトガル語でシダヂ・ヂ・デウ ス Cidade de Deu s ）はリオデジャネイロにある一つのファヴェラの名である 。 199 7 年に同じ作品名の小説が出版された。この小説の著者はパウロ・リンスであり、彼 は7 歳からシダヂ・ヂ・デウスで育てられた。ファヴェラ住民の間には社会的移動が少ないが、リンスは例外的な存在である。小説「シダヂ・ヂ・デウス」はブラジルの現代文学で、最も優れた作品の一つとして評価されている。リンス は 198 6 年 〜 199 3 年の間に「リオデジャネイロ民衆階級における犯罪」という人類学の共同研究プロジェクトに参加し、自分の経験と研究成果に基づいて小説シダヂ・ヂ・デウスを書いた。この小説の内容からブラウリオ・マントヴァニが、映画「シティ・オブ・ゴッド」の脚本を書いた。小説では多くの主人公と多様なストーリーが絡み合っているので、その中から主な主人公を選び、ストーリーをできるだけ分かりやすく紹介することはマントヴァニの困難な仕事であったが、見事に成功した 。 200 2 年に公開された「シティ・オブ・ゴッド」は国際的に評価が高く、例え ば 201 0 年エンパイア誌の「英語圏以外の世界映画ベス ト 10 0 本」 で7 位にランクされた（１位は黒澤明の「七人の侍」である）。  

 　本書で取り上げられている「クアトロ・ディアス」と「セントラル・ステーション」に出演しているマテウス・ナッチェルガリ以外、「シティ・オブ・ゴッド」の出演者のほとんどはファヴェラ住民である。製作スタッフは出演者を選ぶために一年ほどかかった。最初の段階では、ファヴェラにある演劇学校と青年組織の中から選ばれた 約 200 0 人が面接を受けた。面接を合格し た 40 0 人を演劇ワークショップに参加させ、アドリブを中心に演技指導が行われ、最終的にメイレレスやルンヂ監督と製作スタッフが、主な出演 者 6 0 人とエキストラ役 の 15 0 人を選んだ。その後の継続的な演技指導によって、選ばれた出演者は短編映画「パレスⅡ」に出演する。この短編映画は「シティ・オブ・ゴッド」の製作の総合的なリハーサルとなる。  

 　このような出演者の選抜・指導と映画の製作過程は「シティ・オブ・ゴッド」の強烈なリアリズムの一つの要因であるが、映画の中身もリアリズムそのものである。麻薬取引に携わるギャング間の戦いが多いのに、血が流される暴力シーンは少なく、攻撃性は独特な編集を通じて表現される。編集と関わるもう一つ興味深い様相は、映画のペースが徐々に早くなることである。映画の速度とファヴェラの歴史には密接な関係があり、最初の段階ではのどかな生活が残るが、スピードが増すにつれ、徐々 に 198 0 年代前半の「地獄」に移行する。  

 　「黒いオルフェ」と「オルフェ」の解説で、ファヴェラの歴史において三つの段階について論じた。映画「シティ・オブ・ゴッド」のストーリー は 196 0 年代 〜 198 0 年代を含む第二段階に当てはまるが、映画自体は三つの部分で別れて、それぞれはおよ そ 196 0 年代 、 197 0 年代 と 198 0 年代前半の出来事を物語る。各部分では音楽とファッションだけではなく、撮影の形式も変わっていく。この三つの部分では、ファヴェラの歴史において三つの段階の特徴が含まれていると言える 。 196 0 年代には犯罪が存在していたにも関わらず、広いスペースと自然との関係は残り、貧困から脱却できない国内移住者の間に協力的な人間関係が残っていた。映画で見られるように、場合によって犯罪の成果は住民の間に配られていた 。 196 0 年代は激しい学生運動と労働運動 、 196 4 年の軍事クーデターを含む社会的に不安定な時代であった。ファヴェラで苦労していた貧困層の間には日常的な助け合いが残っており、泥棒したものを皆に配るという出来事を説明できるだろう。  

 　リオデジャネイロの地方自治体とブラジルの中央政府によるファヴェラを破壊するという解決策 は 193 0 年代のヴァルガス政権から存在していたが 、 196 0 年代後半になるとこの政策は以前より広く実行された。軍事独裁政権はファヴェラ住民を強制的に別の場所に移動させることを目指していた。他方、リオデジャネイロのファヴェラ住民は強制的な移動に対して以前から抵抗していた。クーデターの前 年 196 3 年にゼ・ケティが作曲した「意見」というサンバの歌詞は「逮捕されても、殴られても、食べ物さえいただかなくても、私の意見は変わらない、丘（ファヴェラがある場所）を後にしない」と訴えていた。ゼ・ケティの「丘の声」と他の曲はネルソン・ペレイラ・ドス・サントス の 195 5 年の「リ オ 4 0 度」 と 195 7 年の「リオ北部」のサウンドトラックに含まれている。  

 　ファヴェラ住民の根本的な問題は失業者とインフォーマルな状況で働いている人が多いということである。リオデジャネイロ、サンパウロなどに移住しても安定的な雇用がなく、移住者はマンションを借りるための購買力がない。移住者の人数と比べると安定雇用は少なかっただけではなく、成長していた製造業で要求されていた労働力の熟練度合いと文盲が少なくなかった移住者の極めて低い教育水準は労働市場のミスマッチを引き起こした。ファヴェラの性質は一時的な住まいから世代を超える長期的な住まいへ徐々に変わっていく。長期雇用を通じて貧困から脱却する道が閉ざされたため、ファヴェラ住民の一部は不法な収入源を得る活動に移転し、そのうち麻薬取引が支配的になる。  

 　 197 0 年代は過渡期であるといえる。リオデジャネイロ で 196 2 年 と 197 4 年の間に 約 1 4 万人がファヴァラから追い出され、彼らの家は破壊され、ファヴェラのコミュニティ・リーダーは拷問され、暗殺されたこともあった。しかし、政府はファヴェラを消滅させることができなくて、逆に北東部から移住者の流れのペースは落ちず、ファヴェラは以前より拡大した。映画「シティ・オブ・ゴッド」では、第二部であ る 197 0 年代に入ると建物の増加と住宅が密集した町の雰囲気が目立つ 。 197 0 年代後半になると軍事政権はファヴェラから手を引き、町づくり政策や社会政策も存在しなくなる。雇用機会が少ないファヴェラ住民の間では不法収入源の要求が拡大し、麻薬取引とそれに伴う暴力や犯罪が広がる。暴力は古里の損失と移住先において社会的排除に伴うフラストレーションと深い関係を持つ。  

 　以上述べた社会的な状況におい て 197 0 年代後半にファヴェラの生活に質的な変化をもたらした出来事はコカイン取引の拡大と武器の大量流入である。この両者 は 197 0 年代を取り上げる「シティ・オブ・ゴッド」の第二部で描かれる。映画の第一部はノスタルジックな気持ちをこめて「天国」として紹介され、第二部は「煉獄」として紹介される。カトリック教会によると「煉獄」に行く魂は一時的に苦労すると、その後天国に上ることができる。映画の第三部は「地獄」として紹介され、関係者にとっては救済の可能性がないという絶望的な状況を表す。「シティ・オブ・ゴッド」では「煉獄」から「地獄」への移行は象徴的にディスコのさよならパーティーにおいてベネの暗殺を通じて表現される。ベネがいないとゼ・ペケノ（リトル・ゼ）の強迫的な暴力を制止できる人がいない。この無規範状況の支配 は 198 0 年頃においてシダヂ・ヂ・デウスの「地獄」時代の開幕を象徴する。「地獄」時代の犯罪は「天国」時代において住民の助け合いと関係なく、犯罪組織以外の市民の生活を悪化させる個人の利害関係に基づいている。  

 　「シティ・オブ・ゴッド」の最初のシーンからブスカペはナレーターを務める。小説を書いたパウロ・リンスと、映画のナレーターである主人公ブスカペは貧困再生産の悪循環から脱却できた。しかし、「シティ・オブ・ゴッド」 は 196 0 年代か ら 198 0 年代に進むにつれ、貧困と暴力の悪循環から逃げるための道が閉ざされていくという印象を与える 。 199 0 年代のリオのファヴェラを描く「オルフェ」では、主人公オルフェは作曲家、歌手とエスコーラ・ヂ・サンバのリーダーとして成功する。しかし、彼はファヴェラからリオの中上流階級が暮らす地域に引っ越さない。ヂエゲス監督によるとオルフェはこの時期に広がるファヴェラで暮らすことに対する住民の自慢を表す。子供の頃のオルフェの親友はルシニョであり、彼は大人になるとファヴェラの麻薬取引のボスになる。あるシーンではオルフェが「ルシニョ、今なら可能だからこの丘から逃げて、この生活から脱して」と言う。ルシニョは「そうしたら何をして生きていけばいい？町の人々が望むように清掃員になったらいいか？他人のトイレ掃除をして人生を送ったらいいか？そして父親のように敗者になってくそまみれの中で死んだらいいか？私はこの年齢ですでに、母親と兄弟に住まいを与えた」と答える。オルフェは「ルシニョ、君は長く生きられないだろう」と言うと、ルシニョは「短い人生でもいい生活を送りたい」と答える。この会話ではルシニョが極めて激しい貧富の差を示すブラジル社会において、世代を超えるファヴェラ住民の屈辱をまとめている。  

 　 199 0 年代には民主政権の下でリオデジャネイロの地方自治体は、大規模になっていたファヴェラの都市化政策を開始し、下水道、上水道、街頭やゴミ収集と言う基本的なインフラやサービスの提供を拡大した。国連の報告書による と 200 0 年代にはブラジル全国でファヴェラの人口 は 1 6 ％減少した 。 201 1 年にはリオデジャネイロのファヴェラ で 17 0 万人を超える人々が暮らし、その人数はリオ総人口 の 1 4 ％を占める。同年、ブラジル全国ではおよ そ 120 0 万人ファヴェラに暮らしていた。
