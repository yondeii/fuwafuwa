　００１　新しい住宅をつくる方法
　－ － － － － － － － － － － － － － －
以前に、あるマンションの広告を見ていたら、部屋の間取りに対して、
ベランダは斜めの構造になっていました。
このようになっているメリットが分かりますでしょうか。
確か広告では、以下のメリットがあると書かれていました。

　〇　玄関前に生まれるデッドスペースが、プライバシーを
　　　確保する空間になる。
　〇　一般的なマンションよりも、ベランダの奥行きを確保できる。

ある建売一戸建て集合住宅の広告では、住宅を平行に並べないで、
目の前の道路を緩いカーブにしていました。
詳細はあまり覚えていませんが、この状態のメリットは、以下のように
書かれていたと思います。

　〇　車のスピードが上がらないので、安全な生活道路になる。
　〇　住宅の日照時間や、近隣との適度な距離感、住宅街の
　　　景観がよくなる。

このように定番の状態や形状でも、少しだけ見方を変えたり工夫しますと、
大きなアイデアになって、魅力的になるのではないでしょうか。
ところで一般的に住宅といえば、以下のようなイメージがあります。

　〇　木造住宅は「子供や健康に良さそうだけど、自然素材なので
　　　寿命が短い」
　〇　コンクリート、鉄筋鉄骨マンションは「丈夫で長持ちそうだけど、
　　　無機質で冷たい」

すると、これらの長所が一つになれば、魅力的な商品になるのではないで
しょうか。

　〇　住宅の外観、骨格は「コンクリート、鉄筋鉄骨」を使う。
　〇　住宅の内装は「木材」を使う。

すると外観は普通の住宅やマンションにも関わらす、玄関に一歩足を
踏み入れると、木の香りがする魅力的な住宅になるのではないでしょうか。
「建築コストが上がってしまうのではないか」と思われるかもしれませんが、
内装の木材は、円安になると安上がりになりますし、国内の間伐材を使うと、
需要も生まれます。
ちなみに、木材を同じ形にしてパネルのようにしますと、内装の取り替えや、
部屋の雰囲気が変えられる、リフォーム用としても使えます。
もし、木材コストが上がりそうな場合は、内装の下部だけに使います。
するとツートン状の内装になりますし、コストも抑えられます。

　００２　パチンコ台が低コストで作れる方法
　－ － － － － － － － － － － － － － － － － － － －
最近のパチンコ台は、芸能人の方や過去に流行ったアニメ、有名な
キャラクターなどを使っているようです。
なかには女性を意識した台も見られますが、以前は考えられなかった状態
ではないでしょうか。
そして最近では、お客さんを飽きさせないためか、以前よりも使用期間が
短いような気がします。
すると肖像権の面から、製造コストが上がってしまうのではないでしょうか。
しかし、その一方では人気が高いので、長年、置かれている台も存在します。
すると話題性によって、お客さんの新規開拓をする台を作る一方で、
従来からある、長く楽しめるタイプも開発しますと、低コストなパチンコ台に
なるのではないでしょうか。

　〇　液晶画面に映るの横向きの車が３分割されて、バラバラに回転
　　　させます。
　　　そして、車体の形が「７７７」のように揃ったら「当たり」
　　　とします。

　〇　その後は、画面内の（「当たり」の）車がレースを始めて、
　　　優勝したら、ふたたび当たる、確率変動に入ります。　

現在では、幅広い年代の方がパチンコをしますが、車好きの世代の方が
多いのではないでしょうか。そこで、これらの方が長く楽しめる台を作ります。
子供の頃は、このようなタイプのコインゲームがありました。
もしパチンコでも同様に、横から見た車体を３分割します。
すると「当たり」の後のレースは、楽しいのではないでしょうか。
現在のパチンコ台の液晶は、かなり綺麗なので、見た目も問題ありません。
そして一度作れば、新車が出てもプログラムの一部変更で対応できると
思います。
また自動車メーカーにとって宣伝になるので、権利関係のコストアップは、
最小限になると思います。
ゲームメーカーさんとの協力があれば、さらに楽しい台になると思います。

　００３　丸い自動掃除機を売る方法
　－ － － － － － － － － － － － － － － －
最近の家電や情報端末は、今までになかった形状や、新しい市場を
生み出す開拓的な製品も見られるのではないでしょうか。
例えば最近売られている掃除機は、コンパクトなだけでなく、デザインや
機能が魅力的なので、高価でも売れているようです。
そして、なかでは自動化されたコンパクトな掃除機が、新しい製品と言える
のではないでしょうか。
おそらく、どのメーカーでも「あの形状」なのは、隅々まで掃除ができるから
だと思いますが、一見では「子供のおもちゃ」にも見えてしまいます。
ここに、製品を売る新しいヒントがあるのではないでしょうか。

　〇　掃除機の外見を「カワイイ系」に変えてしまいます。

例えば形状からイメージして、カワイイ系の生き物（テントウムシなど）に
変えてしまいます。もし何らかのキャラクターを使う場合は、丸い顔（だけ）
でも魅力的になると思います。そして新たに製品の金型（？）を作ると、
コストが高すぎるなら、製品に付ける「カワイイ系」のカバーを作ります。
すると既存の設備や、すでに発売された製品でも使えますし、カバーを
付けることで、よごれの防止や着せ替えの楽しさも出てくるのではないで
しょうか。
このような開発イメージを持ちますと、例えば一人暮らし用の家電製品は、
既存の製品よりもコンパクトになっています。
すると一般的なサイズの製品よりも、カワイイ系に応用しやすいと思います。
その結果、１つの商品が「カワイイ系／一人暮らし用」になります。

また、大手のメーカーでは、様々な家電を取り扱っています。
そこで新たな部署として「カワイイ課」を作って、商品性の高い開発を
行ないますと、機能性と同時に、魅力的な商品を作れると思います。
すでに携帯電話やスマートフォンでは、このような製品があるようです。
これは似たような商品になりやすい家電、工業製品にとっては、新たな
付加価値になると思います。

　００４　新しいピープルブーバーをつくる方法
　－ － － － － － － － － － － － － － － － － － － － －
これからの日本は少子高齢化社会を迎えるので、人口比率が大きく違う
場所ができるのは自然なことです。
そして人が集まる場所では、集客力のある施設や、交通の便も必要ですが、
同時に、快適に移動できる環境も必要ではないでしょうか。
例えば、ある動物園では、山を削った斜面にあるので、園内を移動するバス
があるようですし、大規模な施設では、内部をゆっくりと移動する乗り物が
あります。
これらはビジネスのために置かれていますが、人件費や設備の費用が必要
なので、公的な施設の場合は、予算が回りにくいのではないでしょうか。
そこで大きな公園や外の施設、駅の移動などの移動に使える、低コスト（？）
なピープルムーバーがあればよいと思いました。

　〇　スキー場で使われるリフトを、移動手段として使います。

「人の移動」は、どうしても速さや効率に重点が置かれてしまいます。
しかしエレベーターやエスカレーターのように、ある程度の速さと安全が
あれば十分な場所も多いのではないでしょうか。
そこで、既存の製品であるリフトを利用します。
この場合は、スピードを遅くして椅子の間隔を狭くします。
そして防風などの安全性を確保すれば、人を置く必要のない乗り物になる
かもしれません。
そして、移動中であると同時に「座って、休む時間」も作れます。
すると快適な移動になりますので、例えば、外出の機会が減る高齢者の方に
とっても、体力的な心配がなくなります。
また、お子さんから見ますと遊園地の乗り物に見えるので、集客効果も
あるかもしれません。例えば、駅前や直線的な場所にレールを引きますと、
小さく安全な移動する乗り物になります。
すると「広い場所を移動する」「座って休む」「楽しく移動する」時間が、
同じ行為になります。もしかしたら遊園地にある乗り物なども、
見方を変えると、新しいピープルムーバーとして使えるかもしれません。

　００５　新しい「白いお皿」をつくる方法
　－ － － － － － － － － － － － － － － － － － －
懸賞のキャンペーンや広告に「応募券を集めて、商品が当たる」懸賞は
たくさんあります。
なかには、その商品が「かならず当たる」場合もありますが、ある程度の
流通経路がないと、輸送コストがかかるので、それほど多くないのかも
しれません。その商品が定番になると、高い人気や認知度ができる半面、
企画としては、マンネリ化してしまう面もあるのではないでしょうか。
それに商品の原価も大きく上げるられないので、ここでも変えることが
難しくなっています。
例えば、ある食パンを作るメーカーでは、白いお皿のプレゼントが
長年の定番になっています。これは人気の高い表れだと思いますが、
定番になってしまうと、インパクトは下がってしまうのではないでしょうか。
自宅の食器棚は、白いお皿が多くありますが、サイズの違うタイプは重ねて
置くため、下の「お皿」を使う機会は減っています。
そこで定番のイメージを残しつつ、少しだけプレゼント商品に工夫を
加えるのがよいのではないでしょうか。

　〇　白いお皿に、スプーン、フォークが「ひっかけられる」場所を
　　　作ったり、コーヒーカップやおかず、デザートが載せられる
　　　小さな場所を作ります。

丸いタイプのお皿を使っている時に、スプーンが滑るので思い付きました。
すると定番のイメージを保ちつつ、プレゼントの魅力を上げられます。
現状のように、お皿のサイズや深さや外見の変化だけでは、インパクトが
欠けてしまいます。
そこで定番のイメージを保ちつつ、プレゼントの魅力を上げます。
ちなみに、現在放送されている「焼き肉のタレ」のプレゼント商品は、
タレやお肉、野菜などが乗せられるようになっています。
そして中央部にある「タレを入れる場所」は、ひとつです。
そこで複数のタレを入れる場所を作りますと、いろいろな味が楽しめると
思います。こうすることで、冷蔵庫に置く（焼き肉のタレの）所有数も
上がるので、売り上げも上がるのではないでしょうか。

　００６　農作物の被害を減らす方法
　－ － － － － － － － － － － － － － － －
主に山間部では、野生動物による農作物の被害が増えてるようです。
この状況は、人間の開拓による影響もあるので、単に害のある動物と
決めつけるのは、少し違う気がします。
一部の地域では、その対策として、動物の駆除や食用として販売する
動きも出ていますが、食用として利用できる場合に限られますし、
流通経路や販売対策などが必要なので、多くの時間と費用がかかるのでは
ないでしょうか。そこで、農作物の被害が減られるかもしれない方法を
ご紹介したいと思います。

　〇　畑の周りに「丈夫な釣り糸」を張って、動物が入らないように
　　　囲います。

単に糸を張るだけなら、強度は足りません。
ですが動物によっては「見えない、釣り糸」に恐怖を覚えて、近づかない
可能性も出てくるのではないでしょうか。
これは街中のゴミ置き場などで、カラスの対策になるのではないかと
思いました。例えばカラスの場合は、糸を張らないで、ゴミ置き場の周りに
「糸を垂らす」状態にします。
一般的に鳥類は、羽に何かが当たるのを（命にかかわるので）非常に嫌う
と聞いたことがありますし、鳥目という言葉もあるので、見えない糸が
当たる状態を怖がるのではないでしょうか。
それに張らない状態は、安全性も高いと思います。
テレビで見たのですが、カラスの対策としては、袋に匂いをつけていました。
山間部の対策に話を戻しますと、イノシシは、人工的なモノを嫌うようです。
そこで、使わなくなった中古のタイヤを半分に切って、丸い山が並ぶように
置いても、効果があると新聞記事に書かれいました。
それに水も溜まらないので、蚊の発生も抑えられるのではないでしょうか。

またサルの被害に悩む場所では、大きなサルの「ぬいぐるみ」を置いたり、
ゴム製の蛇を使ってはいかがでしょうか。
野生の猿は、本能的に火や水、蛇などを怖がると聞いたことがあります。
（慣れてしまうと温泉に入ったり、焼き芋を食べる場合もありますが）
そしで現地を歩く場合は、釣り糸などでゴムの蛇を「ひっぱりながら」
歩くと、野生の猿は警戒するかもしれません。
（いつかはバレて、学習してしまうかもしれませんが）

　００７　新しいオーディオをつくる方法
　－ － － － － － － － － － － － － － － － － －
現在のパソコンは、一家に一台あるので、一般的な家電製品になって
ました。ご家庭によっては、仕事や子供用のパソコンがある場合も
多いのではないでしょうか。
そして新しい家電といえば、スマホやＭＰ３プレーヤーによって、自宅や
お部屋での音楽の聞き方が、イヤホンへと変わりつつあるようです。
その結果、以前は普通に置かれていたオーディオ製品が、現在では必需品で
なくなっているようです。家電の広告を見ますと、音響製品が載っていない
場合もありますし、ＭＤを知らない方もいらっしゃるのではないでしょうか。
そして、子供部屋や寝室にはオーディオがなくても、薄型テレビなら
置いてある場合も多いと思います。
ここに、新しいオーディオの可能性があるのではないでしょうか。

　〇　「横長、薄型で、奥行きの浅い」一体型コンポを作ります。

家電量販店に置かれる「小さな」オーディオを見ますと、正面から見ると
コンパクトでも、奥行きが４０センチ近い製品もあります。
これは、家電化したパソコンや薄型テレビから見ますと、時代遅れのサイズ
ではないでしょうか。おそらくは従来品からの代替え需要や、開発コストの
影響だと思いますが、中古でも構わない方なら、安くて質のよい中古品が
買えてしまいます。
すると代替えの市場は、それほど大きくないのではないかと思われます。
そこで「横長、薄型で、奥行きの浅い」オーディオを作ってはいかがで
しょうか。ＤＶＤレコーダーは、サイズの違いはそれほどないので、
上に載せられるサイズをイメージしす。
この時にＤＶＤプレーヤーの金型を使うと、製造コストが下げられます。

ちなみにテレビ番組を録画できるゲーム機はありますが、その逆である－
「ゲームもできる、ＤＶＤレコーダー」は見当たりません。
ＤＶＤレコーダーを買う方には「時々は、ゲームもできる製品」が欲しい
方もいらっしゃるでしょうし、テレビだけでゲームができる製品も需要が
あるかもしれません。
話を戻しますと、ＤＶＤレコーダーの上にオーディオを置いた場合は
「音楽を聞きたい」方と「いい音質で映画を見たい」方がいるので、
専用の純正スピーカーは作りません。
すると薄型テレビでは音質が物足りない方にとっても、単品スピーカを選ぶ
楽しみがありますし、魅力的な製品になるのではないでしょうか。
「サラウンド機器や、低音用スピーカーがあるのではないか」と思われる
かもしれませんが、オーディオに詳しくない方によっては、設定が難しく、
大がかりに思われます。
そこで従来から馴染みのある一体型オーディオを、ＤＶＤレコーダーの上に
置くようにしますと、リビングやお部屋でも、ほとんど邪魔になりません。
このような製品を作りますと「良い音で映画を見たい／音楽が聴きたい」
方々の需要に対応出来るのではないでしょうか。

　００８　会社の経営が上手くなる方法
　－ － － － － － － － － － － － － － － － －
脳の研究をされている方は「人間の社会や環境は、脳内部を具現化している」
とおっしゃっています。例えば政治、経済の分野では、全体をコントロール
する人がいるように、脳内部でも、まとめる個所があります。
ですから、確かに共通した個所があると言えるのではないでしょうか。
ちなみにパソコンは、脳の構造を模して作られているそうです。

ところで「手を使って、何かを取ろうとする時」は、脳の中で何が起こって
いるのでしょうか。ほとんどの方がイメージするのは、まず最初に－
「脳（心）のどこかで、何かを取ろうと考える」意識から始まって、次に－
「体の運動に関係した個所が、反応する」そして、その後は－
「脳の命令が、腕の神経に伝わる・・」順番をイメージすると思います。
実際は、脳内部の「体の運動に関係した（腕を動かそうとする）個所」が、
先に反応するそうです。
この内容を知ったときは、驚き、不思議に思ったのですが、後になって
納得できる面もあるのではないかと思いました。
その理由は、脳内部が役割を分担しつつも、何かをしていることによって
（つまりは「いつでも」です）すべてが物理的に「つながっている」状態
だからです。
すると、もっとも発達している、人間の脳内ネットワークを利用すれば、
会社の経営や組織でも、有効に使えるのではないでしょうか。

　〇　各部署の役割を分担しながらも、上下左右の連携が出来る
　　　ようにしておく。

これは、トップダウンでもボトムアップでもありません。
例えば、何かを作る場合でも、専門的なことは各部署に任せながら、
その中で「まとめ」が必要になります。
そして、通常は小さな「まとめ」が「大きな、まとめ」になって、
組織の上へ向かうと思いますが、これだけでは、もったいないのでは
ないでしょうか。社員が１万人いたら「１万人の頭脳」を使ったほうが、
良いアイデアが生まれやすいからです。
そこで、どの部署でもアイデアを思い付いたら、たとえ部署外の事でも、
内容が上や下へ流れる体制にします。そして会社によい結果が出たら、
アイデアを出した方に対しての報酬があってもよいと思います。
このような組織にしますと、脳の構造に近いネットワークができるので、
上手く利用できるのではないでしょうか。

　００９　新しい自転車を作る方法
　－ － － － － － － － － － － － － － －
最近の自転車は、電動機の付いたタイプが徐々に普及しているようです。
主に高齢者の方が利用してますが、今後の高齢化社会を踏まえますと、
値段が下がっていきながら、普及が進んでいくと思われます。
ですが、ここで問題になる面もあるのではないでしょうか。
現在の電動自転車は、主に坂道を上がる時にメリットがありますが、
高齢の方や、重い荷物がある場合は、ふらつきを起こしやすい状態です。
そこで、新しい自転車を作る方法をご紹介します。

　〇　自転車の後ろの荷台に、小さな子供や買い物の荷物が置ける
　　「三輪の、電動機付き自転車」を作ります。

これからの自転車は、従来のタイプだけでなく、主に高齢の方が安心して
乗られるタイプが必要になると思います。
その場合は「速さ」よりも、安全性の高い、日常の足代わりになるのでは
ないでしょうか。
これからの自転車は、三輪タイプの需要も出てくるのではと思います。
ちなみに原付きも、三輪タイプや屋根の付いたタイプなど、個人で使う
新しいピープルムーバーの需要があると思います。
過去には、屋根付きの原付が発売されていたと記憶しています。
当時は、それほど売れなかったようですが、今後は、このタイプも出てくる
のではないでしょうか。そして荷物が載せられるタイプは、同時に、
お子さんを乗せられるようにすると、一台が二役になります。
すると開発コストが下がるので、量産効果も出てきます。
この場合は、サイズや重さがありますので、足代わりに盗まれることは
少ないのではないでしょうか。
現在の「人が、移動に関わる製品」は、小型車が売れています。
そして自転車では、動力付きのタイプが出てきました。
これは、小型でパーソナルな「新しいピープルムーバー」と言えるのでは
ないでしょうか。すると今後は、自転車と原付きの間に位置するタイプや、
数人乗りの小型自動車の需要が出てくると思います。

　０１０　長持ちする学習机をつくる方法
　－ － － － － － － － － － － － － － － － － －
小学校の入学時に買った学習机を、大人になっても使われている方は、
どのくらいいらっしゃるでしょうか。
おそらくは、ほとんどの方が使っていないと思います。
これは机そのものが必要でなくなった場合もありますが、学習机が
使えないわけでもないのに、新たに買い求めた方もいるのではない
でしょうか。何故、このように使われなくなってしまうのでしょうか。
その理由は、学習机の「奥行き」にあるのではないかと思います。
つまり、ある時期になると、一般的な学習机の奥行きが必要なくなるので
（邪魔になるので）処分してしまうのではないかと思いました。
すると机の「違い」を考えますと、新たな（長持ちする）学習机が出来る
のではないでしょうか。

　〇　奥行きのある／ない、２つの天板を持った学習机を作ります。

最近の学習机はシンプルなものが好まれるらしく、以前のような、
前面にあれこれ付いているタイプは少ないです。
なかには本棚がまるごと取り外せたり、別の家具として使える商品も
あるようです。
これらの商品も、長く使うことを想定しているかもしれませんが、
おそらくは中学高校までの、学生の時期が対象ではないでしょうか。
そこで、あらかじめ２つの天板が付いた、奥行きの変わる学習机を作って
みてはいかがでしょうか。すると値段が張っても「長く使える、机」として、
新たな需要が生まれると思います。
このような商品を作りますと、奥行きのある天板部分は使われなくなります。
そこで、使わない天板は仕切りになったり、新たな本棚にするなど、新たな
工夫が必要になります。

　０１１　新しい住宅をつくる方法
　－ － － － － － － － － － － － － － －
「新しい住宅をつくる方法」は、今までとは違う内装のご提案でしたが、
ここでは主にマンションに対応した、新しい「間取り」について
ご紹介したいと思います。
一般的なマンションの間取りといえば、３／４ＬＤＫが標準になっている
のではないでしょうか。
これは、かつて一般的だった「両親＋子供二人」をイメージした間取りに
なっていると思います。そして、その中でもっとも多く見られるタイプは、
廊下の左右にお部屋が並んで、突き当たりがＬＤＫの状態だと思います。
しかし最近は少子化の影響で、主に都市部の住宅では、従来のタイプが
標準とは言いにくい状況ではないでしょうか。
ちなみに、少子化は「今まで通りの、標準的な家庭人数」と
「子供を持たない、人や夫婦」との二極化になっているようです。

そして、これらの標準タイプを見ますと、リビングが必要以上に（？）
大きなサイズだったり、その一方で（生活空間として）やや小さいお部屋が
付いているように見えます。
建築の法律では、廊下の（最低の）横幅が決まっているそうですが、
以前に比べると、狭くなってしまったようです。
もちろん、これからの高齢化社会を見た場合は、本来ならば車椅子などが
通りやすいように、幅が広いほうが暮らしやすいです。
しかし、以前に読んだコラムには－
「廊下の幅を狭くして、部屋のサイズを広くしたい」要望が通ってしまった
のではないかと、ある建築家が書いていました。
これらの状態から見ますと、主に都市部では、標準的な間取りである、
３／４ＬＤＫの需要は少なくなっているのが現状ではないでしょうか。
そこで「子供が少なくても、対応できる間取り」と
「夫婦のみや一人暮らし」の方に余裕のある間取りを、一体化させて
みてはいかがでしょうか。

　〇　できるだけ廊下のスペースを作らない。
　〇　部屋のサイズに余裕のある「１／２ＬＤＫ」の間取りを作る。
　〇　余裕のある部屋は、いつでも「しっかりとした、仕切り」によって、
　　　部屋数が増やせる状態にする。

「しっかりとした、仕切り」は、ふすまや横開きのドアのような、
スライド式ではありません。
あらかじめ「複数の、お部屋」に変えられる「広い、お部屋」を用意して、
プライバシーが保てるようにします。
つまり、間取りの変更ができるリフォーム（？）が前提の状態です。
すると１つの間取りタイプでも、これからの多種多様な世帯に対応できる
のではないでしょうか。またリビングなどの、お部屋の採光が一面しか
取れない場所は、ベランダや採光窓を「真南」に向けます。
「南に窓があると、夏場は熱いのではないか」と思われるかもしれませんが、
夏場の太陽は、建物の真上を通ります。
すると、真南の窓は「光は多く入っても、直射日光は入りにくい」状態に
なりますし、冬場は、光が部屋の奥まで届くので、光熱費の節約になります。
最近の住宅環境は気密性／保温性が良いので、真南からの採光はメリットが
あるのではないでしょうか。そして一戸建てや角部屋の場合は、東からの採光
もあるので、朝から明るいスペースができます。

すると、このスペースを一日の始まりであるキッチン／ダイニングにするのも
良いのではないでしょうか。
このような特徴は、建物やマンションの「売り」になるかもしれません。
ちなみに玄関の照明は、タイマーなどで（人がいなくても）点灯できると、
防犯高価が高くなると思います。

　０１２　ネットカフェの安全性が高くなる方法
　－ － － － － － － － － － － － － － － － － － － － －
インターネットカフェは、小さな机が並んでいるオープンスペースから、
個室の場合など、店舗によって、さまざまな形態があります。
そして利用される方は、情報収集やインターネットゲームだけでなく、
寝泊まりする場合も少なくありません。
例えば完全な個室は、仮眠を取ったり朝まで過ごすのなら、ある程度の
安全性はあると思います。
ですが、ちょっとした「仕切り」や簡単なドアがあるだけなのに、
このような使い方をする方もいらっしゃるようです。
このようなタイプは、不正に利用されない面と、利用者のプライバシーが
保てる両面があるので、ごく普通に利用する場合は問題ありません。
ですが、終電に遅れて（始発まで）仮眠を取る使い方は、利用者が多ても、
安全性に問題があるのではないでしょうか。

　〇　部屋のドアや、出入り口の「仕切り」を開けると、
　　　小さな音が鳴るように、ドアの一部に鈴を取り付けます。

すると人の出入りが分かりますので、少しだけですが、安全性は高まると
思います。これは、小さな喫茶店やレストランの入り口のように、
周りの人が気付く程度で十分です。

　－ － － － － － － － － － － － － － － － － － － －
　０１３　使いやすいトップページのつくり方
　－ － － － － － － － － － － － － － － － － － － －
以前の携帯電話は、手に「すっぽり」収まるサイズや重さでした。
決して高機能ではありませんが、必要なだけの機能や使いやすいサイズの
魅力があったと思います。
そして現在でも、ベーシックな端末の需要はあるのではと思いましたが、
一部のＰＨＳは、シンプルな機能やサイズを売り物にしています。
ところで現在の携帯やスマートフォンは、非常に高解像度になりました。
それに伴って、トップページのアイコンは増えたのですが、以前よりも
決定する（見つけてタップする）までの時間が長くなっているのでは
ないでしょうか。通常なら、同一画面の選択肢が増えているので、
探している機能は、より早く使えます。
ですがアイコンの数の多さもあってか、使いたい機能を「探す」時間が
長くなっているのは、トップページに理由があるのではないでしょうか。

　〇　トップページのアイコンは「中心になっている、アイコン」
　　　がある配置にします。

コンパクトな形状や、液晶画面がＱＶＧＡ、ＶＧＡ表示の頃の端末は、
トップページのアイコンが、９個や１２個の表示ではないでしょうか。
そして画面を見ますと、中央にアイコンがあったり、並びが３×３なので、
中心のアイコンがあります。
そして押しやすいので、多用する機能がすぐに呼び出せる状態でした。
ところが現在の高解像度の画面では、中心にアイコンが置かれない場合が
ほとんどのようです。つまりアイコンの（縦や横の）並びが偶数になので、

より体感的で面白さが向上するのではないでしょうか。
そして車体がコース上から外れたときは、その車体が「画面の、端に」
移動することでも、体感的な楽しさが出てくると思います。

　〇　人が平行に「並ばない」シート配列にします。

つまり運転席と助手席が、平行になっていないシート配置にします。
この状態は、一人用のシートを単に、ずらして配置するわけではありません。

気にならない状態になりましたが、ここでは下半身の空いている場所を
「詰める」ようにします。
このようなシート配列は、二人乗りやクーペタイプでは適していません。
ですが並んで座るシートよりも横幅を抑えられるので、コンパクトで
デザイン性の高い車や、安全性の高い車ができるのではないでしょうか。

少ないです。それならパーソナル性を重視して、「３人が」余裕で乗れる
タイプや、デザイン、趣味性の高いコンパクトな車を作るのも良いのでは
ないでしょうか。

思い切って、一部の小型車やセダンの後席は「二人乗り」にして

います。

近づける方法です。つまり① とは逆に「スピーカーを、動かして」よい音質が
聴けるようにします。ここで、スピーカーの位置を上げる必要がある場合は－

そこで、この状態や脱水直後に、アイロンを当ててはいかがでしょうか。
すると洗濯機の機能を利用した、効率的な作業になります。
「濡れた衣類では、アイロンが当てにくい」と思われるかもしれません。
そこでアイロンをかける時の、ちょっとした方法をご紹介します。

　　「グローブのような」アイロン台（？）を用意します。

　　　アイロンを当てます。

収納すると、衣類の移し替えがないので、時間が節約できるのでは

　０３９　頭と体の使い方
　－ － － － － － － － － － －
「年を取ったら、ぼーっとするので、認知力が下がるのは当たり前」
「年を取ったら、体力が落ちるのは当たり前」と思われている方が
いるようです。
そして現状を話して、なぜか納得してしまう方もいらっしゃいます。
（何の問題解決にもなっていません）
人間の脳は、他の生物よりも「知識を蓄えて、新しい何かを作る」
ことが出来ます。
そして、ここから「先を、考える」ことも出来てしまいます。
これは、素晴らしい能力である反面「先を、考えられる」ので、
悩みも出来てしまいます。
ですが、能力を生かすほうに目を向けて－
「認知力や体力が落ちないようにするには、どうすればよいか」を
考えますと、「当たり前のこと」が、そうでなくなるかもしれません。

　〇　意識的に「頭を、使う時間」や「体を、使う時間」
　　　を作ります。

人間の五感は、意識によって「受動的な状態」と「能動的な、状態」が
あると思います。
そこで出来るだけ能動的な状態にすると、体力の向上や、能力の低下を
遅らせる可能性があるのではないでしょうか。
例えば、高齢な指揮者さんもいらっしゃいますが、これらの方が
「耳が、遠い」状態とは、考えにくいのではないでしょうか。
これは、たくさんの音を「能動的に」聴く状態なので、耳が鍛えられ、
能力を維持、向上しているのではないかと思いました。
そして職人さんも、経験を積むほど、腕や技術が上がっているので
「老化＝退化」とは、逆の状態になっています。

身近なところでは、ほとんどの方は、階段で「無意識的に」足を動かして
います。これは良い意味での「慣れ」から、動作がスムーズな面もありますが、
残念ながら、この状態は一生続くわけではありません。
体力は、成長期には（自然に）上がりますが、その後は下がってしまう
からです。そして、その結果が「最近は、足がもつれやすくなった」
「以前に比べて、体力が落ちた」と感じる状態かもしれません。
そこで「足を意識的に使う状態＝能動的な状態」にしますと、転びにくく
なったり、筋力の維持になるのではないかと思いました。
以前に見たテレビでは、筋肉の老化は「量」が減少するだけだそうです。
そして量を増やすことは、いくつになっても可能だと紹介していました。
すると意識的に筋肉を使って増やすことは、良い状態を維持できるのでは
ないかと思います。

　－ － － － － － － － － － － － － － － － － － － － －

　〇　室内が３５度以上になったら、設定温度を「３０度」にして

人間の体温は、就寝時には「２度ほど、下がる」と聞いたことがあります。
夏場に売られる涼感グッズは、周りの温度よりも「２度低く、感じる」と
宣伝しています。すると人間は、数度の温度差を感じた時に－
「暑さや、寒さの印象が変わる」と言えるのではないでしょうか。
そこで、エアコンを入れるのは、お部屋の温度が我慢できなくなってからに
します。それまでは、自然の風や扇風機で過ごします。
そして体が汗ばんできたら、涼しさを感じる温度になっている３０度にして
エアコンを使います。
すると２８度以上でも十分に涼しく感じるので、電気代の節約になります。
残念ながら外気の温度が下がると、高い温度設定のままでは、相対的に
暑くなってしまいます。そこで「暑さ」を感じるようになったら、
温度設定は下げますが、２８度以上は維持します。

実際に試したところ、夕方までは３０度でも「暑くない」と感じられました。
ちなみに就寝時の「体温の低下」を考慮しますと、夜中にエアコンが切れる
ようにセットしたり「暑くない」程度の温度のほうが、体調管理には良い
のではないでしょうか。

　０５９　笑いの作り方
　－ － － － － － － － － －
人が笑っている状態を文字で表すと、多くの方は－
「ワッ（アッ）ハッ、ハッ、ハ～」をイメージするのではないでしょうか。
この状態には、ちゃんとした意味があるそうです。
この「ワッ、ハッハッハ～」は「驚き、安心」の順序になっています。
（「モテたい脳、モテない脳」より）
ほとんどの動物は、生き残りのために「驚き＝生活上の危機感」です。
ですから最初の「アッ」（驚き）の後は、危機に対応するだけの行為です。
しかし人間は、生活、生存が脅かされない環境を作ったので、余裕が生まれ、
後からの安心感（ハッ、ハッ、ハ～）が続く、感情の組み合わせ（？）が
生まれたようです。
これは、人類だけが持つ文化と言えるのかもしれません。
すると飼い主さんがペットを驚かすと、ペットは安心感を持つどころか、
警戒するだけかもしれません。
彼らの脳では、安心するまでの判断ができない可能性が高いからです。

－ お笑いの「ツッコミ」のタイミングは－

ところで、テレビのお笑い番組を見ますと－
「この人のツッコミ（のタイミング）は、あの先輩芸人さんと似ている」
と思うことがあります。
これは先輩から研究して、何らかのコツを掴んだのだと思いますが、
技を盗むことが出来るなら、かならず天性の才能が必要なわけではない
表れではないでしょうか。
つまり、ある技術に気が付くかどうかの状態になっています。
ここでは受け継がれる、芸人さんの「ツッコミ」（のタイミング）を
ご紹介したいと思います。

　〇　ボケる側のセリフの「一番最後の、言葉」と、ツッコミ側の
　　　セリフの「一番最初の、言葉」を重ね合わせるようにします。

これは２人組のトークや漫才、相手のセリフが分かっている場合です。
例えばボケ役が「まるで〇〇みたい。」のセリフに対して、ツッコミ役の
セリフは「やかましい！」だったとします。
この時は「い。」を言った瞬間に、「やかましい！」の「や！」を重ねます

－ なぜ「ツッコミ」で、笑いが生まれるのか？－

何故このようなタイミングで、笑いが生まれる（多分）のでしょうか。
これはツッコミの言葉を重ねても、結果的には「重ならない」ことに
理由があります。
「重ね合わせるのに、重ならない？」と不思議に思われるかもしれませんが、
たとえツッコミを重ねても、実際は、若干の時差が生じてしまいます。
具体的には、ボケ側のセリフに対してタイミングを計っても、ツッコミの
脳内では「認識→（今だ！と思う）判断→行動（ツッコミ）」までの時間が
生じます。
すると、わずかですが時間的なタイミングの「ずれ」が発生します。
このような状態から、ツッコミ側のセリフはボケに重なりません。
そして、２つのセリフの間には、間（ま）の無い状態という「間」が生まれ
ます（ややこしい）
関西以外の芸人さんが行ないがちな「間が、無い」ではありません。
そして最初のボケと、後からのツッコミ（否定した言い方）は、繋がって
一体化してしまいます。

すると一体の言葉の流れなのに、結局は否定した状態が、視聴者などの
瞬時の（脳内の）判断と重なって「まさに、その通り！」という
代弁的な笑いが生まれるのではないでしょうか。
ちなみに、一人で行う「ボケツッコミやノリツッコミ（一緒？）」も、
同様の形だと思います。
また「まるで〇〇みたい」での笑いは、後からの「気付き」で生まれる
笑い（面白さ）ではないでしょうか。

　－ 「ツッコミ」が上手くなるには－

もちろん「ツッコミ」のタイミングは、これだけではありません。

先程は間の無い「間」をご紹介しましたが、実際に間がある場合も
あります。例えば「あ、」「いや、」「って、」の間作り（？）や、
溜めを作ることなどです。つまり、さまざまなパターンがあります。
それでは、これらを使った「ツッコミ」は、どうすれば上手くなるので
しょうか。
まず最初に、複数のツッコミのタイミングを、完璧に覚えてしまいます。
そしてネタやコントなどの、セリフが分かっている時は、「笑い」に
効果的なタイミングとセリフを選んでおきます。
（芸人さんがどう見られているか、目の前にお客さんがいるか、
お客さんの年代や、その場の空気によって変わると思います）

その後は相手のトークに対して、一番笑いの取れるタイミングを使います。
例えばセリフのないトーク番組では、相手が何を言うのか分からないにも
関わらず、ツッコミの上手い芸人さんは「笑い」が取れてしまいます。
これは話の予測や、その場の空気を読んで「笑い」の取れる間を、
瞬時に選択、判断しているのではないでしょうか。
そしてキャリアを積んでいくと、自然な（感覚的な）突っ込みが出来る
ようになるのかもしれません。

　○　「｜」のカーソルを文字の端に置いたら「空いていない」側の

有効に使えます。

　０９３　「〇無し、じゃんけんポン！」で勝つ方法
　－ － － － － － － － － － － － － － － － － － － － － － －
子供の頃の遊びは、地域によって、名前やルールが微妙に違います。
例えば「じゃんけん」でも、呼び名が違う地域もあるでしょうし、
チョキではなくてピーとも呼ばれています。
ところで、ある程度の人数から（誰かを）選ぶときは、じゃんけんに
参加しない人が「〇無し、じゃんけんポン！」と言って、二者択一で
行なう方法をご存じの方もいらっしゃるのではないでしょうか。
（ここでの「〇」は、グー、チョキ、パーのどれかになります）
これは、残された（２つの）選択肢から、素早く「勝てる、手」を決める

実は、この時に勝ちやすくなる方法がありますので、ご紹介したいと
思います。

　〇　「チョキ無し・・」と聞いたら、反射的に「パー」を出します。

います。

なりますが、上記のように「〇無し、・・」と言われたときには、勝てる
「手」を考えません。
この時に、並びの「右隣」を、反射的に選んでしまいます。
すると、上記の状態になります。
このようにしますと、わざわざ「勝つ、手」を考えなくても、素早く
「勝てる、手」が判断できるので、これを覚えると（間違えない限り）
負けることはありません。
ただし、他の参加者も知っていたら「相手の、ミス待ち」になるので、
勝敗はミス待ちになってしまいます。
お子様との「遊び」や、ちょっとした決めごとで使えそうでしたら
お試し下さい。

　０９６　英会話の「最初の、ひとこと」が出やすくなる方法
　－ － － － － － － － － － － － － － － － － － － － － － － － － － －

　　　を使いましょう。

と書かれていると思います。
これは、日本語の「すみませんが／失礼ですが」に当てはまりますが、
この一言が無いと、印象は大きく変わるので、重要だと書かれています。
すると、例えば道を聞くときには「ｅｘｃｕｓｅ　ｍｅ、」と話しかける
ことになりますが、英語が母国語でない場合は、言い出しにくい面が
あるかもしれません。

ご紹介したいと思います。

　〇　先に「日本語で話して」、後から「ｅｘｃｕｓｅ　ｍｅ」
　　　を続けます。

「Ｈｅｌｌｏ！コンニチハ！」と話している場面から思い付きました。
具体的には、最初に話すときに「すみません、ｅｘｃｕｓｅ　ｍｅ、」と
言ってしまい、相手からの返答を待ちます。すると、たいていは－
「最初に、何か言ったけど『ｅｘｃｕｓｅ　ｍｅ』だろうな」と、相手が
判断して、その後は、普通に話せると思います。
つまり「スミマセン、」は、相手に話す「最初の、きっかけ」として
割り切ると、「ｅｘｃｕｓｅ　ｍｅ」よりも、話しやすくなります。

ちなみに海外の飲食店などで、日本にいる感覚のまま「コーヒー」と
言ってオーダーすると、たいへんに失礼な「言い方」になるそうです。
お店の方に「日本人は、失礼なお客が多い」と思われないように－
「ｐｌｅａｓｅ」を添えましょうと、英会話書籍に書かれていました。
これは日本語の感覚で「～を下さい」を言わない程度になりますが、
英語圏では、必須の一言になっています。

　０９８　色々と思い付く方法
　－ － － － － － － － － － － － －
他の書籍を書いていた時のことです。
とにかく量が足りなかったので、日常で思い付いたことを、できるだけ
書き留めるようにしていました。
これは書籍の内容を、何処からか「持ってきてる」と思われない為でも
ありますが、気が付けば「何か、思い付かないかな」と考えるクセが
付きました。
そして書籍が完成した後も、思い付いたことを書き集めたのが、本書に
になっています。
このように素人の思い付きでも、ある程度の量になってしまいます。
するとアイデアを出すことは、それほど難しいことではないと言える
のではないでしょうか。
「それなら常に考えていれば、誰でも思い付くのか」と思われるかも
しれません。
これまで読んだ実用書やビジネス書は、似たような内容の思い付き方が
書かれていました。大まかな「流れ」は、以下になります。

　① 　普段から「多くの、モノ」に関心を持つ。
　② 　アイデアや「何か」を思い浮かべることを意識する。
　③ 　「何か」思い浮かんだら、すぐに記録する。

上記の内容が、複数の書籍に載っているということは－
「どの書籍（書かれた方）が、オリジナルか？」ではなくて、アイデアを
出す定番の方法になっています。
そこで、この方法を、もう少し詳しくご紹介したいと思います。
まず① のために、無理に「何か」を行なう必要はありません。
例えば日常生活で見かけた、関心のある情報があったら、自然に情報収集
になります（記事や内容が、かならず必要になるとは限りません）
また情報収集のために書籍を選ぶときでも、単に興味を覚えた内容で
構いません。
そのほうが、ずっと読みやすいですし、読んでいる時に別のアイデアが
思いつくことも多いです。

② は、考える時間を持ちつつも（作るではありません）アイデアが浮かぶ
のを、漠然と待つ程度にします。
最後の③ は、携帯電話やスマートフォンが使えます。
特にスマホは、リアルタイムの情報収集や、遊ぶ面ばかり注目されますが、
思いついたアイデアを（その場で）記録すると、より使えます。
ちなみに、思い付いた内容は「仕事や学習、自身に役立つか？」の基準は
必要ありません。
このような意識が「どうせ無駄／役に立たない」などの、否定的な感情に
なって、思い付きを制限してしまうからです。
たとえ自身の役に立たなくても、何らかの形で公開すれば、誰かに役立つ
かもしれません。

ビジネス書や実用書を見ますと、内容が充実している半面で、悪循環になる
可能性もあるのではないでしょうか。

　〇「書籍に方法が、いっぱい載っている」
　　　→「一体、どれから始めればよいのか分からない」
　　　→「一度に、こんなに出来るはずない」
　　　→「結局、どれも続かない」

このようになる場合は「１つでも試してみて、小さな結果を出す」のも
良いかもしれません。

　○　重曹水でうがい（お口くちゅくちゅ）する時に、頭を左右に
　　　傾けたり、前かがみや軽く上を向いた状態で行ないます。

通常のうがいをするときは、頭が前を向いたままの状態になります。
しかしこの状態では、歯の上部に重曹水がしっかり浸透しないのでは
ないかと思いました。
前を向いてるときは、お口の重曹水は下の歯だけ浸った状態だからです。
しかし、うがいをする時に頭の向きを変えると、あらゆる箇所に重曹水が
浸った状態になります。

具体的には、頭を左に傾けているときは左側だけくちゅくちゅ、右側に
傾けているときは右だけ－ という状態です。すると歯を満遍なく重曹水に
浸すので、重曹うがいが効果的になるのではないでしょうか。

　○　四角いタッパにゆで卵と少量の水を入れて、タッパを
　　　ぐるぐる振り回します。

するとタッパの側面とゆで卵の殻がぶつかって、面白いくらいに卵の殻が

　○　ゆで卵をあらかじめ何箇所か「割ってから」上記の方法を試します。

タッパは比較的柔らかいので、ゆで卵をぶつけるように振っても、最初はなかなか
割れないことがあります。あまり力を入れすぎると、ゆで卵の硬さが足りない
ときに卵がぐちゃぐちゃになったり、お子様が試すときに水がこぼれてしまうので、
最初からゆで卵を割っておきます。すると卵と殻の間に水が入りやすいので、
あとは軽く振るだけで卵の殻が取れます。

これは円筒形の（卵が割れにくそうな）タッパで卵が割れなかったときに
思い付きました。ちなみにゆで卵は、夏場にあまり使わない電気ポットに
水と卵を入れて沸騰（１００度に近ければＯＫ）その後３～４０分ほど放置して
おくと、適度な硬さのゆで卵が作れます（どこかで書いたかもしれません）
