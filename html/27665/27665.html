はじめに

この電子書籍をお買い上げくださりありがとうございます。私はもうすぐ結 婚 1 0 年目になる主婦です。私がこの本をかくきっかけになったのは結婚当初から結 婚 1 0 年目の記念日を憧れていたからです。結婚して間もなかった私に は 1 0 年後がはてしなく先のように思えました。そしてそこにはきらきらと輝くダイヤモンドがあるように思えたのです。（それをいうと夫の目は輝きを失せていましたが・・・）

でも実際結婚生活をスタートさせてみれば毎日あわただしく、日々のやりくり、生活費の節約、将来の貯金、など家計の余裕はなかったです。そして周りも結婚記念日 の C M にでてくるようなスイートテンダイヤを買ってもらった！

というかたに出会っていなくて一体どこにいるの？と疑問をもったからです。むしろ何もなかったという先輩方が多かったのです。

単調な日々の一大イベン ト 1 0 年目の結婚記念日・・・

何にもないだなんてそんなことがあっていいはずがありません。絶対にスルーしないようにしなければ！

それが本書の方法を用いることによってスルーでなく無理やりでもなく、むしろ夫が喜んでブランドの指輪を買ってくれるようになるのです。どこにそんなお金が？と思う人もいるかもしれません。

今の生活、貯金はそのまま で 1 0 年後には手にはいるのです。

この内容は私が日々感じて思ったことややってみたことリサーチをしてみたことから書きました。

あなたの結 婚 1 0 年目が素敵なものとなることが私の願いです。この本を読み終わった後からよきものになると信じています。

 

水木　れま

第１章     　結婚前に描いてい た 1 0 年目をイメージング！　　　　　

 

まずは、結婚前に思い描 く 1 0 年目をイメージングして整理してみることってとても大事だと思います。

"結 婚 1 0 年目・ ・ 1 0 年目のふし目に夜景が見えるレストランでダイヤモンドのサプライズ！ 「 1 0 年たつけどまるでかわらないね」と夫は妻に言う。二人の愛の証、スイートテンダイヤモンド・・・"

 

何 か C M でみて憧れていた一場面です。

夜景ということは 夜9 時ごろでしょうか。子供はどこかに預けているのかな、謎です。でもよく考えたら私、いきなりサプライズで好みじゃないデザインの指輪とか欲しくない・・。スイートテンダイヤモンドっておしゃれなデザインなのかどうかもわからない。むしろおしゃれじゃなくない・・・？ケーキの中からでてきたりしたらクリームでぐちゃぐちゃになるのでは。でもまあいいか、スーツとドレスアップ姿で日本人の庶民の感覚ではないけど何 か 1 0 年目結婚記念日って憧れる、ふぅ・・（フェイドアウト・・）

ではいけません！よく考えて大作戦を練りましょう。

そう、これは戦いなのです。

 

私の場合、整理すると・・・

 

1 0 年目の記念日は前から欲しかったブランド物のダイヤがついている指輪を買って夫と子供とで少しリッチなレストランでお食事をする。そのあときれいなかわいいケーキとお花を買っておうちに帰り楽しく過ごしたい。

そしてその指輪を眺めながら幸せな眠りにつきます。

 

買ってほしい指輪を詳しく考えると、大切なものだからすぐに飽きない物がいい。毎日つけるから洗練されたデザインがいい。夫の親に気兼ねなくつけていられるあまり派手じゃないものがいいけど、わかる人にはわかるいいものが欲しい 。 4 0 代 、 5 0 代になってもつけられるものがいい。

だからハイブランド物のいいものが欲しい。

そして友達に聞かれたら 「 1 0 年目の結婚記念日だから買ってもらっちゃった」と嫌味でない程度に言いたい。

 

そしてハタと気づくのです。ブランド物にダイヤ付きだとお値段がかなりのものになると。

下見をしたらダイヤ付きで小さいもので も 2 0 万 ～ 3 0 万しました。ダイヤなしなら、まだ高いけ ど 1 0 万弱位ならなんとか買えるかもしれない・・・でもダイヤがないと寂しい。年齢が重なると輝きをダイヤで埋めたい！

 

必須！！ダイヤ！！

 

でも今の生活でどうやって手にいれるのだろ？と思うわけです。でもどうやって？は一度置いておいて、

「絶対手に入れる！」と肝に銘じましょう。先に決めてしまいます。

第１０章　心も体もアンチエイジング

 

それから旦那さん に 1 0 年目の物買ってという攻撃をしなく

なったのですごく心がクリーンに保てます。

結婚して日々自由になるお金があまりない中で余裕がもてるってすごく精神的に良いです。

眉間のしわなどない感じなの で 1 0 年分アンチエイジング

です。

そして無理のない程度に月１万の節約を考えてみました。

●通信費のプロバイダーを安いところにかえる。

●ダイエット兼ねて自分のおかし、ケーキをやめる。

●ランチ・カフェを１０００円分やめてみる。

●外食の日を１回減らす。

●ショッピングで安い服１着やめる。

●雑魚のようなものを買わない。

●あまり買いだめしない。

 

もうひとつお勧めは百貨店の友の会制度です。

記念品を百貨店で買う場合ですが、月１万百貨店で貯金して

いくと１年後１２万が１３万円分の商品券としてもらえるという制度です。

１０年これをもし実行すればなんと１０万プラスですが

ブランド物を全部商品券でかうのは恥ずかしいので、いくらか商品券をつかうとお得です。

 

あと、お勧めはパーソナルカラーを調べてもらうといいとおもいます。パーソナルカラーとはその人が似合う色のことです。自分がとても輝く色味をしっておくとこの先の人生すごく節約につながります。余計な服やアクセサリーを買わずにすみます。

色にはイエローベースとブルーベースがあって、どちらに自分があっているかを知るだけでも似合うアクセサリーの地金の色味がわかります。

クローゼットが大幅に整理できてお部屋が美しく保てます。

ファッションもすごくまとまりやすくなって調和します。

色でお顔の色が生き生きと健康的にみえますのでアンチエイジングできます！

興味がある方はいろんな方がパーソナルカラー診断をされているので、診断されてみてはいかがでしょうか。

第１１章　思考はいつか運命になります。

 

私はずっと結婚記念に一方的に妻に贈られるのを夢見ていました。婚約指輪のときのように。

でも、１０年間夫婦どちらも同じ時間を共有してきたのに

なぜ妻にだけご褒美があるのでしょうか？

その思考はふとした時に言葉になり行動になりあたりまえという、感謝とは正反対のところにきてしまいます。

 

最近マザーテレサの言葉を読んではっとしました。

 

「思考に気をつけなさい、それはいつか言葉になるから」

「言葉に気をつけなさい、それはいつか行動になるから」

「行動に気をつけなさい、それはいつか習慣になるから」

「習慣に気をつけなさい、それはいつか性格になるから」

「性格に気をつけなさい、それはいつか運命になるから」

　　　　　　　　　　　　マザーテレサ

 

すべての始まりは思考なのですね。

そのことはすべてのことにあてはまります。些細な憧れや

一場面はいつの日か運命まで変貌していくのです。

いつも偏った考え方をしていないか日々チェックしたいです。いつまでも夫婦で仲良くバランスをとりながら

思考が混乱してきたらクリアにしようと意識して、素敵な

運命にしていきたいです。
